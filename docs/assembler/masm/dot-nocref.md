# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

禁止显示列表在符号表和浏览器文件的符号。  
  
## 语法  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## 备注  
 如果名称指定，则只能被禁止。  和 [.XCREF](../../assembler/masm/dot-xcref.md)相同。
