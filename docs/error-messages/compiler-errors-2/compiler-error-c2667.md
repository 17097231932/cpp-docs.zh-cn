# <a name="compiler-error-c2667"></a>编译器错误 C2667
function： 没有任何数字重载具有最佳转换  
  
 重载的函数调用不明确，无法解析。  
  
 必须严格优于所需的所有其他重载函数的转换需要用来匹配对其中一个重载函数的函数调用中的实际参数转换。  
  
 在函数模板的部分排序，请参阅知识库文章 Q240869 有关详细信息。
