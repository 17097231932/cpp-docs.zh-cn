# <a name="environmental-constants"></a>环境常量

## <a name="syntax"></a>语法

```
#include <stdlib.h>
```

## <a name="remarks"></a>备注

此常量定义字符串的环境长度。

|返回的常量|含义|
|--------------|-------------|
|`_MAX_ENV`|环境字符串的最大字符串大小。|
