# <a name="ltcstdboolgt"></a>&lt;cstdbool&gt;

包含标准 C 库标头 \<stdbool.h> 并将关联名称添加到 `std` 命名空间。

## <a name="syntax"></a>语法

```cpp
#include <cstdbool>

```

## <a name="remarks"></a>备注

包含该标头还将确保使用标准 C 库标头中的外部链接声明的名称在 `std` 命名空间中声明。
