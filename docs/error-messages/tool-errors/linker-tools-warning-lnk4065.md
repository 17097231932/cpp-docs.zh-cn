# 链接器工具警告 LNK4065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法对“function”进行排序；已将其忽略  
  
 给定函数没有编译为封装函数。  使用 [\/Gy](../../build/reference/gy-enable-function-level-linking.md) 重新编译。
