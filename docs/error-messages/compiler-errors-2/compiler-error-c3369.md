# <a name="compiler-error-c3369"></a>编译器错误 C3369

“模块名”: idl_module 已经定义

定义 DLL 的 [Idl_module](../../windows/attributes/idl-module.md) 使用只能在程序中出现一次。

以下示例生成 C3369:

```cpp
// C3369.cpp
// compile with: /c
[idl_module(name="name1", dllname="x.dll")]; // C3369
[idl_module(name="name1", dllname="x.dll")];
```
