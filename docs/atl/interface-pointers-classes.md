# Interface Pointers Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类管理特定接口指针:  
  
-   [CComPtr](../atl/reference/ccomptr-class.md) 执行自动引用计数。  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md) 类似于，`CComPtr`，还要执行自动查询接口。  
  
-   当构造一个数组COM接口指针时，[CInterfaceArray](../atl/reference/cinterfacearray-class.md) 提供有用的方法。  
  
-   当构造列表COM接口指针时，[CInterfaceList](../atl/reference/cinterfacelist-class.md) 提供有用的方法。  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md) 为相关的方法接口指针和全局接口表\(GIT）。
