# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第一个指令将行 *长度* 和过程中列出的 *字符宽度* 。  如果未提供参数，生成分页符。  第二个指令增加节号并重置分页到 1。  
  
## 语法  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
