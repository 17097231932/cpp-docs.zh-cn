# 编译器警告（等级 1）C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

行为更改: 使用显式模板参数导致调用“function”  
  
 如果尚未指定显式参数，则使用显式模板参数对函数进行的调用将调用预期之外的不同函数。
