# <a name="compiler-error-c2163"></a>编译器错误 C2163

function： 不可用作内部函数

`intrinsic`或`function`杂注列出了在内部形式中不可用的函数。 例如，编译以下位置使用 /clr 编程的程序时某些内部函数不可用。
