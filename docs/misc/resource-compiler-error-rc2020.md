# 资源编译器错误 RC2020
基项基数的位数非法  
  
 指定的 `digit` 不是 `radix` 所指定基项的有效数字。 数字或基数可能不正确。  
  
 八进制数字必须是从 0 到 7 的数字，十六进制数字必须是从 0 到 9 的数字或从 A 到 E 的字母。
