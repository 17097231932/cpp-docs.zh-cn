# <a name="walkthroughs-mfc"></a>演练 (MFC)

本节包含的文章为你演练与新的 MFC 库功能相关的各种任务。

## <a name="in-this-section"></a>本节内容

[演练：使用新的 MFC Shell 控件](../mfc/walkthrough-using-the-new-mfc-shell-controls.md)<br/>
在本演练中，你将创建一个类似于文件资源管理器的应用程序。 你将创建一个包含两个窗格的窗口。 左窗格中有一个 [CMFCShellTreeCtrl](../mfc/reference/cmfcshelltreectrl-class.md) 对象，该对象将在层次结构视图中显示您的桌面。 右窗格中有一个 [CMFCShellListCtrl](../mfc/reference/cmfcshelllistctrl-class.md) ，它将在左窗格中选择的文件夹中显示文件。

[演练：将控件置于工具栏上](../mfc/walkthrough-putting-controls-on-toolbars.md)<br/>
新式 MFC 工具栏除简单按钮外，可以承载控件。 本文介绍如何执行此操作。

[演练：向 MFC 项目添加 D2D 对象](../mfc/walkthrough-adding-a-d2d-object-to-an-mfc-project.md)<br/>
演示如何向项目添加 D2D 对象并介绍如何使用 D2D。

[演练：向 MFC 项目添加动画](../mfc/walkthrough-adding-animation-to-an-mfc-project.md)<br/>
演示如何将图形对象添加到项目并介绍如何对其进行动画处理。
