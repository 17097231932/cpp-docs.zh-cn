# <a name="resource-compiler-fatal-error-rc1022"></a>资源编译器错误 RC1022

应输入 "#endif"

`#if`未使用指令终止、 **#ifdef** 或 **#ifndef** 指令 `#endif` 。

在 `#if` 此语句之前，请确保有、 **#ifdef** 或 **#ifndef** 语句生效。
