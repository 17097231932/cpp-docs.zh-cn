# .WHILE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

生成执行 `statements` 块中的代码，当 `condition` 保持为 true 时。  
  
## 语法  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
