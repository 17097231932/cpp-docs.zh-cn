# 架构行集合
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

架构（或元数据）是任何关于数据库结构或组织的信息，如与提供程序、行集合、表、列有关的信息或除实际数据库内容之外的其他信息。  OLE DB 使您可以通过使用架构行集合获取此类信息而不必打开行集合。  
  
 有关如何使用架构行集合的信息，请参见[用架构行集合获取元数据](../../data/oledb/obtaining-metadata-with-schema-rowsets.md)。
