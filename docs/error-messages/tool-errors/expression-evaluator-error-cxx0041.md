# 表达式计算器错误 CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

类元素必须是静态成员或成员函数  
  
 在未指定要使用的类的实例化的情况下，使用了类（或结构或联合）的非静态成员。  
  
 在未指定实例化的情况下，只能使用静态数据成员或成员函数。  
  
 该错误与 CAN0041 相同。
