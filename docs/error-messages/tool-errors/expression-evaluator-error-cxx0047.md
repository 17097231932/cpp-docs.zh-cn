# <a name="expression-evaluator-error-cxx0047"></a>表达式计算器错误 CXX0047

参数列表不匹配函数

一个表达式，它调用具有未与任何函数的形参列表不匹配在程序中定义了一个同名的实际参数列表的函数。

仅当没有参数完全匹配或不需要构造对象的匹配项，可以调用重载的函数。

此错误是与 CAN0047 相同。
