# 资源编译器错误 RC1021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的预处理器命令“string”  
  
 数字记号 \(**\#**\) 后的字符未形成有效的预处理器指令。  
  
 此数字记号不能用作标识符中的第一个字符。
