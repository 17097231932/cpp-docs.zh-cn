# <a name="renaming-with-a-name-that-exists"></a>使用已存在的名称进行重命名

**ANSI 4.9.4.2** 使用新名称的文件在调用 rename 函数之前存在的效果

如果尝试使用现有的名称重命名文件，则 rename 函数将失败并返回错误代码。
