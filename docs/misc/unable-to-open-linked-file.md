# Unable to Open Linked File
为项目添加了一个文件作为链接文件。  但是，当打开该项目时，项目系统无法找到该文件。  
  
 **更正此错误**  
  
-   从解决方案资源管理器删除链接的文件。  
  
     \- 或 \-  
  
-   将该文件放回项目系统应可找到它的位置中。
