# <a name="compiler-warning-level-1-c4030"></a>编译器警告（等级 1）C4030
**第一个长度超过第二个列表的形式参数列表**  
  
 使用不同的正式参数重新声明了一个函数。 编译器使用在第一个声明中给出的正式参数。
