# 编译器错误 C2444
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：使用了 ANSI 原型，找到“type”，应输入“{”或“;”  
  
 函数原型后跟类型。  
  
 缺少分号或大括号可能会引起此错误。
