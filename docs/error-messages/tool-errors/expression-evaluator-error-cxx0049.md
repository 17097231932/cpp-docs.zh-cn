# <a name="expression-evaluator-error-cxx0049"></a>表达式计算器错误 CXX0049

已过时的 OMF-请重新链接程序

该程序使用了旧 OMF （对象模块格式）。

重新使用当前的链接器版本的程序的链接。

此错误是与 CAN0049 相同。
