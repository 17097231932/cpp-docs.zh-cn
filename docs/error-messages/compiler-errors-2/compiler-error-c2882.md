# 编译器错误 C2882
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“name”: 在表达式中非法使用命名空间标识符  
  
 尝试在表达式中使用命名空间的名称。  
  
 下面的示例生成 C2882：  
  
```  
// C2882.cpp  
// compile with: /c  
namespace A {  
   int k;  
}  
  
int i = A;   // C2882, can't assign A to i  
```
