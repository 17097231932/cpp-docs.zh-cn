# MSBuild 错误 MSB4138
**MSB4138: 在注册表位置“\<项\>”处指定了非字符串数据。**  
  
 在自定义工具集配置中指定的注册表位置无效。  
  
### 更正此错误  
  
-   确保工具集定义正确并且指定的注册表项存在。
