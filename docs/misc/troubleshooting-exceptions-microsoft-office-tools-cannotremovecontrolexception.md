# 关于异常的疑难解答：Microsoft.Office.Tools.CannotRemoveControlException
尝试以编程方式移除以非编程方式添加的控件时，将引发 <xref:Microsoft.Office.Tools.CannotRemoveControlException> 异常。  
  
## 相关提示  
 通过编程方式只能移除以编程方式添加的控件。  
 -   通过编程方式只能移除以编程方式添加的控件。 如果控件是在设计时而不是在运行时添加到文档的，则无法移除该控件。
