# <a name="substr"></a>SUBSTR
返回的子字符串*textitem*处开始，*位置*。 *Textitem*可以是文本字符串，前面是一个常数`%`，或通过宏函数返回的字符串。  
  
## <a name="syntax"></a>语法  
  
```  
  
name SUBSTR textitem, position [[, length]]  
```
