# <a name="compiler-error-c2227"></a>编译器错误 C2227
“->member”的左边必须指向类/结构/联合/泛型类型  
  
 `->` 左侧的操作数不是指向类、结构或联合的指针。  
  
 以下示例生成 C2227：  
  
```  
// C2227.cpp  
int *pInt;  
struct S {  
public:  
    int member;  
} s, *pS = &s;  
  
int main() {  
   pInt->member = 0;   // C2227 pInt points to an int  
   pS->member = 0;   // OK  
}  
```
