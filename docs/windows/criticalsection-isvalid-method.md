# CriticalSection::IsValid 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指示当前的临界是否有效。  
  
## 语法  
  
```  
bool IsValid() const;  
```  
  
## 返回值  
 默认情况下，总是返回`true`。  
  
## 要求  
 **标头：**corewrappers.h  
  
 **命名空间：**Microsoft::WRL::Wrappers
