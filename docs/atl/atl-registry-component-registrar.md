# <a name="atl-registry-component-registrar"></a>ATL 注册表组件（注册器）

ATL 注册器通过自定义接口提供对系统注册表的优化访问。 注册器是自由线程的，允许C++客户端的静态代码链接。

> [!NOTE]
> ATL 注册器的源代码可在 atlmfc_include_atliface.h 中找到。
