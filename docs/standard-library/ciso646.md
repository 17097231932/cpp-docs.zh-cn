# <a name="ltciso646gt"></a>&lt;ciso646&gt;

包括 C 标准库标头\<iso646 >, 并将关联名称添加`std`到命名空间。

> [!NOTE]
> 因为此兼容性标头定义的名称是C++中的关键字, 其中包括不起作用的名称。 中\< C++弃用了 iso646 > 标头。 在\<ciso646 c + + 20 标准版中删除了 > 标头。

## <a name="requirements"></a>要求

**标头:** \<ciso646 >

**命名空间：** std

## <a name="remarks"></a>备注

包括此标头可确保在`std`命名空间中声明使用 C 标准库标头中的外部链接声明的名称。
