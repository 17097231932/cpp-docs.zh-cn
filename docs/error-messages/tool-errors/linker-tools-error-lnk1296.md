# 链接器工具错误 LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法加载 filename  
  
 给定 DLL 不可用。  
  
 DLL 的可能位置包括当前目录、系统目录、Windows 目录以及在 PATH 环境变量中指定的目录。
