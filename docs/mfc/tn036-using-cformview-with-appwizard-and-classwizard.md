# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036：将 CFormView 与 AppWizard 和 ClassWizard 结合使用

本技术说明介绍了如何修改应用程序向导生成的应用程序，以便它使用 `CFormView` 而不是默认的 `CView` 作为其主视图类。 此版本的 Visual C++ 直接支持这种情况。
