# 数据绑定控件（ADO 和 RDO）
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

数据绑定是一种允许用户使用从数据库中检索的数据的机制。  
  
 Visual C\+\+ 通过 MFC 项目中的 ActiveX 控件提供可重用的数据绑定机制，该机制使您得以快速开发数据库应用程序。
