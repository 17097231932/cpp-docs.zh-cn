# operator CARRY?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

状态进位标志。  只能在 [.IF](../../assembler/masm/dot-if.md)、 [.WHILE](../../assembler/masm/dot-while.md)或 [.REPEAT](../../assembler/masm/dot-repeat.md) 块并评估了在运行时，不在汇编时。  
  
## 语法  
  
```  
  
CARRY?  
  
```
