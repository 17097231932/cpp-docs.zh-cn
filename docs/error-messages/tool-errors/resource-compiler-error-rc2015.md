# <a name="resource-compiler-error-rc2015"></a>资源编译器错误 RC2015

太多个字符的常量

字符常量包含两个以上的字符。

字符常量被限制为一个字符 （标准字符常数） 或两个字符 （长字符常数）。

请注意，转义序列 (例如，选项卡为 \t) 转换为单个字符。
