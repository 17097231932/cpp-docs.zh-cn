# 资源编译器错误 RC2164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

RCDATA 中的意外值  
  
 **RCDATA** 语句中的 *raw\-data* 值必须是整数或字符串，各个值之间用逗号分隔。  确保在字符串前后没有遗漏逗号或引号。
