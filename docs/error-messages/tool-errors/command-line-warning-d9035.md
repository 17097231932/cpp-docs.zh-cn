# <a name="command-line-warning-d9035"></a>命令行警告 D9035

选项 option 已弃用，并将在未来版本中删除

指定编译器选项，将在未来版本的 Visual c + + 编译器中删除。 此警告后跟警告[D9036](../../error-messages/tool-errors/command-line-warning-d9036.md)是否存在的建议的替换*选项*。

指定的选项仍有效，但应更新现在你的生成配置。 因此，你的项目是更有可能时升级编译器继续生成。
