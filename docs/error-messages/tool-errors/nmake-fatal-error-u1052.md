# <a name="nmake-fatal-error-u1052"></a>NMAKE 错误 U1052
文件 filename 找不到  
  
 NMAKE 找不到与以下项之一指定的文件：  
  
-   **/F**选项  
  
-   **!包括**预处理指令  
  
-   At 符号 (**@**) 响应文件说明符  
  
 检查文件存在以及的文件名拼写正确。
