# 程序终止
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 C\+\+ 中，可通过多种方式退出程序：  
  
-   调用 [exit](../cpp/exit-function.md) 函数。  
  
-   调用 [abort](../cpp/abort-function.md) 函数。  
  
-   从 **main** 执行 [return](../cpp/return-statement-in-program-termination-cpp.md) 语句。
