# <a name="numerical-limits-c"></a>数字限制 (C++)
这两个标准包含文件（LIMITS.H 和 FLOAT.H）定义了数字限制或给定类型的变量可包含的最小值和最大值。 确保这些最小值和最大值可移植到任何 c + + 编译器，使用相同的数据表示作为 ANSI c。限制。H 包括文件定义[对于整数类型的数字限制](../cpp/integer-limits.md)，和 FLOAT。H 定义[浮点类型的数字限制](../cpp/floating-limits.md)。
