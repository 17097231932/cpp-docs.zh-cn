# <a name="platformruntimeinteropservices-namespace"></a>Platform::Runtime::InteropServices 命名空间

此命名空间仅供内部使用，不用于开发。

## <a name="syntax"></a>语法

```cpp
namespace Platform {
   namespace InteropServices {
}}
```

虽然此命名空间供内部使用，但是浏览器可以显示此命名空间的以下成员。

|InterfaceTypeAttribute||

|StuctLayoutAttribute||
|UnmanagedType|枚举|
