# <a name="compiler-error-c2616"></a>编译器错误 C2616

conversion： 不能隐式转换不是 lvalue 的类型 type1 到 type2 不是常量

不能从非左值初始化引用。

这是 ANSI 兼容性错误 ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) 和 Microsoft 扩展下的一条警告 (**/Ze**)。
