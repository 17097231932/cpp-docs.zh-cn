# 编译器警告（等级 4）C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“type name”: 类型名超出了“limit”个字符的元数据限制  
  
 如果类型是托管类型，其名称将在元数据中被截断。  有关更多信息，请参见 [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md)。  
  
 默认情况下关闭此警告。  有关更多信息，请参见[默认情况下处于关闭状态的编译器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。
