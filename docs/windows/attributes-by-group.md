# Attributes by Group
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ 特性归类到以下功能组。  
  
|特性|描述|  
|--------|--------|  
|[COM Attributes](../windows/com-attributes.md)|插入代码支持 COM 开发和 .NET Framework 公共语言运行时开发许多范围。|  
|[IDL Attributes](../windows/idl-attributes.md)|启用 .idl 文件的修改从源代码文件，而不使用向导，也没有该文件结构和语法的知识。|  
|[OLE DB 使用者特性](../windows/ole-db-consumer-attributes.md)|inject 代码根据 [OLE DB 使用者模板](../data/oledb/ole-db-consumer-templates-reference.md)—创建执行任务 \(如打开表，执行命令和访问数据的工作 OLE DB 使用者。|  
|[编译器特性](../windows/compiler-attributes.md)|提供 Visual C\+\+ 编译器。|  
  
 有关如何定义自己的 .NET Framework 应用程序中的自定义特性的信息，请参见 [用户定义的特性](../windows/user-defined-attributes-cpp-component-extensions.md)。
