# <a name="active-document-classes"></a>活动文档类

活动文档可显示在 Web 浏览器的整个客户端窗口中，例如 Internet Explorer 5.5，或在支持活动文档的活动容器（如 Microsoft Office 联编程序）中显示。

[CDocObjectServer](reference/cdocobjectserver-class.md)<br/>
映射活动文档接口，并初始化和激活活动文档对象。

[CDocObjectServerItem](reference/cdocobjectserveritem-class.md)<br/>
专门为活动文档服务器实现 OLE 服务器谓词。

[COleDocObjectItem](reference/coledocobjectitem-class.md)<br/>
实现活动文档包容。
