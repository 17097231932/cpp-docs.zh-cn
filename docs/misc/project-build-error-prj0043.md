# 项目生成错误 PRJ0043
分配给文件“file”的自定义生成规则“rule”的“Outputs”属性是无效的。 该属性包含计算结果为“value”的“string”。  
  
 Outputs 属性的计算结果为空字符串。 自定义生成规则需要生成操作的输出。
