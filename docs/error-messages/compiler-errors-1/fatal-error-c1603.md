# <a name="fatal-error-c1603"></a>错误 C1603
内联程序集分支目标超出范围 number 个字节  
  
 大于 128 个字节之间 JCXZ 或 JECXZ 指令和其指定的目标标签的计算的距离。 更新你的代码，以便标签为更接近于指令。
