# 编译器警告（等级 1）C4402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

必须使用 PTR 运算符  
  
 在内联程序集代码中引用或转换为类型时，用在操作数上的类型没有 PTR 运算符。  
  
 编译器假定为 PTR 运算符。
