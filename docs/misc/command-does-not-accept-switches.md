# Command does not accept switches.
该错误通常发生在将开关提供给了仅识别参数的命令时。  
  
### 更正此错误  
  
1.  请再次阅读该命令的语法要求。  有关更多信息，请参见[带参数的 Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。
