# 针对数组类型的间接寻址
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

对 *n* 维数组类型使用间接寻址运算符 \(**\***\) 将生成 *n*–1 维数组。  如果 *n* 为 1，则将生成标量（或数组元素）。
