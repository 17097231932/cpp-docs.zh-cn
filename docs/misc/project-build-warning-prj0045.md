# 项目生成警告 PRJ0045
未设置分配给文件“file”的自定义生成规则“rule”的“Outputs”属性。 将跳过执行规则。  
  
 未针对自定义生成规则定义“Outputs”属性。 自定义生成规则需要生成操作的输出。
