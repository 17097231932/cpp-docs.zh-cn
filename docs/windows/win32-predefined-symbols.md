# <a name="win32-predefined-symbols"></a>Win32 预定义符号

在 Win32 头文件中定义了这些符号，并且它们支持标准的 Windows 应用程序功能和操作。 这些符号主要用于公共用户界面元素。 当您正在使用资源编辑器中的控件时，这些符号将出现在[属性窗口](/visualstudio/ide/reference/properties-window)与公共控件相关联。 例如，如果您的工具栏应显示应用程序图标，图标将与关联的符号 IDI_SMALL 中**属性**窗口。

|||
|-|-|
|IDABORT|（控件）对话框中，中止按钮|
|IDC_STATIC|（控件）在对话框中的静态文本|
|IDCANCEL|（控件）对话框中，取消按钮|
|IDD_ABOUTBOX|（对话框）产品关于对话框中|
|IDI_PROJECTNAME|（图标）当前项目图标|
|IDI_SMALL|（图标）当前项目小图标|
|IDIGNORE|（控件）用于对话框上的忽略按钮|
|IDM_ABOUT|（菜单项）用于帮助...有关...|
|IDM_EXIT|（菜单项）与文件一起使用...退出...|
|IDNO|（控件）对话框中，没有按钮|
|IDOK|（控件）对话框中，确定按钮|
|IDRETRY|（控件）对话框中，重试按钮|
|IDS_APP_TITLE|（字符串）当前应用程序名称|
|IDYES|（控件）对话框中，是按钮|

## <a name="requirements"></a>要求

Win32
