# 表达式计算器错误 CXX0038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

函数参数计数和\/或类型不匹配  
  
 指定的函数调用与该函数的原型不匹配。  
  
 使用正确的参数数目重新输入调用。  必要时转换每一个参数类型，以便与原型匹配。  
  
 该错误与 CAN0038 相同。
