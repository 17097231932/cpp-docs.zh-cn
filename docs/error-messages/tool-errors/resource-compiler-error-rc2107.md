# <a name="resource-compiler-error-rc2107"></a>资源编译器错误 RC2107

**应为数值命令值**

资源编译器需要**加速器**语句中的数字*idvalue*字段。 请确保已使用 `#define` 常量来指定值，并且常量拼写正确。
