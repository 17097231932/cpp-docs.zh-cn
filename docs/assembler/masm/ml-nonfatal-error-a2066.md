# <a name="ml-nonfatal-error-a2066"></a>ML 非致命错误 A2066

**不兼容的 CPU 模式和段大小**

尝试打开与段**USE16**， **USE32**，或**平面**属性是不兼容，指定的 CPU，或者更改为在 32 位的 16 位 CPU段。

**USE32**并**平面**属性必须加.386 或更高的处理器指令。
