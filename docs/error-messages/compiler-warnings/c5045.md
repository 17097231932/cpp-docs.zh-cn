# <a name="compiler-warning-c5045"></a>编译器警告 C5045

> 如果指定了/Qspectre 开关，编译器将为内存加载插入 Spectre 缓解措施

## <a name="remarks"></a>备注

警告 C5045 可让你查看在指定[/Qspectre](../../build/reference/qspectre.md)编译器选项时，你的代码中的哪些模式会导致 Spectre 缓解操作，例如 LFENCE。 这使你可以确定哪些代码文件受安全问题的影响。 此警告纯粹是信息性的：在使用 **/Qspectre**开关重新编译之前，不会插入缓解措施。 C5045 的功能与 **/Qspectre**开关无关，因此你可以在同一编译中使用它们。

此警告是 Visual Studio 2017 版本15.7 中的新增功能，默认情况下处于关闭状态。 使用[/Wall](../../build/reference/compiler-option-warning-level.md)启用默认情况下处于关闭状态的所有警告，或 __/w__*n*__5045__ ，以将 C5045 启用为级别*n*警告。 在 IDE 中，默认警告等级为 **/W3** ，可以在 "项目**属性页**" 对话框中启用此警告。 打开 "**配置属性**  >  " "**c/c + +**  >  **命令行**"，然后在 "**附加选项**" 框中添加 */w35045*，然后选择 **"确定"**。 有关详细信息，请参阅[默认情况下处于关闭状态的编译器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。 有关如何按编译器版本禁用警告的信息，请参阅编译器[警告（按编译器版本](compiler-warnings-by-compiler-version.md)）。

## <a name="example"></a>示例

下面的示例在 Visual Studio 2017 版本15.7 与 **/Wall**或 **/w35045**和 **/W3**选项一起编译时引发警告 C5045：

```cpp
// C5045.cpp
// Compile with: cl /EHsc /W3 /w35045 C5045.cpp

int G, G1, G2;

__forceinline
int * bar(int **p, int i)
{
    return p[i];
}

__forceinline
void bar1(int ** p, int i)
{
    if (i < G1) {
        auto x = p[i]; // C5045: mitigation here
        G = *x;
    }
}

__forceinline
void foo(int * p)
{
    G = *p;
}

void baz(int ** p, int i)
{
    if (i < G1) {
        foo(bar(p, i + G2));
    }
    bar1(p, i);
}

int main() { }
```

启用警告时，编译器输出如下所示：

```Output
C:\Users\username\source\repos\C5045>cl /W3 /w35045 C5045.cpp
Microsoft (R) C/C++ Optimizing Compiler Version 19.14.26431 for x86
Copyright (C) Microsoft Corporation.  All rights reserved.

C5045.cpp
c:\users\username\source\repos\c5045\c5045.cpp(16) : warning C5045: Compiler will insert Spectre mitigation for memory load if /Qspectre switch specified
c:\users\username\source\repos\c5045\c5045.cpp(15) : note: index 'i' range checked by comparison on this line
c:\users\username\source\repos\c5045\c5045.cpp(17) : note: feeds memory load on this line
Microsoft (R) Incremental Linker Version 14.14.26431.0
Copyright (C) Microsoft Corporation.  All rights reserved.

/out:C5045.exe
C5045.obj
```

警告消息显示已在第16行插入缓解措施。 它还注意到需要缓解，因为第15行的索引 i 在第17行的内存负载为源。 推理是在条形和 bar1 上完成的，但缓解在置于第16行时有效。
