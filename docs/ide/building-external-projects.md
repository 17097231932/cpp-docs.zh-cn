# 生成外部项目
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

外部项目是一个 Visual C\+\+ 项目，它使用 Visual C\+\+ 开发环境以外（外部）的生成文件或其他功能。  
  
 如果您有一个要在 Visual C\+\+ 开发环境中生成的外部项目（如生成文件项目），可创建一个生成文件项目，并在生成文件应用程序向导中指定项目的生成命令和输出。  有关更多信息，请参见[创建生成文件项目](../ide/creating-a-makefile-project.md)。  
  
 请注意，Visual C\+\+ 不再支持从开发环境中导出活动项目的生成文件的功能。  在命令行使用 [Devenv 命令行开关](../Topic/Devenv%20Command%20Line%20Switches.md)生成 Visual Studio 项目。
