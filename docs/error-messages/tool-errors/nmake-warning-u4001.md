# <a name="nmake-warning-u4001"></a>NMAKE 警告 U4001
可以仅从命令行调用命令文件  
  
 命令文件，所调用的 at 符号 (**@**) 说明符，不能包含另一个命令文件的说明。 不允许这样的嵌套。 忽略该规范。
