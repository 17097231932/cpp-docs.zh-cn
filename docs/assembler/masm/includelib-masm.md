# <a name="includelib-masm"></a>INCLUDELIB (MASM)
通知当前模块应与链接链接器*libraryname*。  
  
## <a name="syntax"></a>语法  
  
```  
  
INCLUDELIB libraryname  
```  
  
## <a name="remarks"></a>备注  
 *Libraryname*必须括在尖括号中，如果它包含一个反斜杠，分号，更高的比符号，较少-比符号、 一个单引号或双引号。
