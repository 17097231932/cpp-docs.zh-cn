# 声明符：最大数目
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.5.4** 可以修改算术、结构或联合类型的声明符的最大数量  
  
 Microsoft C 不限制声明符的数目。  该数字仅受可用内存限制。
