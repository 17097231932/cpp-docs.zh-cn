# Container Class::iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  本主题介绍 Visual C\+\+ 文档作为使用标准 C\+\+ 库的容器的非运行的示例。  有关更多信息，请参见 [STL 容器](../standard-library/stl-containers.md)。  
  
 描述可用作一迭代数。为控制序列的对象。  
  
## 语法  
  
```  
  
typedef T5 iterator;  
  
```  
  
## 备注  
 它介绍为此处未指定类型的 **T5**的同义词。  类型 **迭代器** 对象可以转换为对象类型。[const\_iterator](../standard-library/container-class-const-iterator.md)
