# <a name="tab-controls-and-property-sheets"></a>选项卡控件和属性表

属性表为多页对话框或 "选项卡对话框"，可向用户显示最多24个对话框模板资源。 有关属性表的示例，请参阅 Windows 显示属性对话框或以下 MFC 示例应用程序：

[CMNCTRL1：演示公共控件类，第1部分](../overview/visual-cpp-samples.md)

[CMNCTRL2：演示公共控件类，第2部分](../overview/visual-cpp-samples.md)

可以使用 MFC 类 [CPropertySheet](../mfc/reference/cpropertysheet-class.md)轻松实现属性表。
