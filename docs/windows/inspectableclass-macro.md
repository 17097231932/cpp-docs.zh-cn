# <a name="inspectableclass-macro"></a>InspectableClass 宏

设置运行时类名称和信任级别。

## <a name="syntax"></a>语法

```cpp
InspectableClass(
   runtimeClassName,
   trustLevel)  
```

### <a name="parameters"></a>参数

*runtimeClassName*  
运行时类的完整文本名称。

*trustLevel*  
之一[TrustLevel](http://msdn.microsoft.com/library/br224625.aspx)枚举值。

## <a name="remarks"></a>备注

**InspectableClass**宏可以仅可用于 Windows 运行时类型。

## <a name="requirements"></a>要求

**标头：** implements.h

**命名空间：** Microsoft::WRL
