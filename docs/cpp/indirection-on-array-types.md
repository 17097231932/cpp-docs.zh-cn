# <a name="indirection-on-array-types"></a>数组类型的间接寻址

使用间接寻址运算符 (<strong>\*</strong>) 上*n*-一维数组类型生成*n*-1 维数组。 如果*n*为 1，生成标量 （或数组元素）。
