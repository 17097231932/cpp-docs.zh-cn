# 错误 C1602
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支持的内部函数  
  
 该错误表示编译器 .exe 文件中存在不匹配，这可能是由于不完整的安装引起的。  例如，您可能安装了服务包，但没有安装处理器包。  安装所有必需的产品。
