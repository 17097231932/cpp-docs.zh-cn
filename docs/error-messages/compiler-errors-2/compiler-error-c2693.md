# <a name="compiler-error-c2693"></a>编译器错误 C2693

“运算符”：比较托管数组或 WinRT 数组的引用是非法的

你不能测试托管数组或 WinRT 数组的任何不相等情况。 例如，你可以测试托管数组是否相等，但不能测试其中一个数组是大于还是小于另一个数组。
