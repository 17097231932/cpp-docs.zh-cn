# BSCMAKE 错误 BK1512
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename：超出了容量  
  
 BSCMAKE 无法生成浏览信息文件，因为定义、引用、模块或其他信息量超过限制。  
  
### 通过使用下面可能的解决方案进行修复  
  
1.  使用 \/Em、\/Es 或 \/Ei 排除一些信息。  
  
2.  省略 \/Iu 选项。
