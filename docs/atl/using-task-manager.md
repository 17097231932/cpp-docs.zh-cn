# <a name="using-task-manager"></a>使用任务管理器

调试服务的最简单方法之一是使用任务管理器。 当服务正在运行时，启动任务管理器，然后单击 " **进程** " 选项卡。右键单击该 EXE 的名称，然后单击 " **调试**"。 这会启动附加到正在运行的进程 Visual C++。 现在，单击 "**调试**" 菜单上的 "**中断**"，允许您在代码中设置断点。 单击 " **运行** "，运行到选定的断点。
