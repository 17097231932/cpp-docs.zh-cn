# <a name="linker-tools-error-lnk1282"></a>链接器工具错误 LNK1282

无法/REBASE 文件;已对其进行签名

尝试使用 [editbin](../../build/reference/editbin-reference.md)的/REBASE 选项更改已签名程序集的基址。 为此，请先更改基址，然后对程序集进行签名。
