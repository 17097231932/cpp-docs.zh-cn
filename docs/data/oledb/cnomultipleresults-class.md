# <a name="cnomultipleresults-class"></a>CNoMultipleResults 类

用作[CCommand](../../data/oledb/ccommand-class.md)的模板参数（*TMultiple*），以创建处理单个结果集的优化命令。

## <a name="syntax"></a>语法

```cpp
class CNoMultipleResults
```

## <a name="remarks"></a>备注

如果需要命令处理多个结果集，请改用[CMultipleResults](../../data/oledb/cmultipleresults-class.md) 。

## <a name="requirements"></a>要求

**标头:** atldbcli.h
