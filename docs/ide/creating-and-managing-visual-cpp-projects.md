# 创建和管理 Visual C++ 项目
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可使用以下方式来创建 Visual C\+\+ 项目：选择“文件 &#124; 新建 &#124; 项目”，确保在左侧窗格中选中了 Visual C\+\+，然后从中间窗格的项目模板列表中进行选择。 单击模板时，在很多情况下会出现一个向导，让你能够在创建项目前设置各种选项。  
  
 你还可以使用以下方式来新建项目  
  
-   选择“文件 &#124; 新建 &#124; 现有代码中的项目”，然后根据提示添加现有源代码文件。 此选项最适合相对小型简单的项目，或许是具有少量或没有复杂依赖关系的 25 个或以下数量的源代码文件。  
  
-   以生成文件开始，选择“常规”选项卡下的“生成文件项目”模板。  
  
-   创建一个空项目（在“常规”选项卡下），然后通过在解决方案资源管理器中右键单击项目节点并选择“添加 &#124; 现有项”来手动添加源代码文件。  
  
-   使用 [Win32 应用程序向导](../windows/win32-application-wizard.md)。
