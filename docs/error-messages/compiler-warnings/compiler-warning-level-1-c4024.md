# 编译器警告（等级 1）C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 形参和实参“number”的类型不同  
  
 对应的实参与形参的类型不同。  编译器传递实参时未做更改。  接收函数将参数类型转换为预期的类型。
