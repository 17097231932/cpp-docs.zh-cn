# <a name="using-task-manager"></a>使用任务管理器
调试服务的最简单方法之一是通过使用任务管理器。 在运行时服务，启动任务管理器，然后单击**进程**选项卡。右键单击该 exe 文件的名称，然后单击**调试**。 这将启动 Visual c + + 附加到该正在运行的进程。 现在，单击**中断**上**调试**菜单以允许你在你的代码中设置断点。 单击**运行**运行到选定的断点。
