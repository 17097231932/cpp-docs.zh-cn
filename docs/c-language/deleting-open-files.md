# 删除打开的文件
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.4.1** remove 函数对打开的文件的效果  
  
 remove 函数用于删除文件。  如果文件处于打开状态，此函数将失败并返回 \-1。
