# 表达式计算器错误 CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

错误的基数  
  
 C 表达式计算器不识别指定基数。  只有十进制基数、十六进制基数和八进制基数有效。  
  
 该错误与 CAN0023 相同。
