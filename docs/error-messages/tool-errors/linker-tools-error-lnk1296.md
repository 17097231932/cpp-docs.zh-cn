# <a name="linker-tools-error-lnk1296"></a>链接器工具错误 LNK1296
无法加载 filename  
  
 给定的 DLL 是不可用。  
  
 中的 dll 可能的位置是当前目录、 系统目录、 Windows 目录和在 PATH 环境变量中指定的目录。
