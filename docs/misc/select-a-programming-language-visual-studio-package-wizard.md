# 选择编程语言、Visual Studio 包向导
此步骤能让你：  
  
-   选择你想要用于创建 VSPackage 的编程语言。  
  
-   生成或使用现有密钥文件对 VSPackage 程序集进行签名。  
  
 **Visual C\#**  
 在 C\# 中创建你的 VSPackage。  
  
 **Visual Basic**  
 在 Visual Basic 中创建你的 VSPackage。  
  
 **Visual C\+\+**  
 在 Visual C\+\+ 中创建你的 VSPackage。  
  
## 获取程序集签名密钥  
 **生成新密钥文件以对程序集进行签名**  
 将使向导为你生成密钥文件。 生成的密钥文件称为 key.snk。  
  
 **使用以下密钥文件对程序集进行签名**  
 允许你输入现有的密钥文件的路径或单击“浏览”来获取此文件。 该向导将生成文件的副本，并将其命名为 key.snk。  
  
 **取消**  
 取消创建 VSPackage。  
  
 **上一个**  
 移动到该向导的上一页。  
  
 **下一步**  
 移动到向导的下一页。  
  
 **完成**  
 创建 VSPackage。
