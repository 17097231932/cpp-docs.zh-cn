# 错误 C1109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 DLL“dll”中找不到“入口点”  
  
 在编译器所需的[延迟加载 DLL](../../build/reference/linker-support-for-delay-loaded-dlls.md) 中找不到入口点。
