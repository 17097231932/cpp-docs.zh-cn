# <a name="compiler-error-c2255"></a>编译器错误 C2255

“元素”: 不允许位于类定义之外

例如，非成员函数声明 `friend`。

下面的示例生成 C2255:

```
// C2255.cpp
// compile with: /c
class A {
private:
   void func1();
   friend void func2();
};

friend void func1() {}   // C2255
void func2(){}
```
