# 是否可以直接调用 DAO 或 ODBC？
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

照例在 MFC 中，如果需要更好的控制，则除了通过类访问 DAO 或 ODBC 外，还可以直接调用它们。  MFC 尝试简化 Windows 编程，但如果您需要访问基础 API，它还远离您的方式。
