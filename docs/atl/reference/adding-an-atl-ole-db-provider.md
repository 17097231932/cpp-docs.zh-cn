# 添加 ATL OLE DB 提供程序
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用此向导将 ATL OLE DB 提供程序添加到项目中。  ATL OLE DB 提供程序由数据源、会话、命令和行集合类组成。  项目必须已被创建为 ATL COM 应用程序。  
  
### 将 ATL OLE DB 提供程序添加到项目中  
  
1.  在“类视图”中，右击该项目。  在快捷菜单中单击**“添加”**，然后单击**“添加类”**。  
  
2.  在 Visual C\+\+ 文件夹中，双击**“ATL OLE DB 提供程序”**图标，或选择该图标并单击**“打开”**。  
  
     “ATL OLE DB 提供程序向导”打开。  
  
3.  按 [ATL OLE DB 提供程序向导](../../atl/reference/atl-ole-db-provider-wizard.md)中描述的那样定义设置。  
  
4.  单击**“完成”**关闭向导，这样将会在项目中插入新创建的 OLE DB 提供程序代码。
