# 浮点值的截断
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.2.1.4** 在某个浮点数转换为较小的浮点数时截断或舍入的方向  
  
 当出现下溢时，浮点变量的值将向下舍入为 0。  溢出可能导致运行时错误，也可能生成不可预知的值，具体取决于指定的优化。
