# <a name="compiler-error-c3818"></a>编译器错误 C3818

数组属性声明 property1 不会重载索引属性 property2

当其中一个是索引器，另一个是数组属性，是不可能的属性重载。

C3818 才可访问使用已过时的编译器选项 **/clr: oldsyntax**。
