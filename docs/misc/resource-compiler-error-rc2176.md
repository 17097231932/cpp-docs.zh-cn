# 资源编译器错误 RC2176
**文件名中存在旧 DIB。 通过 SDKPAINT 传递它。**  
  
 在指定文件中找到一张旧格式的设备独立位图。 应将其转换为当前格式。 可使用 Windows 3.0 SDK 中提供的 SDKPAINT 应用程序（或等效应用程序）执行此操作。
