# <a name="operator-ne"></a>运算符 NE
如果返回 true (-1) *expression1*不等于*expression2*，或者如果是这样，则返回 false (0)。  
  
## <a name="syntax"></a>语法  
  
```  
  
expression1 NE expression2  
```
