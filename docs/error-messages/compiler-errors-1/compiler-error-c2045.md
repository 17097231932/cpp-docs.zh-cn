# <a name="compiler-error-c2045"></a>编译器错误 C2045
“identifier”：标签已被重定义  
  
 标签出现在同一函数中的多个语句之前。  
  
 以下示例生成 C2045：  
  
```  
// C2045.cpp  
int main() {  
   label: {  
   }  
   goto label;  
   label: {}   // C2045  
}  
```
