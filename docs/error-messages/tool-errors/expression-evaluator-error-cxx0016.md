# 表达式计算器错误 CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

常数太大  
  
 C 表达式计算器无法接受大于 4,294,967,295（十六进制数为 0FFFFFFFF）的无符号整型常数，或绝对值约大于 1.8E\+308 的浮点常数。  
  
 该错误与 CAN0016 相同。
