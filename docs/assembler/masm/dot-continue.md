# <a name="continue-32-bit-masm"></a>.CONTINUE （32位 MASM）

生成要跳转到顶部的代码[。WHILE](dot-while.md)或[。](dot-repeat.md)如果*condition*为 TRUE，则重复 block。 （仅限32位 MASM。）

## <a name="syntax"></a>语法

> **.继续**⟦ **。如果***条件*⟧
