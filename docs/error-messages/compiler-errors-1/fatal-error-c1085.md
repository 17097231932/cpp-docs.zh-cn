# 错误 C1085
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法写入 filetype 文件:“file”: message  
  
### 通过检查以下可能的原因进行修复  
  
1.  驱动器是只读的。  
  
2.  驱动器已满。  
  
3.  共享冲突。  
  
4.  如果消息指出“文件号错误”，说明文件在后台进行编译的同时，可能已在前台关闭。
