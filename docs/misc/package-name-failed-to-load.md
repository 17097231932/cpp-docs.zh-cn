# Package &lt;name&gt; failed to load.
未能打开包。  该错误通常发生在未正确注册包、包缺少 DLL、DLL 已损坏或包要求的另一个组件中缺少 DLL 时。  
  
### 更正此错误  
  
1.  重新安装提供该包的产品。
