# <a name="resource-compiler-warning-rc4002"></a>资源编译器警告 RC4002

“identifier”宏的实参太多

使用给定的标识符指定的实参数量已大于给定标识符的宏定义中的正式参数的数目。

其他的实际参数已收集，但在宏扩展过程中忽略。
