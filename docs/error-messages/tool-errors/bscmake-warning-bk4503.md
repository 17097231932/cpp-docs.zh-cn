# BSCMAKE 警告 BK4503
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

已忽略 .SBR 文件的文件名中的小错误  
  
 此 .sbr 文件包含未暂停生成的错误，但生成的 .bsc 文件可能不正确。  若要解决此问题，请重新编译以创建新的 .sbr 文件。
