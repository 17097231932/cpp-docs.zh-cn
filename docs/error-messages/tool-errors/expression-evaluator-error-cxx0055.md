# <a name="expression-evaluator-error-cxx0055"></a>表达式计算器错误 CXX0055

上下文运算符中有不明确的符号

上下文运算符（ **{}** ）中的符号引用了程序中的多个符号。

范围解析运算符（`::`）可能能够解决歧义。

此错误与 CAN0055 相同。
