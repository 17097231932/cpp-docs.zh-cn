# 编译器警告（等级 4）C4672
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier1”：不明确。 首先被看作“identifier2”  
  
 要在 **try** 块中引发的指定对象不明确。 如果引发了该对象，则无法消除它的歧义。
