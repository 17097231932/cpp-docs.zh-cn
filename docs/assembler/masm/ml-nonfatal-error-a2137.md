# <a name="ml-nonfatal-error-a2137"></a>ML 非致命错误 A2137

**对 INVOKE 的参数太少**

使用传递的参数的数目[INVOKE](../../assembler/masm/invoke.md)指令是数少于所需的参数调用该过程的原型中指定的数。
