# <a name="linker-tools-error-lnk1166"></a>链接器工具错误 LNK1166

无法在 offset = offset，va = 值的情况下调整代码

LINK 无法根据需要填充代码。

某些处理器上不允许某些指令跨页面边界。 链接尝试添加填充来更正此情况。 在这种情况下，LINK 无法解决该问题。
