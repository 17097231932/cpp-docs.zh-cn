# <a name="ml-fatal-error-a1009"></a>ML 错误 A1009
**行太长**  
  
 源文件中的行超过 512 个字符的限制。  
  
 如果多个物理行与行继续符 (\) 进行连接时，生成逻辑行是仍只能进行 512 个字符。
