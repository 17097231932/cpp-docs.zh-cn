# Menu captions cannot contain separators.
该错误通常发生在菜单标题名称中使用了短划线字符 \(\-\) 时。  短划线字符被指定为菜单分隔符。  
  
### 更正此错误  
  
1.  为标题输入不使用短划线字符的新名称。
