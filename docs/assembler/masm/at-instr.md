# <a name="instr"></a>@InStr

找到的第一个匹配项的宏函数*string2*中*string1*，起点*位置*内*string1*。 如果*位置*未显示，在开头开始搜索*string1*。 返回一个位置的整数或 0，如果*string2*找不到。

## <a name="syntax"></a>语法

> @InStr（[[位置]]、 string1，string2）
