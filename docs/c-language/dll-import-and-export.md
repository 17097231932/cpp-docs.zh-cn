# <a name="dll-import-and-export"></a>DLL 导入和导出
**Microsoft 专用**  
  
 dllimport 和 dllexport 存储类修饰符是 C 语言的 Microsoft 专用扩展。 这些修饰符定义其客户端的 DLL 的接口（可执行文件或另一个 DLL）。 有关使用这些修饰符的具体信息，请参阅 [dllexport、dllimport](../cpp/dllexport-dllimport.md)。  
  
 **结束 Microsoft 专用**
