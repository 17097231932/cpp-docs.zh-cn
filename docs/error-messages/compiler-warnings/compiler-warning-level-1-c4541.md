# <a name="compiler-warning-level-1-c4541"></a>编译器警告（等级 1）C4541

使用/GR-的多态类型 "type" 中使用的 "identifier";可能导致不可预知的行为

您尝试使用需要运行时类型信息的功能，而不启用运行时类型信息。 用[/GR](../../build/reference/gr-enable-run-time-type-information.md)进行重新编译。
