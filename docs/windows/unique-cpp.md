# <a name="unique-c"></a>unique (C++)
指定一个唯一指针。  
  
## <a name="syntax"></a>语法  
  
```  
[unique]  
```  
  
## <a name="remarks"></a>备注  
 **唯一**c + + 属性具有相同的功能[唯一](http://msdn.microsoft.com/library/windows/desktop/aa367294)MIDL 特性。
