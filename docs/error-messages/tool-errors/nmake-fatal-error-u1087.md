# <a name="nmake-fatal-error-u1087"></a>NMAKE 错误 U1087

对于同一目标不能有：和：：从属项

不能同时在单冒号（ **：** ）和双冒号（`::`）依赖项中指定目标。

若要指定多个描述块中的目标，请在每个依赖项行中使用 `::`。
