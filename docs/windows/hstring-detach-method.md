# <a name="hstringdetach-method"></a>HString::Detach 方法
解除关联中其基础值的指定的 HString 对象。  
  
## <a name="syntax"></a>语法  
  
```  
HSTRING Detach() throw()  
```  
  
## <a name="return-value"></a>返回值  
 分离操作启动之前的基础 HString 值。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
