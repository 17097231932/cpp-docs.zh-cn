# named_guids
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 专用**  
  
 告知编译器定义和初始化旧样式或 **LIBID\_MyLib**、**CLSID\_MyCoClass**、**IID\_MyInterface** 和 **DIID\_MyDispInterface** 形式的 GUID 变量。  
  
## 语法  
  
```  
named_guids  
```  
  
## 备注  
 **结束 C\+\+ 专用**
