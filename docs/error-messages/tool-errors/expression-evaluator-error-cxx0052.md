# <a name="expression-evaluator-error-cxx0052"></a>表达式计算器错误 CXX0052

成员函数不存在

将成员函数指定为断点，但找不到该函数。 在已内联的函数中设置断点可能导致此错误。

使用内联强制关闭（/Ob0）重新编译文件，以便在此函数中设置断点。

表达式调用了未定义的函数。

此错误与 CAN0052 相同。
