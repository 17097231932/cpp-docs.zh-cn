# <a name="resource-compiler-error-rc2124"></a>资源编译器错误 RC2124
不允许空菜单  
  
 **结束**关键字出现在任何菜单项中进行定义之前**菜单**语句。 资源编译器不允许使用空的菜单。 请确保你没有任何打开用引号引起来**菜单**语句。
