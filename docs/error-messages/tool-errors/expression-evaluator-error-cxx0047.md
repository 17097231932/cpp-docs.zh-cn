# 表达式计算器错误 CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

参数列表与函数不匹配  
  
 表达式调用了一个带实参表的函数，该实参表与程序中定义的任何同名函数的形参表不匹配。  
  
 只有当参数完全匹配或匹配不要求构造对象时才可以调用重载函数。  
  
 该错误与 CAN0047 相同。
