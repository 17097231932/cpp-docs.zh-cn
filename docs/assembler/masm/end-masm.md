# <a name="end"></a>End

标记模块的末尾，还可以选择将程序入口点设置为*procId*。

## <a name="syntax"></a>语法

> **END** ⟦*procId*⟧

## <a name="remarks"></a>备注

*ProcId*参数仅在32位 MASM 中有效。
