# 编译器警告（等级 4）C4408
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

匿名结构或联合不声明任何数据成员  
  
 匿名结构或联合必须具有至少一个数据成员。  
  
 以下示例生成 C4408：  
  
```  
// C4408.cpp  
// compile with: /W4 /LD  
static union  
{  
   // int i;  
};  
// a named union can have no data members  
// } MyUnion;  
```
