# <a name="pushcontext"></a>PUSHCONTEXT
将保存部分或全部当前`context`： 段寄存器假定，基数值、 列表和 cref 标志或处理器/协处理器的值。 `context`可以是**假设**， `RADIX`，**列出**， **CPU**，或**所有**。  
  
## <a name="syntax"></a>语法  
  
```  
  
PUSHCONTEXT context  
```
