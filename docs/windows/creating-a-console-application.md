# <a name="creating-a-console-application"></a>创建控制台应用程序
创建 Win32 控制台应用程序的最简单方法是使用 Win32 应用程序向导。  
  
### <a name="to-create-a-win32-console-application"></a>创建 Win32 控制台应用程序  
  
1.  按照帮助主题中的说明[使用 Visual c + + 应用程序向导创建项目](../ide/creating-desktop-projects-by-using-application-wizards.md)。  
  
2.  在**新项目**对话框中，选择**Win32 控制台项目**在模板窗格中，以打开向导。  
  
3.  定义你[应用程序设置](../windows/application-settings-win-32-project-wizard.md)使用[Win32 应用程序向导](../windows/win32-application-wizard.md)。  
  
    > [!NOTE]
    >  跳过此步骤可保留向导的默认设置。  
  
4.  单击**完成**以关闭向导，新创建的项目将在中打开**解决方案资源管理器**。
