# 链接器工具错误 LNK1287
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

托管入口点函数无效  
  
 入口点对于托管映像无效。  托管入口点函数的返回类型仅可为 `void` 或 `int`。  托管入口点函数的参数类型仅可为 `void` 或 `String []`。
