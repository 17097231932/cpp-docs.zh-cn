# <a name="debugging-support-classes"></a>调试支持类
MFC 提供的以下类，以帮助你调试动态内存分配问题。  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 提供了诊断转储的目标。  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 结构，它提供的内存使用的快照。 此外用于比较早期及更高版本的内存快照。
