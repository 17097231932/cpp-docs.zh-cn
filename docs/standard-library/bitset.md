# <a name="ltbitsetgt"></a>&lt;bitset&gt;

定义用于表示和处理固定大小的位序列的类模板位组和两个支持模板函数。

## <a name="requirements"></a>要求

**标头：**\<bitset>

**命名空间:** std

## <a name="members"></a>成员

### <a name="operators"></a>运算符

|名称|说明|
|-|-|
|[运算符&](../standard-library/bitset-operators.md#op_amp)|执行两个位组间的按位“与”。|
|[运算符<\<](../standard-library/bitset-operators.md#op_lt_lt)|将位序列的文本表示形式插入标准输出流中。|
|[运算符>>](../standard-library/bitset-operators.md#op_gt_gt)|将位序列的文本表示形式插入标准输入流中。|
|[运算符 ^](../standard-library/bitset-operators.md#op_xor)|执行两个位组间的按位“异或”。|
|[运算符&#124;](../standard-library/bitset-operators.md#op_or)|执行两个位组间的按位“或”。|

### <a name="classes"></a>类

|“属性”|说明|
|-|-|
|[位组](../standard-library/bitset-class.md)|类模板描述了一种对象类型，该对象存储由固定数量的位构成的序列，这些位提供一种压缩方法来保留一组项或条件的标志。|
