# <a name="interfaces-atl-control-wizard"></a>接口，ATL 控件向导
在向导的此页将标识该控件支持的接口。 默认情况下，受支持的接口是通常使用的大多数容器。  
  
> [!NOTE]
>  如果你选择**最低限度的控制**上[选项](../../atl/reference/options-atl-control-wizard.md)选项卡上，在默认情况下会出现任何接口**支持**列表框。  
  
 **不支持**  
 指示当前不支持的控件的可用接口。  
  
 **支持**  
 指示当前支持的控件的接口。  
  
|传输按钮|描述|  
|---------------------|-----------------|  
|**>**|将添加到**支持**列表中当前选定的接口名称**不支持**列表。|  
|**>>**|将添加到**支持**列表中可用的所有接口名称**不支持**都列表。|  
|**<**|移除中当前选定的接口名称**支持**列表。|  
|**<<**|删除的所有接口名称中当前列出**支持**列表。|
