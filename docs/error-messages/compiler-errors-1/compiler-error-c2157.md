# <a name="compiler-error-c2157"></a>编译器错误 C2157
“function”：必须先声明才能用于杂注列表  
  
 在 [alloc_text](../../preprocessor/alloc-text.md) 杂注的函数列表中引用之前，未声明函数名。  
  
 下面的示例生成 C2157：  
  
```  
// C2157.cpp  
// compile with: /c  
#pragma alloc_text( "func", func)   // C2157  
  
// OK  
extern "C" void func();  
#pragma alloc_text( "func", func)  
```
