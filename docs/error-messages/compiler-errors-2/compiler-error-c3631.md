# <a name="compiler-error-c3631"></a>编译器错误 C3631

“函数”：无法重载托管事件或 WinRT 事件

无法重载托管事件或 WinRT 事件。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3631。
