# <a name="selecting-foreground-or-background-colors-image-editor-for-icons"></a>选择前景色或背景色（图标的图像编辑器）
除**橡皮擦**上的工具**图像编辑器**使用当前前景色或背景色时分别按左或向右鼠标按钮，工具栏绘图。  
  
### <a name="to-select-a-foreground-color"></a>若要选择前景色  
  
1.  用鼠标左键单击所需的颜色[调色板](../windows/colors-window-image-editor-for-icons.md)。  
  
### <a name="to-select-a-background-color"></a>若要选择背景色  
  
1.  用鼠标右键按钮，单击所需的颜色**颜色**调色板。
