# <a name="fatal-error-c1350"></a>错误 C1350
加载 dll dll 时出错： 没有找到 dll  
  
 找不到支持所尝试的操作的 DLL。  
  
 这表明你的安装问题，应从产品光盘重新安装该 DLL。
