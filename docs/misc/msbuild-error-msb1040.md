# MSBuild 错误 MSB1040
**MSB1040: ToolsVersion 无效。**  
  
 使用 **\/ToolsVersion** 开关指定的 `ToolsVersion` 值无效。  
  
### 更正此错误  
  
-   在命令行中使用 **\/ToolsVersion** 开关生成时，请为 `ToolsVersion` 指定一个有效值。
