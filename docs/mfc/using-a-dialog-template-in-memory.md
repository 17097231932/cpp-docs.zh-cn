不使用在[对话框创建表](../mfc/creating-a-dialog-class-with-code-wizards.md)中指定的方法，您可以从内存中的对话框模板间接创建这两种对话框。 有关详细信息，请参阅*MFC 参考*中的[CDialog](../mfc/reference/cdialog-class.md)类。
