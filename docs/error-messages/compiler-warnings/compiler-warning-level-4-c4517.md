# <a name="compiler-warning-level-4-c4517"></a>编译器警告（等级 4）C4517

不推荐使用访问声明;成员 using 声明提供更好的替代方法

ANSI C++委员会已声明访问声明（在不[使用 using](../../cpp/using-declaration.md)关键字的情况下更改派生类中成员的访问权限）已过时。 的C++未来版本可能不支持访问声明。
