# <a name="compiler-error-c2700"></a>编译器错误 C2700
“identifier”：无法引发（有关详细信息，请使用 /W4）  
  
 无法引发该对象。 使用 [/W4](../../build/reference/compiler-option-warning-level.md) 编译以获得详细诊断信息。
