# Character &lt;character&gt; is not valid for alias names.
该错误通常发生在输入了作为别名组成部分的不正确字符时。  别名不能包含空格、\=、\[、\]、\/、\- 或 "。  
  
### 更正此错误  
  
1.  重新输入别名并省略不正确字符。
