# <a name="expression-evaluator-error-cxx0063"></a>表达式计算器错误 CXX0063
不支持重载运算符 ->  
  
 表达式使用了重载的类成员访问运算符 (**->**)。  
  
 此错误是与 CAN0063 相同。
