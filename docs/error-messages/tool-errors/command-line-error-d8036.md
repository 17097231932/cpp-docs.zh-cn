# 命令行错误 D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不允许“\/option”有多个源文件  
  
 这些编译器选项不能用于多个源文件：  
  
-   命名程序集文件列表 \(\/Fa\)  
  
-   重命名对象文件 \(\/Fo\)  
  
-   创建不含局部变量的源浏览器信息 \(\/Fr\)  
  
-   创建包含局部变量的源浏览器信息 \(\/FR\)
