# <a name="union"></a>UNION

声明一个或多个数据类型的联合。 *字段声明*必须是有效的数据定义。 省略嵌套**联合**定义中的 "[结束](../../assembler/masm/ends-masm.md)*名称*" 标签。

## <a name="syntax"></a>语法

> *名称***联合**⟦*对齐*⟧⟦ __，__ 非**唯一**⟧ \
> *字段声明*\
> ⟦*name*⟧ **end**
