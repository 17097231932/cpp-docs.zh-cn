# <a name="compiler-warning-level-4-c4505"></a>编译器警告（等级 4）C4505
function： 已删除未引用的本地函数  
  
 给定的函数是模块的本地且不被引用中; 正文中因此，该函数是死代码。  
  
 编译器未生成此死函数代码。
