# 编译器错误 C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法更新程序数据库“file”  
  
 编译器无法写入数据库文件。  
  
### 修复此问题  
  
-   确保磁盘具有足够的可用空间。  
  
-   确保文件不是只读的。  
  
-   确保不共享的冲突。
