# 语句：针对 Switch 语句的限制
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** `switch` 语句中的 **case** 值的最大数量  
  
 Microsoft C 未限制 **switch** 语句中的 **case** 值的数量  该数字仅受可用内存限制。
