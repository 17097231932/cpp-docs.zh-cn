# Compiler Error C3456
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\[source\_annotation\_attribute\] 不可用于托管或 WinRT 类声明  
  
 source\_annotation\_attribute 用于定义将被代码分析使用的自定义属性。  
  
 只有当存在理解此属性的相应插件时，这些自定义属性才有用。
