# <a name="attribute-categories"></a>特性类别
属性引用描述各种类型的属性可以在 Visual c + + 中使用。
  
## <a name="related-sections"></a>相关章节
 [COM 特性](../windows/com-attributes.md)  
 将插入代码以支持 COM 开发的许多方面。
  
 [IDL 特性](../windows/idl-attributes.md)  
 插入代码的创建或修改从关联的.idl 文件中不使用向导，也不需要熟悉的结构和语法的该文件的源代码文件。
  
 [OLE DB 使用者特性](../windows/ole-db-consumer-attributes.md)  
 注入代码中，基于[OLE DB 使用者模板](../data/oledb/ole-db-consumer-templates-reference.md)，若要创建使用 OLE DB 使用者执行任务，例如打开表执行命令和访问数据。
  
 [编译器特性](../windows/compiler-attributes.md)  
 将插入代码来执行各种常见编程任务。
  
 [特性化的编程](../windows/attributed-programming-concepts.md)  
 提供指向描述特性化的编程的概念性主题的链接。
