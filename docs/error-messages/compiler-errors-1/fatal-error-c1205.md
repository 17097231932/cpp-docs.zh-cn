# <a name="fatal-error-c1205"></a>错误 C1205

安装的运行时版本不支持这些泛型

编译器所使用的公共语言运行时版本不是当前编译器支持的运行时版本。  例如，泛型功能要求与编译器匹配的运行时。

你的路径说明可能需要修改。
