# 错误 C1108
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找不到 DLL：“dll”  
  
 在路径中找不到指定的 DLL \(***dll***\)。  
  
 若要解决此错误，请重新安装 Visual C\+\+ 或将相应的 .dll 文件从安装 CD 复制到计算机。
