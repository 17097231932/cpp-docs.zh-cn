# <a name="command-line-warning-d9036"></a>命令行警告 D9036

> "*option\_2*" 而不是 "*option\_1*"

## <a name="remarks"></a>备注

警告 D9036 是[命令行警告 D9035](../../error-messages/tool-errors/command-line-warning-d9035.md)的延续。 如果首选编译器选项而不是已弃用的选项，则会在此处列出。 有关指向详细信息的链接，请参阅[已弃用的和已删除的编译器选项](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options)。
