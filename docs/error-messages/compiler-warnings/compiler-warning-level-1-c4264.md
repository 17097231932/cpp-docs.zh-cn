# 编译器警告（等级 1）C4264
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“virtual\_function”: 没有对基“class”中的虚拟成员函数可用的重写；函数被隐藏  
  
 C4264 总是在 [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md) 之后生成。  
  
 默认情况下关闭此警告。  有关更多信息，请参见[默认情况下处于关闭状态的编译器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。
