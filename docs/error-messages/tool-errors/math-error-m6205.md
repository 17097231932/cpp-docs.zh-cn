# <a name="math-error-m6205"></a>数学错误 M6205

"function"： _TLOSS 错误

出现总损失（精度）。

此错误的原因可能是将一个非常大的数字作为 sin、cos 或 tan 的操作数提供，因为操作数必须缩减为介于0到 2 * pi 之间的数字。
