# <a name="nmake-fatal-error-u1077"></a>NMAKE 错误 U1077

"program"：返回代码 "value"

NMAKE 调用的给定命令或程序失败并返回了给定的退出代码。

若要取消此错误并继续 NMAKE 会话，请使用/I 选项 **。忽略** 点指令或短划线 (**-**) 命令修饰符。 若要为依赖关系树的无关部分继续 NMAKE 会话，请使用/K 选项。
