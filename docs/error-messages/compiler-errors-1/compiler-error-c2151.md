# 编译器错误 C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

语言特性多于一个  
  
 函数有多个指定调用约定的关键字（`__cdecl`、`__stdcall` 或 `__fastcall`）。
