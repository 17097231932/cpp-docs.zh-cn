# 编译器错误 CS1555
无法找到为“Main”方法指定的“class”  
  
 已向 [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) 编译器选项指定了一个类，但无法在源代码中找到该类的名称。
