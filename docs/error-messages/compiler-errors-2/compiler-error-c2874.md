# <a name="compiler-error-c2874"></a>编译器错误 C2874
使用声明将导致 symbol 的多个声明  
  
 声明将导致定义两次对同一个项。  
  
 下面的示例生成 C2874:  
  
```  
// C2874.cpp  
namespace Z {  
   int i;  
}  
  
int main() {  
   int i;  
   using Z::i;   // C2874, i already declared  
}  
```
