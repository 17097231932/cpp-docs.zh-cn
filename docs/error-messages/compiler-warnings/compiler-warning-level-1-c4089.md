# <a name="compiler-warning-level-1-c4089"></a>编译器警告 （等级 1） C4089

function： 实际参数 number 形式参数 number 中的不同类型

相应的形式和实际参数具有不同的类型。 实际的参数被传递而无需更改。 该函数将转换为函数定义中指定的类型实参。
