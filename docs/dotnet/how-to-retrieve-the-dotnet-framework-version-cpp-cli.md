# <a name="how-to-retrieve-the-net-framework-version-ccli"></a>如何：检索 .NET Framework 的版本 (C++/CLI)
下面的代码示例演示如何确定与当前安装的.NET Framework 的版本<xref:System.Environment.Version%2A>属性，它是一个指针，到<xref:System.Version>包含版本信息的对象。  
  
## <a name="example"></a>示例  
  
```  
// dotnet_ver.cpp  
// compile with: /clr  
using namespace System;  
int main()   
{  
   Version^ version = Environment::Version;  
   if (version)  
   {  
      int build = version->Build;  
      int major = version->Major;  
      int minor = version->Minor;  
      int revision = Environment::Version->Revision;  
      Console::Write(".NET Framework version: ");  
      Console::WriteLine("{0}.{1}.{2}.{3}",   
            build, major, minor, revision);  
   }  
   return 0;  
}  
```
