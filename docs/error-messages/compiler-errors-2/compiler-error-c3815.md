# <a name="compiler-error-c3815"></a>编译器错误 C3815
方法 get_accessor 返回类型必须与 setter 的最后一个参数的类型匹配  
  
 声明属性的返回值时`get_accessor`方法必须与匹配的声明 set 访问器方法中的最后一个参数。  
  
 C3815 才可访问使用过时的编译器选项**/clr:oldSyntax**。
