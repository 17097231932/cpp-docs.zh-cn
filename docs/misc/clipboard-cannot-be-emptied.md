# Clipboard cannot be emptied.
该错误通常发生在已经复制项而无法将该项放在“剪贴板”上时，因为该“剪贴板”包含了不能移除的信息。  
  
### 更正此错误  
  
1.  关闭其他可能正在使用剪贴板的程序并重试。
