# <a name="drawing-tool-classes"></a>绘图工具类

这些类封装用于在设备上下文中绘制的绘图工具。

[CGdiObject](reference/cgdiobject-class.md)<br/>
GDI 绘图工具的基类。

[CBrush](reference/cbrush-class.md)<br/>
封装一个可以在设备上下文中选择为当前画笔的 GDI 画笔。 画笔用于填充要绘制的对象的内部。

[CPen](reference/cpen-class.md)<br/>
封装可在设备上下文中选择为当前笔的 GDI 笔。 笔用于绘制对象的边框线。

[CFont](reference/cfont-class.md)<br/>
封装可在设备上下文中选择为当前字体的 GDI 字体。

[CBitmap](reference/cbitmap-class.md)<br/>
封装 GDI 位图，提供用于操作位图的接口。

[CPalette](reference/cpalette-class.md)<br/>
封装 GDI 调色板，以用作应用程序与颜色输出设备（如显示）之间的接口。

[CRectTracker](reference/crecttracker-class.md)<br/>
显示并处理调整和移动矩形对象的用户界面。
