# MSBuild 错误 MSB1009
**项目文件不存在。**  
  
 项目文件在指定的目录中不存在；或者，在未指定目录的情况下，项目文件在当前的工作目录中不存在。  
  
### 更正此错误  
  
1.  检查文件名和路径的拼写是否正确。
