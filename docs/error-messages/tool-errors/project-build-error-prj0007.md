# <a name="project-build-error-prj0007"></a>项目生成错误 PRJ0007
无法创建输出目录目录。  
  
 Visual c + + 创建输出目录失败。 原因可能为：  
  
-   已用要创建的输出目录的目录是只读的。  
  
-   计算机的磁盘空间不足。  
  
-   你指定了无效的目录路径。  
  
-   用户权限不能满足需求，创建目录。
