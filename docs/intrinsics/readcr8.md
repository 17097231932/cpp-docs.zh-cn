# <a name="readcr8"></a>__readcr8
**Microsoft 专用**  
  
 读取的 CR8 寄存器并返回其值。  
  
## <a name="syntax"></a>语法  
  
```  
unsigned __int64 __readcr8(void);  
```  
  
## <a name="return-value"></a>返回值  
 中的 CR8 寄存器的值。
