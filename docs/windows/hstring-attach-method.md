# <a name="hstringattach-method"></a>HString::Attach 方法
将指定相关联**HString**对象与当前**HString**对象。  
  
## <a name="syntax"></a>语法  
  
```  
void Attach(  
       HSTRING hstr  
       ) throw()  
```  
  
#### <a name="parameters"></a>参数  
 *hstr*  
 将现有**HString**对象。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
