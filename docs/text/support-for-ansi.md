# <a name="support-for-ansi"></a>对 ANSI 的支持

大多数 MFC 类和方法都支持 ANSI 字符集，尽管 MFC 框架作为整体逐步发展为仅支持 Unicode 字符集。 由于 Windows Vista 和 Windows 公共控件版本6.1 中的持续增强功能，不推荐使用对多个 ANSI 类和方法的支持。  有关详细信息，请参阅 [弃用的 ANSI api](../mfc/deprecated-ansi-apis.md) 和 [Unicode 支持](../text/support-for-unicode.md)。
