# STRUCT (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

声明具有结构的类型指定的 *fielddeclarations*。  每个字段必须是有效的数据定义。  和 [STRUC](../../assembler/masm/struc.md)相同。  
  
## 语法  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
