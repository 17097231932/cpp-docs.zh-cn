# <a name="compiler-warning-level-2-c4051"></a>编译器警告（等级 2）C4051

类型转换；可能丢失数据

表达式包含具有不同的基类型的两个数据项。 转换一个类型导致数据项被截断。

如果将数据项强制转换为适当的类型，则可能会修复此警告。
