# <a name="classfactoryrelease-method"></a>ClassFactory::Release 方法
递减引用计数的当前**ClassFactory**对象。  
  
## <a name="syntax"></a>语法  
  
```  
STDMETHOD_(  
   ULONG,  
   Release  
)();  
```  
  
## <a name="return-value"></a>返回值  
 如果成功，则为 S_OK；否则为描述失败的 HRESULT。  
  
## <a name="requirements"></a>要求  
 **标头：** module.h  
  
 **命名空间：** Microsoft::WRL
