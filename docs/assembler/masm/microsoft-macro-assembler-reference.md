# Microsoft Macro Assembler Reference
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Microsoft 宏汇编程序 \(MASM\) 提供了几种优于内联程序集。  MASM 包含宏语言，如循环、 算术、 和文本字符串处理功能。  MASM 还提供了更好地控制硬件因为它支持 386、 486 和奔腾处理器的指令集。  通过使用 MASM，您还可以减少时间和内存开销。
