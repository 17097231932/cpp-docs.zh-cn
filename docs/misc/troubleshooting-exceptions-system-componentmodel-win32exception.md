# 关于异常的疑难解答：System.ComponentModel.Win32Exception
为 Win32 错误代码引发的异常。  
  
## 备注  
 在显示时，Win32 错误代码被从数值表示形式转换为系统消息。 您可以使用 <xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> 属性访问错误代码的数值表示形式。
