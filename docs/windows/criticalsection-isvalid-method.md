# <a name="criticalsectionisvalid-method"></a>CriticalSection::IsValid 方法

指示当前的临界部分是否有效。

## <a name="syntax"></a>语法

```cpp
bool IsValid() const;
```

## <a name="return-value"></a>返回值

默认情况下始终返回 **，则返回 true**。

## <a name="requirements"></a>要求

**标头：** corewrappers.h

**Namespace:** Microsoft::WRL::Wrappers
