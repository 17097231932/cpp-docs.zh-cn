# <a name="compiler-error-c2802"></a>编译器错误 C2802

静态成员 operator operator 具有没有形参

声明的运算符`static`成员函数必须具有至少一个参数。

下面的示例生成 C2802:

```
// C2802.cpp
// compile with: /clr /c
ref class A {
   static operator+ ();   // C2802
   static operator+ (A^, A^);   // OK
};
```
