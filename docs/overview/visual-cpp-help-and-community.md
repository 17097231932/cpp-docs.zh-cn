# <a name="microsoft-cc-help-and-community"></a>Microsoft C/C++ 帮助和社区

下面提供帮助信息，并介绍如何编写 C++ 代码和使用 Visual Studio 开发工具。

## <a name="samples"></a>示例

- [开发人员代码示例](/samples)：包含来自 Microsoft 和社区参与者的可下载示例代码。

## <a name="product-documentation"></a>产品文档

- [Visual Studio 中的 C++](visual-cpp-in-visual-studio.md)：包含有关 Visual C++ 的引用和概念文档。

- [Windows 开发人员中心](https://developer.microsoft.com/windows/)：包含有关如何使用 C++ 和其他语言来开发面向 Windows 10 的应用的信息。

### <a name="online-and-offline-documentation"></a>联机和脱机文档

可以联机查看 Microsoft 开发者内容。 此内容会定期更新。

你还可以在脱机帮助查看器中本地下载和查看内容。 脱机文档通过相关内容的书籍进行组织，这些书籍还会定期更新。 你可在书籍可用时下载你感兴趣的书籍。 有关详细信息，请参阅 [Microsoft Help Viewer](/visualstudio/ide/microsoft-help-viewer)。

该文档的很多部分还提供 PDF 形式。 这些部分在 docs.microsoft.com 的目录下具有一个“下载 PDF”链接。

## <a name="related-sites"></a>相关站点

- [C++ 团队博客](https://devblogs.microsoft.com/cppblog/)：包含一些博文，它们介绍了专家就 Microsoft 的 C++ 产品团队进行的各种主题。

- [第 9 频道：](https://channel9.msdn.com/)：包含视频访谈和演讲。 使用第 9 频道主页上的搜索框查找 C++ 内容。

- [Visual Studio 网站](https://visualstudio.microsoft.com/)：包含有关 Visual Studio 和相关开发工具的文章和新闻。 此网站还托管 Visual Studio 下载和可再发行文件。

- [Microsoft Docs 问答](/answers/topics/c%2B%2B.html)：官方 Microsoft 论坛，可在上面发布有关 C++ 和 Visual Studio 的问题，并从 Microsoft 和社区内的专家处获取答案。

- [Visual Studio C++ 开发人者社区](https://aka.ms/vsfeedback/browsecpp)：用于报告 C++ 编译器和工具或 IDE 问题，或提出产品建议的论坛。 你可以为最喜欢的想法或 Bug 报告投票，或添加一个新的想法或 Bug 报告。 此站点由开发团队监视。
