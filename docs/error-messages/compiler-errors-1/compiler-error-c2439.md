# <a name="compiler-error-c2439"></a>编译器错误 C2439
identifier： 无法初始化成员  
  
 无法初始化类、 结构或联合成员。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复  
  
1.  尝试初始化间接基类或结构。  
  
2.  尝试初始化类或结构的继承的成员。 必须通过类或结构的构造函数初始化继承的成员。
