# 编译器错误 C3910
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“event”: 必须定义成员“method”  
  
 定义了事件，但该事件不包含所需的指定访问器方法。  
  
 有关更多信息，请参见[事件](../../windows/event-cpp-component-extensions.md)。  
  
 下面的示例生成 C3910：  
  
```  
// C3910.cpp  
// compile with: /clr /c  
delegate void H();  
ref class X {  
   event H^ E {  
      // uncomment the following lines  
      // void add(H^) {}  
      // void remove( H^ h ) {}  
      // void raise( ) {}  
   };   // C3910  
  
   event H^ E2; // OK data member  
};  
```
