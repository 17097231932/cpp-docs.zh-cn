# <a name="platformoutofmemoryexception-class"></a>Platform::OutOfMemoryException 类

## <a name="syntax"></a>语法  
  
```cpp  
public ref class OutOfMemoryException : COMException,    IException,    IPrintable,    IEquatable  
```
