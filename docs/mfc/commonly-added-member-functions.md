# <a name="commonly-added-member-functions"></a>经常添加的成员函数

如果对话框包含确定或取消以外的按键，需要对它们所生成控件通知消息作出响应在对话框类中编写消息处理程序成员函数。 有关示例，请参阅[Scribble](../visual-cpp-samples.md)示例程序。 在对话框中，还可以处理来自其他控件的控件通知消息。
