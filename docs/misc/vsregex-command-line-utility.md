# VsRegEx 命令行实用工具
在 Visual Studio 2010 及更高版本中，VsRegEx 实用程序已替换为[CreateExpInstance 实用程序](../Topic/CreateExpInstance%20Utility.md)。  
  
## 备注  
 在 Visual Studio 2008 中，IDE 的实验实例的配置信息存储在系统注册表中。 在 Visual Studio 2010 及更高版本，它存储在文件系统中。
