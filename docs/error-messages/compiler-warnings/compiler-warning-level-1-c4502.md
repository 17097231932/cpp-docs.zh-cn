# <a name="compiler-warning-level-1-c4502"></a>编译器警告（等级 1）C4502

链接规范需要使用关键字 extern，并且必须位于所有其他说明符之前

指定链接，但未`extern`关键字。 链接到非外部类型无关。

编译器假定`extern`关键字。
