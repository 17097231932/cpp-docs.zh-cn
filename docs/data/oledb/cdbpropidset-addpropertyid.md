# <a name="cdbpropidsetaddpropertyid"></a>CDBPropIDSet::AddPropertyID
将属性 ID 添加到属性 ID 集中。  
  
## <a name="syntax"></a>语法  
  
```cpp
      bool AddPropertyID(DBPROPID propid) throw();  
```  
  
#### <a name="parameters"></a>参数  
 `propid`  
 [in] 要添加到属性 ID 集中的属性 ID。  
  
## <a name="requirements"></a>要求  
 **标头:** atldbcli.h
