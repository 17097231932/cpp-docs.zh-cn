# <a name="working-with-a-header-control"></a>使用标题控件

 ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)) 使用标题控件的简单方法与列表控件结合使用;请参阅本主题中后面的 [使用 CListCtrl](../mfc/using-clistctrl.md) 。 您还可以单独使用标题控件。 MFC `InitCommonControls` 将为您调用。 关键任务如下所示：

- [创建标题控件](../mfc/creating-the-header-control.md)

- [向标题控件添加项](../mfc/adding-items-to-the-header-control.md)

- [对标题控件中的项排序](../mfc/ordering-items-in-the-header-control.md)

- [处理标题控件通知](../mfc/processing-header-control-notifications.md)

如果标题控件对象嵌入在父视图或对话框类中，则在销毁父控件时，控件将被销毁。
