# 编译器错误 C2315
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“type1”：引用由“type2”在行号上捕获  
  
 异常类型由上一个处理程序处理。 第二个 catch 的引用具有的类型与第一个的相同。
