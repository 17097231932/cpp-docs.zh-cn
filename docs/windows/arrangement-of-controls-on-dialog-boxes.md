# Arrangement of Controls on Dialog Boxes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

对话框编辑器提供自动对齐控件和调整控件大小的布局工具。  对于大多数任务，可以使用[“对话框编辑器”工具栏](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)。  所有“对话框编辑器”工具栏命令也可用于“格式”菜单，其中大多数命令都有[快捷键](../mfc/accelerator-keys-for-the-dialog-editor.md)。  
  
 许多对话框布局命令仅在选定一个以上的控件时才可用。  可以选定单个控件或多个控件，如果选定多个控件，则所选的第一个控件默认为“主导”控件。  有关选定控件和主导控件的信息，请参见[选定控件](../mfc/selecting-controls.md)。  
  
 状态栏右下角显示当前控件的位置、高度和宽度。  当选定整个对话框时，状态栏显示对话框的整体位置、高度和宽度。

-   [将对话框上的单选按钮分组](../mfc/grouping-radio-buttons-on-a-dialog-box.md)

-   [沿对话框右侧或底部排列普通按钮](../mfc/arranging-push-buttons-along-the-right-or-bottom-of-a-dialog-box.md)
