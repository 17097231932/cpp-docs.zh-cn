# <a name="nmake-fatal-error-u1071"></a>NMAKE 错误 U1071
在目标 targetname 的依赖关系树中进行循环  
  
 在给定的目标的依赖关系树中存在循环依赖关系。 给定的目标是目标的一个给定的依赖项的依赖。 循环依赖项均无效。
