# 流的定义
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\# 与 C\+\+，不具备内置性能功能。  所有 C\+\+ 编译器，但是一个系统，只不过，面向对象的，" I\/O 包 iostream 类。  流是 iostream 类的核心概念。  您可以将流对象作为一个源和目标的字节的聪明文件。  自定义的插入和运算符提取取决于流的特性由其类和。
