# <a name="unrepresented-character-constants"></a>无代表的字符常量

**ANSI 3.1.3.4** 包含没有以宽字符常量的基本执行字符集或扩展字符集表示的字符或转义序列的整数字符常量的值

所有字符常量或转义序列均可以用扩展字符集表示。
