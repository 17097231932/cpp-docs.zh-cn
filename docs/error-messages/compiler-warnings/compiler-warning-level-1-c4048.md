# <a name="compiler-warning-level-1-c4048"></a>编译器警告（等级1） C4048

声明的数组下标不同： "identifier1" 和 "identifier2"

表达式涉及指向不同大小的数组的指针。 使用指针时不进行转换。

如果将数组显式强制转换为相同或等效的类型，则可能会修复此警告。
