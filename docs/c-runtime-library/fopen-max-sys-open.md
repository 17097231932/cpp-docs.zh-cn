# <a name="fopen_max-_sys_open"></a>FOPEN_MAX、_SYS_OPEN

## <a name="syntax"></a>语法

```
#include <stdio.h>
```

## <a name="remarks"></a>备注

这是可同时打开的文件的最大数目。 `FOPEN_MAX` 是与 ANSI 兼容的名称。 提供 `_SYS_OPEN` 以便与现有代码兼容。
