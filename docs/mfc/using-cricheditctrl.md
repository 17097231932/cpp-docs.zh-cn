# <a name="using-cricheditctrl"></a>使用 CRichEditCtrl
Rich edit 控件是一个窗口，用户可以输入和编辑文本。 文本可以分配字符和段落格式设置，并且可以包含嵌入的 OLE 对象。 Rich edit 控件表示在 MFC 中由[CRichEditCtrl](../mfc/reference/cricheditctrl-class.md)类。
