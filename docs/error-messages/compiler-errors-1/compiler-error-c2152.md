# <a name="compiler-error-c2152"></a>编译器错误 C2152

"identifier"：指向具有不同特性的函数的指针

指向一个函数的指针，该函数具有一个调用约定 (**`__cdecl`** 、 **`__stdcall`** 或 **`__fastcall`**) 被分配给具有其他调用约定的函数的指针。
