# <a name="2-directives"></a>2.指令
指令基于**#pragma** C 和 c + + 标准中定义的指令。  支持的 OpenMP C 和 c + + API 的编译器将包括命令行选项，可激活并允许所有 OpenMP 编译器指令解释。
