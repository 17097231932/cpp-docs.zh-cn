# <a name="compiler-error-c3808"></a>编译器错误 C3808

> *类型*： 具有 ComImport 特性的类不能定义成员*成员*、 仅抽象或允许使用 dllimport 函数

## <a name="remarks"></a>备注

派生自类型<xref:System.Runtime.InteropServices.ComImportAttribute>不能定义*成员*。

**/Clr: pure**和 **/clr: safe**编译器选项是在 Visual Studio 2015 中已过时，在 Visual Studio 2017 中不支持。

## <a name="example"></a>示例

下面的示例生成 C3808。

```cpp
// C3808.cpp
// compile with: /c /clr:pure user32.lib
using namespace System::Runtime::InteropServices;

[System::Runtime::InteropServices::ComImportAttribute()]
ref struct S1 {
   int f() {}   // C3808
   virtual int g() abstract;   // OK

   [DllImport("msvcrt.dll")]
   int printf(System::String ^, int i);   // OK
};
```
