# outp、outpw
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

这些 POSIX 函数被弃用。 请改用与 ISO C\+\+ 一致的 [\_outp、\_outpw、\_outpd](../c-runtime-library/outp-outpw-outpd.md)。  
  
> [!IMPORTANT]
>  这些函数已过时。 从 Visual Studio 2015 开始，CRT 中不再提供这些函数。  
  
> [!IMPORTANT]
>  此 API 不能用于在 [!INCLUDE[wrt](../atl/reference/includes/wrt_md.md)] 中执行的应用程序。 有关详细信息，请参阅[不支持 \/ZW 的 CRT 函数](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx)。
