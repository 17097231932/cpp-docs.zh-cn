# <a name="virtual-specifier"></a>虚拟说明符

[虚拟](../cpp/virtual-cpp.md)关键字可以只能应用于静态类成员函数。 它表示函数的调用绑定将推迟到运行时。 有关详细信息，请参阅[虚函数](../cpp/virtual-functions.md)。
