# <a name="editing-a-message-handler"></a>编辑消息处理程序
一旦定义消息处理程序，您可以转到要添加或修改代码的成员函数的定义。  
  
 若要跳转到成员函数定义从[对话框编辑器](../../windows/dialog-editor.md)，双击一个处理程序已为其定义的控件。 使用它可导航到在其中定义的选定的控件的消息处理程序的文件。  
  
 若要跳转到成员函数定义从[类视图](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925)，双击类视图中的函数名称。
