# RuntimeClassBase 结构
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```  
struct RuntimeClassBase;  
```  
  
## 备注  
 用于检测到 [生成](../windows/make-function.md) 函数的 RuntimeClass。  
  
 RuntimeClassBase 是空结构。  
  
## 继承层次结构  
 `RuntimeClassBase`

 **头文件：**implements.h
