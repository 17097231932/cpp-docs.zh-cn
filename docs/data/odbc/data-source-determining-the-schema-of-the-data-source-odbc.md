# 数据源：确定数据源的架构 (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

本主题适用于 MFC ODBC 类。  
  
 若要在 `CRecordset` 对象中设置数据成员，需要知道正在连接的数据源的架构。  确定数据源的架构包括需要获得数据源中表的列表、每个表中列的列表、每列的数据类型和现有的任何索引。
