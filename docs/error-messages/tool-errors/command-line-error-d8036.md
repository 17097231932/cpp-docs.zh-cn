# <a name="command-line-error-d8036"></a>命令行错误 D8036

不允许 "/option" 用于多个源文件

这些编译器选项不能与多个源文件一起使用：

-  (/Fa) 的名称程序集文件列表

- 重命名对象文件 (/Fo) 

- 创建不含本地变量 (/Fr) 的源浏览器信息

- 创建包含本地变量 (/FR 的源浏览器信息)
