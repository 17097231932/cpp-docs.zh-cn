# <a name="expression-evaluator-error-cxx0045"></a>表达式计算器错误 CXX0045

不是函数

为程序中的符号（不是函数的名称）提供了参数列表。

## <a name="example"></a>示例

```
queue( alpha, beta )
```

如果 `queue` 不是函数，则为。

此错误与 CAN0045 相同。
