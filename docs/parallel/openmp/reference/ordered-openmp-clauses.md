# <a name="ordered-openmp-clauses"></a>ordered（OpenMP 子句）

## <a name="syntax"></a>语法

```
ordered
```

`ordered` 适用于[为](../../../parallel/openmp/reference/for-openmp.md)指令。

有关详细信息，请参阅[2.4.1 for 构造](../../../parallel/openmp/2-4-1-for-construct.md)。
