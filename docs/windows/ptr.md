# ptr
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指定指针作为完整的指针。  
  
## 语法  
  
```  
  
[ptr]  
  
```  
  
## 备注  
 **PTR** C\+\+ 特性具有与 [PTR](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL 属性相同。  
  
## 示例  
 为 [defaultvalue](../windows/defaultvalue.md) 参见示例为 **PTR**的示例使用。  
  
## 要求  
  
### 属性上下文  
  
|||  
|-|-|  
|**适用对象**|接口参数，接口方法， `typedef`|  
|**可重复**|否|  
|**必需的特性**|无|  
|**无效的特性**|无|  
  
 有关属性上下文的更多信息，请参见 [属性上下文](../windows/attribute-contexts.md)。
