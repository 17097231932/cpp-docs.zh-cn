# <a name="compiler-error-c3183"></a>编译器错误 C3183

不能在托管的或 WinRT 类型“type”的内部定义未命名的类、结构或联合

必须命名嵌入托管的或 WinRT 类型中的类型。

下列示例生成 C3183：

```
// C3183a.cpp
// compile with: /clr /c
ref class Test
{
   ref class
   {  // C3183, delete class or name it
      int a;
      int b;
   };
};
```
