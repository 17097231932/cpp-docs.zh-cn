# <a name="iatlautothreadmodule-class"></a>IAtlAutoThreadModule 类

此类表示的接口`CreateInstance`方法。

> [!IMPORTANT]
>  不能在 Windows 运行时中执行的应用程序中使用此类和其成员。

## <a name="syntax"></a>语法

```
__interface IAtlAutoThreadModule
```

## <a name="remarks"></a>备注

该类[CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md)派生自`IAtlAutoThreadModule`，使用它来提供用于创建对象和检索接口指针的代码。

## <a name="requirements"></a>要求

**标头：** atlbase.h
