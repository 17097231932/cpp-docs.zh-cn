# <a name="new-ltdevicegt-image-type-dialog-box-image-editor-for-icons"></a>新&lt;设备&gt;图像类型对话框 （图标的图像编辑器）
使用此选项可创建指定类型的新设备映像。 若要打开**新建\<设备 > 图像**对话框中，单击**新图像类型**上**映像**菜单。  
  
 **目标映像类型**  
 列出可用的图像类型。 选择你想要打开的图像类型：  
  
||||  
|-|-|-|  
|-16 x 16，16 种颜色|-48 x 48，16 种颜色|-x 96，96 16 种颜色|  
|-16 x 16，256 种颜色|-48 x 48，256 种颜色|-x 96，96 256 种颜色|  
|-16 x 16，单色|-48 x 48，单色|-x 96，单色 96|  
|-32 x 32，16 种颜色|-64 x64、 16 种颜色||  
|-32 x 32，256 种颜色|-64 x64、 256 种颜色||  
|-32 x 32，单色|-x 64，单色 64||  
  
> [!NOTE]
>  任何现有的映像将不显示在此列表中。  
  
 **自定义**  
 打开[自定义图像对话框](custom-image-dialog-box-image-editor-for-icons.md)在其中你可以创建新的映像具有自定义的大小和颜色数。
