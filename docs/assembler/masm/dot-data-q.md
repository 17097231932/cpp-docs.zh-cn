# .DATA?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

当使用 [.MODEL](../../assembler/masm/dot-model.md)，开始每个最近的数据段为未初始化的数据 \(段名 \_BSS\)。  
  
## 语法  
  
```  
  
.DATA?  
  
```
