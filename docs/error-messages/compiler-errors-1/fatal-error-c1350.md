# 错误 C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

加载 dll“dll”时出错: 找不到 dll  
  
 找不到支持所尝试的操作的 DLL。  
  
 这表明安装有问题，应从产品 CD 重新安装 DLL。
