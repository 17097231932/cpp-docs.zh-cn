# <a name="ml-nonfatal-error-a2047"></a>ML 非致命错误 A2047
**空 (null) 的字符串**  
  
 一个字符串包含的引号和中的没有字符的分隔对。  
  
 对于有效的字符串，它必须包含 1-255 个字符。
