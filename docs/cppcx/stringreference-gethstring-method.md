# StringReference::GetHSTRING 方法
以 `__abi_HSTRING` 形式返回字符串内容。  
  
## 语法  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## 返回值  
 一个包含字符串数据的 `__abi_HSTRING`。
