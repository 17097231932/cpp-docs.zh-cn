# “清除重写属性”对话框
> [!NOTE]
>  在 [!INCLUDE[vs_dev10_long](../build/includes/vs_dev10_long_md.md)] 上不支持此对话框。  
  
 当将用户定义的属性表（.props 文件）添加到项目配置中时，该项目配置在默认情况下将继承在属性表中定义的所有属性。  使用此对话框清除项目配置不需要的继承属性。
