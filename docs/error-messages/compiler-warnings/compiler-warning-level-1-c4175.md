# <a name="compiler-warning-level-1-c4175"></a>编译器警告（等级 1）C4175

 若要使用[组件](../../preprocessor/component.md)杂注，必须生成在编译期间的浏览信息 ([/FR](../../build/reference/fr-fr-create-dot-sbr-file.md))。
