# NMAKE 错误 U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

目标“targetname”的依赖项树中的循环  
  
 给定目标的依赖项树中存在循环依赖项。  给定目标为该给定目标的一个从属项的从属项。  循环依赖项无效。
