# .CONST
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

当使用 [.MODEL](../../assembler/masm/dot-model.md)，开始每个常数段 \(与段名 CONST\)。  
  
## 语法  
  
```  
  
.CONST  
  
```  
  
## 备注  
 此段都是只读属性。
