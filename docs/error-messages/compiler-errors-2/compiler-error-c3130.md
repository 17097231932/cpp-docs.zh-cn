# 编译器错误 C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

内部编译器错误 : 未能将插入的代码块写入 PDB  
  
 如果编译器未能将插入的代码块写入 .pdb 文件中，则会发生此错误。  这种故障最常见的原因是磁盘空间不足。
