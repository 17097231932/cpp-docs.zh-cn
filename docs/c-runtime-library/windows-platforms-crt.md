# <a name="windows-platforms-crt"></a>Windows 平台 (CRT)
[!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] 的 C 运行库支持 [!INCLUDE[win8](../build/reference/includes/win8_md.md)]、[!INCLUDE[winserver8](../build/reference/includes/winserver8_md.md)]、[!INCLUDE[win7](../build/includes/win7_md.md)]、[!INCLUDE[winsvr08](../build/reference/includes/winsvr08_md.md)]、Windows Vista、适用于 x86 的 [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 3 (SP3)、适用于 x64 的 [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 2 (SP2) 以及同时适用于 x86 和 x64 的 [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Service Pack 2 (SP2)。 所有这些操作系统都支持 Windows 应用程序编程接口 (API) 并提供 Unicode 支持。 此外，所有 Win32 应用程序都能使用多字节字符集 (MBCS)。  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)] 不支持 [!INCLUDE[winxp](../build/includes/winxp_md.md)] 和 [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] 开发。 必须安装 [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] 才能添加对 [!INCLUDE[winxp](../build/includes/winxp_md.md)] 和 [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] 的支持。
