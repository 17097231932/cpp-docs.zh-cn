# <a name="compiler-error-c2969"></a>编译器错误 C2969
语法错误: “symbol”: 成员函数定义应以“}”结尾  
  
 模板成员函数定义具有不匹配的右大括号。  
  
 以下示例生成 C2969：  
  
```  
// C2969.cpp  
// compile with: /c  
class A {  
   int i;  
public:  
   A(int i) {}  
};  
  
A anA(1);  
  
class B {  
   A a;  
   B() : a(anA);   // C2969  
   // try the following line instead  
   // B() : a(anA) {}  
};  
```
