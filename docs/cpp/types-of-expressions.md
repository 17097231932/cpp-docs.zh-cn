# <a name="types-of-expressions"></a>表达式的类型
C++ 表达式分为多个类别：  
  
-   [主表达式](../cpp/primary-expressions.md)。 这些是从中构成所有其他表达式的构造块。  
  
-   [后缀表达式](../cpp/postfix-expressions.md)。 这些是后跟运算符的主表达式 - 例如，数组下标或后缀递增运算符。  
  
-   [使用一元运算符的表达式格式](../cpp/expressions-with-unary-operators.md)。 一元运算符仅作用于表达式中的某个操作数。  
  
-   [使用二元运算符的表达式格式](../cpp/expressions-with-binary-operators.md)。 二元运算符在表达式中操作两个操作数。  
  
-   [带条件运算符的表达式](../cpp/conditional-operator-q.md)。 条件运算符是 C++ 语言中唯一的三元运算符，它操作三个操作数。  
  
-   [常量表达式](../cpp/cpp-constant-expressions.md)。 常量表达式完全由常量数据组成。  
  
-   [使用显式类型转换的表达式](https://msdn.microsoft.com/060ad6b4-9592-4f3e-8509-a20ac84a85ae)。 可以在表达式中使用显式类型转换或“强制转换”。  
  
-   [使用指向成员的指针运算符的表达式](../cpp/pointer-to-member-operators-dot-star-and-star.md)。  
  
-   [强制转换](../cpp/casting.md)。 可在表达式中使用类型安全“强制转换”。  
  
-   [运行时类型信息](../cpp/run-time-type-information.md)。 在程序执行期间确定对象的类型。
