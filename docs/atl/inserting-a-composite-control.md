# <a name="inserting-a-composite-control"></a>插入复合控件

" **添加类** " 对话框允许您将 ATL 对象插入到项目中。 若要访问此对话框，请在解决方案资源管理器中右键单击项目名称，指向 " **添加**"，然后单击 " **添加类**"。

在 " **添加类** " 对话框中，选择 " **ATL 控件**"。 这将启动 [ATL 控件向导](../atl/reference/atl-control-wizard.md)。 若要创建复合控件，请选择 " **选项** " 选项卡，然后单击 " **复合控件** " 复选框。

将创建一个用于查看控件的默认 HTML 页面。
