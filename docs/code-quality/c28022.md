# <a name="c28022"></a>C28022

> 警告 C28022：此函数的函数类 (es) \<classlist1> 与 \<classlist2> 用于定义它的 typedef 上的函数类 (es) 不匹配。

当批注中存在错误时将报告此警告。 Typedef 和函数本身都具有 `_Function_class_` 批注，但不匹配。 如果两者都使用，则它们必须匹配。
