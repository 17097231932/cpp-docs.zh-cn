# “网格设置”对话框（图标的图像编辑器）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

允许您为图像指定网格设置并在编辑的图像上显示网格线。  网格线有助于编辑图像，但不作为图像本身的一部分保存。  
  
 **像素网格**  
 选中时，在图像编辑器中的每个像素周围显示网格。  网格仅以 4× 和更高的分辨率显示。  
  
 **平铺网格**  
 选中时，在图像编辑器中的像素块周围显示网格，像素块由网格间距值指定。  
  
 **宽度**  
 指定每个平铺块的宽度。  这在绘制包含按规则间隔排列的多个图像的位图时非常有用。  
  
 **高度**  
 指定每个平铺块的高度。  这在绘制包含按规则间隔排列的多个图像的位图时非常有用。
