# <a name="containers"></a>容器

容器应用程序是可将嵌入项或者链接项并入自己的文档中的应用程序。 容器应用程序管理的文档必须能够存储和显示 OLE 复合文档组件以及应用程序本身创建的数据。 容器应用程序还必须支持用户插入新项或编辑现有项。
