# <a name="native-and-net-interoperability"></a>本机和 .NET 的互操作性
Visual c + + 支持互操作性功能，允许托管和非托管构造共存和交互操作在相同的程序集中，甚至在同一文件。 此功能，例如 P/Invoke，一小部分支持的其他.NET 语言，但大部分由 Visual c + + 提供的互操作性支持不是有其他语言版本。
