# <a name="recordset-declaring-a-class-for-a-table-odbc"></a>记录集：声明表的类 (ODBC)

本主题适用于 MFC ODBC 类。  
  
最常见的记录集类打开单个表。 若要声明单个表的记录集类，请使用[MFC ODBC 使用者向导](../../mfc/reference/adding-an-mfc-odbc-consumer.md)从**添加类**并选择所需的命名对应的记录集字段数据成员的每个列。  
  
记录集的其他用途包括：  
  
- 联接两个或多个表。  
  
- 包含预定义查询的结果。
