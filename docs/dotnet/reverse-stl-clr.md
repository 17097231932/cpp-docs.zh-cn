# <a name="reverse-stlclr"></a>reverse (STL/CLR)
反转范围中元素的顺序。  
  
## <a name="syntax"></a>语法  
  
```  
template<class _BidIt> inline  
    void reverse(_BidIt _First, _BidIt _Last);  
```  
  
## <a name="remarks"></a>备注  
 此函数的行为与 c + + 标准库函数相同`reverse`。 有关详细信息，请参阅[反向](../standard-library/algorithm-functions.md#reverse)。
