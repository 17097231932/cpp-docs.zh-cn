# <a name="command-line-warning-d9035"></a>命令行警告 D9035

> 选项 "*option*" 已弃用，并将在将来的版本中删除

## <a name="remarks"></a>备注

指定编译器选项，该选项将在编译器的未来版本中删除。 如果有建议的替换*选项*，则此警告后跟警告[D9036](../../error-messages/tool-errors/command-line-warning-d9036.md)。

指定的选项仍然有效，但你现在应更新生成配置。 因此，当你升级编译器时，你的项目更有可能继续生成。
