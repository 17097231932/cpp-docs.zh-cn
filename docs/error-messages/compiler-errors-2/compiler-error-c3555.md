# 编译器错误 C3555
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“decltype”的参数不正确  
  
 `decltype(` `expression` `)` 类型说明符的参数不是有效的表达式。  
  
> [!NOTE]
>  C3555 不大可能发生。 如果编译器发出 C3555，可能是因为发生了内部编译器错误。
