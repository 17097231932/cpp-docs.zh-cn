# CA0064
未执行任何分析，因为指定的规则集不包含任何 FxCop 规则。  
  
 此警告会在下列任一情况下发生：  
  
-   如果您遇到此警告和 CA0063 警告一起显示，则说明加载规则集的文件时出了问题。  有关更多信息，请参见 [CA0063](../misc/ca0063.md) 文档。  
  
-   如果您遇到此错误和 CA0062 警告一起显示，则说明很可能是代码分析找不到包含由规则集指定的规则的程序集。  有关更多信息，请参见 [CA0062](../misc/ca0062.md) 文档。  
  
-   否则，此警告通常发生在您的规则集为空，或子规则集中启用的所有规则都已被禁用时。  使用 [!INCLUDE[vs_current_short](../misc/includes/vs_current_short_md.md)] 规则集编辑器启用您的规则集中的某些规则。  有关更多信息，请参见 [创建自定义规则集](../Topic/Creating%20Custom%20Code%20Analysis%20Rule%20Sets.md)。
