# <a name="c-tokens"></a>C 标记

在 C 源程序中，编译器识别的基本元素是“标记”。 标记是编译器不会分解为组件元素的源程序文本。

## <a name="syntax"></a>语法

*token*: *keyword*

*identifier*

*constant*

*string-literal*

*operator*

punctuator

> [!NOTE]
>  有关 ANSI 语法约定的说明，请参阅 [C 语言语法摘要](../c-language/c-language-syntax-summary.md)的简介。

本节描述的关键字、标识符、常量、字符串文本和运算符是标记的示例。 标点符号（如方括号 ([ ])、大括号 ({ })、圆括号 ( ( ) ) 和逗号 (,) 也是标记。
