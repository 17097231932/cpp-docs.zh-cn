# <a name="eventsourcegetsize-method"></a>EventSource::GetSize 方法
检索与当前 EventSource 对象关联的事件处理程序数量  
  
## <a name="syntax"></a>语法  
  
```  
size_t GetSize() const;  
```  
  
## <a name="return-value"></a>返回值  
 中的事件处理程序数[targets_](../windows/eventsource-targets-data-member.md)。  
  
## <a name="requirements"></a>要求  
 **标头：** event.h  
  
 **命名空间：** Microsoft::WRL
