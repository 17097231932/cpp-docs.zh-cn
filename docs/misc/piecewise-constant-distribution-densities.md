# piecewise_constant_distribution::densities
返回分布的存储密度。  
  
## 语法  
  
```  
std::vector<double> densities() const;  
```  
  
## 备注  
 该成员函数返回从权重参数计算出的存储密度。  
  
## 要求  
 **标头：**\<random\>  
  
 **命名空间:** std
