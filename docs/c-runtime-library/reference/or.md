# <a name="or"></a>或
&#124;&#124; 运算符的替代项。  
  
## <a name="syntax"></a>语法  
  
```  
  
#define or ||  
  
```  
  
## <a name="remarks"></a>备注  
 该宏产生运算符 &#124;&#124;。  
  
## <a name="example"></a>示例  
  
```  
// iso646_or.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   bool a = true, b = false, result;  
  
   boolalpha(cout);  
  
   result= a || b;  
   cout << result << endl;  
  
   result= a or b;  
   cout << result << endl;  
}  
```  
  
```Output  
true  
true  
```

 **标头：** \<iso646.h>
