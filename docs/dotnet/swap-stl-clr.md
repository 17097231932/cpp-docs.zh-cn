# swap (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

交换元素的值对象之间的两种类型的分配，第一个对象到第二个对象的内容和内容第二个到第一个。  
  
## 语法  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## 备注  
 此函数行为与 STL `swap`函数相同。  有关详细信息，请参阅[swap](../Topic/swap.md)。  
  
## 要求  
 **页眉：** \<\/cliext 算法\>  
  
 **命名空间：** cliext
