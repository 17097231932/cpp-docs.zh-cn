# <a name="autohandle-class"></a>auto_handle 类

自动资源管理，这可用于托管类型中嵌入虚拟句柄。

## <a name="syntax"></a>语法

```
template<typename _element_type>
ref class auto_handle;
```

#### <a name="parameters"></a>参数

*_element_type*<br/>
要嵌入的托管的类型。

## <a name="requirements"></a>要求

**标头文件** \<msclr\auto_handle.h >

**Namespace** msclr
