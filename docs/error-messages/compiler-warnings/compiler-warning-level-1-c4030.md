# 编译器警告（等级 1）C4030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**第一个形参表比第二个长**  
  
 使用不同的正式参数重新声明了一个函数。 编译器使用在第一个声明中给出的正式参数。
