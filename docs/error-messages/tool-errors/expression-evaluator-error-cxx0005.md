# <a name="expression-evaluator-error-cxx0005"></a>表达式计算器错误 CXX0005

运算符不受支持

在表达式中指定了不受支持的 C 运算符。 使用支持的 C 运算符编写等效表达式。

此错误与 CAN0005 相同。
