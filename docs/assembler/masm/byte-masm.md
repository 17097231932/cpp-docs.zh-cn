# <a name="byte"></a>BYTE

为每个 *初始值设定项* 分配一个字节的存储，并根据需要对其进行初始化。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **BYTE** *初始值设定* 项⟦__，__ *初始值设定项* .。。⟧

## <a name="remarks"></a>备注

在类型合法的任何位置，还可以用作类型说明符。
