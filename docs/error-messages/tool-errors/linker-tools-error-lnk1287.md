# <a name="linker-tools-error-lnk1287"></a>链接器工具错误 LNK1287

托管入口点函数无效

入口点对于托管映像无效。 托管入口点函数的返回类型只能为 **`void`** 或 **`int`** 。 托管入口点函数的参数类型只能为 **`void`** 或 `String []` 。
