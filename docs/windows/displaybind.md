# displaybind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指示应向用户显示为可绑定的属性。  
  
## 语法  
  
```  
  
[displaybind]  
  
```  
  
## 备注  
 **displaybind** C\+\+ 特性具有与 [displaybind](http://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL 属性相同。  
  
## 示例  
 有关示例的 [可绑定](../windows/bindable.md) 参见示例中使用 **displaybind**。  
  
## 要求  
  
### 属性上下文  
  
|||  
|-|-|  
|**适用对象**|接口方法|  
|**可重复**|否|  
|**必需的特性**|无|  
|**无效的特性**|无|  
  
 有关更多信息，请参见 [属性上下文](../windows/attribute-contexts.md)。
