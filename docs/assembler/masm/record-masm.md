# <a name="record-masm"></a>RECORD (MASM)

声明由指定字段组成的记录类型。 *fieldname*命名字段， *width*指定位数，而*表达式*提供其初始值。

## <a name="syntax"></a>语法

> *recordname* **记录** *fieldname* __：__ *width* ⟦ __=__ *expression*⟧⟦ __，__ *fieldname* __：__ *width* ⟦ __=__ *expression*⟧ .。。⟧
