# 项目生成错误 PRJ0012
系统资源可能严重不足。 无法创建启动生成所需的线程。  
  
 如果系统资源不足：  
  
-   关闭一些正在运行的应用程序。  
  
-   重启 Visual Studio 以释放资源。  
  
-   重新启动。
