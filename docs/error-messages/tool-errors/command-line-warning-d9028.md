# 命令行警告 D9028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

最小重新生成失败，正在转换为标准生成  
  
 项目 .idb 文件已损坏。  删除该文件并重新生成。  
  
 有关最小重建和 .idb 文件的更多信息，请参见 [\/Gm 编译选项](../../build/reference/gm-enable-minimal-rebuild.md)。
