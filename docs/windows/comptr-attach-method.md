# <a name="comptrattach-method"></a>ComPtr::Attach 方法
将此 ComPtr 与由当前模板类型参数指定的接口类型相关联。  
  
## <a name="syntax"></a>语法  
  
```  
void Attach(  
   _In_opt_ InterfaceType* other  
);  
```  
  
#### <a name="parameters"></a>参数  
 `other`  
 接口类型。  
  
## <a name="requirements"></a>惠?  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
