# <a name="atl-project-wizard"></a>ATL 项目向导
活动模板库 (ATL) 是一套基于模板的 c + + 类，用于简化编写小型和快速的 COM 对象。 ATL 项目向导创建的项目与以包含 COM 对象的结构。  
  
## <a name="overview"></a>概述  
 此向导页描述当前[ATL 项目的应用程序设置](../../atl/reference/application-settings-atl-project-wizard.md)你创建。 默认情况下，项目具有以下设置：  
  
-   动态链接库指定你的服务器是 DLL，因此一个进程内服务器。  
  
-   你的项目使用特性的特性化的指定。  
  
 若要更改这些默认设置，请单击**应用程序设置**ATL 项目向导页中的向导并进行更改的左侧列中。  
  
 有关默认项目设置的信息，包括所选的字符集，以及链接默认设置，请参阅[默认 ATL 项目配置](../../atl/reference/default-atl-project-configurations.md)。  
  
 创建新的 ATL 项目后，可以将对象或控件添加到你的项目使用 Visual c + +[代码向导](../../ide/adding-functionality-with-code-wizards-cpp.md)。 你可以进行以下类型的使用代码向导对基本 ATL 项目的增强功能：  
  
-   [将对象和控件添加到 ATL 项目](../../atl/reference/adding-objects-and-controls-to-an-atl-project.md)  
  
-   [ATL 项目中添加新的接口](../../atl/reference/adding-a-new-interface-in-an-atl-project.md)  
  
-   [将 COM + 1.0 组件添加到 ATL 项目](../../atl/reference/adding-an-atl-com-plus-1-0-component.md)  
  
 创建和增强 ATL 项目时，此外，请考虑这些任务：  
  
-   [使 ATL 对象不可创建](../../atl/reference/making-an-atl-object-noncreatable.md)  
  
-   [ATL 项目的编译器进行优化](../../atl/reference/specifying-compiler-optimization-for-an-atl-project.md)  
  
 你可以指定项目属性 (例如，[是否以静态方式链接到 CRT](../../atl/programming-with-atl-and-c-run-time-code.md)) 中[项目属性](../../ide/general-property-page-project.md)页上，并且可以设置[生成配置](/visualstudio/ide/understanding-build-configurations)为ATL 项目。 请参阅[使用项目生成的应用程序](http://msdn.microsoft.com/en-us/3339fa90-bac2-4b95-8361-662a2e0e7dfe)有关详细信息。
