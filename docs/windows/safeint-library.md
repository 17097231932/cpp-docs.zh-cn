# <a name="safeint-library"></a>SafeInt 库

**SafeInt**库有助于防止应用程序执行数学运算时可能会导致整数溢出。 此库的最新版本位于[ https://github.com/dcleblanc/SafeInt ](https://github.com/dcleblanc/SafeInt)。
