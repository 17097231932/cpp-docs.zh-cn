# 资源编译器错误 RC2017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非法的转义序列  
  
 换码序列出现在不应出现的位置上。  
  
 转义序列 \-\- 后跟数字或字母的反斜杠 \( **\\** \)，只能在字符或字符串常数中出现。
