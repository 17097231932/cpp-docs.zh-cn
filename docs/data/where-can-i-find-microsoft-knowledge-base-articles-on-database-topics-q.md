# 哪里可以找到有关数据库主题的 Microsoft 知识库文章？
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

若要查找有关 MFC ODBC 主题的知识库文章，请用关键字 MFCDATABASE 搜索知识库。  
  
 若要查找有关 MFC DAO 主题的知识库文章，请用关键字 MFCDAO 搜索知识库。
