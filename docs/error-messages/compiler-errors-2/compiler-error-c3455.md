# <a name="compiler-error-c3455"></a>编译器错误 C3455
“attribute”：没有任何特性构造函数匹配这些参数  
  
 已使用无效的值声明特性。  有关更多信息，请参见 [attribute](../../windows/attribute.md) 。  
  
## <a name="example"></a>示例  
 以下示例生成 C3455  
  
```  
// C3455.cpp  
// compile with: /clr /c  
using namespace System;  
  
[attribute("MyAt")]   // C3455  
// try the following line instead  
// [attribute(All)]  
ref struct MyAttr {  
   MyAttr() {}  
};  
```
