# <a name="searching-and-sorting"></a>搜索和排序

可使用下列函数进行搜索和排序。

## <a name="searching-and-sorting-functions"></a>搜索和排序函数

|函数|搜索或排序|
|--------------|--------------------|
|[bsearch](../c-runtime-library/reference/bsearch.md)|二进制搜索|
|[bsearch_s](../c-runtime-library/reference/bsearch-s.md)|bsearch 的更安全版本|
|[_lfind](../c-runtime-library/reference/lfind.md)|线性搜索给定值|
|[_lfind_s](../c-runtime-library/reference/lfind-s.md)|_lfind 的更安全版本|
|[_lsearch](../c-runtime-library/reference/lsearch.md)|线性搜索给定值，如果找不到，则已添加至数组|
|[_lsearch_s](../c-runtime-library/reference/lsearch-s.md)|_lsearch 的更安全版本|
|[qsort](../c-runtime-library/reference/qsort.md)|快速排序|
|[qsort_s](../c-runtime-library/reference/qsort-s.md)|qsort 的更安全版本|
