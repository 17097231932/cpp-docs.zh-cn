# <a name="nmake-fatal-error-u1078"></a>NMAKE 错误 U1078

在 expression 的常量溢出

给定的表达式中包含一个常量，它超出了范围-2,147,483,648 到 2,147,483,647。 该常数出现在以下情况之一：

- 使用预处理指令指定的表达式

- 用短划线指定的错误级别 (**-**) 命令修饰符
