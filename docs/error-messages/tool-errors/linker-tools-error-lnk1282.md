# <a name="linker-tools-error-lnk1282"></a>链接器工具错误 LNK1282
无法 /REBASE 文件;已签名  
  
 试图更改了 /REBASE 选项签名的程序集的基址[editbin](../../build/reference/editbin-reference.md)。 若要执行此操作，首先将更改基址，然后登录程序集。
