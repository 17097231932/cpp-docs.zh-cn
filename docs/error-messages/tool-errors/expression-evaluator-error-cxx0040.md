# <a name="expression-evaluator-error-cxx0040"></a>表达式计算器错误 CXX0040
函数需要隐式转换  
  
 C 表达式计算器不支持涉及构造函数调用的隐式转换。  
  
 此错误是与 CAN0040 相同。
