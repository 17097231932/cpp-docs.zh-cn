# <a name="ompgetwtick"></a>omp_get_wtick
返回处理器时钟计时周期之间的秒数。  
  
## <a name="syntax"></a>语法  
  
```  
double omp_get_wtick( );  
```  
  
## <a name="remarks"></a>备注  
 有关详细信息，请参阅[3.3.2 omp_get_wtick 函数](../../../parallel/openmp/3-3-2-omp-get-wtick-function.md)。  
  
## <a name="example"></a>示例  
 请参阅[omp_get_wtime](../../../parallel/openmp/reference/omp-get-wtime.md)有关的使用示例`omp_get_wtick`。
