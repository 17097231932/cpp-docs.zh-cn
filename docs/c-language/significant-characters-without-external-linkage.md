# <a name="significant-characters-without-external-linkage"></a>没有外部链接的重要字符
**ANSI 3.1.2** 不具有外部链接的重要字符的数量  
  
 标识符对 247 个字符是有意义的。 编译器不限制可以在标识符中使用的字符数；它只会忽略限制之外的任何字符。
