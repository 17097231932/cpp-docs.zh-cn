# 编译器警告（等级 1）C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 符号解析为置换寄存器  
  
 标识符是解析为置换寄存器的本地符号，因此可以在操作数上与另一个符号一起使用。
