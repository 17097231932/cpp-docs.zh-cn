# 添加一般 C++ 类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可以使用**“类视图”**来添加一般 C\+\+ 类。  一般 C\+\+ 类是您定义或从您定义的类导出的类。  
  
### 将一般 C\+\+ 类添加到项目  
  
1.  在**“类视图”**中，右击要为其添加新类的项目，单击**“添加”**，然后单击**“类”**。  
  
2.  在[添加类](../ide/add-class-dialog-box.md)对话框中的模板窗格中，单击**“C\+\+ 类”**。  单击**“添加”**以显示[一般 C\+\+ 类向导](../ide/generic-cpp-class-wizard.md)。  
  
3.  在该向导中，提供一个类名，然后定义各项设置或接受默认设置。  
  
4.  若要关闭向导并查看项目中新的一般 C\+\+ 类，请单击**“完成”**。
