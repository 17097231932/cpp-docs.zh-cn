# <a name="compiler-error-c2858"></a>编译器错误 C2858

命令行选项 "/Yc （/Fdfilename）" 与使用 "/Fdfilename" 的预编译头不一致

"使用预编译头（[/yu](../../build/reference/yu-use-precompiled-header-file.md)）" 选项指定的程序数据库不是先前的 "创建预编译标头（[/yc](../../build/reference/yc-create-precompiled-header-file.md)）" 选项指定的数据库。
