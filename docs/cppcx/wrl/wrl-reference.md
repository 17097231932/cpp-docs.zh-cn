# <a name="wrl-reference"></a>WRL 引用

本部分包含 Windows 运行时C++模板库（WRL）的参考信息。

> [!NOTE]
> Windows 运行时C++模板库定义支持 Windows 运行时C++模板库基础结构的功能，但不应在代码中直接使用。 此类功能在本文档中进行了说明。

## <a name="in-this-section"></a>本节内容

[Microsoft::WRL Namespace](microsoft-wrl-namespace.md)<br/>
定义组成 Windows 运行时C++模板库的基本类型。

[Microsoft::WRL::Wrappers 命名空间](microsoft-wrl-wrappers-namespace.md)<br/>
定义简化对象、字符串和句柄的生存期管理的“获取资源即初始化”(RAII) 包装器类型。

[Microsoft::WRL::Wrappers::HandleTraits 命名空间](microsoft-wrl-wrappers-handletraits-namespace.md)<br/>
介绍常见的基于句柄的资源类型的特征。

[Windows::Foundation 命名空间](windows-foundation-namespace.md)<br/>
启用基础 Windows 运行时功能，如对象和工厂创建。

介绍 Windows 运行时C++模板库，这是一个基于 COM 的模板库，提供了一种用于创作和使用 Windows 运行时组件的底层方式。
