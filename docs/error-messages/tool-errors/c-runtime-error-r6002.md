# <a name="c-runtime-error-r6002"></a>C 运行时错误 R6002

未加载浮点支持

所需的浮点库未链接。

> [!NOTE]
> 如果在运行应用程序时遇到此错误消息，则该应用程序已关闭，因为它有一个内部问题。 此错误有几个可能的原因，但通常是由于应用程序的代码中的缺陷或尝试运行的应用程序不是为特定计算机处理器生成的。
>
> 可以尝试以下步骤来修复此错误：
>
> - 使用 **控制面板** 中的 "**应用和功能**" 或 "**程序和功能**" 页来修复或重新安装该程序。
> - 检查 "**控制面板**" 中的 "软件更新" **Windows 更新**。
> - 检查应用的更新版本。 如果问题仍然存在，请与应用程序供应商联系。

**面向程序员的信息**

未链接浮点库时，应用程序中会出现此错误。 检查以下原因之一：

- 或函数的格式字符串 `printf_s` `scanf_s` 包含浮点格式规范，并且程序不包含任何浮点值或变量。 若要解决此问题，请使用浮点参数以与浮点格式规范相对应，或在程序的其他位置执行浮点赋值。 这将导致加载浮点支持。

- 编译器仅在必要时才加载浮点支持来最大程度地降低程序的大小。 编译器无法检测到格式字符串中的浮点运算或浮点格式规范，因此不会加载必要的浮点例程。 若要解决此问题，请使用浮点格式规范并提供浮点参数，或者在程序的其他位置执行浮点赋值。 这将导致加载浮点支持。

- 在混合语言程序中，当程序链接时，将在 FORTRAN 库之前指定 C 库。 重新链接并指定 C 库的最后一个。
