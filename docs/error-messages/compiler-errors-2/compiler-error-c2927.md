# 编译器错误 C2927
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”：必须至少用一个参数调用函数模板  
  
 不使用参数则无法调用模板函数。 模板参数的类型决定将生成的函数的版本。
