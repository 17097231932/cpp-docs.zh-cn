# <a name="specifying-the-dominant-control"></a>指定主导控件
所选控件首先是主导控件。  
  
### <a name="to-specify-the-dominant-control"></a>指定主导控件  
  
1.  按住**CTRL**密钥，然后单击你想要使用的大小或其他控件的位置的影响的控件*第一个*。  
  
     **请注意**空心下级控件的句柄时，是实心主导控件的尺寸控点。 所有进一步调整大小或对齐方式取决于主导控件。  
  
### <a name="to-change-the-dominant-control"></a>若要更改主导控件  
  
1.  清除所有当前所选控件的外部单击当前所选内容。  
  
2.  重复前面的过程，首先选择一个不同的控件。
