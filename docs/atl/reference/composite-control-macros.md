# Composite Control Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些宏定义事件接收器映射和项。  
  
|||  
|-|-|  
|[BEGIN\_SINK\_MAP](../Topic/BEGIN_SINK_MAP.md)|将事件接收器映射的开头该复合控件的。|  
|[END\_SINK\_MAP](../Topic/END_SINK_MAP.md)|将事件接收器映射的关闭该复合控件的。|  
|[SINK\_ENTRY](../Topic/SINK_ENTRY.md)|向事件接收器映射的项。|  
|[SINK\_ENTRY\_EX](../Topic/SINK_ENTRY_EX.md)|向事件接收器映射的项与一个额外的参数。|  
|[SINK\_ENTRY\_INFO](../Topic/SINK_ENTRY_INFO.md)|向事件接收器映射的项与手动所提供的类型信息用于 [IDispEventSimpleImpl](../../atl/reference/idispeventsimpleimpl-class.md)的使用。|
