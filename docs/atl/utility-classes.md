# Utility Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

提供下列MFC独立实用工具选件类:  
  
-   [CImage](../atl-mfc-shared/reference/cimage-class.md) 提供增强的位图在JPEG、GIF、BMP和可移植网络映像\(PNG\)格式支持，包括能够加载和保存图像。  
  
-   [CPoint](../atl-mfc-shared/reference/cpoint-class.md) 用于存储坐标提供的实现\(x，y\)对。  
  
-   [CRect](../atl-mfc-shared/reference/crect-class.md) 用于存储矩形区域坐标提供的实现。  
  
-   [CSize](../atl-mfc-shared/reference/csize-class.md) 用于存储距离、相对位置或匹配的值提供的实现。  
  
-   [CString](../atl-mfc-shared/reference/cstringt-class.md) 用于存储字符字符串提供的实现。  
  
-   用于的[CAdapt](../atl/reference/cadapt-class.md) 的简单模板包装重新定义运算符地址的选件类。  
  
-   [\_U\_RECT](../atl/reference/u-rect-class.md) 允许 `RECT` 指针或引用传递给实现基于指针的参数适配器选件类。
