# <a name="expression-evaluator-error-cxx0041"></a>表达式计算器错误 CXX0041
类元素必须是静态成员或成员函数  
  
 类 （或结构或联合） 的非静态成员而无需指定的实例化的类用于使用。  
  
 而无需指定实例化，可以使用仅静态数据成员或成员函数。  
  
 此错误是与 CAN0041 相同。
