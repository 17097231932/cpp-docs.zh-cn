# <a name="ml-fatal-error-a1016"></a>ML 错误 A1016

**出现内部汇编程序错误**

MASM 驱动程序，名为 ML.exe，生成系统错误。

记录错误发生的情况，并通知 Microsoft Corporation。 产品支持服务位于[ http://support.microsoft.com/ ](http://support.microsoft.com/)。
