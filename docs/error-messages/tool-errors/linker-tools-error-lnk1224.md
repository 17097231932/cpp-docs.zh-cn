# <a name="linker-tools-error-lnk1224"></a>链接器工具错误 LNK1224

无效的映像基址

你为映像指定了无效的基址。 基址的大小必须为64KB， (最后四个十六进制数字必须为零) 并且必须适合32位有符号或无符号值。
