# <a name="dd"></a>DD
分配和 （可选） 初始化双字 （4 个字节） 存储的每个`initializer`。 `DD`同义词[DWORD](../../assembler/masm/dword.md)。  
  
## <a name="syntax"></a>语法  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>备注  
 此外可作为类型说明符类型是合法的任何位置。
