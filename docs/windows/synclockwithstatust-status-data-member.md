# SyncLockWithStatusT::status_ 数据成员
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```  
DWORD status_;  
```  
  
## 备注  
 在基于当前 SyncLockWithStatusT 对象的锁的操作之后保存着基础等待操作的结果。
