# <a name="compiler-warning-level-1-c4153"></a>编译器警告（等级 1）C4153

表达式中的函数/数据指针转换

在函数指针与数据指针之间转换。 在 Microsoft 扩展 (/Ze) 下允许此类转换，但在 ANSI C 下则不允许。
