# 编译器错误 C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 不能用作内部函数  
  
 `intrinsic` 或 `function` 杂注列出了内部形式不可用的函数。  例如，在编译使用 \/clr 编程的程序时某些内部函数不可用。
