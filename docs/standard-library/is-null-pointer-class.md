# <a name="isnullpointer-class"></a>is_null_pointer 类
测试类型是否为 std::nullptr_t。  
  
## <a name="syntax"></a>语法  
  
```
template <class T>
struct is_null_pointer;
```  
  
#### <a name="parameters"></a>参数  
 `T`  
 要查询的类型。  
  
## <a name="remarks"></a>备注  
 如果类型 `T` 是 `std::nullptr_t`，则类型谓词的实例为 true；否则为 false。  
  
## <a name="requirements"></a>要求  
 **标头：**\<type_traits>  
  
 **命名空间：** std
