# 编译器错误 C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“type library”: 类型库看上去似乎是为 32 位指针生成的；请更改“ptrsize”限定符  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 指令的 `ptrsize` 特性提供的显式信息与编译器在类型库中找到的信息不一致。
