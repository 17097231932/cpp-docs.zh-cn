# <a name="compiler-error-c2360"></a>编译器错误 C2360
identifier 的初始化已跳过 case 标签  
  
 初始化`identifier`可以跳过中`switch`语句。 除非声明将封闭的块中，不能跳过具有初始值设定项的声明。 (除非它被声明在块内，始终在范围变量的结束之前一直`switch`语句。)  
  
 下面的示例生成 C2360:  
  
```  
// C2360.cpp  
int main() {  
   int x = 0;  
   switch ( x ) {  
   case 0 :  
      int i = 1;  
      { int j = 1; }  
   case 1 :   // C2360  
      int k = 1;  
   }  
}  
```  
  
 可能的解决方法：  
  
```  
// C2360b.cpp  
int main() {  
   int x = 0;  
   switch ( x ) {  
   case 0 :  
      { int j = 1; int i = 1;}  
   case 1 :  
      int k = 1;  
   }  
}  
```
