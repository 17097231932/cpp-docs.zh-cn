# <a name="project-build-error-prj0027"></a>项目生成错误 PRJ0027
Unicode 日志消息内容包含无法转换为用户的 ANSI 代码页的内容。  
  
 通常，你将仅看到结合中创建批处理和/或响应文件的错误与此警告。  
  
 此错误的解决方法是更新以使用 ANSI 或要在你的计算机上安装的代码页，并将其设置为系统的默认的生成日志的内容。
