# 编译器错误 C2510
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：“::”的左边必须是类\/结构\/联合  
  
 类、结构或联合名称必须显式在范围解析运算符（`::`）运算符的左侧。
