# <a name="linker-tools-warning-lnk4002"></a>链接器工具警告 LNK4002

对象中定义的符号

以修饰形式显示的符号在 `object`中的未修饰形式中指定，但找不到修饰符号的唯一匹配项。 此警告前面总是出现警告[LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) ，并后跟严重错误[LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)。
