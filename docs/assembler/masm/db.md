# DB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化一个字节每 `initializer`的存储。  `DB` 是 [字节](../../assembler/masm/byte-masm.md)同义词  
  
## 语法  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## 备注  
 ，因为类型说明符任何位置类型是非法的，也可以使用。
