# 链接器工具警告 LNK4096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/BASE 值“number”对于 Windows 95 和 Windows 98 无效；映像可能不能运行  
  
 指定的基址无效。  Windows 95 和 Windows 98 可执行文件的基址必须大于 0x400000。  有关基址的更多信息，请参见 [\/BASE](../../build/reference/base-base-address.md) 链接器选项。
