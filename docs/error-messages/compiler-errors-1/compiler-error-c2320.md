# 编译器错误 C2320
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

应在访问说明符“specifier”的后面输入“:”  
  
 关键字 `public`、`protected` 或 `private` 必须后接一个冒号。
