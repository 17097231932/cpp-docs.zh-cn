# <a name="custom-color-selector-dialog-box-image-editor-for-icons"></a>“自定义颜色选择器”对话框（图标的图像编辑器）
此对话框中，可自定义用于你的映像的颜色。 有关详细信息，请参阅[自定义或更改颜色](../windows/customizing-or-changing-colors-image-editor-for-icons.md)。  
  
 **渐变颜色显示**  
 更改所选颜色的值。 你想要更改位置十字线的颜色。 然后移动滑块向上或向下更改的亮度或 RGB 颜色值。  
  
 **亮度栏**  
 设置为您在渐变颜色显示中选择的颜色亮度。 单击并拖动大于亮度的栏的白色箭头键或向下小于。 请注意，颜色框中显示所选的颜色和亮度你设置的影响。  
  
 **颜色**  
 列出你定义的颜色的色调 （颜色滚轮值）。 值的范围从 0 到 240，其中 0 为红色、 60 为黄色、 120 是绿色、 180 是青色、 200 代表洋红色，和 240 为蓝色。  
  
 **色调**  
 列出你定义的颜色的色调 （颜色滚轮值）。 值的范围从 0 到 240，其中 0 为红色、 60 为黄色、 120 是绿色、 180 是青色、 200 代表洋红色，和 240 为蓝色。  
  
 **周六至**  
 指定要定义的颜色饱和度值。 饱和度是量中指定的色调颜色。 值的范围从 0 到 240。  
  
 **亮度**  
 列出你定义的颜色亮度 （亮度）。 值的范围从 0 到 240。  
  
 **红色**  
 指定要定义的颜色的红色值。 值的范围介于 0 到 255 之间。  
  
 **绿色**  
 指定要定义的颜色的绿色值。 值的范围介于 0 到 255 之间。  
  
 **蓝色**  
 指定要定义的颜色的蓝色值。 值的范围介于 0 到 255 之间。
