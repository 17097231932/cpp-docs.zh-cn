# 资源编译器错误 RC1049
无效的数值参数“字符串”  
  
 资源编译器需要数值参数，但接收到了给定的字符串。  
  
 指定十六进制数时未提供必要的 \\x 前缀或浮点数格式不正确，都可能导致此错误。
