# <a name="commonly-added-member-functions"></a>经常添加的成员函数
如果对话框中包含确定或取消以外的按键，你需要对它们生成控件通知消息做出响应对话框类中编写消息处理程序成员函数。 有关示例，请参阅[Scribble](../visual-cpp-samples.md)示例程序。 在对话框中，你还可以处理从其他控件的控件通知消息。
