# <a name="expression-evaluator-error-cxx0058"></a>表达式计算器错误 CXX0058
找不到的重载的运算符  
  
 类类型指定为左操作数在表达式中，但没有为类定义重载的运算符。  
  
 此错误是与 CAN0058 相同。
