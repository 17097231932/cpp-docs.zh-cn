# 使用 CTabCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

选项卡控件类似于笔记本中的分隔卡或档案柜中的标签目录。  使用选项卡控件，由 [CTabCtrl](../mfc/reference/ctabctrl-class.md)类，可显示信息或控件的多个页。用户，一次一个，即使在建议对等类或逻辑关系的各页之间的格式。  
  
 有关选项卡控件的更多信息，请参见" [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)]的 [选项卡控件](http://msdn.microsoft.com/library/windows/desktop/bb760550)。
