# <a name="microsoft-macro-assembler-reference"></a>Microsoft Macro Assembler reference

Microsoft 宏汇编 (MASM) 提供了一些相对于内联程序集的优点。 MASM 包含具有循环、算术和文本字符串处理等功能的宏语言。 MASM 还可以使你更好地控制硬件，因为它支持 386、486 和 Pentium 处理器的指令集。 通过使用 MASM，还可以节省时间和减少内存开销。
