# <a name="openmp-in-visual-c"></a>Visual C++ 中的 OpenMP

通过 OpenMP C 和 c + + 应用程序界面，你可以编写有效地使用多个处理器的应用程序。 Visual C++ 支持 OpenMP 2.0 标准。
