# Unable to customize.
调试应用程序时，如果尝试自定义菜单或工具栏，则会发生此错误。  
  
### 更正此错误  
  
-   等待调试完成，然后进行工具栏或菜单的自定义。  
  
     \- 或 \-  
  
     停止调试，然后进行工具栏或菜单的自定义。
