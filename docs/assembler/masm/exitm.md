# <a name="exitm"></a>EXITM

终止当前重复或宏块的扩展，并在块外开始下一个语句的程序集。

## <a name="syntax"></a>语法

> **EXITM** ⟦*textitem*⟧

## <a name="remarks"></a>备注

在宏函数中， *textitem* 是返回的值。
