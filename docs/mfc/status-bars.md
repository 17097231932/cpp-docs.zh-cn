# <a name="status-bars"></a>状态栏

状态条为应用程序提供一个位置来向用户显示消息和有用的信息而不会中断用户的工作。 状态栏通常显示在窗口的底部，有"窗格，"包括"指示器"和"消息行"。 指示器提供的滚动锁等内容的状态是否宏录制已打开上，依次类推。 状态栏上的消息行可以显示有关程序状态或用户使用鼠标指向工具栏按钮或菜单项的信息。

通过选择在您的程序中创建一个状态栏**初始状态栏**MFC 应用程序向导中的选项。
