# <a name="project-build-error-prj0026"></a>项目生成错误 PRJ0026

> 响应文件 "*file*" 包含无法翻译成用户的 ANSI 代码页的 Unicode 内容。
>
> *文件的 UNICODE 内容*

项目系统在无法正确转换为用户的当前 ANSI 代码页的响应文件中发现了 Unicode 内容。

此错误的解决方法是更新响应文件的内容以使用 ANSI 或在计算机上安装代码页，并将其设置为系统默认值。
