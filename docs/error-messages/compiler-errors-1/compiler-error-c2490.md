# <a name="compiler-error-c2490"></a>编译器错误 C2490
keyword 与 naked 特性的函数中不允许  
  
 函数定义为[裸](../../cpp/naked-cpp.md)不能使用结构化的异常处理。  
  
 下面的示例生成 C2490:  
  
```  
// C2490.cpp  
// processor: x86  
__declspec( naked ) int func() {  
   __try{}   // C2490, structured exception handling  
}  
```
