# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

停止当前重复展开或宏块并开始下一条语句的程序集块之外的。  
  
## 语法  
  
```  
  
EXITM [[textitem]]   
```  
  
## 备注  
 在宏， *textitem* 是返回的值。
