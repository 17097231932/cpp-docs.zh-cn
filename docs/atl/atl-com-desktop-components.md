# <a name="atl-com-desktop-components"></a>ATL COM 桌面组件

ATL 引用记录了活动模板库 (ATL) ，这是一组基于模板的 c + + 类，用于简化组件对象模型 (COM) 对象的编程。 COM 是在 Windows 上创建和使用软件组件的二进制规范。 若要充分利用 ATL，强烈建议使用 COM。 有关 COM 的详细信息，请参阅 [组件对象模型 (COM) ](/windows/win32/com/component-object-model--com--portal)。
