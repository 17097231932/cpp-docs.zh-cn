# Wizard file &lt;name&gt; format is not recognized.
无法打开向导。  此错误通常由损坏的向导文件或不匹配 Visual Studio 版本号的向导引起。  
  
### 更正此错误  
  
1.  验证向导文件的版本号与 Visual Studio 的版本相同，该版本号作为 VSWIZARD 的值列出，例如 `VSWIZARD 7.0`。  
  
     \- 或 \-  
  
     重新安装 Visual Studio 以修复损坏的文件。
