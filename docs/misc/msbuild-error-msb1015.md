# MSBuild 错误 MSB1015
**MSBuild 不在操作系统的此版本上运行。  它仅在 Windows 2000、Windows XP 和更高版本上受支持。**  
  
 操作系统的此版本不支持 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]。  
  
### 更正此错误  
  
1.  在一个支持的操作系统版本上安装和运行 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]。
