# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回 true \(– 1\)，如果 *expression1* 不等于 *expression2*，或者返回错误 \(0\)，则它。  
  
## 语法  
  
```  
  
expression1 NE expression2  
```
