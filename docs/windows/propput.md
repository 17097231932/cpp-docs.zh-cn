# propput
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指定属性设置功能。  
  
## 语法  
  
```  
  
[propput]  
  
```  
  
## 备注  
 **propput** C\+\+ 特性具有与 [propput](http://msdn.microsoft.com/library/windows/desktop/aa367146) MIDL 特性相同的功能。  
  
## 示例  
 请参见 [bindable](../windows/bindable.md) 示例以了解 **propput** 的示例使用。  
  
## 要求  
  
### 特性上下文  
  
|||  
|-|-|  
|**适用对象**|方法|  
|**可重复**|No|  
|**必需的特性**|无|  
|**无效的特性**|**propget**、**propputref**|  
  
 有关特性上下文的详细信息，请参见[特性上下文](../windows/attribute-contexts.md)。
