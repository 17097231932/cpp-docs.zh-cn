# <a name="ismbbgraph-ismbbgraphl"></a>_ismbbgraph、_ismbbgraph_l

确定特定多字节字符是否为图形字符。

## <a name="syntax"></a>语法

```C
int _ismbbgraph (
   unsigned int c
);
int _ismbbgraph_l (
   unsigned int c,
   _locale_t locale
);
```

### <a name="parameters"></a>参数

*c*<br/>
要测试的整数。

*locale*<br/>
要使用的区域设置。

## <a name="return-value"></a>返回值

如果表达式满足以下条件，则返回非零值：

`isctype(c, ( _PUNCT | _UPPER | _LOWER | _DIGIT )) || _ismbbkprint(c)`

为非零*c*，或者，如果它不是 0。 **_ismbbgraph**的任何区域设置相关行为使用当前区域设置。 **_ismbbgraph_l**是完全相同，只不过它改用已传入的区域设置。 有关详细信息，请参阅 [Locale](../../c-runtime-library/locale.md)。

## <a name="requirements"></a>要求

|例程所返回的值|必需的标头|
|-------------|---------------------|
|**_ismbbgraph**|\<mbctype.h>|
|**_ismbbgraph_l**|\<mbctype.h>|
