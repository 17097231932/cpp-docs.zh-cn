# 添加成员函数 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在类视图中，您可以将成员函数添加到任何类中。  当您这么做时，头文件中将添加一个声明，并且添加成员函数体到类实现文件中，可以编辑该文件。  
  
### 向类添加成员函数  
  
1.  在**“类视图”**中，展开项目节点以显示项目中的类。\(若要打开类视图 ，在菜单栏上，选择视图 ，类视图。\)  
  
2.  打开要将成员函数添加到的选件类的快捷菜单，然后选择 添加，添加函数。  
  
3.  提供有关成员函数的适当详细。  有关详细信息，请参阅[添加成员函数向导](../ide/add-member-function-wizard.md)。  
  
4.  选择 完成 按钮生成成员函数代码。
