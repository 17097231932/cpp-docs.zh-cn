# Site Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类允许对象与其站点通信:  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) 检索和设置指向对象的站点。  用于不是控件的对象。  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 检索和设置指向对象的站点。  使用为控件。
