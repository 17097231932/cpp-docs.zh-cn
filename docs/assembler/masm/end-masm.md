# <a name="end-masm"></a>END (MASM)

标记模块的末尾，并选择性地将程序入口点设置为*address*。

## <a name="syntax"></a>语法

> **结束**⟦*地址*⟧

## <a name="remarks"></a>备注

*Address*参数仅在32位 MASM 中有效。
