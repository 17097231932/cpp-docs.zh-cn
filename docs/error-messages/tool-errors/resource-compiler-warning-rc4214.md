# <a name="resource-compiler-warning-rc4214"></a>资源编译器警告 RC4214
代码页无效： 忽略  
  
 .Rc 文件包含的代码页参数和指定的代码页无效。 请参阅[IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674)有关详细信息。
