# <a name="status-bars"></a>状态栏

通过状态栏，你的应用程序可以向用户显示消息和有用信息，而不会中断用户的工作。 状态栏通常显示在窗口的底部，状态栏包含 "窗格"，其中包括 "指示器" 和 "消息行"。 指示器会将此类事件的状态视为滚动锁，宏录制是否打开等。 状态栏上的消息行可以显示有关程序状态的信息，或有关用户通过鼠标指向的工具栏按钮或菜单项的信息。

在 "MFC 应用程序向导" 中选择 " **初始状态栏** " 选项，以在程序中创建状态栏。
