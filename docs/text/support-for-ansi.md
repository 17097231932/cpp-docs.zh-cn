# <a name="support-for-ansi"></a>对 ANSI 的支持
大多数 MFC 类和方法支持 ANSI 字符集，尽管 MFC 框架作为一个整体逐渐不断向支持仅 Unicode 字符集。 由于中正在进行的增强功能[!INCLUDE[windowsver](../build/reference/includes/windowsver_md.md)]和 Windows 公共控件版本 6.1、 支持多个 ANSI 类和方法已弃用。  有关详细信息，请参阅[弃用的 ANSI Api](../mfc/deprecated-ansi-apis.md)和[有关 Unicode 的支持](../text/support-for-unicode.md)。
