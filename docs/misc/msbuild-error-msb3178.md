# MSBuild 错误 MSB3178
**MSB3178: 程序集“\<assembly\>”被错误地指定为一个文件。**  
  
 此警告会在生成应用程序清单时出现以下情况时产生：生成过程检测到文件引用实际为一个（托管或本机）程序集引用。
