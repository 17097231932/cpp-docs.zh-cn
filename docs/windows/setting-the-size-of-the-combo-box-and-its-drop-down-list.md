# <a name="setting-the-size-of-the-combo-box-and-its-drop-down-list"></a>设置组合框及其下拉列表的大小
时将其添加到对话框中，你可以调整大小的组合框。 你还可以指定下拉列表框的大小。  
  
### <a name="to-size-a-combo-box"></a>若要调整大小的组合框  
  
1.  在对话框中选择组合框控件。  
  
     最初，仅左侧和右侧的调整大小控点处于活动状态。  
  
2.  调整大小控点用于设置组合框的宽度。  
  
 你还可以设置组合框的下拉部分的垂直大小。  
  
#### <a name="to-set-the-size-of-the-combo-box-drop-down-list"></a>若要设置的大小的组合框下拉列表  
  
1.  单击组合框右侧的下拉箭头按钮。  
  
     ![MFC 项目中组合框上的箭头](../mfc/media/vccomboboxarrow.gif "vcComboBoxArrow")  
  
     轮廓的控件更改，以显示随着下拉列表框区域的扩展组合框的大小。  
  
2.  使用较低的尺寸控点来更改下拉列表框区域的初始大小。  
  
     ![组合 &#45; 框 MFC 项目中的大小调整](../mfc/media/vccomboboxsizing.gif "vcComboBoxSizing")  
  
3.  单击下拉箭头以关闭组合框的下拉列表框部分。
