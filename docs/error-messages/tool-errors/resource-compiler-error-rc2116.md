# 资源编译器错误 RC2116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

应输入 ID 号  
  
 应为 **DIALOG** 语句中控制语句的 `id` 字段输入数字。  确保此控件 ID 有数字或 `#define` 语句。
