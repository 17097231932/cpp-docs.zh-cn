# 链接器工具错误 LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法推断出子系统，必须定义它  
  
 链接器没有足够的信息推断出您要作为目标的子系统。  
  
 若要修复此错误，请使用 [\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)。
