# <a name="resource-compiler-fatal-error-rc1101"></a>资源编译器错误 RC1101

> 未指定资源二进制文件名

重命名输出 (/fo) 选项后面不是文件名。

对于/fo 选项，请使用以下语法：

> **Rc/fo** *outfilename* <em>infilename</em>
