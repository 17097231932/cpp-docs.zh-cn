# <a name="omp-pragma"></a>omp 杂注

将一个或多个 OpenMP 指令与任何可选指令子句一起使用。

## <a name="syntax"></a>语法

> **#pragma omp***指令*

## <a name="remarks"></a>备注

有关详细信息, 请参阅[OpenMP 指令](../parallel/openmp/reference/openmp-directives.md)。
