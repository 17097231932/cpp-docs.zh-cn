# <a name="compiler-error-c2850"></a>编译器错误 C2850

construct： 仅允许在文件范围内;可能无法在嵌套结构

构造，例如某些杂注只能出现在全局范围内。

下面的示例生成 C2850:

```
// C2850.cpp
// compile with: /c /Yc
// try the following line instead
// #pragma hdrstop
namespace X {
   #pragma hdrstop   // C2850
};
```
