# <a name="project-build-error-prj0027"></a>项目生成错误 PRJ0027

Unicode 日志消息内容包含未能翻译成用户的 ANSI 代码页的内容。

通常将仅看到结合此警告在创建 batch 和/或响应文件时出错。

此错误的解决方法是更新以使用 ANSI 或要在计算机上安装的代码页，并将其设置为系统默认的生成日志的内容。
