# <a name="nmake-fatal-error-u1086"></a>NMAKE 错误 U1086

**推理规则不能有从属项**

冒号 (**:**) 在推理规则必须跟有其中一种：

- 换行字符

- 以分号 (**;**)，可以跟命令

- 数字符号 (**#**)，可以跟一条注释
