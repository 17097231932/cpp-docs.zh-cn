# <a name="hlsl-property-pages-advanced"></a>HLSL 属性页：高级
若要配置 HLSL 编译器 (fxc.exe) 的以下属性，使用其**高级**属性页。 有关如何访问**高级**HLSL 文件夹中的属性页上看到[使用项目属性](../ide/working-with-project-properties.md)。  
  
## <a name="uielement-list"></a>UIElement 列表  
 **取消显示启动版权标志**  
 **是 (/ nologo)**若要取消显示启动版权标志和信息消息; 否则为**否**。 默认情况下，值是**是 (/ nologo)**。  
  
 **将警告视为错误**  
 **是 (/WX)**将所有的编译器警告视为错误; 否则为**否**。
