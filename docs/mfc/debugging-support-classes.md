# <a name="debugging-support-classes"></a>调试支持类

MFC 提供了以下类来帮助调试动态内存分配问题。

[CDumpContext](reference/cdumpcontext-class.md)<br/>
提供诊断转储的目标。

[CMemoryState](reference/cmemorystate-structure.md)<br/>
提供内存使用情况快照的结构。 还用于比较以前的和更高版本的内存快照。
