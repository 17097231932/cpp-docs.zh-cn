# <a name="attributes-by-group"></a>按组分的特性
C + + 特性分为以下功能组。  
  
|特性|描述|  
|---------------|-----------------|  
|[COM 特性](../windows/com-attributes.md)|插入代码以支持的 COM 开发和.NET Framework 公共语言运行时开发的许多方面。|  
|[IDL 特性](../windows/idl-attributes.md)|启用从源代码文件而不使用向导或不包含结构的知识和该文件的语法在.idl 文件的修改。|  
|[OLE DB 使用者特性](../windows/ole-db-consumer-attributes.md)|将代码注入-基于[OLE DB 使用者模板](../data/oledb/ole-db-consumer-templates-reference.md)-若要创建使用 OLE DB 使用者执行任务，例如打开表，执行命令和访问数据。|  
|[编译器特性](../windows/compiler-attributes.md)|由 Visual c + + 编译器提供。|  
  
 有关如何定义为.NET Framework 应用程序的自定义特性的信息，请参阅[用户定义的特性](../windows/user-defined-attributes-cpp-component-extensions.md)。
