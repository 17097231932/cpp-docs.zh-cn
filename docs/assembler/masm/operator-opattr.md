# <a name="operator-opattr"></a>运算符 OPATTR

返回用于定义 *表达式* 的模式和作用域的单词。 低字节与返回的字节完全相同 [。键入](operator-dot-type.md)。 高字节包含附加信息。

## <a name="syntax"></a>语法

> **OPATTR** *表达式*
