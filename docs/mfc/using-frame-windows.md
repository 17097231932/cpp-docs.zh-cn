# 使用框架窗口
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC 框架创建文档框架窗口 — 及其视图和文档 — 作为文件菜单新建和打开命令实现的一部分。  因为框架为您完成大部分的框架窗口工作，在创建、使用和销毁工作中您的工作很少。  但是，您可以显式创建自己的框架窗口和基于特定目的的子窗口。
