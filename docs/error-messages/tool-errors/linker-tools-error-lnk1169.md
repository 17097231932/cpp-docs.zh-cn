# 链接器工具错误 LNK1169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找到一个或多个多重定义的符号  
  
 生成因一个或多个符号的多个定义而失败。  该错误之前为错误 [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)。  
  
 [\/FORCE](../../build/reference/force-force-file-output.md) 或 **\/FORCE:MULTIPLE** 选项将重写此错误。
