# <a name="compiler-warning-level-1-c4034"></a>编译器警告 (等级 1) C4034

sizeof 返回0

**`sizeof`** 运算符应用于大小为零 (空结构、联合、类或枚举类型或类型) 的操作数 **`void`** 。
