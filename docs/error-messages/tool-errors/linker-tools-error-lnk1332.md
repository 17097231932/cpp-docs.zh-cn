# 链接器工具错误 LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

检测到\<count\>在一个模块中导入并在另一个模块中定义的 个 Windows 运行时类型  
  
 在产生了当前目标，链接器检测到 \<Windows`count`\> 运行时类型，其中每个类型一个模块。另一模块中导入和定义。  
  
### 更正此错误  
  
-   按错误消息提供更正建议的每个版本中 LNK2039 错误。
