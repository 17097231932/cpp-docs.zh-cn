# <a name="fatal-error-c1055"></a>错误 C1055
编译器限制： 超出键  
  
 源文件包含太多的符号。 编译器用尽了符号表的哈希键。  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用以下可能的解决方案进行修复  
  
1.  源文件拆分成较小的文件。  
  
2.  消除不必要的标头文件。  
  
3.  重复使用而不是创建新的临时和全局变量。
