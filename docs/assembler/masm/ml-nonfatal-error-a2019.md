# <a name="ml-nonfatal-error-a2019"></a>ML 非致命错误 A2019

**操作数必须是记录类型或字段**

[WIDTH](operator-width.md)或[MASK](operator-mask.md)运算符后面的操作数无效。

[WIDTH](operator-width.md)运算符采用作为字段或记录的名称的操作数。 [掩码](operator-mask.md)运算符采用一个表示字段或记录类型名称的操作数。
