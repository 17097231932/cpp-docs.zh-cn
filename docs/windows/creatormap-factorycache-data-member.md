# <a name="creatormapfactorycache-data-member"></a>CreatorMap::factoryCache 数据成员
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
FactoryCache* factoryCache;  
```
