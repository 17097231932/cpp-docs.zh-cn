# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**只能对齐的次幂 2:表达式**  
  
 该表达式指定与 [对齐](../../assembler/masm/align-masm.md) 指令无效。  
  
 **对齐** 表达式必须是 2 的次幂在 2 和 256 之间，并且必须小于或等于当前段落、结构或联合的对齐方式。
