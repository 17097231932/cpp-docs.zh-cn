# 资源编译器错误 RC2146
LANGUAGE 语句中缺少逗号  
  
 缺少分隔的主要语言和次要语言的逗号。  
  
 **LANGUAGE** 语句必须使用以下语法:  
  
 **LANGUAGE** *primary\_language\_ID*、*secondary\_language\_ID*
