# <a name="compiler-warning-level-4-c4513"></a>编译器警告（等级 4）C4513

class： 未能生成析构函数

编译器无法生成默认析构函数为给定的类;创建没有析构函数。 析构函数是不能由派生类访问基类中。 如果基类具有私有析构函数，使其公共或受保护。
