# <a name="nmake-fatal-error-u1078"></a>NMAKE 错误 U1078
在表达式的常量溢出  
  
 给定的表达式中包含超出范围-2147483648 到 2147483647 的常数。 该常数出现在以下情况之一︰  
  
-   预处理指令与指定的表达式  
  
-   用短划线指定的错误级别 (**-**) 命令修饰符
