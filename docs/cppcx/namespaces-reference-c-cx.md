# 命名空间参考 (C++/CX)
本节文档中的文章介绍支持 [!INCLUDE[cppwrt](../cppcx/includes/cppwrt-md.md)]\([!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)]\) 的编译器的命名空间。  
  
## 编译器提供的命名空间  
 为简化面向 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]的程序代码，[!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)] 编译器及其支持的标头文件提供定义多种类型的命名空间。 命名空间定义内置数值类型、字符串、数组、集合、表示 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]错误的 Visual C\+\+ 异常、对标准 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]类型的特定语言增强。  
  
## 相关主题  
  
|标题|描述|  
|--------|--------|  
|[default 命名空间](../cppcx/default-namespace.md)|包含对内置基本类型的描述。|  
|[Platform 命名空间](../cppcx/platform-namespace-c-cx.md)|包含对可用类型以及仅供编译器基础结构使用的内部类型的描述。|  
|[Windows::Foundation::Collections 命名空间](../cppcx/windows-foundation-collections-namespace-c-cx.md)|包含对 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)]`Windows::Foundation::Collections` 命名空间所做的增强和扩展的描述。|
