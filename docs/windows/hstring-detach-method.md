# <a name="hstringdetach-method"></a>HString::Detach 方法
解除指定的关联**HString**从其基础值的对象。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HSTRING Detach() throw()  
```  
  
## <a name="return-value"></a>返回值  
 基础**HString**分离操作启动之前的值。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
