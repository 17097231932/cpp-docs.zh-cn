# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS：

函数 "% function%" 的指针参数 "% argument%" 可标记为指向 `const`的指针。 请参阅[ C++核心准则 con。 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref)。
