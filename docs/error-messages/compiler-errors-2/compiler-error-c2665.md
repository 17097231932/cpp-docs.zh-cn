# <a name="compiler-error-c2665"></a>编译器错误 C2665
function： 无 number1 重载可以从类型 type 转换参数 number2  
  
 重载函数的参数无法转换为所需的类型。  可能的解决方法：  
  
-   提供一个转换运算符。  
  
-   使用显式转换。  
  
## <a name="example"></a>示例  
 下面的示例生成 C2665。  
  
```  
// C2665.cpp  
void func(short, char*){}  
void func(char*, char*){}  
  
int main() {  
   func(0, 1);   // C2665  
   func((short)0, (char*)1);   // OK  
}  
```
