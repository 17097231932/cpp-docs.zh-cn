# <a name="project-build-warning-prj0042"></a>项目生成警告 PRJ0042

> 文件的自定义生成步骤的输出属性*文件*未设置。 将跳过自定义生成步骤。

未执行的自定义生成步骤，因为未指定输出。

若要解决此错误，请执行下列操作之一：

- 从生成中排除自定义生成步骤。

- 添加输出。

- 删除自定义生成步骤的命令的内容。
