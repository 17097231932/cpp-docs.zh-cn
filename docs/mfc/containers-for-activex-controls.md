# <a name="containers-for-activex-controls"></a>ActiveX 控件的容器

您可以在其他应用程序中使用 Visual C++ 中开发的 ActiveX 控件，只要它们支持 ActiveX 控制控制控制。 许多 Microsoft 应用程序（从列出的版本开始）支持 ActiveX 控制控制。

>[!IMPORTANT]
> ActiveX 是一种不应用于新开发的传统技术。 有关取代 ActiveX 的现代技术的详细信息，请参阅[ActiveX 控件](activex-controls.md)。

> [!NOTE]
> 以下列表不是支持 ActiveX 控件但表示测试中最常用的集的应用程序的完整列表：

支持 ActiveX 控制控制包含的应用程序包括：

- 互联网浏览器 3.x 及更高

- 视觉基本 4.x 及更高

- 视觉C++ 4.x 及更高

- 访问 95 及更高

- Excel 97 及更高

- 字 97 和更高

- 访问 97 及更高

- 前页 97 及更高

- PowerPoint 97 及更高

- 视觉 InterDev 97 及更高

以下是支持 ActiveX 控制遏制的非 Microsoft 应用程序：

- PowerBuilder

- 德 尔 福

- C++建设者

- 网景导航器的 NCompass 插件
