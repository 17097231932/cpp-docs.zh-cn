# <a name="resource-compiler-error-rc2152"></a>资源编译器错误 RC2152

控制字符无效

**ACCELERATORS** 语句中的控制字符无效。 有效控制字符包含一个字母 (仅) 在插入符号 () 之后 **^** 。
