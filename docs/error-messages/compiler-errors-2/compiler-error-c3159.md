# <a name="compiler-error-c3159"></a>编译器错误 C3159
指针︰ 不能声明为指向值类型的指针的数组  
  
 不能声明为值类型的指针的数组。  
  
 C3159 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
