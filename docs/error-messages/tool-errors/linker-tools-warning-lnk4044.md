# <a name="linker-tools-warning-lnk4044"></a>链接器工具警告 LNK4044

无法识别的选项 option;忽略

给定的选项不是此工具的有效选项。 该工具将忽略该选项。

若要查看有效的选项列表，请不带任何参数或选项运行该工具。
