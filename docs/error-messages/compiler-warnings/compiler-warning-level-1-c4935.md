# <a name="compiler-warning-level-1-c4935"></a>编译器警告（等级 1）C4935
程序集访问说明符已从“access”修改  
  
已修改一个类型的程序集可见性。 编译器将使用它所遇到的最后一个说明符。 例如，前向声明的程序集可见性可能与类定义的程序集可见性不同。  
  
C4935 才可访问使用过时的编译器选项 **/clr:oldSyntax**。
