# OLE DB 模板
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

OLE DB 模板使用 OLE DB 数据访问更容易。  Visual C\+\+ 提供实现许多常用 OLE DB 接口的模板类。  可以使用 Visual C\+\+ 使用者模板编写 OLE DB 客户 \(使用者\) 应用程序和提供程序模板编写服务器 \(提供程序\) 应用程序。  
  
## 本节内容

 提供指向使用条件列出的 OLE DB 使用者模板类。  
  
 [OLE DB 提供程序模板](../../data/oledb/ole-db-provider-templates-reference.md)
