# EventTargetArray::Length 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```

```

 获取元素的当前数目在内部数组的事件处理程序。
