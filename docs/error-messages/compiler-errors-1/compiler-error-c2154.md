# <a name="compiler-error-c2154"></a>编译器错误 C2154

“type”：只有枚举类型才允许作为编译器内部类型特征“__underlying_type”的自变量

你只能获取枚举类型的基础类型。

有关详细信息，请参阅[编译器对类型特征的支持](../../extensions/compiler-support-for-type-traits-cpp-component-extensions.md)。
