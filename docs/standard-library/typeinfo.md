# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;

包括标准标头\<typeinfo > 来定义与类型标识运算符关联的几种类型[typeid 运算符](../cpp/typeid-operator.md)，因而产生有关静态和动态类型的信息。

## <a name="syntax"></a>语法

```cpp
#include <typeinfo>

```
