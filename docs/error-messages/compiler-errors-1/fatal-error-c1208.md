# <a name="fatal-error-c1208"></a>错误 C1208

安装的运行时版本不支持在堆栈上分配引用类

使用的编译器是当前版本而公共语言运行时是早期版本时将发生 C1208。

编译器的一些功能在早期的运行时版本中可能无效。

请安装适用于你的编译器的公共语言运行时版本。
