# DT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化 10 个字节每 `initializer`的存储。  ，因为类型说明符任何位置类型是非法的，也可以使用。  `DT` 是 [TBYTE](../../assembler/masm/tbyte.md)同义词。  
  
## 语法  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
