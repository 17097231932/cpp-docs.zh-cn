# <a name="resource-compiler-fatal-error-rc1102"></a>资源编译器错误 RC1102

内部错误： RCPP 的参数太多

向资源编译器预处理器传递的参数太多。 通过在你的源中定义，使用定义符号 (/d) 选项来减少定义的符号数。 使用 "包括搜索路径" 选项 (/i) 指定太多包含文件搜索路径，也可能导致此错误。
