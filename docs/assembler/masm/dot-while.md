# <a name="while-32-bit-masm"></a>.WHILE （32位 MASM）

当*condition*为 true 时，生成执行*语句*块的代码。 （仅限32位 MASM。）

## <a name="syntax"></a>语法

> **.WHILE** *条件*\
> *语句*\
> **.ENDW**
