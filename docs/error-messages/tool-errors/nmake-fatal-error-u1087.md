# <a name="nmake-fatal-error-u1087"></a>NMAKE 错误 U1087
不能有： 和:: 相同的目标的依赖项  
  
 不在这两个单冒号中指定目标 (**:**)，并使用双冒号 (`::`) 依赖项。  
  
 若要在多个描述块中指定目标，使用`::`在每个依赖项的行。
