# <a name="profile-guided-optimization-warning-pg1039"></a>按配置优化警告 PG1039

未能等到空闲时间以进行扫描。

不允许该应用程序[pgosweep](../../build/pgosweep.md)运行。  当应用程序处于空闲状态时，如果可能，请运行 pgosweep。
