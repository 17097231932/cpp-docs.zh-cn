# <a name="listall"></a>.LISTALL

启动列出的所有语句。

## <a name="syntax"></a>语法

> .LISTALL

## <a name="remarks"></a>备注

等效于的组合[。列表](../../assembler/masm/dot-list.md)， [。列表如果](../../assembler/masm/dot-listif.md)，和[。LISTMACROALL](../../assembler/masm/dot-listmacroall.md)。
