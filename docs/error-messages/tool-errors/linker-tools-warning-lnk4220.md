# <a name="linker-tools-warning-lnk4220"></a>链接器工具警告 LNK4220

"链接器选项" 值 "value" 无效;假定为默认值

使用 [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) 选项指定了超出范围的值。 **/TLBID** 的默认值为1。
