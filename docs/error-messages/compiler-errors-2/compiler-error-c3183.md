# 编译器错误 C3183
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不能在托管的或 WinRT 类型“type”的内部定义未命名的类、结构或联合  
  
 必须命名嵌入托管的或 WinRT 类型中的类型。  
  
 下列示例生成 C3183：  
  
```  
// C3183a.cpp  
// compile with: /clr /c  
ref class Test  
{  
   ref class  
   {  // C3183, delete class or name it  
      int a;  
      int b;  
   };  
};  
```  
  
 下列示例生成 C3183：  
  
```  
// C3183b.cpp  
// compile with: /clr:oldSyntax  
#using <mscorlib.dll>  
  
__gc class Test  
{  
   __gc class  
   {  // C3183, delete class or name it  
      int a;  
      int b;  
   };  
};  
  
int main()  
{  
}  
```
