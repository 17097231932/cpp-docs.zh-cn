# MSBuild 错误 MSB1026
**架构文件不存在。**  
  
 指定位置中不存在架构文件。  
  
### 更正此错误  
  
1.  如果您指定了架构文件的绝对位置或相对位置，请检查指定目录中是否存在构架文件。
