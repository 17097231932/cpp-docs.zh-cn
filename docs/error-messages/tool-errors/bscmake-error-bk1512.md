# <a name="bscmake-error-bk1512"></a>BSCMAKE 错误 BK1512

文件名：超出容量

BSCMAKE 无法生成浏览信息文件，因为定义、引用、模块或其他信息的数目超出了限制。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用以下可能的解决方案进行修复

1. 使用/Em、/Es 或/Ei. 排除某些信息

1. 省略/Iu 选项。
