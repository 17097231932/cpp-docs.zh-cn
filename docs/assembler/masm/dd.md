# <a name="dd"></a>DD

分配并根据需要初始化双字 （4 字节） 存储的每个`initializer`。 `DD` 是的同义词[DWORD](../../assembler/masm/dword.md)。

## <a name="syntax"></a>语法

> [[name]]DD 初始值设定项 [[，初始值设定项]]...

## <a name="remarks"></a>备注

也可以用作类型说明符类型是合法的任何位置。
