# <a name="expression-evaluator-error-cxx0067"></a>表达式计算器错误 CXX0067
不支持的函数求值  
  
 表达式包含函数调用。 某些表达式计算器不支持函数调用。  
  
 此错误是与 CAN0067 相同。
