# <a name="ltciso646gt"></a>&lt;ciso646&gt;

包含标准 C 库标头 \<iso646.h> 并将关联名称添加到 `std` 命名空间。

> [!NOTE]
> 使用此标头是不必要在C++。

## <a name="requirements"></a>要求

**标头：** \<ciso646 >

**命名空间：** std

## <a name="remarks"></a>备注

包含该标头还将确保使用标准 C 库标头中的外部链接声明的名称在 `std` 命名空间中声明。
