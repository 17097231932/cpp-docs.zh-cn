# <a name="resource-compiler-error-rc2170"></a>资源编译器错误 RC2170

位图文件的文件名不是 3.00 格式

使用的 Windows 版本 2.x 格式的位图不能用于版本 3.x 资源文件。

若要解决此问题，请重新绘制位图或将其转换为 3.x 格式。
