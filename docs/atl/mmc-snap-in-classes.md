# MMC Snap-In Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类提供了Microsoft Management Console \(MMC\)管理单元元素支持:  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) 实现管理单元节点对象，例如添加菜单项和工具栏和前进管理单元节点的命令为相应的处理程序函数。  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) 实现管理单元属性页对象。
