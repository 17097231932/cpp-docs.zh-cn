# Can I Reuse a Host Window?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

建议不要重复使用宿主窗口。  若要确保代码的可靠性，应附加生存期您的宿主窗口到单个控件的生存期。
