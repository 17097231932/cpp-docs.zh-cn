# <a name="multibyte-characters"></a>多字节字符

**ANSI 2.2.1.2** 多字节字符的移位状态

多字节字符由某些实现（包括 Microsoft C）用来表示基字符集中未表示的外语字符。 但是，Microsoft C 不支持任何依赖于状态的编码。 因此，没有移位状态。 有关详细信息，请参阅[多字节字符和宽字符](../c-language/multibyte-and-wide-characters.md)。
