# <a name="catstr"></a>CATSTR
串联文本项。  
  
## <a name="syntax"></a>语法  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>备注  
 每个文本项可以是文本字符串，前面是一个常数`%`，或通过宏函数返回的字符串。
