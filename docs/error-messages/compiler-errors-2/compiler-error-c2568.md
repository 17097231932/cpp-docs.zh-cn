# <a name="compiler-error-c2568"></a>编译器错误 C2568

"identifier1"：无法解析函数重载

编译器无法确定要调用的重载函数。 传递给函数的实参必须强制转换为与某个重载函数的形参匹配，但没有任何一个匹配明显比所有其他匹配。
