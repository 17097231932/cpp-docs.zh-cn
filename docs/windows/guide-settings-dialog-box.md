# <a name="guide-settings-dialog-box"></a>“参考线设置”对话框
## <a name="layout-guides"></a>版式参考线  
 显示布局参考线设置。  
  
 **无**  
  
 隐藏的布局工具。  
  
 **标尺和参考线**  
  
 启用时，将标尺添加到的布局工具;指南可以放置在标尺。 默认参考线是的边距，可以通过拖动来移动。 单击标尺放置指南中。 控件时控件移通过或旁边的指南的"管理单元"。 一旦将它们附加到它，控件还通过指南中进行移动。 调整控件的控件附加到每一侧的指南，参考线移动时。  
  
 **网格**  
  
 创建布局网格。 新控件将自动对齐到网格中。  
  
## <a name="grid-spacing"></a>网格间距  
 在对话框框单元 (Dlu) 中显示的网格间距的设置。  
  
 **宽度： Dlu**  
  
 Dlu 中设置的布局网格的宽度。 水平 DLU 是由 4 个划分对话框字体的平均宽度。  
  
 **高度： Dlu**  
  
 Dlu 中设置的布局网格的高度。 垂直 DLU 是平均除以八个对话框字体的高度。
