# <a name="_bstr_toperator-"></a>_bstr_t::operator !

**Microsoft 专用**

检查封装的 `BSTR` 是否为 NULL 字符串。

## <a name="syntax"></a>语法

```
bool operator!( ) const throw( );
```

## <a name="return-value"></a>返回值

**`true`** 如果是，则返回，否则返回 **`false`** 。

**结束 Microsoft 专用**
