# <a name="nocref"></a>.NOCREF
禁止显示符号表和浏览器文件中的符号的列表。  
  
## <a name="syntax"></a>语法  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## <a name="remarks"></a>备注  
 如果已指定了名称，则禁止仅给定名称。 与相同[。XCREF](../../assembler/masm/dot-xcref.md)。
