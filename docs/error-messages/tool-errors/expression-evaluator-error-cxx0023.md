# <a name="expression-evaluator-error-cxx0023"></a>表达式计算器错误 CXX0023

错误基数

C 表达式计算器无法识别指定的基数。 只有 decimal、十六进制和八进制 radixes 有效。

此错误与 CAN0023 相同。
