# <a name="const"></a>.CONST
如果用于[。模型](../../assembler/masm/dot-model.md)，启动常量数据段 （与段名称 CONST）。  
  
## <a name="syntax"></a>语法  
  
```  
  
.CONST  
  
```  
  
## <a name="remarks"></a>备注  
 此段具有只读属性。
