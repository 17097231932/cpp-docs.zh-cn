# <a name="expression-evaluator-error-cxx0026"></a>表达式计算器错误 CXX0026

错误的格式字符串

指定的格式字符串不正确。 检查表达式的语法。

此错误与 CAN0026 相同。

有关详细信息，请参阅[ C++中的格式说明符](/visualstudio/debugger/format-specifiers-in-cpp)。
