# <a name="bstrtoperator-"></a>_bstr_t::operator !
**Microsoft 专用**  
  
 检查是否封装`BSTR`是**NULL**字符串。  
  
## <a name="syntax"></a>语法  
  
```  
  
bool operator!( ) const throw( );  
  
```  
  
## <a name="return-value"></a>返回值  
 它将返回**true**如果是， **false**如果不是。
