# <a name="compiler-warning-level-1-c4502"></a>编译器警告（等级 1）C4502

"链接规范" 要求使用关键字 "extern"，并且必须在所有其他说明符之前

未指定关键字的情况下指定了链接 **`extern`** 。 链接与非 extern 类型无关。

编译器假定 **`extern`** 关键字。
