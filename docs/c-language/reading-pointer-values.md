# <a name="reading-pointer-values"></a>读取指针值

**ANSI 4.9.6.2** `fscanf` 函数中的 %p 转换的输入

指定 %p 格式字符时，`fscanf` 函数指针从十六进制 ASCII 值转换为正确地址。
