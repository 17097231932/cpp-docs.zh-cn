# <a name="compiler-error-c2499"></a>编译器错误 C2499
class: 类不能是其自己的基类  
  
 你尝试指定要定义用作基类的类。  
  
 下面的示例生成 C2499:  
  
```  
// C2499.cpp  
// compile with: /c  
class CMyClass : public CMyClass {};   // C2499  
class CMyClass{};   // OK  
```
