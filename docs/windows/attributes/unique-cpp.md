# <a name="unique-c"></a>unique (C++)

指定唯一的指针。

## <a name="syntax"></a>语法

```cpp
[unique]
```

## <a name="remarks"></a>备注

**Unique** c + + 特性具有与[unique](/windows/win32/Midl/unique) MIDL 特性相同的功能。
