# <a name="compiler-error-c2859"></a>编译器错误 C2859

filename 不是创建此预编译头时使用的类型文件，请重新创建预编译头。

必须一起创建项目数据库和预编译标头文件，以确保信息一致。 重新生成项目以重新创建预编译头。

有关预编译头的信息，请参阅 [/y (预编译标头) ](../../build/reference/y-precompiled-headers.md)。
