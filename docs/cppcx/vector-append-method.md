# Vector::Append 方法
在当前向量中的最后一项后插入指定项。  
  
## 语法  
  
```  
  
virtual void Append(  
   T item  
);  
```  
  
#### 参数  
 `index`  
 要插入到向量中的项。`item` 的类型由 *T* 类型名称定义。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
