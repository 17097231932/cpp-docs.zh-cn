# <a name="using-cdatetimectrl"></a>使用 CDateTimeCtrl

日期和时间选取器控件 ([CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)) 实现一种直观且可识别的方法来输入或选择特定日期。 控件的主接口在功能上类似于组合框。 但是，如果用户展开控件，则默认情况下将显示月日历控件 () ，允许用户指定特定日期。 选择日期后，月历控件将自动消失。

> [!NOTE]
> 若要 `CDateTimePicker` `CMonthCalCtrl` 在项目中同时使用和类，则必须包含 AFXDTCTL。H.，通常在项目的 STDAFX.H 中。H 文件。
