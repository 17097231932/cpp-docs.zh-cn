# <a name="compiler-error-c2781"></a>编译器错误 C2781

declaration： 需要至少 value1 参数的值 2 提供

具有变量参数列表的函数模板具有的参数太少。

下面的示例生成 C2781:

```
// C2781.cpp
template<typename T>
void f(T, T, ...){}

int main() {
   f(1);   // C2781

   // try the following line instead
   f(1,1);
}
```
