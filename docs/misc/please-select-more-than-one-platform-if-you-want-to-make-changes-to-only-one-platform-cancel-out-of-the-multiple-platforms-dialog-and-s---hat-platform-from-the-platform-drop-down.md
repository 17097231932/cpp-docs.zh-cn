# 请选择多个平台。 如果你仅要更改一个平台，请退出“多个平台”对话框，然后从平台下拉列表中选择该平台。
当在“多个平台”对话框中选择了单个平台时通常都会发生此错误。  
  
### 更正此错误  
  
1.  选择更多平台。  
  
     \- 或 \-  
  
2.  如果你想更新单个平台，请选择“取消”，然后从“平台”列表中选择正确的平台。
