# <a name="math-error-m6102"></a>数学错误 M6102
不正常  
  
 运算生成很小的浮点数，它是因无效而丢失的基数的倍数。 不正常通常被屏蔽浮点异常，从而导致它们被捕获并且运算。  
  
 程序终止，退出代码为 130。
