# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回定义架构和范围 *表达式的*单词。  这种低字节与 [.TYPE](../../assembler/masm/operator-dot-type.md)返回的字节相同。  高位字节包含附加信息。  
  
## 语法  
  
```  
  
OPATTR expression  
```
