# CA0062
未能找到规则集“*RuleSetName*”中引用的规则“*RuleId*”。  
  
 此错误表示未能找到指定的规则。  
  
 此错误通常因为已手动编辑规则集文件而发生。  若要避免此错误，请使用 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 规则集编辑器来指定规则。  有关详细信息，请参阅[创建自定义规则集](../Topic/Creating%20Custom%20Code%20Analysis%20Rule%20Sets.md)。  
  
 若要解决此问题，请确保对 ruleset 文件指定的所有的 CHECK ID 是有效的。如果使用非标准规则文件，请进行适当的规则，并提示ruleset 文件中指定的路径。
