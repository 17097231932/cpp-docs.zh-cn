# <a name="compiler-error-c2654"></a>编译器错误 C2654
identifier： 尝试在成员函数的外部访问成员  
  
 在声明中访问的成员。 可以仅在成员函数中访问成员数据。  
  
 在尝试初始化声明中的变量时，可以导致此错误。 为此目的使用构造函数。
