# CVTRES 警告 CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**未指定计算机类型；假定为**   
 ***类型***  
  
 CVTRES 未找到计算机规范。  假定为此给定计算机类型。  如果该默认值不正确，则使用 \/MACHINE 选项重新运行 CVTRES。
