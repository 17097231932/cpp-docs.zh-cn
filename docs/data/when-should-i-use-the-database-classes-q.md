# 何时使用数据库类？
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

很多应用程序要求在数据库内进行数据存储，而很多其他应用程序可以从使用数据库中获益。  数据库为您提供了灵活的数据储存库，该库在很多情况下可由多个用户和多个应用程序访问。  数据库可存储大量数据并为查询和更新提供快速数据访问。
