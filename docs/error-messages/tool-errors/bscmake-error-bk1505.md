# BSCMAKE 错误 BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法从文件“filename”中读取 \[: reason\]  
  
 BSCMAKE 无法从文件中读取。  
  
### 通过检查以下可能的原因进行修复  
  
1.  文件损坏。  
  
2.  创建 .sbr 文件时因编译器没有可用的磁盘空间或被中断导致文件截断。
