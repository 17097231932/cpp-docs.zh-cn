# Miscellaneous Files projects can contain &lt;number&gt; items.
“杂项文件”项目最多可以保存 256 项。  出现该错误的条件通常是：在“选项”对话框“环境”选项“文档”窗格上的“最后保存杂项文件项目 \<数字\> 项”框中输入了小于 1 或大于 256 的数字。  
  
### 更正此错误  
  
1.  在框中输入一个介于 1 到 256 之间的数字。
