# <a name="runtimeclassaddref-method"></a>RuntimeClass::AddRef 方法
递增当前 RuntimeClass 对象的引用计数。  
  
## <a name="syntax"></a>语法  
  
```  
STDMETHOD_(  
   ULONG,  
   AddRef  
)();  
```  
  
## <a name="return-value"></a>返回值  
 如果成功，则为 S_OK；否则为指示错误的 HRESULT。  
  
## <a name="requirements"></a>要求  
 **标头：** implements.h  
  
 **命名空间：** Microsoft::WRL
