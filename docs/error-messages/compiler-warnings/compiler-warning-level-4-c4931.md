# 编译器警告（等级 4）C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

我们假定类型库是为 number 位指针生成的  
  
 未与 [\#import](../../preprocessor/hash-import-directive-cpp.md) 指令的 **ptrsize** 特性一同提供显式信息，编译器推断类型库的指针大小为 ***number***。  
  
 默认情况下关闭此警告。  有关更多信息，请参见[默认情况下处于关闭状态的编译器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md)。
