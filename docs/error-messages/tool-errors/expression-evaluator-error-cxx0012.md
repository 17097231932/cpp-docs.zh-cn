# <a name="expression-evaluator-error-cxx0012"></a>表达式计算器错误 CXX0012
缺少左大括号  
  
 该表达式包含的不平衡的大括号。 重新键入用平衡的大括号的表达式。  
  
 此错误是与 CAN0012 相同。
