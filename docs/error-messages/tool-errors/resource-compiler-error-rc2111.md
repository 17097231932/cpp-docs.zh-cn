# 资源编译器错误 RC2111
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的控件类型  
  
 **DIALOG** 语句中的每一个 CONTROL 语句必须为下列之一：3STATE、AUTO3、AUTOCHECK、AUTORADIO、BEDIT、CHECKBOX、COMBOBOX、CONTROL、CTEXT、DEFPUSHBUTTON、EDITTEXT、GROUPBOX、HEDIT、ICON、IEDIT、LISTBOX、LTEXT、PUSHBOX、PUSHBUTTON、RADIOBUTTON、RTEXT、SCROLLBAR、USERBUTTON。  
  
 确保这些 CONTROL 语句拼写正确。
