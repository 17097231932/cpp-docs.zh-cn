# 命令选项，Visual Studio 包向导
此步骤能可定义显示在 Visual Studio“工具”菜单中的命令的名称和命令 ID。  
  
 **命令名：**  
 在此框中的文本将显示：  
  
-   命令的名称。  
  
-   工具栏上的命令按钮的工具提示。  
  
 **命令 ID**  
 -   此文本框中的文本定义该命令的命令 ID。 此 ID 是生成代码中使用的常量，用于指代此菜单项。  
  
 **取消**  
 -   取消并停止该向导。  
  
 **上一个**  
 -   移动到该向导的上一页。  
  
 **完成**  
 -   创建 VSPackage。  
  
 **下一步**  
 -   移动到向导的下一页。
