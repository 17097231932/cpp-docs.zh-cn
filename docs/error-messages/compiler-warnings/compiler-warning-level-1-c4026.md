# 编译器警告（等级 1）C4026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用形参表声明的函数  
  
 函数声明具有形参，而函数定义没有。 对此函数的后续调用将假定该函数不具有参数。
