# <a name="compiler-error-c3246"></a>编译器错误 C3246
“class”: 无法从“type”继承，因为它已声明为“sealed”  
  
标记为 [sealed](../../windows/sealed-cpp-component-extensions.md) 的类不能为任何其他类的基类。  
  
以下示例生成 C3246：  
  
```  
// C3246_2.cpp  
// compile with: /clr /LD  
ref class X sealed {};  
  
ref class Y : public X {}; // C3246  
```
