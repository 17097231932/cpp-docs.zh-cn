# <a name="new-custom-resource-dialog-box"></a>“新建自定义资源”对话框
使用此选项可创建新的自定义资源。  
  
 **资源类型**  
 提供一个文本框，以输入自定义资源类型的名称。 当你退出时，出现 visual c + + 会自动大写名称**新自定义资源**对话框。  
  
## <a name="remarks"></a>备注  
 未指定资源名称或 ID 保留[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。 保留的名称 DESIGNINFO，HWB，并且 TEXTINCLUDE 和保留的 ID 为 255。
