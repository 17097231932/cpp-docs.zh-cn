# 编译器警告（等级 1）C4631
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MSXML 或 XPath 不可用，将不会处理 XML 文档注释。 原因  
  
 你的公共语言运行时安装没有支持处理文档注释的必要文件。 重新安装公共语言运行时。
