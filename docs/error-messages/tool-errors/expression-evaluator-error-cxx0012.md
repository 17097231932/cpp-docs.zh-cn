# 表达式计算器错误 CXX0012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

缺少左大括号  
  
 表达式包含不对称的大括号。  使用对称的大括号重新输入表达式。  
  
 该错误与 CAN0012 相同。
