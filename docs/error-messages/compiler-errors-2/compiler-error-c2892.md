# 编译器错误 C2892
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

局部类不应有成员模板  
  
 在函数中定义的类中，模板成员函数无效。  
  
 下面的示例生成 C2892：  
  
```  
// C2892.cpp  
int main() {  
   struct local {  
      template<class T>   // C2892  
      void f() {}  
   };  
}  
```
