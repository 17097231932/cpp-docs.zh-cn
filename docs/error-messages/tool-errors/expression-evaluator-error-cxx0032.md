# 表达式计算器错误 CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

被 0 除  
  
 表达式包含 0 除数，这是非法的。  该除数可能是数字 0，也可能是计算结果为 0 的表达式。  
  
 该错误与 CAN0032 相同。
