# <a name="character-sets"></a>字符集

**ANSI 3.1.3.4** 源字符集的映射成员

源字符集和执行字符集包含下表中所列的 ASCII 字符。 该表中也显示了转义序列。

## <a name="escape-sequences"></a>转义序列

|转义序列|字符|ASCII 值|
|---------------------|---------------|-----------------|
|&#92;a|提醒/响铃|7|
|&#92;b|Backspace|8|
|&#92;f|换页|12|
|&#92;n|换行符|10|
|&#92;r|回车|13|
|&#92;t|水平制表符|9|
|&#92;v|垂直制表符|11|
|&#92;"|双引号|34|
|&#92;'|单引号|39|
|&#92;&#92;|反斜杠|92|
