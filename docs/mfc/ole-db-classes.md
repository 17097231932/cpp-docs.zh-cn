# OLE DB 类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 MFC 的 OLE DB 当前支持包括 [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md) 类。  通过视图窗体直接连接到 [CRowset](../data/oledb/crowset-class.md) 对象，`COleDBRecordView` 显示控件中数据库记录。  有关 OLE DB 使用者模板的详细信息，请参阅 [OLE DB 使用者模板的列表](../data/oledb/ole-db-consumer-templates-reference.md)。
