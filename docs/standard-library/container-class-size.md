# <a name="container-classsize"></a>Container Class::size

> [!NOTE]
> 该主题位于 Visual C++ 文档内，作为在 C++ 标准库内使用的容器的非功能性示例。 有关详细信息，请参阅 [C++ 标准库容器](../standard-library/stl-containers.md)。

返回受控序列的长度，在常量时间内不考虑受控序列的长度。

## <a name="syntax"></a>语法

```

size_type size() const;
```
