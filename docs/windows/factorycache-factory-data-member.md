# <a name="factorycachefactory-data-member"></a>FactoryCache::factory 数据成员
支持 Windows 运行时 c + + 模板库基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
IUnknown* factory;  
```  
  
## <a name="remarks"></a>备注  
 指向 Windows 运行时或 COM 类工厂。  
  
## <a name="requirements"></a>要求  
 **标头：** module.h  
  
 **Namespace:** Microsoft::WRL::Details
