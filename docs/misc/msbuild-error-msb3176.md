# MSBuild 错误 MSB3176
**MSB3176：指定最低要求的版本高于当前的发布版本。  请指定一个低于或等于当前发布版本的版本。**  
  
 当要求的最低应用程序版本大于部署清单的当前版本时，将产生此错误。  
  
### 更正此错误  
  
-   指定一个低于或等于当前发布版本的最低要求版本。
