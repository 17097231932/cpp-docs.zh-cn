# <a name="compiler-error-c3800"></a>编译器错误 C3800
declaration︰ 不能混合属性和事件  
  
 不能声明为一个属性和事件的构造。  
  
 C3800 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
