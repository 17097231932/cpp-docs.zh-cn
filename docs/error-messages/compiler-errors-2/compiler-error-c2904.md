# <a name="compiler-error-c2904"></a>编译器错误 C2904
“identifier”：名称已经用于当前范围内的模板  
  
 检查代码中是否有重复的名称。  
  
 以下示例生成 C2904：  
  
```  
// C2904.cpp  
// compile with: /c  
void X();  // X is declared as a function  
template<class T> class X{};  // C2904  
```
