# 记录集：声明表的类 (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

本主题适用于 MFC ODBC 类。  
  
 最常用的记录集类打开单个表。  若要声明单个表的记录集类，请使用“添加类”的 [MFC ODBC 使用者向导](../../mfc/reference/adding-an-mfc-odbc-consumer.md)并通过命名相应的记录集字段数据成员来选择所需的每一列。  
  
 记录集的其他用途包括：  
  
-   联接两个或更多个表。  
  
-   包含预定义查询的结果。
