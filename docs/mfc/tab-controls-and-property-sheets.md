# <a name="tab-controls-and-property-sheets"></a>选项卡控件和属性表

属性表为多页对话框或"选项卡对话框"可显示最多 24 的对话框模板资源与用户。 属性表的示例，请参阅 Windows 显示属性对话框中或以下的 MFC 示例应用程序：

[CMNCTRL1： 演示常见的控件类，第 1 部分](../visual-cpp-samples.md)

[CMNCTRL2： 演示常见的控件类，第 2 部分](../visual-cpp-samples.md)

属性表可以实现起来非常容易使用 MFC 类[CPropertySheet](../mfc/reference/cpropertysheet-class.md)。
