# <a name="compiler-error-c3159"></a>编译器错误 C3159

"指针"：不能声明指向值类型的指针数组

不能声明指向值类型的指针的数组。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3159。
