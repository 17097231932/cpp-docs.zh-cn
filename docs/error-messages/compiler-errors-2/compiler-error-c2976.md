# <a name="compiler-error-c2976"></a>编译器错误 C2976
identifier： 类型参数太少  
  
 泛型或模板缺少一个或多个自变量。 检查泛型或模板声明，以查找正确的参数数目。  
  
 可以通过 c + + 标准库组件中缺少模板参数导致此错误。  
  
 下面的示例生成 C2976:  
  
```  
// C2976.cpp  
template <class T>   
struct TC {  
   T t;  
};  
int main() {  
   TC<>* t;   // C2976  
   TC<int>* t2;   // OK  
}  
```  
  
 使用泛型时，也可能发生 C2976:  
  
```  
// C2976b.cpp  
// compile with: /clr  
generic <class T>  
ref struct GC {  
   T t;  
};  
  
int main() {  
   GC<>^ g;   // C2976  
   GC<int>^ g2;   // OK  
}  
```
