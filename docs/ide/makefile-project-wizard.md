# 生成文件项目向导
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用该向导创建供生成引擎执行的命令行。  创建了项目后，可以在项目的[属性页](../ide/working-with-project-properties.md)中查看和编辑项目的属性。  
  
## 概述  
 该向导页描述正在创建的项目的当前项目设置。  默认情况下，项目被设置成生成 *ProjName*.exe 的调试和发布配置。  
  
 若要更改这些默认设置，请单击向导左列中的 [“应用程序设置”](../ide/application-settings-makefile-project-wizard.md) 选项卡并进行所需的更改。
