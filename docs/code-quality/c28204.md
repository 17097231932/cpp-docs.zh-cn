# <a name="c28204"></a>C28204

> 警告 C28204： \<function> ：只有此重载中的一个重载，而 \<filename> () 中的一个 \<line> 是批注的 \<paramname> ：这两种重载都必须批注。

当批注中存在错误时将报告此警告。
