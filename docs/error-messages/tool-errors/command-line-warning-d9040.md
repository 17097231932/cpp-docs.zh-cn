# 命令行警告 D9040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

忽略“\/analyze”选项；“代码分析”警告在此版本的编译器中不可用  
  
 该**\/analyze**的命令行选项不可用在Visual Studio的所有版本。  若要更正此警告，可改用受支持版本的 Visual Studio Team System，或者移除该命令行选项。
