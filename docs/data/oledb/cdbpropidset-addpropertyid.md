# CDBPropIDSet::AddPropertyID
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

将属性添ID加到将属性的 ID。  
  
## 语法  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### 参数  
 `propid`  
 \[in\] 要添加的属性 ID来设置属性的 ID。  
  
## 要求  
 **标头:** atldbcli.h
