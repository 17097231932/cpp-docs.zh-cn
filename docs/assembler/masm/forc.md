# FORC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

标记为 `string`的每个字符一次将重复的块，在当前字符替换每重复 `parameter` 。  
  
## 语法  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## 备注  
 和 [IRPC](../../assembler/masm/irpc.md)相同。
