# <a name="compiler-error-c2913"></a>编译器错误 C2913

显式专用化;declaration 不是类模板的专用化

无法特化非模板类。

下面的示例生成 C2913:

```
// C2913.cpp
// compile with: /c
class X{};
template <class T> class Y{};

template<> class X<int> {};   // C2913
template<> class Y<int> {};
```
