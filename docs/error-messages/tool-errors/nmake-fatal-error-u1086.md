# <a name="nmake-fatal-error-u1086"></a>NMAKE 错误 U1086

**推理规则不能有依赖项**

推理规则中的冒号 (**：**) 必须后跟以下其中之一：

- 换行符

- 分号 (**;**) ，后面可以跟一个命令

- 数字符号 (**#**) ，后面可以跟注释
