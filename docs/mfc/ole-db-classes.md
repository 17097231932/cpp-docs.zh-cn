# <a name="ole-db-classes"></a>OLE DB 类

MFC 中的 OLE DB 支持当前包括类[COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)。 `COleDBRecordView` 在控件中，通过直接连接到的窗体视图显示数据库记录[CRowset](../data/oledb/crowset-class.md)对象。 有关 OLE DB 使用者模板的详细信息，请参阅[列表中的 OLE DB 使用者模板](../data/oledb/ole-db-consumer-templates-reference.md)。
