# <a name="nmake-fatal-error-u1087"></a>NMAKE 错误 U1087

不能有： 和:: 从属项相同的目标

目标不能指定这两个单冒号 (**:**) 和双冒号 (`::`) 依赖项。

若要在多个描述块中指定一个目标，使用`::`中每个依赖行。
