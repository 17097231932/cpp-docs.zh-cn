# Editing Properties for an ActiveX Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

独立供应商提供的 ActiveX 控件可能配备了自己的属性和特性。  ActiveX 控件的属性显示在“属性”窗口中。  另外，ActiveX 控件编写者所创建的任何属性页都显示在“属性页”对话框中（若要查看特定 ActiveX 控件的属性页，请单击[“属性”窗口](../Topic/Properties%20Window.md)中的“属性页”按钮）。  
  
 ActiveX 控件的属性页中显示了各种选项卡，具体取决于作为 ActiveX 控件的一部分提供的属性表。  
  
> [!NOTE]
>  以下过程适用于使用属性页编辑 ActiveX 控件。  还可以在新的“属性”窗口中浏览和编辑 ActiveX 属性。  
  
### 编辑 ActiveX 控件的属性  
  
1.  选择 **ActiveX** 控件。  
  
2.  在**“视图”**菜单上，单击**“属性页”**查看属性。  
  
3.  在属性页中进行所需的更改。  
  
 有关将资源添加到托管项目的信息，请参见“.NET Framework 开发员指南”中的[应用程序中的资源](../Topic/Resources%20in%20Desktop%20Apps.md)。有关手动将资源文件添加到托管项目、访问资源、显示静态资源和将资源字符串分配给属性的信息，请参见[演练：本地化 Windows 窗体](http://msdn.microsoft.com/zh-cn/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。  
  
 要求  
  
 Win32
