# <a name="compiler-error-c2213"></a>编译器错误 C2213

"修饰符"：非法的参数 __based

修改参数 **`__based`** 无效。

下面的示例生成 C2213：

```cpp
// C2213.cpp
// compile with: /c
int i;
int *j;
char __based(i) *p;   // C2213
char __based(j) *p2;   // OK
```
