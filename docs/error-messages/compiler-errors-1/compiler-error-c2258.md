# <a name="compiler-error-c2258"></a>编译器错误 C2258

非法的纯语法，必须为“= 0”

用不正确的语法声明一个纯虚函数。

下面的示例生成 C2258：

```
// C2258.cpp
// compile with: /c
class A {
public:
   void virtual func1() = 1; // C2258
   void virtual func2() = 0;   // OK
};
```
