# MSBuild 错误 MSB3186
**MSB3186：无法从任务输入参数推导出生成的清单的程序集标识。**  
  
 当生成过程无法推断应用程序清单或部署清单的程序集名称时，就会发生此错误。  没有显式给出程序集名称；基础清单中没有标识，也未指定入口点标识。
