# 添加 ATL COM+ 1.0 组件
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

此向导向支持 COM\+ 1.0 服务（包括事务）的项目中添加对象。  
  
### 向项目中添加 ATL COM\+ 1.0 组件  
  
1.  在**“解决方案资源管理器”**或[类视图](http://msdn.microsoft.com/zh-cn/8d7430a9-3e33-454c-a9e1-a85e3d2db925)中，右击要向其添加 ATL COM\+ 1.0 组件的项目的名称。  
  
2.  在快捷菜单上，单击“添加”，然后单击“添加类”。  
  
3.  在[添加类](../../ide/add-class-dialog-box.md)对话框中的“模板”窗格中，单击“ATL COM\+ 1.0 组件”，然后单击“打开”显示 [ATL COM\+ 1.0 组件向导](../../atl/reference/atl-com-plus-1-0-component-wizard.md)。
