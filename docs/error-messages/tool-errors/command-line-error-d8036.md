# <a name="command-line-error-d8036"></a>命令行错误 D8036

/ 选项不允许有多个源文件

这些编译器选项不能用于多个源文件：

- 名称的程序集的文件列表 (/ Fa)

- 重命名对象文件 (/ Fo)

- 创建无局部变量的源浏览器信息 (/ Fr)

- 创建源浏览器信息，包括本地变量 (/ FR)
