# <a name="textequ"></a>TEXTEQU

将 *textitem* 分配给 *name*。 *Textitem* 可以是文本字符串、前面有的常量 `%` 或宏函数返回的字符串。

## <a name="syntax"></a>语法

> *name* **TEXTEQU** ⟦*textitem*⟧
