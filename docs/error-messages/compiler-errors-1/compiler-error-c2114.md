# 编译器错误 C2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“operator”：左侧指针；需要右侧的整数值  
  
 如果 `operator` 的左操作数为一个指针，则右操作数不能为整数值。
