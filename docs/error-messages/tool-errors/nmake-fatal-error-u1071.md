# <a name="nmake-fatal-error-u1071"></a>NMAKE 错误 U1071

目标 "targetname" 的依赖关系树中的循环

给定目标的依赖关系树中存在循环依赖项。 给定目标是给定目标的某个依赖项的依赖项。 循环依赖关系无效。
