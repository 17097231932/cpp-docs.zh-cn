# <a name="using-ctreectrl"></a>使用 CTreeCtrl

由类[CTreeCtrl](../mfc/reference/ctreectrl-class.md)表示的树控件是一个窗口，其中显示项的层次结构列表，例如文档中的标题、索引中的条目或磁盘上的文件和目录。 每个项包含一个标签和一个可选位图图像，且每个项可以有一个与之关联的子项列表。 通过单击项，用户可以展开和折叠关联的子项列表。 资源管理器的左侧窗格中的目录树是树控件的一个示例。
