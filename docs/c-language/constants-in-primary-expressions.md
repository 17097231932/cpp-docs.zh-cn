# <a name="constants-in-primary-expressions"></a>主要表达式中的常量

常量操作数具有它表示的常量的值和类型。 字符常数的类型为 `int`。 整型常数的类型为 `int`、`long`、`unsigned int` 或 `unsigned long`，具体视整型常数大小和指定值的方式而定。 有关详细信息，请参阅[常量](../c-language/c-constants.md)。
