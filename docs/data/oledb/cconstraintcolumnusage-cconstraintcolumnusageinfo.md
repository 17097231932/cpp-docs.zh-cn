# <a name="cconstraintcolumnusage-cconstraintcolumnusageinfo"></a>CConstraintColumnUsage，CConstraintColumnUsageInfo
调用 typedef 类**CConstraintColumnUsage**来实现其参数类**CConstraintColumnUsageInfo**。  
  
## <a name="remarks"></a>备注  
 请参阅[架构行集类和 Typedef 类](../../data/oledb/schema-rowset-classes-and-typedef-classes.md)有关使用 typedef 类的详细信息。  
  
 此类标识列使用的引用约束、 唯一约束、 check 约束和断言、 目录中定义和给定用户所拥有的。  
  
 下表列出了类数据成员及其对应的 OLE DB 列。 请参阅[CONSTRAINT_COLUMN_USAGE 行集](https://msdn.microsoft.com/en-us/library/ms724522.aspx)中*OLE DB 程序员参考*有关的架构和列的详细信息。  
  
|数据成员|OLE DB 列|  
|------------------|--------------------|  
|m_szTableCatalog|TABLE_CATALOG|  
|m_szTableSchema|TABLE_SCHEMA|  
|m_szTableName|TABLE_NAME|  
|m_szColumnName|COLUMN_NAME|  
|m_guidColumn|COLUMN_GUID|  
|m_nColumnPropID|COLUMN_PROPID|  
|m_szConstraintCatalog|CONSTRAINT_CATALOG|  
|m_szConstraintSchema|CONSTRAINT_SCHEMA|  
|m_szConstraintName|CONSTRAINT_NAME|  
  
## <a name="requirements"></a>要求  
 **标头：** atldbsch.h
