# <a name="command-line-warning-d9024"></a>命令行警告 D9024
无法识别的源文件类型 filename 中，对象假定文件  
  
 无法识别指定的文件的扩展名。 该文件被视为一个对象文件，并且传递给链接器。  
  
 识别的以下扩展：  
  
-   .c （C 源文件）  
  
-   .cxx （c + + 源文件）  
  
-   .cpp （c + + 源文件）  
  
-   .obj （对象文件）  
  
-   .lib （库文件）  
  
-   .def （模块定义文件）  
  
-   .exp （链接器导出链接 /LIB 所创建的文件）
