# C 运行时库参考
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft 运行库为编程提供 Microsoft Windows 操作系统中的例程。  这些例程可自动处理许多 C 和 C\+\+ 语言没有提供的常见编程任务。  
  
 示例程序中单个的参考主题包括大多数例程的库中。  
  
## 本节内容  
 [C运行时间库](../c-runtime-library/crt-library-features.md)  
 讨论由 C 运行库的 .lib 文件。  
  
 [按类别分的运行时例程](../c-runtime-library/run-time-routines-by-category.md)  
 按类别提供指向运行库。  
  
 [全局变量和标准类型](../c-runtime-library/global-variables-and-standard-types.md)  
 提供指向运行库和标准提供类型的全局变量。  
  
 [全局常量](../c-runtime-library/global-constants.md)  
 提供指向运行库定义的全局常数。  
  
 [按字母顺序的函数参考](../c-runtime-library/reference/crt-alphabetical-function-reference.md)  
 提供一目录入口点任何 C 运行库函数的按字母顺序排列的列表。  
  
 [一般文本映射](../c-runtime-library/generic-text-mappings.md)  
 提供指向在 Tchar.h 中定义的一般文本映射。  
  
 [语言和国家\/地区字符串](../c-runtime-library/locale-names-languages-and-country-region-strings.md)  
 介绍如何使用 `setlocale` 功能设置语言和国家\/地区字符串。
