# <a name="default-atl-project-configurations"></a>默认 ATL 项目配置
默认情况下，ATL 项目向导创建两个项目配置：  
  
|配置|字符集|ATL 的使用|  
|-------------------|-------------------|----------------|  
|发布|MBCS|DLL|  
|调试|MBCS|DLL|  
  
 **字符集**， **ATL 的使用**和所有，可更改**项目设置**下的对话框**常规**选项卡。此外可以添加您自己使用 Configuration Manager 的配置。 有关详细信息，请参阅[生成配置](/visualstudio/ide/understanding-build-configurations)。
