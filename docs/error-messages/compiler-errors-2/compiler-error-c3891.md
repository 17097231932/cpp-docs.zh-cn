# <a name="compiler-error-c3891"></a>编译器错误 C3891
var: literal 数据成员不能用作左值  
  
 A[文本](../../windows/literal-cpp-component-extensions.md)变量为常量，并在声明中初始化之后，不能更改其值。  
  
 下面的示例生成 C3891:  
  
```  
// C3891.cpp  
// compile with: /clr  
ref struct Y1 {  
   literal int staticConst = 9;  
};  
  
int main() {  
   Y1::staticConst = 0;   // C3891  
}  
```
