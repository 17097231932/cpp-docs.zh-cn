# 错误 C1309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

C2.DLL 和 PGODB\<version\>.DLL 的版本不匹配。  
  
 正用于生成和使用[“按配置优化”](../../build/reference/profile-guided-optimizations.md)的工具集包含不匹配的组件。  如果无法手动解决此错误，请重新安装 Visual C\+\+。
