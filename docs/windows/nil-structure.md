# Nil 结构
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```  
struct Nil;  
```  
  
## 备注  
 用于指示未指定，选项模板参数。  
  
 零是空结构。  
  
## 继承层次结构  
 `Nil`

 **头文件：**implements.h
