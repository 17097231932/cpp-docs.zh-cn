# MSBuild 错误 MSB2011
**元素 \<{0}\> 无效。**  
  
 指定的元素无效。  
  
### 更正此错误  
  
-   检查指定元素的拼写。  
  
-   检查项目文件是否已被修改或损坏。  如果项目已被修改或损坏，请用创建项目所用的 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 版本将项目打开，保存项目，然后尝试再次转换项目。
