# Class Factories Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类实现或支持选件类工厂:  
  
-   [CComClassFactory](../atl/reference/ccomclassfactory-class.md) 为对象创建提供默认选件类工厂。  
  
-   [CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) 控件对象创建通过许可证。  
  
-   [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) 在多个线程池的单元允许对象创建的。  
  
-   [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) 创建一个对象。  
  
-   [CComCoClass](../atl/reference/ccomcoclass-class.md) 定义对象的选件类工厂。
