# <a name="atl-registry-component-registrar"></a>ATL 注册表组件（注册器）
ATL 注册器提供对系统注册表通过自定义接口的优化的访问。 注册机构是自由线程，并且允许静态链接的代码的 c + + 客户端。  
  
> [!NOTE]
>  ATL 注册机构的源代码可以位于 atlmfc\include\atliface.h。
