# <a name="adding-an-atl-ole-db-provider"></a>添加的 ATL OLE DB 提供程序
使用此向导以向项目中添加的 ATL OLE DB 提供程序。 ATL OLE DB 提供程序由数据源、 会话、 命令和行集类组成。 必须为 ATL COM 应用程序创建项目。  
  
### <a name="to-add-an-atl-ole-db-provider-to-your-project"></a>若要向项目中添加的 ATL OLE DB 提供程序  
  
1.  在类视图中，右键单击该项目。 在快捷菜单上，单击**添加**，然后单击**添加类**。  
  
2.  在 Visual c + + 文件夹中，双击**ATL OLE DB 访问接口**图标或选择它，然后单击**打开**。  
  
     ATL OLE DB 提供程序向导将打开。  
  
3.  定义设置中所述[ATL OLE DB 提供程序向导](../../atl/reference/atl-ole-db-provider-wizard.md)。  
  
4.  单击**完成**关闭向导，将在你项目中插入新创建的 OLE DB 提供程序代码。
