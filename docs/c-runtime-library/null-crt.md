# NULL (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**NULL** 是带许多指针运算和函数的空指针值。  它等效于零 。  **NULL** 下头文件中定义：CRTDBG.H LOCALE.H、STDDEF.H、STDIO.H、STDLIB.H、STRING.H、TCHAR.H、TIME.H 和 WCHAR.H。
