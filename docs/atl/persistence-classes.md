# Persistence Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类实现对象持久性:  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) 允许客户端加载和保存对象的属性设置为属性包。  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) 允许客户端加载和保存对象的持久性数据。流。  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) 允许客户端加载和保存对象的持久性数据存储到。
