# operator PTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第一个运算符强制对 *表达式* 视为具有指定的 `type`。  第二个运算符指定指向 `type`。  
  
## 语法  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
