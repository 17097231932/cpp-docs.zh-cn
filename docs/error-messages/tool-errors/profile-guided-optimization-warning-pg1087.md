# <a name="profile-guided-optimization-warning-pg1087"></a>按配置优化警告 PG1087
PGC 文件 file 不是有效的。  
  
 对应的.pgc 文件已损坏，可能是因为异常结束分析会话。  重新运行分析的方案并正常终止。
