# <a name="windowsfoundation-namespace"></a>Windows::Foundation 命名空间

启用基本 Windows 运行时功能，例如对象和工厂创建。

## <a name="syntax"></a>语法

```cpp
namespace Windows::Foundation;
```

|[GetActivationFactory 函数](getactivationfactory-function.md)|检索指定模板参数的类型的激活工厂。|
