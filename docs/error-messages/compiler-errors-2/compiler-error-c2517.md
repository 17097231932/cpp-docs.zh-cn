# <a name="compiler-error-c2517"></a>编译器错误 C2517

"identifier"： "：：" 的右边未定义

范围解析运算符（`::`）右侧的标识符必须是左侧的类、结构或联合的已定义成员。 如果没有命名类、结构或联合，则必须在全局范围内声明权限。
