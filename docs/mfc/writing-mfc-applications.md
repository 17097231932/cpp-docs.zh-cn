# <a name="writing-mfc-applications"></a>编写 MFC 应用程序

本文介绍了开发应用程序的初始步骤。 首先，必须确定要编写的应用程序类型。 [应用程序设计选择](../mfc/application-design-choices.md)中讨论了几个选项。 你的应用程序将是：

- 在 Internet 或 intranet 上运行

- 在客户端或服务器上运行

- 在浏览器中或作为独立的应用程序运行

- 使用 COM 或活动技术

- 使用 WinInet 或异步名字对象下载数据

你的决定将确定哪些类适用于你的应用程序。 你的答案还有助于确定在运行应用程序向导开始构造应用程序时所做的选择。

完成有关 Internet 应用程序的初始设计决策后，可以使用应用程序向导开始使用。 使用应用程序向导创建主干应用程序并修改代码，如以下文章中所述：

- 对于 ActiveX 控件，请参阅 [Internet 上的 Activex 控件](../mfc/activex-controls-on-the-internet.md)。

以下文章还提供了帮助你开始编程任务的说明：

- [应用程序设计选择](../mfc/application-design-choices.md)

- [Internet 上的异步名字对象](../mfc/asynchronous-monikers-on-the-internet.md)

- [WinInet 基础知识](../mfc/wininet-basics.md)
