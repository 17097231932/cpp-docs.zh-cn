# <a name="expression-evaluator-error-cxx0066"></a>表达式计算器错误 CXX0066

静态成员不存在

找不到或未定义类的静态成员。 此错误可能是由声明但未定义的静态类成员导致的，或者仅在不包含调试信息的模块中定义或引用。

此错误与 CAN0066 相同。
