# <a name="compiler-error-c3180"></a>编译器错误 C3180
type name： 名称超过了限制个字符的元数据限制  
  
 编译器截断元数据中的托管类型的名称。 截断将使该类型与不可用`#using`指令 （或在另一种语言的等效项）。  
  
 类型名称限制包括任何命名空间限定。
