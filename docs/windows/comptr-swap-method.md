# <a name="comptrswap-method"></a>ComPtr::Swap 方法
交换当前管理的接口**ComPtr**管理指定的接口与**ComPtr**。  
  
## <a name="syntax"></a>语法  
  
```  
void Swap(  
   _Inout_ ComPtr&& r  
);  
  
void Swap(  
   _Inout_ ComPtr& r  
);  
```  
  
#### <a name="parameters"></a>参数  
 *r*  
 一个**ComPtr**。  
  
## <a name="requirements"></a>要求  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
