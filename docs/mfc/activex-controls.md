# <a name="activex-controls"></a>ActiveX 控件

在 Visual C++ 中，您可以使用 MFC 或 ATL 创建 ActiveX 控件。

>[!IMPORTANT]
> ActiveX 是一种不能用于新开发的旧技术。 可以使用 HTML5 和 JavaScript、新式浏览器扩展或 WebAssembly 模块等新式技术更简单、更安全地执行 ActiveX 控件的许多功能。 有关详细信息，请参阅[过去的中断，第2部分：对 ActiveX、VBScript、onpropertychange 的再见。](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) 和[本机消息传递](/microsoft-edge/extensions/guides/native-messaging)和[Microsoft Edge 扩展](/microsoft-edge/extensions)和[WebAssembly](https://webassembly.org/)。
