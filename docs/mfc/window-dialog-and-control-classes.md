# <a name="window-dialog-and-control-classes"></a>窗口、对话框和控件类

类`CWnd`及其派生的类封装和`HWND`，Windows 窗口的句柄。 `CWnd` 可用于单独或作为基派生新类。 由类库提供的派生的类表示各种类型的 windows。

[CWnd](../mfc/reference/cwnd-class.md)<br/>
所有窗口的基类。 可以使用派生自的类之一`CWnd`或直接从其派生您自己的类。
