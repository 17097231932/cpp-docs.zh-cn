# <a name="2-directives"></a>2.指令

基于指令 **#pragma** C 和 c + + 标准中定义的指令。  OpenMP C 和 c + + API 支持的编译器将包括激活并允许所有 OpenMP 编译器指令的解释的命令行选项。
