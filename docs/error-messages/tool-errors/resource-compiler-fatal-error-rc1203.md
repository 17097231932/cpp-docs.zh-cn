# 资源编译器错误 RC1203
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指定的十六进制默认语言 ID 无效。  
  
 “指定默认语言”\(\/l\) 选项后跟一个无效的十六进制语言 ID。  请参见*《运行库参考》*中[“语言和国家\/地区字符串”](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)内关于有效语言 ID 列表的内容。
