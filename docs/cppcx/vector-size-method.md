# Vector::Size 方法
返回当前向量对象中的元素数目。  
  
## 语法  
  
```  
  
virtual property unsigned int Size;  
```  
  
## 返回值  
 当前 Vector 中的元素数目。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
