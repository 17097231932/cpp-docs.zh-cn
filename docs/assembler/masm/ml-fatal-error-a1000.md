# <a name="ml-fatal-error-a1000"></a>ML 错误 A1000

**无法打开文件：文件名**

汇编程序无法打开源、包含或输出文件。

可能是以下原因之一：

- 文件不存在。

- 文件正由另一个进程使用。

- 文件名无效。

- 已存在输出文件名为的只读文件。

- 当前驱动器已满。

- 当前目录为根，已满。

- 无法写入设备。

- 驱动器未就绪。
