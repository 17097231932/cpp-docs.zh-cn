# <a name="ml-nonfatal-error-a2039"></a>ML 非致命错误 A2039

**行太长**

源文件行超过了512个字符的限制。

如果多个物理线路连接 ( \ ) 的行继续符，则生成的逻辑行仍限制为512个字符。
