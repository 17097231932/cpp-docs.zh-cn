# <a name="nmake-warning-u4007"></a>NMAKE 警告 U4007

文件名 "filename" 太长;截断为8。3

给定文件的基名称包含的字符超过8个，或扩展名包含三个以上的字符。 NMAKE 将名称截断为八个字符的基，并使用三个字符的扩展。

如果文件系统支持长文件名，请将该名称用双引号引起来 (**"**) 。
