# <a name="catan-catanf-catanl"></a>catan、catanf、catanl

检索复数的分支切割超出间隔 [-1; + 1] 沿虚轴的反正切值。

## <a name="syntax"></a>语法

```C
_Dcomplex catan( _Dcomplex z );
_Fcomplex catanf( _Fcomplex z );
_Lcomplex catanl( _Lcomplex z );
```

```cpp
_Fcomplex catan( _Fcomplex z );  // C++ only
_Lcomplex catan( _Lcomplex z );  // C++ only
```

### <a name="parameters"></a>参数

*z*<br/>
表示角度的复数（以弧度为单位）。

## <a name="return-value"></a>返回值

反正切*z*，以弧度为单位。 结果为无限沿虚轴，并位于的间隔 [-π/2; + π/2] 沿实轴。

## <a name="remarks"></a>备注

因为C++允许重载，可以调用的重载**catan**采用并返回 **_Fcomplex**并 **_Lcomplex**的值。 在 C 程序中， **catan**始终采用并返回 **_Dcomplex**值。

## <a name="requirements"></a>要求

|例程所返回的值|C 标头|C++ 标头|
|-------------|--------------|------------------|
|**catan**， **catanf**， **catanl**|\<complex.h>|\<ccomplex>|
