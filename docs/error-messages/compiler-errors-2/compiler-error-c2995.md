# <a name="compiler-error-c2995"></a>编译器错误 C2995
“function”：函数模板已经定义  
  
 确保模板类的每个成员函数只有一个定义。  
  
 以下示例生成 C2995：  
  
```  
// C2995.cpp  
// compile with: /c  
template <class T>  
void Test(T x){}  
  
template <class T> void Test(T x){}   // C2995  
template <class T> void Test2(T x){}   // OK  
```
