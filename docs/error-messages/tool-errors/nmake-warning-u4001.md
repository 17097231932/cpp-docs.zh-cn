# <a name="nmake-warning-u4001"></a>NMAKE 警告 U4001

> 可以仅从命令行调用命令文件

命令文件，由调用 at 符号 (**\@**) 说明符，则不能包含另一个命令文件规范。 不允许此种嵌套。 指定已被忽略。
