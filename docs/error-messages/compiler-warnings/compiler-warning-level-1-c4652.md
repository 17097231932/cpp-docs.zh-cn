# <a name="compiler-warning-level-1-c4652"></a>编译器警告（等级 1）C4652
编译器选项 option 与预编译标头; 不一致当前的命令行选项将重写预编译标头中定义  
  
 给定的命令行选项不同于创建预编译标头 (.pch) 时提供。 使用当前的命令行中指定的选项。  
  
 可以通过重新生成预编译标头与给定的命令行选项避免此警告。
