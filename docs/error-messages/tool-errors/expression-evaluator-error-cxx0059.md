# 表达式计算器错误 CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左边的操作数是类而不是函数名  
  
 函数调用的左操作数是类名，未能将其解析为函数调用。  省略表达式中的成员函数名可导致该错误。  
  
 该错误与 CAN0059 相同。
