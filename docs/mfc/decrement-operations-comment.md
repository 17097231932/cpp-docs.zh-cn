# // Operations 注释
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC 类声明中 `// Operations` 节包含在对象可以调用使其执行操作或执行操作成员函数 \(请执行操作\)。  因为它们通常有副作用，这些函数通常非**const**。  它们可能是虚拟或根据其需要的类。  通常，这些成员称为公共。  
  
 在从 `CStdioFile`类的示例的列表，在 [批注的示例](../mfc/an-example-of-the-comments.md)列表，包含两个成员函数此注释下：`ReadString` 和 `WriteString`。  
  
 使用特性，可进一步操作部分。
