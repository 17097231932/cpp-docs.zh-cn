# 关于异常的疑难解答：System.ArrayTypeMismatchException
系统无法将元素转换成为数组声明的类型时，将引发 <xref:System.ArrayTypeMismatchException> 异常。  
  
## 相关提示  
 **确保该对象类型可转换为数组类型。**  
 两种类型之间必须能够转换。  
  
 有关详细信息，Visual Basic 用户可参阅 [数组](../Topic/Arrays%20in%20Visual%20Basic.md)。
