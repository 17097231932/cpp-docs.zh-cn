# <a name="nmake-fatal-error-u1076"></a>NMAKE 错误 U1076
名称太长  
  
 一个字符串超过一个以下限制：  
  
-   宏名称 1024年个字符。  
  
-   目标路径名的 256 个字符。  
  
-   命令的 2048年个字符。
