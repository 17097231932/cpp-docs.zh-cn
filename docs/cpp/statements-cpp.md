# 语句 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ 控制报表如何以何种顺序去操作对象的程序元素。  本节包括：  
  
-   [概述](../cpp/overview-of-cpp-statements.md)  
  
-   [标记语句](../cpp/labeled-statements.md)  
  
-   语句类别  
  
    -   [表达式语句](../cpp/expression-statement.md)。  这些报表估算其副作用的表达式或为其返回值。  
  
    -   [空语句](../cpp/null-statement.md)。  当报表中要求C\+\+语法时这些报表可以提供，而不需要采取实际操作措施。  
  
    -   [复合语句](../cpp/compound-statements-blocks.md)。  这些报表中的语句组被包含在大括号内 \({}\)。  无论任何一个语句都可以使用它们。  
  
    -   [选择语句](../cpp/selection-statements-cpp.md)。  这些语句执行一个测试；如果测试计算结果为 true \(非零\)他们将执行代码的一部分。  如果测试计算为 FALSE，它们可执行代码的其他部分。  
  
    -   [迭代语句](../cpp/iteration-statements-cpp.md)。  这些报表所提供的代码块将重复执行，直到与指定的终止条件相匹配。  
  
    -   [跳转语句](../cpp/jump-statements-cpp.md)。  这些语句立即将控件转移到函数的其他位置或从函数返回控件。  
  
    -   [声明语句](http://msdn.microsoft.com/zh-cn/14538558-356f-450e-9e1e-3cd62ba952b9)。  声明引入一个名称到程序。\([声明](../misc/declarations.md) 提供有关声明的详细信息。\)  
  
 有关异常处理语句的信息请参见 [异常处理](../cpp/exception-handling-in-visual-cpp.md)。
