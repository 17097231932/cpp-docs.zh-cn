# 尝试为此项目打开或创建应用程序定义文件时出错。“&lt;error&gt;”
尝试打开应用程序定义文件但该文件不可用时，会发生此错误。  它可能已被移动或删除。  
  
### 更正此错误  
  
-   重新创建应用程序定义文件。
