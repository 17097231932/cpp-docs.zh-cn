# <a name="changing-the-magnification-factor-image-editor-for-icons"></a>更改放大因子（图标的图像编辑器）
默认情况下，图像编辑器在实际大小和实际大小的 6 倍在右窗格中的视图的左窗格中显示该视图。 放大因子 （在工作区底部的状态栏中所示） 是映像的实际大小和显示的大小之间的比率。 默认因子为 6 和范围是从 1 到 10。  
  
### <a name="to-change-the-magnification-factor"></a>若要更改放大因子  
  
1.  选择**图像编辑器**窗格中你想要更改其放大因子。  
  
2.  上[图像编辑器工具栏](../windows/toolbar-image-editor-for-icons.md)，单击右侧的箭头[放大工具](../windows/toolbar-image-editor-for-icons.md)和从子菜单中选择放大因子： **1 X**， **2 X**， **6 X**，或**8 X**。  
  
    > [!NOTE]
    >  若要选择列出的这些类型中的放大因子**放大**工具，使用加速键。
