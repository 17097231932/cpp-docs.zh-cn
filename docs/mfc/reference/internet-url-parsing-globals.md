# Internet URL 分析全局函数
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

当客户端发送查询到 Internet 服务器时，可以使用一 URL 分析全局变量提取有关客户端的信息。  
  
### Internet URL 分析全局函数  
  
|||  
|-|-|  
|[AfxParseURL](../Topic/AfxParseURL.md)|分析 URL 字符串并返回服务类型及其组件。|  
|[AfxParseURLEx](../Topic/AfxParseURLEx.md)|分析 URL 字符串并返回服务类型及其组件，和提供用户名和密码一样。|
