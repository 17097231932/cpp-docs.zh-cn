# <a name="erridn"></a>.ERRIDN]

如果文本项是相同的将生成错误。

## <a name="syntax"></a>语法

> .[[I]] ERRIDN *textitem1*， *textitem2* [[，*消息*]]

## <a name="remarks"></a>备注

如果**我**是给定，比较不区分大小写。
