# <a name="switchis"></a>switch_is

指定的表达式或作为联合判别选择联合成员的标识符。

## <a name="syntax"></a>语法

```cpp
[switch_is]
```

## <a name="remarks"></a>备注

**Switch_is** c + + 属性具有相同的功能[switch_is](/windows/desktop/Midl/switch-is) MIDL 特性。
