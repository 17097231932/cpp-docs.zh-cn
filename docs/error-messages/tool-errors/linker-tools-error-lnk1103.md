# <a name="linker-tools-error-lnk1103"></a>链接器工具错误 LNK1103

调试信息损坏;重新编译模块

导致此错误的原因可能是编译在创建有效对象文件之前被终止。

重新编译给定的对象文件。 如果重新编译不能解决问题，请注意错误的情况，尝试隔离问题并创建可重现的测试用例。 然后，查看技术支持帮助文件或手册中的技术支持部分。
