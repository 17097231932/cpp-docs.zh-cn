# MSBuild 错误 MSB3168
**MSB3168: 重复项“\<程序包\>”将被忽略。**  
  
 当指定引导程序应安装两个完全相同的引导程序包时，便会出现此警告。  在这种情况下，引导程序将仅安装一个实例。
