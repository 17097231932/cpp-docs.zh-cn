# <a name="ml-nonfatal-error-a2034"></a>ML 非致命错误 A2034

**必须在段块中**

在段块外发现了以下内容之一：

- 指令

- 标签定义

- [此](../../assembler/masm/operator-this.md)运算符

- [$](../../assembler/masm/dollar.md)运算符

- 过程定义

- [ALIGN](../../assembler/masm/align-masm.md)指令

- [组织](../../assembler/masm/org.md)指令
