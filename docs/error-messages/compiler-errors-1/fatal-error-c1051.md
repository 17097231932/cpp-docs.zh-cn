# <a name="fatal-error-c1051"></a>错误 C1051

程序数据库文件 "pdbfile" 具有过时的格式，将其删除并重新编译

编译器无法更新具有较旧版本号的程序数据库文件。 删除文件，然后用 **/zi** 或 **/zi** 重新编译程序。 有关详细信息，请参阅 [/Z7、/zi、/zi (调试信息格式) ](../../build/reference/z7-zi-zi-debug-information-format.md)
