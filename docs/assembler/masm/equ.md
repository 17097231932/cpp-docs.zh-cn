# <a name="equ"></a>EQU

第一个指令将分配数值*表达式*到*名称*。

## <a name="syntax"></a>语法

> *名称*EQU*表达式*
>
> *name* EQU \<*text*>

## <a name="remarks"></a>备注

*名称*更高版本不能重新定义。

指定第二个指令分配*文本*到*名称*。 *名称*可以分配不同*文本*更高版本。 请参阅[TEXTEQU](../../assembler/masm/textequ.md)。
