# 编译器错误 C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?': 左边的操作数无效，应为算术类型或指针类型  
  
 `?` 左侧的表达式不能比较为零。  必须使用可以比较为零的算术或指针表达式。
