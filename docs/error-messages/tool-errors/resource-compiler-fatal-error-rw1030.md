# <a name="resource-compiler-fatal-error-rw1030"></a>资源编译器错误 RW1030

输出错误

如果资源编译器无法创建目标文件，则可能会导致此错误。 请确保有足够的磁盘空间，并且您对该卷具有写入权限。
