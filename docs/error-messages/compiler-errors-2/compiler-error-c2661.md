# <a name="compiler-error-c2661"></a>编译器错误 C2661

function： 没有重载的函数接受数字参数

可能的原因：

1. 函数调用中的实际参数不正确。

1. 缺少函数声明。

下面的示例生成 C2661:

```
// C2661.cpp
void func( int ){}
void func( int, int ){}
int main() {
   func( );   // C2661 func( void ) was not declared
   func( 1 );   // OK func( int ) was declared
}
```
