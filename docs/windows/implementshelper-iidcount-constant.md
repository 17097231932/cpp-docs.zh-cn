# <a name="implementshelperiidcount-constant"></a>ImplementsHelper::IidCount 常量
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
  
static const unsigned long IidCount;  
```  
  
## <a name="remarks"></a>备注  
 保存当前的实现对象中的实现接口 Id 的数量。

 **标头：** implements.h  
  
 **Namespace:** Microsoft::WRL::Details
