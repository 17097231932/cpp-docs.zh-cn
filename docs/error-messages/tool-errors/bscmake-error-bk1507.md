# <a name="bscmake-error-bk1507"></a>BSCMAKE 错误 BK1507
无法打开临时文件 filename [: 原因]  
  
 BSCMAKE 无法打开临时文件。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复  
  
1.  **TMP**环境变量未设置为有效的驱动器和目录。  
  
2.  磁盘已满。
