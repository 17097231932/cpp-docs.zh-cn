# <a name="operator-parity"></a>运算符 PARITY?
奇偶校验标志的状态。 只能在使用[。如果](../../assembler/masm/dot-if.md)， [。虽然](../../assembler/masm/dot-while.md)，或[。重复](../../assembler/masm/dot-repeat.md)拦截并在运行时，不是在程序集时计算。  
  
## <a name="syntax"></a>语法  
  
```  
  
PARITY?  
  
```
