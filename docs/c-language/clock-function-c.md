# <a name="clock-function-c"></a>clock 函数 (C)

**ANSI 4.12.2.1** `clock` 函数的纪元

当 C 程序开始执行时，`clock` 函数的纪元将以值 0 开始。 它返回以 1/CLOCKS_PER_SEC（对于 Microsoft C 等于 1/1000）度量的时间。
