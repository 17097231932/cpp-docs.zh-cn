# ComPtr::ptr_ 数据成员
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

包含一个指向相关，并由此 ComPtr 托管接口。  
  
## 语法  
  
```  
InterfaceType *ptr_;  
```  
  
## 备注  
 `ptr_` 是一个内部的，保护的数据成员。  
  
## 要求  
 **标头：**client.h  
  
 **命名空间:** Microsoft::WRL
