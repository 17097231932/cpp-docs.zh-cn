# <a name="trackers-implementing-trackers-in-your-ole-application"></a>跟踪器：在您的 OLE 应用程序内实现跟踪器

跟踪器提供了一个供用户与 OLE 项客户端交互的图形界面。 利用不同的跟踪器样式，可以采用阴影边框、大小调整手柄或者各种其他视觉效果显示 OLE 客户端项。 本文介绍：

- [如何在代码中实现跟踪](../mfc/how-to-implement-tracking-in-your-code.md)。

- [橡胶分级和跟踪](../mfc/rubber-banding-and-trackers.md)器。

本文还介绍了如何对跟踪器使用样式。 此外，它还对 MFC OLE 示例 [OCLIENT](../overview/visual-cpp-samples.md)进行了多个引用。
