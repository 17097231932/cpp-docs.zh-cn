# <a name="compiler-error-c3196"></a>编译器错误 C3196
keyword： 使用多个一次  
  
 不止一次使用的关键字。  
  
 下面的示例生成 C3196:  
  
```  
// C3196.cpp  
// compile with: /clr  
ref struct R abstract abstract {};   // C3196  
ref struct S abstract {};   // OK  
```
