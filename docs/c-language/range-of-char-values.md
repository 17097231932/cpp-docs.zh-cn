# <a name="range-of-char-values"></a>字符值的范围
**ANSI 3.2.1.1**：“纯”**char** 的值范围与 **signed char** 或 `unsigned char` 的值范围是否相同  
  
 所有带符号的字符值的范围都介于 -128 和 127 之间。 所有无符号的字符值的范围介于 0 和 255 之间。  
  
 /J 编译器选项将默认值从 **signed** 更改为 `unsigned`。
