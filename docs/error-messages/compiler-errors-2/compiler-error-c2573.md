# 编译器错误 C2573
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“class”：不能删除指向此类对象的指针；该类没有“operator delete”的非位置重载。  
  
 该类缺少非位置 delete 运算符。
