# <a name="outword"></a>__outword

**Microsoft 专用**

将生成`out`指令，将发送一词`Data`出指定的 I/O 端口`Port`。

## <a name="syntax"></a>语法

```
void __outword(
   unsigned short Port,
   unsigned short Data
);
```

#### <a name="parameters"></a>参数

*端口*<br/>
[in]要向其发送数据的端口。

*Data*<br/>
[in]要发送的数据。

## <a name="requirements"></a>要求

|内部函数|体系结构|
|---------------|------------------|
|`__outword`|x86、x64|

**标头文件** \<intrin.h >

## <a name="remarks"></a>备注

此例程仅可用作内部函数。
