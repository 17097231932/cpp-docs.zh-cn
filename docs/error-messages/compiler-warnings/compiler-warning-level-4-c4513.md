# <a name="compiler-warning-level-4-c4513"></a>编译器警告（等级 4）C4513

"class"：未能生成析构函数

编译器无法为给定的类生成默认析构函数;未创建析构函数。 析构函数位于派生类不可访问的基类中。 如果基类具有私有析构函数，请将其设为公共或受保护。
