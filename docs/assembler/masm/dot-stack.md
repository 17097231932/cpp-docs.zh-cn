# <a name="stack-32-bit-masm"></a>.STACK （32位 MASM）

与一起使用时[。模型](../../assembler/masm/dot-model.md)，定义堆栈段（带有分段名称**stack**）。 可选*大小*为堆栈指定字节数（默认值为1024）。 **。STACK**指令自动关闭 stack 语句。 （仅限32位 MASM。）

## <a name="syntax"></a>语法

> **.STACK** ⟦*size*⟧
