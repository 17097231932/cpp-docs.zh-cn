# <a name="container-classvalue_type"></a>Container Class::value_type

> [!NOTE]
> 本主题在 Microsoft c + + 文档中作为 c + + 标准库中使用的容器的非功能性示例。 有关详细信息，请参阅 [C++ 标准库容器](../standard-library/stl-containers.md)。

可用作模板参数 *Ty* 的同义词。

## <a name="syntax"></a>语法

```cpp
typedef T4 value_type;
```

## <a name="remarks"></a>备注

此处描述为未指定类型的同义词 `T4` (通常 `Alloc::value_type`) 。
