# <a name="sqword"></a>SQWORD

为每个*初始值设定*分配和（可选）初始化8个有符号的存储。 还可在类型合法的任何位置用作类型说明符。

## <a name="syntax"></a>语法

> *名称* **SQWORD** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
