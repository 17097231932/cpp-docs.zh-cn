# &lt;name&gt; is not valid editor for this file type.
该错误通常发生在在“命令”窗口或“查找\/命令”框中为 \/e:*editorname* 参数输入了不正确的值时。  
  
### 更正此错误  
  
1.  在文档中查看 \/e:*editorname* 的正确语法并重新输入该命令。
