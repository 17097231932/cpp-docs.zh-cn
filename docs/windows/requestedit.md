# <a name="requestedit"></a>requestedit
该值指示属性是否支持**OnRequestEdit**通知。  
  
## <a name="syntax"></a>语法  
  
```  
  
[requestedit]  
  
```  
  
## <a name="remarks"></a>备注  
 **Requestedit** c + + 属性具有相同的功能[requestedit](http://msdn.microsoft.com/library/windows/desktop/aa367155) MIDL 特性。
