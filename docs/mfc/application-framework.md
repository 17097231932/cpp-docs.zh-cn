# 应用程序框架
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft 基础类 \(MFC\) \(MFC\) 库的核心是 Windows API 的一个主要组件的封装C\+\+ 窗体。  库类表示窗口、对话框、设备上下文、公用 GDI 对象 \(如画笔、钢笔，标准 Windows 控件和其他项。  这些类提供了一项便利 C\+\+ 函数接口来封装这些成员的窗口结构。  有关更多有关使用这些类，请参见 [窗口主题对象](../mfc/window-objects.md)。  
  
 但是，MFC 库还提供有关 Windows API 的 C\+\+ 生成封装附加的应用程序功能层。  此层是提供大多数常见用户界面的 Windows 程序，包括工具栏、状态栏、打印、打印预览、数据库支持 ActiveX 和支持 Windows 的工作应用程序框架。  [使用类编写 Windows 的应用程序](../mfc/using-the-classes-to-write-applications-for-windows.md) 详细解释框架。
