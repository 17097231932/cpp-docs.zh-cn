# <a name="compiler-error-c2174"></a>编译器错误 C2174

function： 实参具有 void 类型： 参数 number1，参数列表 number2

参数`number1`传递给参数列表`number2`是`void`参数。 参数不能有类型`void`。 请改用 `void*`。
