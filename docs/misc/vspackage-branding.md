# Vspackage 外观方案
VSPackage 可以在 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 初始屏幕和“帮助主题”对话框中显示关于产品的图标和文本信息（合称为徽标）。 对于作为 VSPackage 添加的每个产品，你都可以添加一个徽标。 初始屏幕和“帮助主题”对话框中包含 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 核心产品和其他已安装产品的徽标。
