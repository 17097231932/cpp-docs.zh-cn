# <a name="compiler-error-c3248"></a>编译器错误 C3248

“function1”: 声明为 "__sealed" 的函数无法被“function2”重写

派生的类试图重写 **__sealed** 虚方法。

可能出现 c3248 错误才可访问使用已过时的编译器选项 **/clr: oldsyntax**。
