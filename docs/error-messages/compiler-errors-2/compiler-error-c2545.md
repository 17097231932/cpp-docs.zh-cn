# <a name="compiler-error-c2545"></a>编译器错误 C2545

"operator"：找不到重载运算符

运算符不能与提供的操作数一起使用。 必须向重载运算符提供所需的操作数。

如果操作数的类型不正确，则可能导致此错误。

如果定义的转换运算符或构造函数采用单个参数，则可能会修复此错误。
