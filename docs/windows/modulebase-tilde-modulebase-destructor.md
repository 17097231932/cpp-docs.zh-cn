# <a name="modulebasemodulebase-destructor"></a>ModuleBase::~ModuleBase 析构函数
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
virtual ~ModuleBase();  
```  
  
## <a name="remarks"></a>备注  
 取消初始化 ModuleBase 类的当前实例。  
  
## <a name="requirements"></a>惠?  
 **标头：** implements.h  
  
 **Namespace:** Microsoft::WRL::Details
