# <a name="compiler-error-c3076"></a>编译器错误 C3076

instance： 不能在本机类型中嵌入引用类型，type 的实例

本机类型不能包含 CLR 类型的实例。

有关详细信息，请参阅[的引用类型的 c + + 堆栈语义](../../dotnet/cpp-stack-semantics-for-reference-types.md)。

## <a name="example"></a>示例

下面的示例生成 C3076。

```
// C3076.cpp
// compile with: /clr /c
ref struct U {};

struct V {
   U y;   // C3076
};

ref struct W {
   U y;   // OK
};
```
