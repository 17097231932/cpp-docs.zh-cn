# <a name="expression-evaluator-error-cxx0032"></a>表达式计算器错误 CXX0032

除以0

表达式包含零除数，这是非法的。 此除数可以是文本数字零，也可以是计算结果为零的表达式。

此错误与 CAN0032 相同。
