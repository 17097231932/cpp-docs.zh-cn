# <a name="default-atl-project-configurations"></a>默认 ATL 项目配置

默认情况下，ATL 项目向导创建两个项目配置：

|Configuration|字符集|ATL 的使用|
|-------------------|-------------------|----------------|
|发布|MBCS|DLL|
|调试|MBCS|DLL|

**字符集**、 **ATL 的使用** 和可以在 "**常规**" 选项卡下的 "**项目设置**" 对话框中进行更改。你还可以使用 Configuration Manager 添加自己的配置。 有关详细信息，请参阅 [生成配置](/visualstudio/ide/understanding-build-configurations)。
