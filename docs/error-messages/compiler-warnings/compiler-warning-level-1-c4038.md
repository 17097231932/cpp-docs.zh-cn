# <a name="compiler-warning-level-1-c4038"></a>编译器警告（等级 1）C4038
“modifier”：非法的环境类修饰符  
  
 具有 **dllimport** 或 [dllexport](../../cpp/dllexport-dllimport.md) 特性的类不能使用此修饰符。
