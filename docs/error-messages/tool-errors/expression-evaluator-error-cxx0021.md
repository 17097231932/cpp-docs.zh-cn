# <a name="expression-evaluator-error-cxx0021"></a>表达式计算器错误 CXX0021
结构或联合用作标量  
  
 在表达式中，使用结构或联合，但未指定任何元素。  
  
 当操作结构或联合变量，变量的名称可能单独出现，而无需域限定符。 如果在表达式中使用结构或联合，必须用所需的特定元素进行限定。  
  
 指定要在表达式中使用其值的元素。  
  
 此错误是与 CAN0021 相同。
