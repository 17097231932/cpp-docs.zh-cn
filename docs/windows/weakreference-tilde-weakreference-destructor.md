# WeakReference::~WeakReference 析构函数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
virtual ~WeakReference();  
```  
  
## <a name="return-value"></a>返回值  
  
## <a name="remarks"></a>备注  
 取消初始化 WeakReference 类的当前实例。
