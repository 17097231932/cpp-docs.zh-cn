# 使用工具窗口中的渐变服务
由于 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 工具窗口使用 windows presentation foundation \(WPF\) 生成，不必调用代码中的渐变服务。  相反，在表示工具窗口的设计器图面上，选择 `StackPanel` 元素，然后，在 **属性** 窗口中，修改 **背景** 属性设置渐变。  有关更多信息，请参见 [设置颜色、渐变和不透明度](../misc/setting-colors-gradients-and-opacity.md)。
