# <a name="regex_traitsltchargt-class"></a>regex_traits&lt;char&gt; 类

的专用 `regex_traits` 化 **`char`** 。

## <a name="syntax"></a>语法

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>备注

类是对类型元素的类模板[regex_traits](../standard-library/regex-traits-class.md)的显式专用化 **`char`** （以便它可以充分利用操作此类型对象的库函数）。
