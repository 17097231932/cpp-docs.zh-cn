# <a name="fatal-error-c1093"></a>错误 C1093

> API 调用 "*function name*" 未能*在运行时* *的位置*":" 文本

对 .NET 函数的调用失败。 *运行时字符串中的文本*可能由 COM 运行时提供, 也可能不提供。

有关系统错误消息的详细信息, 请参阅 winerror.h 系统文件和[FormatMessage](/windows/win32/api/winbase/nf-winbase-formatmessage)。
