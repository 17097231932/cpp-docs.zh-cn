# <a name="expression-evaluator-error-cxx0028"></a>表达式计算器错误 CXX0028

not struct/union 元素

指定了 "Struct. Member" 或 ">pStruct" 成员的表达式，但 \<member> 它不是该结构的元素。

表达式的表达式可能不正确。

此错误与 CAN0028 相同。
