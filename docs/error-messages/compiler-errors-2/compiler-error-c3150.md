# <a name="compiler-error-c3150"></a>编译器错误 C3150

'element': attribute 仅应用于类、 接口、 数组或指针

`__gc` 仅在类、 接口或数组上使用。

C3150 才可访问使用已过时的编译器选项 **/clr: oldsyntax**。
