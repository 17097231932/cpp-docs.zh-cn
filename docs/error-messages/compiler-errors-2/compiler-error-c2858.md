# 编译器错误 C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

命令行选项“\/Yc \(\/Fdfilename\)”与使用“\/Fdfilename”的预编译头不一致  
  
 “使用预编译头”\([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) 选项指定的程序数据库不是前一个“创建预编译头”\([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\) 选项指定的程序数据库。
