# <a name="compiler-error-c3632"></a>编译器错误 C3632
event： 非法样式事件 for 构造  
  
 [__event](../../cpp/event.md)声明并非在所有构造中有效。  
  
 C3632 才可访问使用过时的编译器选项 **/clr:oldSyntax**。
