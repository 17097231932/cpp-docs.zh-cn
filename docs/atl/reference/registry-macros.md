# Registry Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些宏定义有用的类型库和注册表结构。  
  
|||  
|-|-|  
|[\_ATL\_STATIC\_REGISTRY](../Topic/_ATL_STATIC_REGISTRY.md)|指示您希望在对象中注册代码可以以避免在对象的ATL.DLL的依赖项。|  
|[DECLARE\_LIBID](../Topic/DECLARE_LIBID.md)|为ATL提供了一种获取该类型库的 *libid*。|  
|[DECLARE\_NO\_REGISTRY](../Topic/DECLARE_NO_REGISTRY.md)|避免默认ATL注册。|  
|[DECLARE\_REGISTRY](../Topic/DECLARE_REGISTRY.md)|输入或移除在系统注册表的主要目的项。|  
|[DECLARE\_REGISTRY\_APPID\_RESOURCEID](../Topic/DECLARE_REGISTRY_APPID_RESOURCEID.md)|指定信息自动需要注册 *appid*。|  
|[DECLARE\_REGISTRY\_RESOURCE](../Topic/DECLARE_REGISTRY_RESOURCE.md)|在其中查找这种名为资源并运行注册表脚本。|  
|[DECLARE\_REGISTRY\_RESOURCEID](../Topic/DECLARE_REGISTRY_RESOURCEID.md)|在其中找到了该资源标识由ID号并运行注册表脚本。|
