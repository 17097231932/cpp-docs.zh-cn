# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>MFC DLL 向导生成的类和函数

MFC DLL 向导生成的代码取决于要创建的 DLL 类型和所选的选项。 MFC DLL 向导为两种形式的规则 MFC Dll 生成相同的代码。

|DLL 类型|选项|类|函数|
|-----------------|------------|-------------|---------------|
|[扩展名](../../build/extension-dlls-overview.md)|无|无|`DllMain`|
|[常规](../../build/regular-dlls-dynamically-linked-to-mfc.md)|无|从派生的应用程序类 `CWinApp`|无|
|[常规](../../build/regular-dlls-dynamically-linked-to-mfc.md)|自动化|从派生的应用程序类 `CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[扩展名](../../build/extension-dlls-overview.md)|窗口套接字|无|`DllMain`|
|[常规](../../build/regular-dlls-dynamically-linked-to-mfc.md)|窗口套接字|从派生的应用程序类 `CWinApp`|`InitInstance` 包含对的调用 `AfxSocketInit`|
