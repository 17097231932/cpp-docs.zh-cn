# <a name="resource-compiler-error-rc2017"></a>资源编译器错误 RC2017

非法的转义序列

转义序列出现在不应出现的位置。

转义序列（反斜杠 ( **\\** ) 后跟数字或字母）只能在字符或字符串常量中出现。
