# <a name="word"></a>WORD

分配并根据需要为每个初始化存储一个单词 （2 个字节） `initializer`。 也可以用作类型说明符类型是合法的任何位置。

## <a name="syntax"></a>语法

> [[*名称*]] WORD*初始值设定项*[[，*初始值设定项*]]...
