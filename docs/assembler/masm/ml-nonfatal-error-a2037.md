# <a name="ml-nonfatal-error-a2037"></a>ML 非致命错误 A2037

**不允许在结构定义语句**

结构定义中包含了无效的语句。

结构不能包含说明、 标签、 过程、 控制流指令[。启动](../../assembler/masm/dot-startup.md)，或[。退出](../../assembler/masm/dot-exit.md)。
