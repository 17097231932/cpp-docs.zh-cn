# <a name="compiler-error-c3456"></a>编译器错误 C3456

[source_annotation_attribute] 不可用于托管或 WinRT 类声明

source_annotation_attribute 用于定义将被代码分析使用的自定义属性。

只有当存在理解此属性的相应插件时，这些自定义属性才有用。
