# <a name="compiler-error-c2121"></a>编译器错误 C2121

"#"：无效字符：可能是宏扩展的结果

使用标记粘贴运算符 ( # # ) 而不是字符串化运算符 ( # ) 中的错误宏可能插入了无效的 # 字符。
