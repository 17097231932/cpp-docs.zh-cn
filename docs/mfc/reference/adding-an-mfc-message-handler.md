# <a name="adding-an-mfc-message-handler"></a>添加 MFC 消息处理程序

你可以[使用属性窗口添加消息处理程序](../../mfc/reference/mapping-messages-to-functions.md)（处理 Windows 消息的成员函数） 到消息处理程序类和映射 Windows 消息。 您还可以添加[任何对话框控件的事件处理程序](../../windows/adding-event-handlers-for-dialog-box-controls.md)。

通过使用属性窗口来定义消息和事件处理函数，您可以自动更新的消息调度表 （或消息映射） 和类的标头文件。

> [!NOTE]
>  可以将消息处理程序添加到 ATL 类使用属性窗口中;但是，某些结果可能会有所不同。 有关详细信息，请参阅 ATL 主题[添加 ATL 消息处理程序](../../atl/adding-an-atl-message-handler.md)。
