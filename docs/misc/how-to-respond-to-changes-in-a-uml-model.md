# 如何：响应 UML 模型中的更改
在 Visual Studio 中的 UML 模型中出现更改时可以编写已执行的代码。  它将同样会响应直接通过该用户，以及通过其他 Visual Studio 扩展所做的更改。  若要查看支持 UML 模型的 Visual Studio 版本，请参阅[对体系结构和建模工具的版本支持](../Topic/What's%20new%20for%20design%20in%20Visual%20Studio.md#VersionSupport)。  
  
> [!WARNING]
>  这些技术不受 UML API 支持。  它们可能无法在将来版本的 Visual Studio 中使用。  
  
 示例代码可在 [UML：通过使用事件和规则响应 UML 模型中的更改](http://code.msdn.microsoft.com/UML-Responding-to-changes-c024cd4b)中找到
