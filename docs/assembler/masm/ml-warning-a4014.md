# ML Warning A4014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 BSS 段和初始化的数据不支持的命令  
  
 尝试定义在 BSS 节中的初始化的数据。  BSS 部分定义，因为名称是 BSS 的类。  这包括简化的段 `.data?`。
