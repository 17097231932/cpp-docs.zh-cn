# <a name="fatal-error-c1107"></a>错误 C1107
找不到程序集 file： 请指定程序集搜索路径中使用 /AI 或通过设置 LIBPATH 环境变量  
  
 元数据文件已传递到[#using](../../preprocessor/hash-using-directive-cpp.md)编译器找不到的指令。  
  
 LIBPATH 中的主题所述`#using`，和[/AI](../../build/reference/ai-specify-metadata-directories.md)编译器选项，你可以指定在其中编译器将查找引用元数据文件的目录。
