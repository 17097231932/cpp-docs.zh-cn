# <a name="handletisvalid-method"></a>HandleT::IsValid 方法
指示是否当前**HandleT**对象表示的句柄。  
  
## <a name="syntax"></a>语法  
  
```cpp  
bool IsValid() const;  
```  
  
## <a name="return-value"></a>返回值  
 **true**如果**HandleT**表示的句柄; 否则为**false**。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
