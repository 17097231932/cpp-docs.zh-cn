# <a name="cframewndex-class"></a>CFrameWndEx 类

实现 Windows 单文档界面 (SDI) 重叠式或弹出框架窗口功能，并提供管理窗口的成员。 它扩展了[CFrameWnd](../../mfc/reference/cframewnd-class.md)类。

## <a name="syntax"></a>语法

```
class CFrameWndEx : public CFrameWnd
```
