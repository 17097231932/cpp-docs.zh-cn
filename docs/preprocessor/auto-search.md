# auto_search
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 专用**  
  
 指定当使用 \#import 引用了类型库，并且它本身引用了另一个类型库时，编译器可以为其他类型库执行隐式 \#import。  
  
## 语法  
  
```  
auto_search  
```  
  
## 备注  
 **结束 C\+\+ 专用**
