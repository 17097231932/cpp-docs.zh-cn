# 编译器错误 C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“\#”: 无效字符 : 可能是宏展开的结果  
  
 无效的 \# 字符可能已由不正确的宏插入，该宏使用标记粘贴运算符 \(\#\#\) 而不是字符串化运算符 \(\#\)。
