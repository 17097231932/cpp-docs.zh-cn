# Using a Template Library
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

模板是有些类似于宏。  与宏，调用模板使其展开\(与适当的参数替换\)代码编写。  但是，模板进一步比此转到允许根据您将作为参数类型的选件类的创建。  执行用您的模板代码表示的操作这些新选件类实现的类型安全方法。  
  
 模板库\(如ATL与传统C\+\+选件类库不同它们仅通常会提供作为源代码\(或作为一个略加的源代码，支持运行时\)并不从本质上而言是或命令分层的性质。  而不是从派生选件类以获取所需的功能，您实例化模板的选件类。
