# Editing Accelerator Tables
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在 Visual C\+\+ 中，可以直接使用快捷键编辑器中的就地编辑功能来编辑快捷键对应表。  
  
 下面的过程涉及标准属性页的使用，然而，就地编辑和属性页方法具有相同的结果。  使用属性页或就地编辑所做的更改立即反映在快捷键对应表中。  
  
-   [编辑快捷键对应表](../windows/editing-in-an-accelerator-table.md)  
  
-   [在快捷键对应表中查找项](../windows/finding-an-entry-in-an-accelerator-table.md)  
  
-   [向快捷键对应表添加项](../windows/adding-an-entry-to-an-accelerator-table.md)  
  
-   [从快捷键对应表删除项](../windows/deleting-an-entry-from-an-accelerator-table.md)  
  
-   [移动或复制快捷键对应表项](../windows/moving-or-copying-an-accelerator-table-entry-to-another-resource-script-file.md)  
  
-   [更改多个快捷键的属性](../windows/changing-the-properties-of-multiple-accelerator-keys.md)  
  
 有关将资源添加到托管项目的信息，请参见“.NET Framework 开发员指南”中的[应用程序中的资源](../Topic/Resources%20in%20Desktop%20Apps.md)。有关手动将资源文件添加到托管项目、访问资源、显示静态资源和将资源字符串分配给属性的信息，请参见[演练：本地化 Windows 窗体](http://msdn.microsoft.com/zh-cn/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
