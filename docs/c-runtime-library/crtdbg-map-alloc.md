# _CRTDBG_MAP_ALLOC
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

当 **\_CRTDBG\_MAP\_ALLOC** 标志。应用程序的调试版本中定义，堆函数的基版本映射直接对它们的调试版本。  在标志 Crtdbg.h 执行映射。  在 [\_DEBUG](../c-runtime-library/debug.md) 标志在应用程序中，定义此标志才可用。  
  
 有关使用调试版本的更多信息和堆函数的基版本，请参见 [使用调试版本与基版本](../Topic/Debug%20Versions%20of%20Heap%20Allocation%20Functions.md)。
