# 使用 CComboBoxEx
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

扩展组合框控件是为项图像提供本机支持标准组合框控件的扩展。  这些图像可以用于指示各项状态组合框中，如当前选择和取消选中的项。  若要使项图像，提供易于访问控件图像列表支持。  
  
 使用此组合框控件提供的功能，而无需手动绘制项图形。
