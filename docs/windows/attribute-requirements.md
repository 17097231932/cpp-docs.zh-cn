# Attribute Requirements
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

列出要求的 C\+\+ 特性描述项类型、编译器设置和其他信息所需的属性来工作。  邮件类下述。  
  
> [!NOTE]
>  使用在从类派生还可以使用属性的类的属性不受支持。  
  
## Header  
 此字段列表必须包括的头文件，才能使用属性。  
  
## Project  
 此字段描述可以使用属性的项目类型。  
  
## 编译器  
 此字段提供必须存在要使了此属性的编译器选项。
