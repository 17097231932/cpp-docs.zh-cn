# <a name="sqword"></a>SQWORD

分配并根据需要初始化每个 8 字节有符号的存储`initializer`。 此外可以用作类型说明符类型是合法的任何位置。

## <a name="syntax"></a>语法

> 名称 SQWORD 初始值设定项 [[，初始值设定项]]...
