# 关于异常的疑难解答：System.IO.DirectoryNotFoundException
当未找到部分文件路径或目录时，会引发 <xref:System.IO.DirectoryNotFoundException> 异常。  
  
## 相关提示  
 **验证目录是否出现在指定位置。**  
 检查目录是否存在于指定位置。  
  
 **使用相对路径时，请确保当前目录是正确的。**  
 如果采用了不正确的当前目录，则路径可能不正确。
