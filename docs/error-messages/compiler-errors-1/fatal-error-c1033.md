# <a name="fatal-error-c1033"></a>错误 C1033

无法打开程序数据库 pdb

此错误可能是由以下原因引起的：磁盘错误、防病毒程序创建的临时锁、未完全关闭的上一个调试器实例，或尝试访问同一文件的并行生成 mspdbsrv.exe 进程，以及其他可能的原因。
