# 编译器警告（等级 1）C4045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“array”：数组界限溢出  
  
 数组的初始值设定项太多。 忽略多余的初始值设定项。  
  
 请确保数组元素与初始值设定项在大小和数量上匹配。
