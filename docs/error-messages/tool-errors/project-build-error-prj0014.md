# <a name="project-build-error-prj0014"></a>项目生成错误 PRJ0014

用于控制生成过程的作业对象失败。  生成无法继续。

开发环境出错。

若要解决此错误，请关闭并重新加载项目。 如有必要，请退出并重新启动 Visual Studio。
