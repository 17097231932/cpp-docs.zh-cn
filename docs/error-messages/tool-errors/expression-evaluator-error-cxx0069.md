# <a name="expression-evaluator-error-cxx0069"></a>表达式计算器错误 CXX0069

变量需要堆栈帧

表达式计算器无法计算变量，因为它未出现在堆栈帧中。 这可能是由于声明为内联函数的一部分的变量所致。
