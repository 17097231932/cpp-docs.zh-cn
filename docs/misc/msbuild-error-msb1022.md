# MSBuild 错误 MSB1022
**响应文件不存在。**  
  
 指定位置中不存在响应文件。  
  
### 更正此错误  
  
1.  如果指定了响应文件的绝对位置或相对位置，请检查指定目录中是否存在响应文件。
