# <a name="application-settings-makefile-project-wizard"></a>应用程序设置，生成文件项目向导
提供项目的信息。 指定的选项将反映在项目的[属性页](../ide/working-with-project-properties.md)。 一旦你创建项目时，你可以查看和编辑每个中的下列选项**Nmake**的项目的属性页的页。  
  
 **生成命令行**  
 指定的命令行运行时用户选择**生成**从**生成**菜单。 显示在**生成命令行**字段上**Nmake**的项目的属性页的页。  
  
 **输出**  
 指定将包含命令行的输出文件的名称。 默认情况下，此选项根据项目名称。 显示在**输出**字段上**Nmake**的项目的属性页的页。  
  
 **清除命令**  
 指定的命令行运行时用户选择**清理**从**生成**菜单。 显示在**干净的命令行**字段上**Nmake**的项目的属性页的页。  
  
 **重新生成命令行**  
 指定的命令行运行时用户选择**重新生成**从**生成**菜单。 显示在**重新生成所有命令行**字段上**Nmake**的项目的属性页的页。
