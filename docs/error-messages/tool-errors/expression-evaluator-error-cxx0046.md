# <a name="expression-evaluator-error-cxx0046"></a>表达式计算器错误 CXX0046
所需的成员函数的自变量列表  
  
 一个表达式调用成员函数，但未指定任何实际的参数。  
  
 此错误是与 CAN0046 相同。
