# <a name="string-and-text-classes"></a>字符串和文本类

这些类提供对字符串和文本字符串转换的支持。

- [CA2AEX](../atl/reference/ca2aex-class.md) 此类由字符串转换宏 CA2TEX 和 CT2AEX 以及 typedef CA2A 使用。

- [CA2CAEX](../atl/reference/ca2caex-class.md) 此类由字符串转换宏 CA2CTEX 和 CT2CAEX 以及 typedef CA2CA 使用。

- [CA2WEX](../atl/reference/ca2wex-class.md) 此类由字符串转换宏 CA2TEX、CA2CTEX、CT2WEX、CT2CWEX 和 typedef CA2W 使用。

- [CW2AEX](../atl/reference/cw2aex-class.md) 此类由字符串转换宏 CT2AEX、CW2TEX、CW2CTEX、CT2CAEX 和 typedef CW2A 使用。

- [CW2CWEX](../atl/reference/cw2cwex-class.md) 此类由字符串转换宏 CW2CTEX 和 CT2CWEX 以及 typedef CW2CW 使用。

- [CW2WEX](../atl/reference/cw2wex-class.md) 此类由字符串转换宏 CW2TEX 和 CT2WEX 以及 typedef CW2W 使用。

- [CComBSTR](../atl/reference/ccombstr-class.md) 此类是 Bstr 的包装器。

- [_U_STRINGorID](../atl/reference/u-stringorid-class.md) 此参数适配器类允许将资源名称 (`LPCTSTR`) 或资源 id (UINTs) 传递到函数，而无需调用方使用 MAKEINTRESOURCE 宏将 ID 转换为字符串。
