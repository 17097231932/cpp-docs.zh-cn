# “新建自定义资源”对话框
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使您得以创建新自定义资源。  
  
 **资源类型**  
 提供用于输入自定义资源类型名称的文本框。  当您退出**“新建自定义资源”**对话框时，Visual C\+\+ 会自动将名称变为大写。  
  
## 备注  
 不要指定 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 保留的资源名称或 [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)]。  保留名称为 DESIGNINFO、HWB 和 TEXTINCLUDE，保留的 [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] 为 255。
