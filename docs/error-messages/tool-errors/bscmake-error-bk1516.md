# <a name="bscmake-error-bk1516"></a>BSCMAKE 错误 BK1516

bscfile 损坏;不能以增量方式更新

.Bsc 文件已损坏，可能是由于在生成期间在系统出现故障。 删除.bsc 文件，重新生成所有的.sbr 文件，然后重新生成.bsc 文件。
