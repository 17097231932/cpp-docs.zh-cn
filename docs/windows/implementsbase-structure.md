# <a name="implementsbase-structure"></a>ImplementsBase 结构
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
struct ImplementsBase;  
```  
  
## <a name="remarks"></a>备注  
 用于验证中的模板参数类型[Implements 结构](../windows/implements-structure.md)。  
  
 **ImplementsBase**结构为空的设计。
