# <a name="resource-compiler-error-rc2152"></a>资源编译器错误 RC2152
无效的控制字符  
  
 **ACCELERATORS** 语句中的控制字符无效。 一个有效的控制字符包含插入符号 （仅限） 后跟一个字母 (**^**)。
