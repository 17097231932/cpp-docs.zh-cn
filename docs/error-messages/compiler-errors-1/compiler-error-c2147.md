# <a name="compiler-error-c2147"></a>编译器错误 C2147

语法错误: identifier 是一个新的关键字

使用现在是保留的关键字的语言标识符。

下面的示例生成 C2147:

```
// C2147.cpp
// compile with: /clr
int main() {
   int gcnew = 0;   // C2147
   int i = 0;   // OK
}
```
