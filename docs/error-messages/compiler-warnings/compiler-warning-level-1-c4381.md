# <a name="compiler-warning-level-1-c4381"></a>编译器警告（等级 1）C4381

"function1"：非公共方法 "function2" 将不实现接口方法

类必须实现接口中的所有函数。 如果某个类的一个基类实现了此函数，则该类可以满足此条件。 但是，该函数必须作为公共函数实现。
