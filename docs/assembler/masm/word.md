# <a name="word"></a>WORD

为每个*初始值设定项*分配一个字（2个字节），并根据需要对其进行初始化。 在类型合法的任何位置，还可以用作类型说明符。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **WORD** *初始值设定*项⟦ __，__ *初始值设定项*.。。⟧
