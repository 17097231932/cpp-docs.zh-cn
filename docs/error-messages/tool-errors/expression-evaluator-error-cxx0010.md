# 表达式计算器错误 CXX0010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

缺少左尖括号  
  
 表达式包含不对称的方括号。  使用对称的方括号重新输入表达式。  
  
 该错误与 CAN0010 相同。
