# <a name="expression-evaluator-error-cxx0034"></a>表达式计算器错误 CXX0034

**与运算符不兼容的类型**

指定的操作数类型对运算是非法的。

例如，指针不能与任何值相乘。

可能需要键入将操作数转换为与运算符兼容的类型。
