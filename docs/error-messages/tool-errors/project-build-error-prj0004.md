# <a name="project-build-error-prj0004"></a>项目生成错误 PRJ0004

> 无法为 "*工具*" 工具生成命令行。

以这样一种方式指定了一个或多个属性，以使对 *工具* 调用的语法无效：

- 您可能指定了格式错误或未知的宏。

- 计算机的可用磁盘空间可能不足。

您可能想要通过查看 " [命令行" 属性页](../../build/reference/command-line-property-pages.md)来查看该工具的设置。
