# The application cannot undo.
该错误通常发生在尝试对一个与另一个文件有链接撤消的项执行撤消操作时。  如果在链接的修改后执行了操作，Visual Studio 将无法回滚更改。  
  
### 更正此错误  
  
1.  手动反转更改。
