# <a name="compiler-error-c2325"></a>编译器错误 C2325
type： 右侧的 name 的意外的类型  
  
 对不正确的类型的析构函数进行调用。  
  
 下面的示例生成 C2325:  
  
```  
// C2325.cpp  
// compile with: /c  
class A {};  
typedef A* pA_t;  
void f() {  
    A** ppa = new A *;  
    ppa->~A*;   // C2325  
  
   pA_t *ppa2 = new pA_t;  
   ppa2->~pA_t();   // OK  
}  
```
