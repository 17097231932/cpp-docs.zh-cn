# <a name="cvtres-warning-cvt4001"></a>CVTRES 警告 CVT4001

> 未指定计算机类型;假定 *类型*

CVTRES 未找到计算机规范。 它假定给定的计算机类型。 如果默认值不正确，请使用/MACHINE 选项重新运行 CVTRES。
