# <a name="active-template-library-atl-tutorial"></a>活动模板库 (ATL) 教程
ATL 旨在简化创建高效、 灵活、 轻型控件的过程。 本教程将引导你逐步完成创建 ActiveX 控件，演示许多 ATL 和 COM 的基础知识。  
  
 通过遵循本教程，你将学习如何将控件添加到绘制圆形和填充的多边形的 ATL 项目。 然后，你将添加一个属性以指示多少条边，多边形将拥有，并创建在属性变化时更新控件的绘图代码。 控件将显示在网页上使用某些 VBScript 以使其响应事件。  
  
 本教程分为七个步骤。 后续步骤依赖于以前已完成的任务，你应按顺序执行每个步骤。 在开始之前，你应确认你拥有注册您的特定计算机上的 ActiveX 组件所需的权限。 如果你正在运行 Visual Studio.NET 通过终端服务连接，这通常是仅一个问题。  
  
-   [步骤 1： 创建项目](../atl/creating-the-project-atl-tutorial-part-1.md)  
  
-   [步骤 2： 将控件添加到你的项目](../atl/adding-a-control-atl-tutorial-part-2.md)  
  
-   [步骤 3。将属性添加到你的控件](../atl/adding-a-property-to-the-control-atl-tutorial-part-3.md)  
  
-   [步骤 4： 更改控件的绘图代码](../atl/changing-the-drawing-code-atl-tutorial-part-4.md)  
  
-   [步骤 5： 添加事件](../atl/adding-an-event-atl-tutorial-part-5.md)  
  
-   [步骤 6： 添加属性页](../atl/adding-a-property-page-atl-tutorial-part-6.md)  
  
-   [步骤 7： 将你在网页上的控件放](../atl/putting-the-control-on-a-web-page-atl-tutorial-part-7.md)
