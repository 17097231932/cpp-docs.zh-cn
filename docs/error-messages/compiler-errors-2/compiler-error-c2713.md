# <a name="compiler-error-c2713"></a>编译器错误 C2713

只有一个窗体的每个函数允许异常处理

不能使用结构化的异常处理 (`__try`/`__except`) 和 c + + 异常处理 (`try`/`catch`) 在同一函数中。
