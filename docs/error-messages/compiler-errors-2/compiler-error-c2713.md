# <a name="compiler-error-c2713"></a>编译器错误 C2713
只有一种形式的允许每个函数的异常处理  
  
 不能使用结构化的异常处理 (`__try`/`__except`) 和 c + + 异常处理 (`try`/`catch`) 相同的函数中。
