# 错误 C1098
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

版本与“编辑并继续”引擎不匹配  
  
 正在使用的调试器版本与用来创建可执行文件的编译器不匹配。 如果重新编译不能解决问题，则可能需要重新安装 Visual C\+\+，以确保拥有正确的调试器和编译器版本。
