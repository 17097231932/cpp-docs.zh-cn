# <a name="ole-db-classes"></a>OLE DB 类

MFC 中的 OLE DB 支持当前包含[COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)类。 `COleDBRecordView` 通过直接连接到[CRowset](../data/oledb/crowset-class.md)对象的窗体视图显示控件中的数据库记录。 有关 OLE DB 使用者模板的详细信息，请参阅[OLE DB 使用者模板列表](../data/oledb/ole-db-consumer-templates-reference.md)。
