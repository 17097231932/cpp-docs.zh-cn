# <a name="_heapmin"></a>_heapmin

将未使用的堆内存释放到操作系统。

## <a name="syntax"></a>语法

```C
int _heapmin( void );
```

## <a name="return-value"></a>返回值

如果成功，_heapmin返回 0;如果成功，**则返回**0。否则，函数将返回 -1 并将**errno**设置到**ENOSYS**。

有关此代码及其他返回代码的详细信息，请参阅 [_doserrno、errno、_sys_errlist 和 _sys_nerr](../../c-runtime-library/errno-doserrno-sys-errlist-and-sys-nerr.md)。

## <a name="remarks"></a>备注

**_heapmin**函数通过将未使用的堆内存释放到操作系统，将堆最小化。 如果操作系统不支持 **_heapmin（** 例如，Windows 98），则函数将返回 -1 并将**errno**设置到**ENOSYS**。

默认情况下，此函数的全局状态范围为应用程序。 要更改此情况，请参阅[CRT 中的全局状态](../global-state.md)。
