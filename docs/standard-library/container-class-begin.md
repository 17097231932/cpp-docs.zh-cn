# <a name="container-classbegin"></a>Container Class::begin

> [!NOTE]
> 本主题位于 Visual Studio C++文档中，作为在C++标准库中使用的容器的非功能性示例。 有关详细信息，请参阅 [C++ 标准库容器](../standard-library/stl-containers.md)。

该成员函数返回一个迭代器，指向序列的第一个元素（或刚超出空序列末尾的位置）。

## <a name="syntax"></a>语法

```cpp
const_iterator begin() const;

iterator begin();
```
