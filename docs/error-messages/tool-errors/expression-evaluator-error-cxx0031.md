# <a name="expression-evaluator-error-cxx0031"></a>表达式计算器错误 CXX0031

不可扩展的表达式

表达式计算器遇到内部错误。

您可能能够编写可以正确地计算的等效表达式。

此错误是与 CAN0031 相同。
