# 项目生成错误 PRJ0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未能继续挂起的进程。生成已失败。  
  
 开发环境中发生错误。  
  
 若要解决此错误，请关闭并重新加载项目。  如有必要，请退出并重新启动 Visual Studio。
