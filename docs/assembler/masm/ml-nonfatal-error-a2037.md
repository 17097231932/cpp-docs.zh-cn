# ML Nonfatal Error A2037
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**语句不允许的内部结构定义**  
  
 结构定义包含了无效语句。  
  
 结构不能包含命令、标签、过程、控制流指令、 [.STARTUP](../../assembler/masm/dot-startup.md)或 [.EXIT](../../assembler/masm/dot-exit.md)。
