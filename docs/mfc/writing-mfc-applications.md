# <a name="writing-mfc-applications"></a>编写 MFC 应用程序
此文章介绍了执行来开发应用程序的初始步骤。 首先，必须确定哪种应用程序正在编写。 多个选项所述[应用程序设计选择](../mfc/application-design-choices.md)。 将为你的应用程序：  
  
-   Internet 或 intranet 上运行  
  
-   在客户端或服务器上运行  
  
-   运行在浏览器中或作为独立应用程序  
  
-   使用 COM 或处于活动状态的技术  
  
-   下载使用 WinInet 或异步名字对象的数据  
  
 您的决策确定哪些类是适用于你的应用程序。 您的回答还有助于确定运行应用程序向导，开始构建您的应用程序时所做的选择。  
  
 您的初始设计决策进行有关 Internet 应用程序之后，可以使用应用程序向导来开始。 使用应用程序向导创建主干应用程序和修改代码，如以下文章中所述：  
  
-   ActiveX 控件，请参阅[Internet 上的 ActiveX 控件](../mfc/activex-controls-on-the-internet.md)。  
  
 以下文章还提供了说明，帮助你开始编程任务：  
  
-   [应用程序设计选项](../mfc/application-design-choices.md)  
  
-   [Internet 上的异步名字对象](../mfc/asynchronous-monikers-on-the-internet.md)  
  
-   [WinInet 基础知识](../mfc/wininet-basics.md)
