# <a name="moduleincrementobjectcount-method"></a>Module::IncrementObjectCount 方法
递增模块所跟踪对象的数量。  
  
## <a name="syntax"></a>语法  
  
```  
virtual long IncrementObjectCount();  
```  
  
## <a name="return-value"></a>返回值  
 递增操作之前的计数。  
  
## <a name="requirements"></a>要求  
 **标头：** module.h  
  
 **命名空间：** Microsoft::WRL
