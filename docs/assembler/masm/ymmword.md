# <a name="ymmword"></a>YMMWORD

用于包含 Intel 高级矢量扩展（AVX）说明的256位多媒体操作数。

## <a name="syntax"></a>语法

> **YMMWORD**

## <a name="remarks"></a>备注

**YMMWORD**用于表示与 AVX 内部函数的 `__m256` 相同的类型。
