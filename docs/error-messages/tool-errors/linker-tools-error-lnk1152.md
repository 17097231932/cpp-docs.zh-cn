# <a name="linker-tools-error-lnk1152"></a>链接器工具错误 LNK1152

无法解析一个或多个未修饰符号

对于无法解析的每个未修饰符号，此错误前面会出现一个警告 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) ，对于为未修饰符号找到的重复符号，此错误至少有两个警告 [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) 。
