# <a name="operator-dup"></a>运算符 DUP

指定 *initialvalue* 的 *声明数。*

## <a name="syntax"></a>语法

> *计数* **DUP** __(__ *initialvalue* ⟦__，__ *initialvalue* .。。⟧__)__
