# 默认日期和时间
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.8.8** 转换的日期和时间不可用时应遵循的 \_DATE\_ 和 \_TIME\_ 的定义  
  
 当操作系统不提供转换的时间和日期时，\_DATE\_ 和 \_TIME\_ 的默认值为 `May 03 1957` 和 `17:00:00"`。
