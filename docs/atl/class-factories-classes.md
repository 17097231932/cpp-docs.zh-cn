# <a name="class-factories-classes"></a>类工厂类

以下类实现或支持类工厂：

- [CComClassFactory](../atl/reference/ccomclassfactory-class.md) 为对象创建提供默认类工厂。

- [CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) 通过许可证控制对象的创建。

- [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) 允许在多个线程共用单元中创建对象。

- [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) 创建单个对象。

- [CComCoClass](../atl/reference/ccomcoclass-class.md) 定义对象的类工厂。
