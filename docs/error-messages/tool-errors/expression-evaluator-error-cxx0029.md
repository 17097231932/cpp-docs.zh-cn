# <a name="expression-evaluator-error-cxx0029"></a>表达式计算器错误 CXX0029

not struct 指针

成员选择运算符 (**->**) 应用于不是指向结构的指针的表达式。

检查整个表达式是否已正确地加括号，或将地址表达式强制转换为相应的结构指针类型。

此错误与 CAN0029 相同。
