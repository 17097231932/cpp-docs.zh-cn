# <a name="microsoft-macro-assembler-reference"></a>Microsoft 宏汇编程序参考

Microsoft Macro Assembler (MASM) 提供了几大优势，内联程序集。 MASM 包含具有功能，例如循环、 算术、 和文本的字符串处理的宏语言。 MASM 还可以更好地控制硬件因为它支持的 386、 486 和 Pentium 处理器的指令集。 通过使用 MASM，您还可以减少时间和内存开销。
