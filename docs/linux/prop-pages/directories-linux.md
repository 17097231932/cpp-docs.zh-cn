# <a name="vc-directories-linux-c"></a>VC++ 目录 (Linux C++)

属性 | 描述
--- | ---
包含目录 | 构建 VC++ 项目期间，搜索包含文件时使用的路径。  与环境变量 INCLUDE 相对应。
库目录 | 构建 VC++ 项目期间，搜索库文件时使用的路径。  与环境变量 LIB 相对应。
源目录 | 搜索用于 IntelliSense 的源文件时使用的路径。
排除目录 | 搜索扫描依赖项时要跳过的路径。
