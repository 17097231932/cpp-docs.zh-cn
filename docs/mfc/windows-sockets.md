# <a name="windows-sockets"></a>Windows 套接字

本系列文章介绍了 Windows 套接字 MFC 实现。 MFC 提供两个类以支持使用 Windows 套接字 API 的编程网络应用程序。 类[CAsyncSocket](../mfc/reference/casyncsocket-class.md)封装一个 Windows 套接字 API，为高级的网络编程人员提供最强大功能和灵活性。 类[CSocket](../mfc/reference/csocket-class.md)来回序列化数据提供了简化的界面[CArchive](../mfc/reference/carchive-class.md)对象。
