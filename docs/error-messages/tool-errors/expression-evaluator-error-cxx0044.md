# <a name="expression-evaluator-error-cxx0044"></a>表达式计算器错误 CXX0044

使用 _based （void）指针需要： > 运算符

基于的指针 **`void`** 不能直接使用。 必须使用 **： >** 运算符建立完整的指针。

此错误与 CAN0044 相同。
