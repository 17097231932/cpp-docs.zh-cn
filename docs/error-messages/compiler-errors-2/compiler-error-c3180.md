# <a name="compiler-error-c3180"></a>编译器错误 C3180

"type name"：名称超出了 "限制" 字符的元数据限制

编译器截断了元数据中托管类型的名称。 截断将使类型无法使用 `#using` 指令 (或其他语言) 中的等效项。

类型名称限制包括任何命名空间限定。
