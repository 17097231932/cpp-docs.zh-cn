# <a name="compiler-error-c3697"></a>编译器错误 C3697

限定符： 不能使用此限定符 ^

跟踪句柄 (^) 应用于为其不针对的限定符。

下面的示例生成 C3697:

```
// C3697.cpp
// compile with: /clr
using namespace System;
int main() {
   String ^__restrict s;   // C3697
   String ^ s2;   // OK
}
```
