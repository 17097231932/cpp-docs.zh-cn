# <a name="compiler-warning-level-4-c4505"></a>编译器警告（等级 4）C4505

"function"：未引用的本地函数已移除

给定的函数是局部的，在模块的主体中未引用;因此，该函数是死代码。

编译器未生成此停滞函数的代码。
