# <a name="expression-evaluator-error-cxx0072"></a>表达式计算器错误 CXX0072

错误： 缺少类型信息或未知

未链接的.pch 文件，或者代码对不是用 /Zi 编译模块中的类型的引用。

此错误是与 CAN0067 相同。
