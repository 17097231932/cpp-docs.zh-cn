# <a name="fatal-error-c1046"></a>错误 C1046
编译器限制： 结构嵌套太深  
  
 结构、 联合或类超出嵌套限制，这是 15 个级别。 重写的定义，以减少嵌套级别。 结构、 联合或类通过拆分成两个或多个部分使用`typedef`定义一个或多个嵌套的结构。
