# <a name="stack"></a>.STACK

与一起使用时[。模型](../../assembler/masm/dot-model.md)，定义一个堆栈段 （使用段名称堆栈）。 可选`size`指定的堆栈 （默认值为 1024） 的字节数。 `.STACK`指令会自动关闭堆栈语句。

## <a name="syntax"></a>语法

> .堆栈 [[size]]
