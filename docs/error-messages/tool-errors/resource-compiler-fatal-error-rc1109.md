# <a name="resource-compiler-fatal-error-rc1109"></a>资源编译器错误 RC1109
创建资源名称时出错  
  
 无法创建指定的.res 文件。 请确保它不创建只读驱动器上。 /V 选项用于找出是否创建该文件。
