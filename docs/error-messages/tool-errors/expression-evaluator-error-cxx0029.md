# <a name="expression-evaluator-error-cxx0029"></a>表达式计算器错误 CXX0029

不是结构指针

成员选择运算符 (**->**) 已应用于不是指向一个结构的指针的表达式。

检查整个表达式是用圆括号括起来正确，或者类型强制转换为适当结构指针类型在地址表达式。

此错误是与 CAN0029 相同。
