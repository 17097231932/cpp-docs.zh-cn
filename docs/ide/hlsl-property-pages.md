# <a name="hlsl-property-pages"></a>“HLSL”属性页
HLSL 编译器 (fxc.exe) 属性页可用于配置文件生成的个别 HLSL 着色器。 属性分为三个类别：  
  
-   [HLSL 属性页：常规](../ide/hlsl-property-pages-general.md)  
  
     提供可指定常用 HLSL 编译器选项属性。  
  
-   [HLSL 属性页：高级](../ide/hlsl-property-pages-advanced.md)  
  
     提供可指定 HLSL 编译器的其他选项属性。  
  
-   [HLSL 属性页：输出文件](../ide/hlsl-property-pages-output-files.md)  
  
     提供的属性来指定如何通过 HLSL 编译器的输出文件。  
  
 你可以通过使用指定 HLSL 编译器命令行自变量**其他选项**属性**命令行**属性页; 这包括不能使用其他配置的自变量HLSL 属性页中的属性。 HLSL 编译器有关的信息，请参阅[效果编译器工具](http://go.microsoft.com/fwlink/p/?LinkID=258285&clcid=0x409)
