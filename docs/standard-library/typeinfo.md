# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;
包括标准标头 `<typeinfo>` 以定义与类型标识运算符 [typeid 运算符](../cpp/typeid-operator.md)（可生成静态和动态类型的相关信息）关联的几种类型。  
  
## <a name="syntax"></a>语法  
  
```  
#include <typeinfo>  
  
```
