# <a name="compiler-warning-level-1-c4821"></a>编译器警告（等级 1）C4821

> 无法确定 Unicode 编码类型，请用签名 (BOM) 保存文件

编译器无法确定文件的编码类型。 若要解决此警告，请使用字节顺序标记保存该文件。 请参阅[带有编码的管理文件](/sql/ssms/solution/manage-files-with-encoding)有关详细信息。
