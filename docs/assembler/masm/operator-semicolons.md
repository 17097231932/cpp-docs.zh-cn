# <a name="operator-"></a>运算符 ;;
将*文本*作为中的宏，仅在宏定义中出现的注释。 该列表不显示*文本*扩展宏。  
  
## <a name="syntax"></a>语法  
  
```  
  
;;  
text  
  
```
