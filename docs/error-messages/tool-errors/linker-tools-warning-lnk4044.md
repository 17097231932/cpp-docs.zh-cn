# 链接器工具警告 LNK4044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法识别的选项“option”；被忽略  
  
 给定选项不是此工具的有效选项。  工具忽略了此选项。  
  
 若要查看有效选项列表，请不带任何参数或选项运行工具。
