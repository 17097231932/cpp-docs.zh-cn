# <a name="linker-tools-warning-lnk4229"></a>链接器工具警告 LNK4229

找到无效的指令/directive;掉

通过 [注释](../../preprocessor/comment-c-cpp.md) 杂注传递的指令无效。 链接器将忽略 `/directive` 。
