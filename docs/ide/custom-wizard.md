# 自定义向导
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+“自定义向导”是一个特定类型的向导，旨在帮助您创建自己的向导。  配置了向导后，它立即生成项目所需的起始文件。  
  
## 概述  
 该向导页描述正创建的自定义向导项目的当前应用程序设置。  默认情况下，该向导按如下方式创建项目：  
  
-   将向导的友好名称设置为在**“新建项目”对话框**中指示的项目名称。  
  
-   向导具有用户界面。  
  
-   向导的用户界面只有一页。  
  
 可以在向导的[应用程序设置](../ide/application-settings-custom-wizard.md)页中更改这些默认设置。
