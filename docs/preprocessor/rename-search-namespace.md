# rename_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 专用**  
  
 与 [rename\_namespace](../preprocessor/rename-namespace.md) 特性的功能相同，但在类型库中使用时，可以将 \#import 指令与 [auto\_search](../preprocessor/auto-search.md) 特性结合使用。  
  
## 语法  
  
```  
rename_search_namespace("NewName")  
```  
  
#### 参数  
 `NewName`  
 命名空间的新名称。  
  
## 备注  
 **结束 C\+\+ 专用**
