# Registry Data Exchange Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些宏执行注册表数据交换的操作。  
  
|||  
|-|-|  
|[BEGIN\_RDX\_MAP](../Topic/BEGIN_RDX_MAP.md)|标记注册表数据交换的映射的开头。|  
|[END\_RDX\_MAP](../Topic/END_RDX_MAP.md)|标记注册表数据交换的映射结束。|  
|[RDX\_BINARY](../Topic/RDX_BINARY.md)|将指定的注册表项与类型字节的指定成员变量。|  
|[RDX\_CSTRING\_TEXT](../Topic/RDX_CSTRING_TEXT.md)|将指定的注册表项与类型CString的指定成员变量。|  
|[RDX\_DWORD](../Topic/RDX_DWORD.md)|将指定的注册表项与类型DWORD的指定成员变量。|  
|[RDX\_TEXT](../Topic/RDX_TEXT.md)|将指定的注册表项与类型TCHAR的指定成员变量。|
