# <a name="compiler-warning-level-1-c4379"></a>编译器警告（等级 1）C4379

此编译器不支持公共语言运行时版本。 使用此版本可能会导致意外的结果。

你的计算机上有以前版本的公共语言运行时，而不是当前版本。 若要解析 C4379，请安装编译器附带的公共语言运行时版本。
