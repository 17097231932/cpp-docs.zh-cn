# 关于异常的疑难解答：System.DirectoryServices.Protocols.DsmlInvalidDocumentException
当 DSML（目录服务标记语言）请求或响应文档不是格式良好的 XML，或者无法使用 DSMLV2 架构进行验证时引发的异常。
