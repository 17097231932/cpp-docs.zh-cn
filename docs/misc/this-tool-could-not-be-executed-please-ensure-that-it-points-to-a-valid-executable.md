# This tool could not be executed. Please ensure that it points to a valid executable.
该错误通常发生在尝试启动外部工具并且该工具运行失败时。  该错误可能是由指定的文件不是有效的可执行文件或批处理文件所致。  
  
### 更正此错误  
  
1.  利用可从**工具**菜单访问的**外部工具**对话框检查，确保该外部工具为有效的可执行文件或批处理文件。
