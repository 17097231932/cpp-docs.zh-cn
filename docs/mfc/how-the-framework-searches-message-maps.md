框架在消息映射表中搜索与传入消息匹配的消息。 为每条消息编写消息映射项后，需要类处理并写入相应的处理程序，框架会自动调用处理程序。 以下主题介绍消息映射搜索：

- [在哪里可以找到消息映射](where-to-find-message-maps.md)

- [派生的消息映射](derived-message-maps.md)

- [将消息范围、命令 Id 或控件 Id 映射到一个处理程序](handlers-for-message-map-ranges.md)
