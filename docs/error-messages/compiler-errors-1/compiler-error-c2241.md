# <a name="compiler-error-c2241"></a>编译器错误 C2241

“identifier”：成员访问受限制

代码试图访问私有或受保护成员。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用以下可能的解决方案进行修复

1. 更改成员的访问级别。

1. 声明 **`friend`** 访问函数的成员。
