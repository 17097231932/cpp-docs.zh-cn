# <a name="criticalsectioncriticalsection-destructor"></a>CriticalSection::~CriticalSection 析构函数
取消初始化和销毁当前**CriticalSection**对象。  
  
## <a name="syntax"></a>语法  
  
```  
WRL_NOTHROW ~CriticalSection();  
```  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
