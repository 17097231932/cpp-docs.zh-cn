# <a name="cusageprivileges-cusageprivilegeinfo"></a>CUsagePrivileges，CUsagePrivilegeInfo
调用 typedef 类**CUsagePrivileges**来实现其参数类**CUsagePrivilegeInfo**。  
  
## <a name="remarks"></a>备注  
 请参阅[架构行集类和 Typedef 类](../../data/oledb/schema-rowset-classes-and-typedef-classes.md)有关使用 typedef 类的详细信息。  
  
 此类将标识对在目录中定义的对象，适用于或给定用户授予的使用权限。  
  
 下表列出了类数据成员及其对应的 OLE DB 列。 请参阅[USAGE_PRIVILEGES 行集](https://msdn.microsoft.com/en-us/library/ms722743.aspx)中*OLE DB 程序员参考*有关的架构和列的详细信息。  
  
|数据成员|OLE DB 列|  
|------------------|--------------------|  
|m_szGrantor|GRANTOR|  
|m_szGrantee|GRANTEE|  
|m_szObjectCatalog|OBJECT_CATALOG|  
|m_szObjectSchema|OBJECT_SCHEMA|  
|m_szObjectName|OBJECT_NAME|  
|m_szObjectType|OBJECT_TYPE|  
|m_szPrivilegeType|PRIVILEGE_TYPE|  
|m_bIsGrantable|IS_GRANTABLE|  
  
## <a name="requirements"></a>要求  
 **标头：** atldbsch.h
