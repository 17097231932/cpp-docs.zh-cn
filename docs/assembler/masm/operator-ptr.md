# <a name="operator-ptr"></a>运算符 PTR

第一个运算符强制*表达式*被视为具有指定*类型*。 第二个运算符指定一个指向*类型*。

## <a name="syntax"></a>语法

> *类型*PTR*表达式*
>
> \[*距离*] PTR*类型*
