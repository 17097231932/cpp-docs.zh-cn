# 链接器工具错误 LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

映像大小\(“size”\)超过允许的最大大小 \(80000000\)  
  
 链接器确定了输出文件的大小将超过程序图像的最大可能大小。  可能需要将程序处理成多个 DLL。  有关程序可执行文件 \(PE\) 的更多信息，请参见 [MSDN Library](http://go.microsoft.com/fwlink/?linkid=556)中的文章“Peering Inside the PE: A Tour of the Win32 Portable Executable File Format”（PE 内部窥探：Win32 可移植可执行文件格式漫谈）。
