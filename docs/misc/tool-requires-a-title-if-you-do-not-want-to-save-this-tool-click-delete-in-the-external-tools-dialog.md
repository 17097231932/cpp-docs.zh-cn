# 工具需要标题。 如果不想保存此工具，请在“外部工具”对话框中单击“删除”。
当新工具已添加到“外部工具”对话框中但尚未指定标题时，通常会发生此错误。  
  
### 更正此错误  
  
1.  如果要保存该工具，请单击**确定**，然后在**外部工具**对话框中为该工具输入标题。  
  
2.  如果不希望保存该工具，请单击**确定**，然后在**外部工具**对话框中单击**删除**。
