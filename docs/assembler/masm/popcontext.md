# <a name="popcontext"></a>POPCONTEXT
还原部分或全部当前`context`(由保存[PUSHCONTEXT](../../assembler/masm/pushcontext.md)指令)。 `context`可以是**假设**， `RADIX`，**列出**， **CPU**，或**所有**。  
  
## <a name="syntax"></a>语法  
  
```  
  
POPCONTEXT context  
```
