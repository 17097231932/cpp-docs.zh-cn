# <a name="linker-tools-warning-lnk4039"></a>链接器工具警告 LNK4039

用/SECTION 选项指定的 "name" 部分不存在

[DUMPBIN](../../build/reference/dumpbin-reference.md) 或 [EDITBIN](../../build/reference/editbin-reference.md) 未能 `name` 在输入文件中找到名为的部分。

若要查看文件中的部分，请使用 [/HEADERS](../../build/reference/headers.md) 选项运行 DUMPBIN。
