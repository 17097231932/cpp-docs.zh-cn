# <a name="compiler-warning-level-4-c4639"></a>编译器警告（等级 4）C4639
MSXML 错误，将不会处理注释的 XML 文档。 原因  
  
 因多种原因会出现此警告。  
  
 若要解决此警告：  
  
-   重新编译。  
  
-   通过重新安装公共语言运行时重新安装 MSXML。  
  
-   编辑或删除该警告并重新编译导致的文档注释。  
  
 颁发 C4639 时，禁用所有更多的 XML 注释处理，并不会生成.xdc 文件。
