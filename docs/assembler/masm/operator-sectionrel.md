# <a name="operator-sectionrel"></a>运算符 SECTIONREL
返回相对于包含最终可执行文件中的目标的部分的表达式的部分相对偏移量。  
  
## <a name="syntax"></a>语法  
  
```  
SECTIONREL expression  
```  
  
## <a name="remarks"></a>备注  
 SECTIONREL 时仅使用 COFF 对象发出。
