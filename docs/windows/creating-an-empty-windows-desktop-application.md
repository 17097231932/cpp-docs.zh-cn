# 正在创建空的 Windows 桌面应用程序
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 创建空的 Windows 桌面应用程序  
  
1.  从**“文件”**菜单中选择**“新建项目”**。  
  
2.  在**“新建项目”**对话框的左窗格中，单击**“Win32”**，然后在中央窗格中选择**“Win32 控制台应用程序”**。  
  
3.  输入新项目的名称以及项目目录的路径，然后单击“确定”。  
  
4.  在 [Win32 应用程序向导](../windows/win32-application-wizard.md)中，单击“应用程序设置”页。 选择要使用源代码文件创建的“应用程序类型”，然后在“附加选项”下选中“空项目”复选框。  
  
5.  单击“确定”。  
  
 项目会出现在“解决方案资源管理器”中，具有三个目录，用于包含源文件、头文件和资源文件。  
  
 接下来，可以[将文件添加到空的 Visual C\+\+ 项目](../windows/adding-files-to-an-empty-win32-applications.md)。
