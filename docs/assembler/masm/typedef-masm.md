# <a name="typedef"></a>TYPEDEF

定义名为 " *name*" 的新类型或 **PROTO** ，它等效于 *type* 或 *protoDefinition*。

## <a name="syntax"></a>语法

> *名称* **TYPEDEF** *类型*  |  **协议** *protoDefinition*
