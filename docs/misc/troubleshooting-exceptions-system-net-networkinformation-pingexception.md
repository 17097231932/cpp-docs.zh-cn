# 关于异常的疑难解答：System.Net.NetworkInformation.PingException
当 <xref:System.Net.NetworkInformation.Ping.Send%2A> 或 <xref:System.Net.NetworkInformation.Ping.SendAsync%2A> 方法调用的方法引发异常时引发的异常。 例如，服务器可能不支持 ping，或者防火墙可能阻止了该操作。
