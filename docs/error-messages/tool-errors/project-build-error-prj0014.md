# <a name="project-build-error-prj0014"></a>项目生成错误 PRJ0014
失败的作业对象，用于控制生成的进程。  无法继续生成。  
  
 在开发环境中出错。  
  
 若要解决此错误，请关闭并重新加载项目。 如有必要，退出并重新启动 Visual Studio。
