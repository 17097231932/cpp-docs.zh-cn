# <a name="ltdequegt"></a>&lt;deque&gt;

定义容器类模板 deque 和多个支持模板。

## <a name="requirements"></a>要求

**标头**： \<deque>

> [!NOTE]
> \<deque>该库还使用该 `#include <initializer_list>` 语句。
