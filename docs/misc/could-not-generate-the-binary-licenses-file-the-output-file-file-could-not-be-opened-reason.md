# 无法生成二进制许可证文件。 无法打开输出文件“file”。 &lt;reason&gt;
项目系统无法删除已过时的许可证文件副本。 作为准备许可证的一部分，项目系统会将 .licx 文本文件转换为 .NET 授权系统可识别的二进制许可证文件。  
  
 **更正此错误**  
  
-   重新启动 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
     如果发生此错误，则生成过程将失败。
