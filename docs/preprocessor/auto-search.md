# <a name="auto_search-import-attribute"></a>auto_search 导入属性

**C++ 专用**

指定在引用类型库时， `#import` 它本身引用另一个类型库，编译器可以 `#import` 为其他类型库执行隐式转换。

## <a name="syntax"></a>语法

> **#import** *类型-库* **auto_search**

**结束 C++ 专用**
