# <a name="eventtargetarraylength-method"></a>EventTargetArray::Length 方法
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
size_t Length();  
```  
  
## <a name="return-value"></a>返回值  
 事件处理程序在内部数组的元素的当前数目。  
  
## <a name="remarks"></a>备注  
 获取事件处理程序在内部数组元素的当前数目。  
  
## <a name="requirements"></a>惠?  
 **标头：** event.h  
  
 **Namespace:** Microsoft::WRL::Details
