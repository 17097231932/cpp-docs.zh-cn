# <a name="expression-evaluator-error-cxx0006"></a>表达式计算器错误 CXX0006
缺少左的括号  
  
 在表达式中找到括号不对称。 重新键入带平衡括号表达式。  
  
 此错误是与 CAN0006 相同。
