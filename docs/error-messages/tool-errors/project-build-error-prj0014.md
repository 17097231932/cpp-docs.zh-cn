# 项目生成错误 PRJ0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

用于控制生成过程的作业对象已失败。生成无法继续进行。  
  
 开发环境中发生错误。  
  
 若要解决此错误，请关闭并重新加载项目。  如有必要，请退出并重新启动 Visual Studio。
