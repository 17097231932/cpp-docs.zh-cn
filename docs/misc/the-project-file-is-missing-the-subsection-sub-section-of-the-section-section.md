# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
子节 \(subsection\) 是 Include，而节 \(section\) 是 Files。  
  
 此错误很可能由手动编辑项目文件引起。  
  
 **更正此错误**  
  
-   以**“显示所有文件”**模式（通过在解决方案资源管理器工具栏中单击该按钮）工作并包括相关文件。  
  
     如果得到此错误，项目将打开，但是不向其中添加任何文件。
