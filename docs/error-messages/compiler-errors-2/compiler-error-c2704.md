# 编译器错误 C2704
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：\_\_va\_start 内部函数只能在 varargs 中  
  
 `__va_start` 内部函数用于具有固定数量的参数的函数声明中。
