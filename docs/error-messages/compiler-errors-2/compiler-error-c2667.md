# 编译器错误 C2667
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 所有的 number 重载都没有最好的转换  
  
 重载函数调用不明确，无法解析。  
  
 为匹配对一个重载函数的函数调用中的实际参数，所要求的转换必须确实比所有其他重载函数所要求的转换更好。  
  
 有关函数模板部分排序的更多信息，请参见知识库文章 Q240869。
