# <a name="c28351"></a>C28351

> 警告 C28351：批注介绍了在 \<annotation> 条件中无法使用动态值 (变量) 的情况。

当批注应用于所检查的 C/C++ 类型位置时，将报告此警告。

`_When_` 中的表达式的计算结果应为常数。 `_When_`忽略。
