# <a name="nmake-fatal-error-u1000"></a>NMAKE 错误 U1000
语法错误:) 宏调用中缺少  
  
 左的括号， **(**，却没有匹配的右括号， **)**，宏调用中。 正确的窗体是 **$(***名称***)**;`$` *n*所允许的一个字符的名称。
