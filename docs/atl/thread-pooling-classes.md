# <a name="thread-pooling-classes"></a>线程池类

以下类支持线程池：

- [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) 实现 EXE 模块，支持多线程池单元。

- [CComApartment](../atl/reference/ccomapartment-class.md) 在线程池 EXE 模块中管理单元。

- [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) 管理 EXE 模块的线程选择。
