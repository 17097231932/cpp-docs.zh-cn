# Add-in &lt;name&gt; caused an access violation. Remove it from the Add-in Manager?
该外接程序已发生故障并可能导致集成开发环境中出现其他问题。  
  
### 更正此错误  
  
1.  选择“是”从可用外接程序列表中移除该外接程序。  
  
     \- 或 \-  
  
     选择“否”。  
  
    > [!NOTE]
    >  请与外接程序的作者联系报告此问题。
