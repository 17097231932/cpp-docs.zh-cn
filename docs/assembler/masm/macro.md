# <a name="macro"></a>MACRO

将标记调用一个宏块*名称*，并建立*参数*时该宏调用传递参数的占位符。

## <a name="syntax"></a>语法

> *name* MACRO [[*parameter* [[:REQ | :=*default* | :VARARG]] ]] ...<br/>
> *statements*<br/>
> ENDM [[*值*]]

## <a name="remarks"></a>备注

宏函数将返回*值*给调用语句。
