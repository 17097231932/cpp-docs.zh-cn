# <a name="listmacro"></a>.LISTMACRO

启动生成的代码或数据的宏扩展语句的列表。

## <a name="syntax"></a>语法

> .LISTMACRO

## <a name="remarks"></a>备注

这是默认设置。 与相同[。XALL](../../assembler/masm/dot-xall.md)。
