# 编译器错误 C2018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未知字符“hexnumber”  
  
 源文件包含一个意外的 ASCII 字符，该字符由其十六进制数标识。  若要解决该错误，请移除该字符。
