# <a name="odbc-driver-list"></a>ODBC 驱动程序列表

VisualC++提供对以下数据库的 ODBC 驱动程序：

- SQL Server

- Microsoft Access

- Microsoft Excel

- dBASE

- Paradox

- Microsoft Oracle ODBC

- 文本文件

有关 ODBC 驱动程序可从 Microsoft 和其他公司，包括 ODBC 驱动程序包，请与 Microsoft 产品支持服务联系。
