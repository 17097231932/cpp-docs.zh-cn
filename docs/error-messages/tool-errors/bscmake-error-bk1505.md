# <a name="bscmake-error-bk1505"></a>BSCMAKE 错误 BK1505

无法从文件 "filename" [： reason] 读取

BSCMAKE 无法从该文件中读取。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复

1. 文件损坏。

1. 文件截断，导致编译器磁盘空间不足或在创建 .sbr 文件时被中断。
