# <a name="compiler-error-c2873"></a>编译器错误 C2873
symbol︰ 不能在使用声明中使用符号  
  
 一个`using`指令是缺少[命名空间](../../cpp/namespaces-cpp.md)关键字。 这将导致编译器错误地解释为代码[使用声明](../../cpp/using-declaration.md)而不是[using 指令](../../cpp/namespaces-cpp.md#using_directives)。
