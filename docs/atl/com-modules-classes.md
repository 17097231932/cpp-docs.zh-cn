# COM Modules Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类提供对COM模块支持:  
  
-   [CAtlBaseModule](../atl/reference/catlbasemodule-class.md) 此选件类在每个ATL项目实例化。  
  
-   [CAtlComModule](../atl/reference/catlcommodule-class.md) 此选件类实现一个COM服务器模块。  
  
-   [CAtlModule](../atl/reference/catlmodule-class.md) 此选件类提供了若干ATL模块选件类的方法。  
  
-   [CAtlModuleT](../atl/reference/catlmodulet-class.md) 此选件类实现一个ATL模块。  
  
-   [CAtlExeModuleT](../atl/reference/catlexemodulet-class.md) 此选件类表示应用程序的模块。  
  
-   [CAtlServiceModuleT](../atl/reference/catlservicemodulet-class.md) 此选件类实现一服务。  
  
-   此 选件类提供ATL多窗口组件支持的[CAtlWinModule](../atl/reference/catlwinmodule-class.md)。  
  
-   [CComModule](../atl/reference/ccommodule-class.md) 此选件类实现一个DLL或EXE模块。  过时在ATL 7.0。  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) 此选件类实现一个EXE模块，并对多个线程池的双工支持。  过时在ATL 7.0。
