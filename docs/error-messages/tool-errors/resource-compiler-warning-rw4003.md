# <a name="resource-compiler-warning-rw4003"></a>资源编译器警告 RW4003
无 VIRTKEY 的情况下使用 SHIFT 或 CONTROL  
  
 在快捷键对应表资源中，SHIFT 或 CONTROL 需要 VIRTKEY。 因为 SHIFT 键和 CONTROL 键在 VIRTKEY 类型快捷键中显示为标志位，其不能独立于 VIRTKEY 存在。
