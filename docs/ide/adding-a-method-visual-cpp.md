# 添加方法 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可以使用[添加方法向导](../ide/add-method-wizard.md)向项目中的接口添加方法。  如果项目包含与接口关联的类，则向导还将修改此类。  
  
### 向对象添加方法  
  
1.  在“类视图”中，展开项目节点以显示要向其中添加方法的接口。  
  
    > [!NOTE]
    >  也可以向调度接口中添加方法，调度接口位于库节点之下（除非项目已特性化）。  
  
2.  右击接口的名称。  
  
3.  在快捷菜单上单击“添加”，然后单击“添加方法”。  
  
4.  在“添加方法向导”中提供创建方法的信息。  
  
5.  在向导的 [IDL 特性](../ide/idl-attributes-add-method-wizard.md)页中，为此方法指定任何接口定义语言设置。  
  
6.  单击“完成”添加此方法。
