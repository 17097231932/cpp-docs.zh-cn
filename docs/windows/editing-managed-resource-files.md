# <a name="editing-managed-resource-files-c"></a>编辑托管资源文件 （c + +）

托管的资源 (.resx) 文件是 XML 文件。 当将托管的资源文件添加到你的项目从**添加新项**对话框中，**托管资源编辑器**默认情况下打开。
