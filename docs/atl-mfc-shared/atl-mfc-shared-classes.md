# ATL/MFC Shared Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

从Visual C\+\+ .NET 2002中启动，一些现有MFC实用工具选件类复盖或修改减少它们在其他MFC选件类的依赖项。  这些实用工具选件类现在可用于任何本机C\+\+项目。  本节包含仅是前面提供对MFC项目和现在共享的选件类，以及一些新选件类与 `CString`中的更改相关。  不包括ATL选件类，可用于任何本机C\+\+项目类型使用适当的标头的中。
