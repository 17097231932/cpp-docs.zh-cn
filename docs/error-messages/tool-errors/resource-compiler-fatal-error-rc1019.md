# <a name="resource-compiler-fatal-error-rc1019"></a>资源编译器错误 RC1019

意外的 "#else"

`#else`指令未出现在 `#if` 、 **#ifdef** 或 **#ifndef** 构造内。

在 `#if` 此语句之前，请确保有、 **#ifdef** 或 **#ifndef** 语句生效。
