# <a name="open-database-connectivity-odbc"></a>开放式数据库连接 (ODBC)

有关使用不带 MFC 的 ODBC API 的详细信息，请参阅 [开放式数据库连接 (ODBC) ](/sql/odbc/microsoft-open-database-connectivity-odbc) 文档。
