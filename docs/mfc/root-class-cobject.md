# <a name="root-class-cobject"></a>根类：CObject

Microsoft 基础类 (MFC) 库中的大多数类都是从类层次结构的根处的单个基类派生的。 `CObject` 为从其派生的所有类提供很多有用的功能，开销非常低。 有关及其功能的详细信息 `CObject` ，请参阅 [使用 CObject](../mfc/using-cobject.md)。

[CObject](../mfc/reference/cobject-class.md)<br/>
大多数 MFC 类的最终基类。 支持对数据进行序列化和获取有关类的运行时信息。

[CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)<br/>
用于在运行时确定对象的确切类的结构。
