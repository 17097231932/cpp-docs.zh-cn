# <a name="data-types-classes"></a>数据类型类

下面的类包装C++数据类型：

- [CComBSTR](../atl/reference/ccombstr-class.md)包装`BSTR`数据类型。

- [CComVariant](../atl/reference/ccomvariant-class.md)包装`VARIANT`数据类型。

- [CComCurrency](../atl/reference/ccomcurrency-class.md)包含用于创建和管理的方法和运算符`CURRENCY`对象。
