# <a name="sword"></a>SWORD

分配每个 *初始值设定项*，并选择性地为每个初始值 (2 个字节的存储) 。 在类型合法的任何位置，还可以用作类型说明符。

## <a name="syntax"></a>语法

> *名称***剑***初始值设定项*⟦__，__ *初始值设定项*.。。⟧
