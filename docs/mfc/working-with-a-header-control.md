# <a name="working-with-a-header-control"></a>使用标题控件

若要使用的标头控件的简单方法 ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)) 是与列表控件结合使用; 请参阅[使用 CListCtrl](../mfc/using-clistctrl.md)本系列主题中更高版本。 您可以单独使用标头控件。 MFC 调用`InitCommonControls`为您。 关键任务如下所示：

- [创建标题控件](../mfc/creating-the-header-control.md)

- [将项添加到标头控件](../mfc/adding-items-to-the-header-control.md)

- [对标题控件中的项排序](../mfc/ordering-items-in-the-header-control.md)

- [处理标题控件通知](../mfc/processing-header-control-notifications.md)

如果标头控件对象嵌入父视图或对话框类中，在销毁父级时销毁控件。
