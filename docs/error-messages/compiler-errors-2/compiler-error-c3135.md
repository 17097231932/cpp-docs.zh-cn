# 编译器错误 C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“property”: 属性不能有“const”或“volatile”类型  
  
 不允许在属性上使用 [const](../../cpp/const-cpp.md) 和 [volatile](../../cpp/volatile-cpp.md) 关键字。
