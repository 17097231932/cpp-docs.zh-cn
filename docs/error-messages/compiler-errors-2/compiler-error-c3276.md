# <a name="compiler-error-c3276"></a>编译器错误 C3276
keyword: __finally/finally 块中的跳转出现不可确定的行为在终止处理过程  
  
 此错误是相同[C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md)警告。 但是，当使用 /clr 时，此条件不能禁用与[警告](../../preprocessor/warning.md)杂注。
