# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

启用所有命令程序集 \(包括特权\)。 80386 个处理器的;禁用命令程序集引入的最新处理器。  
  
## 语法  
  
```  
  
.386P  
  
```  
  
## 备注  
 并启用 80387 的命令。
