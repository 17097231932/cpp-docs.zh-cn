# 编译器错误 C2077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非标量字段初始值设定项“identifier”  
  
 尝试用非标量（结构、联合、数组或类）初始化位域。  使用整数值或浮点值。
