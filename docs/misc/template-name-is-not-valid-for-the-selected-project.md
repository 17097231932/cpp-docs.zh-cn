# Template &lt;name&gt; is not valid for the selected project.
该错误通常发生在向**新建文件**命令 `File.NewFile` 或**添加新解决方案项**命令 `File.AddNewSolutionItem` 输入了未知模板名时。  
  
### 更正此错误  
  
1.  重新输入命令及其参数。  
  
     \- 或 \-  
  
2.  检查“新文件命令”或“添加新解决方案项命令”主题中的正确语法。
