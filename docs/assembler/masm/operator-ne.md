# <a name="operator-ne"></a>运算符 NE

如果 *表达式* 1 不等于 *表达式* 2，则返回 true (-1) ; 否则，返回 false (0) 。

## <a name="syntax"></a>语法

> *表达式* 2 **NE**  2
