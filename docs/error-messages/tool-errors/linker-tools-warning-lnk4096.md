# <a name="linker-tools-warning-lnk4096"></a>链接器工具警告 LNK4096

/BASE 值 "number" 对于 Windows 95 和 Windows 98 无效;映像可能无法运行

指定的基址无效。 Windows 95 和 Windows 98 可执行文件的基址必须大于0x400000 处。 有关基址的详细信息，请参阅[/base](../../build/reference/base-base-address.md)链接器选项。
