# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

`CMFCTabCtrl`类提供选项卡控件的功能。 选项卡控件在其顶部或底部显示具有平面或三维选项卡的可停靠窗口。 选项卡可以显示文本和图像，并可在处于活动状态时更改颜色。

## <a name="syntax"></a>语法

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>备注

`CMFCTabCtrl`类支持:

- 使用共享水平滚动条的选项卡控件样式, 包括3D、平面和平面。

- 位于窗口顶部或底部的选项卡。

- 显示文本、图像、文本和图像的选项卡。

- 选项卡处于活动状态时更改颜色的选项卡。

- 可调选项卡的边框大小改变。

- 可分离的选项卡式窗口。

`CMFCTabCtrl`类可与对话框一起使用, 但适用于使用停靠控件条 (如 Microsoft Excel 和 Visual Studio) 的应用程序。 有关详细信息, 请参阅[CDockablePane 类](../../mfc/reference/cdockablepane-class.md)。

按照以下步骤在你的应用程序中添加可调整大小的停靠选项卡控件:

1. 创建[CTabbedPane 类](../../mfc/reference/ctabbedpane-class.md)的实例。

1. 调用[CDockablePane:: Create](../../mfc/reference/cdockablepane-class.md#create)。

1. 使用[CBaseTabbedPane:: AddTab](../../mfc/reference/cbasetabbedpane-class.md#addtab)或[CMFCBaseTabCtrl:: InsertTab](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab)添加新选项卡。

1. 调用[CBasePane:: EnableDocking](../../mfc/reference/cbasepane-class.md#enabledocking) , 使当前停靠选项卡控件可以停靠在主框架窗口中。

1. 调用[CFrameWndEx::D ockpane](../../mfc/reference/cframewndex-class.md#dockpane)将选项卡式窗口停靠在主框架。

有关如何创建选项卡式窗口作为停靠控件条的示例, 请参阅[CTabbedPane 类](../../mfc/reference/ctabbedpane-class.md)。 若要`CMFCTabCtrl`用作非停靠控件, 请创建一个对象, 然后调用[CMFCTabCtrl:: create。](#create) `CMFCTabCtrl`

##  <a name="create"></a>CMFCTabCtrl:: Create

创建选项卡控件, 并将其附加`CMFCTabCtrl`到对象。

```
BOOL Create(
    Style style,
    const RECT& rect,
    CWnd* pParentWnd,
    UINT nID,
    Location location=LOCATION_BOTTOM,
    BOOL bCloseBtn=FALSE);
```

### <a name="parameters"></a>参数

*style*<br/>
中选项卡控件的样式。 有关更多信息，请参见“备注”。

*rect*<br/>
中限定选项卡控件的矩形。

*pParentWnd*<br/>
中指向父窗口的指针。 不得为 NULL。

*nID*<br/>
中选项卡控件的 ID。

*location*<br/>
中选项卡的位置。 默认值为 LOCATION_BOTTOM。 有关更多信息，请参见“备注”。

*bCloseBtn*<br/>
中若要在选项卡上显示 "关闭" 按钮, 则为 TRUE;否则为 FALSE。 默认值是 FALSE。

### <a name="return-value"></a>返回值

如果成功, 则为 TRUE;否则为 FALSE。

|STYLE_FLAT|使用平面选项卡创建选项卡控件。|
|STYLE_FLAT_SHARED_HORZ_SCROLL|使用平面选项卡和滚动条创建选项卡控件 (如果这些选项卡被父窗口剪裁)。|
|STYLE_3D_ONENOTE|创建 Microsoft OneNote 样式的选项卡控件。|
|STYLE_3D_VS2005|创建 Microsoft Visual Studio 2005 样式的选项卡控件。|
|STYLE_3D_ROUNDED|使用 Microsoft Visual Studio 2005 样式的舍入选项卡创建选项卡控件。|
|STYLE_3D_ROUNDED_SCROLL|使用 Microsoft Visual Studio 2005 样式的圆角选项卡和滚动按钮创建选项卡控件。|

下表列出了可为*location*参数指定的值。

##  <a name="setimagelist"></a>CMFCTabCtrl:: SetImageList

指定图像列表。

```
virtual BOOL SetImageList(
    UINT uiID,
    int cx=15,
    COLORREF clrTransp=RGB(255, 0, 255));

virtual BOOL SetImageList(HIMAGELIST hImageList);
```

### <a name="parameters"></a>参数

*uiID*<br/>
中包含图像列表的位图资源的 ID。

*cx*<br/>
中每个图像的宽度 (以像素为单位)。 默认值为 15。

*clrTransp*<br/>
中透明图像颜色。 此颜色的图像部分将是透明的。 默认值为洋红色、RGB (255、0255)。

*hImageList*<br/>
中预加载的图像列表的句柄。

### <a name="return-value"></a>返回值

如果此方法成功, 则为 TRUE。 如果选项卡控件是使用平面样式创建的, 或者如果第一种方法重载无法加载*uiID*参数指定的位图, 则为 FALSE。

### <a name="remarks"></a>备注

使用此方法可设置选项卡控件的图像列表。 图像列表中的图像显示在选项卡标签旁边。 此方法重新计算 tab 键的高度, 以便调整选项卡的大小以同时包含图像和文本。

使用选项卡控件继承的[CMFCBaseTabCtrl:: AddTab](../../mfc/reference/cmfcbasetabctrl-class.md#addtab)方法来指定要显示的图像的索引。

##  <a name="setresizemode"></a>CMFCTabCtrl:: SetResizeMode

指定如何调整当前选项卡控件的大小并重新计算控件。

```
void SetResizeMode(ResizeMode resizeMode);
```

### <a name="parameters"></a>参数

*resizeMode*<br/>
中`CMFCTabCtrl::ResizeMode`枚举值之一, 指定如何调整选项卡控件的大小。 有关可能值的列表, 请参阅 "备注" 中的表。

### <a name="remarks"></a>备注

*System.windows.resizemode>* 参数可以是以下`ResizeMode`枚举值之一。

|name|描述|
|----------|-----------------|
|RESIZE_NO|无法调整选项卡控件的大小。|
|RESIZE_VERT|选项卡控件可以垂直调整大小, 但不能水平调整大小。|
|RESIZE_HORIZ|选项卡控件可以水平调整大小, 但不能垂直调整大小。|

##  <a name="synchronizescrollbar"></a>CMFCTabCtrl:: SynchronizeScrollBar

### <a name="parameters"></a>参数

### <a name="remarks"></a>备注
