# <a name="operator-mask"></a>运算符 MASK
返回的位掩码顺序中的位*recordfieldname*或*记录*设置并清除所有其他位。  
  
## <a name="syntax"></a>语法  
  
```  
  
MASK {recordfieldname | record}  
```
