# 错误 C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在“location of call”API 调用“function name”失败 :“text from run\-time”  
  
 .NET 函数调用失败。  COM 运行时可能提供也可能不会提供 ***text from run\-time***。  
  
 有关系统错误信息的更多信息，请参见 winerror.h 系统文件和 [FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351)。
