# <a name="omp"></a>omp
将一个或多个 OpenMP 指令与任何可选指令子句一起使用。

## <a name="syntax"></a>语法

```
#pragma omp directive
```

## <a name="remarks"></a>备注

请参阅[OpenMP 指令](../parallel/openmp/reference/openmp-directives.md)有关详细信息。
