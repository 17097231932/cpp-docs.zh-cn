# <a name="expression-evaluator-error-cxx0018"></a>表达式计算器错误 CXX0018
错误的寄存器名  
  
 指定的注册不存在或无法显示。  
  
 监视窗口可以显示的以下寄存器：  
  
 AX SP DS IP  
  
 BX BP ES FL  
  
 CX SI SS GS  
  
 DX DI CS SS  
  
 EAX ESP DS EIP  
  
 EBX EBP ES 推崇  
  
 ECX ESI FS  
  
 EDX EDI CS  
  
 此错误是与 CAN0018 相同。
