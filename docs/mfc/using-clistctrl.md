# 使用 CListCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用列表控件显示带标签图标的安排，正如在带或没带图标的文件浏览器，或文本栏列表。  有关四个可能的“视图的”描述 \(不要与MFC 视图混淆\)，可以在具有列表控件 \- 视图图标、小图标视图、列表视图和报告视图 \- 请参阅在 [CListCtrl](../mfc/reference/clistctrl-class.md) 类概述的视图。  
  
 在一些视图中，用户可以拖动图标到不同的位置或版本图标标签。  例如，请参阅在文件资源管理器的右侧窗格中，nondialog 窗口使用的列表控件。  可以尝试在资源视图的菜单中的可用视图。  
  
 有关相关信息，请参阅 [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)]的 [有关列表视图控件](http://msdn.microsoft.com/library/windows/desktop/bb774735)。  
  
> [!NOTE]
>  [!INCLUDE[winSDK](../atl/includes/winsdk_md.md)] 引用列表控件作为“列表视图控件”。“视图”此用法不引用 MFC 视图类，特别地 `CListView`。  有关详细信息，请参阅 [列表控件以及列表视图](../mfc/list-control-and-list-view.md)。
