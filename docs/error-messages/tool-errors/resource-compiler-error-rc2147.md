# <a name="resource-compiler-error-rc2147"></a>资源编译器错误 RC2147

子语言 ID 不是数字

子语言 ID 值必须是数字。

**LANGUAGE** 语句必须使用以下语法:

**LANGUAGE** *primary_language_ID*、*secondary_language_ID*

有效的子语言 Id 在 WINNT .h 文件中定义为 **SUBLANG_** 常量。
