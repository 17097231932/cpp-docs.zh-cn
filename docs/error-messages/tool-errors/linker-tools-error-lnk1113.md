# 链接器工具错误 LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的计算机类型 type  
  
 在对象头中指定的计算机类型无效。  
  
 如果文件已损坏，则可能导致此错误。  重新生成对象。
