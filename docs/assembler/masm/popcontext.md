# <a name="popcontext"></a>POPCONTEXT

还原当前*上下文*的部分或全部（由[PUSHCONTEXT](../../assembler/masm/pushcontext.md)指令保存）。 *上下文*可以是**假设**、**基数**、**列出**、 **CPU**或**全部**。

## <a name="syntax"></a>语法

> **POPCONTEXT** *上下文*
