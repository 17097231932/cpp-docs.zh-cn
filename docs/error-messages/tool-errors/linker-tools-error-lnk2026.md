# <a name="linker-tools-error-lnk2026"></a>链接器工具错误 LNK2026

SAFESEH 映像的模块不安全

已指定[/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) ，但模块与安全异常处理功能不兼容。 如果希望将此模块用于 **/SAFESEH**，则需要重新编译该模块。
