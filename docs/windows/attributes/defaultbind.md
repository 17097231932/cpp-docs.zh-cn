# <a name="defaultbind"></a>defaultbind

指示最能代表对象的单个、 可绑定属性。

## <a name="syntax"></a>语法

```cpp
[defaultbind]
```

## <a name="remarks"></a>备注

**Defaultbind** c + + 属性具有相同的功能[defaultbind](/windows/desktop/Midl/defaultbind) MIDL 特性。
