# <a name="resource-compiler-error-rc2109"></a>资源编译器错误 RC2109

应为数值对话框常量

**对话** 语句要求 " *x"、"y"、"宽度*" 和 "*高度*" 字段的整数值。 请确保在 **DIALOG** 关键字后面包含这些值，并且这些值不是负数。
