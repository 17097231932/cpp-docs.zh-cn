# <a name="command-line-warning-d9040"></a>命令行警告 D9040

忽略选项 "/analyze";代码分析警告在此版本的编译器中不可用

在所有版本的 Visual Studio 中都不提供 " **/analyze**命令行" 选项。 若要解决此警告，请切换到受支持版本的 Visual Studio 或移除该命令行选项。
