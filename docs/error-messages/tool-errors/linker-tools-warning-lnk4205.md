# <a name="linker-tools-warning-lnk4205"></a>链接器工具警告 LNK4205

filename 缺少引用模块中; 当前调试信息正在链接对象，如同没有调试信息一样

.Pdb 文件已过时的信息。 链接器将继续链接没有调试信息的对象。 你可能想要重新编译对象文件使用[/Zi](../../build/reference/z7-zi-zi-debug-information-format.md)选项。
