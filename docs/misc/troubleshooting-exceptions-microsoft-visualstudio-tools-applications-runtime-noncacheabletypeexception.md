# 关于异常的疑难解答：Microsoft.VisualStudio.Tools.Applications.Runtime.NonCacheableTypeException
当尝试缓存某种类型，而该类型不是 <xref:System.Data.DataSet>、<xref:System.Data.DataTable>，也不是满足 [Caching Data](http://msdn.microsoft.com/zh-cn/e60dc731-2cb0-4de6-a861-0f0d131d780a) 中指定的要求的类型时，引发的异常。
