# <a name="project-build-error-prj0015"></a>项目生成错误 PRJ0015
从你的系统缺少 NULL 设备。 我们都无法启动生成。  
  
 可以通过系统资源不足导致此错误。 关闭一些应用程序或重新启动。  
  
 如果你的计算机上没有足够的特权的 NULL 设备，还可能会导致该错误。
