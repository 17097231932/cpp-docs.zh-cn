# MSBuild 错误 MSB3484
**MSB3484: 未能找到签名目标“\<文件\>”。**  
  
 此错误在未找到尝试进行签名的文件时产生。  
  
### 更正此错误  
  
-   确保为签名指定了有效的文件。
