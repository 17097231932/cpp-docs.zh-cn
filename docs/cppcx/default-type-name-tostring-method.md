# <a name="defaulttypenametostring-method"></a>default::(type_name)::ToString 方法
返回表示当前类型的字符串。  
  
## <a name="syntax"></a>语法  
  
```cpp  
  
String^ ToString()  
```  
  
### <a name="return-value"></a>返回值  
 表示当前对象的字符串。  
  
### <a name="requirements"></a>要求  
 **支持的最低客户端：** Windows 8  
  
 **支持的最低服务器：** Windows Server 2012  
  
 **命名空间：** 默认  
  
 **标头：** vccorlib.h
