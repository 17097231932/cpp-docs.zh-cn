# <a name="compiler-error-c2667"></a>编译器错误 C2667

function： 没有任何数字重载具有最佳转化

重载的函数调用不明确，无法解析。

匹配对其中一个重载函数的函数调用中的实际参数所需的转换必须是严格优于所需的所有其他重载函数的转换。
