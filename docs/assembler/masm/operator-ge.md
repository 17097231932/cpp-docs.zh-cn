# <a name="operator-ge"></a>运算符 GE
如果返回 true (-1) *expression1*大于或等于*expression2*，或如果它不返回 false (0)。  
  
## <a name="syntax"></a>语法  
  
```  
  
expression1 GE expression2  
```
