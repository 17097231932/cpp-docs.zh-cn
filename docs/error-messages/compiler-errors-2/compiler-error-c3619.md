# <a name="compiler-error-c3619"></a>编译器错误 C3619
不能在托管或 WinRT 类型中声明模板  
  
不允许在托管或 WinRT 类或接口使用类模板。  
  
C3619 才可访问使用过时的编译器选项**/clr:oldSyntax**。
