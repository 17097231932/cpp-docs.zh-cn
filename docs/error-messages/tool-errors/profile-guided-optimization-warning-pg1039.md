# 按配置优化警告 PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未能等到空闲时间以进行清扫。  
  
 应用程序不允许 [pgosweep](../../build/reference/pgosweep.md) 运行。如果可能，在应用程序处于空闲状态时运行 pgosweep。
