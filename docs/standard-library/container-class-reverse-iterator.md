# Container Class::reverse_iterator
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  本主题介绍 Visual C\+\+ 文档作为使用标准 C\+\+ 库的容器的非运行的示例。  有关更多信息，请参见 [STL 容器](../standard-library/stl-containers.md)。  
  
 描述可用作一进行反向迭代数。为控制序列的对象。  
  
## 语法  
  
```  
  
typedef T7 reverse_iterator;  
  
```  
  
## 备注  
 它介绍为此处未指定类型的 **T7** \(通常为 `reverse_iterator` **\<**\)[迭代器](../standard-library/container-class-iterator.md)**\>**的同义词。
