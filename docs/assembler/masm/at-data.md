# <a name="data"></a>\@数据

默认数据组的名称。 对于所有模型（**平面**除外），计算结果为 DGROUP。 在**平面**内存模型（文本宏）下计算为**平面**。

## <a name="syntax"></a>语法

> **\@数据**
