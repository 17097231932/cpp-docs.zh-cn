# Tab name &lt;name&gt; already exists.
“工具箱”中的每个选项卡必须有唯一的名称。  该错误通常发生在输入的“工具箱”选项卡名称和“工具箱”中已有的选项卡同名时。  
  
### 更正此错误  
  
1.  为选项卡输入唯一的名称。
