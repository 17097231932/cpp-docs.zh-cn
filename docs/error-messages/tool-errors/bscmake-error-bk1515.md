# <a name="bscmake-error-bk1515"></a>BSCMAKE 错误 BK1515

bscfile： 版本不兼容： 不能以增量方式更新

创建.bsc 文件时使用 BSCMAKE 的另一个版本。 重新创建.bsc 文件。
