# <a name="expression-evaluator-error-cxx0016"></a>表达式计算器错误 CXX0016

常量太大

C 表达式计算器无法接受无符号的整数常量大小超过 4294967295 (0FFFFFFFF) 或其位数大于大约 1.8 e + 308 的浮点常数。

此错误是与 CAN0016 相同。
