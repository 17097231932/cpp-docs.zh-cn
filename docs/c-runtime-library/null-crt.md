# <a name="null-crt"></a>NULL (CRT)

NULL 是与许多指针操作和函数一起使用的 null 指针值。 它等效于 0。 NULL 在下列头文件中进行定义：CRTDBG.H、LOCALE.H、STDDEF.H、STDIO.H、STDLIB.H、STRING.H、TCHAR.H、TIME.H 和 WCHAR.H。
