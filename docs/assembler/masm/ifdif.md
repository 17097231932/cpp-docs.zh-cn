# <a name="ifdif-ifdifi"></a>IFDIF、IFDIFI

如果文本项不同，则授予程序集。

## <a name="syntax"></a>语法

> **IFDIF** *textitem1*__，__ *textitem2*\
> **IFDIFI** *textitem1*__，__ *textitem2*

## <a name="remarks"></a>备注

如果给定了 **IFDIFI** ，则比较不区分大小写。 [若](if-masm.md)要获取完整的语法，请参阅。
