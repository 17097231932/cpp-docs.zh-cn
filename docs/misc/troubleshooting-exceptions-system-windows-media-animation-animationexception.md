# 关于异常的疑难解答：System.Windows.Media.Animation.AnimationException
在对属性进行动画处理的过程中发生错误时，将引发 <xref:System.Windows.Media.Animation.AnimationException> 异常。  
  
## 备注  
 此类不用于与计时相关的错误，例如，用负的 <xref:System.Windows.Media.Animation.Timeline.Duration%2A> 或无效的 <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> 和 <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> 设置来创建动画。
