# 无法为结构“&lt;structurename&gt;”编制索引，因为它没有默认属性
试图对没有默认属性的结构使用默认属性语法。  
  
 **错误 ID：**BC30690  
  
### 更正此错误  
  
-   使用标准属性语法访问该结构的属性。
