# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

托管资源文件 \(.resx\) 是 XML 文件。  从**“添加新项”**对话框中向项目添加托管资源文件时，默认打开**“托管资源编辑器”**。  
  
 有关更多信息，请参见 [.Resx 文件格式中的资源](http://msdn.microsoft.com/zh-cn/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5)。
