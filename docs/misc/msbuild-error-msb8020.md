# MSBuild 错误 MSB8020
找不到 *\<platform\>*（平台工具集 \=“*\<version\>*”）的生成工具。 若要使用 *\<version\>* 生成工具进行生成，请安装 *\<platform\>* 生成工具。  或者，可以通过选择“项目”菜单或右键单击解决方案升级到当前 Visual Studio 工具，然后选择“重定解决方案目标”。
