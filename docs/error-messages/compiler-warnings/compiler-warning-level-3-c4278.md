# <a name="compiler-warning-level-3-c4278"></a>编译器警告（等级 3）C4278
identifier： 类型库 tlb 中的标识符已宏;使用重命名限定符  
  
 使用时[#import](../../preprocessor/hash-import-directive-cpp.md)，您要导入类型库中的标识符尝试声明标识符***标识符***。 但是，这已是有效的符号。  
  
 使用`#import`**重命名**属性要分配给对类型库中的符号的别名。
