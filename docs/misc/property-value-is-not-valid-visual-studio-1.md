# Property value is not valid. (Visual Studio) (1)
该错误通常发生在向属性输入了无效值时。  例如，如果属性为布尔型属性，其有效值为 TRUE 和 FALSE，值 MAYBE 对该属性是无效的。  
  
### 更正此错误  
  
1.  输入该属性的有效值。  
  
    > [!TIP]
    >  在联机文档中搜索该属性名以查找有效值。
