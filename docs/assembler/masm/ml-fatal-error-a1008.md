# <a name="ml-fatal-error-a1008"></a>ML 错误 A1008

**不匹配的宏嵌套**

宏在文件末尾之前未终止，或在宏块外发现了终止指令 [ENDM](endm.md) 。

此错误的一个原因就是在之前省略点 [。重复](dot-repeat.md) 或 [。WHILE](dot-while.md)。
