# <a name="real4"></a>REAL4

分配并选择性地为每个*初始值设定项*初始化单精度（4字节）浮点数。

## <a name="syntax"></a>语法

> *名称*  **REAL4** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
