# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
`Boolean` 属性的持久值不是以下值之一：  
  
-   1 或 0  
  
-   On 或 Off  
  
-   true 或 false  
  
 此错误很可能由手动编辑项目文件引起。  
  
 **更正此错误**  
  
-   再次编辑项目文件，将其恢复为所满意的条件。  
  
     受影响的属性的值将被设置为 false。
