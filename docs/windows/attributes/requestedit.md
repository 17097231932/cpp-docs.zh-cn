# <a name="requestedit"></a>requestedit

指示该属性支持 `OnRequestEdit` 通知。

## <a name="syntax"></a>语法

```cpp
[requestedit]
```

## <a name="remarks"></a>备注

**Requestedit** c + + 特性具有与 [requestedit](/windows/win32/Midl/requestedit) MIDL 特性相同的功能。
