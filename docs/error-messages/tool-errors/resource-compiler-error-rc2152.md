# 资源编译器错误 RC2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的控制字符  
  
 在 **ACCELERATORS** 语句中的控制字符无效。  一个有效的控制字符由插入符号 \(**^**\) 后跟一个字母（并且只有一个）组成。
