# reverse (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

撤消的元素顺序范围内的。  
  
## 语法  
  
```  
template<class _BidIt> inline  
    void reverse(_BidIt _First, _BidIt _Last);  
```  
  
## 备注  
 此函数行为与 STL `reverse`函数相同。  有关详细信息，请参阅[reverse](../Topic/reverse.md)。  
  
## 要求  
 **页眉：** \<\/cliext 算法\>  
  
 **命名空间：** cliext
