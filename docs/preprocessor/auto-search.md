# <a name="autosearch"></a>auto_search
**C + + 专用**  
  
指定，使用引用类型库时`#import`和本身引用了另一个类型库，则编译器可以隐式`#import`其他类型库。  
  
## <a name="syntax"></a>语法  
  
```  
auto_search  
```  
  
## <a name="remarks"></a>备注  
 **结束 c + + 专用**
