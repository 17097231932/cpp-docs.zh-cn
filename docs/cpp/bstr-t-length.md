# _bstr_t::length
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 专用**  
  
 返回 `_bstr_t` 中的字符数，但不包括封装的 `BSTR` 的终止 null 字符。  
  
## 语法  
  
```  
  
unsigned int length ( ) const throw( );  
  
```
