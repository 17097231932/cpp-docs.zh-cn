# Command &lt;name&gt; does not accept arguments or switches.
该错误通常发生在随不使用参数或开关的命令或别名输入了开关或参数时。  如果输入了别名，在错误信息中将显示完整命令名而不是别名。  有关使用开关和参数的有效命令的列表，请参见 [Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。  
  
### 更正此错误  
  
1.  重新输入命令和参数或输入一个新命令。
