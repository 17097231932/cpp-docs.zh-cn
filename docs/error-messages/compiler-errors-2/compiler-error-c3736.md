# <a name="compiler-error-c3736"></a>编译器错误 C3736

event： 必须是一种方法，或在托管事件中可以选择的数据成员

本机和 COM 事件必须是方法。 .NET 事件也可以是数据成员。

下面的示例生成 C3736:

```
// C3736.cpp
struct A {
   __event int e();
};

struct B {
   int f;   // C3736
   // The following line resolves the error.
   // int f();
   B(A* a) {
      __hook(&A::e, a, &B::f);
   }
};

int main() {
}
```
