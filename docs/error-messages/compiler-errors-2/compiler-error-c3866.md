# <a name="compiler-error-c3866"></a>编译器错误 C3866

函数调用缺少参数列表

如果在非静态成员函数的析构函数或终结器调用没有参数列表。

下面的示例生成 C3866:

```
// C3866.cpp
// compile with: /c
class C {
   ~C();
   void f() {
      this->~C;   // C3866
      this->~C();   // OK
   }
};
```
