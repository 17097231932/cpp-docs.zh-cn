# <a name="codesize"></a>@CodeSize

0 表示**微小**，**小型**， **COMPACT**，并**平面**模型，而 1**中等**， **大型**，并**非常巨大**模型 （数字等同）。

## <a name="syntax"></a>语法

> @CodeSize
