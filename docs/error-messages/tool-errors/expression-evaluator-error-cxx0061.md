# <a name="expression-evaluator-error-cxx0061"></a>表达式计算器错误 CXX0061
函数的嵌套深度超过  
  
 该表达式包含嵌套深度超过限制的函数。 修改表达式以减少嵌套深度。  
  
 此错误是与 CAN0061 相同。
