# <a name="expression-evaluator-error-cxx0010"></a>表达式计算器错误 CXX0010
缺少左的括号  
  
 该表达式包含不均衡的方括号。 重新键入该表达式与平衡方括号。  
  
 此错误是与 CAN0010 相同。
