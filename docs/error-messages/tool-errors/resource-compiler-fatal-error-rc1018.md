# <a name="resource-compiler-fatal-error-rc1018"></a>资源编译器错误 RC1018

意外的 #elif

`#elif`中不显示指令`#if`， **#ifdef**，或 **#ifndef**构造。

请确保是否有`#if`， **#ifdef**，或 **#ifndef**实际上此语句前的语句。
