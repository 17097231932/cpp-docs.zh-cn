# <a name="compiler-error-c2692"></a>编译器错误 C2692
function_name: C 编译器中所需的完全原型函数 ' / clr 选项  
  
 编译的.NET 托管代码时，C 编译器需要 ANSI 函数声明。 此外，如果函数不采用任何参数，它必须显式声明`void`作为参数类型。
