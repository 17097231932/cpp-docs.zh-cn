# <a name="compiler-warning-level-1-c4024"></a>编译器警告（等级1） C4024

"function"：形参 "number" 的类型不同

对应的形参和实参具有不同的类型。 编译器无需更改即可传递实际参数。 接收函数将参数类型转换为所需的类型。
