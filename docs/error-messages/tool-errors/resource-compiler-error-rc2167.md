# <a name="resource-compiler-error-rc2167"></a>资源编译器错误 RC2167

无法识别的 VERSIONINFO 字段;BEGIN 或应为逗号

无法识别的字段中找到**FIXED**的一部分**VERSIONINFO**结构声明。 一个**VERSIONINFO**字段必须是用逗号分隔的 dword 值。
