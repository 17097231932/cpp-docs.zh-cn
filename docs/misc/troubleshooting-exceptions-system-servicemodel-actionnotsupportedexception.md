# 关于异常的疑难解答：System.ServiceModel.ActionNotSupportedException
当与调用的操作相关的动作与服务器中操作的任何动作都不匹配时，通常将在客户端上引发 <xref:System.ServiceModel.ActionNotSupportedException> 异常。
