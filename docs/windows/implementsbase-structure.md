# ImplementsBase 结构
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```  
struct ImplementsBase;  
```  
  
## 备注  
 用于验证模板参数输入。[Implements 结构](../windows/implements-structure.md)  
  
 ImplementsBase 空结构是设计使然。  
  
## 继承层次结构  
 `ImplementsBase`
