# Windows 套接字
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

此系列文章介绍 Windows 套接字的 MFC 实现。  MFC 提供了两类支持使用 Windows 套接字 API 编程的网络应用程序中。  [CAsyncSocket](../mfc/reference/casyncsocket-class.md) 类封装一种 Windows 套接字一个 API，为网络程序员最高级的功能和灵活性。  [CSocket](../mfc/reference/csocket-class.md) 类用于序列化数据提供了一种简化的接口进出 [CArchive](../mfc/reference/carchive-class.md) 对象。
