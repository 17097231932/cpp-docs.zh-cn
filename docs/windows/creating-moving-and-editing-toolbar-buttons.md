# <a name="creating-moving-and-editing-toolbar-buttons"></a>创建、移动和编辑工具栏按钮

您可以轻松地创建、 移动、 复制和编辑工具栏按钮。

默认情况下，新的或空白按钮显示工具栏的右侧。 您可以在编辑前移动此按钮。 当创建一个新按钮时，另一个空白按钮将出现编辑按钮右侧。 当您保存一个工具栏时，空白按钮不会保存。

- [创建新的工具栏按钮](creating-a-new-toolbar-button.md)

- [移动工具栏按钮](moving-a-toolbar-button.md)

- [从工具栏资源复制按钮](../windows/copying-buttons-from-a-toolbar.md)

- [删除工具栏按钮](deleting-a-toolbar-button.md)

- [工具栏资源上的按钮之间插入空格](../windows/inserting-a-space-between-buttons-on-a-toolbar.md)

- [在工具栏按钮之间的间隔](../windows/removing-space-between-buttons-on-a-toolbar.md)

- [编辑工具栏的属性页](../windows/changing-the-properties-of-a-toolbar-button.md)

有关将资源添加到托管项目的信息，请参阅[桌面应用中的资源](/dotnet/framework/resources/index)中 *.NET Framework 开发人员指南*。 有关手动将资源文件添加到托管项目、 访问资源、 显示静态资源和将资源字符串分配给属性的信息，请参阅[桌面应用中创建资源文件](/dotnet/framework/resources/creating-resource-files-for-desktop-apps)。 全球化和本地化的托管应用中的资源的信息，请参阅[Globalizing and Localizing.NET Framework Applications](/dotnet/standard/globalization-localization/index)。

## <a name="requirements"></a>要求

MFC 或 ATL
