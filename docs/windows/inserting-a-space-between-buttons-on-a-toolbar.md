# <a name="inserting-a-space-between-buttons-on-a-toolbar"></a>在工具栏按钮之间插入间隔
一般情况下，若要插入按钮之间留一个空格，只需将它们拖开另一个在工具栏上。 若要删除的空间，请将它们拖动向彼此。  
  
### <a name="to-insert-a-space-before-a-button-that-is-not-followed-by-a-space"></a>若要插入不跟一个空格的按钮前一个空格  
  
1.  将按钮拖动到右或向下直到与下一步按钮重叠大约一半。  
  
### <a name="to-insert-a-space-before-a-button-which-is-followed-by-a-space-and-to-retain-that-trailing-space"></a>若要插入的按钮之后的空间前一个空格，并保留该尾随空格  
  
1.  拖动按钮，直到右侧或底部边缘刚好接触下一步按钮，或只需重叠。  
  
### <a name="to-insert-a-space-before-a-button-that-is-followed-by-a-space-and-close-up-that-following-space"></a>若要插入跟一个空格的按钮前一个空格和近距离该以下空间  
  
1.  将按钮拖动到右或向下直到与下一步按钮重叠大约一半。
