# 编译器错误 C2830
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

只有“operator new”的位置参数可以有默认值  
  
 [new 运算符](../Topic/operator%20new%20\(%3Cnew%3E\).md)的标准形参不能具有默认值。  只有用户定义的位置参数可指定默认值。
