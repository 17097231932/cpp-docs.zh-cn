# <a name="compiler-error-c2634"></a>编译器错误 C2634

“&class::member”: 指向引用成员的指针是非法的

声明指向引用成员的指针。

下面的示例生成 C2634:

```
// C2634.cpp
int mem;
struct S {
   S() : rf(mem) { }
   int &rf;
};
int (S::*pdm) = &S::rf;   // C2634
```
