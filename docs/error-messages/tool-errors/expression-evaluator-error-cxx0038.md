# <a name="expression-evaluator-error-cxx0038"></a>表达式计算器错误 CXX0038

函数参数计数和/或类型不匹配

指定的函数调用与函数的原型不匹配。

用正确的参数数目重新键入调用。 根据需要，类型转换每个参数以匹配原型。

此错误与 CAN0038 相同。
