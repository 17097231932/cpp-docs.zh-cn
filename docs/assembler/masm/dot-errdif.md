# <a name="errdifi"></a>.ERRDIF [[I]]
如果文本项不同，则生成错误。  
  
## <a name="syntax"></a>语法  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>备注  
 如果**我**是给定，比较不区分大小写。
