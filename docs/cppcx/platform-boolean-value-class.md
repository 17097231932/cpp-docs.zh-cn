# Platform::Boolean 值类
表示布尔值。`bool` 的等效项。  
  
## 语法  
  
```cpp  
public value struct Boolean  
```  
  
## 成员  
 Boolean 具有从 [Platform::Object 类](../cppcx/platform-object-class.md)派生的 Equals\(\)、GetHashCode\(\) 和 ToString\(\) 方法，以及从 [Platform::Type 类](../cppcx/platform-type-class.md)派生的 GetTypeCode\(\) 方法。
