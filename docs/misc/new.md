# new
简要介绍 `new` 关键字的用法，并提供指向更多信息的链接。  
  
|标题|说明|  
|--------|--------|  
|[operator new](../c-runtime-library/operator-new-crt.md)|从堆中分配内存块。|  
|[operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md)|供 new\-expression 调用用来为单个对象分配存储的函数。|  
|[new 运算符](../cpp/new-operator-cpp.md)|为自由存储中的对象或对象数组分配内存，并返回指向对象的适当类型化的非零指针。|  
|[新的 \(在 vtable 的新槽\)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)|在 C\+\+\/CLI 项目中，指示虚拟成员将获取 vtable 中的新槽。  换言之，函数不会重写基类方法。|  
|[gcnew](../windows/ref-new-gcnew-cpp-component-extensions.md)|在 [!INCLUDE[cppwrt_short](../build/reference/includes/cppwrt_short_md.md)] \(`ref new`\) 或 C\+\+\/CLI \(`gcnew`\) 项目中，此复合关键字创建引用计数的对象。|
