# <a name="expression-evaluator-error-cxx0016"></a>表达式计算器错误 CXX0016
常量太大  
  
 无符号的整数常量大于 4294967295 (0FFFFFFFF) 或其量值大于大约 1.8 e + 308 的浮点常数，则不能接受 C 表达式计算器。  
  
 此错误是与 CAN0016 相同。
