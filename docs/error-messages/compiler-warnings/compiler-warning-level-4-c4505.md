# 编译器警告（等级 4）C4505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 未引用的本地函数已移除  
  
 给定函数是局部的且不在模块体内被引用；因此该函数是死代码。  
  
 编译器没有为此死函数生成代码。
