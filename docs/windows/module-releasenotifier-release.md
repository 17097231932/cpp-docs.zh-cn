# <a name="modulereleasenotifierrelease"></a>Module::ReleaseNotifier::Release
如果对象是使用 `true` 的参数构造的，则删除当前 Module::ReleaseNotifier 对象。  
  
## <a name="syntax"></a>语法  
  
```cpp  
void Release() throw();  
```  
  
## <a name="requirements"></a>要求  
 **标头：** module.h  
  
 **命名空间：** Microsoft::WRL
