# 表达式计算器错误 CXX0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

错误的格式字符串  
  
 格式字符串指定不正确。  检查表达式的语法。  
  
 该错误与 CAN0026 相同。  
  
 有关详细信息，请参阅[C\+\+ 中的格式说明符](../Topic/Format%20Specifiers%20in%20C++.md)。
