# 资源编译器错误 RC2169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

资源文件 filename 的格式并非 2.03 格式  
  
 指定的资源使用了早于 2.03 版的格式。  必须使用 3.00 版或更高版本的格式转换或重新创建资源文件。
