# <a name="elseif2"></a>ELSEIF2

如果**选项： SETIF2**为**TRUE**，则对每个程序集 pass 计算[ELSEIF](../../assembler/masm/elseif-masm.md)块。

## <a name="syntax"></a>语法

> **ELSEIF2**

## <a name="remarks"></a>备注

有关 MASM 5.1 与 MASM 6.1 中的2次通过行为的详细信息，请参阅[IF1 和 IF2](if2.md)。
