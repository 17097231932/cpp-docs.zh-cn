# <a name="expression-evaluator-error-cxx0038"></a>表达式计算器错误 CXX0038
函数自变量计数和/或类型不匹配  
  
 为指定的函数调用与函数原型不匹配。  
  
 重新键入调用具有参数的正确数目。 类型强制转换以匹配的原型，根据需要每个自变量。  
  
 此错误是与 CAN0038 相同。
