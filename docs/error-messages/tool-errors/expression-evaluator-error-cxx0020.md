# 表达式计算器错误 CXX0020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

此操作的操作数类型错误  
  
 运算符所应用于的表达式含有该运算符的无效类型。  
  
 例如，采用寄存器的地址或用浮点表达式作为数组下标是无效的。  
  
 该错误与 CAN0020 相同。
