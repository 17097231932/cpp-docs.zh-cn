# 灰色和抖色位图函数
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**灰色位图函数**  
  
 MFC 提供了两个函数为位图提供已禁用控件的外观。  
  
 ![灰色图标版本与原始图标版本之比较](../../mfc/reference/media/vcgraybitmap.png "vcGrayBitmap")  
  
|||  
|-|-|  
|[AfxDrawGrayBitmap](../Topic/AfxDrawGrayBitmap.md)|绘制灰色版本的位图。|  
|[AfxGetGrayBitmap](../Topic/AfxGetGrayBitmap.md)|复制灰色版本的位图。|  
  
 **抖色位图函数**  
  
 MFC 还提供了两个函数以将位图背景替换为抖色样式。  
  
 ![抖动图标版本与原始图标版本之比较](../../mfc/reference/media/vcditheredbitmap.png "vcDitheredBitmap")  
  
|||  
|-|-|  
|[AfxDrawDitheredBitmap](../Topic/AfxDrawDitheredBitmap.md)|绘制抖色背景位图。|  
|[AfxGetDitheredBitmap](../Topic/AfxGetDitheredBitmap.md)|复制抖色背景位图。|
