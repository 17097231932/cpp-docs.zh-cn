# <a name="resource-compiler-error-rc2015"></a>资源编译器错误 RC2015

常量中的字符太多

字符常量包含两个以上的字符。

字符常量限制为一个字符 (标准字符常量) 或两个字符 (长字符常数) 。

请注意，转义序列 (例如，将制表符) 的 \t 转换为单个字符。
