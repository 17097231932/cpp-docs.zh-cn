# 使用您的旧工具栏
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

如果您使用 Visual C\+\+ 早期版本创建自定义工具栏，[CToolBar](../mfc/reference/ctoolbar-class.md) 类的新实现可能导致您的问题。  因此无需丢弃旧工具栏使用新的功能，以前的实现仍受支持。  
  
 DOCKTOOL 示例不使用旧式工具栏，只有样式的工具栏。  
  
 您无法编辑与工具栏资源编辑器的旧式工具栏。
