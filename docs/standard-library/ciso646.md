# <a name="ltciso646gt"></a>&lt;ciso646&gt;

包括 C 标准库标头 \<iso646.h> ，并将关联名称添加到 `std` 命名空间。

> [!NOTE]
> 因为此兼容性标头定义了作为 c + + 中的关键字的名称，因此它不起作用。 \<iso646.h>C + + 中弃用了标头。 \<ciso646>该标头在草案 c + + 20 标准版中删除。

## <a name="requirements"></a>要求

**标头：**\<ciso646>

**命名空间:** std

## <a name="remarks"></a>备注

包括此标头可确保在命名空间中声明使用 C 标准库标头中的外部链接声明的名称 `std` 。
