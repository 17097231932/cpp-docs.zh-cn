# <a name="c28272"></a>C28272

> 警告 C28272：检查时，函数、参数的批注与函数声明不一致

此警告表明批注而不是所分析的代码中存在错误。 显示在函数定义上的批注与显示在声明上的批注不一致。 这两个批注在解析后应匹配。
