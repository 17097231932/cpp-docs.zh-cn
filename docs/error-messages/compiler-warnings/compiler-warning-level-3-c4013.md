# <a name="compiler-warning-level-3-c4013"></a>编译器警告（等级3） C4013

未定义 "function";假定 extern 返回 int

编译器遇到了对未定义的函数的调用。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复

1. 函数名称的拼写不正确

1. 不作为原型的外部函数**`extern`**
