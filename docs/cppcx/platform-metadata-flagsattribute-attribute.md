# <a name="platformmetadataflagsattribute-attribute"></a>Platform::Metadata::FlagsAttribute 特性

指示可将枚举视为位域（即一组标志）。

## <a name="syntax"></a>语法

```cpp
public ref class Flags abstract : Attribute
```
