# <a name="project-build-error-prj0036"></a>项目生成错误 PRJ0036
Web 部署工具的附加文件属性包含的无效条目。  
  
 Web 部署属性页上的其他文件属性包含错误，可能是因为宏计算问题。 此错误还可能表示该路径格式错误，包含字符或在文件路径中是非法的字符的组合。  
  
 若要解决此错误，请修复宏或修复所指定的路径。 计算出的路径是从项目目录的绝对路径。  
  
 此错误还可能表示该引用的文件之一不存在。
