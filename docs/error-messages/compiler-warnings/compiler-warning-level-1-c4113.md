# <a name="compiler-warning-level-1-c4113"></a>编译器警告 （等级 1） C4113

identifier1 与 identifier2 的参数列表中的差异

函数指针分配给另一个函数指针，但函数的形参列表不一致。 分配的编译时不进行修改。
