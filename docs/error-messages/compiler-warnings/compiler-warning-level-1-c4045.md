# <a name="compiler-warning-level-1-c4045"></a>编译器警告（等级 1）C4045

“array”：数组界限溢出

数组的初始值设定项太多。 忽略多余的初始值设定项。

请确保数组元素与初始值设定项在大小和数量上匹配。
