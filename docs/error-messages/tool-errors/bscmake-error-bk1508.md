# BSCMAKE 错误 BK1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法删除临时文件“文件名”\[: 原因\]  
  
 BSCMAKE 无法删除其临时文件之一。  
  
### 通过检查以下可能的原因进行修复  
  
1.  另一进程已打开该文件。  
  
2.  硬件错误。
