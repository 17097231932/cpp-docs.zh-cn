# <a name="compiler-error-c3229"></a>编译器错误 C3229
“type”：不允许泛型类型参数上的间接寻址  
  
 泛型参数不能和 `*`、 `^`或 `&`一起使用。  
  
## <a name="example"></a>示例  
 以下示例生成 C3229。  
  
```  
// C3229.cpp  
// compile with: /clr /c  
generic <class T>  
ref class C {  
   T^ t;   // C3229  
};  
  
// OK  
generic <class T>  
ref class D {  
   T u;  
};  
```  
  
## <a name="example"></a>示例  
 以下示例生成 C3229。  
  
```  
// C3229_b.cpp  
// compile with: /clr /c  
generic <class T>   // OK  
ref class Utils {  
   static void sort(T elems[], size_t size);   // C3229  
   static void sort2(T elems, size_t size);   // OK  
};  
```
