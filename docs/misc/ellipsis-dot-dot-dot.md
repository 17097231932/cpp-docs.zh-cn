# 省略号 (...)
省略号运算符 \(`...`\) 在 C 和 C\+\+ 中具有许多用途，具体取决于上下文。  它最初在 C 中用作变量函数参数列表的抽象声明符，现在还以 variadic 宏的形式用于预处理器宏编程中。  在 C\+\+ 中，`...` 用于异常处理 `catch` 块中，而在 C\+\+11 中，则用于可变参数模板。
