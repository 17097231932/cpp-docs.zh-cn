# <a name="profile-guided-optimization-warning-pg1039"></a>按配置优化警告 PG1039
失败等待空闲时间以进行扫描。  
  
 不允许应用程序[pgosweep](../../build/reference/pgosweep.md)运行。  当应用程序处于空闲状态时，如果可能，运行 pgosweep。
