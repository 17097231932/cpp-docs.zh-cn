# <a name="using-frame-windows"></a>使用框架窗口

在实现 "文件" 菜单上的 "新建" 和 "打开" 命令的过程中，MFC 框架会创建文档框架窗口及其视图和文档。 由于框架会为你执行大部分的框架窗口工作，因此在创建、使用和销毁这些窗口时只扮演了一个小型角色。 不过，您可以出于特殊目的显式创建自己的框架窗口和子窗口。
