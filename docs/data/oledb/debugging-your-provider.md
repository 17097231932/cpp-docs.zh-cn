# <a name="debugging-your-provider"></a>调试提供程序

调试提供程序有两种方法：

- 因为提供程序是在过程中创建的，所以你可以使用 OLE DB 使用者模板来创建一些使用者代码，并按常规单步执行访问接口。

- 可使用视觉对象C++附带的各种实用程序。

## <a name="to-use-debugging"></a>使用调试

1. 打开提供程序项目。

1. 在 "**项目**" 菜单上，单击 "**属性**"。

1. 在 "**属性页**" 对话框中，单击 "**调试**" 选项卡。

1. 选择所需的选项，然后单击 **"确定"** 。

1. 设置断点，然后照常调试。
