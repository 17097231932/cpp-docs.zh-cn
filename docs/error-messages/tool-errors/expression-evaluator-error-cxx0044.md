# 表达式计算器错误 CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

使用 \_based\(void\) 指针需要:\> 运算符  
  
 无法直接使用基于 `void` 的指针。  必须使用 **:\>**  运算符构造完整的指针。  
  
 该错误与 CAN0044 相同。
