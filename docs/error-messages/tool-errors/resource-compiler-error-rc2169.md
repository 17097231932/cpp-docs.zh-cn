# <a name="resource-compiler-error-rc2169"></a>资源编译器错误 RC2169

资源文件文件名不是2.03 格式

指定的资源使用了早于版本2.03 的格式。 必须使用3.00 版或更高版本的格式转换或重新创建资源文件。
