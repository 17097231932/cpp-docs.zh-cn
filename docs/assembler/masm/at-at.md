# <a name=""></a>@@:

定义代码标签仅之间可识别*label1*并*label2*，其中*label1*是代码的开始位置或上一个`@@:`标签和*label2*是代码的末尾，或者下一步是`@@:`标签。 请参阅[ @B ](../../assembler/masm/at-b.md)并[ @F ](../../assembler/masm/at-f.md)。
