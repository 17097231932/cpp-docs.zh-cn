# <a name="compiler-warning-level-3-c4013"></a>编译器警告 （等级 3） C4013

function 未定义;假设外部返回 int

编译器遇到对未定义的函数的调用。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复

1. 函数名称拼写不正确

1. 外部函数不原型 `extern`
