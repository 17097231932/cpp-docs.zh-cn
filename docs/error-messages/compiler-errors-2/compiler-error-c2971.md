# <a name="compiler-error-c2971"></a>编译器错误 C2971

class： 模板参数 param: arg： 局部变量不能用作非类型参数

不能作为模板参数中使用的名称或本地变量的地址。

下面的示例生成 C2971:

```
// C2971.cpp
template <int *pi>
class Y {};

int global_var = 0;

int main() {
   int local_var = 0;
   Y<&local_var> aY;   // C2971
   // try the following line instead
   // Y<&global_var> aY;
}
```
