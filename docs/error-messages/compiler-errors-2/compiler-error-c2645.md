# <a name="compiler-error-c2645"></a>编译器错误 C2645
指向成员的指针的限定的名 (找到:: *)  
  
 指向成员的指针的声明未指定类。  
  
 下面的示例生成 C2645:  
  
```  
// C2645.cpp  
class A {};  
int main() {  
   int B::* bp;   // C2645 B not defined  
   int A::* ap;   // OK  
}  
```
