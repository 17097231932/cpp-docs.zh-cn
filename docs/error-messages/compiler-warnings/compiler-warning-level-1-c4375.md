# <a name="compiler-warning-level-1-c4375"></a>编译器警告（等级 1）C4375

非公共方法 "method2" 不重写 "method1"

实现另一个类型的类型定义了重写方法，但重写不是公共的。 因此，该方法不重写基类型方法。
