## <a name="remarks"></a>备注  
 **CHtmlEditCtrlBase**提供用于 web 浏览器的 HTML 编辑命令，如成员函数[加粗](#bold)。 (或者，你可以调用[ExecCommand](#execcommand)执行**IDM_BOLD**命令。)  
  
 **CHtmlEditCtrlBase**不是为了独立的。 它旨在作为公开的 HTML 编辑功能的 webbrowser 控件中的派生类的基类 (请参阅[CHtmlEditCtrl](../../mfc/reference/chtmleditctrl-class.md)和[CHtmlEditView](../../mfc/reference/chtmleditview-class.md))。

##  <a name="addtoglyphtable"></a>CHtmlEditCtrlBase::AddToGlyphTable  
 指定要为特定标记在设计模式下显示的图像的标志符号表中添加一个条目。  
  
```  
HRESULT AddToGlyphTable(
    LPCTSTR szTag,  
    LPCTSTR szImgUrl,  
    unsigned short nTagType,  
    unsigned short nAlignment,  
    unsigned short nPosInfo,  
    unsigned short nDirection,  
    unsigned int nImgWidth,  
    unsigned int nImgHeight) const;  
```  
  
### <a name="parameters"></a>参数  
 `szTag`  
 标记名称 （例如，"P"或"表"）。  
  
 *szImgUrl*  
 图像 URL。  
  
 *nTagType*  
 标记类型︰ 0 表示图像为的开始标记。 1 表示映像是在结束标记。 2 表示图像的开始和结束标记。 必须用设置为 0 的标记类型添加 br 和注释等的单个标记。  
  
 *nAlignment*  
 对齐方式 （仅限矩形元素）︰ 此参数指示该图像是为具有对齐特性的元素。 左 = 0 时，中心 = 1、 右 = 2，和未定义 = 3。 左、 右或居中属性必须显式设置的元素上。  
  
 *nPosInfo*  
 定位信息。 确定哪些级联样式表 (CSS) 定位值标志符号应用的其中静态定位 = 0，绝对定位 = 1，相对定位 = 2，并且所有 = 3。 此字段使你可以指定一个标志符号时它不定位的标记和另一个标志符号，以定位到该标记时显示一个定位点。  
  
 *nDirection*  
 方向。 此参数指定的图像的基于当前语言从右向左阅读顺序标记。 0 指定从左到右、 1 指定从右到左、 2 指定自上而下的顺序、 3 指定底部到顶部，和 4 指定所有。 通常将此字段设置为 4。  
  
 *nImgWidth*  
 图像的宽度以像素为单位。  
  
 *nImgHeight*  
 图像高度 （像素）。  
  
### <a name="return-value"></a>返回值  
 返回成功，则为 S_OK 或失败的错误 HRESULT。  
  
### <a name="remarks"></a>备注  
 参数的详细信息，请参阅"标志符号表字符串格式"中[使用编辑标志符号](https://msdn.microsoft.com/library/aa969614.aspx)。  
  
 此方法可发送[IDM_ADDTOGLYPHTABLE 命令 ID](https://msdn.microsoft.com/library/aa769891.aspx)到 web 浏览器控件。
