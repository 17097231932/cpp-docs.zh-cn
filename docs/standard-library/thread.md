# <a name="ltthreadgt"></a>&lt;thread&gt;

包含标准标头 \<thread> 以定义类 `thread` 和各种支持函数。

## <a name="syntax"></a>语法

```cpp
#include <thread>
```

## <a name="remarks"></a>备注

> [!NOTE]
> 在使用 **/clr** 编译的代码中，此标头被阻止。

`__STDCPP_THREADS__`此宏定义为一个非零值，指示此标头支持线程。

|名称|描述|
|----------|-----------------|
|[thread 类](../standard-library/thread-class.md)|定义一个对象，该对象用于观察和管理应用程序中的执行线程。|
