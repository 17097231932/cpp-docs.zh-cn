# <a name="compiler-error-c2691"></a>编译器错误 C2691
数据类型︰ 托管或 WinRTarray 不能具有此元素类型  
  
 托管数组元素或 WinRT 数组元素的类型可以是值类型或引用类型。  
  
 以下示例生成 C2691：  
  
```  
// C2691a.cpp  
// compile with: /clr  
class A {};  
  
int main() {  
   array<A>^ a1 = gcnew array<A>(20);   // C2691  
   array<int>^ a2 = gcnew array<int>(20);   // value type OK  
}  
```
