# <a name="compiler-error-c2223"></a>编译器错误 C2223

-> 标识符的左侧必须指向结构/联合

`->` 左侧的操作数不是指向类、结构或联合的指针。

此错误可能引起是未定义的变量的左操作数 (因此键入`int`)。
