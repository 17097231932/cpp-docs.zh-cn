# <a name="compiler-warning-level-3-c4306"></a>编译器警告（等级 3）C4306
**“**   
 ***标识符*： 从转换**   
 ***type1*收件人**   
 ***type2*更大的**  
  
 标识符的类型强制转换为更大的指针。 未填充的新类型的高位将用零填充。  
  
 此警告可能指示不需要的转换。 生成的指针可能不是有效的。
