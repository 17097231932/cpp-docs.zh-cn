# The application cannot run.
内存不足，Visual Studio 无法运行。  
  
### 更正此错误  
  
1.  验证计算机满足运行 Visual Studio 的最低要求。  
  
2.  关闭计算机上运行的其他程序以释放内存。  
  
     \- 或 \-  
  
     重新安装 Visual Studio。
