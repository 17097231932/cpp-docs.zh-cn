# Container Class::const_reference
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  本主题介绍 Visual C\+\+ 文档作为使用标准 C\+\+ 库的容器的非运行的示例。  有关更多信息，请参见 [STL 容器](../standard-library/stl-containers.md)。  
  
 描述可以充当对控制序列元素的常量引用的对象。  
  
## 语法  
  
```  
  
typedef T3 const_reference;  
  
```  
  
## 备注  
 它介绍为此处未指定类型的 **T3** \(通常是 **Alloc::const\_reference**的同义词。\)
