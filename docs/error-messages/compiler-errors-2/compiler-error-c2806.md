# <a name="compiler-error-c2806"></a>编译器错误 C2806

"operator operator" 的形参太多

重载运算符的参数太多。

下面的示例生成 C2806：

```cpp
// C2806.cpp
// compile with: /c
class X {
public:
   X operator++ ( int, int );   // C2806 more than 1 parameter
   X operator++ ( int );   // OK
} ;
```
