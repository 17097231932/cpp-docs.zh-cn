# 编译器警告（等级 1）C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**构造函数\/析构函数的非法调用约定**  
  
 构造函数和析构函数不能具有平台默认以外的调用约定（显式指定 **\_\_clrcall** 的情况除外）。
