# 表达式计算器错误 CXX0003
表达式计算器中的内部错误  
  
 表达式计算器遇到内部错误。  
  
 记录错误发生的情况，并通知 Microsoft Corporation。 你可以在 [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) 上在线报告错误，也可以致电 Microsoft 产品支持服务。 有关详细信息，请单击 Visual C\+\+ 中**“帮助”**菜单上的**“技术支持”**。  
  
 此错误与错误 CAN0003 相同。
