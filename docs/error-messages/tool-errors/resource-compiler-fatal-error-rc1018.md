# <a name="resource-compiler-fatal-error-rc1018"></a>资源编译器错误 RC1018

意外的 "#elif"

`#elif` 指令未出现在 `#if`、 **#ifdef**或 **#ifndef**结构内。

请确保在此语句之前有 `#if`、 **#ifdef**或 **#ifndef**语句生效。
