# 编译器警告（等级 1）C4541
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”用在了带 \/GR\- 的多态类型“type”上；可能导致不可预知的行为  
  
 尝试在未启用运行时类型信息的情况下使用需要运行时类型信息的功能。  用 [\/GR](../../build/reference/gr-enable-run-time-type-information.md) 重新编译。
