# <a name="compiler-error-c2828"></a>编译器错误 C2828
operator operator 不能全局重写以与二进制格式  
  
 运算符不能有外部对象的二进制格式。  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用以下可能的解决方案进行修复  
  
1.  对象的本地做重载的运算符。  
  
2.  选择要重载的适当的一元运算符。
