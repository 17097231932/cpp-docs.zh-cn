# <a name="operator-"></a>运算符 .

第一个运算符返回*表达式*加上偏移量*字段*其结构或联合中。 第二个运算符返回值指向的位置*注册*加上偏移量*字段*其结构或联合中。

## <a name="syntax"></a>语法

> *表达式* __。__ *字段* \[ __。__ *字段*]...
>
> __\[__*注册*__]__ __。__ *字段* \[ __。__ *字段*]...
