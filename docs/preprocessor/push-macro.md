# <a name="push_macro-pragma"></a>push_macro 杂注

将*宏名称*宏的值保存在此宏的堆栈顶部。

## <a name="syntax"></a>语法

> **#pragma push_macro ("** _宏名_ **")**

## <a name="remarks"></a>备注

可以通过`pop_macro`检索的*宏名*的值。

有关示例, 请参阅[pop_macro](../preprocessor/pop-macro.md) 。
