# <a name="compiler-error-c3745"></a>编译器错误 C3745

function： 只有事件才能被引发

仅使用定义的函数[__event](../../cpp/event.md)关键字可以传递给[__raise](../../cpp/raise.md)关键字。

下面的示例生成 C3745:

```
// C3745.cpp
struct E {
   __event void func();
   void func(int) {
   }

   void func2() {
   }

   void bar() {
      __raise func();
      __raise func(1);   // C3745
      __raise func2();   // C3745
   }
};

int main() {
   E e;
   __raise e.func();
   __raise e.func(1);   // C3745
   __raise e.func2();   // C3745
}
```
