# 应用程序设置，生成文件项目向导
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

提供项目的信息。  指定的选项反映在项目的[属性页](../ide/working-with-project-properties.md)中。  创建了项目后，可以在项目属性页的 **Nmake** 页中查看和编辑下列每个选项。  
  
 **生成命令行**  
 指定当用户从“生成”菜单中选择“生成”时运行的命令行。  显示在项目属性页的 **Nmake** 页上的“生成命令行”字段中。  
  
 **Output**  
 指定将包含命令行输出的文件名。  默认情况下，此选项基于项目名称。  显示在项目属性页的 **Nmake** 页上的“输出”字段中。  
  
 **清除命令**  
 指定当用户从“生成”菜单中选择“清除”时运行的命令行。  显示在项目属性页的 **Nmake** 页上的“清除命令”字段中。  
  
 **重新生成命令行**  
 指定当用户从“生成”菜单中选择“重新生成”时运行的命令行。  显示在项目属性页的 **Nmake** 页上的“重新生成命令行” 字段中。
