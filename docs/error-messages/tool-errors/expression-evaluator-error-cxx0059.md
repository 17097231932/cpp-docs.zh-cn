# <a name="expression-evaluator-error-cxx0059"></a>表达式计算器错误 CXX0059

左操作数是类而不是函数名

函数调用的左操作数为类名，未能解析为函数调用。 省略表达式中成员函数的名称可能导致此错误。

此错误与 CAN0059 相同。
