向字形表添加一个条目，该字形指定在设计模式下为特定标记显示的图像。

```
HRESULT AddToGlyphTable(
    LPCTSTR szTag,
    LPCTSTR szImgUrl,
    unsigned short nTagType,
    unsigned short nAlignment,
    unsigned short nPosInfo,
    unsigned short nDirection,
    unsigned int nImgWidth,
    unsigned int nImgHeight) const;
```

### <a name="parameters"></a>参数

*什塔格*<br/>
标记名称（例如，"P"或"表"）。

*什伊姆图尔*<br/>
图像 URL。

*nTagType*<br/>
标记类型：0 表示图像仅适用于打开标记。 1 表示图像仅用于关闭标记。 2 表示图像同时用于打开和关闭标记。 单个标记（如 br 和注释）必须添加，标记类型设置为 0。

*n对齐*<br/>
对齐（仅限矩形元素）：此参数指示图像适用于具有对齐属性的元素。 左侧 = 0，中心 = 1，右侧 = 2，未定义 = 3。 必须在元素上显式设置左、右或中心属性。

*nPosInfo*<br/>
定位信息。 确定字形适用的级联样式表 （CSS） 定位值，其中静态定位 = 0、绝对定位 = 1、相对定位 = 2 和所有 = 3。 此字段使您能够在标记未定位时为其指定一个字形，在定位标记时为另一个字形指定锚点。

*n方向*<br/>
方向。 此参数基于当前语言的读取顺序指定标记的图像。 0 从左到右指定，1 指定从右到左，2 指定从上到下，3 指定从下到上，4 指定全部。 通常将此字段设置为 4。

*nImgWidth*<br/>
图像宽度（以像素为单位）。

*尼姆格高地*<br/>
图像高度（以像素为单位）。
