# 项目生成错误 PRJ0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配给文件“file”的自定义生成规则“rule”的“附加依赖项”属性无效。该属性包含计算为“value”的“string”。  
  
 **“附加依赖项”**属性的计算结果为一个空字符串或一个包含无效字符（任何不能在文件名或目录名中出现的字符）的字符串。  自定义生成规则需要生成操作的输出结果。  
  
 有关详细信息，请参阅[指定自定义生成工具](../../ide/specifying-custom-build-tools.md)。
