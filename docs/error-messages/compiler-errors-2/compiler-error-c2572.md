# <a name="compiler-error-c2572"></a>编译器错误 C2572
class::member： 重定义的默认参数： 参数 param  
  
 默认参数不能重新定义。 如果你需要另一个参数的值，默认参数应保持为未定义状态。  
  
 下面的示例生成 C2572:  
  
```  
// C2572.cpp  
// compile with: /c  
void f(int i = 1);   // function declaration  
  
// function definition  
void f(int i = 1) {}   // C2572  
  
// try the following line instead  
// void f(int i) {}  
```
