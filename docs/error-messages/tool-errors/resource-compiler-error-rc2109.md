# 资源编译器错误 RC2109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

应输入数值对话框常数  
  
 对于 *x、y、width* 和 *height* 字段，**DIALOG** 语句要求整数值。  确保这些值包括在 **DIALOG** 关键字后并且不为负数。
