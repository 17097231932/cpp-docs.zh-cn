# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**操作数必须是记录类型或字段**  
  
 按照 [宽度](../../assembler/masm/operator-width.md) 或 [掩码](../../assembler/masm/operator-mask.md) 运算符的操作数无效。  
  
 [宽度](../../assembler/masm/operator-width.md) 运算符采用字段或记录名称的操作数。  [掩码](../../assembler/masm/operator-mask.md) 运算符采用字段或一条记录类型名称的操作数。
