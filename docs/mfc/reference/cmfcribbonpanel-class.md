# <a name="cmfcribbonpanel-class"></a>CMFCRibbonPanel 类

实现包含一组功能区元素的面板。 在绘制面板时，系统将根据面板的大小显示尽可能多的元素。

有关更多详细信息，请参阅中的源代码**VC\\atlmfc\\src\\mfc**的 Visual Studio 安装文件夹。

## <a name="syntax"></a>语法

```
class CMFCRibbonPanel : public CObject
```

## <a name="remarks"></a>备注

功能区面板是创建功能区类别中的相关任务的逻辑分组。 为功能区更改的大小，面板布局自动调整以尽可能显示尽可能多的元素。

获取功能区面板中功能区类别包含通过调用[CMFCRibbonCategory::GetPanel](../../mfc/reference/cmfcribboncategory-class.md#getpanel)方法。

##  <a name="addtoolbar"></a>  CMFCRibbonPanel::AddToolBar

将工具栏添加到功能区面板。

```
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,
UINT uiColdResID = 0,
UINT uiHotResID = 0,
UINT uiDisabledResID = 0);
```

### <a name="parameters"></a>参数

*uiToolbarResID*<br/>
[in]指定要添加工具栏的资源 ID。

*uiColdResID*<br/>
[in]指定工具栏的冷图像的资源 ID。

*uiHotResID*<br/>
[in]指定工具栏的热图像的资源 ID。

*uiDisabledResID*<br/>
[in]指定工具栏的已禁用映像的资源 ID。

### <a name="return-value"></a>返回值

调用此方法以将工具栏添加到功能区面板。 工具栏将添加到先前的调用添加的功能区元素旁边[cmfcribbonpanel:: Add](#add)。

### <a name="remarks"></a>备注

有关工具栏、 热映像、 冷映像和已禁用的映像的详细信息，请参阅[CMFCToolBar 类](../../mfc/reference/cmfctoolbar-class.md)。

### <a name="example"></a>示例

下面的示例演示如何使用`SetElementRTCByID`方法：

```

// Load and add toolbar with standard buttons. This toolbar
// should display a custom color button with id ID_CHAR_COLOR:

pPanel->AddToolBar(IDR_MAINFRAME,
    IDB_MAINFRAME256);

CMFCRibbonColorButton* pColorButton =
(CMFCRibbonColorButton*)pPanel->SetElementRTCByID(
ID_CHAR_COLOR,
    RUNTIME_CLASS (CMFCRibbonColorButton));

// SetElementRTCByID sets runtime class and returns a pointer
// to the newly created custom button,
    which can be set up immediately:
pColorButton->EnableAutomaticButton(_T("Automatic"),
    RGB (0,
    0,
    0));
```
