# <a name="compiler-warning-level-1-c4098"></a>编译器警告（等级1） C4098

"function"： void 函数返回值

使用返回类型[void](../../cpp/void-cpp.md)声明的函数包含一个 **`return`** 返回值的语句。 编译器假设函数返回类型为的值 **`int`** 。
