# <a name="compiler-error-c2541"></a>编译器错误 C2541

删除： 删除： 不能删除不是指针的对象

[删除](../../cpp/delete-operator-cpp.md)运算符不是指针的对象上使用。

下面的示例生成 C2541:

```
// C2541.cpp
int main() {
   int i;
   delete i;   // C2541 i not a pointer

   // OK
   int *ip = new int;
   delete ip;
}
```
