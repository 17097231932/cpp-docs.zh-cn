# <a name="linker-tools-warning-lnk4073"></a>链接器工具警告 LNK4073
无法创建.ilk 文件; 的映射非增量链接  
  
 无法使用大量的链接以创建增量状态 (.ilk) 文件共享内存中连续空间不足。 链接执行非增量生成。
