# &lt;typeindex&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定义类和函数的标准头 \<包括 typeindex\> 支持 [type\_info](../cpp/type-info-class.md)类索引对象。  
  
## 语法  
  
```cpp  
#include <typeindex>  
```  
  
## 备注  
 [hash 结构](../standard-library/hash-structure.md) 定义适用于映射值类型的分配。`hash function` 的索引。[type\_index](../standard-library/type-index-class.md)  
  
 `type_index` 包装类的指针到帮助的 `type_info` 对象在索引。
