# <a name="compiler-error-c3130"></a>编译器错误 C3130
内部编译器错误： 无法写入 PDB 插入的代码块  
  
 如果编译器无法插入的代码块写入.pdb 文件，则会发生此错误。 失败的最常见原因是缺乏磁盘空间。
