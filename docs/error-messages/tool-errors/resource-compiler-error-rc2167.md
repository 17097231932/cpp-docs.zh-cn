# 资源编译器错误 RC2167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法识别的 VERSIONINFO 字段；应输入 BEGIN 或逗号  
  
 **VERSIONINFO** 结构声明的 **FIXED** 部分中发现无法识别的字段。  **VERSIONINFO** 字段必须是逗号分隔的 DWORDS。
