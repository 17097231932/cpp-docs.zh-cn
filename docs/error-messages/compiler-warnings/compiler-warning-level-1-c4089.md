# <a name="compiler-warning-level-1-c4089"></a>编译器警告 (等级 1) C4089

"function"：实参 "number"、形参 "number" 中的类型不同

对应的形参和实参具有不同的类型。 未更改即可传递实际参数。 函数将实参转换为在函数定义中指定的类型。
