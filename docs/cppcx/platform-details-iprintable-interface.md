# <a name="platformdetailsiprintable-interface"></a>Platform::Details::IPrintable 接口

此接口仅供内部使用，不用于开发。

## <a name="syntax"></a>语法

```cpp
interface class IPrintable;
```

### <a name="remarks"></a>备注

仅出于完整性的考虑提供此接口，因为可以通过浏览器对其进行检查。
