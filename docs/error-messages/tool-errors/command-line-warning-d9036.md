# <a name="command-line-warning-d9036"></a>命令行警告 D9036

> '*选项\_2*而不是 of*选项\_1*

## <a name="remarks"></a>备注

警告 D9036 是的延续[命令行警告 D9035](../../error-messages/tool-errors/command-line-warning-d9035.md)。 如果一个编译器选项是首选而不是不推荐使用的选项，则它是此处列出。 有关详细信息的链接，请参阅[已弃用并删除的编译器选项](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options)。
