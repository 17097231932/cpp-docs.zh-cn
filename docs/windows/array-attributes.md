# <a name="array-attributes"></a>数组特性
下列属性适用于数组或**SAFEARRAY**s。  
  
|特性|描述|  
|---------------|-----------------|  
|[library_block](../windows/library-block.md)|将.idl 文件的库块中的构造。|  
|[satype](../windows/satype.md)|指定的 SAFEARRAY 结构的数据类型。|  
|[字符串](../windows/string-cpp.md)|指示的一维`char`， `wchar_t`，**字节**（或等效） 必须作为一个字符串处理数组或指向这样的数组的指针。|
