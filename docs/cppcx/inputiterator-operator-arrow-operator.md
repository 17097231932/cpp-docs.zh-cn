# InputIterator::operator-&gt; 运算符
检索当前 InputIterator 指定的元素的地址。  
  
## 语法  
  
```  
pointer operator->() const;  
```  
  
## 返回值  
 当前 InputIterator 指定的元素的地址。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
