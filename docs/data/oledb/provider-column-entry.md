# <a name="providercolumnentry"></a>PROVIDER_COLUMN_ENTRY
表示提供程序支持的特定列。  
  
## <a name="syntax"></a>语法  
  
```cpp
PROVIDER_COLUMN_ENTRY (name  
, ordinal, member )  
```  
  
#### <a name="parameters"></a>参数  
 *name*  
 [in]列名称。  
  
 `ordinal`  
 [in] 列号。 除非列是书签列，列数必须不为 0。  
  
 `member`  
 [in]中的成员变量`dataClass`与列对应。  
  
## <a name="requirements"></a>惠?  
 **标头：** atldb.h
