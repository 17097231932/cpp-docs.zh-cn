# <a name="asyncbasefirecompletion-method"></a>AsyncBase::FireCompletion 方法
调用完成事件处理程序，或重置内部进行委托。  
  
## <a name="syntax"></a>语法  
  
```  
void FireCompletion(  
   void  
) override;  
  
virtual void FireCompletion();  
```  
  
## <a name="remarks"></a>备注  
 第一个版本**FireCompletion()** 重置内部进度委托变量。 如果异步操作已完成，第二个版本调用完成事件处理程序。  
  
## <a name="requirements"></a>要求  
 **标头：** async.h  
  
 **命名空间：** Microsoft::WRL
