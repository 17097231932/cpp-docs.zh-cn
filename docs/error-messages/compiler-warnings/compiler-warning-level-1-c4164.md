# 编译器警告（等级 1）C4164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 未声明的内部函数  
  
 指定内部函数未声明；可能需要对相应的头文件执行 \#include 操作。
