# auto_rename
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 专用**  
  
 通过将两个下划线 \(\_\_\) 追加到变量名称来重命名 C\+\+ 保留字，从而解决可能的名称冲突。  
  
## 语法  
  
```  
auto_rename  
```  
  
## 备注  
 当导入将一个或多个 C\+\+ 保留字（关键字或宏）用作变量名称的类型库时，使用此特性。  
  
 **结束 C\+\+ 专用**
