# 编译器错误 C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“data type”:“operator”的左边必须是类、结构或联合  
  
 若要将模板参数用作类，请使用类而不是 integral 类型专用化该类模板。
