# <a name="defaulttypenamegethashcode-method"></a>default::(type_name)::GetHashCode 方法
返回此实例的哈希代码。  
  
## <a name="syntax"></a>语法  
  
```cpp  
public:int GetHashCode()  
```  
  
### <a name="return-value"></a>返回值  
 此实例的哈希代码。

 **支持的最低服务器：** Windows Server 2012

 **标头：** vccorlib.h
