# <a name="expression-evaluator-error-cxx0059"></a>表达式计算器错误 CXX0059
左的操作数是类不是函数的名称  
  
 函数调用的左的操作数是类名，无法解析到函数调用。 忽略在表达式中的成员函数的名称可能会导致此错误。  
  
 此错误是与 CAN0059 相同。
