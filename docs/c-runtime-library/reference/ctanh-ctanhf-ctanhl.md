# <a name="ctanh-ctanhf-ctanhl"></a>ctanh、ctanhf、ctanhl
计算复数的复杂双曲正切值。  
  
## <a name="syntax"></a>语法  
  
```  
_Dcomplex ctanh(   
   _Dcomplex z   
);  
_Fcomplex ctanh(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex ctanh(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex ctanhf(   
   _Fcomplex z   
);  
_Lcomplex ctanhl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>参数  
 `z`  
 表示角度的复数（以弧度为单位）。  
  
## <a name="return-value"></a>返回值  
 `z` 的复杂双曲正切值。  
  
|输入|SEH 异常|`_matherr` 异常|  
|-----------|-------------------|--------------------------|  
|± ∞、QNAN、IND|无|_DOMAIN|  
|± ∞ （tan、 tanf）|INVALID|_DOMAIN|  
  
## <a name="remarks"></a>备注  
 由于 C++ 允许重载，因此你可以调用采用并返回 `ctanh` 和 `_Fcomplex` 值的 `_Lcomplex` 重载。 在 C 程序中，`ctanh` 始终采用并返回 `_Dcomplex` 值。
