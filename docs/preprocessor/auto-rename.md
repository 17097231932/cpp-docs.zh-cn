# <a name="auto_rename-import-attribute"></a>auto_rename 导入属性

**C++相关**

通过将两个下划线 (__) 追加到变量名称来重命名 C++ 保留字，从而解决可能的名称冲突。

## <a name="syntax"></a>语法

> **#import***类型库***auto_rename**

## <a name="remarks"></a>备注

当导入将一个或多个 C++ 保留字（关键字或宏）用作变量名称的类型库时，使用此特性。

**结束C++特定**
