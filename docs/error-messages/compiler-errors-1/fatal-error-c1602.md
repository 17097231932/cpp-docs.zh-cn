# <a name="fatal-error-c1602"></a>错误 C1602
不支持的内部函数  
  
 此错误指示编译器.exe 文件，这可能是由于不完整的安装中不匹配。 例如，你可能已安装 service pack 但不是处理器包。 安装所有必需的产品。
