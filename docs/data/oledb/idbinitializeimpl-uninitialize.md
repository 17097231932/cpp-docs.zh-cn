# <a name="idbinitializeimpluninitialize"></a>IDBInitializeImpl::Uninitialize
位置数据通过释放内部资源的属性支持如源中未初始化状态的对象。  
  
## <a name="syntax"></a>语法  
  
```cpp
      STDMETHOD(Uninitialize)(void);  
```
