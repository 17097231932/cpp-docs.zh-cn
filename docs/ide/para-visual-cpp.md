# &lt;para&gt; (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

\<para\> 标记用于诸如 [\<summary\>](../ide/summary-visual-cpp.md)、[\<remarks\>](../ide/remarks-visual-cpp.md) 或 [\<returns\>](../ide/returns-visual-cpp.md) 等标记内，使您得以将结构添加到文本中。  
  
## 语法  
  
```  
<para>content</para>  
```  
  
#### 参数  
 `content`  
 段落文本。  
  
## 备注  
 使用 [\/doc](../build/reference/doc-process-documentation-comments-c-cpp.md) 进行编译可以将文档注释处理到文件中。  
  
## 示例  
 有关使用 \<para\> 的示例，请参见 [\<summary\>](../ide/summary-visual-cpp.md)。
