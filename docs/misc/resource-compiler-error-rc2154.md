# 资源编译器错误 RC2154
控制字符超出范围 \[^A \- ^Z\]  
  
 **ACCELERATORS** 语句中的控制字符无效。 插入符号 \(**^**\) 后面的字符必须介于 A 和 Z（包含 A 和 Z）。
