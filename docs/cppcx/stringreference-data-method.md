# StringReference::Data 方法
以 char16 值数组形式返回此 `StringReference` 的内容。  
  
## 语法  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## 返回值  
 char16 UNICODE 文本字符的数组。
