# File &lt;filename&gt; cannot be created.
当尝试创建代码注释 Web 报告但当前用户没有写访问权时，通常会发生此错误。  
  
### 更正此错误  
  
1.  确认您拥有写访问权。  
  
2.  确认有足够的空间用于保存页。
