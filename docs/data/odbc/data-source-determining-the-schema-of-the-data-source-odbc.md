# <a name="data-source-determining-the-schema-of-the-data-source-odbc"></a>数据源：确定数据源的架构 (ODBC)

本主题适用于 MFC ODBC 类。

若要在对象中设置数据成员 `CRecordset` ，需要知道要连接到的数据源的架构。 确定数据源的架构涉及获取数据源中的表的列表、每个表中的列列表、每个列的数据类型以及是否存在任何索引。
