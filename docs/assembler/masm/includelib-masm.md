# <a name="includelib"></a>INCLUDELIB

通知链接器，当前模块应与 *libraryname* 链接。

## <a name="syntax"></a>语法

> **INCLUDELIB** *libraryname*

## <a name="remarks"></a>备注

如果 *libraryname* 包括反斜杠、分号、大于符号、小于符号、单引号或双引号，则必须将其括在尖括号中。
