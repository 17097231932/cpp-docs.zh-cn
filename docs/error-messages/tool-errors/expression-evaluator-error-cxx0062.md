# <a name="expression-evaluator-error-cxx0062"></a>表达式计算器错误 CXX0062

不支持构造函数调用

表达式调用了构造函数。 表达式不能显式调用构造函数，也不能进行需要调用构造函数的转换。

此错误与 CAN0062 相同。
