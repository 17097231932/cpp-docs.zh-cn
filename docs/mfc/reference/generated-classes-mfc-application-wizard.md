# <a name="generated-classes-mfc-application-wizard"></a>MFC 应用程序向导的生成的类

本主题列出了基本的类和你的项目生成的文件的名称。 默认情况下，名称基于你在中指定的项目名称**新建项目对话框**。 如下所述，您可以更改这些名称的大多数：

- **生成的类**

   为项目创建的类的名称。 默认情况下，名称基于项目名称。 默认 MFC 项目将创建一个 C*ProjName*查看类、 一个*ProjName*App 类，C*ProjName*Doc 类、 一个 CMainFrame 类和一个 CChildFrame 类。 此页上的所有其他值包含有关中当前所选类的信息**生成的类**列表。

   若要更改类名称，请使用**类名**文本框。

- **类名**

   中当前所选类的名称**生成的类**列表。 如果此框处于活动状态，可以更改类名称。 当你更改中的焦点**类名**框中，对所选的类名称的任何更改将出现在**生成的类**列表。

- **.h 文件**

   中当前所选类的头文件的名称**生成的类**列表。 如果在文本框处于活动状态，可以更改标头文件的名称。

- **基类**

   中的当前所选类的基类的名称**生成的类**列表。 如果此框处于活动状态，您可以从列表中选择另一个类的基类。

- **.cpp 文件**

   与所选类关联源代码文件的名称。 如果在文本框处于活动状态，可以更改的实现文件的名称。
