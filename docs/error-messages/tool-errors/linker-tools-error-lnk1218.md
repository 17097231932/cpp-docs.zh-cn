# 链接器工具错误 LNK1218
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

警告被视为错误；未生成输出文件  
  
 使用 **\/WX** 进行链接时，任何链接器警告都将被视为错误，且不会创建任何输出文件。  
  
 有关更多信息，请参见[\/WX（将链接器警告视为错误）](../../build/reference/wx-treat-linker-warnings-as-errors.md)。
