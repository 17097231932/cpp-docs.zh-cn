# <a name="comptrinternalrelease-method"></a>ComPtr::InternalRelease 方法
对与此 ComPtr 关联的接口执行 COM 释放操作。  
  
## <a name="syntax"></a>语法  
  
```  
void InternalRelease();  
```  
  
## <a name="remarks"></a>备注  
 此方法是受保护。  
  
## <a name="requirements"></a>惠?  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
