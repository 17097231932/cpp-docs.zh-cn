# <a name="renamesearchnamespace"></a>rename_search_namespace
**C + + 专用**  
  
 具有相同的功能[rename_namespace](../preprocessor/rename-namespace.md)属性，但在使用与 #import 指令的类型库上使用[auto_search](../preprocessor/auto-search.md)属性。  
  
## <a name="syntax"></a>语法  
  
```  
rename_search_namespace("NewName")  
```  
  
#### <a name="parameters"></a>参数  
 `NewName`  
 命名空间的新名称。  
  
## <a name="remarks"></a>备注  
 **结束 c + + 专用**
