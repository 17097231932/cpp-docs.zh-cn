# <a name="compiler-error-c2503"></a>编译器错误 C2503

class： 基类不能包含大小为零的数组

基类或结构包含一个大小为零的数组。 在类中的数组必须具有至少一个元素。

下面的示例生成 C2503:

```
// C2503.cpp
// compile with: /c
class A {
   public:
   int array [];
};

class B : A {};    // C2503

class C {
public:
   int array [10];
};

class D : C {};
```
