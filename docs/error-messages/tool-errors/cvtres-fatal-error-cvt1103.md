# <a name="cvtres-fatal-error-cvt1103"></a>CVTRES 错误 CVT1103
无法读取 filename  
  
 CVTRES 尝试读取给定的文件时发生了不可恢复的错误。  
  
 可以通过文件损坏导致此错误。
