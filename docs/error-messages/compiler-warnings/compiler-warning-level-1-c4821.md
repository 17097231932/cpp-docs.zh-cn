# 编译器警告（等级 1）C4821
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法确定 Unicode 编码类型，请用签名\(BOM\)保存文件  
  
 编译器未能确定文件的编码类型。  若要解决此警告，请用字节顺序标记保存该文件。  有关更多信息，请参见[管理带编码的文件](http://msdn.microsoft.com/zh-cn/ab3a74da-5c8f-4680-8882-f1def4909e82)。
