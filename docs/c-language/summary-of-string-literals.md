# <a name="summary-of-string-literals"></a>字符串文本的摘要
string-literal：  
 “ s-char-sequence opt”  
  
 L” s-char-sequence opt”  
  
 s-char-sequence：  
 s-char  
  
 s-char-sequence s-char  
  
 s-char：  
 除双引号 (")、反斜杠 (\\) 或换行符 escape-sequence 之外的源字符集的任何成员
