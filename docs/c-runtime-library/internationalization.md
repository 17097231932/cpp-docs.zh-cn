# <a name="internationalization"></a>国际化

Microsoft 运行时库提供许多例程，可用于为国际市场自定义应用，例如：
- [与区域设置相关的例程](../c-runtime-library/locale.md)
- 宽字符例程
- 多字节字符例程
- 一般文本例程

为方便起见，大多数区域设置相关的例程也按它们的用途进行分类。

多字节例程和宽字符例程在其对应的单字节字符的对应项（当存在时）进行描述。

还包括 ISO646 运算符替代项。
