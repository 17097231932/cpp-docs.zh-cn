# <a name="resource-compiler-error-rc2114"></a>资源编译器错误 RC2114
预期的控件类名  
  
 `class`字段中的控制语句**对话框**语句必须是以下类型之一： 按钮、 组合框、 编辑、 列表框、 滚动条、 静态的或用户定义。 请确保类的拼写正确。
