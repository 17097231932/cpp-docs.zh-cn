# <a name="displaybind"></a>displaybind

指示应显示给用户作为可绑定的属性。

## <a name="syntax"></a>语法

```cpp
[displaybind]
```

## <a name="remarks"></a>备注

**Displaybind** c + + 属性具有相同的功能[displaybind](/windows/desktop/Midl/displaybind) MIDL 特性。
