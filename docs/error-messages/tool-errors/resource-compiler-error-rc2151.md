# <a name="resource-compiler-error-rc2151"></a>资源编译器错误 RC2151

不能重复使用字符串常量

在**STRINGTABLE**语句中两次使用相同的值。 请确保不会混合重叠的十进制和十六进制值。

**STRINGTABLE**中的每个 ID 都必须是唯一的。 为了获得最大效率，请使用以16的倍数开始的连续常量。
