# <a name="compiler-warning-levels-1-and-4-c4223"></a>编译器警告（等级 1 和等级 4）C4223

使用了非标准扩展：将非 lvalue 数组转换为指针

在标准 C 中，不能将非左值数组转换为指针。  ([/ze](../../build/reference/za-ze-disable-language-extensions.md)) 的默认 Microsoft 扩展，你可以。
