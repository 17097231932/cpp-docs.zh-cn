# Selecting Foreground or Background Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

分别按下鼠标左键或鼠标右键时，除了**橡皮擦**之外，**图像编辑器**工具栏上的工具都能使用当前前景色或背景色绘图。  
  
### 若要选择前景色  
  
1.  用鼠标左键在[调色板](../windows/colors-window-image-editor-for-icons.md)上单击你想要的颜色。  
  
### 若要选择背景色  
  
1.  用鼠标右键在**调色板**上单击你想要的颜色。  
  
 有关将资源添加到托管项目的信息，请参阅“.NET Framework 开发员指南”中的[应用程序中的资源](../Topic/Resources%20in%20Desktop%20Apps.md)。

 无
