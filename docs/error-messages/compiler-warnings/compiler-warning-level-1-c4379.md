# 编译器警告（等级 1）C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

此编译器不支持 Version 版本的公共语言运行时。使用此版本可能会引起意外的结果。  
  
 计算机上已安装了早期版本的公共语言运行时，而不是当前版本。  若要解决 C4379，请安装编译器附带的公共语言运行时版本。
