# <a name="linker-tools-error-lnk1215"></a>链接器工具错误 LNK1215

元数据操作失败 (HRESULT) ：错误

尝试通过 .NET 运行时执行元数据更新时，链接器从 .NET 运行时收到错误。

`HRESULT` 是 .NET 运行时方法中的 HRESULT。 `error` 是。NET 提供的文本。

您可能有不匹配的链接器和 .NET 运行时;重新安装 Visual C++。
