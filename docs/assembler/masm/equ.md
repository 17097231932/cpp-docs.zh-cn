# <a name="equ"></a>EQU

第一个指令将 *expression* 的数值赋给 *name*。

## <a name="syntax"></a>语法

> *名称***等于***表达式*
>
> *名称***等于**__\<__*text*__>__

## <a name="remarks"></a>备注

稍后不能重定义该 *名称* 。

第二个指令为 *名称* 分配指定的 *文本*。 稍后可以为该 *名称* 分配不同的 *文本* 。 请参阅 [TEXTEQU](textequ.md)。
