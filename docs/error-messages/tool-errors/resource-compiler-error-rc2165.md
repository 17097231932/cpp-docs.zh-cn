# 资源编译器错误 RC2165
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 DLGINCLUDE 语句中未找到字符串  
  
 该语句未指定有效的包含文件。  
  
 **DLGINCLUDE** 语句必须使用以下语法：  
  
 **DLGINCLUDE "*filename*.h"**
