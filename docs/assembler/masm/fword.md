# FWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化 6 个字节每 `initializer`的存储。  
  
## 语法  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## 备注  
 可以用作类型说明符任何位置类型是非法的。
