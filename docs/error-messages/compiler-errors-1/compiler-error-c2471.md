# <a name="compiler-error-c2471"></a>编译器错误 C2471

无法更新程序数据库 file

编译器无法写入数据库文件。

### <a name="to-fix-the-problem"></a>修复此问题

- 确保磁盘具有足够的可用空间。

- 确保文件不是只读的。

- 确保没有共享冲突。
