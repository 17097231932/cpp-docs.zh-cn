# <a name="schema-rowsets"></a>架构行集

架构（或元数据）是有关数据库结构或组织的任何信息，如有关提供程序、行集、表、列的信息，或者与实际数据库内容有关的其他信息。 OLE DB 允许您通过使用架构行集来获取此类信息，而无需打开行集。

有关如何使用架构行集的信息，请参阅[使用架构行集获取元数据](../../data/oledb/obtaining-metadata-with-schema-rowsets.md)。
