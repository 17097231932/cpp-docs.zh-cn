# BSCMAKE 错误 BK1507
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法打开临时文件“文件名”\[: 原因\]  
  
 BSCMAKE 无法打开临时文件。  
  
### 通过检查以下可能的原因进行修复  
  
1.  未将 **TMP** 环境变量设置为有效驱动器和目录。  
  
2.  磁盘已满。
