# Enumerators and Collections Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类提供对COM集合和枚举支持:  
  
-   [CComEnum](../atl/reference/ccomenum-class.md) 定义基于数组的COM enumerator对象。  
  
-   [CComEnumImpl](../atl/reference/ccomenumimpl-class.md) 对枚举项存储在数组中的COM枚举器提供了接口实现。  
  
-   [CComEnumOnSTL](../atl/reference/ccomenumonstl-class.md) 定义基于STL集合的COM enumerator对象。  
  
-   [IEnumOnSTLImpl](../atl/reference/ienumonstlimpl-class.md) 对枚举项在一个STL兼容容器存储的COM枚举器提供了接口实现。  
  
-   [ICollectionOnSTLImpl](../atl/reference/icollectiononstlimpl-class.md) 为 **Count**、 **Item**和集合接口的 `_NewEnum` 属性提供实现。
