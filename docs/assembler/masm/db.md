# <a name="db"></a>DB

为每个*初始值设定项*分配一个字节的存储，并根据需要对其进行初始化。 **DB**是[字节](byte-masm.md)的同义词。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **DB** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧

## <a name="remarks"></a>备注

在类型合法的任何位置，还可以用作类型说明符。
