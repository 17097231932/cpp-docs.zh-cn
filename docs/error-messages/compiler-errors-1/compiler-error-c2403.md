# 编译器错误 C2403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 寄存器必须为“context”中的基\/索引寄存器  
  
 间接存储操作数中使用的寄存器必须是此上下文中的基\/索引寄存器。
