# &lt;cvt/wstring&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual Studio 早期版本中的标头 `<cvt/wstring>` 在 stdext::cvt 命名空间中定义模板类 [wstring\_convert 类](../standard-library/wstring-convert-class.md)。  保留标头以实现向后兼容性。  新代码应使用在 `std` 命名空间的 [\<locale\>](../standard-library/locale.md) 中定义的类的版本  
  
## 语法  
  
```  
  
#include <cvt/wstring>  
  
```
