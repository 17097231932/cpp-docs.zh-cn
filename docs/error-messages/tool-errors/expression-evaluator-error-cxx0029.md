# 表达式计算器错误 CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非结构指针  
  
 成员选择运算符 \(**\-\>**\) 应用到的表达式不是结构指针。  
  
 检查整个表达式是否正确地使用了括号，或者将地址表达式类型转换为适当的结构指针类型。  
  
 该错误与 CAN0029 相同。
