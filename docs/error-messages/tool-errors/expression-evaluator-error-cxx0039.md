# 表达式计算器错误 CXX0039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

符号不明确  
  
 C 表达式计算器无法确定表达式中要使用的符号的实例。  在继承树中该符号出现不止一次。  
  
 必须使用范围解析运算符 \(`::`\) 显式指定表达式中要使用的实例。  
  
 该错误与 CAN0039 相同。
