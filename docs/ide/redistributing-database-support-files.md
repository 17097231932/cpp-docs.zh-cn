# <a name="redistributing-database-support-files"></a>重新分发数据库支持文件
数据访问对象 (DAO) 和 Microsoft 数据访问 SDK 中的数据库技术，可以重新发布支持文件。  
  
## <a name="installing-dao-support-files"></a>安装 DAO 支持文件  
 若要获取 DAO 的最新版本，请参阅[文章 239114： 如何获取 Microsoft Jet 4.0 数据库引擎的最新 service pack](http://go.microsoft.com/fwlink/?LinkId=198014) Microsoft 支持网站上。  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>安装 Microsoft 数据访问 SDK 支持文件  
 使用 Mdac_typ.exe 安装对 ODBC、 OLE DB、 ActiveX 数据对象 (ADO) 和远程数据服务 (RDS) 的支持。 Mdac_typ.exe 位于 Visual Studio 安装媒体上的 ..\WCU\MDAC28\ 文件夹中。 你也可以下载 Mdac_typ.exe 从[Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=198015)网站。  
  
 有关详细信息，在[MSDN](http://go.microsoft.com/fwlink/?LinkId=198016)网站中，搜索"重新分发 MDAC 2.8 SP1"。 如果使用 Visual Studio 安装程序项目来部署应用程序，请参阅[部署和依赖关系](http://msdn.microsoft.com/en-us/49e9b84d-bd6a-4388-b9ac-46ea79cf0733)。
