# Vector::RemoveAt 方法
从当前向量删除指定索引标识的元素。  
  
## 语法  
  
```  
  
virtual void RemoveAt(  
   unsigned int index  
);  
```  
  
#### 参数  
 `index`  
 从零开始的无符号整数，用于指定 Vector 对象中的特定元素。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
