# ATL Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

若要按类别查找ATL功能，请参见下列主题。  
  
 [COM映射全局函数](../../atl/reference/com-map-global-functions.md)  
 可以通过为COM映射 **IUnknown** 实现支持。  
  
 [复合控件全局函数](../../atl/reference/composite-control-global-functions.md)  
 可以通过支持创建对话框以及创建，承载和授权ActiveX控件。  
  
 [连接点全局函数](../../atl/reference/connection-point-global-functions.md)  
 可以通过为连接点和接收映射支持。  
  
 [调试和错误报告全局函数](../../atl/reference/debugging-and-error-reporting-global-functions.md)  
 提供有用的调试和跟踪功能。  
  
 [设备上下文全局函数](../../atl/reference/device-context-global-functions.md)  
 创建特定设备的设备上下文。  
  
 [处理全局函数的事件](../../atl/reference/event-handling-global-functions.md)  
 提供一个事件处理程序。  
  
 [封送处理全局函数](../../atl/reference/marshaling-global-functions.md)  
 可以通过是封送处理和转换封送数据支持为接口指针。  
  
 [Pixel\/HIMETRIC转换全局函数](../../atl/reference/pixel-himetric-conversion-global-functions.md)  
 可以通过对转换支持来回像素和HIMETRIC单元。  
  
 [注册表和类型库全局函数](../../atl/reference/registry-and-typelib-global-functions.md)  
 可以通过为加载和注册类型支持库。  
  
 [安全全局函数](../../atl/reference/security-global-functions.md)  
 可以通过为修改的SID和ACL对象支持。  
  
 [安全标识符\(sid\)全局函数](../../atl/reference/security-identifier-global-functions.md)  
 返回常见已知的SID对象。  
  
 [服务器注册全局函数](../../atl/reference/server-registration-global-functions.md)  
 可以通过提供注册的支持，并在对象的注销的服务器对象映射。  
  
 [WinModule全局函数](../../atl/reference/winmodule-global-functions.md)  
 可以通过为 `_AtlCreateWndData` framework操作支持。
