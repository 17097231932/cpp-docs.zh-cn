# VectorViewIterator::operator* 运算符
检索对当前 VectorViewIterator 指定的元素的引用。  
  
## 语法  
  
```  
reference operator*() const;  
```  
  
## 返回值  
 当前 VectorViewIterator 指定的元素。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
