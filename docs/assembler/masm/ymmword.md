# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

用于 256 位与 Intel 的多媒体操作数高级矢量扩展 \(AVX\) 命令。  
  
## 语法  
  
```  
YMMWORD  
```  
  
## 备注  
 `YMMWORD` 用于表示类型和 AVX 内部的 `__m256` 相同。
