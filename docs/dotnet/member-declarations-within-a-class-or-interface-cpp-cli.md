# 类或接口中的成员声明 (C++/CLI)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

从 C\+\+ 托管扩展到 [!INCLUDE[cpp_current_long](../Token/cpp_current_long_md.md)]，属性和运算符的声明已经过大范围的改写，隐藏了在托管扩展设计中公开的基础实现详细信息。  还修改了事件声明。  
  
 在不具有托管扩展支持的更改类别下，现在可外联定义静态构造函数（在托管扩展中要求对其进行内联定义），而且引入了委托构造函数的概念。
