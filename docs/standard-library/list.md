# &lt;list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定义容器模板类列表和几个支持模板。  
  
## <a name="syntax"></a>语法  
  
```  
#include <list>  
```  
  
### <a name="operators"></a>运算符  
  
|||  
|-|-|  
|[运算符 ！ =](../Topic/%3Clist%3E%20operators.md#operator_neq)|测试运算符左侧的列表对象是否不等于右侧的列表对象。|  
|[运算符 <](../Topic/%3Clist%3E%20operators.md#operator_lt_)|测试运算符左侧的列表对象是否小于右侧的列表对象。|  
|[运算符 \< =](../Topic/%3Clist%3E%20operators.md#operator_lt__eq)|测试运算符左侧的列表对象是否小于或等于右侧的列表对象。|  
|[运算符 = =](../Topic/%3Clist%3E%20operators.md#operator_eq_eq)|测试运算符左侧的列表对象是否等于右侧的列表对象。|  
|[运算符 >](../Topic/%3Clist%3E%20operators.md#operator_gt_)|测试运算符左侧的列表对象是否大于右侧的列表对象。|  
|[运算符 > =](../Topic/%3Clist%3E%20operators.md#operator_gt__eq)|测试运算符左侧的列表对象是否大于或等于右侧的列表对象。|
