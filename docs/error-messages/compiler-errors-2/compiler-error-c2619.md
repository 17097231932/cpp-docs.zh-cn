# <a name="compiler-error-c2619"></a>编译器错误 C2619
“标识符”：匿名结构或联合中不允许使用静态数据成员  
  
 匿名结构或联合的成员声明为 `static`。  
  
 下面的示例生成 C2619，并演示如何通过删除 static 关键字修复此错误。  
  
```  
// C2619.cpp  
int main() {  
   union { static int j; };  // C2619  
   union { int j; };  // OK  
}  
```
