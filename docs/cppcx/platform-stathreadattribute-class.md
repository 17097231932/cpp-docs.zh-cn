# Platform::STAThreadAttribute 类
指示应用程序的线程模型是单线程单元 \(STA\)。  
  
## 语法  
  
```  
public ref class STAThreadAttribute sealed : Attribute  
```  
  
## 成员  
  
### 公共构造函数  
  
|名称|描述|  
|--------|--------|  
|[STAThreadAttribute 构造函数](../cppcx/stathreadattribute-constructor-1.md)|初始化类的新实例。|  
  
### 公共方法  
 STAThreadAttribute 属性继承自 [Platform::Object 类](../cppcx/platform-object-class.md)。 STAThreadAttribute 还会重载或具有以下成员：  
  
|名称|说明|  
|--------|--------|  
|[STAThreadAttribute::Equals](../cppcx/stathreadattribute-equals.md)|确定指定的对象是否等于当前对象。|  
|[STAThreadAttribute::GetHashCode](../cppcx/stathreadattribute-gethashcode.md)|返回此实例的哈希代码。|  
|[STAThreadAttribute::ToString](../cppcx/stathreadattribute-tostring.md)|返回表示当前对象的字符串。|
