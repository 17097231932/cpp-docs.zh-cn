# <a name="compiler-error-c2962"></a>编译器错误 C2962
语法错误 :“token”: 模板类成员函数定义应以 "}" 结尾  
  
 该标记导致模板声明中出现语法错误。  
  
 此错误可能是由分隔符不匹配引起的
