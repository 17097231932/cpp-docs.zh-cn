# 数学错误 M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不正常  
  
 运算生成一个非常小的浮点数，由于有效位丢失而无效。  不正常的浮点异常通常被屏蔽，导致异常被捕获并对其进行操作。  
  
 程序以退出代码 130 终止。
