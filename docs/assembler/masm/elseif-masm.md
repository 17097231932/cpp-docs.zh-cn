# <a name="elseif"></a>ELSEIF

将 [ELSE](else-masm.md) 和 [IF](if-masm.md) 组合到一个语句中。

## <a name="syntax"></a>语法

> **ELSEIF** *constantExpression*\
> *前瞻性*\
> ⟦**ELSE**\
> *else-语句*⟧ \
> **ENDIF**
