# CMFCVisualManagerOfficeXP Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`CMFCVisualManagerOfficeXP` 为应用程序Microsoft Office XP外观。  
  
## 语法  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## 成员  
 所有这些选件类的成员是从此选件类的上级派生的虚函数，[CMFCVisualManager Class](../../mfc/reference/cmfcvisualmanager-class.md)。  
  
## 备注  
 使用 `CMFCVisualManagerOfficeXP` 选件类更改应用程序的视觉外观类似于Microsoft Office XP。
