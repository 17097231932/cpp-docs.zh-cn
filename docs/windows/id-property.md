# <a name="id-property"></a>ID 属性

ID 属性引用在程序代码中的每个快捷键对应表项。 这是当用户按下的加速键或键组合，程序将接收的命令值。 若要使快捷键与菜单项相同，使其 Id 相同 （只要快捷键对应表的 ID 是菜单资源的 ID 相同）。
  
## <a name="requirements"></a>要求

Win32
