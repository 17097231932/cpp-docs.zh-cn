# BYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化一个字节每 `initializer`的存储。  
  
## 语法  
  
```  
[[name]] BYTE initializer [[, initializer]] ...  
```  
  
## 备注  
 ，因为类型说明符任何位置类型是非法的，也可以使用。
