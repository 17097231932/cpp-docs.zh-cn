# <a name="compiler-error-c3417"></a>编译器错误 C3417
member： 值类型不能包含用户定义的特殊成员函数  
  
 值类型不能包含如默认实例构造函数、 析构函数或复制构造函数的函数。  
  
 下面的示例生成 C3517:  
  
```  
// C3417.cpp  
// compile with: /clr /c  
value class VC {  
   VC(){}   // C3417  
  
   // OK  
   static VC(){}  
   VC(int i){}  
};  
```
