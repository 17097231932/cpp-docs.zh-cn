# <a name="ml-nonfatal-error-a2137"></a>ML 非致命错误 A2137

**调用的参数太少**

使用[INVOKE](../../assembler/masm/invoke.md)指令传递的参数数量少于所调用过程的原型中指定的所需参数的数目。
