# 链接器工具警告 LNK4039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

用 \/SECTION 选项指定的节“name”不存在  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) 或 [EDITBIN](../../build/reference/editbin-reference.md) 未能在输入文件中找到名为 `name` 的节。  
  
 若要查看文件中的节，请带 [\/HEADERS](../../build/reference/headers.md) 选项运行 DUMPBIN。
