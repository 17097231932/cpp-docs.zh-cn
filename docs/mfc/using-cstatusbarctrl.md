# <a name="using-cstatusbarctrl"></a>使用 CStatusBarCtrl

您可以使用状态栏控件（[CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)）创建一个控制窗口，该窗口反映有关应用程序的各种状态信息。 状态窗口可以分成多个部分，以显示多种类型的信息。
