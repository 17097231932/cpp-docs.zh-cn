# <a name="stock-properties-atl-control-wizard"></a>常用属性，ATL 控件向导
向导的此页可标识控件支持的常用属性。 默认情况下，没有属性进行标识。  
  
 **不支持**  
 指示当前不支持的控件的可用属性。  
  
 **支持**  
 指示当前支持的控件的属性。  
  
|传输按钮|描述|  
|---------------------|-----------------|  
|**>**|将添加到**支持**列表中当前选定的属性名称**不支持**列表。|  
|**>>**|将添加到**支持**列表中可用的所有属性名**不支持**都列表。|  
|**<**|移除中当前选定的属性名**支持**列表。|  
|**<<**|移除当前中列出的所有属性名**支持**列表。|
