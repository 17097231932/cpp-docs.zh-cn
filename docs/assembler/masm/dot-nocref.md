# <a name="nocref"></a>.NOCREF

禁止显示符号表和浏览器文件中的符号列表。

## <a name="syntax"></a>语法

> **.Nocref** ⟦*name*⟦ __，__ *name* 。⟧⟧

## <a name="remarks"></a>备注

如果指定了名称，则仅禁止给定的名称。 与相同[.XCREF](dot-xcref.md)。
