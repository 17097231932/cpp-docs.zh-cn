# 无法创建 XML 文档文件“&lt;名称&gt;”：&lt;消息&gt;
由于出现错误，无法创建 XML 文档文件。  
  
 **错误 ID：**BC42311  
  
 以下消息可能会与该错误一起出现：  
  
-   [XML 注释块不能与任何支持应用 XML 文档注释的语言元素相关联](../misc/xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments.md)。  
  
-   ["WriteOnly" 属性中不允许出现 XML 注释标记 "returns"](../misc/xml-comment-tag-returns-is-not-permitted-on-a-writeonly-property.md)。  
  
-   [XML 注释无法在一个分部 \<type\> 上应用多次](../misc/xml-comment-cannot-be-applied-more-than-once-on-a-partial-type.md)。  
  
-   ["declare sub" 语言元素中不允许出现 XML 注释标记 "returns"](../misc/xml-comment-tag-returns-is-not-permitted-on-a-declare-sub-language-element.md)。  
  
-   [XML 文档分析错误：开始标记“\<tag\>”没有匹配的结束标记](../misc/xml-documentation-parse-error-start-tag-tag-doesn-t-have-a-matching-end-tag.md)。  
  
-   [XML 注释类型形参 \<parameter\> 与相应 \<keyword\> 语句上的类型形参不匹配](../misc/xml-comment-type-parameter-parameter-does-not-match-a-type-parameter-on-the-corresponding-keyword-statement.md)。  
  
-   [XML 注释类型参数必须具有 "name" 属性](../misc/xml-comment-type-parameter-must-have-a-name-attribute.md)。  
  
-   [XML 注释异常必须具有“cref”特性](../Topic/XML%20comment%20exception%20must%20have%20a%20'cref'%20attribute.md)。  
  
### 更正此错误  
  
-   有关建议，请查看所提供的消息。
