# <a name="compiler-warning-level-1-c4506"></a>编译器警告（等级 1）C4506

内联函数 "function" 没有定义

为内联声明了给定函数并标记了该函数，但未定义它。

编译器未内联函数。

请确保用关键字声明要内联的外部函数 **`extern`** 。
