# 编译器错误 C3167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法初始化 .NET Framework: 确保它已经安装  
  
 此计算机上未安装 .NET Framework；请安装 .NET Framework。
