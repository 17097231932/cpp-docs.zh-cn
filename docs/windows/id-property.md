# ID Property
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ID 属性在程序代码中引用每一个快捷键对应表项。  这是当用户按快捷键或组合键时程序将接收的命令值。  若要使快捷键与菜单项相同，则应使它们的 ID 相同（只要快捷键对应表的 ID 与菜单资源的 ID 相同）。  
  
## 要求  
 Win32
