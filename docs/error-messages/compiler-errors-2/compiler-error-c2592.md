# <a name="compiler-error-c2592"></a>编译器错误 C2592

“class”：“base_class_2” 继承自“base_class_1”，无法重新指定

你仅可指定不是继承自其他基类的基类。 在这种情况下，仅 `base_class_1` 在规范中需要， **`class`** 因为 `base_class_1` 已继承 `base_class_2` 。
