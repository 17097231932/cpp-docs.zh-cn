# <a name="include-masm"></a>INCLUDE (MASM)

插入从给定的源文件的源代码*文件名*到程序集在当前的源文件。

## <a name="syntax"></a>语法

> 包含文件名

## <a name="remarks"></a>备注

*文件名*必须括在尖括号中，如果它包含一个反斜杠，分号，更高版本的符号，较少-比符号、 单引号或双引号。
