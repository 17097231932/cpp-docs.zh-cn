# <a name="resource-compiler-fatal-error-rc1009"></a>资源编译器错误 RC1009
编译器限制： 宏嵌套太深 macro  
  
 此文件超过了宏嵌套的资源编译器限制。 修改指定的源文件，以减少其宏的嵌套深度。
