# <a name="atexit-function-c"></a>atexit 函数 (C)
**ANSI 4.10.4.3** 如果参数的值为零、EXIT_SUCCESS 或 EXIT_FAILURE 之外的其他值，则为 `atexit` 函数返回的状态  
  
 如果成功，则 `atexit` 函数返回零；否则返回非零值。
