# <a name="ml-warning-a4014"></a>ML 警告 A4014

BSS 段不支持的说明和初始化数据

尝试在 BSS 部分中定义已初始化的数据。  BSS 部分定义为名称为 BSS 的类。  这包括简化段 `.data?`。
