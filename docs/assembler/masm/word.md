# <a name="word"></a>WORD
分配和 （可选） 为每个初始化一个字 （2 个字节） 存储的`initializer`。 此外可作为类型说明符类型是合法的任何位置。  
  
## <a name="syntax"></a>语法  
  
```  
[[name]] WORD initializer [[, initializer]]...  
```
