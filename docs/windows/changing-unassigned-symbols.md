# Changing Unassigned Symbols
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在[“资源符号”对话框](../windows/resource-symbols-dialog-box.md)中，你可以编辑或删除现有的且尚未分配给资源或对象的符号。  
  
### 更改未分配的符号  
  
1.  在**“名称”**框中，选择未分配的符号，然后单击**“更改”**。  
  
2.  编辑**“更改符号”**对话框中提供的框中的符号名称或值。  
  
    > [!NOTE]
    >  若要更改*已*分配给资源或对象的符号，必须使用资源编辑器或“属性”窗口。  有关详细信息，请参阅[更改符号或符号名称](../windows/changing-a-symbol-or-symbol-name-id.md)。  
  
 有关将资源添加到托管项目的信息，请参阅“.NET Framework 开发员指南”中的[应用程序中的资源](../Topic/Resources%20in%20Desktop%20Apps.md)。  
  
 要求  
  
 Win32
