# <a name="dt"></a>DT

分配并根据需要为每个初始化的存储空间的 10 个字节`initializer`。 也可以用作类型说明符类型是合法的任何位置。 `DT` 是的同义词[TBYTE](../../assembler/masm/tbyte.md)。

## <a name="syntax"></a>语法

> [[name]]DT 初始值设定项 [[，初始值设定项]]...
