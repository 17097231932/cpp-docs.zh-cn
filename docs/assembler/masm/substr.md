# <a name="substr"></a>SUBSTR

返回*textitem*的子字符串，从该*位置*开始。 *Textitem*可以是文本字符串、前面带有 `%`的常量或宏函数返回的字符串。

## <a name="syntax"></a>语法

> *name* **SUBSTR** *textitem* __，__ *position* ⟦ __，__ *length*⟧
