# <a name="for"></a>FOR

将为每个*参数*重复一次的块标记为使用当前*参数*替换每个重复的*参数*。

## <a name="syntax"></a>语法

> **对于***参数*⟦ **：** 请求 |  __： =__ *default*⟧ **，\<** _参数_⟦ __，__ *argument* .。。⟧ **>** \
> *语句*\
> **ENDM**

## <a name="remarks"></a>备注

与[IRP](irp.md)相同。
