# “&lt;项目名&gt; 的项目加载警告”对话框
如果没有正确加载 IDE 中已打开项目的设置，则将显示此对话框。  
  
 **请更正指定的错误，然后重新加载项目**  
 提供有关出现的错误的信息，如无效的路径或文件名。  若要解决错误，请从项目的属性页中移除或替换所有无效的引用。  保存项目，然后在 IDE 中重新加载该项目。  
  
 **在此会话中不再显示该对话框**  
 如果选择该选项，则在下次打开 IDE 之前\<Projectname\>对话框将不再显示。
