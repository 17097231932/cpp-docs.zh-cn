# <a name="ml-warning-a4012"></a>ML 警告 A4012

**段的行号信息没有类 "CODE"**

段中的说明中没有以 "CODE" 结尾的类名。 汇编程序未生成这些说明的 CodeView 信息。

CodeView 无法处理包含代码的模块，这些段中的类名称不以 "CODE" 结尾。
