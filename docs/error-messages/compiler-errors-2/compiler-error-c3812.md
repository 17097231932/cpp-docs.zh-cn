# <a name="compiler-error-c3812"></a>编译器错误 C3812
属性必须为属性声明中的第一个标记  
  
 声明属性时`__property`关键字必须是在行上的第一个标记。  
  
 C3812 才可访问使用过时的编译器选项**/clr:oldSyntax**。
