# ML Nonfatal Error A2039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**行太长**  
  
 源文件行已超过 512 个字符的限制。  
  
 如果多个物理行连接以行继续符 \(\\\)，则发生的逻辑行仍被限制为 512 个字符。
