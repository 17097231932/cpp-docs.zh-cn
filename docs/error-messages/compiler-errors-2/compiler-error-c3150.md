# <a name="compiler-error-c3150"></a>编译器错误 C3150

"element"： "attribute" 只能应用于类、接口、数组或指针

**`__gc`** 只能在类、接口或数组上使用。

只能使用过时的编译器选项 **/clr： oldSyntax**来访问 C3150。
