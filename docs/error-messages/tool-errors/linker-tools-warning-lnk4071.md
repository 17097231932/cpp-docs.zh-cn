# <a name="linker-tools-warning-lnk4071"></a>链接器工具警告 LNK4071

无法增量链接到后面的链接上

LINK 找到一个或多个符号的多个定义，但[/force](../../build/reference/force-force-file-output.md)或 **/FORCE：多个**用于创建输出文件，而不考虑错误。 链接已删除增量状态（. .ilk）文件。
