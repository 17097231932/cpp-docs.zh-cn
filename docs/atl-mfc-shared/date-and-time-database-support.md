# Date and Time: Database Support
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

自4.0版中，MFC数据库编程的使用 [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) 和 [COleDateTimeSpan](../atl-mfc-shared/reference/coledatetimespan-class.md) 选件类表示日期和时间数据。  这些选件类，还用于自动化，从选件类 [COleVariant](../mfc/reference/colevariant-class.md)派生。  他们发现 [CTime](../atl-mfc-shared/reference/ctime-class.md) 和 [CTimeSpan](../atl-mfc-shared/reference/ctimespan-class.md)提供管理的日期和时间数据更好的支持。
