# .FARDATA?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

当使用 [.MODEL](../../assembler/masm/dot-model.md)，开始每更数据段为未初始化的数据 \(段名 FAR\_BSS 或 *名称*\)。  
  
## 语法  
  
```  
  
.FARDATA? [[name]]  
```
