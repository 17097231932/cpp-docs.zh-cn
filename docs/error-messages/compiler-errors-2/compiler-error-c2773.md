# <a name="compiler-error-c2773"></a>编译器错误 C2773
\#导入和 #using 仅在 c + + 编译器中可用  
  
 C 编译器不能识别`#import`预处理器指令。 编译为 c + + 源文件。 使用[/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md)如有必要。
