# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
未能生成对特定项目的项目到项目的引用。  
  
 引用生成 .exe 文件的项目是此错误的最常见的原因。  此错误还指示某项目引用了其输出不是程序集（例如，本机 C\+\+ 应用程序）的项目。
