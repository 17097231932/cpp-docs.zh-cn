# <a name=""></a>\@\@：

定义只能在*label1*和*label2*之间识别的代码标签，其中*label1*是代码的开头或上一个 `@@:` 标签，而*label2*是代码的结尾或下一个 `@@:` 标签。 请参阅[\@B](../../assembler/masm/at-b.md)和[\@F](../../assembler/masm/at-f.md)。

## <a name="syntax"></a>语法

> **\@\@：**
