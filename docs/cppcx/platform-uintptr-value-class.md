# <a name="platformuintptr-value-class"></a>Platform::UIntPtr 值类
表示其大小适合平台（32 位或 64 位）的无符号指针。  
  
## <a name="syntax"></a>语法  
  
```cpp  
public value struct UintPtr  
```
