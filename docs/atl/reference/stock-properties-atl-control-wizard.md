# 常用属性，ATL 控件向导
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

此向导页标识控件支持的常用属性。  默认情况下，不标识属性。  
  
 **不支持**  
 指示控件当前不支持的可用属性。  
  
 **是否支持**  
 指示控件当前支持的属性。  
  
|传输按钮|说明|  
|----------|--------|  
|**\>**|向“支持”列表中添加“不支持”列表中当前选定的属性名。|  
|**\>\>**|向“支持”列表中添加“不支持”列表中可用的所有属性名。|  
|**\<**|移除“支持”列表中当前选定的属性名。|  
|**\<\<**|移除“支持”列表中当前列出的所有属性名。|
