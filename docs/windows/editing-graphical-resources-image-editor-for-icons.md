# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

图形资源是为应用程序定义的图像。  可以手工绘制或使用形状绘制。  可以选择图像的某些部分以编辑、翻转或调整大小，或者可以从图像的选定部分创建自定义画笔并用此画笔进行绘制。  可以定义图像属性、以不同格式保存图像以及将图像从一种格式转换为另一种格式。  
  
 除创建新的图形资源外，还可以[导入现有资源](../windows/how-to-import-and-export-resources.md)进行编辑，然后将它们添加到项目中。  还可以打开和编辑不属于项目的图像进行[独立运行图像编辑](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md)。
