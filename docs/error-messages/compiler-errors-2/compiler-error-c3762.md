# 编译器错误 C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法处理特性“attribute”  
  
 从 `System.Security.Permissions.SecurityAttribute` 中继承的用户定义的特性正被用于定义安全特性。  这种特性不能在定义它的同一程序集内使用。
