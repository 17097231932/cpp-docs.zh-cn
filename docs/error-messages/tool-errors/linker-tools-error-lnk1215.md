# 链接器工具错误 LNK1215
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

元数据操作失败 \(HRESULT\): 错误  
  
 链接器在尝试通过 .NET 运行时执行元数据更新时收到 .NET 运行时错误。  
  
 `HRESULT` 是 .NET 运行时方法中的 HRESULT，  而 `error` 是 .NET 提供的文本。  
  
 链接器和 .NET 运行时可能不匹配；重新安装 Visual C\+\+。
