# 关于异常的疑难解答：System.Data.SqlServerCe.SqlCeTransactionInProgressException
当应用程序在另一个事务正在进行期间尝试修改数据库时，将发生 `System.Data.SqlServerCe.SqlCeTransactionInProgressException` 异常。 有关详细信息，请参阅 [SQL Server Compact 3.5 类库](http://go.microsoft.com/fwlink/?LinkID=102595)。
