# <a name="linker-tools-error-lnk1166"></a>链接器工具错误 LNK1166

不能调整代码偏移量位置处 = 偏移量，va = value

链接无法填充所需的代码。

某些指令不能在某些处理器中跨页边界。 链接尝试将添加填充代码来纠正这种情况。 在这种情况下，链接不可能解决此问题。
