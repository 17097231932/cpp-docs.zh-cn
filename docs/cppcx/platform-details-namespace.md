# <a name="platformdetails-namespace"></a>Platform::Details 命名空间

此命名空间仅供内部使用，不用于开发。

## <a name="syntax"></a>语法

```cpp
namespace Platform {
   namespace Details {
}}
```

### <a name="members"></a>成员

虽然此命名空间供内部使用，但是浏览器可以显示此命名空间的以下成员。

|名称|备注|
|----------|------------|
|控制台|类。 在单位测试中显示输出。|
|_GUID|结构|
|堆|类|
|HeapAllocationTrackingLevel|枚举|
|HeapEntryHandler|委托|
|IActivationFactory|接口|
|IAgileObject|接口|
|IClassFactory|接口|
|IEquatable|接口|
|IPrintable|接口|
|IWeakReference|接口|
|IWeakReferenceSource|接口|
