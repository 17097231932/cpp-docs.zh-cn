# <a name="compiler-error-c2345"></a>编译器错误 C2345

align(value)：非法的对齐值

向 [align](../../cpp/align-cpp.md) 关键字传递了允许范围之外的值。

下面的代码生成 C2345

```
// C2345.cpp
// compile with: /c
__declspec(align(0)) int a;   // C2345
__declspec(align(1)) int a;   // OK
```
