# <a name="compiler-warning-level-4-c4639"></a>编译器警告（等级 4）C4639

MSXML 错误，将不会处理 XML 文档注释。 reason

出现此警告的原因有很多。

若要解决此警告：

- 重新编译.

- 通过重新安装公共语言运行时重新安装 MSXML。

- 编辑或删除导致警告和重新编译的文档注释。

发出 C4639 时，将禁用所有进一步的 XML 注释处理，并且不生成 .xdc 文件。
