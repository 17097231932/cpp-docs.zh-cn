# <a name="public-masm"></a>PUBLIC (MASM)

使指定为*名称*的每个变量、标签或绝对符号可用于程序中的所有其他模块。

## <a name="syntax"></a>语法

> **PUBLIC** ⟦*language-type*⟧ *name* ⟦ __，__ ⟦*language-type*⟧ *name* .。。⟧

## <a name="remarks"></a>备注

*语言类型*参数仅在32位 MASM 中有效。
