# <a name="resource-compiler-error-rc2167"></a>资源编译器错误 RC2167

无法识别的 VERSIONINFO 字段;应为开始或逗号

在 **VERSIONINFO** 结构声明的 **固定** 部分中找到了无法识别的字段。 **VERSIONINFO** 字段必须是由逗号分隔的 dword。
