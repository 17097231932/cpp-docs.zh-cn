# Building and Testing the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

按照 [插入复合控件](../atl/inserting-a-composite-control.md)所述，其中一个项目的初始元素是承载您的新复合控件的默认HTML页。  在完成修改该复合控件后，单击 **生成解决方案** 或 **Rebuild Solution** 从 **生成** 菜单。  对于项目成功生成，加载HTML页，位于您的复合控件项目根目录，对于Internet Explorer或其他浏览器和测试您的控件的功能。  
  
 使用测试容器工具，还可以测试您的复合控件，或者可以承载ActiveX控件的其他应用程序。  有关如何访问测试容器的信息，请参见[用测试容器测试属性和事件](../mfc/testing-properties-and-events-with-test-container.md)。
