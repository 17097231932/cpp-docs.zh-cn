# <a name="ml-nonfatal-error-a2034"></a>ML 非致命错误 A2034
**必须在段块**  
  
 段块之外找到以下项之一：  
  
-   指令  
  
-   标签定义  
  
-   A [THIS](../../assembler/masm/operator-this.md)运算符  
  
-   A [ $ ](../../assembler/masm/dollar.md)运算符  
  
-   过程定义  
  
-   [ALIGN](../../assembler/masm/align-masm.md)指令  
  
-   [ORG](../../assembler/masm/org.md)指令
