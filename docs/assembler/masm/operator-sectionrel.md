# <a name="operator-sectionrel"></a>运算符 SECTIONREL

返回表达式相对于包含最终可执行文件中的目标的部分的部分相对偏移量。

## <a name="syntax"></a>语法

> SECTIONREL 表达式

## <a name="remarks"></a>备注

仅支持 COFF 对象发出 SECTIONREL。
