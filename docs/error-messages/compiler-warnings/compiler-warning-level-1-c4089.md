# <a name="compiler-warning-level-1-c4089"></a>编译器警告 （等级 1） C4089
function： 不同类型中实际参数 number 形参数字  
  
 对应的形参和实参参数具有不同的类型。 实际的参数传递时未经更改。 该函数将转换为函数定义中指定的类型实参。
