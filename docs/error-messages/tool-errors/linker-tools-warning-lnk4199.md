# 链接器工具警告 LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

已忽略 \/DELAYLOAD:dllname；未找到来自 dllname 的导入  
  
 链接器忽略 `dllname`，因为它不需要 `dllname` 导出的任何函数。
