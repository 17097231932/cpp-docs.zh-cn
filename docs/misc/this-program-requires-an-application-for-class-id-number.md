# This program requires an application for class ID &lt;number&gt;.
Visual Studio 未能找到您尝试使用的应用程序。  此消息是一个操作系统错误。  
  
### 更正此错误  
  
1.  重新安装受影响的应用程序并重试。  
  
     \- 或 \-  
  
     重新安装 Visual Studio，然后重试。
