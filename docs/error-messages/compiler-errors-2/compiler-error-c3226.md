# <a name="compiler-error-c3226"></a>编译器错误 C3226

泛型声明内不允许出现模板声明

在泛型类中使用泛型声明。

以下示例生成 C3226：

```cpp
// C3226.cpp
// compile with: /clr
generic <class T>
ref class C {
   template <class T1>   // C3226
   ref struct S1 {};
};
```

以下示例演示了可能的解决方法：

```cpp
// C3226b.cpp
// compile with: /clr /c
generic <class T>
ref class C {
   generic <class T1>
   ref struct S1 {};
};
```
