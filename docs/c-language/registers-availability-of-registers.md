# <a name="registers-availability-of-registers"></a>寄存器：寄存器的可用性
**ANSI 3.5.1** 通过使用寄存器存储类说明符可将对象实际放置在寄存器中的范围  
  
 编译器不接受对寄存器变量的用户请求。 相反，在进行优化时，它将自行做出选择。
