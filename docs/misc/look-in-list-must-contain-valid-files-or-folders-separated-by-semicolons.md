# Look In list must contain valid files or folders, separated by semicolons.
当“查找范围”列表中列出的目录或文件包含无效字符时，通常会发生此错误。  
  
### 更正此错误  
  
1.  使用“查找范围”对话框定位到要搜索的目录或文件名，而不是键入路径或文件名。  通过选择“查找范围”列表旁边的省略号按钮，可访问“查找范围”对话框。
