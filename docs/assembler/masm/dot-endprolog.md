# .ENDPROLOG
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

通知 Prolog 声明的结尾。  
  
## 语法  
  
```  
.ENDPROLOG  
```  
  
## 备注  
 它是使用任何一个的错误 \(声明在 [PROC](../../assembler/masm/proc.md) 框架和 .ENDPROLOG 之间的界限。  
  
 有关更多信息，请参见 [MASM for x64 \(ml64.exe\)](../../assembler/masm/masm-for-x64-ml64-exe.md)。
