# 资源编译器错误 RC2147
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

SUBLANGUAGE ID 不是数字  
  
 SUBLANGUAGE ID 值必须是数字。  
  
 **LANGUAGE** 语句必须使用以下语法：  
  
 **LANGUAGE** *primary\_language\_ID*,*secondary\_language\_ID*  
  
 有效的 SUBLANGUAGE ID 被定义为 WINNT.h 文件中的 **SUBLANG\_** 常数。
