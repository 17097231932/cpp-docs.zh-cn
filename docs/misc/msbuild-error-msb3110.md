# MSBuild 错误 MSB3110
**MSB3110: 程序集“\<assembly\>”与标识“\<AssemblyIdentity\>”不匹配，文件名应为:“\<file\>”。**  
  
 此错误会在从程序集标识获取的程序集文件名与实际文件名不匹配时产生。  
  
### 更正此错误  
  
-   更正程序集文件名，使其与程序集标识匹配。
