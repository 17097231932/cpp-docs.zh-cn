# <a name="auto_search-import-attribute"></a>auto_search 导入属性

**C++相关**

指定在引用`#import`类型库时, 它本身引用另一个类型库, 编译器可以为其他类型库执行隐`#import`式转换。

## <a name="syntax"></a>语法

> **#import***类型库***auto_search**
