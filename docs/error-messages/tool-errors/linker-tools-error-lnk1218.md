# <a name="linker-tools-error-lnk1218"></a>链接器工具错误 LNK1218

视为错误的警告;未生成输出文件

链接到 **/wx**后，所有链接器警告将被视为错误，并且不会创建任何输出文件。

有关详细信息，请参阅[/wx （将链接器警告视为错误）](../../build/reference/wx-treat-linker-warnings-as-errors.md) 。
