# 关于异常的疑难解答：System.IdentityModel.Tokens.SecurityTokenValidationException
当接收的安全标记无效时，将引发 <xref:System.IdentityModel.Tokens.SecurityTokenValidationException> 异常。 该异常通常由 <xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken%2A> 方法引发。
