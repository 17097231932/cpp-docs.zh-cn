# <a name="resource-compiler-error-rc2124"></a>资源编译器错误 RC2124

不允许空菜单

在 **菜单** 语句中定义任何菜单项之前，会出现一个 **END** 关键字。 资源编译器不允许空菜单。 请确保 **MENU** 语句中没有任何左引号。
