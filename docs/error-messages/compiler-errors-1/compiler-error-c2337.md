# <a name="compiler-error-c2337"></a>编译器错误 C2337

“attribute name”：未找到特性

你使用了此版本的 Visual c++ 不支持的特性。

以下示例生成 C2337：

```
// C2337.cpp
// compile with: /c
[emitidl];
[module(name="x")];
[grasshopper]   // C2337, not a supported attribute
class a{};
```
