# <a name="linker-tools-error-lnk1215"></a>链接器工具错误 LNK1215

元数据操作失败 (HRESULT): 错误

链接器尝试进行通过.NET 运行时元数据更新时收到的.NET 运行时错误。

`HRESULT` 是从.NET 运行时方法的 HRESULT。 `error` 是。NET 提供文本。

您可能具有不匹配的链接器和.NET 运行时;重新安装 Visual C++。
