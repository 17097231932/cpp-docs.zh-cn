# <a name="expression-evaluator-error-cxx0041"></a>表达式计算器错误 CXX0041

类元素必须是静态成员或成员函数

非静态成员的类 （或结构或联合） 而无需指定的实例化的类用于使用。

仅静态数据成员或成员函数可以使用而无需指定实例化。

此错误是与 CAN0041 相同。
