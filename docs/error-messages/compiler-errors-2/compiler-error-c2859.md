# 编译器错误 C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename 不是创建该预编译头时使用的类型文件，请重新创建预编译头。  
  
 必须一起创建项目数据库和预编译头文件，以确保信息的一致性。  重新生成项目以便重新创建预编译头。  
  
 有关预编译头的信息，请参见 [\/Y（预编译头）](../../build/reference/y-precompiled-headers.md)。
