# <a name="adding-an-mfc-class"></a>添加 MFC 类

若要将从 Microsoft 基础类派生的类添加到项目 (MFC) 库类，请使用 [类向导](mfc-class-wizard.md)中的 "**添加类**" 按钮。 指定新类的名称，选择基类，并选择与 (关联的对话框的 ID （如果有任何) ）。 向导将创建一个标头文件和一个实现文件并将其添加到你的项目中。

> [!NOTE]
> 如果最初 [使用 mfc 支持创建了应用程序](../../atl/reference/mfc-support-in-atl-projects.md)，则可以将 MFC 类添加到 ATL COM 应用程序。 还可以将 MFC 类添加到具有 MFC 支持的 Win32 项目。

### <a name="to-add-an-mfc-class-to-your-project"></a>向项目添加 MFC 类

在类视图中，右键单击项目名称。 单击 " **类向导** "，然后单击 " **添加类** " 以打开 " [添加 MFC 类](mfc-add-class-wizard.md) " 对话框。 使用 **类向导** 添加消息处理程序或新方法。
