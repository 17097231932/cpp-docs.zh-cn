# Unable to read the resource information from the resx file
项目系统未能读取 .resx 文件。  
  
 操作系统特定的原因（如果有的话）将追加到此消息的结尾。  同时还追加未能读取的文件的文件名。  
  
 此错误的可能原因还包括缺少读取此文件的权限，或（对于 Web 项目）失去与 Web 服务器之间的通信。  
  
 此错误将使生成进程失败。
