# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配 *textitem* 到的名称。   *textitem* 可以是一个文本字符串、 `%`后的常数和宏函数返回的字符串。  
  
## 语法  
  
```  
  
name TEXTEQU [[textitem]]  
```
