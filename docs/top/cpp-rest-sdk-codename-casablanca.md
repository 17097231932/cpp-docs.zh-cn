# C++ REST SDK (Codename &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

通过使用 C\+\+ REST SDK \(codename "Casablanca"\)，你可以更轻松地写入与基于云的服务连接的现代异步 C\+\+ 代码。  
  
 Casablanca 项目不再附带 Visual Studio。  文档和资源在 CodePlex 上进行托管并偶尔进行更新。  有关本项目的更多详细信息，请参阅 [C\+\+ REST SDK \(codename "Casablanca"\)](http://casablanca.codeplex.com/)。
