这些类使您能够与使用 Windows 套接字或 Win32 Internet (WinInet) 的其他计算机交换信息。 还有一组类用于操作 Windows 套接字。

下列类别的类支持连接。

- [Windows 套接字类](../mfc/windows-sockets-classes.md)

- [Win32 Internet 类](../mfc/win32-internet-classes.md)
