# operator :
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

重写 *表达式的* 默认段与 `segment`的。  `segment` 可以是段寄存器、组名称、段名或段落表达式。  该 *表达式* 必须为常量。  
  
## 语法  
  
```  
  
segment  
: expression  
```
