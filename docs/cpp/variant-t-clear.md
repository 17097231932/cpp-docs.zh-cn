# _variant_t::Clear
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 专用**  
  
 清除封装的 **VARIANT** 对象。  
  
## 语法  
  
```  
  
void Clear( );  
  
```  
  
## 备注  
 对封装的 **VARIANT** 对象调用 **VariantClear**。
