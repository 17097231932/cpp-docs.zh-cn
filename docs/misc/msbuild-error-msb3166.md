# MSBuild 错误 MSB3166
**MSB3166: 未能找到项“\<程序包\>”所需的文件“\<文件\>”。**  
  
 缺少最终用户许可协议 \(EULA\) 或外部检查文件时会产生此错误。  确保磁盘上存在该文件。  
  
### 更正此错误  
  
-   重新安装 Visual Studio SDK  
  
     \- 或 \-  
  
-   将所需的文件复制到相应的位置。
