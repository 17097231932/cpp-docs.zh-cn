# 文件名不存在、无效或包含无法验证的参数。 是否更改你的输入?
在向“添加程序”对话框中输入不正确的路径或文件名时通常发生该错误。  
  
### 更正此错误  
  
1.  选择“浏览”以导航到正确的文件。  
  
     \- 或 \-  
  
2.  重新键入路径和文件名。
