# 编译器错误 C2088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“operator”: 对于“class\-key”非法  
  
 没有为结构或联合定义该运算符。  该错误只对 C 代码有效。  
  
 下面的示例生成三次 C2088：  
  
```  
// C2088.c  
struct S {  
   int m_i;   
} s;  
  
int main() {  
   int i = s * 1;   // C2088  
   struct S s2 = +s;   // C2088  
   s++;   // C2088  
}  
```
