## <a name="compiler-supplied-namespaces"></a>编译器提供的命名空间

若要简化的 Windows 运行时，为目标的程序代码C++/CX 编译器和及其支持的标头文件提供定义多种类型的命名空间。 命名空间定义内置数值类型;字符串、 数组和集合;VisualC++表示 Windows 运行时错误; 异常和标准的 Windows 运行时类型的特定于语言的增强功能。
