# XML 注释类型参数必须具有 &quot;name&quot; 属性
XML 注释类型参数必须具有 "name" 属性。 将忽略 XML 注释。  
  
 XML 注释标记需要 `name` 特性，但该特性缺失或不完整。  
  
 **错误 ID：**BC42318  
  
### 更正此错误  
  
-   向标记添加或补充所需的 `name` 特性。
