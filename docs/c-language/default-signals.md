# <a name="default-signals"></a>默认信号
**ANSI 4.7.1.1** 调用信号处理程序前未执行 signal (sig, SIG_DFL) 的等效项时，对执行的信号的阻止  
  
 程序开始运行时，信号将设置为其默认状态。
