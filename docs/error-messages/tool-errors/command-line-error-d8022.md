# 命令行错误 D8022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法打开“messagefile”  
  
 给定文件不在当前目录或 PATH 环境变量所指定的目录中。  该消息文件包含编译器命令行语法和选项的简要总结。  
  
 将该文件移动到当前目录或当前路径下的目录中。  如果找不到该文件，则运行 SETUP 程序从分发磁盘上复制它。
