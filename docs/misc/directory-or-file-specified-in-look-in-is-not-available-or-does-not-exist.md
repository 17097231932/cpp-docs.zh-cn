# Directory or file specified in Look In is not available or does not exist.
该错误通常发生在找不到“浏览”列表中指定的路径或文件名时。  
  
### 更正此错误  
  
1.  重新输入路径和文件名。  
  
     \- 或 \-  
  
2.  用“浏览”列表和“可用项”框定位到正确的文件或文件夹。
