# 编译器错误 C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier1”: C 链接函数无法返回 C\+\+ 类“identifier2”  
  
 用 C 链接定义的函数无法返回用户定义的类型。
