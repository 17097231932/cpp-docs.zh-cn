# File is currently in use and cannot be overwritten.
无法保存当前文件。  该错误通常发生在尝试用同时处于打开状态的文件的名称保存一个文件时。  
  
### 更正此错误  
  
1.  关闭同名文件并重试。  
  
     \- 或 \-  
  
     重命名该文件然后保存它。
