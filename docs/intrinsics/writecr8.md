# <a name="writecr8"></a>__writecr8
**Microsoft 专用**  
  
 将值`Data`CR8 注册。  
  
## <a name="syntax"></a>语法  
  
```  
void writecr8(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>参数  
 [in] `Data`  
 要写入的 CR8 寄存器的值。

## <a name="remarks"></a>备注
