# “打开 &lt;设备&gt; 图像”对话框（图标的图像编辑器）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用此对话框打开设备图像。  它列出当前资源中的现有设备图像（即作为当前资源一部分的图像）。  
  
 **当前图像**  
 列出包括在资源中的图像。  选择要打开的图像类型。  
  
## 要求  
 无
