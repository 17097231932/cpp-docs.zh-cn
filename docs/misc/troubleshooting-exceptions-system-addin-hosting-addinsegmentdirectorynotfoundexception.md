# 关于异常的疑难解答：System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
当管线目录结构中缺少段目录时，将引发 <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> 异常。  
  
## 备注  
 当激活外接程序时，将构造外接程序到宿主的管线。 外接程序系统要求将管线段置于具有指定目录名称的指定目录结构中，如[管线开发要求](http://msdn.microsoft.com/zh-cn/ef9fa986-e80b-43e1-868b-247f4c1d9da5)中所述。
