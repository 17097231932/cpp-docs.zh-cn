# <a name="readeflags"></a>__readeflags
读取程序状态和控制 (EFLAGS) 注册。  
  
## <a name="syntax"></a>语法  
  
```  
unsigned     int __readeflags(void);  
unsigned __int64 __readeflags(void);  
```  
  
## <a name="return-value"></a>返回值  
 EFLAGS 寄存器的值。 返回值为 32 位长时间在 32 位平台上和 64 位长时间在 64 位平台上。  
  
## <a name="remarks"></a>备注  
 这些例程只能用作内部函数不可用。
