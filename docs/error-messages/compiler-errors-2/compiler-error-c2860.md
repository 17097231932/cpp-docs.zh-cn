# <a name="compiler-error-c2860"></a>编译器错误 C2860
void 不能是自变量类型，除外 (void)  
  
 类型`void`不能用作参数类型与其他参数。  
  
 下面的示例生成 C2860:  
  
```  
// C2860.cpp  
// compile with: /c  
void profunc1(void, int i);   // C2860  
void func10(void);   // OK  
```
