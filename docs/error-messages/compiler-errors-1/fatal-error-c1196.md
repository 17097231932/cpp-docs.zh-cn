# 错误 C1196
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：在类型库“typelib”中找到的标识符不是有效的 C\+\+ 标识符  
  
 类型库中有一个标识符不是有效的 C\+\+ 标识符。 类型库不可与 [\#import](../../preprocessor/hash-import-directive-cpp.md) 一起使用。
