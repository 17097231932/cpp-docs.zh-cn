# <a name="linker-tools-error-lnk1189"></a>链接器工具错误 LNK1189

LIBTOOMANYMEMBERS:: 已超出数字对象的限制的库

已超出库中 65535 个对象或成员的限制。

若要解决此问题，请将你的库分为两个或多个使用更少对象或成员的库。
