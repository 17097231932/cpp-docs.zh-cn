# <a name="linker-tools-error-lnk2023"></a>链接器工具错误 LNK2023

错误的 dll 或入口点 \<dll or entry point>

链接器正在加载 msobj90.dll 的不正确版本。 请确保路径中的 link.exe 和 msobj90.dll 具有相同的版本。

msobj90.dll 的依赖项可能不存在。 msobj90.dll 的依赖项列表为：

- Msvcr90.dll

- Kernel32.dll

检查计算机中是否有任何其他可能过期的 msobj90.dll 副本。
