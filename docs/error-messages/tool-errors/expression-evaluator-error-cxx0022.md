# <a name="expression-evaluator-error-cxx0022"></a>表达式计算器错误 CXX0022
_main 前的函数调用  
  
 调试器已进入函数之前，C 表达式计算器无法评估函数 **_main**。 该程序未正确初始化直到 **_main**已调用。  
  
 此错误是与 CAN0022 相同。
