# operator LROFFSET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回偏移量 *表达式*。  和 **偏移量**相同，但是，它生成一加载程序解析的偏移量，允许窗口定位代码段。  
  
## 语法  
  
```  
  
LROFFSET expression  
```
