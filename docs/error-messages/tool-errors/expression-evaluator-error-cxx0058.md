# 表达式计算器错误 CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未找到重载运算符  
  
 类类型被指定为了表达式中的左操作数，但没有为该类定义重载运算符。  
  
 该错误与 CAN0058 相同。
