# <a name="stock-properties-atl-control-wizard"></a>常用属性，ATL 控件向导

向导的此页标识控件支持的常用属性。 默认情况下，不会标识任何属性。

- **不支持**

   指示控件当前不支持的可用属性。

- **支持**

   指示控件当前支持的属性。

|传输按钮|说明|
|---------------------|-----------------|
|**>**|将当前选择的属性名称添加到 **受支持****的列表中**。|
|**>>**|将所有属性名称添加到 **受支持****的列表** 中。|
|**\<**|删除 **受支持** 列表中当前选定的属性名称。|
|**\<\<**|删除 **受支持** 列表中当前列出的所有属性名称。|
