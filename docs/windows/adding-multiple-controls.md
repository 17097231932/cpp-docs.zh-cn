# <a name="adding-multiple-controls"></a>添加多个控件
### <a name="to-add-multiple-controls"></a>若要添加多个控件  
  
1.  按住**CTRL**密钥，选择一个控件中的[工具箱窗口](/visualstudio/ide/reference/toolbox)。  
  
2.  版本**CTRL**密钥，然后单击你想要添加特定的控件多次对话框。  
  
3.  按**ESC**停止放置控件。
