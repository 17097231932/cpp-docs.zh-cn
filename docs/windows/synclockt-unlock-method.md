# <a name="synclocktunlock-method"></a>SyncLockT::Unlock 方法

支持 WRL 基础结构，不应在代码中直接使用。

## <a name="syntax"></a>语法

```cpp
void Unlock();
```

## <a name="remarks"></a>备注

释放当前占用的资源的控制**SyncLockT**对象，如果有的话。
