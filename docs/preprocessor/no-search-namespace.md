# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 专用**  
  
 与 [no\_namespace](../preprocessor/no-namespace.md) 特性的功能相同，但在类型库中使用时，可以将 \#import 指令与 [auto\_search](../preprocessor/auto-search.md) 特性结合使用。  
  
## 语法  
  
```  
no_search_namespace  
```  
  
## 备注  
 **结束 C\+\+ 专用**
