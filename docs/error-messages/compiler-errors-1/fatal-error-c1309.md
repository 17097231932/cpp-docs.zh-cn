# <a name="fatal-error-c1309"></a>错误 C1309

C2.DLL 和 PGODB 的版本不匹配 \<version> 。.DLL

用于生成和使用 [按配置文件优化](../../build/profile-guided-optimizations.md) 的工具集包含不匹配的组件。  如果无法手动解决此错误，请重新安装 Visual C++。
