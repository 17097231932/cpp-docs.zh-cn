# <a name="linker-tools-warning-lnk4039"></a>链接器工具警告 LNK4039
节使用 /SECTION 选项指定 name 不存在  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md)或[EDITBIN](../../build/reference/editbin-reference.md)找不到名为一节`name`输入文件中。  
  
 若要查看文件的部分内容，请运行与 DUMPBIN [/HEADERS](../../build/reference/headers.md)选项。
