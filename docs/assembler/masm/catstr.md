# CATSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

连接文本项。  
  
## 语法  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## 备注  
 每个文本项可以是一个文本字符串、 `%`后的常数和宏函数返回的字符串。
