# <a name="largest-array-size"></a>最大数组大小

**ANSI 3.3.3.4、4.1.1** 保存数组的最大大小所需的整数类型，即 **size_t** 的大小

`size_t` typedef 是 32 位 x86 平台上的一个 `unsigned int`。 在 64 位平台上，`size_t` typedef 是 `unsigned __int64`。
