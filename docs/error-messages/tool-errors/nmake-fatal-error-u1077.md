# <a name="nmake-fatal-error-u1077"></a>NMAKE 错误 U1077
程序： 返回代码 value  
  
 由 NMAKE 失败，返回给定的退出代码的调用，给定的命令或程序。  
  
 若要禁止显示此错误并继续 NMAKE 会话，请使用 /I 选项、 **。忽略**点指令或短划线 (**-**) 命令修饰符。 若要继续 NMAKE 会话不相关的依赖关系树的部分，使用 /K 选项。
