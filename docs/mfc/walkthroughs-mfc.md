# <a name="walkthroughs-mfc"></a>演练 (MFC)
本节包含的文章为你演练与新的 MFC 库功能相关的各种任务。  
  
## <a name="in-this-section"></a>本节内容  
 [演练：使用新的 MFC Shell 控件](../mfc/walkthrough-using-the-new-mfc-shell-controls.md)  
 在本演练中，您将创建一个类似文件资源管理器的应用程序。 你将创建包含两个窗格的窗口。 左窗格中将包含[CMFCShellTreeCtrl](../mfc/reference/cmfcshelltreectrl-class.md)对象将在层次结构视图中显示你的桌面。 右窗格中将包含[CMFCShellListCtrl](../mfc/reference/cmfcshelllistctrl-class.md) ，将显示在左窗格中选择的文件夹中的文件。  
  
 [演练：将控件置于工具栏上](../mfc/walkthrough-putting-controls-on-toolbars.md)  
 现代 MFC 工具栏可以承载除简单按钮以外的控件。 此文章介绍了如何执行此操作。  
  
 [演练：向 MFC 项目添加 D2D 对象](../mfc/walkthrough-adding-a-d2d-object-to-an-mfc-project.md)  
 演示如何向项目添加 D2D 对象，并介绍了如何使用 D2D。  
  
 [演练：向 MFC 项目添加动画](../mfc/walkthrough-adding-animation-to-an-mfc-project.md)  
 演示如何将图形对象添加到项目，并介绍了如何对它进行动画处理。
