# <a name="pushcontext"></a>PUSHCONTEXT

保存当前的部分或全部*上下文*：段寄存器假设、基数值、列表和 cref 标志，或者处理器/协处理器值。 *上下文*可以是**假设**、**基数**、**列出**、 **CPU**或**全部**。

## <a name="syntax"></a>语法

> **PUSHCONTEXT** *上下文*
