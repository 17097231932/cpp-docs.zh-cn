# 编译器错误 C3276
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“keyword”: 在终止处理期间从 \_\_finally 块的跳出的行为未定义  
  
 此错误与 [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) 警告相同。  但是，当您使用 \/clr 时，不能使用 [warning](../../preprocessor/warning.md) 杂注禁用此条件。
