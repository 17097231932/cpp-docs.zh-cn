# 编译器警告（等级 3）C4013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”未定义；假设外部返回 int  
  
 编译器遇到对未定义函数的调用。  
  
### 通过检查以下可能的原因进行修复  
  
1.  函数名称拼写不正确  
  
2.  原型非 `extern` 的外部函数
