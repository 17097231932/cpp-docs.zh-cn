# <a name="creating-an-atl-dhtml-control"></a>创建 ATL DHTML 控件

ATL 控件向导会自动执行创建 DHTML 控件的过程。 它生成必要的资源文件，包括包含示例代码的 HTML 文件。

## <a name="to-create-an-atl-dhtml-control"></a>创建 ATL DHTML 控件

1. 按照 [创建 ATL 项目](../atl/reference/creating-an-atl-project.md)中的步骤进行操作。

1. 在 **类视图** 中，右键单击项目节点，指向 " **添加**"，然后单击快捷菜单中的 " **添加类** "。 在 " **添加类** " 对话框中，双击 " [ATL 控件向导](../atl/reference/atl-control-wizard.md)"。 在 **ATL 控件向导** 中，单击 " **选项** " 选项卡，然后选择 " **DHTML 控制**"。

你现在可以 [测试默认控件](../atl/testing-the-atl-dhtml-control.md)。
