# <a name="ml-nonfatal-error-a2039"></a>ML 非致命错误 A2039

**行太长**

源文件行超过 512 个字符的限制。

如果多个物理行与行继续符 (\) 进行连接，然后生成逻辑行仍被限制为 512 个字符。
