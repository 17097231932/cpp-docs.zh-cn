# 关于异常的疑难解答：System.ServiceModel.PoisonMessageException
当消息被视为“有害”消息时，将引发 <xref:System.ServiceModel.PoisonMessageException> 异常。 如果反复尝试发送消息失败，则将该消息视为有害消息。
