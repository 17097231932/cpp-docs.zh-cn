# Snap-In Object Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些宏提供管理单元扩展支持。  
  
|||  
|-|-|  
|[BEGIN\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/BEGIN_EXTENSION_SNAPIN_NODEINFO_MAP.md)|标记管理单元扩展数据类映射的开头管理单元对象的。|  
|[BEGIN\_SNAPINTOOLBARID\_MAP](../Topic/BEGIN_SNAPINTOOLBARID_MAP.md)|标记工具栏映射的开头管理单元对象的。|  
|[END\_EXTENSION\_SNAPIN\_NODEINFO\_MAP](../Topic/END_EXTENSION_SNAPIN_NODEINFO_MAP.md)|标记管理单元扩展数据类映射的结束管理单元对象的。|  
|[END\_SNAPINTOOLBARID\_MAP](../Topic/END_SNAPINTOOLBARID_MAP.md)|标记工具栏映射的结束管理单元对象的。|  
|[EXTENSION\_SNAPIN\_DATACLASS](../Topic/EXTENSION_SNAPIN_DATACLASS.md)|创建管理单元扩展的数据类的数据成员。|  
|[EXTENSION\_SNAPIN\_NODEINFO\_ENTRY](../Topic/EXTENSION_SNAPIN_NODEINFO_ENTRY.md)|输入管理单元扩展数据类管理单元对象的管理单元扩展数据类映射。|  
|[SNAPINMENUID](../Topic/SNAPINMENUID.md)|声明管理单元对象使用的上下文菜单的ID。|  
|[SNAPINTOOLBARID\_ENTRY](../Topic/SNAPINTOOLBARID_ENTRY.md)|转到工具栏管理单元对象的工具栏映射。|
