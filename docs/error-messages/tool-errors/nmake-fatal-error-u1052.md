# <a name="nmake-fatal-error-u1052"></a>NMAKE 错误 U1052

> 找不到文件 "*filename*"

NMAKE 找不到通过下列方式之一指定的文件：

- **/F** 选项

- **!INCLUDE** 预处理指令

- 响应文件的 At 符号 (**\@**) 说明符

请检查该文件是否存在以及文件名是否拼写正确。
