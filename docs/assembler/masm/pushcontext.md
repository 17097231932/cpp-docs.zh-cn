# <a name="pushcontext"></a>PUSHCONTEXT

保存当前的部分或全部*上下文*：段寄存器假设、基数值、列表和 cref 标志，或者处理器/协处理器值。 *上下文*可以**假设**为（仅限32位 masm）、**基数**、**列表**、 **CPU** （仅限32位 masm）或**全部**。

## <a name="syntax"></a>语法

> **PUSHCONTEXT** *上下文*
