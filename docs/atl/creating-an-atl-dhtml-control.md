# <a name="creating-an-atl-dhtml-control"></a>创建 ATL DHTML 控件

ATL 控件向导会自动创建 DHTML 控件的过程。 它生成必需的资源文件，其中包括包含示例代码的 HTML 文件。

## <a name="to-create-an-atl-dhtml-control"></a>创建 ATL DHTML 控件

1. 按照中的步骤[创建 ATL 项目](../atl/reference/creating-an-atl-project.md)。

1. 在中**类视图**，右键单击项目节点，指向**添加**，然后单击**添加类**从快捷菜单。 在中**添加类**对话框中，双击[ATL 控件向导](../atl/reference/atl-control-wizard.md)。 在中**ATL 控件向导**，单击**选项**选项卡并选择**DHTML 控件**。

现在可以[测试的默认控件](../atl/testing-the-atl-dhtml-control.md)。
