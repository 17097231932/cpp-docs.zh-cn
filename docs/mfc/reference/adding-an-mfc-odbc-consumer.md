# <a name="adding-an-mfc-odbc-consumer"></a>添加 MFC ODBC 使用者

MFC ODBC 使用者由 ODBC 记录集类和访问数据源所需的数据绑定组成。

### <a name="to-add-an-mfc-odbc-consumer"></a>添加 MFC ODBC 使用者

1. 在“类视图”中，右键单击项目。 在快捷菜单中，依次单击“添加”和“添加类”。

1. 在 Visual C++ 文件夹中，双击 **MFC ODBC 使用者** 图标或选择它并单击 " **打开**"。

   MFC ODBC 使用者向导将打开。

1. 定义 [MFC ODBC 使用者向导](../../mfc/reference/mfc-odbc-consumer-wizard.md)中所述的设置。

1. 单击 " **完成** " 关闭向导，这会在项目中插入新创建的 ODBC 使用者代码。
