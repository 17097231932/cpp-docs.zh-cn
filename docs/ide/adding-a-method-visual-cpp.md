# <a name="adding-a-method--visual-c"></a>添加方法 (Visual C++)
你可以使用[添加方法向导](../ide/add-method-wizard.md)添加到你的项目中的接口的方法。 如果项目包含与接口关联的类，则向导将太修改类。  
  
### <a name="to-add-a-method-to-your-object"></a>若要将方法添加到你的对象  
  
1.  在类视图中，展开项目节点以显示你想要将方法添加的接口。  
  
    > [!NOTE]
    >  你还可以将方法添加到调度接口，该项目已特性化，除非位于库节点下。  
  
2.  右键单击该接口的名称。  
  
3.  在快捷菜单上，单击**添加**，然后单击**添加方法**。  
  
4.  在添加方法向导中，提供的信息，以便创建方法。  
  
5.  指定此方法在任何接口定义语言设置[IDL 特性](../ide/idl-attributes-add-method-wizard.md)向导页。  
  
6.  单击**完成**添加此方法。
