# WriteOnlyArray::Length 属性
返回调用方分配的数组中的元素数目。  
  
## 语法  
  
```cpp  
property unsigned int Length{  
   unsigned int get() const;  
}  
```  
  
## 返回值  
 数组中的元素数。

 **命名空间：**Platform
