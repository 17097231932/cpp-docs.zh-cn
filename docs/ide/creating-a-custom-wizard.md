# 创建自定义向导
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+“自定义向导”是需要创建新自定义向导时使用的工具。  创建自定义向导的最简单方法是使用[自定义向导](../ide/custom-wizard.md)。  
  
> [!TIP]
>  自定义向导模板不可用的学习版。  
  
### 使用“自定义向导”创建向导  
  
1.  按照帮助主题[用 Visual C\+\+ 应用程序向导创建项目](../ide/creating-desktop-projects-by-using-application-wizards.md)中的说明进行操作。  
  
2.  在 **新建项目** 对话框中，选择" **Visual C\+\+&#124;常规** "模板"窗格中 **自定义向导** 图标打开"自定义向导。  
  
3.  使用“自定义向导”定义[应用程序设置](../ide/application-settings-custom-wizard.md)。  
  
4.  单击“完成”关闭向导并在开发环境中打开新项目。  
  
 创建项目后，可以在**解决方案资源管理器**中查看所创建的文件。  有关为“自定义向导”创建的文件的更多信息，请参见[为向导创建的文件](../ide/files-created-for-your-wizard.md)。
