# <a name="compiler-warning-level-2-c4007"></a>编译器警告（等级2） C4007

"function"：必须是 "attribute"

函数的必需特性未显式声明。 例如，函数**main**必须具有 **`__cdecl`** 特性。 编译器强制属性。
