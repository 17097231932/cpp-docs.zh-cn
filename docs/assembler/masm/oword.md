# <a name="oword"></a>OWORD

当需要16字节的数据类型时，用作类型说明符。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **OWORD**  *初始值设定项* ⟦__，__ *初始值设定项* .。。⟧
