# <a name="compiler-warning-level-1-c4166"></a>编译器警告（等级 1）C4166

**构造函数/析构函数的非法调用约定**

构造函数和析构函数不能具有平台默认以外的调用约定（显式指定 **__clrcall** 的情况除外）。
