# <a name="ptr"></a>ptr
将一个指针指定为完整的指针。  
  
## <a name="syntax"></a>语法  
  
```  
  
[ptr]  
  
```  
  
## <a name="remarks"></a>备注  
 **Ptr** c + + 属性具有相同的功能[ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL 特性。
