# 尚未正确安装 .NET Framework 调试服务。 最可能的原因是未正确注册 mscordbi.dll。 单击“帮助”以获取如何修复 .NET Framework 安装程序的详细信息。
当 Visual Studio 找不到与公共语言运行时调试服务一起安装的文件 mscordbi.dll 时，通常会发生此错误。  
  
### 更正此错误  
  
-   修复你计算机上的 .NET Framework 安装。
