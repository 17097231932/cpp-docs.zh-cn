# <a name="compiler-warning-level-4-c4517"></a>编译器警告（等级 4）C4517
访问声明已弃用;成员使用声明提供更好的选择  
  
 ANSI c + + 委员会已经宣布访问声明 (更改而无需派生类中的成员的访问权限[使用](../../cpp/using-declaration.md)关键字) 已过期。 C + + 的未来版本可能不支持访问声明。
