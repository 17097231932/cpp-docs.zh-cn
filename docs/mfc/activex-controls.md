# <a name="activex-controls"></a>ActiveX 控件

在 Visual C++ 中，您可以使用 MFC 或 ATL 创建 ActiveX 控件。

>[!IMPORTANT]
> ActiveX 是一项传统技术，不应使用新的开发。 可以使用现代的技术，例如 HTML5 和 JavaScript、 现代浏览器扩展或 WebAssembly 模块以更简单、 更安全的方式执行的许多功能的 ActiveX 控件。 有关详细信息，请参阅[推陈出新，第 2 部分： 告别 ActiveX、 VBScript、 attachEvent...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) 并[本机消息传送](/microsoft-edge/extensions/guides/native-messaging)并[Microsoft Edge 扩展](/microsoft-edge/extensions)并[WebAssembly](https://webassembly.org/)。
