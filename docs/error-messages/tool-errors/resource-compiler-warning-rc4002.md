# <a name="resource-compiler-warning-rc4002"></a>资源编译器警告 RC4002
“identifier”宏的实参太多  
  
 使用给定标识符指定的实际参数的数目高于标识符的宏定义中给定的正式参数的数目。  
  
 其他实际参数已收集，但在宏扩展过程中忽略。
