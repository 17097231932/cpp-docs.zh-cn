# <a name="fatal-error-c1085"></a>错误 C1085

无法写入 filetype 文件: file： 消息

### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复

1. 驱动器是只读的。

1. 驱动器已满。

1. 共享冲突。

1. 如果消息指出“文件号错误”，说明此文件在后台进行编译的同时，可能已在前台关闭。
