# <a name="compiler-error-c2824"></a>编译器错误 C2824
operator new 必须是返回类型 void *  
  
 使用非基于指针重载的运算符`new`必须返回`void *`。  
  
 下面的示例生成 C2824:  
  
```  
// C2824.cpp  
// compile with: /c  
class   A {  
   A* operator new(size_t i, char *m);   // C2824  
   // try the following line instead  
   // void* operator new(size_t i, char *m);  
};  
```
