# <a name="ml-nonfatal-error-a2066"></a>ML 非致命错误 A2066

**不兼容的 CPU 模式和段大小**

尝试使用与指定 CPU 不兼容的 **USE16**、 **USE32** 或 **扁平** 属性打开段，或在32位段中更改为16位 CPU。）。

**USE32** 和 **平面** 属性前面必须加上386或更高的处理器指令。
