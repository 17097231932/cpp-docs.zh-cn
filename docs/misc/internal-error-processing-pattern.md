# Internal error processing pattern.
当正则表达式已包括在搜索字符串中而该正则表达式的语法包含错误时，通常会发生此错误。  
  
### 更正此错误  
  
1.  使用正确的正则表达式语法重新输入搜索字符串。  有关更多信息，请参见[正则表达式](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)。
