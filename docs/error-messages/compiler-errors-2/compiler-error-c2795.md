# 编译器错误 C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“super::function”不是成员函数  
  
 每当尝试使用 [super](../../cpp/super.md) 访问成员而不是成员函数时，都会出现此错误信息。
