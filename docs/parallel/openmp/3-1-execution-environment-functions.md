# <a name="31-execution-environment-functions"></a>3.1 执行环境函数

在本部分中所述的函数会影响，并监视线程、 处理器和并行环境：

- **omp_set_num_threads**函数。

- **omp_get_num_threads**函数。

- **omp_get_max_threads**函数。

- **omp_get_thread_num**函数。

- **omp_get_num_procs**函数。

- **omp_in_parallel**函数。

- **omp_set_dynamic**函数。

- **omp_get_dynamic**函数。

- **omp_set_nested**函数。

- **omp_get_nested**函数。
