# 资源编译器错误 RC1005
找不到文件：filename  
  
 找不到资源编译器命令行中指定的文件。 检查以确定文件是否已被移动到另一个目录，以及是否正确键入文件名或路径。  
  
 使用 **INCLUDE** 环境变量或 Visual C\+\+ 目录设置搜索了文件。
