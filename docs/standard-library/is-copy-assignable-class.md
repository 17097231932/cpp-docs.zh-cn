# <a name="iscopyassignable-class"></a>is_copy_assignable 类
测试是否可以在赋值时复制类型。  
  
## <a name="syntax"></a>语法  
  
```
template <class Ty>
struct is_copy_assignable;
```  
  
#### <a name="parameters"></a>参数  
 `Ty`  
 要查询的类型。  
  
## <a name="remarks"></a>备注  
 如果类型 `Ty` 是具有复制赋值运算符的类，则类型谓词的实例为 true；否则为 false。 等效于 is_assignable\<Ty&, const Ty&>。  
  
## <a name="requirements"></a>要求  
 **标头：**\<type_traits>  
  
 **命名空间：** std
