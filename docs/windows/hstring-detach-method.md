# HString::Detach 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

将指定的 HString 对象从其基础值分离。  
  
## 语法  
  
```  
HSTRING Detach() throw()  
```  
  
## 返回值  
 在启动的分离操作之前的 HString 基础值。  
  
## 要求  
 **标头：**corewrappers.h  
  
 **命名空间：**Microsoft::WRL::Wrappers
