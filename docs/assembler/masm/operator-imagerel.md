# operator IMAGEREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回图像相对偏移量 `expression`。  
  
## 语法  
  
```  
IMAGEREL expression  
```  
  
## 备注  
 得到的值通常称为 RVA 或相对虚拟地址。  
  
 IMAGEREL 为 COFF 对象显示可用。
