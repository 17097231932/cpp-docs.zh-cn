# <a name="compiler-error-c3832"></a>编译器错误 C3832
type library： 类型库看上去就好像它为 32 位指针生成;请更改 ptrsize 限定符  
  
 显式信息随附`ptrsize`属性[#import](../../preprocessor/hash-import-directive-cpp.md)指令未与类型库中找到的编译器不一致。
