# 编译器错误 C2819
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“type”类型没有重载成员“operator \-\>'  
  
 需要定义 `operator->()` 以使用此指针运算。  
  
 下面的示例生成 C2819：  
  
```  
// C2819.cpp  
// compile with: /c  
class A {  
   public:  
      int i;  
};  
  
class B {};  
  
void C(B j) {  
   j->i;   // C2819  
}  
  
class D {  
   A* pA;  
  
   public:  
      A* operator->() {  
         return pA;  
      }  
};  
  
void F(D j) {  
   j->i;  
}  
```  
  
 使用 [参考类型的 C\+\+ 堆栈语义](../../dotnet/cpp-stack-semantics-for-reference-types.md) 时也可能发生 C2819 错误。  下面的示例生成 C2819：  
  
```  
// C2819_b.cpp  
// compile with: /clr  
ref struct R {  
   void Test() {}  
};  
  
int main() {  
   R r;  
   r->Test();   // C2819  
   r.Test();   // OK  
}  
```
