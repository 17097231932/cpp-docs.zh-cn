# <a name="ole-mfc"></a>OLE (MFC)

在您的程序中实现 OLE 功能将以多种方式影响您的用户界面：

- 可视化编辑 (就地激活) 在程序的 windows 中显示其他程序的用户界面，并通过其他程序的项来修改程序的菜单。

- 拖放允许用户在窗口内、甚至在程序之间拖动对象。

- 跟踪器提供可视化编辑和拖放过程中对象状态的视觉提示。
