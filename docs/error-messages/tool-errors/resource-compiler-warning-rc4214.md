# <a name="resource-compiler-warning-rc4214"></a>资源编译器警告 RC4214

代码页无效：已忽略

.Rc 文件包含代码页参数，但指定的代码页无效。 有关详细信息，请参阅[IsValidCodePage](/windows/win32/api/winnls/nf-winnls-isvalidcodepage) 。
