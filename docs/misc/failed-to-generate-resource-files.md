# Failed to generate resource files
在作为创建 .NET 资源管理器能够理解的二进制 .resources 文件的过程的一部分读取 .resx 文件时，发生项目系统的一般性错误。  
  
 此错误通常由 .resx 文件错误引起。  
  
 该警告不会导致生成失败。  但是，为其显示此错误的 .resx 文件不会作为程序集清单资源包含在项目中。
