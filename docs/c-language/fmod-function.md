# fmod 函数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.6.4** 当 `fmod` 函数的第二个参数为零时，是发生域错误还是返回零  
  
 当 `fmod` 函数的第二个参数为零时，该函数将返回零。
