# <a name="sbyte-masm"></a>SBYTE (MASM)
分配和 （可选） 为每个初始化存储一个有符号的字节`initializer`。 此外可作为类型说明符类型是合法的任何位置。  
  
## <a name="syntax"></a>语法  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
