# 编译器警告（等级 4）C4206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**使用了非标准扩展 : 翻译单元为空**  
  
 预处理后文件为空。  
  
 该扩展可以防止将代码移植到其他编译器。  它在 ANSI 兼容性 \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) 下生成错误，且只应用于 C 源代码。
