# ML Fatal Error A1017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**缺少源文件**  
  
 语言找不到聚合或传递给链接器。  
  
 此错误，当您提供语言命令行选项时，如果不指定文件名操作。  若要程序集没有一个 .asm 扩展名的文件，请使用 **\/Ta** 命令行选项。  
  
 此错误可能通过调用语言还会生成不带参数，如果语言环境变量包含命令行选项。
