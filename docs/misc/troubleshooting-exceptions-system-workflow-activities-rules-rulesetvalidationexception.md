# 关于异常的疑难解答：System.Workflow.Activities.Rules.RuleSetValidationException
当由于无法验证某个规则集而无法继续处理时，将引发 <xref:System.Workflow.Activities.Rules.RuleSetValidationException> 异常。 如果使用的 <xref:System.Workflow.Activities.Rules.RuleEngine> 具有验证错误，则 <xref:System.Workflow.Activities.Rules.RuleSet> 构造函数将引发此异常。
