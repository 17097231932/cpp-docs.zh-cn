# <a name="ml-nonfatal-error-a2206"></a>ML 非致命错误 A2206

**在表达式中缺少运算符**

无法计算表达式，因为它缺少运算符。 此错误消息也可能是前面的程序错误的副作用。

以下行将生成此错误：

```asm
value1 = ( 1 + 2 ) 3
```
