# <a name="compiler-error-c3047"></a>编译器错误 C3047
OpenMP“sections”区域中的结构化块的前面必须是“#pragma omp section”  
  
 中引入的代码块的任何代码[部分](../../parallel/openmp/reference/sections-openmp.md)指令必须是中引入的代码块`section`指令。  
  
 下面的示例生成 C3047：  
  
```  
// C3047.cpp  
// compile with: /openmp /c  
#include "omp.h"  
  
int main() {  
   int n2 = 2, n3 = 3;  
  
   #pragma omp parallel  
   {  
      ++n2;  
  
      #pragma omp sections  
      {  
  
         #pragma omp section  
         {  
            ++n3;  
         }  
  
         ++n2;   // C3047 not enclosed in #pragma omp section  
      }  
   }  
}  
```
