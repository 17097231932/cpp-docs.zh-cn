# <a name="expression-evaluator-error-cxx0016"></a>表达式计算器错误 CXX0016

常量太大

C 表达式计算器不能接受大于4294967295的无符号整数常量（0FFFFFFFF 十六进制），也不能接受其数量级大于大约 1.8 E + 308 的浮点型常量。

此错误与 CAN0016 相同。
