# <a name="using-task-manager"></a>使用任务管理器

若要调试服务的最简单方式之一是通过使用任务管理器。 运行该服务时，启动任务管理器，然后单击**进程**选项卡。右键单击的 EXE 的名称，然后单击**调试**。 这将启动 VisualC++附加到该正在运行的进程。 现在，单击**中断**上**调试**菜单，以便您可以在代码中设置断点。 单击**运行**运行到选定的断点。
