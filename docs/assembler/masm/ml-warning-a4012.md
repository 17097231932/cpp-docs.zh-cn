# ML Warning A4012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**线段的行号信息没有类 “CODE**  
  
 在没有一个类名称以 “结尾的代码段的命令”。汇编不生成这些命令的 CodeView 信息。  
  
 CodeView 无法处理的代码的模块在具有以 “代码不关闭的类名的段”。
