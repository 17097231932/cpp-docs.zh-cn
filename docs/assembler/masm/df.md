# <a name="df"></a>DF
分配和 （可选） 为每个初始化 6 字节存储`initializer`。 `DF` 同义词[FWORD](../../assembler/masm/fword.md)。  
  
## <a name="syntax"></a>语法  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>备注  
 此外可以用作类型说明符类型是合法的任何位置。
