# <a name="_variant_tclear"></a>_variant_t::Clear

**Microsoft 专用**

清除封装的 `VARIANT` 对象。

## <a name="syntax"></a>语法

```
void Clear( );
```

## <a name="remarks"></a>备注

对封装的 `VARIANT` 对象调用 `VariantClear`。
