# 追加模式中的文件位置
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** 追加模式流的文件位置指示器最初是位于文件的开头还是结尾  
  
 在追加模式中打开文件时，文件位置指示器最初指向文件的末尾。
