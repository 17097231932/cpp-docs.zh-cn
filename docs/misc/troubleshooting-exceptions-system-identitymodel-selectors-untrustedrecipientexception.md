# 关于异常的疑难解答：System.IdentityModel.Selectors.UntrustedRecipientException
当用户检查正在请求标记的实体中的信息并决定不信任该实体时，将引发 <xref:System.IdentityModel.Selectors.UntrustedRecipientException> 异常。 用户单击 CardSpace 用户界面中的**“禁用 Windows CardSpace”**。
