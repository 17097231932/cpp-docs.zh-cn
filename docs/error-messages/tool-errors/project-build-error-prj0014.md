# <a name="project-build-error-prj0014"></a>项目生成错误 PRJ0014

用于控制生成的过程的作业对象失败。  生成无法继续。

在开发环境中出错。

若要解决此错误，请关闭并重新加载项目。 如果有必要，请退出并重新启动 Visual Studio。
