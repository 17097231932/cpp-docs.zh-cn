# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指定属性访问器函数。  
  
## 语法  
  
```  
  
[propget]  
  
```  
  
## 备注  
 **propget** C\+\+ 特性具有与 [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL 属性相同。  
  
## 示例  
 为 [可绑定](../windows/bindable.md) 参见示例为 **propget**的示例使用。  
  
## 要求  
  
### 属性上下文  
  
|||  
|-|-|  
|**适用对象**|方法|  
|**可重复**|否|  
|**必需的特性**|无|

 有关属性上下文的更多信息，请参见 [属性上下文](../windows/attribute-contexts.md)。
