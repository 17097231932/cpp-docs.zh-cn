# <a name="flipping-an-image-image-editor-for-icons"></a>翻转图像（图标的图像编辑器）

您可以翻转或旋转图像：

- 创建原始的镜像。

- 将图像上下颠倒。

- 一次将右侧图像旋转 90 度。

### <a name="to-flip-the-image-horizontally-mirror-image"></a>若要水平翻转图像 （镜像）

1. 从**图像**菜单中，选择**水平翻转**。

### <a name="to-flip-the-image-vertically-turn-upside-down"></a>若要垂直翻转图像 （倒置）

1. 从**图像**菜单中，选择**垂直翻转**。

### <a name="to-rotate-the-image-90-degrees"></a>若要将图像旋转 90 度

1. 从**图像**菜单中，选择**旋转 90 度**。

   > [!NOTE]
   > 此外可以使用[加速 （快捷） 键](../windows/accelerator-keys-image-editor-for-icons.md)这些命令或从快捷菜单 （外部图像与图像编辑器中单击） 访问这些命令。
