# <a name="label-masm"></a>LABEL (MASM)

通过将分配的当前位置计数器值创建一个新标签和给定*类型*到*名称*。

## <a name="syntax"></a>语法

> *名称*标签*类型*
>
> *名称*标签\[NEAR |得 |PROC] PTR \[*类型*]
