# 无法使用当前的 Delimiter 分析行 &lt;number&gt;
无法分析指定的行，因为它使用的不是那些指定的分隔符。  
  
### 更正此错误  
  
-   调整 `Delimiters` 以便正确分析此行，或插入异常处理代码以处理此行。
