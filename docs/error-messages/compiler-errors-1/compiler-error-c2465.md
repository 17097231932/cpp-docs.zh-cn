# <a name="compiler-error-c2465"></a>编译器错误 C2465
不能在括号内定义匿名类型  
  
 在带括号的表达式中定义了匿名结构、联合或枚举类型。 这在 C++ 中无效，因为该定义在函数范围中没有意义。
