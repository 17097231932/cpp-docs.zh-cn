# <a name="compiler-error-c3762"></a>编译器错误 C3762

无法处理特性 attribute

用户定义的特性继承`System.Security.Permissions.SecurityAttribute`用于定义安全特性。 此类属性不能使用在同一程序集中定义的位置。
