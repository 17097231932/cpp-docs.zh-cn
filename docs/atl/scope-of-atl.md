# Scope of ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL可以轻松地创建COM对象、自动化服务器和ActiveX控件。  ATL提供内置用于许多基本COM接口支持。  
  
 ATL交付为您在应用程序中包含的源代码。  ATL还允许一个DLL可用\(atl90.dll\)，包含代码可以在元素之间共享。  但是，此DLL不是必需的。
