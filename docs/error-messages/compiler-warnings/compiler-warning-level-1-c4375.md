# 编译器警告（等级 1）C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非公共方法“method2”不重写“method1”  
  
 实现另一个类型的类型定义了一个重写方法，但重写不是公共的。  因此，该方法不重写基类型方法。
