# <a name="label-masm"></a>LABEL (MASM)

通过将当前位置计数器值和给定*类型*分配给*名称*来创建新标签。

## <a name="syntax"></a>语法

> *名称***标签***类型*
>
> *名称***标签**⟦**NEAR** | **远处** | **PROC**⟧ **PTR** ⟦*类型*⟧
