# <a name="immediatebind"></a>immediatebind
指示数据库将立即收到通知的数据绑定对象的属性的所有更改。  
  
## <a name="syntax"></a>语法  
  
```  
  
[immediatebind]  
  
```  
  
## <a name="remarks"></a>备注  
 **Immediatebind** c + + 属性具有相同的功能[immediatebind](http://msdn.microsoft.com/library/windows/desktop/aa367045) MIDL 特性。
