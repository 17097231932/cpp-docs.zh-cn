# <a name="ole-db-templates"></a>OLE DB 模板

OLE DB 模板简化了 OLE DB 数据访问。 Visual C++ 提供了实现许多常用 OLE DB 接口的模板类。 可以使用 Visual C++ 使用者模板来编写 OLE DB 客户端（使用者）应用程序，并使用提供程序模板来编写服务器（提供程序）应用程序。
