# Visual Studio 反馈策略
感谢你的反馈。  开发人员编写团队会使用你的反馈改进文档。  在我们查看你的反馈期间，我们可能会向你发送电子邮件以询问有关解决方案的说明或反馈。  我们不会将你的电子邮件地址用于任何其他用途，并且会在完成查看之后删除它。  有关 Microsoft 的隐私策略的进一步信息，请参阅 [http:\/\/privacy.microsoft.com\/default.aspx](http://privacy.microsoft.com/default.aspx)。
