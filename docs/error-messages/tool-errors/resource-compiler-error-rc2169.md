# <a name="resource-compiler-error-rc2169"></a>资源编译器错误 RC2169
资源文件的文件名不是 2.03 格式  
  
 指定的资源使用早于 2.03 版的格式。 资源文件必须转换或重新创建使用的格式对 3.00 或更高版本。
