# <a name="public-masm"></a>PUBLIC (MASM)

使每个变量、 标签或指定为绝对符号*名称*提供给程序中的所有其他模块。

## <a name="syntax"></a>语法

> 公共 [langtype 名称 [[，[langtype 名称]]...
