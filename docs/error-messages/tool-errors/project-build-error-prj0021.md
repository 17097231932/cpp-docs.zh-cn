# <a name="project-build-error-prj0021"></a>项目生成错误 PRJ0021

工具*工具*的属性*属性*中指定的文件名无效。

可能使用了未知或无效的宏。

有关如何访问项目属性的信息，请参阅[设置视觉对象C++属性](../../build/working-with-project-properties.md)。
