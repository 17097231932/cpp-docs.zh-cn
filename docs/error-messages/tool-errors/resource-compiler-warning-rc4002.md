# 资源编译器警告 RC4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”宏的实参太多  
  
 用给定标识符指定的实参数大于该标识符宏定义中给定的形参数。  
  
 在宏扩展期间收集但忽略了其他的实参。
