# <a name="generated-classes-mfc-application-wizard"></a>MFC 应用程序向导的生成的类
本主题列出的基类，这些类和你的项目生成的文件的名称。 默认情况下，名称基于中指定的项目名称**新建项目对话框**。 你可以更改大部分这些名称，如下所述：  
  
 **生成的类**  
 为项目创建的类的名称。 默认情况下，名称基于项目名称。 默认 MFC 项目创建 C*ProjName*查看 C 类*ProjName*App 类，C*ProjName*文档类、 CMainFrame 类，并 CChildFrame 类。 此页上的所有其他值包含有关中当前选定的类信息**生成的类**列表。  
  
 若要更改类名称，请使用**类名**文本框。  
  
 **类名**  
 中当前选定的类名称**生成的类**列表。 如果框中处于活动状态，你可以更改类名称。 当你更改焦点从**类名**框中，对所选的类名称的任何更改将出现在**生成的类**列表。  
  
 **.h 文件**  
 中当前选定的类标头文件的名称**生成的类**列表。 如果文本框处于活动状态，你可以更改标头文件的名称。  
  
 **基类**  
 中的当前所选类的基类的名称**生成的类**列表。 如果框中处于活动状态，你可以从列表中选择另一个类的基类。  
  
 **.cpp 文件**  
 与所选类的源代码文件的名称。 如果文本框处于活动状态，你可以更改实现文件的名称。
