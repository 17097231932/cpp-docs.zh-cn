# <a name="real8"></a>REAL8

分配并选择性地为每个 *初始值设定项* 初始化双精度 (8 字节) 浮点数。

## <a name="syntax"></a>语法

> *名称* **REAL8** *初始值设定项* ⟦__，__ *初始值设定项* .。。⟧
