# <a name="compiler-warning-level-1-c4015"></a>编译器警告 （等级 1） C4015
identifier： 位域的类型必须是整型  
  
 位域未声明为整数类型。 编译器将假定将无符号的位字段的基类型。 位域必须声明为无符号的整数类型。
