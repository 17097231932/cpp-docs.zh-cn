# A failure occurred while attempting to start the compilation
向 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 或 [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)] 编译器发出**“启动生成”**请求时遇到了未知错误。  
  
 **更正此错误**  
  
-   重新启动 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。  
  
     此错误将使生成进程失败。
