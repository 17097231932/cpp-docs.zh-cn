# 资源编译器错误 RC2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

对话框中应有 END  
  
 **END** 关键字必须出现在 **DIALOG** 语句的结尾。  确保没有前一语句剩下的左引号。
