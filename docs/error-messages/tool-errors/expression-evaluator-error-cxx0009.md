# 表达式计算器错误 CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在字符串末尾缺少 '  
  
 字符常数后缺少应有的单引号。  重新输入表达式，并用单引号将字符常数括起来。  
  
 该错误与 CAN0009 相同。
