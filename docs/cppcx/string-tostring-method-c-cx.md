# String::ToString 方法 (C++/CX)
返回一个字符串对象，其值与当前字符串相同。  
  
## 语法  
  
```cpp  
  
String^ String::ToString()  
```  
  
## 返回值  
 其值与当前字符串相同的字符串对象。
