# <a name="expression-evaluator-error-cxx0020"></a>表达式计算器错误 CXX0020

此操作的操作数类型错误

运算符对该运算符已应用于具有无效类型的表达式。

例如，不能用来获取注册或用浮点表达式作为数组下标的地址。

此错误是与 CAN0020 相同。
