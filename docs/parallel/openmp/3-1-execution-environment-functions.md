# 3.1 Execution Environment Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

本节描述的功能会影响和监视线程、处理器和并行环境:  
  
-   **omp\_set\_num\_threads** 功能。  
  
-   **omp\_get\_num\_threads** 功能。  
  
-   **omp\_get\_max\_threads** 功能。  
  
-   **omp\_get\_thread\_num** 功能。  
  
-   **omp\_get\_num\_procs** 功能。  
  
-   **omp\_in\_parallel** 功能。  
  
-   **omp\_set\_dynamic** 功能。  
  
-   **omp\_get\_dynamic** 功能。  
  
-   **omp\_set\_nested** 功能。  
  
-   **omp\_get\_nested** 功能。
