# <a name="editable-file-types-for-resources"></a>资源的可编辑文件类型
可以打开下列类型的文件，并可以编辑它们包含的资源。  
  
|文件名|描述|  
|---------------|-----------------|  
|.rc|资源脚本文件。|  
|.rct|资源模板文件。|  
|.res|资源文件。|  
|.resx|托管资源文件。|  
|.exe|可执行文件。 只能在 Windows 2000 和更高版本的操作系统上进行编辑。|  
|.dll|动态链接库文件。 只能在 Windows 2000 和更高版本的操作系统上进行编辑。|  
|.bmp、.ico、.dib 和 .cur|位图、图标、工具栏和光标文件。|

 Win32
