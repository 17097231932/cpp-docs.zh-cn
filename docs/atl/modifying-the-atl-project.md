# <a name="modifying-the-atl-project"></a>修改 ATL 项目

此时，您的复合控件项目实现为复合控件必需的对象。 下一步是添加任何控件复合控件将包含并处理任何必要的事件。

若要添加其他 ActiveX 或 Windows 控件，添加一个新的资源脚本，然后使用对话框编辑器。 添加控件 （和相关的任务） 的详细信息，请参阅[对话框编辑器](../windows/dialog-editor.md)。

若要处理任何必要的事件从 ActiveX 控件，请参阅[将功能添加到复合控件](../atl/adding-functionality-to-the-composite-control.md)。
