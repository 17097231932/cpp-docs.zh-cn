# <a name="randmax"></a>RAND_MAX

## <a name="syntax"></a>语法

```
#include <stdlib.h>
```

## <a name="remarks"></a>备注

常量 `RAND_MAX` 是 `rand` 函数可返回的最大值。 `RAND_MAX` 定义为值 0x7fff。
