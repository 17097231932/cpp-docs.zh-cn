# <a name="hstringisvalid-method"></a>HString::IsValid 方法
指示当前 HString 对象是否为空。  
  
## <a name="syntax"></a>语法  
  
```  
bool IsValid() const throw()  
```  
  
#### <a name="parameters"></a>参数  
 如果当前对象 HString 不为空，则为 `true`；否则为 `false`。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
