# <a name="attributes-by-group"></a>按组分的特性

这些C++特性分为以下功能组。

|Attribute|说明|
|---------------|-----------------|
|[COM 特性](com-attributes.md)|注入代码以支持许多 COM 开发区域和 .NET Framework 公共语言运行时开发。|
|[IDL 特性](idl-attributes.md)|在不使用向导的情况下，从源代码文件中启用对 .idl 文件的修改，并且不知道该文件的结构和语法。|
|[OLE DB 使用者特性](ole-db-consumer-attributes.md)|插入代码（基于[OLE DB 使用者模板](../../data/oledb/ole-db-consumer-templates-reference.md)）用于创建一个工作 OLE DB 使用者，该使用者执行打开表、执行命令和访问数据等任务。|
|[编译器特性](compiler-attributes.md)|由 Microsoft C++编译器提供。|

有关如何为 .NET Framework 应用程序定义自定义属性的信息，请参阅[用户定义的属性](../../extensions/user-defined-attributes-cpp-component-extensions.md)。
