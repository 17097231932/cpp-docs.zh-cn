# <a name="expression-evaluator-error-cxx0053"></a>表达式计算器错误 CXX0053

无函数符号匹配绑定断点时

用作一个断点的符号不是一个函数。 为断点指定的数据成员可能导致此错误。

此错误是与 CAN0053 相同。
