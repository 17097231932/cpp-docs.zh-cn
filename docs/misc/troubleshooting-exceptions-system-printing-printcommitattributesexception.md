# 关于异常的疑难解答：System.Printing.PrintCommitAttributesException
当错误条件阻止 <xref:System.Printing.PrintCommitAttributesException> 对象向其表示的实际计算机、打印机或设备提交某些特性时，将引发 <xref:System.Printing.PrintSystemObject> 异常。
