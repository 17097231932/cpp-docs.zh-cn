# 编译器警告（等级 1）C4088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 实参“number”和形参“number”中的指针不匹配  
  
 对应的形参和实参间接寻址的级别不同。  实参传递时未经更改。  所调用函数将其值解释为指针。
