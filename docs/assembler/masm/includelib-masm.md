# <a name="includelib-masm"></a>INCLUDELIB (MASM)

通知链接器的当前模块应与链接*libraryname*。

## <a name="syntax"></a>语法

> INCLUDELIB libraryname

## <a name="remarks"></a>备注

*Libraryname*必须括在尖括号中，如果它包含一个反斜杠，分号，更高版本的符号，较少-比符号、 单引号或双引号。
