# 编译器错误 C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

命令行选项“option”与预编译头不一致  
  
 给定命令行选项与用于创建预编译头的选项不同。  请使用给定的命令行选项重新生成预编译头。
