# ML Nonfatal Error A2004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**符号类型冲突:标识符**  
  
 [EXTERNDEF](../../assembler/masm/externdef.md) 或 [标签](../../assembler/masm/label-masm.md) 的指令中定义在同一个模块，但使用不同类型的变量、符号、数据结构或标签改用。
