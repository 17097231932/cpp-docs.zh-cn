# Error getting information on languages supported by online search providers.
一般情况下，尝试从 [Help, Environment, Options Dialog Box](http://msdn.microsoft.com/zh-cn/a2a5dff5-70a5-4f14-bf4c-66793894de6a) 的**“联机语言”**列表中选择一种语言，却又不能访问联机服务时，会出现此错误。  
  
### 更正此错误  
  
1.  确认您有活动的 Internet 连接。  
  
2.  如果使用代理服务器，请验证代理服务器配置是否正确。  
  
3.  再次选择**“联机语言”**下拉列表。
