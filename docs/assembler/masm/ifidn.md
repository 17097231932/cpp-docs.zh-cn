# <a name="ifidn-ifidni"></a>IFIDN、IFIDNI

如果文本项相同，则授予程序集。

## <a name="syntax"></a>语法

> **IFIDN** *textitem1*__，__ *textitem2*\
> **IFIDNI** *textitem1*__，__ *textitem2*

## <a name="remarks"></a>备注

如果给定了 **IFIDNI** ，则比较不区分大小写。 [若](if-masm.md)要获取完整的语法，请参阅。
