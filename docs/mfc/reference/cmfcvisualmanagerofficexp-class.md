# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP 类

`CMFCVisualManagerOfficeXP`向应用程序提供了 Microsoft Office XP 的外观。

## <a name="syntax"></a>语法

```
class CMFCVisualManagerOfficeXP : public CMFCVisualManager
```

## <a name="members"></a>成员

所有此类的成员都是派生自该类的上级的虚函数[CMFCVisualManager 类](../../mfc/reference/cmfcvisualmanager-class.md)。

## <a name="remarks"></a>备注

您使用`CMFCVisualManagerOfficeXP`类更改为类似于 Microsoft Office XP 的应用程序的可视外观。
