# .CONTINUE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

生成代码。 [.WHILE](../../assembler/masm/dot-while.md) 的顶部跳转或 [.REPEAT](../../assembler/masm/dot-repeat.md) 块，如果 `condition` 为 true。  
  
## 语法  
  
```  
  
.CONTINUE [[.IF condition]]  
```
