# 关于异常的疑难解答：Microsoft.Office.Tools.ControlNameAlreadyExistsException
如果尝试以编程方式创建与项目中其他控件同名的控件，则将引发 <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> 异常。  
  
## 相关提示  
 **为每个控件使用不同的名称。**  
 控件名称必须唯一。
