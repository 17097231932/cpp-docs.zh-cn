# <a name="linker-tools-error-lnk1221"></a>链接器工具错误 LNK1221

子系统无法推断，必须定义

链接器没有足够的信息来推断出哪些子系统你想要为目标。

若要修复此错误，请使用[/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)。
