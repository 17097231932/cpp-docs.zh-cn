# <a name="math-error-m6110"></a>数学错误 M6110

堆栈溢出

浮点表达式导致浮点堆栈溢出。

除8087/287/387 协处理器通常支持的八个级别外，堆栈溢出浮点异常将捕获到7个级别的限制。

程序终止，退出代码为138。
