# <a name="compiler-error-c3661"></a>编译器错误 C3661

显式重写列表没有发现要重写任何方法

显式重写指定一个或多个类型名称。  但是，没有具有所需的类型匹配重写函数的签名中的任何函数。  如果你尝试重写基于类型名称，重写的函数的签名匹配的指定类型中必须有一个或多个虚函数。

有关详细信息，请参阅[显式重写](../../extensions/explicit-overrides-cpp-component-extensions.md)。
