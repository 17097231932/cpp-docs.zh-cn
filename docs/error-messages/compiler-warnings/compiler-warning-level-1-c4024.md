# <a name="compiler-warning-level-1-c4024"></a>编译器警告 （等级 1） C4024

function： 形参和实参的参数 number 的类型不同

对应的形式和实际参数具有不同的类型。 编译器将传递的实际参数，而无需更改。 接收函数将参数类型转换为预期的类型。
