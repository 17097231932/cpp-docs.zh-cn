# operator EQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回 true \(– 1\)，如果 *expression1* 等于 *expression2*，或者返回错误 \(0\); 如果未。  
  
## 语法  
  
```  
  
expression1 EQ expression2  
```
