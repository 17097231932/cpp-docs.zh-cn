# What Is AtlAxWin100?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`AtlAxWin100` 是帮助提供ATL的控件承载函数窗口选件类的名称。  当您创建此选件类实例，窗口过程将自动使用控件承载API创建宿主对象与windows和用您指定为窗口的标题的控件填充它。
