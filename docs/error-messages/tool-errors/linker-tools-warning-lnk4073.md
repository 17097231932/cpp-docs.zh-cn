# <a name="linker-tools-warning-lnk4073"></a>链接器工具警告 LNK4073

无法创建 .ilk 文件的映射;非增量链接

共享内存中没有足够大的连续空间，无法创建增量状态（. .ilk）文件。 链接已执行非增量生成。
