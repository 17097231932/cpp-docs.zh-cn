# <a name="namespaces-reference-ccx"></a>命名空间参考 (C++/CX)

文档的本节中的文章介绍了支持C++/cx 编译器的命名空间

## <a name="compiler-supplied-namespaces"></a>编译器提供的命名空间

为了简化针对 Windows 运行时的程序的编码， C++/cx 编译器及其支持的标头文件提供了定义各种类型的命名空间。 命名空间定义内置数值类型;字符串、数组和集合;C++表示 Windows 运行时错误的异常;和对标准 Windows 运行时类型的特定语言增强。
