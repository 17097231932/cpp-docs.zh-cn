# SDWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化签名双字 \(4 字节\) 每 `initializer`的存储。  可以用作类型说明符任何位置类型是非法的。  
  
## 语法  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
