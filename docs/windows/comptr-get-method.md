# <a name="comptrget-method"></a>ComPtr::Get 方法
检索指向与此 ComPtr 相关联的接口。  
  
## <a name="syntax"></a>语法  
  
```  
T* Get() const;  
```  
  
## <a name="return-value"></a>返回值  
 指向与此 ComPtr 相关联的接口指针。  
  
## <a name="requirements"></a>要求  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
