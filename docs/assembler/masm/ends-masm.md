# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

标记从 [段](../../assembler/masm/segment.md)、 [STRUCT](../../assembler/masm/struct-masm.md)、 [联合](../../assembler/masm/union.md)或一个简化的段指令\) 开始时间段结束，结构或联合 *名* 。  
  
## 语法  
  
```  
  
name ENDS  
```
