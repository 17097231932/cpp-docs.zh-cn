# <a name="adding-an-mfc-class"></a>添加 MFC 类

要将派生自 Microsoft 基础类 （MFC） 库类的类添加到项目，请使用[类向导](mfc-class-wizard.md)中的 **"添加类**"按钮。 指定新类的名称，选择基类，然后选择与其关联的对话框的 ID（如果有）。 该向导创建一个标头文件和一个实现文件，并将它们添加到您的项目中。

> [!NOTE]
> 如果最初[使用 MFC 支持创建了应用程序，](../../atl/reference/mfc-support-in-atl-projects.md)则可以将 MFC 类添加到 ATL COM 应用程序。 您还可以向支持 MFC 的 Win32 项目添加 MFC 类。

### <a name="to-add-an-mfc-class-to-your-project"></a>向项目添加 MFC 类

在类视图中，右键单击项目名称。 单击 **"类向导**"，然后单击"**添加类**"以打开"[添加 MFC 类"](mfc-add-class-wizard.md)对话框。 使用**类向导**添加消息处理程序或新方法。
