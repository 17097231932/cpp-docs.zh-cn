# <a name="profile-guided-optimization-warning-pg1039"></a>按配置优化警告 PG1039

未能等待空闲时间以进行扫描。

应用程序不允许运行[pgosweep](../../build/pgosweep.md) 。  如果可能，请在应用程序处于空闲状态时运行 pgosweep。
