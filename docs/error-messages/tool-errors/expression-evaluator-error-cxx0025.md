# <a name="expression-evaluator-error-cxx0025"></a>表达式计算器错误 CXX0025

运算符需要结构/联合

采用 `struct` 或**联合**类型表达式的运算符被应用于非 `struct` 或**union**表达式。

类、结构或联合变量的组件必须具有完全限定的名称。 不能以完全规范输入组件。

此错误与 CAN0025 相同。
