# <a name="compiler-warning-level-2-c4007"></a>编译器警告 （等级 2） C4007
function： 必须是属性  
  
 未显式指定的函数所需的属性。 例如，函数**主要**必须具有`__cdecl`属性。 编译器强制属性。
