# Evaluation stack overflow. The pattern may be too complex.
当在搜索字符串中使用集成开发环境 \(IDE\) 无法处理的复杂正则表达式时，通常会发生此错误。  
  
### 更正此错误  
  
1.  减少所包含内容和子句的数量，然后再试一次。
