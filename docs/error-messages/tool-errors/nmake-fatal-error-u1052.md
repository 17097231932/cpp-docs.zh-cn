# <a name="nmake-fatal-error-u1052"></a>NMAKE 错误 U1052

> 文件*文件名*找不到

NMAKE 找不到具有以下值之一指定的文件：

- **/F**选项

- **!包括**预处理指令

- At 符号 (**\@**) 的响应文件说明符

检查存在该文件，文件名的拼写正确。
