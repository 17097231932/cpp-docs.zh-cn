# The following components could not be browsed: &lt;name&gt;
该错误通常发生在尝试将不支持对象浏览的文件类型（如 .htm 和 .txt 文件）添加到“组件选择器”对话框时。  “对象浏览器”支持 .bsc、.olb、.tlb、.dll、.exe、.ocx 等文件类型。  
  
### 更正此错误  
  
1.  将不支持的文件类型从“选择”组件列表中移除并选择另一个的文件。
