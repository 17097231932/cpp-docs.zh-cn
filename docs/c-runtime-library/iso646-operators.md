# <a name="iso646-operators"></a>ISO646 运算符

提供某些运算符或标点符号的可读替代项。 标准标头 \<iso646.h> 在独立实现中也可用。

## <a name="macros"></a>宏

|名称|说明|
|-|-|
|[and](../c-runtime-library/reference/and.md)|&& 运算符的替代项。|
|[and_eq](../c-runtime-library/reference/and-eq.md)|&= 运算符的替代项。|
|[bitand](../c-runtime-library/reference/bitand.md)|& 运算符的替代项。|
|[bitor](../c-runtime-library/reference/bitor.md)|&#124; 运算符的替代项。|
|[compl](../c-runtime-library/reference/compl.md)|~ 运算符的替代项。|
|[not](../c-runtime-library/reference/not.md)|! 的替代项 运算符。|
|[not_eq](../c-runtime-library/reference/not-eq.md)|!= 运算符的替代项。|
|[or](../c-runtime-library/reference/or.md)|&#124;&#124; 运算符的替代项。|
|[or_eq](../c-runtime-library/reference/or-eq.md)|&#124;= 运算符的替代项。|
|[运算](../c-runtime-library/reference/xor.md)|^ 运算符的替代项。|
|[xor_eq](../c-runtime-library/reference/xor-eq.md)|^= 运算符的替代项。|
