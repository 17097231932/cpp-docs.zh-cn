# <a name="editable-file-types-for-resources-c"></a>资源 （c + +） 的可编辑的文件类型

可以打开下列类型的文件，并可以编辑它们包含的资源。

|文件名|描述|
|---------------|-----------------|
|.rc|资源脚本文件。|
|.rct|资源模板文件。|
|.res|资源文件。|
|.resx|托管资源文件。|
|.exe|可执行文件。|
|.dll|动态链接库文件。|
|.bmp、.ico、.dib 和 .cur|位图、图标、工具栏和光标文件。|

## <a name="requirements"></a>要求

Win32
