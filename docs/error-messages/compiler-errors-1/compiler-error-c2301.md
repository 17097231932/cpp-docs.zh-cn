# 编译器错误 C2301
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"\-\>~identifier" 的左侧必须指向类\/结构\/联合  
  
 `->` 运算符左侧表达式的计算结果不是指向类、结构或联合的指针。
