# <a name="window-dialog-and-control-classes"></a>窗口、对话框和控件类

类 `CWnd` 及其派生类封装一个 `HWND` ，即 Windows 窗口的句柄。 `CWnd` 可以单独使用，也可以用作派生新类的基础。 类库提供的派生类表示各种类型的窗口。

[CWnd](../mfc/reference/cwnd-class.md)<br/>
所有窗口的基类。 你可以使用从派生的类之一 `CWnd` 或直接从中派生你自己的类。
