# <a name="sdword"></a>SDWORD

分配并选择性地为每个*初始值设定项*初始化有符号双字（4字节）的存储。 还可在类型合法的任何位置用作类型说明符。

## <a name="syntax"></a>语法

> *名称* **SDWORD** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
