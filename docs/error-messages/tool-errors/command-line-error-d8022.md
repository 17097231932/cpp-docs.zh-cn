# <a name="command-line-error-d8022"></a>命令行错误 D8022
无法打开邮件文件  
  
 给定的文件有不在当前目录或在指定的目录中的 PATH 环境变量。 消息文件包含编译器命令行语法和选项的简短的摘要。  
  
 将此文件移到当前目录或当前路径中的目录。 如果找不到此文件，运行安装程序从分发磁盘复制它。
