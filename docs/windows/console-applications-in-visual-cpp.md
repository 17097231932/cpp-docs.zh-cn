# <a name="console-applications-in-visual-c"></a>Visual C++ 中的控制台应用程序
控制台应用程序接受输入并向控制台发送输出，输入和输出数据也称为命令提示。 可以创建控制台应用以执行基本操作或非常复杂的任务。 还可以使用控制台应用对你稍后想要合并到 Windows 桌面应用程序或通用 Windows 平台应用中的功能进行概念证明演示。 控制台应用可通过管道或其他 RPC 机制与其他桌面应用进行通信。
