# <a name="ltinitializer_listgt"></a>&lt;initializer_list&gt;

定义容器类模板 `initializer_list` 和多个支持模板。

## <a name="syntax"></a>语法

```cpp
#include <initializer_list>
```

### <a name="classes"></a>类

|类|描述|
|-|-|
|[initializer_list](../standard-library/initializer-list-class.md)|提供访问元素数组的权限，其中数组的每个成员均具有指定的类型。|
