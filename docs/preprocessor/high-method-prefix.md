# high_method_prefix
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 专用**  
  
 指定用于命名高级属性和方法的前缀。  
  
## 语法  
  
```  
high_method_prefix("Prefix")  
```  
  
#### 参数  
 `Prefix`  
 要使用的前缀。  
  
## 备注  
 默认情况下，由未使用前缀命名的成员函数公开高级错误处理属性和方法。  这些名称来自类型库。  
  
 **结束 C\+\+ 专用**
