# 链接器工具错误 LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

已损坏或无效的 COFF 符号表（未定义的静态或标签符号）  
  
 给定对象的某一部分中由编译器生成的内容看来已损坏。  
  
 此错误可能表示编译器的失败。
