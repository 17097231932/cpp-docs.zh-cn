# <a name="ml-nonfatal-error-a2063"></a>ML 非致命错误 A2063

**仅为 2 的幂可以对齐： 表达式**

使用指定的表达式[对齐](../../assembler/masm/align-masm.md)指令无效。

**对齐**表达式必须是介于 2 和 256 之间，2 的幂，并且必须是小于或等于当前段、 结构或联合的对齐方式。
