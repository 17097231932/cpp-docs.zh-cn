# <a name="ymmword"></a>YMMWORD
使用 256 位多媒体操作数与 Intel 高级矢量扩展 (AVX) 说明。  
  
## <a name="syntax"></a>语法  
  
```  
YMMWORD  
```  
  
## <a name="remarks"></a>备注  
 `YMMWORD` 用于表示与相同的类型`__m256`对 AVX 内部函数。
