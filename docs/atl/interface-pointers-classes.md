# <a name="interface-pointers-classes"></a>接口指针类
以下类管理给定的接口指针：  
  
-   [CComPtr](../atl/reference/ccomptr-class.md)执行自动引用计数。  
  
-   [CComQIPtr](../atl/reference/ccomqiptr-class.md)与类似`CComPtr`，但还要执行自动查询的接口。  
  
-   [CInterfaceArray](../atl/reference/cinterfacearray-class.md)提供方法构造的 COM 数组时有用接口指针。  
  
-   [CInterfaceList](../atl/reference/cinterfacelist-class.md)提供方法构造的 COM 列表时有用接口指针。  
  
-   [CComGITPtr](../atl/reference/ccomgitptr-class.md)提供接口指针和全局接口表 (GIT) 的方法来处理。
