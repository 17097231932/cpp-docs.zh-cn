# <a name="operator-sectionrel"></a>运算符 SECTIONREL

返回相对于在最终可执行文件中包含目标的节的表达式相对偏移区。

## <a name="syntax"></a>语法

> **SECTIONREL** *表达式*

## <a name="remarks"></a>备注

**SECTIONREL**仅可用于显示 COFF 对象。
