# <a name="file-position-in-append-mode"></a>追加模式中的文件位置

**ANSI 4.9.3** 追加模式流的文件位置指示器最初是位于文件的开头还是结尾

在追加模式中打开文件时，文件位置指示器最初指向文件的末尾。
