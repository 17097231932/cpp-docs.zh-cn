# <a name="compiler-error-c2245"></a>编译器错误 C2245

将不存在的成员函数“function”指定为友元(成员函数签名与所有重载都不匹配)

编译器未找到指定为友元的函数。

下面的示例生成 C2245：

```
// C2245.cpp
// compile with: /c
class B {
   void f(int i);
};

class A {
   int m_i;
   friend void B::f(char);   // C2245
   // try the following line instead
   // friend void B::f(int);
};

void B::f(int i) {
   A a;
   a.m_i = 0;
}
```
