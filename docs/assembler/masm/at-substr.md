# <a name="substr"></a>\@SubStr

一个宏函数，该函数返回从*位置*开始的子字符串。

## <a name="syntax"></a>语法

> **\@SubStr （** *字符串* __，__ *位置*⟦ __，__ *长度*⟧ **）**
