# “项目加载失败”对话框
如果无法加载项目文件，则将出现此对话框。  
  
 **请更正指定的错误，然后重新加载项目。**  
 提供与错误有关的信息。  使用此对话框中的信息查找并解决项目文件中的错误。  保存项目，然后在 IDE 中重新加载该项目。  
  
 **在此会话中不再显示该对话框**  
 如果选择此选项，在下次打开 IDE 之前将不会出现**“项目加载失败”**对话框。
