# 编译器错误 C2648
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 将成员作为默认参数使用要求静态成员  
  
 非静态成员被用作默认参数。  
  
 下面的示例生成 C2648：  
  
```  
// C2648.cpp  
// compile with: /c  
class C {  
public:  
   int i;  
   static int j;  
   void func1( int i = i );  // C2648  i is not static  
   void func2( int i = j );  // OK  
};  
```
