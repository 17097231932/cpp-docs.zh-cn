# 编译器错误 C3501
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

没有为 ProgID“progid”注册的类型库  
  
 给定的 progid 的类 ID 没有关联的类型库。  因此，不能向 `#import` 语句传递此 progid。
