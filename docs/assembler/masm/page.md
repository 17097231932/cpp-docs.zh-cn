# <a name="page"></a>PAGE

第一个指令设置程序列表的行 *长度* 和字符 *宽度* 。 如果未提供任何参数，则将生成分页符。 第二个指令递增节号，并将页码重置为1。

## <a name="syntax"></a>语法

> **PAGE** ⟦*length*⟧⟦__，__ *width*⟧

> **PAGE +**
