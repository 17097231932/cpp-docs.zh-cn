# 没有外部链接的重要字符
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** 不具有外部链接的重要字符数目  
  
 标识符对 247 个字符是有意义的。  编译器不限制可以在标识符中使用的字符数；它只会忽略限制之外的任何字符。
