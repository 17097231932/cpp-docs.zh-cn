# 资源编译器错误 RC2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

应输入控件类名  
  
 **DIALOG** 语句中的 CONTROL 语句的 `class` 字段必须为下列类型之一：BUTTON、COMBOBOX、EDIT、LISTBOX、SCROLLBAR、STATIC 或用户定义的。  确保正确拼写此类。
