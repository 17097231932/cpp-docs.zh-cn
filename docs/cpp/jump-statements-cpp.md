# 跳转语句 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ 跳转语句执行控制的即时本地转换。  
  
## 语法  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
