# 资源编译器错误 RC1023
无法打开源文件“filename”  
  
 指定的文件不存在、无法打开或找不到。  
  
 确保环境设置有效，并且已为该文件指定了正确的路径。  
  
 如果出现错误但未出现错误消息，则资源编译器已用完文件句柄。
