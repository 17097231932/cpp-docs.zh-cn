# “常规”属性页（文件）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

在**“解决方案资源管理器”**中选定一个文件后，**“配置属性”**节点下的**“常规”**属性页包含下面的属性：  
  
 **从生成中排除**  
 指定文件是否应在当前配置的生成中。  
  
 若要以编程方式访问此属性，请参见 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild%2A>。  
  
 **工具**  
 将用于生成此文件的工具。  有关更多信息，请参见[指定自定义生成工具](../ide/specifying-custom-build-tools.md)。  
  
 若要以编程方式访问此属性，请参见 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool%2A>。  
  
 有关如何访问**“配置属性”**节点下的**“常规”**属性页的信息，请参见 [如何：用属性页指定项目属性](../misc/how-to-specify-project-properties-with-property-pages.md)。
