# The application cannot shut down.
此错误信息通常出现在外部程序（例如任务管理器）尝试关闭 Visual Studio，而 Visual Studio 仍有打开的模式对话框时。  
  
### 更正此错误  
  
1.  关闭有模式对话框并重试。
