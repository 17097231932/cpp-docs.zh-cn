# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

将列映射项的结束。  
  
## 语法  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## 备注  
 它用于行集的一个访问器。  完成 `BEGIN_COLUMN_MAP` 宏与 `END_COLUMN_MAP` 宏。  
  
## 示例  
 参见 [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md)。  
  
## 要求  
 **标头:** atldbcli.h
