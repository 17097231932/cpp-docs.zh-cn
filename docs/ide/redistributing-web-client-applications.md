# 重新发布 Web 客户端应用程序
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

如果应用程序使用实现 WebBrowser 控件的 MFC 类（例如，`CHtmlView` 或 `CHtmlEditView`），则必须在目标计算机上至少最小安装 Microsoft Internet Explorer 4.0 或更高版本。  
  
 安装 Internet Explorer 的最新版本也将确保目标计算机有最新的公共控件文件。  
  
 有关安装最小限度的 Internet Explorer 组件的信息可以在下面的知识库文章中找到：  
  
-   “Q185375, HOWTO: Create a Single EXE Install of Internet Explorer”（Q185375，如何创建 Internet Explorer 的单个 EXE 安装）\([http:\/\/support.microsoft.com\/support\/kb\/articles\/q185\/3\/75.asp](http://support.microsoft.com/support/kb/articles/q185/3/75.asp)\)  
  
 可以在 MSDN Library 中或 [http:\/\/support.microsoft.com\/default.aspx?ln\=zh\-cn](http://support.microsoft.com/default.aspx?ln=zh-cn) 上查找知识库文章。
