# <a name="compiler-warning-level-1-c4555"></a>编译器警告（等级 1）C4555

表达式无效；应输入带副作用的表达式

当表达式不起作用时，此警告将通知您。

默认情况下，此警告处于关闭状态。 请参阅 [默认情况下处于关闭状态的编译器警告](../../preprocessor/compiler-warnings-that-are-off-by-default.md) 了解详细信息。

例如：

```cpp
// C4555.cpp
// compile with: /W1
#pragma warning(default:4555)

void func1()
{
   1;   // C4555
}

void func2()
{
   int x;
   x;   // C4555
}

int main()
{
}
```
