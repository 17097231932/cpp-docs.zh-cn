# <a name="compiler-error-c3159"></a>编译器错误 C3159
指针： 不能声明为指向值类型的指针的数组  
  
 无法声明指向值类型的指针的数组。  
  
 C3159 才可访问使用过时的编译器选项**/clr:oldSyntax**。
