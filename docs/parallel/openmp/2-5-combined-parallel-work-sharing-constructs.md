# <a name="25-combined-parallel-work-sharing-constructs"></a>2.5 组合的并行工作共享构造
组合的并行工作共享构造是用于指定并行区域包含只有一个工作共享构造的快捷方式。 这些指令的语义相等的显式指定**并行**指令跟 single 的工作共享构造。  
  
 下列各节描述组合并行工作共享构造：  
  
-   **为并行**指令。  
  
-   **并行部分**指令。
