# 安装 ATL 数据库支持
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

作为基准安装的一部分，安装程序安装支持 OLE DB 编程所必需的所有组件。  OLE DB 接口作为 MDAC SDK 的一部分被安装。  OLE DB 模板支持在 Visual C\+\+ .NET 中进行 OLE DB 编程，它是作为 ATL 库的一部分安装的。  Visual C\+\+ .NET 安装程序同时安装 MDAC SDK（2.8 版）和 ATL 库。
