# <a name="compiler-warning-level-3-c4283"></a>编译器警告（等级 3）C4283

并通过类型 type

警告此延续[C4281](../../error-messages/compiler-warnings/compiler-warning-level-3-c4281.md)表明： 通过 **-> 运算符**调用其本身通过`type`。
