# CMultipleResults 类
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果您想让命令处理多个结果集，对于 `CCommand` 模板参数 *TMultiple*请使用 `CMultipleResults`。  
  
## 语法  
  
```  
class CMultipleResults  
```  
  
## 备注  
 若要处理多个结果集，[CCommand](../../data/oledb/ccommand-class.md) 必须从此类继承。  
  
## 要求  
 **标头:** atldbcli
