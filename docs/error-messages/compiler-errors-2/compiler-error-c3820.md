# <a name="compiler-error-c3820"></a>编译器错误 C3820

> "*name*"：初始值设定项必须是托管的

## <a name="remarks"></a>备注

托管类型的 *name* 初始值设定项也必须是托管的。 若要更正此错误，请指定托管初始值设定项方法。
