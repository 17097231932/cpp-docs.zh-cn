##  <a name="addpane"></a>CMDIFrameWndEx::AddPane  
 注册的扩展管理器窗格。  
  
```  
BOOL AddPane(
    CBasePane* pControlBar,  
    BOOL bTail=TRUE);
```  
  
### <a name="parameters"></a>参数  
 [in] `pControlBar`  
 指针，指向注册窗格中。  
  
 [in] `bTail`  
 指定是否将此窗格添加到列表的末尾。  
  
### <a name="return-value"></a>返回值  
 如果成功注册窗格中，则，返回非零值。 如果窗格中已注册的扩展管理器，返回 0。  
  
### <a name="remarks"></a>备注  
 必须注册每个窗格[CDockingManager 类](../../mfc/reference/cdockingmanager-class.md)它可以参与停靠布局之前。 使用此方法来通知你想要将特定窗格停靠停靠管理器。 该窗格注册后，扩展管理器将其基于其对齐方式设置和维护的扩展管理器窗格的列表中的位置对齐。

##  <a name="adjustdockinglayout"></a>CMDIFrameWndEx::AdjustDockingLayout  
 重新计算所有停靠窗格的布局。  
  
```  
virtual void AdjustDockingLayout(HDWP hdwp=NULL);
```  
  
### <a name="parameters"></a>参数  
 [in] `hdwp`

### <a name="remarks"></a>备注  
 调用该成员函数以重新计算所有窗格停靠在框架窗口的布局。

##  <a name="enablemditabbedgroups"></a>CMDIFrameWndEx::EnableMDITabbedGroups  
 启用或禁用框架窗口的 MDI 选项卡式的组功能。

```  
void EnableMDITabbedGroups(
    BOOL bEnable,  
    const CMDITabInfo& params);
```  
  
### <a name="parameters"></a>参数  
 [in] `bEnable`  
 如果`TRUE`，启用了 MDI 选项卡式的组功能; 如果`FALSE`，MDI 选项卡式的组功能已禁用。

 当所有选项卡式的窗口水平对齐，或只有一个选项卡式的窗口时，您可以创建新的水平选项卡组。
