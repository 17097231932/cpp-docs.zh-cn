# <a name="linker-tools-error-lnk1141"></a>链接器工具错误 LNK1141

在导出文件生成过程失败

[链接](../../build/reference/linking.md)无法生成导出 (.exp) 文件。

此错误的原因选项或模块定义语句中包括系统出现问题，如没有足够的内存和语法错误。

此错误之前会提供附加信息的另一个错误。
