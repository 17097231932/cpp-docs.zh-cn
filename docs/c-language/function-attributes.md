# 函数特性
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 专用**  
  
 可选的 *attribute\-seq* 非终止符允许您逐个函数地选择调用约定。  您还可将函数指定为 `__fastcall` 或 `__inline`。
