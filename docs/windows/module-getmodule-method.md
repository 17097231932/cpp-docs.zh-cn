# <a name="modulegetmodule-method"></a>Module::GetModule 方法
创建模块的实例。  
  
## <a name="syntax"></a>语法  
  
```cpp  
static Module& GetModule();  
WRL_NOTHROW static Module& GetModule();  
```  
  
## <a name="return-value"></a>返回值  
 对模块的引用。  
  
## <a name="requirements"></a>要求  
 **标头：** module.h  
  
 **命名空间：** Microsoft::WRL
