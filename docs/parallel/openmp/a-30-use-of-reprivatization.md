# <a name="a30---use-of-reprivatization"></a>A.30   重新私有化的使用
下面的示例演示变量的 reprivatization。 可以标记私有变量`private`再次嵌套指令中。 它们无需在封闭的并行区域中共享。  
  
```  
int i, a;  
...  
#pragma omp parallel private(a)  
{  
  ...  
  #pragma omp parallel for private(a)  
  for (i=0; i<10; i++)  
     {  
       ...  
     }  
}  
```
