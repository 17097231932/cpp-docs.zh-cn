# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

`CMFCTabCtrl`类提供的选项卡控件的功能。 选项卡控件在其顶部或底部显示具有平面或三维选项卡的可停靠窗口。 选项卡可以显示文本和图像，并可在处于活动状态时更改颜色。

## <a name="syntax"></a>语法

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>备注

`CMFCTabCtrl`类支持：

- 包括 3D，平面，和固定的共享水平滚动条控件样式的选项卡。

- 选项卡位于顶部或底部的窗口。

- 选项卡来显示文本、 图像或文本和图像。

- 选项卡处于活动状态时更改颜色的选项卡。

- 可调整的选项卡边框大小更改。

- 可拆分的选项卡式的窗口。

`CMFCTabCtrl`类可用于对话框中，但适用于应用程序使用停靠控件条像 Microsoft Excel 和 Visual Studio 进行的。 有关详细信息，请参阅[CDockablePane 类](../../mfc/reference/cdockablepane-class.md)。

请按照这些步骤添加可调整大小，停靠在应用程序中的选项卡控件：

1. 创建的实例[CTabbedPane 类](../../mfc/reference/ctabbedpane-class.md)。

1. 调用[CDockablePane::Create](../../mfc/reference/cdockablepane-class.md#create)。

1. 使用[cbasetabbedpane:: Addtab](../../mfc/reference/cbasetabbedpane-class.md#addtab)或[cmfcbasetabctrl:: Inserttab](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab)添加新选项卡。

1. 调用[CBasePane::EnableDocking](../../mfc/reference/cbasepane-class.md#enabledocking) ，以便当前的停靠选项卡控件可以停靠在主框架窗口。

1. 调用[CFrameWndEx::DockPane](../../mfc/reference/cframewndex-class.md#dockpane)停靠在主框架在选项卡式的窗口。

有关如何创建作为停靠控件条选项卡式的窗口的示例，请参阅[CTabbedPane 类](../../mfc/reference/ctabbedpane-class.md)。 若要使用`CMFCTabCtrl`为非停靠控件，创建`CMFCTabCtrl`对象，然后调用[CMFCTabCtrl::Create](#create)。

##  <a name="create"></a>  CMFCTabCtrl::Create

创建选项卡控件，并将其附加到`CMFCTabCtrl`对象。

```
BOOL Create(
    Style style,
    const RECT& rect,
    CWnd* pParentWnd,
    UINT nID,
    Location location=LOCATION_BOTTOM,
    BOOL bCloseBtn=FALSE);
```

### <a name="parameters"></a>参数

*style*<br/>
[in]选项卡控件的样式。 有关更多信息，请参见“备注”。

*rect*<br/>
[in]限定选项卡控件的矩形。

*pParentWnd*<br/>
[in]指向父窗口的指针。 不能为 NULL。

*nID*<br/>
[in]选项卡控件的 ID。

*location*<br/>
[in]选项卡的位置。 默认值为 LOCATION_BOTTOM。 有关更多信息，请参见“备注”。

*bCloseBtn*<br/>
[in]若要显示的选项卡; 上的关闭按钮，则返回 TRUE否则为 FALSE。 默认值是 FALSE。

### <a name="return-value"></a>返回值

如果成功，则为 TRUE否则为 FALSE。
