# 表达式计算器错误 CXX0069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

变量需要堆栈帧  
  
 表达式计算器无法计算变量，因为它不出现在堆栈帧中。  这可能是由作为内联函数一部分声明的变量引起的。
