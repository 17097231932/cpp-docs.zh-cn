# <a name="interfaces-atl-control-wizard"></a>接口，ATL 控件向导

向导的此页标识控件支持的接口。 默认情况下，支持的接口是大多数容器通常使用的接口。

> [!NOTE]
> 如果在 "[选项](../../atl/reference/options-atl-control-wizard.md)" 选项卡上选择了 "**最小控制**"，则默认情况下，不会在 **受支持** 的列表框中显示

- **不支持**

   指示控件当前不支持的可用接口。

- **支持**

   指示控件当前支持的接口。

   |传输按钮|说明|
   |---------------------|-----------------|
   |**>**|将当前在 **不受支持** 的列表中选择的接口名称添加到 **受支持** 的列表中。|
   |**>>**|将所有接口名称添加到 **受支持****的列表** 中。|
   |**\<**|删除 **受支持** 列表中当前选定的接口名称。|
   |**\<\<**|删除 **受支持** 列表中当前列出的所有接口名称。|
