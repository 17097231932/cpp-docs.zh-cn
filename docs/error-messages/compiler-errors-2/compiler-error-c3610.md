# <a name="compiler-error-c3610"></a>编译器错误 C3610
valuetype： 在调用方法 method 之前必须将值类型装箱  
  
 默认情况下，值类型不是托管堆上。 你可以调用的方法从.NET 运行时类，如之前`Object`，你需要将值类型移至托管堆。  
  
 C3610 才可访问使用过时的编译器选项**/clr:oldSyntax**。
