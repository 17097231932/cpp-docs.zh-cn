# <a name="compiler-error-c3818"></a>编译器错误 C3818
数组属性声明 property1 应重载相等 property2' 的索引属性  
  
 重载是不可能的属性，一个是一个索引器，另一个是数组属性。 
  
 C3818 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
