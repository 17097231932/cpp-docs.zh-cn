# <a name="linker-tools-warning-lnk4044"></a>链接器工具警告 LNK4044

无法识别的选项 "option";掉

给定选项不是此工具的有效选项。 此工具忽略了选项。

若要查看有效选项的列表，请运行不带参数或选项的工具。
