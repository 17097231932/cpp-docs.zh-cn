# <a name="statements-limits-on-switch-statements"></a>语句：对 Switch 语句的限制

ANSI 3.6.4.2 `switch` 语句中的 `case` 值数量上限

Microsoft C 没有限制 `switch` 语句中的 `case` 值数量。 该数字仅受可用内存限制。
