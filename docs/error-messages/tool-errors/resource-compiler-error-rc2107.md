# <a name="resource-compiler-error-rc2107"></a>资源编译器错误 RC2107

**应输入数字的命令值**

资源编译器需要数值*idvalue*字段中**加速器**语句。 请确保你已使用`#define`常量，以指定的值和常量拼写正确。
