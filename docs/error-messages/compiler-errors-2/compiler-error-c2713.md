# <a name="compiler-error-c2713"></a>编译器错误 C2713

> 每个函数只允许一种形式的异常处理

不能在同一函数中使用结构化异常处理 (**`__try`** / **`__except`**) 和 c + + 异常处理 (**`try`** / **`catch`**) 。
