# <a name="compiler-error-c2121"></a>编译器错误 C2121
'#': 无效字符： 可能宏展开的结果  
  
 无效的 # 字符可能由不正确的宏，而不是字符串化运算符使用标记粘贴运算符 （#） 插入 （#）。
