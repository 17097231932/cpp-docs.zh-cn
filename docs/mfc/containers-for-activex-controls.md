# <a name="containers-for-activex-controls"></a>ActiveX 控件的容器

可以使用 ActiveX 控件在视觉对象中开发C++中其他应用程序，只要它们支持 ActiveX 控件包容。 大量的 Microsoft 应用程序，从列出，版本支持 ActiveX 控件包容。

>[!IMPORTANT]
> ActiveX 是一项传统技术，不应使用新的开发。 有关取代 ActiveX 的现代技术的详细信息，请参阅[ActiveX 控件](activex-controls.md)。

> [!NOTE]
>  以下列表并不支持 ActiveX 控件，但表示一组测试中最常用的应用程序的完整列表：

支持 ActiveX 控件包含的应用程序包括：

- Internet Explorer 3.x 及更高版本

- Visual 基本 4.x 及更高版本

- Visual C++ 4.x 及更高版本

- 访问 95 及更高版本

- Excel 97 及更高版本

- Word 97 及更高版本

- 访问 97 及更高版本

- FrontPage 97 及更高版本

- PowerPoint 97 及更高版本

- Visual InterDev 97 及更高版本

以下是支持 ActiveX 控件包含的非 Microsoft 应用程序：

- PowerBuilder

- Delphi

- C++生成器

- NCompass 插件用于 Netscape Navigator
