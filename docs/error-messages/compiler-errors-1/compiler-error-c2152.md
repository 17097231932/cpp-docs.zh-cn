# <a name="compiler-error-c2152"></a>编译器错误 C2152
identifier： 指向具有不同的属性的函数的指针  
  
 使用一个调用约定函数指针 (`__cdecl`， `__stdcall`，或`__fastcall`) 分配给使用另一个调用约定函数指针。
