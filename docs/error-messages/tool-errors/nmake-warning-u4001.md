# <a name="nmake-warning-u4001"></a>NMAKE 警告 U4001

> 命令文件只能从命令行调用

由 at 符号 () 说明符调用的命令文件 **\@** 不能包含另一个命令文件的规范。 不允许这样的嵌套。 已忽略该规范。
