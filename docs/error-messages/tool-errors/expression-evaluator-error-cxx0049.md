# 表达式计算器错误 CXX0049
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

已过时的 OMF \- 请重新链接程序  
  
 程序使用了旧的 OMF（对象模块格式）。  
  
 使用当前的链接器版本重新链接程序。  
  
 该错误与 CAN0049 相同。
