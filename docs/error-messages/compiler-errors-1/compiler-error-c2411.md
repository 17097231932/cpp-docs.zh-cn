# 编译器错误 C2411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”:“context”中的非法结构\/联合成员  
  
### 通过检查以下可能的原因进行修复  
  
1.  该标识符不是此上下文中可视结构或联合的成员。  
  
2.  该标识符不是用成员选择运算符 \(.\) 指定的结构或联合的成员。
