# _bstr_t::Detach
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 专用**  
  
 返回 `_bstr_t` 包装的 `BSTR` 并从 `_bstr_t` 中分离 `BSTR`。  
  
## 语法  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## 返回值  
 由 `_bstr_t` 包装的 `BSTR`。  
  
## 示例  
 有关使用 **Detach** 的示例，请参阅 [\_bstr\_t::Assign](../cpp/bstr-t-assign.md)。
