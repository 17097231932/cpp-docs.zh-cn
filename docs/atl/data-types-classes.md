# Data Types Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类包装C\+\+数据类型:  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) 包装 `BSTR` 数据类型。  
  
-   [CComVariant](../atl/reference/ccomvariant-class.md) 包装 **VARIANT** 数据类型。  
  
-   [CComCurrency](../atl/reference/ccomcurrency-class.md) 包括方法和运算符创建并管理的 `CURRENCY` 对象。
