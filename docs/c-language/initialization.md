# <a name="initialization"></a>初始化
“初始值设定项”是要赋给正在声明的变量的一个值或一系列值。 通过在变量声明中将初始值设定项应用于声明符，可以将变量设置为初始值。 初始值设定项的值将赋给变量。  
  
 以下各节介绍如何初始化[标量](../c-language/initializing-scalar-types.md)、[聚合](../c-language/initializing-aggregate-types.md)和[字符串](../c-language/initializing-strings.md)类型的变量。 “标量类型”包括所有算术类型，还包括指针。 “聚合类型”包括数组、结构和联合。
