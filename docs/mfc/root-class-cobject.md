# 根类：CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

大多数类在 Microsoft 基础类 \(MFC\) \(MFC\) 库中将在类层次结构的根的一个基类中派生。  `CObject` 提供了许多有用功能从其派生的所有类，使用太低开销。  有关 `CObject` 及其功能的更多信息，请参见 [使用 CObject](../mfc/using-cobject.md)。  
  
 [CObject](../mfc/reference/cobject-class.md)  
 大多数 MFC 类的基类。  序列化数据并获取有关类支持的运行时信息。  
  
 [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)  
 用于确定配置的对象的具体类在运行时。
