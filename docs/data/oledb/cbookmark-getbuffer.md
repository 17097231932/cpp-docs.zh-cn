# CBookmark::GetBuffer
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## 语法  
  
```  
  
virtual BYTE* GetBuffer( ) const throw( );  
  
```  
  
## 返回值  
 向书签缓冲区的指针。
