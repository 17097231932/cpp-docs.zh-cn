# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回部分相对偏移量表达式相对于包含该目标的部分在最终可执行文件中。  
  
## 语法  
  
```  
SECTIONREL expression  
```  
  
## 备注  
 SECTIONREL 为 COFF 对象显示可用。
