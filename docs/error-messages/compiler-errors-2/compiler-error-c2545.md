# <a name="compiler-error-c2545"></a>编译器错误 C2545

operator： 无法查找重载运算符

运算符不能与提供的操作数一起使用。 必须提供与所需的操作数重载的运算符。

如果操作数具有不正确的类型，可以导致此错误。

如果定义了转换运算符或构造函数采用单个参数，则可能会修复此错误。
