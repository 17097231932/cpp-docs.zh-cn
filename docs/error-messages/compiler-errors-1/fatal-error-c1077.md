# <a name="fatal-error-c1077"></a>错误 C1077

编译器限制：命令行选项数不得超过其限定值

命令行选项数超过了内部限制。

可能多用 [/D](../../build/reference/d-preprocessor-definitions.md)定义的符号太多。 （改为将定义放置在头文件中。）
