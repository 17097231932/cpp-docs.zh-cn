# <a name="expression-evaluator-error-cxx0049"></a>表达式计算器错误 CXX0049

过时的 OMF-请重新链接程序

程序使用旧的 OMF (对象模块格式) 。

使用当前的链接器版本重新链接程序。

此错误与 CAN0049 相同。
