# <a name="fatal-error-c1206"></a>错误 C1206

安装的运行时版本不支持 per-appdomain 数据

某些功能（如每个应用程序域数据）只由支持该功能的公共语言运行库支持。

C1206 指出计算机上未安装运行库的最新版本。 请安装适用于你的编译器的公共语言运行时版本。

有关更多信息，请参见 [appdomain](../../cpp/appdomain.md) 。
