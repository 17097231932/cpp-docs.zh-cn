# <a name="compiler-error-c3333"></a>编译器错误 C3333
“type library”：无法 #import 损坏的类型库  
  
 编译器无法读取 `#import` 语句中指定的类型库。 建议重新生成类型库（如可能），或从你的供应商请求提供新的类型库。 建议使用随 Visual C++ 提供的 OLE 查看器来查看类型库文件，以了解发生了什么情况。
