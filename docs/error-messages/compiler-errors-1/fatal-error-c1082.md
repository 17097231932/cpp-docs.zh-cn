# 错误 C1082
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法关闭 filetype 文件：“file”：消息  
  
 如果消息指出“文件号错误”，说明此文件在后台进行编译的同时，可能已在前台关闭。
