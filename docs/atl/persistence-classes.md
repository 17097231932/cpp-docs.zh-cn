# <a name="persistence-classes"></a>持久性类

以下类实现对象持久性：

- [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) 允许客户端将对象的属性加载并保存到属性包。

- [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) 允许客户端将对象的持久性数据加载并保存到流中。

- [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) 允许客户端将对象的持久性数据加载并保存到存储中。
