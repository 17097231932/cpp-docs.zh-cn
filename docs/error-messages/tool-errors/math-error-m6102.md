# <a name="math-error-m6102"></a>数学错误 M6102

非常规

操作生成非常小的浮点数，它是因无效而丢失基数。 非常规通常被屏蔽浮点异常，从而导致它们被捕获并且操作。

程序终止，退出代码为 130。
