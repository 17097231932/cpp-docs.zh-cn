# 使用 CProgressCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

进度栏控件用于指示较长操作的进度。  该控件由一个矩形构成，当操作进行时，该矩形逐渐填充系统突出显示颜色。  
  
 进度在 MFC 类由 [CProgressCtrl](../mfc/reference/cprogressctrl-class.md)。  
  
 在最初创建进度时，应指定其大小和位置、父窗口 \(通常对话框 ID.\) 和  通过 `dwStyle` 参数，可以为控件指定不同窗口样式和样式。它如何填充。
