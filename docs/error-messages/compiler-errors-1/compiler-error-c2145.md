# 编译器错误 C2145
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

语法错误：标识符的前面缺少“token”  
  
 编译器预期 `token`，却找到了标识符。  
  
 块中的最后一个声明后，缺少分号可能会导致此错误。
