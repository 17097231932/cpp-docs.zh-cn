# <a name="deferlockt-structure"></a>defer_lock_t 结构
表示一种类型，它定义 [defer_lock](../standard-library/mutex-functions.md#defer_lock) 对象以用于选择其中一个 [unique_lock](../standard-library/unique-lock-class.md) 的重载构造函数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
struct defer_lock_t;
```
