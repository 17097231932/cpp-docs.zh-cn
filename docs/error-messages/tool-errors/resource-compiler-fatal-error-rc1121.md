# 资源编译器错误 RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

读取文件时出现 I\/O 错误  
  
 资源编译器无法读取文件。  检查包含此文件的驱动器是否可用以及此文件是否有效。
