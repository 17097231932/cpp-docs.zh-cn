# <a name="386-32-bit-masm"></a>. 386 （32）

为80386处理器启用非特权指令的程序集;禁用由更高版本的处理器引入的指令的程序集。 （仅限32位 MASM。）

## <a name="syntax"></a>语法

> **.386**

## <a name="remarks"></a>备注

还启用80387说明。
