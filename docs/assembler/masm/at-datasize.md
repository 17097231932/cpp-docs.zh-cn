# <a name="datasize"></a>\@DataSize

**对于小型、** **小型**、**中型**和**平面**模型，为 0; 对于**COMPACT**和**大型**模型，则为 1; 对于**大**模型，则为2（数值等于）。

## <a name="syntax"></a>语法

> **\@DataSize**
