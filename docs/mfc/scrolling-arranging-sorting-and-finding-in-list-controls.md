# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>在列表控件中滚动、排列、排序和查找
列表控件 ([CListCtrl](../mfc/reference/clistctrl-class.md)) 是默认情况下可滚动。 有关详细信息，请参阅[滚动位置](/windows/desktop/Controls/using-list-view-controls)Windows SDK 中并[滚动](../mfc/reference/clistctrl-class.md#scroll)成员函数。  
  
 您可以调用`CListCtrl`成员函数来排列列表项在控件中，对项进行排序和查找特定项。 有关详细信息，请参阅[排列、 排序和查找](https://msdn.microsoft.com/library/windows/desktop/bb774736)Windows SDK 中并[CListCtrl](../mfc/reference/clistctrl-class.md)成员[排列](../mfc/reference/clistctrl-class.md#arrange)， [SortItems](../mfc/reference/clistctrl-class.md#sortitems)并[FindItem](../mfc/reference/clistctrl-class.md#finditem)。
