# csin，csinf csinl
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

检索一个复杂的数字的正弦值。  
  
## 语法  
  
```  
_Dcomplex csin(   
   _Dcomplex z   
);  
_Fcomplex csin(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex csin(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex csinf(   
   _Fcomplex z   
);  
_Lcomplex csinl(   
   _Lcomplex z   
);  
```  
  
#### 参数  
 `z`  
 表示的角度以弧度为单位的复数。  
  
## 返回值  
 正弦值 `z`, ，以弧度为单位。  
  
## 备注  
 由于 C\+\+ 允许重载，因此你可以调用采用并返回 `csin` 和 `_Fcomplex` 值的 `_Lcomplex` 重载。 在 C 程序中， `csin` 始终采用并返回 `_Dcomplex` 值。  
  
## 要求  
  
|例程|C 标头|C\+\+ 标头|  
|--------|----------|--------------|  
|`csin`, `csinf`, `csinl`|\<complex.h\>|\< \> ccomplex|
