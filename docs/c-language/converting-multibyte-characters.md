# <a name="converting-multibyte-characters"></a>转换多字节字符
**ANSI 3.1.3.4** 用于将多字节字符转换为宽字符常量的相应宽字符（代码）的当前区域设置  
  
 默认情况下，当前区域设置为“C”区域设置。 可以使用 #pragma setlocale[](../preprocessor/setlocale.md) 更改该设置。
