# <a name="ml-nonfatal-error-a2063"></a>ML 非致命错误 A2063

**只能与2： expression 的幂对齐**

用 [ALIGN](align-masm.md) 指令指定的表达式无效。

**ALIGN** 表达式必须是2到256之间的2的幂，并且必须小于或等于当前段、结构或联合的对齐方式。
