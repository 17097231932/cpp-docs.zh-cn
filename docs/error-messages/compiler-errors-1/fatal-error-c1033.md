# <a name="fatal-error-c1033"></a>错误 C1033

无法打开程序数据库 pdb

此错误可能引起了磁盘错误，通过防病毒程序，尚未完全关闭的前一个调试器实例创建的临时锁或并行生成 mspdbsrv.exe 过程尝试访问同一文件中，在其他可能的原因。
