# Type library &lt;name&gt; was not found.
没有 DTE.olb 和 vsext.tlb 类型库 Visual Studio 就无法运行。  该错误通常发生在这两个文件已移除或删除时。  
  
### 更正此错误  
  
1.  重新安装 Visual Studio。
