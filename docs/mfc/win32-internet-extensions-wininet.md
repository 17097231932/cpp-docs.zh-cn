# <a name="win32-internet-extensions-wininet"></a>Win32 Internet 扩展 (WinInet)

Internet 客户端应用程序是一个程序，从网络数据源 （服务器） 使用 Internet 协议，如 gopher、 FTP 或 HTTP 访问的信息。 Internet 客户端应用程序可能会访问服务器以检索数据，例如天气映射、 股票价格或新闻标题即可，例如。 Internet 客户端可以通过外部网络 (Internet) 或 （有时称为 intranet） 的内部网络访问服务器。

MFC 包含 Win32 Internet 扩展或 WinInet，用于创建 Internet 客户端应用程序。 MFC 封装这些 Internet 扩展功能的一组标准的、 易于使用的类。 您可以通过调用 Win32 函数直接或通过使用 MFC WinInet 类编写 WinInet 客户端应用程序。

Microsoft Win32 Internet 函数 (WinInet) 可帮助促进 Internet 的任何应用程序不可或缺的一部分。 WININET 中包含的新函数。DLL，简化访问使用 HTTP （超文本传输协议）、 FTP （文件传输协议），以及 gopher Internet。
