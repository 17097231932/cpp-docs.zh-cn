# SWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化一个签名的字符串 \(2 字节\) 的每 `initializer`的存储。  ，因为类型说明符任何位置类型是非法的，也可以使用。  
  
## 语法  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
