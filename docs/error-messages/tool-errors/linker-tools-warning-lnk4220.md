# 链接器工具警告 LNK4220
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的“linker option”值“value”；假定为默认值  
  
 用 [\/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) 选项指定了超出范围的值。  **\/TLBID** 的默认值为 1。
