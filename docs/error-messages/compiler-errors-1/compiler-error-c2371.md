# <a name="compiler-error-c2371"></a>编译器错误 C2371
“identifier”：重定义；不同的基类型  
  
 已声明该标识符。  
  
 以下示例生成 C2371：  
  
```  
// C2371.cpp  
int main() {  
   int i;  
   float i;   // C2371, redefinition  
   float f;   // OK  
}  
```
