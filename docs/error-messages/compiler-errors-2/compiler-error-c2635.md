# <a name="compiler-error-c2635"></a>编译器错误 C2635
无法将转换 identifier1 * 为 identifier2\*; 隐式转换从虚拟基类  
  
 此转换所需从强制转换`virtual`基类派生的类，这不允许。  
  
 下面的示例生成 C2635:  
  
```  
// C2635.cpp  
class B {};  
class D : virtual public B {};  
class E : public B {};  
  
int main() {  
   B b;  
   D d;  
   E e;  
  
   D * pD = &d;  
   E * pE = &e;  
   pD = (D*)&b;   // C2635  
   pE = (E*)&b;   // OK  
}  
```
