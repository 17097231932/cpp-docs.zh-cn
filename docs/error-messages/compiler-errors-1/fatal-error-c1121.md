# 错误 C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

调用 CryptoAPI 失败  
  
 编译器调用 CryptoAPI 且调用失败。  重新安装 Visual Studio，可能还要重新安装操作系统。
