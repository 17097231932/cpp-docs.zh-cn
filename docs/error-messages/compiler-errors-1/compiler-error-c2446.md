# 编译器错误 C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“operator”: 没有从“type1”到“type2”的转换  
  
 编译器无法从 `type1` 转换到 `type2`。  该转换可能没有意义，因为它违反 C\/C\+\+ 语义。
