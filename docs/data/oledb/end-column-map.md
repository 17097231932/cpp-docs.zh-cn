# <a name="endcolumnmap"></a>END_COLUMN_MAP
标记列映射条目的末尾。  
  
## <a name="syntax"></a>语法  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## <a name="remarks"></a>备注  
 它使用单个访问器行集上。 `BEGIN_COLUMN_MAP` 宏以 `END_COLUMN_MAP` 宏结束。
