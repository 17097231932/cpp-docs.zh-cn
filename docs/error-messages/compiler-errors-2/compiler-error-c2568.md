# 编译器错误 C2568
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier1”: 无法解析函数重载  
  
 编译器无法确定调用哪一个重载函数。  必须转换传递给该函数的实际参数，以匹配一个重载函数的形参，但没有一个明显比所有其他重载函数更匹配的重载函数。
