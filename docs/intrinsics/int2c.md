# <a name="__int2c"></a>__int2c

**Microsoft 专用**

生成 `int 2c` 指令，这会触发 `2c` 中断。

## <a name="syntax"></a>语法

```C
void __int2c(void);
```

## <a name="requirements"></a>要求

|Intrinsic|体系结构|
|---------------|------------------|
|`__int2c`|x86、x64|

**结束 Microsoft 专用**
