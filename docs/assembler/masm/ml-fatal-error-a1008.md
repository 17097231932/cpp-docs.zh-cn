# <a name="ml-fatal-error-a1008"></a>ML 错误 A1008
**不匹配的宏嵌套**  
  
 文件或终止指令结束之前未终止任一宏[ENDM](../../assembler/masm/endm.md)找宏块之外。  
  
 此错误的一个原因是该点之前省略[。重复](../../assembler/masm/dot-repeat.md)或[。虽然](../../assembler/masm/dot-while.md)。
