# <a name="beginparammap"></a>BEGIN_PARAM_MAP
标记参数映射项的开始。  
  
## <a name="syntax"></a>语法  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>参数  
 *x*  
 [in] 用户记录类的名称。
