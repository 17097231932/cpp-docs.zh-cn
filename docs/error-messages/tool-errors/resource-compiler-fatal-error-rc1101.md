# 资源编译器错误 RC1101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未指定资源二进制文件名  
  
 “重命名输出”\(\/fo\) 选项后没有文件名。  
  
 使用以下 \/fo 选项语法：  
  
 **RC \/fo**   
 ***outfilename infilename.* rc**
