# <a name="instr"></a>INSTR

在*textitem1*中查找*textitem2*的第一个匹配项。

## <a name="syntax"></a>语法

> *name* **INSTR** ⟦*position* __，__ ⟧ *textitem1* __，__ *textitem2*

## <a name="remarks"></a>备注

起始*位置*是可选的。 每个文本项可以是文本字符串、前面带有 **%** 的常量或宏函数返回的字符串。
