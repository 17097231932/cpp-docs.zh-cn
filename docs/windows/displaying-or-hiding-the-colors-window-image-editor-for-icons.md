# 显示或隐藏“颜色”窗口（图标的图像编辑器）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 显示“颜色”窗口  
  
1.  在“图像编辑器”窗格中右击并从快捷菜单中选择“显示颜色窗口”。  
  
     \- 或 \-  
  
2.  在[“图像”菜单](../mfc/image-menu-image-editor-for-icons.md)上单击“显示颜色窗口”。  
  
### 隐藏“颜色”窗口  
  
1.  断开窗口。  这允许窗口在不使用时自动隐藏。  有关更多信息，请参见[排列窗口](../misc/how-to-arrange-and-dock-windows.md)。  
  
 \- 或 \-  
  
-   单击“关闭”按钮。  
  
 要求  
  
 无
