
使用 `CMFCPropertyGridProperty` 对象表示属性，然后将该属性添加到属性列表控件。 有关详细信息，请参阅 [CMFCPropertyGridCtrl 类](../../mfc/reference/cmfcpropertygridctrl-class.md)。

## <a name="cmfcpropertygridpropertyaddoption"></a><a name="addoption"></a> CMFCPropertyGridProperty：： AddOption

将新的列表项添加到属性列表控件中。

```
BOOL AddOption(
    LPCTSTR lpszOption,
    BOOL bInsertUnique=TRUE);
```

### <a name="parameters"></a>parameters

*lpszOption*<br/>
中要添加的列表项 (选项) 。

*bInsertUnique*<br/>
中如果只添加列表项，则为 TRUE; 否则为 false。否则为 FALSE。 默认值为 TRUE。

### <a name="return-value"></a>返回值

TRUE，表示添加列表项。 否则为 FALSE，这意味着不会添加列表项，因为 *bInsertUnique* 参数为 TRUE，并且 *lpszOption* 参数指定的列表项已存在。

### <a name="remarks"></a>备注

## <a name="cmfcpropertygridpropertyaddsubitem"></a><a name="addsubitem"></a> CMFCPropertyGridProperty：： AddSubItem

将子项添加到属性。

```
BOOL AddSubItem(CMFCPropertyGridProperty* pProp);
```

### <a name="parameters"></a>parameters

*pProp*<br/>
中指向要添加的属性的指针。

### <a name="return-value"></a>返回值

如果指定的属性已成功添加为子属性，则为 TRUE。 如果该属性已在父属性中出现，则为 FALSE。

### <a name="remarks"></a>备注

使用此方法创建父属性和子属性的层次结构列表。 添加子属性后，parent 属性会自动显示由加号 (+) 指定的展开框控件。 当用户单击加号时，父属性将展开并显示任何子属性项。

## <a name="cmfcpropertygridpropertyadjustbuttonrect"></a><a name="adjustbuttonrect"></a> CMFCPropertyGridProperty：： AdjustButtonRect

由父属性列表控件调用，用来指示属性调整嵌入按钮的边框的大小。

```
virtual void AdjustButtonRect();
```

### <a name="remarks"></a>备注

默认情况下，此方法为：

- 将按钮宽度调整为等于按钮高度加上3个像素。

- 将按钮的边框移动到属性的右边缘

- 将按钮1像素移动到属性上边缘的下边缘。

## <a name="cmfcpropertygridpropertyadjustinplaceeditrect"></a><a name="adjustinplaceeditrect"></a> CMFCPropertyGridProperty：： AdjustInPlaceEditRect

检索用于设置属性值的文本框和可选的数值调节钮控件的边界。

```
virtual void AdjustInPlaceEditRect(
    CRect& rectEdit,
    CRect& rectSpin);
```

### <a name="parameters"></a>parameters

*rectEdit*<br/>
弄此方法返回时，指定属性值文本框的边界的矩形。

*rectSpin*<br/>
弄此方法返回时，指定属性值的数值调节钮控件的边界的矩形。 或者，如果属性不支持数值调节钮，则为空矩形。

### <a name="remarks"></a>备注
