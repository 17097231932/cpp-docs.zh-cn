# <a name="ml-fatal-error-a1016"></a>ML 错误 A1016

**内部汇编程序错误**

名为 ML.exe 的 MASM 驱动程序生成了系统错误。

记录错误发生的情况，并通知 Microsoft Corporation。 提供产品支持服务 [https://support.microsoft.com/](https://support.microsoft.com/) 。
