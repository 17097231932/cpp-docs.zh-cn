# <a name="cpanecontainer-class"></a>CPaneContainer 类

类`CPaneContainer`是 MFC 实现的停靠模型的基本组件。 此类对象存储指向两个停靠窗格或两个 `CPaneContainer.` 实例的指针。它还存储分隔窗格（或容器）的分隔线的指针。 通过嵌套容器内部的容器，框架可以生成表示复杂停靠布局的二叉树。 二进制树的根存储在[CPane 容器管理器](../../mfc/reference/cpanecontainermanager-class.md)对象中。

有关详细信息，请参阅位于 Visual Studio 安装的**VC\\\\atlmfc src\\mfc**文件夹中的源代码。

## <a name="syntax"></a>语法

```
class CPaneContainer : public CObject
```
