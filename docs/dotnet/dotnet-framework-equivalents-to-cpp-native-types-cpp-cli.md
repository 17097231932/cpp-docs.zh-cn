# <a name="net-framework-equivalents-to-c-native-types-ccli"></a>对应于 C++ 本机类型的 .NET Framework 类型 (C++/CLI)
下表显示了内置的 Visual c + + 类型，它们是预定义的类型的别名关键字中**系统**命名空间。  
  
|Visual c + + 类型|.NET Framework 类型|  
|-----------------------|-------------------------|  
|**bool**|**System.Boolean**|  
|**签名 char** (请参阅[/J](../build/reference/j-default-char-type-is-unsigned.md)有关详细信息)|**System.SByte**|  
|**unsigned char**|**System.Byte**|  
|**wchar_t**|**System.Char**|  
|**double**和**长双精度**|**System.Double**|  
|**float**|**System.Single**|  
|**int**，**带符号整型**，**长**，和**签名长时间**|**System.Int32**|  
|**无符号的整数**和**无符号长**|**System.UInt32**|  
|**__int64**和**签名 __int64**|**System.Int64**|  
|unsigned __int64|**System.UInt64**|  
|**短**和**简单地说签名**|**System.Int16**|  
|**unsigned short**|**System.UInt16**|  
|**void**|**System.Void**|
