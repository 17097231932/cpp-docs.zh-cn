# <a name="forc"></a>FORC

将对*字符串*中的每个字符重复一次的块标记为每个重复的当前字符替换*参数*。

## <a name="syntax"></a>语法

> **强制***参数* __\<__ *string* __>__ \
> *语句*\
> **ENDM**

## <a name="remarks"></a>备注

与[IRPC](../../assembler/masm/irpc.md)相同。
