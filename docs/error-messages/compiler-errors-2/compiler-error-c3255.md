# 编译器错误 C3255
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“value type”: 在本机堆上无法动态分配此值类型对象  
  
 可以在堆栈上但不能在堆上创建包含托管成员的值类型的实例（请参见 [类和结构 \(托管\)](../../windows/classes-and-structs-cpp-component-extensions.md)）。  
  
 下面的示例生成 C3255：  
  
```  
// C3255.cpp  
// compile with: /clr  
using namespace System;  
value struct V {  
   Object^ o;  
};  
  
value struct V2 {  
   int i;  
};  
  
int main() {  
   V* pv = new V;   // C3255  
   V2* pv2 = new V2;  
   V v2;  
}  
```  
  
 **C\+\+ 托管扩展**  
  
 下面的示例生成 C3255：  
  
```  
// C3255b.cpp  
// compile with: /clr:oldSyntax  
#using <mscorlib.dll>  
using namespace System;  
  
__value struct V {  
   Object* o;  
};  
  
__value struct V2 {  
   int i;  
};  
  
int main() {  
   V* pv = __nogc new V;   // C3255  
   V2* pv2 = __nogc new V2;   // OK  
}  
```
