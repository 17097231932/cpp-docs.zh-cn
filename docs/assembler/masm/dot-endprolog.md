# <a name="endprolog"></a>.ENDPROLOG

指示序言声明的结尾。

## <a name="syntax"></a>语法

> **.ENDPROLOG**

## <a name="remarks"></a>备注

在[过程](proc.md)**框架**和之间使用区域外的任何序言声明都是错误的 **。ENDPROLOG**。

有关详细信息，请参阅[MASM for x64 （ml64.exe）](masm-for-x64-ml64-exe.md)。
