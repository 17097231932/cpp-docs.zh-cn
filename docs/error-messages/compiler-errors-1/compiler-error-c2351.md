# <a name="compiler-error-c2351"></a>编译器错误 C2351

已过时的 c + + 构造函数初始化语法

在新式的初始化列表中的构造函数，必须显式命名每个直接基类，即使它是唯一的基类。

下面的示例生成 C2351:

```
// C2351.cpp
// compile with: /c
class B {
public:
   B() : () {}   // C2351
   B() {}   // OK
};
```
