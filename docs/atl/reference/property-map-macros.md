# Property Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些宏定义属性映射和项。  
  
|||  
|-|-|  
|[BEGIN\_PROP\_MAP](../Topic/BEGIN_PROP_MAP.md)|标记ATL属性映射的开头。|  
|[PROP\_DATA\_ENTRY](../Topic/PROP_DATA_ENTRY.md)|指示区域或维度，ActiveX控件。|  
|[PROP\_ENTRY\_TYPE](../Topic/PROP_ENTRY_TYPE.md)|输入属性声明、属性DISPID和属性页CLSID属性映射。|  
|[PROP\_ENTRY\_TYPE\_EX](../Topic/PROP_ENTRY_TYPE_EX.md)|输入属性声明、属性DISPID，属性页CLSID和 `IDispatch` IID属性映射。|  
|[PROP\_PAGE](../Topic/PROP_PAGE.md)|输入属性页CLSID属性映射。|  
|[END\_PROP\_MAP](../Topic/END_PROP_MAP.md)|标记ATL属性映射的结尾。|
