# <a name="compiler-error-c2873"></a>编译器错误 C2873

"symbol"：符号不能用在 using 声明中

`using` 指令缺少[命名空间](../../cpp/namespaces-cpp.md)关键字。 这将导致编译器将代码误认为为[使用声明](../../cpp/using-declaration.md)，而不是[使用 using 指令](../../cpp/namespaces-cpp.md#using_directives)。
