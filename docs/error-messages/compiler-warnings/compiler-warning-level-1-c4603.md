# 编译器警告（等级 1）C4603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“\<说明符\>”：未定义宏或在预编译头使用后定义发生改变  
  
 在使用使用预编译器头后，*标识符*占位符所指定的宏不同，或者不再定义。
