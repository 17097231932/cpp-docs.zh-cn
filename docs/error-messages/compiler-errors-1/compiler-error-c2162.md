# 编译器错误 C2162
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

应输入宏形参  
  
 跟在字符串化运算符 \(\#\) 后面的标记不是一个形参名。  
  
## 示例  
 下面的示例生成 C2162：  
  
```  
// C2162.cpp  
// compile with: /c  
#include <stdio.h>  
  
#define print(a) printf_s(b)   // OK  
#define print(a) printf_s(#b)    // C2162  
```
