# MSBuild 错误 MSB3171
**MSB3171：生成清单时出现问题。'  \<错误\>”**  
  
 这是在获取有关被引用的文件或程序集的信息时发生的意外错误。  在生成应用程序清单或部署清单时，可能会产生此消息。
