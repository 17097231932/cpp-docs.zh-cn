# <a name="compiler-warning-level-1-c4411"></a>编译器警告（等级 1）C4411
identifier： 符号将解析为置换寄存器  
  
 标识符是将解析为置换寄存器并因此可以在与另一个符号操作数一起使用的本地符号。
