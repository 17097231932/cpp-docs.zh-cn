# <a name="ltcinttypesgt"></a>&lt;cinttypes&gt;

包含标准 C 库标头 \<inttypes.h>。 包含此标头还将包括 \<cstdint>。

## <a name="syntax"></a>语法

```cpp
#include <cinttypes>

```

## <a name="remarks"></a>备注

包含该标头还将确保使用标准 C 库标头中的外部链接声明的名称在 `std` 命名空间中声明。
