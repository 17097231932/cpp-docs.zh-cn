# <a name="visual-studio-2017-documentation-for-visual-c"></a>有关 Visual C++ 的 Visual Studio 2017 文档

欢迎访问! 此仓库包含正在完善的 Visual C++ 技术文档的相关源文件。文档的内容发布在 [Visual C++ 文档站点](https://docs.microsoft.com/cpp)上。

此仓库已于 2017 年 6 月 23 日从 Microsoft/vcppdocs 移至 MicrosoftDocs/cpp-docs-pr。

Visual Basic 和 Visual C# 的相关文档位于 [http://github.com/dotnet/core-docs](http://github.com/dotnet/core-docs) 下的不同仓库中，Visual Studio 2017 的相关文档位于 [http://github.com/Microsoft/visualstudio-docs](http://github.com/Microsoft/visualstudio-docs) 下的仓库中。

## <a name="contributing-to-the-documentation"></a>参与改进文档

若要参与改进此文档，请参阅[参与指南](CONTRIBUTING.md)。
我们欢迎你参与帮助我们改进 Visual C++ 文档。 此仓库中的所有文章均使用 GitHub 风格的Markdown语言。

Visual Studio 的多个类别的功能在此仓库中拥有自己的文件夹，例如关于 C++ 标准库主题的 **standard-library**、关于 Visual Studio 交互式开发环境 (IDE) 主题的 **ide** 等等。 每个文件夹中的 **/media** 子文件夹包含主题的剪贴画文件。 [参与指南](CONTRIBUTING.md)中有更多详细信息。
