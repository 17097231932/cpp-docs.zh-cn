# <a name="nmake-warning-u4007"></a>NMAKE 警告 U4007
文件名 filename 太长;截断为 8.3  
  
 给定的文件的基名称具有多于 8 个字符，或扩展具有多个三个字符。 NMAKE 截断到 8 个字符基类型和扩展名为三个字符的名称。  
  
 如果您的文件系统支持长文件名，将名称括在双引号 (**"**)。
