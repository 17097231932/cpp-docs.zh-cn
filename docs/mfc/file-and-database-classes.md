# 文件和数据库类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

这些类使您可以将信息存储到数据库或磁盘文件。  有三组数据库类 \- 提供类似功能的 OLE DB、ODBC 和 DAO。  使用 OLE DB实现OLE DB 组件 并与OLE DB 使用者模板协作，使用数据访问对象实现DAO 组件，使用开放式数据库连接实现 ODBC 组件。  还有一组类来操作标准文件、激活流和 HTML 流。  
  
 以下类的类别支持持久性数据。  
  
-   [文件 I\/O 类](../mfc/file-i-o-classes.md)  
  
-   [OLE DB 类](../mfc/ole-db-classes.md)  
  
-   [DAO 类](../mfc/dao-classes.md)  
  
-   [ODBC 类](../mfc/odbc-classes.md)
