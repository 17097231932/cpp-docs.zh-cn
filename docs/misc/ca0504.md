# CA0504
规则目录因不存在而被忽略  
  
 如果未能找到项目或生成配置文件中的 **CodeAnalysisRuleDirectories** 属性所指定的目录，则会生成警告 CA504。  若要解决此问题，请输入一个有效的目录路径。
