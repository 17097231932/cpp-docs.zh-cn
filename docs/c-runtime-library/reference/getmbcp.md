# <a name="getmbcp"></a>_getmbcp

检索当前代码页。

## <a name="syntax"></a>语法

```C
int _getmbcp( void );
```

## <a name="return-value"></a>返回值

返回当前多字节代码页。 返回值为 0 表示单字节代码页正在使用中。

## <a name="requirements"></a>要求

|例程|必需的标头|
|-------------|---------------------|
|**_getmbcp**|\<mbctype.h>|

有关更多兼容性信息，请参阅 [兼容性](../../c-runtime-library/compatibility.md)。
