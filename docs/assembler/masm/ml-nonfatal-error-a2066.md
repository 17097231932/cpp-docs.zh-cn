# ML Nonfatal Error A2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不兼容的 CPU 架构和段落范围**  
  
 尝试打开与 **USE16**、 **USE32**或与指定的 CPU 兼容的 **简单** 属性的一段，或者更改为 16 位 CPU，在 32 位段时。  
  
 **USE32** 和 **简单** 属性必须在 .386 或更大的处理器指令之后。
