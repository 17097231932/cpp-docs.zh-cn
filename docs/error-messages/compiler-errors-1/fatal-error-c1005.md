# <a name="fatal-error-c1005"></a>错误 C1005

字符串过大，无法缓冲

编译器中间文件内的字符串溢出了缓冲区。

当你传递到 [/Fd](../../build/reference/fd-program-database-file-name.md) 或 [/Yl](../../build/reference/yl-inject-pch-reference-for-debug-library.md) 编译器选项的参数大于 256 个字节时，可能会遇到此错误。
