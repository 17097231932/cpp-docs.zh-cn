# Switch &lt;name&gt; specified with conflicting values.
当在“命令”窗口中输入同一个开关两次，并且为该开关指定的值在每个实例中都不同时，会发生此错误。  
  
### 更正此错误  
  
-   重新输入命令，但每个开关都仅使用一次。
