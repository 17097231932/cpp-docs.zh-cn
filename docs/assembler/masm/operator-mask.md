# <a name="operator-mask"></a>运算符 MASK

返回一个位掩码，其中设置了*recordfieldname*或*记录*中的位，并清除所有其他位。

## <a name="syntax"></a>语法

> **掩码**{*recordfieldname* | *记录*}
