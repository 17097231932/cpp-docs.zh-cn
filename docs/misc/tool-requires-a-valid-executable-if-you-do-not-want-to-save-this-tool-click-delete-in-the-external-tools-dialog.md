# 工具需要有效的可执行文件。 如果不想保存此工具，请在“外部工具”对话框中单击“删除”。
当工具已添加到“外部工具”对话框中但尚未指定可执行文件时，通常会发生此错误。  
  
### 更正此错误  
  
1.  如果要保存该工具，请单击**确定**，然后在**命令**框中输入该工具的文件名，或通过选择省略号按钮定位到一个文件。  
  
2.  如果不希望保存该工具，请单击**确定**，然后在**外部工具**对话框中单击**删除**。
