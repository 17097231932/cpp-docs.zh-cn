# <a name="handletisvalid-method"></a>HandleT::IsValid 方法
指示当前 HandleT 对象是否表示句柄。  
  
## <a name="syntax"></a>语法  
  
```  
bool IsValid() const;  
```  
  
## <a name="return-value"></a>返回值  
 如果 HandleT 表示句柄，则为 `true`；否则为 `false`。  
  
## <a name="requirements"></a>惠?  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
