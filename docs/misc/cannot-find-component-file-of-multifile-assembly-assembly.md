# Cannot find component &#39;file&#39; of multifile assembly &#39;assembly&#39;
未能在多文件程序集中找到一个文件。  多文件程序集引用一个或多个文件，如模块或资源文件。  
  
 正在复制的程序集可能是一个直接引用，在“解决方案资源管理器”中的**“引用”**节点可见，或者该程序集可能是某个直接引用的依赖项。  
  
 **更正此错误**  
  
-   重新安装该组件。  
  
     项目可能生成，但是可能无法运行。
