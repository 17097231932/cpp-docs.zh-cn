# 表达式计算器错误 CXX0067
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支持函数求值  
  
 表达式包含了函数调用。  某些表达式计算器不支持函数调用。  
  
 该错误与 CAN0067 相同。
