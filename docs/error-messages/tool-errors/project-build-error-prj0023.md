# <a name="project-build-error-prj0023"></a>项目生成错误 PRJ0023

> 工具 "*工具*"，"未知" 属性包含无效文件名 "*file*"。

为工具 *工具* 指定的文件名 *文件* 无效。

可能使用了未知或无效的宏。

有关如何访问项目属性的信息，请参阅 [设置 Visual C++ 项目属性](../../build/working-with-project-properties.md) 。
