# <a name="page"></a>PAGE

第一个指令设置行*长度*和字符*宽度*的程序列表。 如果不给定任何参数时，将生成一个分页符。 第二个指令的节号会递增，并且将页码重置为 1。

## <a name="syntax"></a>语法

> 页 [[*长度*]] [[，*宽度*]]<br/><br/>
> 页 +
