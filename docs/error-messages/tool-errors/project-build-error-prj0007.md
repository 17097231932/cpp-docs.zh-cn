# 项目生成错误 PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未能创建输出目录“directory”。  
  
 Visual C\+\+ 未能创建输出目录。  可能的原因包括：  
  
-   在其中创建输出目录的目录是只读的。  
  
-   计算机的磁盘空间不足。  
  
-   指定了无效的目录路径。  
  
-   用户权限不够，无法创建目录。
