# 编译器警告（等级 4）C4709
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

逗号运算符在数组索引表达式内  
  
 当数组索引表达式中出现逗号时，编译器使用最后一个逗号后面的值。  
  
## 示例  
 下面的示例生成 C4709：  
  
```  
// C4709.cpp  
// compile with: /W4  
#include <stdio.h>  
  
int main()   
{  
    int arr[2][2];  
    arr[0][0] = 10;  
    arr[0][1] = 11;  
  
    // Prints 10, not 11  
    printf_s("\n%d",arr[0][1,0]);   // C4709  
}  
```
