# <a name="compiler-error-c2104"></a>编译器错误 C2104
位域上的“&”被忽略  
  
 不能采用位字段的地址。  
  
 下面的示例生成 C2104:  
  
```  
// C2104.cpp  
struct X {  
   int sb : 1;  
};  
  
int main() {  
   X x;  
   &x.sb;   // C2104   
   x.sb;   // OK  
}  
```
