# <a name="operator-ptr"></a>运算符 PTR
第一个运算符强制*表达式*被视为具有指定`type`。 第二个运算符指定指向的指针`type`。  
  
## <a name="syntax"></a>语法  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
