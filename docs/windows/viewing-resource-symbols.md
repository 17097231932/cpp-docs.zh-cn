# <a name="viewing-resource-symbols"></a>查看资源符号
### <a name="to-view-resource-symbols"></a>查看资源符号  
  
1.  在[资源视图](../windows/resource-view-window.md)，右键单击.rc 文件。  
  
    > [!NOTE]
    >  如果你的项目尚未包含 .rc 文件，请参阅 [创建新的资源脚本文件](../windows/how-to-create-a-resource-script-file.md)。  
  
2.  选择**资源符号**从快捷菜单，若要查看资源符号表中的[资源符号对话框](../windows/resource-symbols-dialog-box.md)。  
  
    > [!NOTE]
    >  若要查看预定义的符号，请**显示只读符号**复选框。  
  

  
 **要求**  
  
 Win32
