# CCommonDialog Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

封装Windows通用对话框的函数的选件类的基类。  
  
## 语法  
  
```  
class CCommonDialog : public CDialog  
```
