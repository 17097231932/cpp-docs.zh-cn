# <a name="creating-desktop-projects-by-using-application-wizards"></a>使用应用程序向导创建桌面项目
每种 Visual C++ 项目类型都有应用程序向导，可帮助快速容易地生成仿效项目模板的新项目。  若要打开应用程序向导，请使用**新项目**对话框可以指定项目属性，例如项目，或目录和解决方案的名称在你的项目将驻留的。  
  
#### <a name="to-open-a-visual-c-application-wizard"></a>打开 Visual C++ 应用程序向导  
  
1.  在 **“文件”** 菜单上，单击 **“新建”**，然后单击 **“项目”**。  
  
     此时将出现 “新建项目” 对话框。  
  
2.  在项目类型窗格中，选择**Visual c + + 项目**文件夹。 每种 C++ 项目类型的图标显示在“模板”窗格中。  
  
3.  在“模板”窗格中，选择图标以选择项目类型。 一条信息同时显示在两个窗格下，指示将要创建的项目类型。  
  
4.  指定项目属性，或跳过这一步以使用 Visual Studio 默认项目属性。 请参阅[使用项目属性](../ide/working-with-project-properties.md)有关详细信息。  
  
5.  单击**确定**，和你的项目类型打开的向导。 如果你想要使该向导，请阅读帮助主题，请按**F1**。  
  
 创建项目时，可以创建新的解决方案或将项目添加到现有的解决方案。 有关详细信息，请参阅[如何：通过现有代码创建 C++ 项目](../ide/how-to-create-a-cpp-project-from-existing-code.md)。
