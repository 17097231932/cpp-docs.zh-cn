# 编译器错误 C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“operator”: 无法查找重载运算符  
  
 该运算符无法与提供的操作数一起使用。  必须与所需操作数一起提供重载运算符。  
  
 如果操作数具有错误的类型，可能导致此错误。  
  
 如果定义采用单个参数的转换运算符或构造函数，可能会修复此错误。
