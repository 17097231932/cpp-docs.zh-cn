# <a name="group-32-bit-masm"></a>GROUP （32位 MASM）

将指定*段*添加到名为的*组。* （仅限32位 MASM。）

## <a name="syntax"></a>语法

> *名称***组***段*⟦ __，__ *段*.。。⟧
