# <a name="expression-evaluator-error-cxx0034"></a>表达式计算器错误 CXX0034

**类型与运算符不兼容**

指定的操作数类型不是合法的操作的。

例如，指针不能乘以的任何值。

可能需要键入强制转换运算符与兼容的类型的操作数。
