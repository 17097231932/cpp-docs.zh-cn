# <a name="in-c"></a>in (C++)
指示参数是要从调用过程传递给被调用过程。  
  
## <a name="syntax"></a>语法  
  
```  
[in]  
```  
  
## <a name="remarks"></a>备注  
 **中**c + + 属性具有相同的功能[中](http://msdn.microsoft.com/library/windows/desktop/aa367051)MIDL 特性。  
  
## <a name="example"></a>示例  
 请参阅[可绑定](../windows/bindable.md)有关如何使用的示例**中**。
