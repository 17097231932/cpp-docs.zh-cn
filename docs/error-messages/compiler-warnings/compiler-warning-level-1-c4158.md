# 编译器警告（等级 1）C4158
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

假定 \#pragma pointers\_to\_members\(full\_generality, inheritance\)  
  
 发出 **\#pragma pointers\_to\_members \(***single&#124;multiple&#124;virtual***\)** 时未发出 **\#pragma pointers\_to\_members \(full\_generality\)**。
