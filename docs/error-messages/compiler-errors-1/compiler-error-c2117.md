# <a name="compiler-error-c2117"></a>编译器错误 C2117
identifier： 数组界限溢出  
  
 数组具有初始值设定项太多：  
  
-   数组元素和初始值设定项在大小和数量不匹配。  
  
-   字符串中 null 终止符的占用任何空间。  
  
 下面的示例生成 C2117:  
  
```  
// C2117.cpp  
int main() {  
   char abc[4] = "abcd";   // C2117  
   char def[4] = "abd";   // OK  
}  
```
