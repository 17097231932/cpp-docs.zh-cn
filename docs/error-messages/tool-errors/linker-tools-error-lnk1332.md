# <a name="linker-tools-error-lnk1332"></a>链接器工具错误 LNK1332

检测到 \<count> 在一个模块中导入并在另一个模块中定义的 Windows 运行时类型

当它生成当前目标时，链接器检测到 <`count`> Windows 运行时类型，其中每个类型都导入一个模块中，并在另一个模块中定义。

### <a name="to-correct-this-error"></a>更正此错误

- 请根据错误消息中建议更正每个生成中的 LNK2039 错误。
