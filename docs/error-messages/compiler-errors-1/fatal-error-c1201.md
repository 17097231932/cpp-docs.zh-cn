# <a name="fatal-error-c1201"></a>错误 C1201
类模板定义中出现语法错误后无法继续  
  
 分析类模板定义时出现意外错误。  
  
 修复任何其他错误并重新编译。 如果该失败，请注意错误的情况下尝试找出问题并创建可重现的测试用例，请联系[Microsoft 产品支持服务](/visualstudio/ide/talk-to-us)。
