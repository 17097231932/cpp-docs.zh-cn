# <a name="pointer-subtraction"></a>指针减法

**ANSI 3.3.6, 4.1.1** 保留两个指向同一数组 ptrdiff_t 的元素的指针之间的差所需的整数类型

`ptrdiff_t` typedef 是 32 位 x86 平台上的一个 `int`。 在 64 位平台上，`ptrdiff_t` typedef 是 `__int64`。
