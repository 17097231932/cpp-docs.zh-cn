# 表达式计算器错误 CXX0045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不是函数  
  
 将参数列表提供给了程序中不是函数名的符号。  
  
## 示例  
  
```  
queue( alpha, beta )  
```  
  
 当 `queue` 不是一个函数时。  
  
 该错误与 CAN0045 相同。
