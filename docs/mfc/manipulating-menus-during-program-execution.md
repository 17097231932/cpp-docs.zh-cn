# <a name="manipulating-menus-during-program-execution"></a>在程序执行过程中操作菜单

使用类 `CMenu` 即时操作菜单和菜单项。 `CMenu`封装 Windows HMENU 句柄并提供用于使用菜单的成员函数。

有关详细信息，请参阅类[CMenu](reference/cmenu-class.md)概述。
