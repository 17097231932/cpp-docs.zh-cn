## <a name="remarks"></a>备注

该`CMFCPropertyGridCtrl`类显示一个属性网格控件，该控件包含从[CMFCPropertyGridProperty](../../mfc/reference/cmfcpropertygridproperty-class.md)类派生的可编辑属性。 每个属性可以表示一个类型，并且它可以包含子项。 属性网格控件支持底部的可调整大小的区域，可以显示所选属性的说明。

要使用属性网格控件，请构造对象`CMFCPropertyGridCtrl`，然后调用[CMFCPropertyGridCtrl：：创建](#create)方法。 使用[CMFC 属性网格Ctrl：addProperty](#addproperty)方法将属性添加到列表中。

## <a name="cmfcpropertygridctrlsetgroupnamefullwidth"></a><a name="setgroupnamefullwidth"></a>CMFC 属性网格Ctrl：：组名称全宽

指定是否在当前属性网格控件中显示一组属性的类别名称的完整宽度。

```cpp
void SetGroupNameFullWidth(
    BOOL bGroupNameFullWidth = TRUE,
    BOOL bRedraw = TRUE);
```

### <a name="parameters"></a>参数

*b组名全宽*<br/>
[在]TRUE 显示类别名称的完整宽度，而不考虑属性名称列的宽度。 FALSE 将类别名称的宽度限制为属性名称列的宽度。 默认值为 TRUE。

*bredraw*<br/>
[在]TRUE 立即更新属性网格控制;FALSE 以在下一次重绘事件发生时更新控件。 默认值为 TRUE。

### <a name="remarks"></a>备注

属性网格控件由可调整大小的*属性名称*列和*属性值*列组成。 名称列的末尾也是值列的开头。 要调整列的大小，请拖动列之间的边框。

术语*组名称*和*类别名称*在此方法中可互换使用。 类别名称显示在一行上，该行将领导一组相关属性和值。 此方法指定属性名称列的宽度是否也指定显示的类别名称的宽度。
