# 使用 CToolTipCtrl
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[CToolTipCtrl](../mfc/reference/ctooltipctrl-class.md)封装“工具提示控件”功能，此控件是一个小型弹出窗口，显示说明应用程序中工具用途的单行文本。  工具提示的大多数时间隐藏，仅在用户的工具和对象中将光标放在它有大约一半一秒的。  当用户单击按钮或将鼠标光标工具，工具提示在光标周围将出现然后消失。
