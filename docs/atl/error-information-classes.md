# <a name="error-information-classes"></a>Error 信息类

下面的类指示如何处理错误消息：

- [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) 确定对象是否支持 [IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo) 接口。 `IErrorInfo` 允许将错误信息传播回客户端。
