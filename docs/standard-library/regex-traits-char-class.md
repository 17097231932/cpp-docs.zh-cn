# <a name="regex_traitsltchargt-class"></a>regex_traits&lt;char&gt; 类

**Char**的 `regex_traits` 专用化。

## <a name="syntax"></a>语法

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>备注

类是[regex_traits](../standard-library/regex-traits-class.md)类型的元素的类模板的显式专用**化（以便**它可以充分利用操作此类型对象的库函数）。

## <a name="requirements"></a>要求

**标头：** \<regex 1>

**命名空间:** std
