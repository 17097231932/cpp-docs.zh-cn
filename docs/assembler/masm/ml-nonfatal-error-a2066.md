# <a name="ml-nonfatal-error-a2066"></a>ML 非致命错误 A2066
**不兼容的 CPU 模式和分段大小**  
  
 尝试打开的段**USE16**， **USE32**，或**平面**未与指定的 CPU，或将更改为在 32 位的 16 位 CPU 兼容的属性段。  
  
 **USE32**和**平面**特性的前面必须是.386 或更高的处理器指令。
