# <a name="compiler-error-c3120"></a>编译器错误 C3120
method_name： 接口方法不能接受变量自变量列表  
  
 接口方法不能采用变量自变量列表。 例如，下面的接口定义生成 C3120:  
  
```  
// C3120.cpp  
__interface A {  
int X(int i, ...);    // C3120  
};  
  
int main(void) { return(0); }  
```
