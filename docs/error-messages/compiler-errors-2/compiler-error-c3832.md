# <a name="compiler-error-c3832"></a>编译器错误 C3832

"类型库"：类型库看起来就像是为32位指针生成的;请更改 "ptrsize" 限定符

随[#import](../../preprocessor/hash-import-directive-cpp.md)指令的 `ptrsize` 属性一起提供的显式信息未与编译器在类型库中找到的信息一致。
