# <a name="abort-function-c"></a>abort 函数 (C)

**ANSI 4.10.4.1** 有关打开的文件和临时文件的 abort  函数的行为

 abort 函数不关闭已打开的或临时的文件。 它不刷新流缓冲区。 有关详细信息，请参阅[中止](../c-runtime-library/reference/abort.md)。
