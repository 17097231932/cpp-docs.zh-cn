# <a name="c-language-syntax-summary"></a>C 语言语法摘要

本节提供 C 语言和 Microsoft 特定的 C 语言功能的完整说明。 可以使用本节中的语法表示法，确定任意语言组件的确切语法。 该语法的说明将显示在本手册讨论主题的节中。

> [!NOTE]
>  此语法摘要不是 ANSI C 标准的一部分，但包括在此仅用作信息参考。 Microsoft 特定的语法在该语法后面的注释中进行了注明。
