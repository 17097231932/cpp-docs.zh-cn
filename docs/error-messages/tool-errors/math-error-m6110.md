# <a name="math-error-m6110"></a>数学错误 M6110

堆栈溢出

浮点表达式导致浮点堆栈溢出。

最多为七个级别除了通常支持 8087/287/387 协处理器的八个级别，可以捕获堆栈溢出浮点异常。

程序终止，退出代码为 138。
