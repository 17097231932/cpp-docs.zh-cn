# 编译器错误 C3396
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“attribute”: 未能在“namespace”中发现自定义特性  
  
 C3396 指示你正在使用的不是最新的公共语言运行时。  重新安装 CLR 以解决问题。
