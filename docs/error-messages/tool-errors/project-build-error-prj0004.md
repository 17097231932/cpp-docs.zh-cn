# <a name="project-build-error-prj0004"></a>项目生成错误 PRJ0004

> 无法生成命令行*工具*工具。

在此类的方式使调用语法指定的一个或多个属性*工具*非法：

- 你可能已指定的格式不正确或未知的宏。

- 您的计算机的可用磁盘空间可能不足。

你可能想要查看该工具的设置，通过查看[命令行属性页](../../ide/command-line-property-pages.md)。
