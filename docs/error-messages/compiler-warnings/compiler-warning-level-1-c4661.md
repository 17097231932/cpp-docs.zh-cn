# <a name="compiler-warning-level-1-c4661"></a>编译器警告（等级 1）C4661

identifier： 没有提供显式模板实例化请求适当的定义

未定义此模板类的成员。

## <a name="example"></a>示例

```
// C4661.cpp
// compile with: /W1 /LD
template<class T> class MyClass {
public:
   void i();   // declaration but not definition
};
template MyClass< int >;  // C4661
```
