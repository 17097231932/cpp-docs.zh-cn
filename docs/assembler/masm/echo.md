# <a name="echo"></a>ECHO

将*消息*显示到标准输出设备（默认情况下为屏幕）。

## <a name="syntax"></a>语法

> **回显***消息*

## <a name="remarks"></a>备注

与[% OUT](percent-out.md)相同。
