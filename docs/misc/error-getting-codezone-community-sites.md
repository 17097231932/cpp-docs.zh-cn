# Error getting Codezone Community sites.
当找不到与联机 Web 服务的连接时，会发生此错误。  Web 服务填充**“联机”**页中的信息，该页位于**“选项”**对话框的**“环境”**中的**“帮助”**下。  
  
### 更正此错误  
  
1.  确认您有活动的 Internet 连接。  
  
     \- 或 \-  
  
     如果通过代理服务器连接到 Internet，请检查代理服务器的设置。  
  
2.  尝试再次访问“联机”页。
