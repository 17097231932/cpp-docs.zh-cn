# <a name="compiler-error-c3241"></a>编译器错误 C3241

method： 此方法不由 interface 引入

显式重写函数，函数签名必须与要重写的函数声明完全匹配。

下面的示例生成 C3241:

```
// C3241.cpp
#pragma warning(disable:4199)

__interface IX12A {
   void mf();
};

__interface IX12B {
   void mf(int);
};

class CX12 : public IX12A, public IX12B { // C3241
   void IX12A::mf(int);
};
```
