# ML Nonfatal Error A2034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**必须在段块**

-   标签定义  
  
-   [这](../../assembler/masm/operator-this.md) 运算符  
  
-   [$](../../assembler/masm/dollar.md) 运算符  
  
-   程序定义  
  
-   [对齐](../../assembler/masm/align-masm.md) 指令  
  
-   [ORG](../../assembler/masm/org.md) 指令
