# <a name="ml-nonfatal-error-a2019"></a>ML 非致命错误 A2019

**操作数必须是记录类型或字段**

操作数以下[宽度](../../assembler/masm/operator-width.md)或[掩码](../../assembler/masm/operator-mask.md)运算符无效。

[宽度](../../assembler/masm/operator-width.md)运算符采用的操作数是一个字段或记录的名称。 [掩码](../../assembler/masm/operator-mask.md)运算符采用是名称的字段或记录类型的操作数。
