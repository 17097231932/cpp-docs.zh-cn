# OLE 自动化类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

这些类支持的自动化客户程序 \(控制其他应用程序\) 的应用程序。  可能由其他应用程序控制\) 的自动化服务器应用程序 \(通过 [计划映射](../mfc/reference/dispatch-maps.md)支持。  
  
 [COleDispatchDriver](../mfc/reference/coledispatchdriver-class.md)  
 用于调用自动化客户的自动化服务器。  添加类时，此类用于创建提供一类型库的自动化服务器的类型安全的类。  
  
 [COleDispatchException](../mfc/reference/coledispatchexception-class.md)  
 异常引起错误在 OLE 自动化时。  自动化异常由自动化服务器引发并通过自动化客户端捕获。
