# <a name="math-error-m6205"></a>数学错误 M6205
function: _TLOSS 错误  
  
 总丢失基数 （精度） 发生。  
  
 此错误可能由与 sin、 操作数给定的非常大的数 cos 或 tan，因为该操作数必须减小为介于 0 和 2 之间的数字 * pi。
