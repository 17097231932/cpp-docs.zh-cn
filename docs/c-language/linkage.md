# <a name="linkage"></a>链接

标识符名称可引用不同范围内的各个标识符。 在不同的范围内或在同一范围内多次声明的标识符可以通过称为“链接”的过程来引用同一标识符或函数。 链接确定可在其中引用标识符的程序的部分（其“可见性”）。 有三种链接：[内部](../c-language/internal-linkage.md)、[外部](../c-language/external-linkage.md)和[无链接](../c-language/no-linkage.md)。
