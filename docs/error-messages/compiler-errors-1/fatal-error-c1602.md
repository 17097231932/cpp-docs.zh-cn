# <a name="fatal-error-c1602"></a>错误 C1602

不支持的内部函数

此错误表示你的编译器 .exe 文件不匹配，可能是由于安装不完整而发生的。 例如，您可能安装了 Service Pack，但不是处理器包。 安装所有必需的产品。
