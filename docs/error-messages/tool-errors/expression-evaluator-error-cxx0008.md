# 表达式计算器错误 CXX0008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**在字符串末尾缺少 "**  
  
 字符串末尾缺少应有的双引号。  重新输入表达式，并用双引号将字符串括起来。  
  
 该错误与 CAN0008 相同。
