# <a name="ltlistgt"></a>&lt;list&gt;
定义容器模板类列表和几个支持模板。  
  
## <a name="syntax"></a>语法  
  
```  
#include <list>  
```  
  
### <a name="operators"></a>运算符  
  
|||  
|-|-|  
|[operator!=](../standard-library/list-operators.md#operator_neq)|测试运算符左侧的列表对象是否不等于右侧的列表对象。|  
|[operator<](../standard-library/list-operators.md#operator_lt_)|测试运算符左侧的列表对象是否小于右侧的列表对象。|  
|[operator\<=](../standard-library/list-operators.md#operator_lt__eq)|测试运算符左侧的列表对象是否小于或等于右侧的列表对象。|  
|[operator==](../standard-library/list-operators.md#operator_eq_eq)|测试运算符左侧的列表对象是否等于右侧的列表对象。|  
|[operator>](../standard-library/list-operators.md#operator_gt_)|测试运算符左侧的列表对象是否大于右侧的列表对象。|  
|[operator>=](../standard-library/list-operators.md#operator_gt__eq)|测试运算符左侧的列表对象是否大于或等于右侧的列表对象。|  
  
### <a name="classes"></a>类  
  
|||  
|-|-|  
|[list 类](../standard-library/list-class.md)|用于保持其元素为线性排列并允许在序列内任何位置上的高效插入和删除的序列容器的模板类。|
