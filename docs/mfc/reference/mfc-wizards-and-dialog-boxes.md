# MFC 向导和对话框
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Microsoft 基础类 \(MFC\)向导为各种组件和对象生成样板代码。  您可以在**解决方案资源管理器**中，通过打开项目的快捷菜单中的选择**添加类**来运行向导。
