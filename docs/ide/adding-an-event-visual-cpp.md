# 添加事件 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

从类视图中，只能使用[添加事件向导](../ide/add-event-wizard.md)将事件添加到 [MFC ActiveX 控件](../mfc/reference/creating-an-mfc-activex-control.md)项目中的控件类。  如果要将事件添加到其他项目类型，使用[“属性”窗口](../Topic/Properties%20Window.md)中的“事件”按钮。  
  
### 将事件添加到 MFC ActiveX 控件项目  
  
1.  在“类视图”中，展开项目节点以显示项目中的类。  
  
2.  右击项目的控件类。  
  
3.  在快捷菜单上单击“添加”，然后单击**“添加事件”**以显示“添加事件向导”。  
  
4.  在适当的向导框中提供事件信息。  
  
5.  单击“完成”将事件添加到项目。
