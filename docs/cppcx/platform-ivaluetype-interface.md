# <a name="platformivaluetype-interface"></a>Platform::IValueType 接口

`Platform::IValueType` 是由值类和值结构实现的基础结构接口。 不能在代码中显式使用。

## <a name="syntax"></a>语法

```cpp
interface class IValueType
```
