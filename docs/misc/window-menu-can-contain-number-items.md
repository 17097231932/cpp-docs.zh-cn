# Window menu can contain &lt;number&gt; items.
“窗口”菜单中的“窗口列表”仅能显示最多 24 个窗口名称。  出现该错误的情况通常是：在“选项”对话框“环境”选项“常规”窗格上的**窗口菜单中显示 \<number\> 项**框中输入了一个小于 1 或大于 24 的数字。  
  
### 更正此错误  
  
1.  在框中输入一个介于 1 到 24 之间的数字。
