# 资源编译器错误 RC2135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未找到文件：filename  
  
 资源编译器命令行中指定的文件未找到。  检查该文件是否已被移动到其他目录以及文件名或路径是否正确键入。  
  
 使用 **INCLUDE** 环境变量或 Visual C\+\+ **INCLUDE** 设置搜索文件。
