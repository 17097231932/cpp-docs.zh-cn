# <a name="pushcontext"></a>PUSHCONTEXT

将保存当前的部分或全部`context`： 段寄存器假定基数值、 列表和 cref 标志或处理器/协处理器值。 `context`可以是**假设**， `RADIX`，**列出**， **CPU**，或**所有**。
