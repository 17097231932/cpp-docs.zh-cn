# <a name="makefile-project-wizard"></a>生成文件项目向导
使用此向导创建的生成引擎执行的命令行。 一旦你创建项目时，你可以查看和编辑项目的属性在项目的[属性页](../ide/working-with-project-properties.md)。  
  
## <a name="overview"></a>概述  
 此向导页描述你创建的项目的当前项目设置。 默认情况下，项目设置为生成这两个调试和发布的配置*ProjName*.exe。  
  
 若要更改这些默认设置，请单击[应用程序设置](../ide/application-settings-makefile-project-wizard.md)向导的左侧列中选项卡并记所需的更改。
