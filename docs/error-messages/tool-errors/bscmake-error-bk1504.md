# BSCMAKE 错误 BK1504
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法在文件“filename”中定位 \[: reason\]  
  
 BSCMAKE 未能移动到文件中某一位置。  
  
### 通过检查以下可能的原因进行修复  
  
1.  磁盘已满。  
  
2.  硬件错误。  
  
3.  创建 .sbr 文件时因磁盘空间不足或编译器中断导致文件截断。
