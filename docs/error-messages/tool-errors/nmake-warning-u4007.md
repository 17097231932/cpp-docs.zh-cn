# <a name="nmake-warning-u4007"></a>NMAKE 警告 U4007

文件名 filename 太长;截断为 8.3

给定的文件的基名称具有超过八个字符或扩展具有三个以上的字符。 NMAKE 截断到了八个字符的基数和三个字符扩展的名称。

如果您的文件系统支持长文件名，将名称括在双引号引起来 (**"**)。
