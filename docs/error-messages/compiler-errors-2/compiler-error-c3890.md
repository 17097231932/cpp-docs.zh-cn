# 编译器错误 C3890
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“var”: 无法获取 literal 数据成员的地址  
  
 垃圾回收堆上已存在 literal 数据成员。由于垃圾回收堆上的对象可以移动，因此获取地址是没有意义的。  
  
 下面的示例生成 C3890：  
  
```  
// C3890.cpp  
// compile with: /clr  
ref struct Y1 {  
   literal int staticConst = 9;  
};  
  
int main() {  
   int p = &Y1::staticConst;   // C3890  
   int p2 = Y1::staticConst;   // OK  
}  
```
