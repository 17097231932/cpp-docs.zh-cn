# DDX_DHtml Helper Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

DDX\_DHtml helper宏允许轻松访问控件常用属性在HTML页中。  
  
### 数据交换的宏  
  
|||  
|-|-|  
|[DDX\_DHtml\_ElementValue](../Topic/DDX_DHtml_ElementValue.md)|从选定的控件设置或检索属性值。|  
|[DDX\_DHtml\_ElementInnerText](../Topic/DDX_DHtml_ElementInnerText.md)|设置或检索在当前元素的开始和结束标记之间的文本。|  
|[DDX\_DHtml\_ElementInnerHtml](../Topic/DDX_DHtml_ElementInnerHtml.md)|设置或检索在当前元素的开始和结束标记之间的HTML。|  
|[DDX\_DHtml\_Anchor\_Href](../Topic/DDX_DHtml_Anchor_Href.md)|设置或检索目标URL或定位点。|  
|[DDX\_DHtml\_Anchor\_Target](../Topic/DDX_DHtml_Anchor_Target.md)|设置或检索目标窗口或框架。|  
|[DDX\_DHtml\_Img\_Src](../Topic/DDX_DHtml_Img_Src.md)|设置或检索图像的名称或文档中的视频剪辑。|  
|[DDX\_DHtml\_Frame\_Src](../Topic/DDX_DHtml_Frame_Src.md)|设置或检索关联的帧的URL。|  
|[DDX\_DHtml\_IFrame\_Src](../Topic/DDX_DHtml_IFrame_Src.md)|设置或检索关联的帧的URL。|  
  
## 要求  
 **Header:** afxdhtml.h
