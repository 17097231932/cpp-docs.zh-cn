# <a name="using-a-template-library"></a>使用模板库

模板与宏有些类似。 与宏一样，调用模板会使其使用适当的参数替换将 (扩展) 你编写的代码。 但是，模板比此更进一步，以允许基于作为参数传递的类型创建新类。 这些新类实现了以类型安全的方式执行在模板代码中表示的操作。

ATL 等模板库不同于传统的 c + + 类库，因为它们通常仅作为源代码提供 (或作为源代码提供，但支持运行时) ，并不是本质上或不一定有层次结构。 从模板实例化类，而不是从类派生以获取所需的功能。
