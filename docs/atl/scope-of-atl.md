# <a name="scope-of-atl"></a>ATL 的作用域
ATL，可轻松地创建 COM 对象、 自动化服务器和 ActiveX 控件。 ATL 提供对许多基本的 COM 接口的内置支持。  
  
 ATL 附带作为源代码包括在你的应用程序。 ATL 还会使 DLL 可用 (atl90.dll)，其中包含可以在组件之间共享的代码。 但是，此 DLL 是不必要的。
