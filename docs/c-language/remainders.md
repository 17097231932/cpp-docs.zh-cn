# <a name="remainders"></a>余数

**ANSI 3.3.5** 整除的余数的符号

余数的符号与被除数的符号相同。 例如，应用于对象的

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
