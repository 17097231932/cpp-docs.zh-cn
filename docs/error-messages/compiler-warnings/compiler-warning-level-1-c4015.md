# 编译器警告（等级 1）C4015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 位域的类型必须是整型  
  
 位域未声明为 integer 类型。  编译器假定位域的基类型无符号。  位域必须声明为无符号的 integer 类型。
