# <a name="compiler-error-c2780"></a>编译器错误 C2780
“声明”: 应输入 N 个自变量，却提供了 M 个  
  
 函数模板的自变量太少或太多。  
  
 下面的示例生成 C2780，并演示如何修复此错误：  
  
```  
// C2780.cpp  
template<typename T>  
void f(T, T){}  
  
int main() {  
   f(1);  // C2780  
   // try the following line instead  
   // f(1,2);  
}  
```
