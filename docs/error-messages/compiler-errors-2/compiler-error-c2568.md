# <a name="compiler-error-c2568"></a>编译器错误 C2568

identifier1： 无法解析函数重载

编译器无法确定哪一个重载函数调用。 传递给函数的实参必须强制转换为匹配的重载函数之一的正式参数但不匹配，明确优于与所有其他用户。
