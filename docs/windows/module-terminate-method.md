# Module::Terminate 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

导致关闭模块实例化的所有工厂。  
  
## <a name="syntax"></a>语法  
  
```  
void Terminate();  
```  
  
## <a name="remarks"></a>备注  
 释放缓存中的工厂。  
  
## <a name="requirements"></a>要求  
 **标头︰** module.h  
  
 **命名空间：** Microsoft::WRL
