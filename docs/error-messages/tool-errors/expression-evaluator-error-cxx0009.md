# <a name="expression-evaluator-error-cxx0009"></a>表达式计算器错误 CXX0009

缺少在字符串末尾

单引号字符常量的末尾不应缺失。 重新键入表达式，用单引号括住字符常量。

此错误是与 CAN0009 相同。
