# 项目生成错误 PRJ0021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

工具“tool”、属性“property”包含无效文件名。  
  
 在属性 ***property*** 中为工具 ***tool*** 指定的文件名无效。  
  
 可能使用了未知或无效的宏。  
  
 有关如何访问项目属性的信息，请参见[设置 Visual C\+\+ 项目属性](../../ide/working-with-project-properties.md)。
