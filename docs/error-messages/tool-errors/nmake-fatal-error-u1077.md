# <a name="nmake-fatal-error-u1077"></a>NMAKE 错误 U1077

程序： 返回代码 value

由 NMAKE 失败，返回给定的退出代码调用给定的命令或程序。

若要禁止显示此错误并继续 NMAKE 会话，请使用 /I 选项 **。忽略**点指令或短划线 (**-**) 命令修饰符。 若要继续的依赖关系树不相关部分的 NMAKE 会话，请使用 /K 选项。
