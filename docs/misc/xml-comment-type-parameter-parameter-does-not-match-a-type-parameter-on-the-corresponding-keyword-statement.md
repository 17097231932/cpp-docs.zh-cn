# XML 注释类型形参 &lt;parameter&gt; 与相应 &lt;keyword&gt; 语句上的类型形参不匹配
XML 注释形参 \<parameter\> 与相应 \<keyword\> 语句上的类型形参不匹配。 将忽略 XML 注释。  
  
 为方法中的某些（而非所有）类型形参指定了 XML 注释标记。  
  
 **错误 ID：**BC42317  
  
### 更正此错误  
  
-   为方法中的所有类型形参提供 XML 注释标记。
