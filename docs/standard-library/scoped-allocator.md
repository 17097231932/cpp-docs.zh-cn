# <a name="ltscoped_allocatorgt"></a>&lt;scoped_allocator&gt;

定义容器类模板 scoped_allocator。

## <a name="syntax"></a>语法

```cpp
#include <scoped_allocator>
```

### <a name="operators"></a>运算符

|运算符|描述|
|-|-|
|[operator!=](../standard-library/scoped-allocator-operators.md#op_neq)|测试运算符左侧的 scoped_allocator 对象是否不等于右侧的列表对象。|
|[operator==](../standard-library/scoped-allocator-operators.md#op_eq_eq)|测试运算符左侧的 scoped_allocator 对象是否等于右侧的列表对象。|

### <a name="classes"></a>类

|实例|描述|
|-|-|
|[scoped_allocator_adaptor 类](../standard-library/scoped-allocator-adaptor-class.md)|一个类模板，它封装一个或多个分配器的嵌套。|
