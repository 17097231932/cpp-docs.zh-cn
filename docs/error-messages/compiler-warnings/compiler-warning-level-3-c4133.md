# 编译器警告（等级 3）C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“type”: 从“type1”到“type2”的类型不兼容  
  
 尝试对两个不同类型的指针相减，则可能导致出现此警告。  
  
 若要避免该警告，请提供适当的类型转换。
