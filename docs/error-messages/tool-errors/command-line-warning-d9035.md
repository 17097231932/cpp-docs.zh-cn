# <a name="command-line-warning-d9035"></a>命令行警告 D9035

> 选项*选项*已弃用，将在未来版本中删除

## <a name="remarks"></a>备注

指定编译器选项，将在未来版本的编译器中删除。 如果没有的建议的替换*选项*，此警告后跟警告[D9036](../../error-messages/tool-errors/command-line-warning-d9036.md)。

指定的选项仍有效，但应更新现在你的生成配置。 因此，你的项目是更有可能时升级编译器继续生成。
