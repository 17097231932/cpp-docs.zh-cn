# MSBuild 错误 MSB1039
**MSB1039: 指定工具集的版本。**  
  
 使用 **\/ToolsVersion** 开关时未指定任何值。  
  
### 更正此错误  
  
-   使用 **\/ToolsVersion** 开关时，请确保为 `ToolsVersion` 指定一个有效的非空值。
