# <a name="c28224"></a>C28224

> 警告 C28224：批注需要参数

此警告意味着在不使用参数的情况下使用命名批注，且至少需要一个参数。 如果对批注使用标准宏，则这种情况应该是不可能的；正常使用时不应出现此警告。
