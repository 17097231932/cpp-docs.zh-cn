# 关于异常的疑难解答：System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException
当筛选器或筛选器表尝试访问未缓冲的消息体时，将引发 <xref:System.ServiceModel.Dispatcher.FilterInvalidBodyAccessException> 异常。
