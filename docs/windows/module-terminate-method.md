# <a name="moduleterminate-method"></a>Module::Terminate 方法
导致关闭模块实例化的所有工厂。  
  
## <a name="syntax"></a>语法  
  
```cpp  
void Terminate();  
```  
  
## <a name="remarks"></a>备注  
 释放缓存中的工厂。

 **命名空间：** Microsoft::WRL
