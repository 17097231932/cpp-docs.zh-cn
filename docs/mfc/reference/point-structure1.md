# <a name="point-structure1"></a>点结构 1
**点**结构定义 x *-* 和 y 坐标的点。  
  
## <a name="syntax"></a>语法  
  
```  
typedef struct tagPOINT {  
    LONG x;  
    LONG y;  
} POINT;  
```  
  
#### <a name="parameters"></a>参数  
 *x*  
 指定的点的 x 坐标。  
  
 *y*  
 指定的点的 y 坐标。
