# UnorderedMap::Remove 方法
从 UnorderedMap 对象中删除指定的键值对。  
  
## 语法  
  
```cpp  
  
virtual void Remove(  
   K key);  
```  
  
#### 参数  
 `key`  
 键值对中的键部分。`key` 的类型名称为 *K*。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
