# <a name="compiler-error-c2531"></a>编译器错误 C2531
identifier： 对有点字段非法引用  
  
 不允许对位域的引用。  
  
 下面的示例生成 C2531:  
  
```  
// C2531.cpp  
// compile with: /c  
class P {  
   int &b1 : 10;   // C2531  
   int b2 : 10;   // OK  
};  
```
