# <a name="compiler-error-c2267"></a>编译器错误 C2267
function： 具有块范围的静态函数是非法的  
  
 声明的本地函数`static`。 静态函数必须具有全局作用域。  
  
 下面的示例生成 C2267:  
  
```  
// C2267.cpp  
static int func2();   // OK  
int main() {  
    static int func1();   // C2267  
}  
```
