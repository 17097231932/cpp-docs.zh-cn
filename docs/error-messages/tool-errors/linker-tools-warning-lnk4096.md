# <a name="linker-tools-warning-lnk4096"></a>链接器工具警告 LNK4096

/ 对于 Windows 95 和 Windows 98; 基本值"数字"无效映像可能无法运行

指定的基址是无效的。 Windows 95 和 Windows 98 的可执行文件必须具有大于 0x400000 处的基址。 基址的详细信息，请参阅[/base](../../build/reference/base-base-address.md)链接器选项。
