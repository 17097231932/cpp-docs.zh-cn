# <a name="nmake-fatal-error-u1097"></a>NMAKE 错误 U1097
文件名部分语法要求相关  
  
 当前的依赖项不具有显式依赖项或隐式的从属项。 使用百分比的文件名部分语法 (`%`) 说明符，表示当前的目标的第一个从属项组件。
