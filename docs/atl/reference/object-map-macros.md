# Object Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些宏定义对象映射和项。  
  
|||  
|-|-|  
|[DECLARE\_OBJECT\_DESCRIPTION](../Topic/DECLARE_OBJECT_DESCRIPTION.md)|允许您指定选件类对象的文本说明，将输入对象映射。|  
|[OBJECT\_ENTRY\_AUTO](../Topic/OBJECT_ENTRY_AUTO.md)|输入ATL对象对象映射，更新注册表，并创建对象的实例。|  
|[OBJECT\_ENTRY\_NON\_CREATEABLE\_EX\_AUTO](../Topic/OBJECT_ENTRY_NON_CREATEABLE_EX_AUTO.md)|允许您指定应注册和初始化对象，但是，不应该是外部可通过 `CoCreateInstance`。|
