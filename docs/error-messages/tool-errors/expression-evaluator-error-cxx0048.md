# <a name="expression-evaluator-error-cxx0048"></a>表达式计算器错误 CXX0048

不支持的调用顺序

在表达式中指定的函数使用的 C 表达式计算器不支持以调用序列。 您不能在监视窗口表达式中调用此函数。

此错误是与 CAN0048 相同。
