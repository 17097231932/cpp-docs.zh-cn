# <a name="compiler-error-c3812"></a>编译器错误 C3812

"property" 必须是属性声明中的第一个标记

声明属性时， `__property` 关键字必须是行中的第一个标记。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3812。
