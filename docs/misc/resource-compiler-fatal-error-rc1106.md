# 资源编译器错误 RC1106
无效开关: 选项  
  
 该选项不是有效的资源编译器选项。  
  
 有效选项是：  
  
 \/r  
  
 发出 .res 文件  
  
 \/v  
  
 详细（显示进度消息）  
  
 \/d  
  
 定义符号  
  
 \/fo  
  
 重命名 .res 文件  
  
 \/l  
  
 十六进制形式的默认语言 ID  
  
 \/I  
  
 为 **INCLUDE** 搜索添加路径  
  
 \/x  
  
 忽略 **INCLUDE** 环境变量  
  
 \/c  
  
 定义 NLS 转换使用的代码页
