# <a name="security-classes"></a>安全类

这些类是常见 Win32 安全类和对象的包装器。

- [CAccessToken](../atl/reference/caccesstoken-class.md) 此类是访问令牌的包装器。

- [CAcl](../atl/reference/cacl-class.md) 此类是 `ACL` (访问控制列表) 结构的包装器。

- [CDacl](../atl/reference/cdacl-class.md) 此类是 `DACL` (自由访问控制列表) 结构的包装器。

- [CPrivateObjectSecurityDesc](../atl/reference/cprivateobjectsecuritydesc-class.md) 此类表示私有对象安全描述符对象。

- [CSacl](../atl/reference/csacl-class.md) 此类是 `SACL` (系统访问控制列表) 结构的包装器。

- [CSecurityAttributes](../atl/reference/csecurityattributes-class.md) 此类是结构的精简包装器 `SECURITY_ATTRIBUTES` 。

- [CSecurityDesc](../atl/reference/csecuritydesc-class.md) 此类是结构的包装器 `SECURITY_DESCRIPTOR` 。

- [CSid](../atl/reference/csid-class.md) 此类是 `SID` (安全标识符) 结构的包装器。

- [CTokenGroups](../atl/reference/ctokengroups-class.md) 此类是结构的包装器 `TOKEN_GROUPS` 。

- [CTokenPrivileges](../atl/reference/ctokenprivileges-class.md) 此类是结构的包装器 `TOKEN_PRIVILEGES` 。
