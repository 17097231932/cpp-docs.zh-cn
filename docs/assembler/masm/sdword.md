# <a name="sdword"></a>SDWORD

分配并根据需要为每个初始化存储一个有符号的双字 （4 字节） `initializer`。 此外可以用作类型说明符类型是合法的任何位置。

## <a name="syntax"></a>语法

> 名称 SDWORD 初始值设定项 [[，初始值设定项]]...
