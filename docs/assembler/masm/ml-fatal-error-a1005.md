# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**汇编限制:宏完全参数名表**  
  
 许多参数、局部变量或宏标签用于宏定义。  它没有无空间在宏名表中。  
  
 定义更短或更少的名称或移除不必要的宏。
