# 创建控制台应用程序
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

创建 Win32 控制台应用程序最容易的方法是使用 Win32 应用程序向导。  
  
### 创建 Win32 控制台应用程序  
  
1.  按照帮助主题[用 Visual C\+\+ 应用程序向导创建项目](../ide/creating-desktop-projects-by-using-application-wizards.md)中的说明进行操作。  
  
2.  在**“新建项目”**对话框中，选择“模板”窗格中的**“Win32 控制台项目”**打开向导。  
  
3.  使用 [Win32 应用程序向导](../windows/win32-application-wizard.md)定义[应用程序设置](../windows/application-settings-win-32-project-wizard.md)。  
  
    > [!NOTE]
    >  跳过此步骤可保留向导的默认设置。  
  
4.  单击**“完成”**关闭向导，新创建的项目随即在**“解决方案资源管理器”**中打开。
