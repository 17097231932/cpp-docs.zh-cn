# <a name="expression-evaluator-error-cxx0032"></a>表达式计算器错误 CXX0032

被 0 除

表达式包含为零，这是非法的除数。 此除数可以是数字 0，也可能是一个表达式，计算结果为零。

此错误是与 CAN0032 相同。
