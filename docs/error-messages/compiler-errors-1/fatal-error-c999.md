# <a name="fatal-error-c999"></a>错误 C999

未知消息请选择 Visual c + + 帮助菜单上的技术支持命令或打开技术支持帮助文件了解详细信息

此错误通常表示你有具有来自编译器不同版本的混合文件，或者你的编译器安装已损坏。 使用**程序和功能**中修复或重新安装该产品的控制面板小程序。
