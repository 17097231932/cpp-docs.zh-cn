# 链接器工具错误 LNK1152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法解析一个或多个未修饰符号  
  
 在该错误之前，对于每个未能解析的未修饰符号都有一个 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) 警告，而且对于找到的未修饰符号的重复符号，至少有两个 [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) 警告。
