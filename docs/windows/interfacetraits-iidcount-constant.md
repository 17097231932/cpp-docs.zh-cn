# InterfaceTraits::IidCount 常量
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```  
static const unsigned long IidCount = 1;  
```  
  
## 备注  
 包含接口 ID 的数量与当前 InterfaceTraits 对象。

 **头文件：**implements.h
