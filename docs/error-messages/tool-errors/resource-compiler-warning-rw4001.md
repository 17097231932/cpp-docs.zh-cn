# 资源编译器警告 RW4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

.EXE 处理选项（\/L \/M \/P \/T \/K \/E \/31 或 \/30）  
  
 您已指定 EXE 处理选项，但没有要处理的可执行文件。  不要将这些选项用于 .res 文件。
