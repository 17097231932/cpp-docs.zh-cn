# <a name="compiler-error-c3202"></a>编译器错误 C3202
“arg name”：模板参数“parameter”的默认参数无效，应为类模板  
  
 你向模板参数传递的默认参数无效。  
  
 下面的示例生成 C3202：  
  
```  
// C3202.cpp  
template<typename T>  
class X  
{  
};  
  
class Z  
{  
};  
  
template<template<typename U> class T1 = Z, typename T2> // C3202  
class Y  
{  
};  
```
