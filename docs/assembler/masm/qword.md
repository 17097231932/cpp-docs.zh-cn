# <a name="qword"></a>QWORD

为每个*初始值设定*分配和（可选）初始化8个字节的存储。 还可在类型合法的任何位置用作类型说明符。 **QWORD**是[DQ](dq.md)的同义词。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **QWORD** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
