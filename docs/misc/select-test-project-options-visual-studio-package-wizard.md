# 选择“测试项目”选项，Visual Studio 包向导
此步骤能让你：  
  
-   为你的 VSPackage 生成集成测试项目。  
  
-   为你的 VSPackage 生成单元测试项目。  
  
 **集成测试项目**  
 创建集成测试项目。 集成测试在 Visual Studio 的实验实例中运行，并使用自动化模型在 Visual Studio 中测试你的包集成。  
  
 **单元测试项目**  
 创建单元测试项目。 单元测试是小型 MSTest 实例，可用于测试包中的单个类和成员。  
  
 **取消**  
 取消并停止新项目命令。  
  
 **上一步**  
 移动到上一步。  
  
 **完成**  
 创建 VSPackage。
