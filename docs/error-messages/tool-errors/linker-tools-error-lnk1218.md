# <a name="linker-tools-error-lnk1218"></a>链接器工具错误 LNK1218

警告视为错误;未生成输出文件

与链接 **/WX**，任何链接器警告会视为错误，并将创建任何输出文件。

请参阅[/WX （将链接器警告视为错误）](../../build/reference/wx-treat-linker-warnings-as-errors.md)有关详细信息。
