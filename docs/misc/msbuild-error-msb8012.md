# MSBuild 错误 MSB8012
*{0}\(\\{1\\}\)* 与 *\\{2\\}* 的 OutputFile 属性值 \(*\\{3\\}*\) 不匹配。 这可能导致你的项目无法正确构建。 若要纠正此问题，请确保 $\(OutDir\)、$\(TargetName\) 和 $\(TargetExt\) 属性值与 %\(\\{4\\}.OutputFile\) 中指定的值相匹配。
