# <a name="c26495-member_uninit"></a>C26495 MEMBER_UNINIT

变量 "% variable%" 未初始化。 务必初始化成员变量。 请参阅[ C++核心准则类型 6](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)。
