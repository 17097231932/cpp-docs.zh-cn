# <a name="finding-binary-data"></a>查找二进制数据
你可以搜索 ASCII 字符串或十六进制字节。 例如，若要查找"Hello"，你可以搜索是否存在字符串"Hello"或"48 65 6C 6 C 6F"（十六进制的等效项）。  
  
### <a name="to-find-binary-data"></a>若要查找二进制数据  
  
1.  从**编辑**菜单上，单击[查找](/visualstudio/ide/reference/find-command)。  
  
2.  在**查找内容**框中，从下拉列表中选择以前的搜索字符串或键入你想要查找的数据。  
  
3.  选择任一**查找**选项。  
  
4.  单击**查找下一个**。
