# <a name="string-literals-in-primary-expressions"></a>主要表达式中的字符串文本

“字符串”是字符、宽字符或包含在双引号内的相邻字符序列。 由于它们不是变量，因此字符串和其任一元素都不能作为赋值运算中的左操作数。 字符串文本的类型为 `char` 的数组（或对于宽字符串文本，类型为 `wchar_t` 的数组）。 表达式中的数组将转换为指针。 有关字符串的详细信息，请参阅[字符串文本](../c-language/c-string-literals.md)。
