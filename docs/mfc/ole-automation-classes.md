# <a name="ole-automation-classes"></a>OLE 自动化类
这些类支持自动化客户端 （控制其他应用程序的应用程序）。 自动化服务器 （可以由其他应用程序控制的应用程序） 支持通过[调度映射](../mfc/reference/dispatch-maps.md)。  
  
 [COleDispatchDriver](../mfc/reference/coledispatchdriver-class.md)  
 用于从你的自动化客户端调用自动化服务器。 添加类别时，此类用于创建针对提供的类型库的自动化服务器的类型安全类。  
  
 [COleDispatchException](../mfc/reference/coledispatchexception-class.md)  
 OLE 自动化期间出错导致的异常。 自动化异常由自动化服务器引发，由自动化客户端捕获。
