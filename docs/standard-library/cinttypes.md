# &lt;cinttypes&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

包含标准 C 库标头 \<inttypes.h\>。  包含此标头还将包括 \<cstdint\>。  
  
## 语法  
  
```  
  
#include <cinttypes>  
  
```  
  
## 备注  
 包含该标头还将确保使用标准 C 库标头中的外部链接声明的名称在 `std` 命名空间中声明。
