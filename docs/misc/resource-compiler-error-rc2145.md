# 资源编译器错误 RC2145
主语言 ID 太大  
  
 主语言 ID 必须是有效的十六进制语言 ID。 有关有效的语言 ID 的列表，请参阅[《运行时库参考》](../c-runtime-library/locale-names-languages-and-country-region-strings.md)中的*语言和国家\/地区字符串*。
