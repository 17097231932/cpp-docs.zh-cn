# &lt;number&gt; power closure argument &#39;n&#39; must be from 1 to 256.
当指数超过可接受的范围 1 到 256 时，通常会发生此错误。  
  
### 更正此错误  
  
1.  重新输入一个介于 1 到 256 之间的数字，然后再试一次。  有关语法的更多信息，请参见[正则表达式](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)。
