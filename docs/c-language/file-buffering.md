# 文件缓冲
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** 文件缓冲的特性  
  
 将完全缓冲通过标准 I\/O 函数访问的磁盘文件。  默认情况下，缓冲区包含 512 个字节。
