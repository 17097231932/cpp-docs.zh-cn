# <a name="bscmake-error-bk1514"></a>BSCMAKE 错误 BK1514
全部。SBR 文件截断，文件名中找不到  
  
 更新指定的.sbr 文件不是原始的浏览信息 (.bsc) 文件的一部分。 若要查找导致此错误的.sbr 文件的名称，请阅读[BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md)它前面的警告。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复  
  
1.  为.sbr 或.bsc 指定的文件名出错。  
  
2.  .Bsc 文件损坏所需 BSCMAKE 重新生成它。
