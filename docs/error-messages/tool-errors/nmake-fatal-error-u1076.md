# <a name="nmake-fatal-error-u1076"></a>NMAKE 错误 U1076

名称太长

一个字符串超过一项的以下限制：

- 宏名为 1024年个字符。

- 目标路径名的 256 个字符。

- 命令为 2048 个字符。
