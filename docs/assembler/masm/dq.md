# <a name="dq"></a>DQ

分配并根据需要为每个初始化 8 个字节存储`initializer`。 此外可以用作类型说明符类型是合法的任何位置。 `DQ` 是的同义词[QWORD](../../assembler/masm/qword.md)。

## <a name="syntax"></a>语法

> [[name]]DQ 初始值设定项 [[，初始值设定项]]...
