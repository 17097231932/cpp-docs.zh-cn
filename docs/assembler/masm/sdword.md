# <a name="sdword"></a>SDWORD
分配和 （可选） 为每个初始化存储一个有符号的双字 （4 个字节） `initializer`。 此外可以用作类型说明符类型是合法的任何位置。  
  
## <a name="syntax"></a>语法  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
