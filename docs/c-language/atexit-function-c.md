# atexit 函数 (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** 如果参数的值为零、EXIT\_SUCCESS 或 EXIT\_FAILURE 之外的其他值，则为 `atexit` 函数返回的状态  
  
 如果成功，则 `atexit` 函数返回零；否则返回非零值。
