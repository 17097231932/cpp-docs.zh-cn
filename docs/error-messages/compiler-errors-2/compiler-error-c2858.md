# <a name="compiler-error-c2858"></a>编译器错误 C2858
命令行选项 /Yc (/ Fdfilename) 不一致，出现预编译标头，使用 / Fdfilename  
  
 使用预编译标头指定的程序数据库 ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) 选项不是指定的上一个创建预编译头 ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) 选项。
