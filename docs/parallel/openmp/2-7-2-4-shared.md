# <a name="2724-shared"></a>2.7.2.4 shared
此子句共享中出现的变量*变量列表*团队中的所有线程间。 在团队中的所有线程都访问相同的存储区域，以进行**共享**变量。  
  
 语法**共享**子句是，如下所示：  
  
```  
shared(variable-list)  
```
