# <a name="compiler-warning-level-2-c4948"></a>编译器警告（等级 2）C4948

"访问器" 的返回类型与相应的 setter 的最后一个参数类型不匹配

编译器发现要获取的数据类型与为索引属性设置的数据类型不匹配。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C4948。
