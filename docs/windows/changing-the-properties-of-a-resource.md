# <a name="changing-the-properties-of-a-resource"></a>更改资源属性
### <a name="to-edit-the-properties-of-a-resource"></a>编辑资源的属性  
  
1.  在[资源视图](../windows/resource-view-window.md)，右键单击你想要编辑和选择的资源**属性**从快捷菜单。  
  
     **注意：** 如果你的项目尚未包含 .rc 文件，请参阅 [创建新资源脚本文件](../windows/how-to-create-a-resource-script-file.md)。  
  
2.  在[属性窗口](/visualstudio/ide/reference/properties-window)，更改你的资源的属性。  
  
### <a name="to-undo-a-change-youve-made-to-the-properties-of-a-resource"></a>撤消对资源属性所做的更改  
  
1.  确保资源在“资源视图”中具有焦点。  
  
2.  选择**撤消**从**编辑**菜单。  
  

  
 **要求**  
  
 无
