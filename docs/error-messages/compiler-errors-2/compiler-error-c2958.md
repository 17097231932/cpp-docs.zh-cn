# 编译器错误 C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左侧分隔符（在“location”中找到）未能正确匹配  
  
 分隔符不匹配。  
  
 引起此错误的可能原因是函数模板声明中有不匹配的括号。
