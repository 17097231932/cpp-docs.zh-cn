# <a name="sign-of-bit-fields"></a>位域的符号
**ANSI 3.5.2.1**“纯”`int` 域被视为 signed int 位域还是 unsigned int 位域  
  
 位域可以带符号也可以不带符号。 纯位域被视为带符号的。
