# <a name="compiler-error-c2095"></a>编译器错误 C2095
function： 实参具有类型 void： 参数 number  
  
 参数传递给函数的类型为`void`，这不允许。 使用指向 void 的指针 ( `void *`) 相反。  
  
 `number`指示哪些参数是`void`。
