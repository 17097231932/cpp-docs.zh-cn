# <a name="comptr"></a>com::ptr
可用作 CLR 类的成员的 COM 对象的包装器。 包装还会自动调用其析构函数时释放对象上拥有的引用的 COM 对象的生存期管理。 类似于[CComPtr 类](../atl/reference/ccomptr-class.md)。  
  
## <a name="syntax"></a>语法  
  
```  
#include <msclr\com\ptr.h>  
```  
  
## <a name="remarks"></a>备注  
 [com:: ptr 类](../dotnet/com-ptr-class.md)中定义\<msclr\com\ptr.h > 文件。
