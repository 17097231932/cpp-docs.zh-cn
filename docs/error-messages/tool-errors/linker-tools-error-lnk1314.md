# <a name="linker-tools-error-lnk1314"></a>链接器工具错误 LNK1314

损坏或无效的 COFF 符号表 （未定义的静态或标签符号）

给定的对象中的某个部分的编译器生成内容似乎已损坏。

此错误可能表示失败的编译器。
