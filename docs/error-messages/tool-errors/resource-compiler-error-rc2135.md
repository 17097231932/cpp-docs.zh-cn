# <a name="resource-compiler-error-rc2135"></a>资源编译器错误 RC2135

找不到文件：filename

找不到资源编译器命令行中指定的文件。 检查以确定文件是否已被移动到另一个目录，以及是否正确键入文件名和路径。

使用 **include** 环境变量或 Visual C++ **包括** 设置搜索文件。
