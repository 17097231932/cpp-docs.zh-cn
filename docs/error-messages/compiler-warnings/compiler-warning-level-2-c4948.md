# <a name="compiler-warning-level-2-c4948"></a>编译器警告（等级 2）C4948
访问器返回类型与相应的 setter 的最后一个参数类型不匹配  
  
编译器发现何种数据类型在获取和设置索引的属性之间的不匹配。  
  
C4948 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
