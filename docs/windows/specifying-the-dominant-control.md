# Specifying the Dominant Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

首先选定的控件是主导控件。  
  
### 指定主导控件  
  
1.  按住 **Ctrl** 键，并*首先* 单击要用来影响其他控件的大小或位置的控件。  
  
     **注意** 主导控件的尺寸柄是实心的，而从属控件的尺寸柄是空心的。  所有进一步的大小调整或对齐都基于主导控件。  
  
### 更改主导控件  
  
1.  单击当前选定的所有控件的外部以清除当前的选定。  
  
2.  重复上述过程，首先选定一个不同的控件。
