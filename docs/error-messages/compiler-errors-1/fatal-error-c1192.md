# <a name="fatal-error-c1192"></a>错误 C1192

\#在 "文件" 上使用失败

只能将 Microsoft 中间语言 (MSIL) 格式的文件传递到 [#using](../../preprocessor/hash-using-directive-cpp.md) 指令。 使用 [/clr](../../build/reference/clr-common-language-runtime-compilation.md) 编译器选项可以创建 MSIL 输出文件。 其他 Visual Studio 语言还会生成 MSIL 文件。
