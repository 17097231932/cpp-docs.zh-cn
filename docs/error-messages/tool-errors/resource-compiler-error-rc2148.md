# <a name="resource-compiler-error-rc2148"></a>资源编译器错误 RC2148

子语言 ID 太大

子语言 ID 值超出范围。

**LANGUAGE** 语句必须使用以下语法:

**LANGUAGE** *primary_language_ID*、*secondary_language_ID*

有效的子语言 Id 在 WINNT .h 文件中定义为 **SUBLANG_** 常量。
