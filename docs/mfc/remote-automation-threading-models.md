# <a name="remote-automation-threading-models"></a>远程自动化线程模型
远程自动化支持 COM 的单线程模型和单元线程模型。 在 32 位和 64 位平台（均受 Visual C++ 支持）上，应将线程模型设置为“单元线程”。 有关详细信息和执行此操作的位置，请参阅[创建使用远程自动化的程序](../mfc/creating-programs-that-use-remote-automation.md)。
