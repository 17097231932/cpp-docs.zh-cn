# <a name="project-build-error-prj0015"></a>项目生成错误 PRJ0015

系统中缺少空设备。 我们无法启动生成。

此错误可能是由于系统资源不足引起的。 关闭一些应用程序或重新启动。

如果对计算机上的空设备没有足够的权限，也可能会导致此情况。
