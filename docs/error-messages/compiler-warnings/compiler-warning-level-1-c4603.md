# <a name="compiler-warning-level-1-c4603"></a>编译器警告（等级 1）C4603
\<标识符 1>︰ 未定义宏或在预编译标头使用后定义发生改变  
  
 在使用使用预编译器头后， *标识符* 占位符所指定的宏不同，或者不再定义。
