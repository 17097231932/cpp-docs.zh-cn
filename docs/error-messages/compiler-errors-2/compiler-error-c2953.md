# <a name="compiler-error-c2953"></a>编译器错误 C2953
“identifier”：类模板已经定义  
  
 检查源文件并包含其他定义的文件。  
  
 以下示例生成 C2953：  
  
```  
// C2953.cpp  
// compile with: /c  
template <class T>  class A {};  
template <class T>  class A {};   // C2953  
template <class T>  class B {};   // OK  
```
