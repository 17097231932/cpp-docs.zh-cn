# 资源编译器错误 RC2180
无法打开临时文件  
  
 资源编译器\/Visual C\+\+ 无法打开临时文件。 可能的原因是没有目录的写入权限，或目录不存在。 资源编译器\/Visual C\+\+ 尝试在 **TMP** 环境变量指定的目录或当前目录（如果未指定任何目录）中使用这些文件。
