# <a name="project-build-error-prj0044"></a>项目生成错误 PRJ0044

分配给文件 "file" 的自定义生成规则 "rule" 的 "附加依赖项" 属性无效。 该属性包含计算结果为“value”的“string”。

**附加依赖项** 属性的计算结果为空字符串，或包含无效字符 (任何不能位于文件名或目录名称) 中的字符的字符串。 自定义生成规则需要生成操作的输出。

有关详细信息，请参阅 [指定自定义生成工具](../../build/specifying-custom-build-tools.md)。
