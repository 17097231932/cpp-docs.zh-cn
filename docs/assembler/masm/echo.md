# <a name="echo"></a>ECHO
显示*消息*到标准输出设备 （默认情况下，屏幕）。  
  
## <a name="syntax"></a>语法  
  
```  
  
ECHO   
message  
  
```  
  
## <a name="remarks"></a>备注  
 与相同[%out](../../assembler/masm/percent-out.md)。
