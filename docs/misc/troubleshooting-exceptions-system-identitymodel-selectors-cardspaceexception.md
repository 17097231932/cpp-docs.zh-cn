# 关于异常的疑难解答：System.IdentityModel.Selectors.CardSpaceException
当 CardSpace 服务级别上发生了一个或多个异常时，将引发 <xref:System.IdentityModel.Selectors.CardSpaceException> 异常。 错误的原因记录在事件日志中。  
  
## 备注  
 导致此异常的条件包括加载存储区或帐目时发生错误。
