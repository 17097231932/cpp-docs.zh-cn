# <a name="signal-function-c"></a>signal 函数 (C)
**ANSI 4.7.1.1** signal 函数的信号集  
  
 传递给 signal 的第一个参数必须为《运行时库参考》中描述的 signal 函数的符号常量之一。 《运行时库参考》中的信息还列出了每个信号的操作模式支持。 SIGNAL.H 中也定义了这些常量。
