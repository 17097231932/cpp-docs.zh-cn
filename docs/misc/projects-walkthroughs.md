# 项目演练
Visual Studio 项目是开发人员用来组织源代码文件和其他属性的容器。  项目可以将组织，生成，调试，并且，部署源代码并创建对 web 服务、数据库和其他资源。  使用 **解决方案资源管理器**，您可以查看项目及其内容。  
  
 本节的演练演示如何创建和扩展 Visual Studio 项目。  有关 Visual Studio 项目的更多信息以及它们可以执行，请参见 [扩展项目](../Topic/Extending%20Projects.md)。
