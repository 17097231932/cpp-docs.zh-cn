# 表达式计算器错误 CXX0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

缺少右圆括号  
  
 在表达式中发现不对称的圆括号。  使用对称圆括号重新输入表达式。  
  
 该错误与 CAN0007 相同。
