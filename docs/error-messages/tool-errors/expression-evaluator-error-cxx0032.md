# <a name="expression-evaluator-error-cxx0032"></a>表达式计算器错误 CXX0032
被 0 除  
  
 该表达式包含除数为零，这是非法。 此除数可能文本数字零，或者它可能是表达式计算结果为零。  
  
 此错误是与 CAN0032 相同。
