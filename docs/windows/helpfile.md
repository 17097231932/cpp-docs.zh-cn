# <a name="helpfile"></a>helpfile
设置类型库的帮助文件的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
      [ helpfile(  
   "filename"  
) ]  
```  
  
#### <a name="parameters"></a>参数  
 *filename*  
 包含的帮助主题文件的名称。  
  
## <a name="remarks"></a>备注  
 **Helpfile** c + + 属性具有相同的功能[helpfile](http://msdn.microsoft.com/library/windows/desktop/aa366853) MIDL 特性。
