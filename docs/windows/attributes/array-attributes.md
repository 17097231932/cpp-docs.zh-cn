# <a name="array-attributes"></a>数组特性

以下属性适用于数组或 `SAFEARRAY` 。

|Attribute|说明|
|---------------|-----------------|
|[library_block](library-block.md)|将构造置于 .idl 文件的库块内。|
|[satype](satype.md)|指定结构的数据类型 `SAFEARRAY` 。|
|[string](string-cpp.md)|指示 **`char`** 必须将一维、 **`wchar_t`** 、 `byte` （或等效）数组或指向此类数组的指针视为字符串。|
