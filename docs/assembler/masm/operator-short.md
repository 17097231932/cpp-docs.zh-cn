# <a name="operator-short"></a>运算符 SHORT

将 *标签* 的类型设置为 short。 跳转到 *标签的* 所有跳转必须介于-128 到 + 127 字节范围内的 (，才能 *标记*) 。

## <a name="syntax"></a>语法

> **短***标签*
