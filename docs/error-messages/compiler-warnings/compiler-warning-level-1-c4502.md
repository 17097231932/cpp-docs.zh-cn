# 编译器警告（等级 1）C4502
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“链接规范”要求使用关键字“extern”，并且该关键字必须在所有其他说明符之前  
  
 没有用 `extern` 关键字指定链接。  链接与非外部类型无关。  
  
 编译器假定为 `extern` 关键字。
