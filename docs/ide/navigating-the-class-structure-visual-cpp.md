# <a name="navigate-the-class-structure"></a>导航类结构

可使用“类视图”或“对象浏览器”导航项目类结构。 项目类结构可能包括函数、继承和接口。 有关详细信息，请参阅[查看代码的结构](/visualstudio/ide/viewing-the-structure-of-code)。

要转到函数声明，请在“类视图”中双击函数名称。 要转到类定义，请在“类视图”中双击类名。
