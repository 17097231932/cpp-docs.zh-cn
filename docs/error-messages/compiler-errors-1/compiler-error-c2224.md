# <a name="compiler-error-c2224"></a>编译器错误 C2224

". identifier" 的左侧必须具有结构/联合类型

句点 ( 的左操作数。 ) 不是类、结构或联合。

此错误可能是由于左操作数是未定义的变量 (因此键入 **`int`**) 。
