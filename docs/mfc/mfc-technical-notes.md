# <a name="mfc-technical-notes"></a>MFC 技术说明

技术说明是程序员为程序员编写的文档。

每篇技术说明描述了 MFC 文档范围之外的一个问题或一个功能。 提供的技术说明反映了用户的信息请求，以及 MFC 开发人员预计高级用户需要的专业信息。

浏览技术说明的方式有两种：

- [按编号的技术说明](technical-notes-by-number.md)

- [按类别列出的技术说明](technical-notes-by-category.md)
