# <a name="columnentrytype"></a>COLUMN_ENTRY_TYPE
表示数据库中的特定列的绑定。 支持`type`参数。  
  
## <a name="syntax"></a>语法  
  
```  
  
COLUMN_ENTRY_TYPE (  
nOrdinal  
,   
wType  
,   
data  
 )  
  
```  
  
#### <a name="parameters"></a>参数  
 `nOrdinal`  
 [in] 列号。  
  
 `wType`  
 [in]数据类型列条目。  
  
 `data`  
 [in] 用户记录中的对应数据成员。  
  
## <a name="remarks"></a>备注  
 此宏是专用的变体[COLUMN_ENTRY](../../data/oledb/column-entry.md)提供一种指定数据类型方法的宏。
