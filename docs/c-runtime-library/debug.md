# _DEBUG
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

当指定 \/MTd 或 \/MDd 选项时，编译器会定义 `_DEBUG`。  这些选项指定调试 C 运行库的版本。  
  
 有关更多信息，请参见[CRT 调试技术](../Topic/CRT%20Debugging%20Techniques.md)。
