# 编译器错误 C2338
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”ATL 特性提供程序  
  
 该特性引用了未声明的标识符。 你可能会看到与 [ATL 提供程序错误 ATL2010](http://msdn.microsoft.com/zh-cn/27a25924-b52b-4971-80d3-c3b3e9ffbe41) 关联的这项错误。
