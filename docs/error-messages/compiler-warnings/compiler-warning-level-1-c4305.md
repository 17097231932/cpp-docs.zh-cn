# <a name="compiler-warning-level-1-c4305"></a>编译器警告（等级 1）C4305
identifier： 从 type1 到 type2 截断  
  
 标识符转换为较小的类型，从而导致信息丢失。  
  
 如果你使用 ATL 连接点向导，请参阅知识库文章：  
  
-   Q250847: ATL 连接点向导生成的代码具有 variant 类型的值的自变量的事件获得 C4305 警告
