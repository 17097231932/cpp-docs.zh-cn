# NMAKE 警告 U4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

命令文件只能从命令行调用  
  
 At 符 \(**@**\) 说明符所调用的命令文件不能包含另一个命令文件的规范。  不允许这样的嵌套。  忽略该规范。
