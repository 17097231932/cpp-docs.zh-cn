# <a name="ml-warning-a4012"></a>ML 警告 A4012
**行号信息而不需要类代码的段**  
  
 没有没有结尾"的代码。"的类名称的段中的说明 汇编程序没有生成这些说明的 CodeView 信息。  
  
 CodeView 无法处理具有带未以"代码。 结束的类名称段中的代码模块
