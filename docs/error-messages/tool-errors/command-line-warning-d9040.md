# <a name="command-line-warning-d9040"></a>命令行警告 D9040
选项忽略 '/ 分析';代码分析警告不在此版本的编译器中可用  
  
 **/ 分析**命令行选项不是所有版本的 Visual Studio 中提供。 若要解决此警告，请切换到受支持版本的 Visual Studio 或移除该命令行选项。
