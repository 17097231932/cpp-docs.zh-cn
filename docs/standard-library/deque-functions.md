## <a name="swap"></a>  swap

交换两个 deque 的元素。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>参数

*left*<br/>
一个 `deque` 类型的对象。

*right*<br/>
一个 `deque` 类型的对象。
