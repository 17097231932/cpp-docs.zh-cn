# 编辑消息处理程序
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

定义了消息处理程序后，可以转到成员函数定义以添加或修改代码。  
  
 若要从[对话框编辑器](../../mfc/dialog-editor.md)跳转到成员函数定义，请双击为其定义了处理程序的控件。  这使您定位到在其中定义选定控件的消息处理程序的文件。  
  
 若要从[“类视图”](http://msdn.microsoft.com/zh-cn/8d7430a9-3e33-454c-a9e1-a85e3d2db925)跳转到成员函数定义，请在“类视图”中双击函数名。
