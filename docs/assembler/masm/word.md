# <a name="word"></a>WORD

为每个 *初始值设定项* 分配和（可选）初始化一个单词 (2 字节的存储) 。 在类型合法的任何位置，还可以用作类型说明符。 [DW](dw.md) 是 **WORD** 的同义词。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **WORD** *初始值设定* 项⟦__，__ *初始值设定项* .。。⟧
