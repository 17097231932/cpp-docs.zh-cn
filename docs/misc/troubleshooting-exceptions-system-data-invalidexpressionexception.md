# 关于异常的疑难解答：System.Data.InvalidExpressionException
当尝试向 <xref:System.Data.InvalidExpressionException> 添加包含无效 <xref:System.Data.DataColumn> 的 <xref:System.Data.DataColumn.Expression%2A> 时，将引发 <xref:System.Data.DataColumnCollection> 异常。  
  
## 相关提示  
 **检查表达式的语法。**  
 <xref:System.Data.DataColumn.Expression%2A> 属性用于计算一列的值或创建聚合列。
