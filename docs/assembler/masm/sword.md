# <a name="sword"></a>SWORD

分配每个*初始值设定项*并根据需要对其进行初始化（2个字节）。 在类型合法的任何位置，还可以用作类型说明符。

## <a name="syntax"></a>语法

> *名称***剑***初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
