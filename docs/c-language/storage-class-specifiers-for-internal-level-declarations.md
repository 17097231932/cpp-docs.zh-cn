# <a name="storage-class-specifiers-for-internal-level-declarations"></a>内部级别声明的存储类说明符
可以对内部级别的变量声明使用四个 storage-class-specifier 终止符中的任何一个。 如果省略此类声明中的 storage-class-specifier，则默认的存储类将是 auto。因此，很少在 C 程序中显示关键字 auto。
