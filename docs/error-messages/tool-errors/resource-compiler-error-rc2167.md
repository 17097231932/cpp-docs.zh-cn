# <a name="resource-compiler-error-rc2167"></a>资源编译器错误 RC2167
无法识别的 VERSIONINFO 字段;开始或需要逗号  
  
 不可识别的字段位于**固定**属于**VERSIONINFO**结构声明。 A **VERSIONINFO**字段必须是用逗号分隔的 dword 值。
