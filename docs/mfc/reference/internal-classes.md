# 内部类
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

下面的类内部 MFC 中使用。  为了保持完整性，本节将说明这些内部类，但这些内部类不在代码中直接使用。  
  
## 本节内容  
  
|类|说明|  
|-------|--------|  
|`CCommandManager Class`|`CCommandManager` 类管理命令及其关联带图像的。|  
|`CDialogImpl Class (MFC)`|提供执行详细信息。基于对话框的类，如 `CDialogEx`。|  
|`CDocumentAdapter Class`|实现需要搜索所需的 IDocument 接口和组织处理程序。|  
|`CFrameImpl Class`|以下框架窗口的 `CFrameImpl` 类处理工具栏自定义类：[CFrameWndEx Class](../../mfc/reference/cframewndex-class.md)[CMDIFrameWndEx 类](../../mfc/reference/cmdiframewndex-class.md)[COleIPFrameWndEx Class](../../mfc/reference/coleipframewndex-class.md)。|  
|`CFullScreenImpl Class`|实现满屏的共有功能 SDI、MDI 应用程序。|  
|`CMDIClientAreaWnd Class`|`CMDIClientAreaWnd` 类简化是 MDI 选项卡和 MDI 制表符分隔的团队实现的帮助程序类。|  
|`CMemDC Class`|内存设备上下文的帮助器类。  内存设备上下文支持屏幕外的绘图。|  
|`CMenuHash Class`|读取 `CMFCToolBar` 对象的状态从存档或写状态到存档。|  
|`CMenuImages Class`|提供功能显示预定义的图像 \(例如关闭按钮，最大化按钮、单选按钮和箭头。|  
|`CMFCCaptionButtonEx Class`|保留系统标题按钮非工作区。|  
|`CMFCCaptionMenuButton Class`|表示该任务窗格或工具栏的标题区域中的菜单按钮。|  
|`CMFCColorPropertySheet Class`|[CMFCColorDialog Class](../../mfc/reference/cmfccolordialog-class.md) 用于提供一个选项卡式对话框。|  
|`CMFCControlBarImpl Class`|提供执行详细信息为控件基于横条的类，如 `CMFCReBar` 和 `CMFCToolBar`。|  
|`CMFCControlRenderer Class`|帮助程序类处理该图像。|  
|`CMFCControlRendererInfo Class`|传递的参数的帮助程序类。`CMFCControlRenderer` 类。|  
|`CMFCCustomizeButton Class`|支持用于启动 用户对话框的菜单命令或工具栏按钮。|  
|`CMFCCustomizeMenuButton Class`|表示自定义菜单上的一个按钮。|  
|`CMFCDropDownListBox Class`|提供下拉列表框功能添加到功能区到组合框。|
