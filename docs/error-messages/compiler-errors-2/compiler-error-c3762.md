# <a name="compiler-error-c3762"></a>编译器错误 C3762

无法处理特性 "attribute"

继承自的用户定义的属性 `System.Security.Permissions.SecurityAttribute` 正用于定义安全特性。 此类特性不能在定义它的同一程序集中使用。
