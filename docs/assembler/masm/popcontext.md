# POPCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

还原部件或所有当前 `context` \(保存的 [PUSHCONTEXT](../../assembler/masm/pushcontext.md) 指令\)。  `context` 可以是 **假定**、 `RADIX`、 **列表**、 **CPU**或 **任何**。  
  
## 语法  
  
```  
  
POPCONTEXT context  
```
