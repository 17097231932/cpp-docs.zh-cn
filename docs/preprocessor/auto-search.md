# <a name="autosearch"></a>auto_search
**C + + 专用**  
  
 指定当使用 #import 引用了类型库，并且它本身引用了另一个类型库时，编译器可以为其他类型库执行隐式 #import。  
  
## <a name="syntax"></a>语法  
  
```  
auto_search  
```  
  
## <a name="remarks"></a>备注  
 **结束 c + + 专用**
