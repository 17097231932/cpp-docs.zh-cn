# MSBuild 错误 MSB3157
**MSB3157: 未能匹配项“\<程序包\>”的区域性“\<区域性\>”。  改用区域性“\<区域性\>”。**  
  
 指定产品无法找到使用指定区域性的包清单文件 \(package.xml\) 时会产生此警告。  
  
### 更正此错误  
  
-   提供相应的程序包清单文件。
