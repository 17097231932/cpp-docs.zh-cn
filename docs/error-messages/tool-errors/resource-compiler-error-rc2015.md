# 资源编译器错误 RC2015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

常数中字符太多  
  
 一个字符常数包含的字符多于两个。  
  
 字符常数限于一个字符（标准字符常数）或两个字符（长字符常数）。  
  
 请注意，换码序列（例如，\\t 对应制表符）被转换为单个字符。
