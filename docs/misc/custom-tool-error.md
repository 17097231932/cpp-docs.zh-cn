# Custom Tool Error
自定义工具生成错误或警告。  自定义工具是可用于在设计时将文件从一种类型转换为另一种类型的组件。  例如，自定义工具可以是一个数据集代码生成器，它在一个 XML 架构 \(.xsd\) 文件中读取，在以编程方式公开其表和列的代码文件中生成类。  
  
 **更正此错误**  
  
-   参考自定义工具供应商所提供的文档。
