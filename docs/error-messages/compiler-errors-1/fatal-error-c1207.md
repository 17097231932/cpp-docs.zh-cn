# <a name="fatal-error-c1207"></a>错误 C1207

安装的运行时版本不支持托管模板

使用的编译器是当前版本而公共语言运行时是早期版本时将发生 C1207 错误。

编译器的一些功能在早期的运行时版本中可能无效。

若要解决 C1207 问题，请安装你的编译器专用的公共语言运行时版本。
