# <a name="resource-compiler-warning-rw4001"></a>资源编译器警告 RW4001
.EXE 处理选项 (/L /M/P /T /K /E /31 或/30)  
  
 您已指定 EXE 处理选项，但没有任何要处理的可执行文件。 不要将用于.res 文件的这些选项。
