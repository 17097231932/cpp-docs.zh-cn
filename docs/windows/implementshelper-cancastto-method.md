# ImplementsHelper::CanCastTo 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

支持 WRL基础结构，不应在代码中直接使用。  
  
## 语法  
  
```  
  
HRESULT CanCastTo(  
   REFIID riid,  
   _Deref_out_ void **ppv  
);  
  
HRESULT CanCastTo(  
   _In_ const IID &iid,  
   _Deref_out_ void **ppv  
);  
```  
  
#### 参数  
 `riid`  
 对接口 ID. 的引用  
  
 `ppv`  
 如果该操作成功，给接口的指针`riid` 点由 `iid`指定。  
  
 `iid`  
 对接口 ID. 的引用  
  
## 返回值  
 S\_OK，如果成功；否则，一错误的 HRESULT。  
  
## 备注  
 获取特定接口 ID的指针。
