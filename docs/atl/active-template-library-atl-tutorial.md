# <a name="active-template-library-atl-tutorial"></a>活动模板库 (ATL) 教程

ATL 旨在简化创建高效、灵活的轻型控件的过程。 本教程将引导你完成创建 ActiveX 控件的过程，其中演示了许多 ATL 和 COM 的基础知识。

通过学习本教程，您将了解如何将控件添加到用于绘制圆形和填充多边形的 ATL 项目。 然后，您将添加一个属性以指示多边形将具有的边数，并创建在属性更改时用于更新控件的绘图代码。 然后，该控件将在网页上显示，并使用某些 VBScript 来响应事件。

本教程分为七个步骤。 你应按顺序执行每个步骤，因为后续步骤取决于先前完成的任务。 在开始之前，应确认你具有在特定计算机上注册 ActiveX 组件所需的权限。 如果在终端服务连接上运行 Visual Studio .NET，通常只需考虑这一点。

- [步骤1：创建项目](creating-the-project-atl-tutorial-part-1.md)

- [步骤2：将控件添加到项目](adding-a-control-atl-tutorial-part-2.md)

- [步骤3。向控件添加属性](adding-a-property-to-the-control-atl-tutorial-part-3.md)

- [步骤4：更改控件的绘图代码](changing-the-drawing-code-atl-tutorial-part-4.md)

- [步骤5：添加事件](adding-an-event-atl-tutorial-part-5.md)

- [步骤6：添加属性页](adding-a-property-page-atl-tutorial-part-6.md)

- [步骤7：将控件置于网页上](putting-the-control-on-a-web-page-atl-tutorial-part-7.md)
