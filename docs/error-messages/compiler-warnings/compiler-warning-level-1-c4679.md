# 编译器警告（等级 1）C4679
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

编译器遇到了它无法支持的结构，该结构无法从元数据中导入。  
  
 **“**   
 ***member* ”：未能导入成员**  
  
 不要尝试使用该结构。
