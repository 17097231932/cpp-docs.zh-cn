# 表达式计算器错误 CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**类型与运算符不匹配**  
  
 指定的操作数类型对于该操作不合法。  
  
 例如，指针不能与任何值相乘。  
  
 可能需要将操作数类型转换为与运算符匹配的类型。
