# <a name="compiler-warning-level-1-c4113"></a>编译器警告（等级1） C4113

"identifier1" 在 "identifier2" 的参数列表中不同

将函数指针分配给另一个函数指针，但函数的形参列表不一致。 不修改即可编译分配。
