# <a name="compiler-warning-level-4-c4324"></a>编译器警告（等级 4）C4324
struct_name： 结构已填充由于 __declspec(align())  
  
 因为你指定在结构的末尾添加填充[__declspec(align)](../../cpp/align-cpp.md)值。  
  
 例如，下面的代码生成 C4324:  
  
```  
// C4324.cpp  
// compile with: /W4  
struct __declspec(align(32)) A  
{  
   char a;  
};   // C4324  
  
int main()  
{  
}  
```
