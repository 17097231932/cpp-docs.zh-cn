# <a name="compiler-warning-level-1-c4344"></a>编译器警告（等级 1）C4344
行为更改: function 的调用中的显式模板自变量结果的使用  
  
 对使用显式模板自变量函数的调用不会像未指定显式参数一样调用其他函数
