# <a name="linker-tools-warning-lnk4199"></a>链接器工具警告 LNK4199

/DELAYLOAD:dllname 忽略;从 dll 名称中找到任何导入

链接器将忽略`dllname`因为它不需要的任何函数的`dllname`导出。
