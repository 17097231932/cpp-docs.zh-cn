# 编译器警告（等级 1）C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier1”和“identifier2”的参数列表不同  
  
 将一个函数指针分配给另一个函数指针，但这些函数的形参表不一致。  编译该分配时不进行修改。
