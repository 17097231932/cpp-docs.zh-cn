# <a name="grid-settings-dialog-box-image-editor-for-icons"></a>“网格设置”对话框（图标的图像编辑器）
可以指定你的映像的网格设置和编辑图像上显示网格线。 行可用于编辑映像，但不是保存为图像本身的一部分。  
  
 **像素网格**  
 选中时，图像编辑器中显示一个网格，每个像素周围。 仅在 4 × 及更高分辨率显示网格。  
  
 **磁贴网格**  
 选中时，显示的像素为单位的块周围的网格中网格间距值所指定的图像编辑器。  
  
 **宽度**  
 指定每个磁贴块的宽度。 绘制位图包含按固定间隔排列的多个映像时，这很有用。  
  
 **高度**  
 指定每个磁贴块的高度。 绘制位图包含按固定间隔排列的多个映像时，这很有用。  
  
## <a name="requirements"></a>要求  
 无
