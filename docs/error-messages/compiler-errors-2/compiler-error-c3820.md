# <a name="compiler-error-c3820"></a>编译器错误 C3820

“name”: 初始值设定项必须是托管的

托管类型的 *name* 初始值设定项也必须是托管的。

### <a name="to-correct-this-error"></a>更正此错误

1. 指定托管初始值设定项方法。
