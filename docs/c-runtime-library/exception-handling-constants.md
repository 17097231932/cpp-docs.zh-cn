# 异常处理常量
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

常数`EXCEPTION_CONTINUE_SEARCH`, `EXCEPTION_CONTINUE_EXECUTION`, 或者 `EXCEPTION_EXECUTE_HANDLER`返回当**try\-except**语句中守卫部分执行时发生异常。  返回值确定如何处理异常。  更多信息，请参见[try\-except Statement](../cpp/try-except-statement.md) 在 *C\+\+ Language Reference*中。
