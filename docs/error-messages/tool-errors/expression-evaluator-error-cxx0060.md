# <a name="expression-evaluator-error-cxx0060"></a>表达式计算器错误 CXX0060

注册不可用

表达式指定的寄存器不能使用。 尝试访问运行的计算机上不存在的注册时，可能会导致此错误。

此错误与 CAN0060 相同。
