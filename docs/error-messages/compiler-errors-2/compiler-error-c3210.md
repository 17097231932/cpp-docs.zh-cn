# <a name="compiler-error-c3210"></a>编译器错误 C3210

type： 访问声明可以仅应用于基类成员

一个[using 声明](../../cpp/using-declaration.md)未正确指定。

## <a name="example"></a>示例

下面的示例生成 C3210。

```
// C3210.cpp
// compile with: /c
struct A {
protected:
   int i;
};

struct B {
   using A::i;   // C3210
};

struct C : public A {
   using A::i;   // OK
};
```
