# <a name="compiler-warning-level-1-c4506"></a>编译器警告（等级 1）C4506

内联函数 function 没有定义

给定的函数声明并将其标记为内联，但未定义。

编译器没有内联函数。

请确保用声明外部函数内联`extern`关键字。
