# <a name="resource-compiler-fatal-error-rc1015"></a>资源编译器错误 RC1015

无法打开包含文件 filename

给定的包含文件不存在、 无法打开或找不到。

确保环境设置有效，并且已为该文件指定了正确的路径。 请确保足够的文件句柄可供资源编译器。 如果文件位于网络驱动器上，请确保您有权打开该文件。

即使包含文件的配置属性中附加包含目录-> 的资源所指定的目录中存在，可能发生 RC1015-> 常规属性页;指定包含文件的完整路径。
