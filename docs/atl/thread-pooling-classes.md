# Thread Pooling Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类支持线程池:  
  
-   [CComAutoThreadModule](../atl/reference/ccomautothreadmodule-class.md) 实现一个EXE模块，并对多个线程池的双工支持。  
  
-   [CComApartment](../atl/reference/ccomapartment-class.md) 管理在一个线程池的EXE模块的一个单元。  
  
-   [CComSimpleThreadAllocator](../atl/reference/ccomsimplethreadallocator-class.md) 管理EXE模块的线程选择。
