# <a name="strspnp-wcsspnp-mbsspnp-mbsspnpl"></a>_strspnp、_wcsspnp、_mbsspnp、_mbsspnp_l

返回指向给定字符串（不在另一个给定字符串）中的第一个字符的指针。

> [!IMPORTANT]
> **_mbsspnp**和 **_mbsspnp_l**不能用于在 Windows 运行时中执行的应用程序。 有关详细信息，请参阅[通用 Windows 平台应用中不支持的 CRT 函数](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md)。

## <a name="syntax"></a>语法

```C
char *_strspnp(
   const char *str,
   const char *charset
);
wchar_t *_wcsspnp(
   const unsigned wchar_t *str,
   const unsigned wchar_t *charset
);
unsigned char *_mbsspnp(
   const unsigned char *str,
   const unsigned char *charset
);
unsigned char *_mbsspnp_l(
   const unsigned char *str,
   const unsigned char *charset,
   _locale_t locale
);

```

### <a name="parameters"></a>参数

*str*<br/>
要搜索的 null 终止的字符串。

*字符集*<br/>
null 终止的字符集。

*locale*<br/>
要使用的区域设置。

## <a name="return-value"></a>返回值

**_strspnp**， **_wcsspnp**，和 **_mbsspnp**返回指向第一个字符的指针*str* ，不属于组字符*charset*。 其中每个函数返回**NULL**如果*str*完全由字符从组成*charset*。 对于这些例程，都不会保留返回值来指示错误。

## <a name="remarks"></a>备注

**_Mbsspnp**函数将指针返回到中的第一个字符的多字节字符*str* ，不属于组中的字符*charset*。 **_mbsspnp**识别多字节字符序列根据[多字节代码页](../../c-runtime-library/code-pages.md)当前正在使用。 搜索不包括结尾的 null 字符。

如果任一*str*或*charset*是 null 指针，此函数将调用无效参数处理程序，如中所述[参数验证](../../c-runtime-library/parameter-validation.md)。 如果允许执行继续，函数将返回**NULL**和设置**errno**到**EINVAL**。

### <a name="generic-text-routine-mappings"></a>一般文本例程映射

|Tchar.h 例程|未定义 _UNICODE 和 _MBCS|已定义 _MBCS|已定义 _UNICODE|
|---------------------|--------------------------------------|--------------------|-----------------------|
|**_tcsspnp**|**_strspnp**|**_mbsspnp**|**_wcsspnp**|

**_strspnp**和 **_wcsspnp**是单字节字符和宽字符版本的 **_mbsspnp**。 **_strspnp**和 **_wcsspnp**行为 **_mbsspnp**否则为; 它们仅为此映射提供并不用于任何其他原因。 有关详细信息，请参阅[使用一般文本映射](../../c-runtime-library/using-generic-text-mappings.md)和[一般文本映射](../../c-runtime-library/generic-text-mappings.md)。

**_mbsspnp_l**具有完全相同，只不过它改用已传入的区域设置参数。 有关详细信息，请参阅 [Locale](../../c-runtime-library/locale.md)。

## <a name="example"></a>示例

```C
// crt_mbsspnp.c
#include <mbstring.h>
#include <stdio.h>

int main( void ) {
   const unsigned char string1[] = "cabbage";
   const unsigned char string2[] = "c";
   unsigned char *ptr = 0;
   ptr = _mbsspnp( string1, string2 );
   printf( "%s\n", ptr);
}
```

### <a name="output"></a>输出

```Output
abbage
```
