# <a name="compiler-error-c2222"></a>编译器错误 C2222
意外类型“type”：应为基类或成员  
  
 初始值设定项列表只能初始化某个类型的基类或成员。 要修复此错误，请确保在初始值设定项列表中仅初始化该类型的基类或成员。
