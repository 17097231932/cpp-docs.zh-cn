# <a name="compiler-error-c3140"></a>编译器错误 C3140

不能在同一编译单元中具有多个 module 特性

[模块](../../windows/module-cpp.md)属性只能定义一次每个项目。

下面的示例生成 C3140:

```
// C3140.cpp
// compile with: /c
[emitidl];
[module(name = "MyLibrary")];
[module(name = "MyLibrary2")];   // C3140
```
