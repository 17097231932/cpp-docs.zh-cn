# Control Containment Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类提供包容为承载控件支持的:  
  
-   [CAxWindow](../atl/reference/caxwindow-class.md) 用于操作承载一个ActiveX控件的窗口的方法。  
  
-   [CAxWindow2T](../atl/reference/caxwindow2t-class.md) 用于操作承载一个ActiveX控件并为承载授权的ActiveX控件支持的windows提供方法。  
  
-   [IAxWinAmbientDispatch](../atl/reference/iaxwinambientdispatch-interface.md) 调用此接口的方法设置环境属性可用于承载的控件。  
  
-   [IAxWinHostWindow](../atl/reference/iaxwinhostwindow-interface.md) 调用此接口的方法创建和\/或控件附加到宿主对象，或者从获取对所承载的控件的接口。
