# &lt;forward_list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

容器定义模板类 forward\_list 和若干支持的模板。  
  
## 语法  
  
```  
#include <forward_list>  
```
