# <a name="creatormapservername-data-member"></a>CreatorMap::serverName 数据成员

支持 WRL 基础结构，不应在代码中直接使用。

## <a name="syntax"></a>语法

```cpp
const wchar_t* serverName;
```

## <a name="remarks"></a>备注

存储 CreatorMap 的服务器名称。

## <a name="requirements"></a>要求

**标头：** module.h

**Namespace:** Microsoft::WRL::Details
