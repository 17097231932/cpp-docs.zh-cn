# <a name="compiler-error-c2493"></a>编译器错误 C2493
__based 非法形式  
  
 A`__based`表达式必须基于指针。  
  
 下面的示例生成 C2493:  
  
```  
// C2493.cpp  
// compile with: /c  
char mybase;  
int __based(mybase) ptr;   // C2493  
  
// OK  
char * mybase;  
int __based(mybase) * ptr;  
```
