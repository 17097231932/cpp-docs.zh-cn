# <a name="compiler-error-c2036"></a>编译器错误 C2036
identifier： 未知的大小  
  
 上的操作`identifier`需要的数据对象，无法确定大小。  
  
## <a name="example"></a>示例  
 下面的示例生成 C2036。  
  
```  
// C2036.c  
// a C program  
struct A* pA;  
struct B { int i; } *pB;  
int main() {  
   pA++;   // C2036, size of A not known  
   ((char*)pA)++;   // OK  
  
   pB++;   // OK  
}  
```  
  
## <a name="example"></a>示例  
 下面的示例生成 C2036。  
  
```  
// C2036_2.cpp  
// a C++ program  
struct A* pA;  
int main() {  
   pA++;   // C2036, size of A not known  
   ((char*&)pA)++;   // OK, if sizeof(A) == sizeof(char)  
}  
```
