# <a name="compiler-error-c2808"></a>编译器错误 C2808

一元 operator operator 具有形参太多

一元运算符具有非 void 参数列表。

下面的示例生成 C2808:

```
// C2808.cpp
// compile with: /c
class X {
public:
   X operator! ( X );   // C2808 nonvoid parameter list
   X operator! ( void );   // OK
};
```
