# <a name="2724-shared"></a>2.7.2.4 shared

此子句共享中出现的变量*变量列表*在团队中的所有线程之间。 发生在团队内的所有线程都访问相同的存储区域以进行**共享**变量。

语法**共享**子句如下所示：

```
shared(variable-list)
```
