# Top-level menu names cannot contain separators.
该错误通常发生在尝试将包含分隔符（如短划线 \(\-\)）的文本拖动到菜单名称中时。  顶层菜单不能包含该字符，它是为菜单项的分隔设置保留的。  
  
### 更正此错误  
  
1.  输入不使用分隔符的文本。
