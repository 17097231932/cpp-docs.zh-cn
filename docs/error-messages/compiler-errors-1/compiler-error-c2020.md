# 编译器错误 C2020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“member”:“class”成员重定义  
  
 从基类或结构继承的成员被重定义。  不能重定义继承成员，除非它在基类中被声明为 `virtual`。
