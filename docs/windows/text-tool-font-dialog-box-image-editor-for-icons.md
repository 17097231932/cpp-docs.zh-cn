# “文本工具”-“字体”对话框（图标的图像编辑器）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用[图形编辑器](../mfc/image-editor-for-icons.md)的[“文本工具”对话框](../mfc/text-tool-dialog-box-image-editor-for-icons.md)时指定字体设置。  
  
 此对话框可用于：  
  
-   [更改图像上文本的字体](../mfc/changing-the-font-of-text-on-an-image-image-editor-for-icons.md)  
  
 要访问此对话框，请单击“文本工具”对话框中的**“字体”**按钮。  
  
 **字体**  
 列出可用的字体。  
  
 **Font Style**  
 列出指定字体的可用样式。  
  
 **大小**  
 列出指定字体的可用磅值。  
  
 **示例**  
 显示使用指定字体设置的文本将是什么样子的示例。  
  
 **Script**  
 列出指定字体的可用字符集。  当您选择某个不同的字符集时，该语言的字符集就可用来创建多语言文档。
