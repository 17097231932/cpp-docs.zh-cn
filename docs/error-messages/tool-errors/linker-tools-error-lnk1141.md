# 链接器工具错误 LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

导出文件生成期间失败  
  
 [LINK](../../build/reference/linker-command-line-syntax.md) 未能生成导出文件 \(.exp\)。  
  
 导致该错误的原因包括系统问题（如内存不足）和选项或模块定义语句中的语法错误。  
  
 在该错误之前有另一个提供附加信息的错误。
