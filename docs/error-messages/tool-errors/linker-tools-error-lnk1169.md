# <a name="linker-tools-error-lnk1169"></a>链接器工具错误 LNK1169
一个或多个多重找到定义的符号  
  
 由于多个定义的一个或多个符号，生成失败。 此错误之前错误[LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)。  
  
 [/强制](../../build/reference/force-force-file-output.md)或**/FORCE:MULTIPLE**选项将重写此错误。
