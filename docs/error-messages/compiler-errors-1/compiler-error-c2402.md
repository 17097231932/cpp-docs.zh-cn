# 编译器错误 C2402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：寄存器必须为“context”中的索引  
  
 间接存储操作数中使用的寄存器必须是此上下文中的索引寄存器。
