# 表达式计算器错误 CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

表达式不可扩展  
  
 表达式计算器遇到内部错误。  
  
 您可以写能被正确计算的等效表达式。  
  
 该错误与 CAN0031 相同。
