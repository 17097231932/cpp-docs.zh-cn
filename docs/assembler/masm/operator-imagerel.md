# <a name="operator-imagerel"></a>运算符 IMAGEREL

返回的映像相对偏移量`expression`。

## <a name="syntax"></a>语法

> IMAGEREL 表达式

## <a name="remarks"></a>备注

生成的值通常称为 RVA 或相对虚拟地址。

仅支持 COFF 对象发出 IMAGEREL。
