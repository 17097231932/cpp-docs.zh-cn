# 编译器警告（等级 1）C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof 返回 0  
  
 `sizeof` 运算符应用于零大小的操作数（空结构、联合、类、枚举类型或 `void` 类型）。
