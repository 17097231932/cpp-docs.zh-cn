# <a name="linker-tools-error-lnk1136"></a>链接器工具错误 LNK1136

无效或损坏的文件

输入文件具有损坏的标头，或者为零大小或异常小。

如果文件被错误地指定为附加的依赖项，也可能导致 LNK1136。
