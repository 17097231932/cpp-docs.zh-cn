# 错误 C1038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

编译器限制：function：控制流状态太复杂；请简化函数  
  
 该函数具有的控制流状态比编译器能够处理的多。 简化控制流或者将该函数拆分为更小的函数。
