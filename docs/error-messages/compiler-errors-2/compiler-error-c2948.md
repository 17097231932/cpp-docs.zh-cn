# 编译器错误 C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

显式实例化；专用化中不允许存储类说明符“specifier”  
  
 在专用化以前显式实例化的模板类时不能使用存储类说明符（如 `extern`）。
