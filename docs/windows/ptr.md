# <a name="ptr"></a>ptr

将一个指针，指定为完整的指针。

## <a name="syntax"></a>语法

```cpp
[ptr]
```

## <a name="remarks"></a>备注

**Ptr** c + + 属性具有相同的功能[ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL 特性。

## <a name="example"></a>示例

有关示例，请参阅[defaultvalue](../windows/defaultvalue.md)的示例使用**ptr**。
