# Command &lt;name&gt; is not valid.
该错误通常发生在在**查找\/命令**框或**命令**窗口中输入 Visual Studio 不识别的命令或命令的别名时。  如果输入了别名，在错误信息中将显示完整命令名而不是别名。  
  
### 更正此错误  
  
1.  重新输入该命令名或选择一个有效的命令名。
