# <a name="compiler-error-c3672"></a>编译器错误 C3672
伪析构函数表达式仅用作函数调用的一部分  
  
 未正确调用析构函数。  有关详细信息，请参阅[析构函数](../../cpp/destructors-cpp.md)。  
  
## <a name="example"></a>示例  
 下面的示例生成 C3672。  
  
```  
// C3672.cpp  
template<typename T>  
void f(T* pT) {  
   &pT->T::~T;   // C3672  
   pT->T::~T();   // OK  
};  
  
int main() {  
   int i;  
   f(&i);  
}  
```
