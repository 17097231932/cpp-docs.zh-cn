# 编译器错误 C2487
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 不能使用 dll 接口声明 dll 接口类的成员  
  
 可以使用 DLL 接口声明整个类或非 DLL 接口类的某些成员。  但不能使用 DLL 接口声明某个类后再用 DLL 接口声明该类的成员。
