# <a name="character-constants-and-conditional-inclusion"></a>字符常量和条件包含

**ANSI 3.8.1** 控制条件包含的常量表达式中的单个字符的字符常量的值是否与执行字符集中的相同字符常量的值匹配。 此类字符常量是否可具有负值

预处理器语句中使用的字符集与执行字符集相同。 预处理器识别负字符值。
