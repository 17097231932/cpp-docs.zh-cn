# <a name="compiler-warning-level-2-c4653"></a>编译器警告（等级 2）C4653
编译器选项 option 与预编译标头; 不一致忽略当前命令行选项  
  
 使用预编译标头指定的选项 ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) 选项不一致的预编译标头创建时指定的选项。 此编译使用预编译标头创建时指定的选项。  
  
 当包结构选项的不同值时，会出现此警告 ([/Zp](../../build/reference/zp-struct-member-alignment.md)) 的预编译标头在编译期间指定。
