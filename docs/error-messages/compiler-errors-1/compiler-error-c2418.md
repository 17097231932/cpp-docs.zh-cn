# 编译器错误 C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法删除浏览器文件: filename  
  
 编译器未能删除浏览器文件。  
  
### 通过检查以下可能的原因进行修复  
  
1.  文件为只读的。  
  
2.  文件正由另一个进程使用。
