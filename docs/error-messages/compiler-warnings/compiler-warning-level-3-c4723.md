# <a name="compiler-warning-level-3-c4723"></a>编译器警告（等级 3）C4723
潜在的被 0 除  
  
 计算结果为零编译时，给出了未定义的结果 divide 操作中的第二个操作数。  
  
 仅在使用时，将发出此警告[/Og](../../build/reference/og-global-optimizations.md)或暗指 /Og 的优化选项。  
  
 编译器可能生成的零个操作数。
