# <a name="compiler-error-c2212"></a>编译器错误 C2212

"identifier"： __based 不可用于指向函数的指针

不能声明指向函数 **`__based`** 的指针。 如果需要基于代码的数据，请使用 **`__declspec`** 关键字或 `data_seg` 杂注。
