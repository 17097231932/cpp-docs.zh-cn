# <a name="ml-fatal-error-a1005"></a>ML 错误 A1005

**汇编程序限制：宏参数名称表已满**

为宏定义的参数、局部变量或宏标签太多。 宏名称表中没有更多空间。

定义较短或更少的名称，或删除不必要的宏。
