# 编译器错误 C2753
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“class”: 已定义模板类  
  
 如果模板参数列表与参数列表匹配，则编译器将其视作同一模板。  不允许定义同一模板两次。  
  
 下面的示例生成 C2753：  
  
```  
// C2753.cpp  
template<class T>  
struct A {};  
  
template<class T>  
struct A<T> {};   // C2753  
// try the following line instead  
// struct A<int> {};  
  
template<class T, class U, class V, class W, class X>  
struct B {};  
```
