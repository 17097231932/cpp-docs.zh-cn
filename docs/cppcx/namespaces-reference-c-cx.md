## <a name="compiler-supplied-namespaces"></a>编译器提供的命名空间  
 若要简化面向 Windows 运行时 C + 程序的编码 + /cli CX 编译器及其支持的标头文件提供定义多种类型的命名空间。 命名空间定义内置数值类型、字符串、 数组和集合;表示 Windows 运行时错误; visual c + + 异常和标准的 Windows 运行时类型的特定语言增强功能。
