# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP 类
`CMFCVisualManagerOfficeXP` ，Microsoft Office XP 外观的应用程序。  
  
## <a name="syntax"></a>语法  
  
```  
class CMFCVisualManagerOfficeXP : public CMFCVisualManager  
```  
  
## <a name="members"></a>成员  
 所有此类的成员都派生自的此类上级的虚函数[CMFCVisualManager 类](../../mfc/reference/cmfcvisualmanager-class.md)。  
  
## <a name="remarks"></a>备注  
 你使用`CMFCVisualManagerOfficeXP`类，以更改你的应用程序类似于 Microsoft Office XP 的可视外观。
