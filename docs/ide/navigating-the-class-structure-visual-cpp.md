# 导航类结构 (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可使用[“类视图”](http://msdn.microsoft.com/zh-cn/8d7430a9-3e33-454c-a9e1-a85e3d2db925)或[对象浏览器](http://msdn.microsoft.com/zh-cn/f89acfc5-1152-413d-9f56-3dc16e3f0470)导航项目类结构，包括函数、继承和接口。  
  
 在“类视图”中，双击函数名可以转到函数声明。  在“类视图”中，双击类名可以转到类定义。
