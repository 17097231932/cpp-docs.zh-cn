# A failure occurred writing to the resources file
项目系统未能写入 .resources 输出文件。  作为准备资源过程的一部分，项目系统将带 .resx 扩展名的 XML 文件转换成二进制 .resources 文件供 .NET 资源管理器使用。  
  
 操作系统特定的原因（如果有的话）将追加到此消息的结尾。  同时还追加 .resx 文件的文件名。  
  
 此错误将使生成进程失败。
