# <a name="linker-tools-error-lnk1141"></a>链接器工具错误 LNK1141

导出文件生成期间失败

[LINK](../../build/reference/linking.md)无法生成导出（.exp）文件。

此错误的原因包括系统问题，例如内存不足和选项或模块定义语句中的语法错误。

此错误之前有另一个错误，该错误提供了其他信息。
