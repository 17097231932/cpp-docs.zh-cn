# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**无法打开文件:文件名**  
  
 此时将汇编无法打开数据源，包括或输出文件。  
  
 操作之一可能原因是:  
  
-   该文件不存在。  
  
-   文件正由另一个的是进程使用。  
  
-   文件名无效。  
  
-   具有输出文件名的只读文件已存在。  
  
-   当前驱动器已满。  
  
-   当前目录是根并已满。  
  
-   计算机未能写入。  
  
-   该驱动程序未就绪。
