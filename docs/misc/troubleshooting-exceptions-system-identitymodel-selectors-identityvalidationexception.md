# 关于异常的疑难解答：System.IdentityModel.Selectors.IdentityValidationException
当收件人证书无效时，将引发 <xref:System.IdentityModel.Selectors.IdentityValidationException> 异常。 如果证书未成功链接到根证书，则会发生此异常。
