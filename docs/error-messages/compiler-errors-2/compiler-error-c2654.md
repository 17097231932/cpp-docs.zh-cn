# <a name="compiler-error-c2654"></a>编译器错误 C2654

"identifier"：尝试访问成员函数的外部成员

在声明中访问成员。 只能在成员函数中访问成员数据。

尝试在声明中初始化变量时，可能会导致此错误。 使用构造函数来实现此目的。
