# CA0057
无法将冲突映射到当前目标和规则组  
  
 如果项目文件或导入的 XML 报告具有无法映射到当前目标和规则组的冲突，则对它手动调用 FxCopCmd.exe 时会发生此错误。  
  
 仅在使用该工具的命令行版本进行分析的情况下才会发生此错误。
