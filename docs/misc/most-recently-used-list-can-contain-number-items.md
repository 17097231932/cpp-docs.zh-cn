# Most recently used list can contain &lt;number&gt; items.
MRU 列表可以包含最多 24 项。  出现该错误的条件通常是：在“选项”对话框“环境”选项“常规”窗格上的“最近使用列表可以包含 \<数字\> 项”框中输入了小于 1 或大于 24 的数字。  
  
### 更正此错误  
  
1.  在框中输入一个介于 1 到 24 之间的数字。
