# 编译器错误 C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

重载的“operator \-\>的无效返回类型“type”  
  
 `->` 运算符必须返回一个类、结构或联合，或者返回对其中之一的引用。
