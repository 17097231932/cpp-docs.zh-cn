# CA0067
找不到目录。  
  
 未找到 FxCopCmd 命令行中 **\/directory** 选项的值。  
  
 如果要使用 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] 的 **CodeAnalysisDependentAssemblyPaths** 属性来指定其他引用程序集搜索路径而且这些路径之一不存在，则可能会出现此警告。  
  
-   如果警告 CA0067 和 [CA0058](../misc/ca0058.md) 错误或 [CA0060](../misc/ca0060.md) 警告一起显示，解决其他错误通常可解决 CA0067。  
  
-   如果警告 CA0067 不和其他错误或警告一起显示，通常可以忽略该警告，因为不需要目录。
