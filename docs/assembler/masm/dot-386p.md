# <a name="386p-32-bit-masm"></a>. .386P （32）

启用80386处理器的所有说明的程序集（包括特权）;禁用由更高版本的处理器引入的指令的程序集。 （仅限32位 MASM。）

## <a name="syntax"></a>语法

> .386P

## <a name="remarks"></a>备注

还启用80387说明。
