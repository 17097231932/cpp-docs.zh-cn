# <a name="project-build-error-prj0007"></a>项目生成错误 PRJ0007

未能创建输出目录“”。

Visual C++ 无法创建输出目录。 原因可能为：

- 要在其中创建输出目录的目录是只读的。

- 计算机的磁盘空间不足。

- 指定的目录路径无效。

- 用户权限不足，无法创建目录。
