# <a name="command-line-warning-d9024"></a>命令行警告 D9024

无法识别的源文件类型 "filename"，假定为对象文件

无法识别指定文件的扩展名。 该文件被认为是对象文件，并被传递到链接器。

识别以下扩展：

- .c （C 源文件）

- . .cxx （C++源文件）

- .cpp （C++源文件）

- .obj （对象文件）

- .lib （库文件）

- .def （模块定义文件）

- exp （链接器导出文件，由链接/LIB 创建）
