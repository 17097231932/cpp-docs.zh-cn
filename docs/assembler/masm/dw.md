# <a name="dw"></a>DW

为每个 *初始值设定项* 分配和（可选）初始化一个单词 (2 字节的存储) 。 在类型合法的任何位置，还可以用作类型说明符。 **DW** 是 [WORD](word.md)的同义词。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **DW** *初始值设定项* ⟦__，__ *初始值设定项* .。。⟧
