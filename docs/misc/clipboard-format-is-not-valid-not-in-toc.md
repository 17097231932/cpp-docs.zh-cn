# Clipboard format is not valid.
剪贴板仅接受指定格式的数据。  当尝试将组件中的数据放入剪贴板，但这些数据的格式与剪贴板不兼容时，通常会发生此错误。  
  
### 更正此错误  
  
1.  参考组件的文档，确定是否能使用“剪贴板”传输组件中的数据。  
  
     \- 或 \-  
  
2.  选择其他项，然后重试。
