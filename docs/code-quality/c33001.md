# <a name="c33001"></a>C33001

> 警告 C33001：变量 "var" 在 (表达式 "expr" 未初始化时被清除 ) 

如果将未初始化的变体传递到需要初始化变体的 API （例如 VariantClear），则会触发此警告。

## <a name="example"></a>示例

```cpp
#include <Windows.h>

HRESULT foo(bool some_condition)
{
    VARIANT var;

    if (some_condition)
    {
        //...
        VariantInit(&var);
        //...
    }

    VariantClear(&var);     // C33001
}
```

通过确保仅为正确初始化的变量调用 VariantClear，可以更正这些警告：
```cpp
#include <Windows.h>

HRESULT foo(bool some_condition)
{
    VARIANT var;

    if (some_condition)
    {
        //...
        VariantInit(&var);
        //...
        VariantClear(&var);     // C33001
    }
}
```
