# <a name="nmake-fatal-error-u1000"></a>NMAKE 错误 U1000

> 语法错误：宏调用中缺少 "）"

宏调用中的左括号 **（** ，显示没有匹配的右括号 **）** 。 正确的形式是 **$ （** <em>name</em> **）** ;单字符名称允许 **$** <em>n</em> 。
