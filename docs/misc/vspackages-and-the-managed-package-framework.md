# VSPackage 和托管包框架
通过使用 COM 互操作类，您可以通过创建 VSPackage 减少开发时间的托管包结构 \(MPF\)类而不是。  
  
 有两种创建托管 VSPackage:  
  
-   使用 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 包 " 项目模板  
  
     有关更多信息，请参见 [演练：使用 Visual Studio 包模板创建菜单命令](../Topic/Walkthrough:%20Creating%20a%20Menu%20Command%20By%20Using%20the%20Visual%20Studio%20Package%20Template.md)。  
  
-   生成 VSPackage，而无需 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 包 " 项目模板  
  
     例如，可以复制示例 VSPackage 和更改 GUID 和名称。  可以找到在 VSX 节中的示例。 [Code Gallery](http://code.msdn.microsoft.com/vsx/)
