# <a name="cmfcdropdownframe-class"></a>CMFCDropDownFrame 类


## <a name="cmfcdropdownframecreate"></a><a name="create"></a> CMFCDropDownFrame：： Create

创建一个 `CMFCDropDownFrame` 对象。

```
virtual BOOL Create(
    CWnd* pWndParent,
    int x,
    int y,
    CMFCDropDownToolBar* pWndOriginToolbar);
```

### <a name="parameters"></a>参数

*pWndParent*\
中下拉框的父窗口。

*x-blade*\
中下边框位置的水平屏幕坐标。

*误差*\
中下边框位置的垂直屏幕坐标。

*pWndOriginToolbar*\
中具有此方法用来填充新下拉框架对象的下拉按钮的工具栏。

### <a name="return-value"></a>返回值

如果已成功创建下拉框架，则为 TRUE;否则为 FALSE。

### <a name="remarks"></a>注解

此方法调用基本 [CMiniFrameWnd：： CreateEx](../../mfc/reference/cminiframewnd-class.md#createex) 方法来创建具有 WS_POPUP 样式的下拉框窗口。 下拉框架窗口将出现在指定的屏幕坐标中。 如果 [CMiniFrameWnd：： CreateEx](../../mfc/reference/cminiframewnd-class.md#createex) 方法返回 FALSE，则此方法将失败。

`CMFCDropDownFrame`类创建提供的参数的副本 `CMFCDropDownToolBar` 。 此方法从参数将按钮图像和按钮状态复制 `pWndOriginToolbar` 到 `m_pWndOriginToolbar` 数据成员。

## <a name="cmfcdropdownframegetparentmenubar"></a><a name="getparentmenubar"></a> CMFCDropDownFrame::GetParentMenuBar

检索下拉框的父菜单栏。

```
CMFCMenuBar* GetParentMenuBar() const;
```

### <a name="return-value"></a>返回值

指向下拉框的父菜单栏的指针; 如果该框架没有父项，则为 NULL。

### <a name="remarks"></a>注解

此方法从父按钮检索父菜单栏。 如果下拉框没有父按钮或父按钮没有父菜单栏，则此方法返回 NULL。

## <a name="cmfcdropdownframegetparentpopupmenu"></a><a name="getparentpopupmenu"></a> CMFCDropDownFrame::GetParentPopupMenu

检索下拉框的父弹出菜单。

```
CMFCDropDownFrame* GetParentPopupMenu() const;
```

### <a name="return-value"></a>返回值

指向下拉框的父下拉菜单的指针，如果该框架没有父项，则为 NULL。

### <a name="remarks"></a>注解

此方法从父按钮检索父菜单。 如果下拉框没有父按钮或父按钮没有父菜单，则此方法返回 NULL。

## <a name="cmfcdropdownframerecalclayout"></a><a name="recalclayout"></a> CMFCDropDownFrame：： RecalcLayout

重新定位下拉框。

```
virtual void RecalcLayout(BOOL bNotify = TRUE);
```

### <a name="parameters"></a>参数

*bNotify*\
中用.

### <a name="remarks"></a>注解

创建下拉框架或调整父窗口大小时，框架将调用此方法。 此方法使用父窗口的位置和大小来计算下拉框的位置和大小。

## <a name="cmfcdropdownframesetautodestroy"></a><a name="setautodestroy"></a> CMFCDropDownFrame::SetAutoDestroy

设置子下拉工具栏窗口是否自动销毁。

```cpp
void SetAutoDestroy(BOOL bAutoDestroy = TRUE);
```

### <a name="parameters"></a>参数

*bAutoDestroy*<br/>
中若要自动销毁关联的下拉工具栏窗口，则为 TRUE;否则为 FALSE。

### <a name="remarks"></a>注解

如果 *bAutoDestroy* 为 TRUE，则 `CMFCDropDownFrame` 析构函数会销毁关联的下拉工具栏窗口。 默认值为 TRUE。
