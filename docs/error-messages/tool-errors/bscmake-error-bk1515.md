# <a name="bscmake-error-bk1515"></a>BSCMAKE 错误 BK1515
bscfile： 版本不兼容： 无法以增量方式更新  
  
 .Bsc 文件是使用 BSCMAKE 的另一个版本创建的。 重新创建.bsc 文件。
