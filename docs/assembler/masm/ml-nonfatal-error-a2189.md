# <a name="ml-nonfatal-error-a2189"></a>ML 非致命错误 A2189

**与段对齐方式的组合无效**

**ALIGN** 或指令指定的对齐方式 `EVEN` 大于由 **segment** 指令指定的当前段对齐方式。
