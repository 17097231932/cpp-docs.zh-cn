# <a name="compiler-error-c3817"></a>编译器错误 C3817
declaration： 属性可以仅应用于函数  
  
 `property`关键字只能应用于函数定义。  
  
 C3817 才可访问使用过时的编译器选项**/clr:oldSyntax**。
