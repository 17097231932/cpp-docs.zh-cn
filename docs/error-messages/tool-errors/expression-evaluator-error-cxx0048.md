# 表达式计算器错误 CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支持调用序列  
  
 表达式中指定的函数使用 C 表达式计算器不支持的调用顺序。  不能在“监视”窗口表达式中调用该函数。  
  
 该错误与 CAN0048 相同。
