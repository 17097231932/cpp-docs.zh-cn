# 编译器警告（等级 1）C4900
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“tool1”版本“version1”和“tool2”版本“version2”之间的中间语言不匹配  
  
 *tool1* 和 *tool2* 中使用的中间语言不匹配。 检查是否已安装各工具的最新版本。
