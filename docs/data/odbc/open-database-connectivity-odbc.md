# <a name="open-database-connectivity-odbc"></a>开放式数据库连接 (ODBC)

有关使用 ODBC API 而不使用 MFC 的详细信息，请参阅[开放式数据库连接 (ODBC)](/sql/odbc/microsoft-open-database-connectivity-odbc) MSDN 中的 SDK。
