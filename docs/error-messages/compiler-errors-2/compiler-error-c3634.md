# <a name="compiler-error-c3634"></a>编译器错误 C3634
function： 不能定义托管或 WinRTclass 的一个抽象方法  
  
 可在托管或 WinRT 类中声明一个抽象方法，但不能定义它。  
  
## <a name="example"></a>示例  
以下示例生成 C3634：  
  
```  
// C3634.cpp  
// compile with: /clr  
ref class C {  
   virtual void f() = 0;  
};  
  
void C::f() {   // C3634 - don't define managed class' pure virtual method  
}  
```
