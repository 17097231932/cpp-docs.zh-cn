# 表达式计算器错误 CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支持构造函数调用  
  
 表达式调用了构造函数。  表达式不能显式调用构造函数或进行需要调用构造函数的转换。  
  
 该错误与 CAN0062 相同。
