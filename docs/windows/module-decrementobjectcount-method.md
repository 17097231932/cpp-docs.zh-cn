# Module::DecrementObjectCount 方法
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

递减模块所跟踪对象的数量。  
  
## <a name="syntax"></a>语法  
  
```  
virtual long DecrementObjectCount();  
```  
  
## <a name="return-value"></a>返回值  
 递减操作之前的计数。  
  
## <a name="requirements"></a>要求  
 **标头︰** module.h  
  
 **命名空间：** Microsoft::WRL
