# BSCMAKE 错误 BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bscfile 损坏；无法增量更新  
  
 可能因生成过程中系统故障而导致此 .bsc 文件损坏。  删除此 .bsc 文件，重新生成所有 .sbr 文件，然后重新生成此 .bsc 文件。
