# <a name="operator-lroffset"></a>运算符 LROFFSET

返回*表达式*的偏移量。 与**offset**相同，但它会生成加载器解析的偏移量，从而允许 Windows 重新定位代码段。

## <a name="syntax"></a>语法

> **LROFFSET** *表达式*
