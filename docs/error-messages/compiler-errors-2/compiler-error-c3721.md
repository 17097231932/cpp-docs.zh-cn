# <a name="compiler-error-c3721"></a>编译器错误 C3721
签名： 事件不兼容的签名  
  
 未正确声明事件。 有关详细信息，请参阅[__event](../../cpp/event.md)。  
  
 C3721 才可访问使用过时的编译器选项**/clr:oldSyntax**。
