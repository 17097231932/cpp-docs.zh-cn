# <a name="scope-of-atl"></a>ATL 的范围

ATL 允许您轻松创建 COM 对象、自动化服务器和 ActiveX 控件。 ATL 提供对许多基本 COM 接口的内置支持。

ATL 作为源代码包含在您的应用程序中。 ATL 还会使 DLL 可用 ( # A0) ，其中包含可跨组件共享的代码。 但是，此 DLL 不是必需的。
