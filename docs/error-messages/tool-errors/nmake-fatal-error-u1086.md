# NMAKE 错误 U1086
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**推理规则不能有从属项**  
  
 推理规则中的冒号 \(**:**\) 后面必须跟下面一种符号：  
  
-   换行符  
  
-   分号 \(**;**\)，后面可跟命令  
  
-   数字记号 \(**\#**\)，后面可跟注释
