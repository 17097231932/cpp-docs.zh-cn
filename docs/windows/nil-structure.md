# <a name="nil-structure"></a>Nil 结构
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
struct Nil;  
```  
  
## <a name="remarks"></a>备注  
 用于指示未指定可选的模板参数。  
  
 **Nil**是一个空的结构。  
  
## <a name="inheritance-hierarchy"></a>继承层次结构  
 `Nil`  
  
## <a name="requirements"></a>要求  
 **标头：** implements.h  
  
 **Namespace:** Microsoft::WRL::Details
