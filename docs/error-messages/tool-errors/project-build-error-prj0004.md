# 项目生成错误 PRJ0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未能为“tool”工具生成命令行。  
  
 指定了一个或多个属性，但所采用的方式使 ***tool*** 调用的语法不合法：  
  
-   可能指定了格式错误的宏或未知的宏。  
  
-   计算机的可用磁盘空间可能不足。  
  
 可能需要通过查看[命令行属性页](../../ide/command-line-property-pages.md)来检查工具的设置。
