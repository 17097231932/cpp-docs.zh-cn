# <a name="compiler-error-c2943"></a>编译器错误 C2943
“class”：type-class-id 重新定义为模板的类形参数  
  
 你不能使用泛型类或模板类，而是改用符号作为泛型或模板类型参数。  
  
 以下示例生成 C2943：  
  
```  
// C2943.cpp  
// compile with: /c  
template<class T>  
class List {};  
  
template<class List<int> > class MyList;   // C2943  
template<class T >  class MyList;  
```
