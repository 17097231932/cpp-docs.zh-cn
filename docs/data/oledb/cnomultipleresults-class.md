# CNoMultipleResults 类
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

作为模板参数使用 \(*TMultiple*\)为[CCommand](../../data/oledb/ccommand-class.md)来创建一个已优化的命令处理单一结果集。  
  
## 语法  
  
```  
class CNoMultipleResults  
```  
  
## 备注  
 如果您想让命令处理多个结果集，请使用 [CMultipleResults](../../data/oledb/cmultipleresults-class.md)。  
  
## 要求  
 **Header:** atldbcli.h
