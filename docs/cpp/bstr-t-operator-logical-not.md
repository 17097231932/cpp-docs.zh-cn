# <a name="bstrtoperator-"></a>_bstr_t::operator !

**Microsoft 专用**

检查封装`BSTR`是一个 NULL 字符串。

## <a name="syntax"></a>语法

```
bool operator!( ) const throw( );
```

## <a name="return-value"></a>返回值

如果是，则返回 TRUE，如果没有，则返回 FALSE。
