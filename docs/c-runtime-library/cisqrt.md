# <a name="cisqrt"></a>_CIsqrt

计算堆栈顶部值的平方根。

## <a name="syntax"></a>语法

```
void __cdecl _CIsqrt();
```

## <a name="remarks"></a>备注

此版本的 `sqrt` 函数具有编译器理解的专用化调用约定。 它将加快执行的速度，因为它可防止生成副本和帮助注册表分配。

生成的值被将被推送到堆栈顶部。

## <a name="requirements"></a>要求

**平台：** x86
