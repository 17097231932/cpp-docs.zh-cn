# <a name="default-date-and-time"></a>默认日期和时间

ANSI 3.8.8 转换的日期和时间不可用时应遵循的 `__DATE__` 和 `__TIME__` 的定义

当操作系统不提供转换的时间和日期时，`__DATE__` 和 `__TIME__` 的默认值为 `May 03 1957` 和 `17:00:00`。
