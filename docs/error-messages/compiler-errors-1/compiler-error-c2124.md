# <a name="compiler-error-c2124"></a>编译器错误 C2124

被零除或对零求模

常量表达式的分母为零。 若要解决此错误，请勿将零作为除数。

下面的示例生成 C2124：

```
// C2124.cpp
int main() {
  int i = 1 / 0;   // C2124  do not divide by zero
  int i2 = 12 / 2;   // OK
}
```
