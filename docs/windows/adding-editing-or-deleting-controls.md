# <a name="adding-editing-or-deleting-controls"></a>添加、编辑或删除控件
使用对话框编辑器，可以添加、 调整大小、 编辑和删除控件在对话框中。 你还可以编辑的属性的控件，例如其 ID，或者它是否是最初可见在运行时。  
  
 你可以使用对话框编辑器中的快捷菜单快速将已注册的 ActiveX 控件添加到对话框中，并可以将 ActiveX 控件添加到工具箱中用于快速访问 (有关详细信息，请参阅[自定义工具箱对话框](http://msdn.microsoft.com/en-us/bd07835f-18a8-433e-bccc-7141f65263bb))。
