# <a name="ml-nonfatal-error-a2063"></a>ML 非致命错误 A2063
**可以仅为 2 的幂 ALIGN： 表达式**  
  
 使用指定的表达式[ALIGN](../../assembler/masm/align-masm.md)指令无效。  
  
 **ALIGN**表达式必须是介于 2 和 256 之间，2 的幂，并且必须小于或等于当前段、 结构或联合的对齐方式。
