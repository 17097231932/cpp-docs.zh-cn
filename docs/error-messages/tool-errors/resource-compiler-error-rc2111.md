# <a name="resource-compiler-error-rc2111"></a>资源编译器错误 RC2111
无效的控件类型  
  
 在每个控件语句**对话框**语句必须是以下项之一： 3STATE，AUTO3、 AUTOCHECK、 AUTORADIO、 b 编辑、 复选框、 组合框、 控件、 CTEXT、 DEFPUSHBUTTON、 EDITTEXT、 GROUPBOX、 HEDIT、 图标、 IEDIT、 列表框、 LTEXT，PUSHBOX、 按钮、 单选按钮、 RTEXT、 滚动条、 USERBUTTON。  
  
 请确保这些控制语句拼写正确。
