# ML Fatal Error A1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**不对宏嵌套**  
  
 ，在文件或停止的指令 [ENDM](../../assembler/masm/endm.md) 结束位于宏外部块之前，宏尚未被终止。  
  
 此错误的一个原因是该点的 " 在 [.REPEAT](../../assembler/masm/dot-repeat.md) 或 [.WHILE](../../assembler/masm/dot-while.md)之前。
