# <a name="command-line-warning-d9024"></a>命令行警告 D9024

无法识别的源文件类型 filename 中，对象文件假定

无法识别指定的文件的扩展名。 该文件都假定为对象文件，并传递给链接器。

识别下列扩展名：

- .c （C 源代码文件）

- .cxx （c + + 源文件）

- .cpp （c + + 源文件）

- .obj （对象文件）

- .lib （库文件）

- .def （模块定义文件）

- .exp （链接器将导出链接 /LIB 所创建的文件）
