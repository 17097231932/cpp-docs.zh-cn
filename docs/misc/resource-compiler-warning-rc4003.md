# 资源编译器警告 RC4003
“identifier”宏的实参不足  
  
 使用给定标识符指定的实参的数目小于标识符的宏定义中给定的形参数目。  
  
 在定义中引用形参而未提供对应实参时，将在宏扩展中替换空文本。
