# <a name="expression-evaluator-error-cxx0020"></a>表达式计算器错误 CXX0020

此操作的操作数类型错误

运算符应用于对该运算符具有无效类型的表达式。

例如，使用寄存器地址或使用浮点表达式作为数组的下标是无效的。

此错误与 CAN0020 相同。
