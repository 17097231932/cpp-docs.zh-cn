# 编译器错误 C2406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：在 “context” 中未定义名称  
  
 一个未定义的标识符与 `SIZE`、`LENGTH` 或成员选择 \(.\) 一起使用 运算符。
