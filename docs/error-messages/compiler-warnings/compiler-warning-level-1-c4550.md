# 编译器警告（等级 1）C4550
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

表达式计算为缺少参数列表的函数  
  
 指向函数的取消引用的指针缺少参数列表。  
  
## 示例  
  
```  
// C4550.cpp  
// compile with: /W1  
bool f()  
{  
   return true;  
}  
  
typedef bool (*pf_t)();  
  
int main()  
{  
   pf_t pf = f;  
   if (*pf) {}  // C4550  
   return 0;  
}  
```
