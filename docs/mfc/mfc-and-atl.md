# MFC 和 ATL
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft 基础类 \(MFC\) 通过 Win32 提供面向 C\+\+ 对象的包装，以便实现本机桌面应用程序的快速开发。 活动模板库 \(ATL\) 是一个包装库，它简化了 COM 开发，广泛用于创建 ActiveX 控件。  
  
 可以使用 Visual Studio Community 或更高版本创建 MFC 或 ATL 程序。 Express 版本不支持 MFC 或 ATL。 在 Visual Studio 2015 中，Visual C\+\+ 是一个可选组件，MFC 和 ATL 组件是 Visual C\+\+ 下的可选子组件。 如果首次安装 Visual Studio 时未选择这些组件，则在你第一次尝试创建或打开 MFC 或 ATL 项目时，系统将提示你安装它们。
