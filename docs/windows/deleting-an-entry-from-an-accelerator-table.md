# Deleting an Entry From an Accelerator Table
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 若要从快捷键对应表删除项  
  
1.  通过双击[资源视图](../windows/resource-view-window.md)中的快捷键对应表图标将其打开。  
  
    > [!NOTE]
    >  如果你的项目尚未包含 .rc 文件，请参阅[创建新资源脚本文件](../windows/how-to-create-a-resource-script-file.md)。  
  
2.  选择要删除的项。  （在单击选择多个项的同时，按住 **CTRL** 或 **SHIFT** 键。）  
  
3.  右键单击并从快捷菜单选择**“删除”**（或从**“编辑”**菜单选择**“删除”**）。  
  
 \- 或 \-  
  
-   按删除键。  
  
 有关将资源添加到托管项目的信息，请参阅“.NET Framework 开发员指南”中的[应用程序中的资源](../Topic/Resources%20in%20Desktop%20Apps.md)。  
  
 **要求**  
  
 Win32
