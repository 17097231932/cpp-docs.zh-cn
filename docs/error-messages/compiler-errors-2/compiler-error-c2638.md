# <a name="compiler-error-c2638"></a>编译器错误 C2638

identifier: __based 修饰符在指向成员的指针上非法

`__based`修饰符不能用于指向成员的指针。

下面的示例生成 C2638:

```
// C2638.cpp
void *a;

class C {
public:
   int i;
   int j;
   int func();
};
int __based (a) C::* cpi = &C::i;  // C2638
int (__based (a) C::* cpf)() = &C::func; // c2638
```
