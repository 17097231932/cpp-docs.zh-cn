# 编译器错误 C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的 ProgID“progid”  
  
 使用 `#import` 语句指定了无效的 progid。  检查 Windows 注册表以确保您指定的 progid 有效。
