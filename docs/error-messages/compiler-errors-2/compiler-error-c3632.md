# <a name="compiler-error-c3632"></a>编译器错误 C3632

"event"：非法的构造事件样式

[__event](../../cpp/event.md) 声明在所有构造中无效。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3632。
