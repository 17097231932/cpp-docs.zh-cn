# <a name="compiler-error-c2256"></a>编译器错误 C2256

非法使用友元说明符在 function

析构函数或构造函数不能指定为[友元](../../cpp/friend-cpp.md)。

下面的示例生成 C2256:

```
// C2256.cpp
// compile with: /c
class C {
public:
   friend ~C();   // C2256
   ~C();   // OK
};
```
