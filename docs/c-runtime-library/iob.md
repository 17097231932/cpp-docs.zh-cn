# <a name="_iob"></a>_iob

stdio 控件结构的数组。

## <a name="syntax"></a>语法

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>备注

`IOB_ENTRIES` 在 stdio.h 中定义为 20。
