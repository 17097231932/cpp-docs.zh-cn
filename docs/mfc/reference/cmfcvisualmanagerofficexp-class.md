# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP 类

为 `CMFCVisualManagerOfficeXP` 应用程序提供 MICROSOFT OFFICE XP 外观。

## <a name="syntax"></a>语法

```
class CMFCVisualManagerOfficeXP : public CMFCVisualManager
```

## <a name="members"></a>成员

此类的所有成员都是派生自此类（ [CMFCVisualManager 类](../../mfc/reference/cmfcvisualmanager-class.md)）的祖先的虚函数。

## <a name="remarks"></a>备注

使用类可以 `CMFCVisualManagerOfficeXP` 将应用程序的可视外观更改为类似于 MICROSOFT OFFICE XP。
