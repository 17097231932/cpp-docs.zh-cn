# CriticalSection::~CriticalSection 析构函数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Deinitializes 和销毁当前 CriticalSection 对象。  
  
## 语法  
  
```  
WRL_NOTHROW ~CriticalSection();  
```  
  
## 要求  
 **标头：**corewrappers.h  
  
 **命名空间：**Microsoft::WRL::Wrappers
