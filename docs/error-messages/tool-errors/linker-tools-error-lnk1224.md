# <a name="linker-tools-error-lnk1224"></a>链接器工具错误 LNK1224
无效图像的基址  
  
 指定映像无效的基址。 基址必须是对齐的 64 KB （最后四个十六进制数字必须是零） 和基本映像必须适应 32 位有符号或无符号值。
