# 编译器警告（等级 1）C4143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支持杂注“same\_seg”；使用 \_\_based 分配  
  
 **\#Pragma same\_seg** 不再受支持。 请改用 [\_\_based](../../cpp/based-pointers-cpp.md) 关键字。
