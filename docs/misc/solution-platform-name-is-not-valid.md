# Solution platform name is not valid.
解决方案平台名称不能包含下列字符：  
  
-   井号 \(\#\)  
  
-   百分号 \(%\)  
  
-   “and”符 \(&\)  
  
-   星号 \(\*\)  
  
-   竖线 \(&#124;\)  
  
-   反斜杠 \(\\\)  
  
-   冒号 \(:\)  
  
-   双引号 \("\)  
  
-   小于号 \(\<\)  
  
-   大于号 \(\>\)  
  
-   句点 \(.\)  
  
-   问号 \(?\)  
  
-   正斜杠 \(\/\)  
  
-   前导或尾随空格（“”）  
  
-   为 Windows 或 DOS 保留的名称 （nul、aux、con、com1、lpt1 等）  
  
### 更正此错误  
  
1.  输入不包含上面所列字符的解决方案平台名称。
