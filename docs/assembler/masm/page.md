# <a name="page"></a>PAGE
第一个指令将设置行*长度*和字符*宽度*的程序列表。 如果未提供参数，将生成一个分页符。 第二个指令形式递增的节号，并将页码重置为 1。  
  
## <a name="syntax"></a>语法  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
