# <a name="compiler-warning-level-3-c4723"></a>编译器警告（等级 3）C4723

潜在的被 0 除

计算结果为零，在编译时，为未定义的结果提供除运算中的第二个操作数。

仅在使用时，将发出此警告[/Og](../../build/reference/og-global-optimizations.md)或意味着 /Og 的优化选项。

编译器可能生成的零个操作数。
