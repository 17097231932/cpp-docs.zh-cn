# <a name="compiler-error-c3003"></a>编译器错误 C3003

directive:在指令子句之后不允许使用 OpenMP 指令名称

OpenMP 指令名称不能位于 OpenMP 指令子句之后。

下面的示例生成 C3003：

```
// C3003.c
// compile with: /openmp
int main()
{
   int x, y, z;
   #pragma omp parallel shared(x, y, z) for   // C3003
}
```
