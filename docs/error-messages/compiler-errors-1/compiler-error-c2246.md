# <a name="compiler-error-c2246"></a>编译器错误 C2246
“identifier”：局部定义的类中出现非法静态数据成员  
  
 具有局部范围的类、结构或联合的成员被声明 `static`。  
  
 下面的示例生成 C2246：  
  
```  
// C2246.cpp  
// compile with: /c  
void func( void ) {  
   class A { static int i; };   // C2246  i is local to func  
   static int j;   // OK  
};  
```
