# <a name="ltctgmathgt"></a>&lt;ctgmath&gt;
实际上包括 C++ 标准库标头 \<ccomplex> 和 \<cmath>，用于提供等同于 \<tgmath.h> 的泛型类型算术宏。  
  
## <a name="syntax"></a>语法  
  
```  
#include <ctgmath>  
  
```  
  
## <a name="remarks"></a>备注  
 标准 C 库标头 \<tgmath.h> 的功能通过 \<ccomplex> 和 \<cmath> 中的重载来提供。  
  
 包含该标头还将确保使用标准 C 库标头中的外部链接声明的名称在 `std` 命名空间中声明。
