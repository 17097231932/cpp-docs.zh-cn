# 关于异常的疑难解答：Microsoft.Office.Tools.RangeNotOnWorksheetException
当以编程方式创建视图控件时，如果该控件的范围不在与控件集合对应的工作表之内，则引发 <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException> 异常。  
  
## 相关提示  
 确保指定的范围对应于正确的工作表。  
 当以编程方式创建视图控件时，如果该控件的范围不在与控件集合对应的工作表之内，则引发该异常。
