# <a name="compiler-error-c2418"></a>编译器错误 C2418

无法删除浏览器文件：文件名

编译器无法删除浏览器文件。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复

1. 文件是只读的。

1. 文件正由另一个进程使用。
