# MSBuild 错误 MSB3114
**MSB3114: 未能找到由程序集“\<程序集\>”引用的文件“\<文件\>”。**  
  
 当现有清单中发现了无法解析的引用时会产生此错误。  
  
### 更正此错误  
  
-   检查您的清单中是否存在文件引用冲突。
