# 程序和链接 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

程序包括链接在一起的一个或多个翻译单元。  执行（从概念上说）在包含函数 **main** 的翻译单元中开始。  （有关翻译单元的详细信息，请参阅《预处理器参考》[](../preprocessor/phases-of-translation.md "Phases of Translation")中的*翻译阶段*。） 有关 **main** 函数的详细信息，请参阅[程序启动：main 函数](../cpp/main-program-startup.md)。）
