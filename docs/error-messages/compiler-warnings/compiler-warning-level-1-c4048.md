# <a name="compiler-warning-level-1-c4048"></a>编译器警告 （等级 1） C4048
不同的声明数组下标: identifier1 和 identifier2  
  
 表达式涉及到不同的大小的数组的指针。 指针用于无需转换。  
  
 如果你显式强制转换为相同或等效类型数组，则可能会修复此警告。
