# 1.5 Normative References
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

-   ISO\/IEC 9899:1999， *信息技术 \- 编程语言 \- C。* 此 OpenMP API 规范引用 ISO\/IEC 9899:1999 作为 C99。  
  
-   ISO\/IEC 9899:1990， *信息技术 \- 编程语言 \- C。* 此 OpenMP API 规范引用 ISO\/IEC 9899:1990 作为 C90。  
  
-   ISO\/IEC 14882:1998， *信息技术 \- 编程语言 \- C\+\+*。  此 OpenMP API 规范引用 ISO\/IEC 14882:1998 为 C\+\+。  
  
 如果此 OpenMP API 规范引用 C，请引用了实现支持的这种基本语言。
