# <a name="fatal-error-c1603"></a>错误 C1603

内联程序集分支目标超出了 "number" 字节范围

JCXZ 或 JECXZ 指令与其指定目标标签之间的计算距离大于128字节。 更新代码，以便标签更接近指令。
