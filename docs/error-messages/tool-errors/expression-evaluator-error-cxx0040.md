# 表达式计算器错误 CXX0040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

函数需要隐式转换  
  
 C 表达式计算器不支持涉及到构造函数调用的隐式转换。  
  
 该错误与 CAN0040 相同。
