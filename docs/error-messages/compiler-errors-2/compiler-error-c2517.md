# 编译器错误 C2517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”:“::”的右边未定义  
  
 在范围解析运算符 \(`::`\) 右边的标识符必须是左边的类、结构或联合的已定义成员。  如果没有命名类、结构或联合，则必须用全局范围声明右边的标识符。
