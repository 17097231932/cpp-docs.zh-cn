# <a name="no_search_namespace-import-attribute"></a>no_search_namespace 导入属性

**C++ 专用**

具有与 [no_namespace](../preprocessor/no-namespace.md) 特性相同的功能，但在将 `#import` 指令与 [auto_search](../preprocessor/auto-search.md) 特性结合使用的类型库上使用。

## <a name="syntax"></a>语法

> **#import** *类型-库* **no_search_namespace**

**结束 C++ 专用**
