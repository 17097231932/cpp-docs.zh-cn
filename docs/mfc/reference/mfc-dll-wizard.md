# <a name="mfc-dll-wizard"></a>MFC DLL 向导

当您使用 MFC DLL 向导创建 MFC DLL 项目时，您将获得一个带有内置功能的工作入门应用程序，编译后该功能将实现 [DLL](../../build/dlls-in-visual-cpp.md)的基本功能。 MFC starter 程序包含 c + + 源 ( .cpp) 文件、资源 ( .rc) 文件和项目 () 文件。 这些初学者文件中生成的代码基于 MFC。 有关更多详细信息，请参阅在 Visual Studio 中为你的项目生成的 Readme.txt 中的文件详细信息，以及 [MFC DLL 向导生成的类和函数](../../mfc/reference/classes-and-functions-generated-by-the-mfc-dll-wizard.md)

## <a name="overview"></a>概述

此向导页描述你正在创建的 [MFC DLL 项目的当前应用程序设置](../../mfc/reference/application-settings-mfc-dll-wizard.md) 。 默认情况下，该项目将创建为 (MFC 共享) 项目的常规 MFC DLL，而无其他设置。

若要更改这些默认值，请单击向导左列中的 " **应用程序设置** "，并在 MFC DLL 向导的相应页中进行更改。

创建 MFC DLL 项目后，可以使用 Visual C++ [代码向导](../../ide/adding-functionality-with-code-wizards-cpp.md)将对象或控件添加到项目。

您可以对基本 MFC DLL 项目执行以下任务和增强类型：

- [从 DLL 导出](../../build/exporting-from-a-dll.md)

- [将可执行文件链接到 DLL](../../build/linking-an-executable-to-a-dll.md)

- [初始化 DLL](../../build/run-time-library-behavior.md#initializing-a-dll)
