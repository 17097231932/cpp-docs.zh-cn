# 关于异常的疑难解答：System.IdentityModel.Selectors.StsCommunicationException
如果在与安全标记服务通信时出现问题，将引发 <xref:System.IdentityModel.Selectors.StsCommunicationException> 异常。  
  
 该问题可由下列条件引起：  
  
-   网络故障  
  
-   无法到达 MEX 终结点  
  
-   安全标记服务没有有效的终结点标识
