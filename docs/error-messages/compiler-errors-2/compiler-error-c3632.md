# <a name="compiler-error-c3632"></a>编译器错误 C3632
'event': 非法样式事件 for 构造  
  
 [__event](../../cpp/event.md)声明并非在所有构造函数中有效。  
  
 C3632 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
