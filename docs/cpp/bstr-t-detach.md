# <a name="_bstr_tdetach"></a>_bstr_t::Detach

**Microsoft 专用**

返回 `BSTR` 包装的 `_bstr_t` 并从 `BSTR` 中分离 `_bstr_t`。

## <a name="syntax"></a>语法

```
BSTR Detach( ) throw;
```

## <a name="return-value"></a>返回值

由 `BSTR` 包装的 `_bstr_t`。

## <a name="example"></a>示例

有关使用**分离**的示例，请参阅[_Bstr_t：： Assign](../cpp/bstr-t-assign.md) 。
