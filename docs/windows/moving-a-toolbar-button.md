# <a name="moving-a-toolbar-button"></a>移动工具栏按钮
### <a name="to-move-a-toolbar-button"></a>若要移动工具栏按钮  
  
1.  在**工具栏视图**窗格中，拖动想要移动到其新位置，在工具栏上的按钮。
