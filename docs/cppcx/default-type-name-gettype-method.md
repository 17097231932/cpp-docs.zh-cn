# <a name="defaulttype_namegettype-method"></a>default::(type_name)::GetType 方法

返回表示当前类型的 Platform::Type^。

## <a name="syntax"></a>语法

```cpp
Platform::Type^ GetType();
```

### <a name="return-value"></a>返回值

表示当前对象的 [Platform::Type](../cppcx/platform-type-class.md)^ 对象。

### <a name="requirements"></a>要求

**支持的最低客户端：** Windows 8

**支持的最低服务器：** Windows Server 2012

**命名空间：** 默认

**标头：** vccorlib.h
