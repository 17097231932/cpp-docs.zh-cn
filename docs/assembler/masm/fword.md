# <a name="fword"></a>FWORD
分配和 （可选） 为每个初始化 6 字节存储`initializer`。  
  
## <a name="syntax"></a>语法  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>备注  
 此外可以用作类型说明符类型是合法的任何位置。
