# 错误 C1053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“\<identifier\>”：函数太大  
  
 该函数太大，无法编译。  
  
### 使用以下可能的解决方案进行修复  
  
1.  请尝试编译但不优化。  
  
2.  将函数拆分为较小的函数。  
  
3.  减少对内联函数的调用。
