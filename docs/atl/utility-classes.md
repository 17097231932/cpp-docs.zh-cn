# <a name="utility-classes"></a>实用程序类

提供以下与 MFC 无关的实用工具类：

- [CImage](../atl-mfc-shared/reference/cimage-class.md) 提供增强的位图支持，其中包括将图像加载并保存到 JPEG、GIF、BMP 和可移植网络图形 (PNG) 格式的功能。

- [CPoint](../atl-mfc-shared/reference/cpoint-class.md) 提供用于存储坐标 (x，y) 对的实现。

- [CRect](../atl-mfc-shared/reference/crect-class.md) 提供一个实现，用于存储矩形区域的坐标。

- [CSize](../atl-mfc-shared/reference/csize-class.md) 提供用于存储距离、相对位置或成对值的实现。

- [CString](../atl-mfc-shared/reference/cstringt-class.md) 提供用于存储字符串的实现。

- [CAdapt](../atl/reference/cadapt-class.md) 用于包装重新定义运算符地址的类的简单模板。

- [_U_RECT](../atl/reference/u-rect-class.md) 允许 `RECT` 将指针或引用传递给在指针方面实现的函数的自变量适配器类。
