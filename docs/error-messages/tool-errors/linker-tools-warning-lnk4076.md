# <a name="linker-tools-warning-lnk4076"></a>链接器工具警告 LNK4076

增量状态文件 "filename" 无效;非增量链接

LINK 无法写入增量状态（. .ilk）文件。 `filename` 已损坏或不是增量链接数据库。 删除文件并重新链接。
