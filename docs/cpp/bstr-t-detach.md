# `_bstr_t::Detach`

**Microsoft 专用**

返回 `BSTR` 包装的 `_bstr_t` 并从 `BSTR` 中分离 `_bstr_t`。

## <a name="syntax"></a>语法

```cpp
BSTR Detach( ) throw;
```

## <a name="return-value"></a>返回值

返回 `BSTR` 封装的 `_bstr_t` 。

## <a name="example"></a>示例

[`_bstr_t::Assign`](../cpp/bstr-t-assign.md)有关使用的示例，请参阅 **`Detach`** 。
