# <a name="ymmword"></a>YMMWORD

用于具有 Intel 高级矢量扩展的256位多媒体操作数 (AVX) 说明。

## <a name="syntax"></a>语法

> **YMMWORD**

## <a name="remarks"></a>备注

**YMMWORD** 用于表示与 `__m256` AVX 内部函数相同的类型。
