# NMAKE 错误 U1076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

名称太长  
  
 字符串已超过下列限制之一：  
  
-   宏名最长为 1024 个字符。  
  
-   目标路径名最长为 256 个字符。  
  
-   命令最长为 2048 个字符。
