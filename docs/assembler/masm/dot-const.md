# <a name="const-32-bit-masm"></a>.CONST （32位 MASM）

与一起使用时[。模型](../../assembler/masm/dot-model.md)，则启动常量数据段（带有段名**CONST**）。

## <a name="syntax"></a>语法

> **.CONST**

## <a name="remarks"></a>备注

此段具有只读特性。
