# 表达式计算器错误 CXX0021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

结构或联合用作标量  
  
 表达式中使用结构或联合，但未指定任何元素。  
  
 当操作结构或联合变量时，变量名可能单独出现，不带域限定符。  如果表达式中使用结构或联合，则必须由所需的特定元素限定。  
  
 指定要在表达式中使用其值的元素。  
  
 该错误与 CAN0021 相同。
