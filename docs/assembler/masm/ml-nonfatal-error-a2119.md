# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**必须指定语言类型**  
  
 未给出程序定义或原型一个语言类型。  
  
 默认语言，如果未指定类型，在每个程序定义或原型必须声明语言类型。  使用 [.MODEL](../../assembler/masm/dot-model.md) 指令， **选项 LANG**或语言命令行选项 **\/Gc** 或 **\/Gd**，默认语言类型设置。
