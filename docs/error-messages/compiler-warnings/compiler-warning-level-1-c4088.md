# <a name="compiler-warning-level-1-c4088"></a>编译器警告 (等级 1) C4088

"function"：实参 "number"、形参 "number" 中的指针不匹配

对应的形参和实参具有不同的间接级别。 未更改即可传递实际参数。 被调用的函数将其值解释为指针。
