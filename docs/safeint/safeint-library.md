# <a name="safeint-library"></a>SafeInt 库

SafeInt**** 是可以与 MSVC、GCC 或 Clang 结合使用的可移植库，有助于防止在应用程序执行数学运算时可能会出现的整数溢出。 此库的最新版本位于 [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt) 。

|[SafeInt 函数](safeint-functions.md)|不创建 SafeInt**** 对象即可使用的函数。|
|[SafeIntException 类](safeintexception-class.md)|与 SafeInt**** 类相关的异常类。|
