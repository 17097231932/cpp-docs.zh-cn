# <a name="operator-"></a>运算符 .

第一个运算符返回*表达式*以及*字段*在其结构或联合中的偏移量。 第二个运算符通过在其结构或联合中*注册*和*字段*的偏移量，在所指向的位置返回值。

## <a name="syntax"></a>语法

> *表达式* __。__ *字段*⟦ __。__ *字段*.。。⟧
>
> **\[** _注册_ **]** __。__ *字段*⟦ __。__ *字段*.。。⟧
