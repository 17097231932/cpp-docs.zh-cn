# ML Nonfatal Error A2078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**命令不允许更间接寻址**  
  
 一个条件分支或循环不能采用内存操作数。  必须为其提供相对地址或标签。
