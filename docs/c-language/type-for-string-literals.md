# 字符串文本的类型
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

字符串具有 `char`（即 **char\[ \]**）的类型数组。（宽字符字符串具有 `wchar_t`（即**wchar\_t\[ \]**）的类型数组。）这意味着字符串是带有 `char` 类型的元素的数组。  数组中的元素数等于字符串中的字符数加上结尾的 null 字符。
