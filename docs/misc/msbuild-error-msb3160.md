# MSBuild 错误 MSB3160
**MSB3160: 文件“\<file\>”中的 Xml 验证警告: \<error\>**  
  
 此警告在产品文件或 package.xml 文件不符合 XML 架构时产生。  此警告的后半部分是具体的警告消息。  
  
### 更正此错误  
  
-   更新 XML 文件以解决该验证错误。
