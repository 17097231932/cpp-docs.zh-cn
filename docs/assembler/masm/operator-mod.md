# operator MOD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回余数 \(取模\) 之间的整数值，当部件 *expression1* 由 *expression2 时*。  
  
## 语法  
  
```  
  
expression1 MOD expression2  
```
