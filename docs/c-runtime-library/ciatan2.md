# <a name="ciatan2"></a>_CIatan2

计算 *x* / *y* 的反正切值，其中 *x* 和 *y* 是堆栈顶部的值。

## <a name="syntax"></a>语法

```
void __cdecl _CIatan2();
```

## <a name="remarks"></a>备注

此版本的 `atan2` 函数具有编译器理解的专用化调用约定。 它将加快执行的速度，因为它可防止生成副本和帮助注册表分配。

生成的值被将被推送到堆栈顶部。

## <a name="requirements"></a>要求

**平台：** x86
