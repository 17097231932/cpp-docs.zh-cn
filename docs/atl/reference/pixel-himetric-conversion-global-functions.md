# Pixel/HIMETRIC Conversion Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

> [!IMPORTANT]
>  下表中列出的函数不能在 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]执行的应用程序。  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../Topic/AtlHiMetricToPixel.md)|转换HIMETRIC单元\(每个单位是0.01毫米\)与像素。|  
|[AtlPixelToHiMetric](../Topic/AtlPixelToHiMetric.md)|为HIMETRIC单位的转换像素\(每个单位是0.01毫米）。|
