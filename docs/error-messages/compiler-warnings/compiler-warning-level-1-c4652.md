# <a name="compiler-warning-level-1-c4652"></a>编译器警告（等级 1）C4652

编译器选项 "option" 与预编译头不一致;当前命令行选项将重写预编译头中定义的

给定的命令行选项与创建 ( .pch) 预编译标头时给定的命令行选项不相同。 使用了当前命令行中指定的选项。

可以通过使用给定的命令行选项重新生成预编译标头来避免此警告。
