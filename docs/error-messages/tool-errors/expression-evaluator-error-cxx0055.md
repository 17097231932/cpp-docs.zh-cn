# 表达式计算器错误 CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在上下文运算符中符号不明确  
  
 上下文运算符 \(**{}**\) 中的一个符号引用了程序中的多个符号。  
  
 范围解析运算符 \(`::`\) 可能能够解析多义性。  
  
 该错误与 CAN0055 相同。
