# <a name="ctable-class"></a>CTable 类
提供一种方法直接访问简单行集合 （一个不带任何参数）。  
  
## <a name="syntax"></a>语法  
  
```  
template <   
   class TAccessor = CNoAccessor,    
   template <typename T> class TRowset = CRowset    
>  
class CTable :    
   public CAccessorRowset <   
      TAccessor,    
      TRowset    
   >  
```  
  
#### <a name="parameters"></a>参数  
 `TAccessor`  
 一个访问器类。  
  
 `TRowset`  
 行集类。
