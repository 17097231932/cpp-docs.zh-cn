# Server Registration Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

这些功能提供用于注册的支持，并在对象的注销的服务器对象映射。  
  
> [!IMPORTANT]
>  下表中列出的函数不能在 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]执行的应用程序。  
  
|||  
|-|-|  
|[AtlComModuleRegisterServer](../Topic/AtlComModuleRegisterServer.md)|此功能称为注册在对象映射的每个对象。|  
|[AtlComModuleUnregisterServer](../Topic/AtlComModuleUnregisterServer.md)|此函数调用注销在对象映射的每个对象。|  
|[AtlComModuleRegisterClassObjects](../Topic/AtlComModuleRegisterClassObjects.md)|此功能来注册选件类对象。|  
|[AtlComModuleRevokeClassObjects](../Topic/AtlComModuleRevokeClassObjects.md)|此功能称为取消从COM模块的选件类对象。|  
|[AtlComModuleGetClassObject](../Topic/AtlComModuleGetClassObject.md)|此功能称为获取选件类对象。|
