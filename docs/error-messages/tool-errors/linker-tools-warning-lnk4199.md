# <a name="linker-tools-warning-lnk4199"></a>链接器工具警告 LNK4199
/DELAYLOAD:dllname 忽略;没有找到 dll 名称从的导入  
  
 链接器忽略`dllname`因为它不需要任何函数，`dllname`导出。
