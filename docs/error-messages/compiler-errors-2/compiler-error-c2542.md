# 编译器错误 C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 类对象没有用于初始化的构造函数  
  
 没有所带参数列表与初始化匹配的构造函数。  
  
 初始化时参数有误会导致此错误。
