# <a name="evening-the-spacing-between-controls"></a>使控件之间的间距相等
对话框编辑器使您能够选择最外层控件之间均匀地空间控件。  
  
### <a name="to-even-the-spacing-between-controls"></a>若要控件之间的间距相等  
  
1.  选择你想要重新排列这些的控件。  
  
2.  从**格式**菜单上，选择**均匀隔开**，然后选择以下间距对齐方式之一：  
  
    -   **跨**： 空间最左边和右边所选控件之间均匀地的控件。  
  
    -   **向下**： 上面的控件与所选的最底端控件之间均匀间隔控件。
