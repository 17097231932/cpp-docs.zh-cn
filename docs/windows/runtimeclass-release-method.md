# <a name="runtimeclassrelease-method"></a>RuntimeClass::Release 方法
对当前 RuntimeClass 对象执行 COM 版本操作。  
  
## <a name="syntax"></a>语法  
  
```  
STDMETHOD_(  
   ULONG,  
   Release  
)();  
```  
  
## <a name="return-value"></a>返回值  
 如果成功，则为 S_OK；否则为指示错误的 HRESULT。  
  
## <a name="remarks"></a>备注  
 如果引用计数变为零，则将删除 RuntimeClass 对象。

 **命名空间：** Microsoft::WRL
