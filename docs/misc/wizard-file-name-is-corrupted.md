# Wizard file &lt;name&gt; is corrupted.
向导的 .vsz 文件格式不正确。  该错误发生在 .vsz 文件损坏或该文件来自 Visual Studio 以外的产品时。  
  
### 更正此错误  
  
1.  验证该向导确实为 Visual Studio 向导。  
  
     \- 或 \-  
  
     重新安装 Visual Studio 以替换损坏的文件。
