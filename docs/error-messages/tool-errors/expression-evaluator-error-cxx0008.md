# <a name="expression-evaluator-error-cxx0008"></a>表达式计算器错误 CXX0008
**"在字符串的末尾缺少**  
  
 缺少应有末尾的字符串文本的双引号。 重新键入封闭文本字符串用双引号括起来的表达式。  
  
 此错误是与 CAN0008 相同。
