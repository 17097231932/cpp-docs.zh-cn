# <a name="compiler-warning-levels-1-and-4-c4223"></a>编译器警告（等级 1 和等级 4）C4223
使用的非标准扩展： 非左值数组转换为指针  
  
 在标准 C 中，无法将非左值数组转换为指针。 与默认 Microsoft 扩展 ([/Ze](../../build/reference/za-ze-disable-language-extensions.md))，你可以。
