# VectorIterator::operator* 运算符
检索当前 VectorIterator 指定的元素的地址。  
  
## 语法  
  
```  
reference operator*() const;  
```  
  
## 返回值  
 当前 VectorIterator 指定的元素。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
