# <a name="comptrswap-method"></a>ComPtr::Swap 方法
交换由当前 ComPtr 托管的接口与由指定 ComPtr 托管的接口。  
  
## <a name="syntax"></a>语法  
  
```  
void Swap(  
   _Inout_ ComPtr&& r  
);  
  
void Swap(  
   _Inout_ ComPtr& r  
);  
```  
  
#### <a name="parameters"></a>参数  
 `r`  
 ComPtr。  
  
## <a name="requirements"></a>要求  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
