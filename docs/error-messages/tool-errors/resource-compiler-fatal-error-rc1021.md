# <a name="resource-compiler-fatal-error-rc1021"></a>资源编译器错误 RC1021

无效的预处理器命令“string”

数字符号后 () 的字符 **#** 未构成有效的预处理器指令。

不能将数字符号用作标识符中的第一个字符。
