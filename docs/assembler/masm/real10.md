# <a name="real10"></a>REAL10

分配每个*初始值设定项*并根据需要初始化10字节浮点数。

## <a name="syntax"></a>语法

> *名称* **REAL10** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
