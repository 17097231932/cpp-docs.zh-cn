# 表达式计算器错误 CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

寄存器不可用  
  
 表达式指定了无法使用的寄存器。  尝试访问在运行的计算机上不存在的寄存器时会导致该错误。  
  
 该错误与 CAN0060 相同。
