# 引用的对象“&lt;objectname&gt;”的值为“Nothing”
尚未设置引用的对象，或已将其值设置为 `Nothing`。  
  
 **错误 ID：**BC30705  
  
### 更正此错误  
  
-   验证对象名称的拼写是否正确，并验证是否设置了该对象。
