# <a name="ml-nonfatal-error-a2037"></a>ML 非致命错误 A2037

**结构定义内不允许使用语句**

结构定义包含无效的语句。

结构不能包含指令、标签、过程和控制流指令[。启动](../../assembler/masm/dot-startup.md)或[。退出](../../assembler/masm/dot-exit.md)。
