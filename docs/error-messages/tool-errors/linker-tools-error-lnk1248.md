# <a name="linker-tools-error-lnk1248"></a>链接器工具错误 LNK1248

> 图像大小 (*大小*) 超过了允许的最大大小 (80000000)

链接器确定输出文件的大小将超过 32 位程序映像的可能的最大大小。 你可能想要将程序处理成多个 Dll。
