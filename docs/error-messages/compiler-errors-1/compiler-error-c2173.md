# 编译器错误 C2173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”：实参不是指针：参数 number1，参数列表 number2  
  
 传递给参数列表 `number2` 的参数 `number1` 不是指针。 该函数需要一个指针。
