# <a name="compiler-error-c2583"></a>编译器错误 C2583

identifier: this 指针是非法的构造函数/析构函数的 const/volatile

声明构造函数或析构函数`const`或`volatile`。 这是不允许的。

下面的示例生成 C2583:

```
// C2583.cpp
// compile with: /c
class A {
public:
   int i;
   A() const;   // C2583

   // try the following line instead
   // A();
};
```
