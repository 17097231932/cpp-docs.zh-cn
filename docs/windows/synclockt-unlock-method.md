# <a name="synclocktunlock-method"></a>SyncLockT::Unlock 方法
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
void Unlock();  
```  
  
## <a name="remarks"></a>备注  
 释放由当前 SyncLockT 对象占用的资源的控制，如果有的话。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers::Details
