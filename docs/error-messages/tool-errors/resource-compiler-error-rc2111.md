# <a name="resource-compiler-error-rc2111"></a>资源编译器错误 RC2111

控件类型无效

**对话** 语句中的每个控制语句必须是下列其中一项：3STATE、AUTO3、AUTOCHECK、AUTORADIO、BEDIT、CHECKBOX、COMBOBOX、CONTROL、CTEXT、DEFPUSHBUTTON、EDITTEXT、分组框、HEDIT、ICON、IEDIT、LISTBOX、LTEXT、PUSHBOX、按钮、单选按钮、RTEXT、滚动条、USERBUTTON。

请确保这些控制语句拼写正确。
