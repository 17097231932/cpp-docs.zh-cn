# <a name="compiler-error-c2877"></a>编译器错误 C2877

symbol 不能从 class 访问

派生自的基类的所有成员必须都是可在派生类中访问。

下面的示例生成 C2877:

```
// C2877.cpp
// compile with: /c
class A {
private:
   int a;
};

class B : public A {
   using A::a;   // C2877
};
```
