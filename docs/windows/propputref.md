# <a name="propputref"></a>propputref
指定使用的引用，而不是值的属性设置函数。  
  
## <a name="syntax"></a>语法  
  
```  
  
[propputref]  
  
```  
  
## <a name="remarks"></a>备注  
 **Propputref** c + + 属性具有相同的功能[propputref](http://msdn.microsoft.com/library/windows/desktop/aa367147) MIDL 特性。
