# <a name="project-build-error-prj0015"></a>项目生成错误 PRJ0015

从您的系统缺少 NULL 设备。 我们将无法启动生成。

系统资源不足可导致此错误。 关闭一些应用程序或重新启动。

如果您的计算机上没有足够的特权来 NULL 设备也可能导致该问题。
