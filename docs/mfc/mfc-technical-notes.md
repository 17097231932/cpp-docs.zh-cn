# MFC 技术说明
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

一种声明适用于程序员编写的文档由程序员。  
  
 每种声明描述超出 MFC 文档的范围之外的问题或功能。  提供技术声明反映应用程序信息从用户的，以及开发人员所需用户需要高级 MFC 专用的信息。
