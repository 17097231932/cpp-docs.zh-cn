# <a name="bits-per-character"></a>每字符的位数
**ANSI 2.2.4.2.1** 字符中的位数  
  
 字符中的位数由清单常量 CHAR_BIT 表示。 LIMITS.H 文件将 CHAR_BIT 定义为 8。
