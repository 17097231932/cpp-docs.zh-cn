# <a name="compiler-error-c3006"></a>编译器错误 C3006

“clause”：OpenMP“directive”指令上的子句缺少应有的参数

OpenMP 指令没有应有的参数。

下面的示例生成 C3006：

```
// C3006.c
// compile with: /openmp
int main()
{
   #pragma omp parallel shared   // C3006
   // Try the following line instead:
   // #pragma omp parallel shared(x) {}

}
```
