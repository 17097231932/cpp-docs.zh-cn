# <a name="compiler-error-c2529"></a>编译器错误 C2529
name： 引用的引用是非法的  
  
 ： 使用指针的语法，并声明对指针的引用，则可能修复此错误。  
  
 下面的示例生成 C2529:  
  
```  
// C2529.cpp  
// compile with: /c  
int i;  
int &ri = i;  
int &(&rri) = ri;   // C2529  
```
