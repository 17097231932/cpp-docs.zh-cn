# <a name="setting-the-dialog-boxs-background-color"></a>设置对话框的背景色

可以通过处理对话框窗口的 WM_CTLCOLOR 消息设置对话框的背景色。 您设置的颜色仅用于指定的对话框。

请参阅[codexpert 博客](http://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/)有关的示例。
