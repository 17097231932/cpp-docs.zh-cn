# Inserting a Composite Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Add Class** 对话框允许插入ATL对象到项目中。  通过右击"解决方案资源管理器中的项目名称访问此对话框，指向 **Add**，然后单击的 **Add Class**。  
  
 在 **Add Class** 对话框中，选择 **ATL控件**。  这将启动 [ATL控件向导](../atl/reference/atl-control-wizard.md)。  若要创建复合控件，请选择 **选项** 选项，然后单击 **Composite control** 复选框。  
  
 默认HTML页用于查看控件被创建。
