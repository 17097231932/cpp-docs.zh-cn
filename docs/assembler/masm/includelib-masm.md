# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

通知链接器应与 *libraryname*链接当前模块。  
  
## 语法  
  
```  
  
INCLUDELIB libraryname  
```  
  
## 备注  
 ，如果它包含一个反斜杠，分号，大于号，小于符号、单引号或双引号，在尖括号必须将 *libraryname* 。
