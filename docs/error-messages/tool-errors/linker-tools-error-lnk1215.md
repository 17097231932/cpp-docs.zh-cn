# <a name="linker-tools-error-lnk1215"></a>链接器工具错误 LNK1215
元数据操作失败 (HRESULT): 错误  
  
 链接器错误时，从收到的.NET 运行时尝试的操作通过.NET 运行时的元数据更新。  
  
 `HRESULT` 是从.NET 运行时方法的 HRESULT。 `error` 是。NET 提供的文本。  
  
 你可能有不匹配的链接器并使用.NET 运行时;重新安装 Visual c + +。
