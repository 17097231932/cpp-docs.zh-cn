# UnorderedMap::Clear 方法
从当前 UnorderedMap 对象中移除所有键值对。  
  
## 语法  
  
```cpp  
  
virtual void Clear();   
```  
  
## 要求  
 [Platform::Collections::UnorderedMap 类](../cppcx/platform-collections-unorderedmap-class.md)**标头：**collection.h  
  
 **命名空间：**Platform::Collections
