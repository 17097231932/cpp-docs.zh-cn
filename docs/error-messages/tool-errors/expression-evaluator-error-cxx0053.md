# <a name="expression-evaluator-error-cxx0053"></a>表达式计算器错误 CXX0053

绑定断点时的 nonfunction 符号匹配

用作断点的符号不是函数。 将数据成员指定为断点会导致此错误。

此错误与 CAN0053 相同。
