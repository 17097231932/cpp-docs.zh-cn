# <a name="fword"></a>FWORD

为每个*初始值设定*分配6个字节的存储，并根据需要对其进行初始化。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **FWORD** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧

## <a name="remarks"></a>备注

还可在类型合法的任何位置用作类型说明符。
