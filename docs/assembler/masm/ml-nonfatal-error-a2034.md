# <a name="ml-nonfatal-error-a2034"></a>ML 非致命错误 A2034

**必须是段块中**

段块之外找到以下项之一：

- 指令

- 标签定义

- 一个[THIS](../../assembler/masm/operator-this.md)运算符

- 一个[ $ ](../../assembler/masm/dollar.md)运算符

- 过程定义

- [对齐](../../assembler/masm/align-masm.md)指令

- [组织](../../assembler/masm/org.md)指令
