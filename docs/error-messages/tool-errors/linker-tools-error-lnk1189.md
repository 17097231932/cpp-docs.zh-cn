# 链接器工具错误 LNK1189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

LIBTOOMANYMEMBERS::已超出数字对象的库限制  
  
 已超出库中 65535 个对象或成员的限制。  
  
 若要解决此问题，请将你的库分为两个或多个使用更少对象或成员的库。
