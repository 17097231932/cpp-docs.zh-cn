# 编译器错误 C2224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”的左侧必须有结构\/联合类型  
  
 句点 \(.\) 左边的操作数不是类、结构或联合。  
  
 导致此错误的原因可能是由于左边的操作数是未定义的变量（因此为 `int` 类型）。
