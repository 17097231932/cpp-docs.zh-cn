# <a name="compiler-error-c2646"></a>编译器错误 C2646
全局或命名空间范围的匿名结构或联合必须声明为静态  
  
 匿名结构或联合具有全局或命名空间范围，但未被声明为 `static`。  
  
 下面的示例生成 C2646，并演示如何修复此错误：  
  
```  
// C2646.cpp  
// compile with: /c  
union { int i; };   // C2646 not static  
  
// OK  
static union { int j; };  
union U { int i; };  
```
