# Syntax error in pattern.
当搜索字符串包括两个或多个正则表达式而至少其中一个正则表达式包含语法错误时，通常会发生此错误。  
  
### 更正此错误  
  
1.  再次输入该字符串。  有关语法的更多信息，请参见[正则表达式](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)。
