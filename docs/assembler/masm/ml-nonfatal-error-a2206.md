# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**表达式的缺少运算符**  
  
 ，因为它缺少运算符，表达式不能进行计算。  此错误消息也有副作用一之前编程错误。  
  
 以下行将生成此错误:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
