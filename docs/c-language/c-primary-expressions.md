## 语法  
 *primary\-expression*:  
 *identifier*  
  
 *constant*  
  
 *string\-literal*  
  
 **\(**  *expression*  **\)**  
  
 *expression*:  
 *assignment\-expression*  
  
 *expression*  **,**  *assignment\-expression*
