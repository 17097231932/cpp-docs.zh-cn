# <a name="wrl-reference"></a>WRL 引用

本部分包含有关 Windows 运行时 c + + 模板库 (WRL) 的参考信息。
  
> [!NOTE]
> Windows 运行时 c + + 模板库定义支持的 Windows 运行时 c + + 模板库基础结构，不适合直接在代码中使用的功能。 在本文档中记录了此类功能。
  
## <a name="in-this-section"></a>本节内容

[Microsoft::WRL Namespace](../windows/microsoft-wrl-namespace.md)<br/>
定义构成了 Windows 运行时 c + + 模板库的基础类型。
  
[Microsoft::WRL::Wrappers 命名空间](../windows/microsoft-wrl-wrappers-namespace.md)<br/>
定义简化对象、字符串和句柄的生存期管理的“获取资源即初始化”(RAII) 包装器类型。
  
[Microsoft::WRL::Wrappers::HandleTraits 命名空间](../windows/microsoft-wrl-wrappers-handletraits-namespace.md)<br/>
介绍了常见的基于句柄的资源类型的特征。
  
[Windows::Foundation 命名空间](../windows/windows-foundation-namespace.md)<br/>
启用基本 Windows 运行时功能，例如对象和工厂创建。

引入了 Windows 运行时 c + + 模板库，一个基于 COM 的模板库，提供低级别的方式来创作和使用 Windows 运行时组件。
