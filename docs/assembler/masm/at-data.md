# <a name="data"></a>\@数据

默认数据组的名称。 对于所有模型（ **平面** 除外），计算结果为 DGROUP。 计算结果为 **平面内存模型** 下的 **平面** (文本宏) 。

## <a name="syntax"></a>语法

> **\@数据**
