# <a name="fatal-error-c1051"></a>错误 C1051

程序数据库文件，pdbfile 具有过时的格式，将其删除并重新编译

编译器无法更新程序数据库文件的较旧的版本号。 删除该文件并重新编译您的程序与 **/Zi**或 **/ZI**。 有关详细信息，请参阅[/Z7、 /Zi、 /ZI （调试信息格式）](../../build/reference/z7-zi-zi-debug-information-format.md)
