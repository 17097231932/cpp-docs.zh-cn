# <a name="new-custom-resource-dialog-box"></a>“新建自定义资源”对话框
可以创建新的自定义资源。  
  
### <a name="resource-type"></a>资源类型  
 提供一个文本框，以输入自定义资源类型的名称。 Visual c + + 在退出时自动将大写名称**新的自定义资源**对话框。  
  
## <a name="remarks"></a>备注  
 未指定资源名称或 ID 由预留[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]。 保留的名称为 DESIGNINFO、 HWB 和 TEXTINCLUDE，保留的 ID 为 255。
