# <a name="expression-evaluator-error-cxx0005"></a>表达式计算器错误 CXX0005
不支持的运算符  
  
 在表达式中指定了不受支持的 C 运算符。 编写使用支持的 C 运算符的等效表达式。  
  
 此错误是与 CAN0005 相同。
