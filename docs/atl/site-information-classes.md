# <a name="site-information-classes"></a>站点信息类

以下类允许对象与其站点通信：

- [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) 检索和设置指向对象的站点的指针。 用于不是控件的对象。

- [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 检索和设置指向对象的站点的指针。 用于控件。
