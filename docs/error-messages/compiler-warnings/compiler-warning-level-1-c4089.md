# 编译器警告（等级 1）C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 实参“number”、形参“number”类型不同  
  
 相应的形参与实参的类型不同。  实参传递时未经更改。  函数将实参转换为函数定义中指定的类型。
