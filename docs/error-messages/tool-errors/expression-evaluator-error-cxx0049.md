# <a name="expression-evaluator-error-cxx0049"></a>表达式计算器错误 CXX0049
已过时的 OMF-请重新链接程序  
  
 程序使用了旧的 OMF （对象模块格式）。  
  
 重新链接程序使用的当前链接器版本。  
  
 此错误是与 CAN0049 相同。
