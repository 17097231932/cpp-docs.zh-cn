# <a name="expression-evaluator-error-cxx0055"></a>表达式计算器错误 CXX0055
上下文运算符中的不明确符号  
  
 上下文运算符中的符号 (**{}**) 引用的程序中的多个符号。  
  
 范围解析运算符 (`::`) 可能能够解析多义性。  
  
 此错误是与 CAN0055 相同。
