# 属性表 (MFC)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

此系列文章介绍如何实现在属性表支持的 MFC 应用程序。  属性表，也称为"选项卡"对话框中，提供一种管理在对话框中大量控件。  属性表包含属性页，一个基于对话框模板资源的每个。  可划分的对话框控件逻辑分组和自己将每个组属性页。
