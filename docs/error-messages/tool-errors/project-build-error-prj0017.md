# 项目生成错误 PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

当前的工作目录无效。  
  
 当前工作目录的路径比 \_MAXPATH 长。  若要解决该错误，请不要用这么深的层级创建项目。
