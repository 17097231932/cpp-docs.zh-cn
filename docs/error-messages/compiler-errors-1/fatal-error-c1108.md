# <a name="fatal-error-c1108"></a>错误 C1108

> 找不到 DLL： "*dll 名称*"

在路径中找不到指定的 DLL （*dll 名称*）。

若要解决此错误，请C++重新安装视觉对象，或将相应的 .dll 文件从安装中复制到您的计算机。
