# <a name="cbytearray-class"></a>CByteArray 类

支持字节的动态数组。

## <a name="syntax"></a>语法

```
class CByteArray : public CObject
```

## <a name="members"></a>成员

的成员函数 `CByteArray` 类似于类 [CObArray](../../mfc/reference/cobarray-class.md)的成员函数。 由于此相似性，因此你可以使用 `CObArray` 参考文档获取成员函数细节。 无论你在何处看到 `CObject` 作为函数参数或返回值的指针，都将替换为一个字节。

`CObject* CObArray::GetAt( int <nIndex> ) const;`

例如，转换为

`BYTE CByteArray::GetAt( int <nIndex> ) const;`

## <a name="remarks"></a>备注

`CByteArray` 合并 IMPLEMENT_SERIAL 宏，以支持其元素的序列化和转储。 如果使用重载插入 () 运算符或成员函数将字节数组存储到存档， **<<** `Serialize` 则每个元素依次序列化。

> [!NOTE]
> 在使用数组之前，先使用 `SetSize` 建立其大小并为其分配内存。 如果不使用 `SetSize`，则向数组添加元素会导致它经常重新分配和复制。 经常重新分配和复制会降低效率而且会产生内存碎片。

如果需要从数组中的单个元素调试输出，则必须将对象的深度设置 `CDumpContext` 为1或更大。

有关使用的详细信息 `CByteArray` ，请参阅文章 [集合](../../mfc/collections.md)。
