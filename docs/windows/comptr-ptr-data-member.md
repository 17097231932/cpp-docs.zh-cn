# <a name="comptrptr-data-member"></a>ComPtr::ptr_ 数据成员
包含指向相关联，并管理此接口的指针**ComPtr**。  
  
## <a name="syntax"></a>语法  
  
```  
InterfaceType *ptr_;  
```  
  
## <a name="remarks"></a>备注  
 **ptr_** 是一个内部、 受保护的数据成员。  
  
## <a name="requirements"></a>要求  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
