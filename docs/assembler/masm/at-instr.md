# <a name="instr"></a>\@InStr

用于在*string1*中从*位置*开始查找*string2*中第一个匹配项的宏*函数。* 如果未出现位置，搜索*将*在*string1*开头开始。 如果未找到*string2* ，则返回一个位置整数或0。

## <a name="syntax"></a>语法

> **\@InStr （** ⟦*位置*⟧ __，__ *string1* __，__ *string2* **）**
