# <a name="struct-masm"></a>STRUCT (MASM)
声明具有指定的结构类型*fielddeclarations*。 每个字段必须是一个有效的数据定义。 与相同[STRUC](../../assembler/masm/struc.md)。  
  
## <a name="syntax"></a>语法  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
