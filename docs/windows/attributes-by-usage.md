# Attributes by Usage
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

本主题列出了根据要应用的 C\+\+ 语言元素的属性。  
  
 如果属性之前不是属性的作用域中的 Visual C\+\+ 元素，则该属性块视为注释。  
  
|特性|说明|  
|--------|--------|  
|[模块属性](../windows/module-attributes.md)|适用于[模块](../windows/module-cpp.md)属性。|  
|[接口属性](../windows/interface-attributes.md)|适用于 [\_\_interface](../cpp/interface.md) C\+\+ 关键字。|  
|[类属性](../windows/class-attributes.md)|适用于 C\+\+ 关键字。|  
|[方法属性](../windows/method-attributes.md)|适用于类、 coclass 或接口中的方法。|  
|[参数属性](../windows/parameter-attributes.md)|应用于类或接口中的方法的参数。|  
|[数据成员属性](../windows/data-member-attributes.md)|适用于类、 coclass 或接口中的数据成员。|  
|[Typedef、 枚举、 联合和结构属性](../windows/typedef-enum-union-and-struct-attributes.md)|适用于 C\+\+ 关键字。|  
|[数组属性](../windows/array-attributes.md)|适用于阵列或 **SAFEARRAY**s。|  
|[独立属性](../windows/stand-alone-attributes.md)|运行更像一个代码行，但不会无法运行在 C\+\+ 关键字。  独立属性语句要求在行尾的分号。|  
|[自定义特性](../windows/custom-attributes-cpp.md)|允许用户扩展元数据。|
