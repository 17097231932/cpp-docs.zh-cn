# <a name="ml-nonfatal-error-a2119"></a>ML 非致命错误 A2119

**必须指定语言类型**

过程定义或原型没有给定的语言类型。

如果未指定默认语言类型，必须在每个过程定义或原型中声明的语言类型。 使用设置的默认语言类型[。模型](../../assembler/masm/dot-model.md)指令，却**选项 LANG**，或机器学习命令行选项 **/Gc**或 **/Gd**。
