# <a name="command-line-warning-d9028"></a>命令行警告 D9028

最小重新生成失败，正在转换到常规的生成

项目的.idb 文件已损坏。 删除该文件并重新生成。

最小重新生成和.idb 文件的详细信息，请参阅[/Gm 编译器选项](../../build/reference/gm-enable-minimal-rebuild.md)。
