# <a name="project-build-error-prj0035"></a>项目生成错误 PRJ0035
XML 文件 file 包含无法转换为用户的 ANSI 代码页的 Unicode 内容。  
  
 ***UNICODE 文件的内容***  
  
 ***文件***是作为 Web 部署工具命令行创建的 XML 文件。  
  
 项目系统正确不能转换为 ANSI 的 Web 部署属性页上，在某些属性中找到 Unicode 字符。  
  
 此错误的解决方法是更新以使用 ANSI 或要在你的计算机上安装的代码页，并将其设置为系统默认值的属性的内容。
