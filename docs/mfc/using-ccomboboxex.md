# <a name="using-ccomboboxex"></a>使用 CComboBoxEx

扩展组合框控件是为项图像提供本机支持的标准组合框控件的扩展。 这些图像可用于指示组合框中各个项的状态，例如当前选定和未选定的项。 为了使项映像易于访问，控件提供了图像列表支持。

使用此控件可提供组合框的功能，而无需手动绘制项图形。
