# <a name="project-build-error-prj0035"></a>项目生成错误 PRJ0035

> XML 文件*文件*包含未能翻译成用户的 ANSI 代码页的 Unicode 内容。
>
> *UNICODE 文件的内容*

*文件*是作为对 Web 部署工具命令行创建的 XML 文件。

项目系统无法正确转换为 ANSI 的 Web 部署属性页上，某些属性中找到的 Unicode 字符。

此错误的解决方法是属性的更新以使用 ANSI 或要在计算机上安装的代码页，并将其设置为系统默认值的内容。
