# <a name="regex_traitsltwchar_tgt-class"></a>regex_traits&lt;wchar_t&gt; 类

为**wchar_t**`regex_traits` 专用化。

## <a name="syntax"></a>语法

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>备注

类是类型为**wchar_t**的元素的类模板[regex_traits](../standard-library/regex-traits-class.md)的显式专用化（以便它可以充分利用操作此类型对象的库函数）。

## <a name="requirements"></a>要求

**标头：** \<regex 1>

**命名空间:** std
