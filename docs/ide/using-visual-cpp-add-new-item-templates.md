# <a name="using-visual-c-add-new-item-templates"></a>使用 Visual C++ 的“添加新项”模板
你可以轻松地添加通过使用 Visual c + + 项目所共有的项**添加新项**命令。 当你使用**添加新项**命令，`Add New Item`对话框中将显示的项模板，将相应的文件添加到你的项目列表。  
  
 下表是 Visual c + + 添加新项模板的字母顺序列表。  
  
|模板|描述|  
|--------------|-----------------|  
|程序集资源文件 (.resx)|创建包含 CLR 资源的文件。|  
|位图文件 (.bmp)|创建 Win32 位图文件。|  
|C + + 文件 (.cpp)|创建 c + + 源文件。|  
|类图 (.cd)|创建一个空的类图。|  
|代码分析规则集 (.ruleset)|创建用于配置代码分析的设置文件。|  
|配置文件 (app.config)|创建一个空的配置文件。|  
|组件类|添加使用 CLR 功能的组件类。|  
|光标 (.cur) 文件|创建 Win32 光标文件。|  
|发现文件，静态 (.disco)|创建一个静态发现文件，其中是一个 XML 文档，包含描述 XML Web 服务的其他资源的链接、 启用的 XML Web 服务以编程方式发现。|  
|框架集 (.htm)|添加承载多个 HTML 页的 HTML 文件。|  
|标头文件 (.h)|创建 c + + 头文件。|  
|HTML 页 (.htm)|创建一个空的 HTML 文件。|  
|图标文件 (.ico)|创建 Win32 图标文件。|  
|安装程序类|添加了一个类继承自[安装程序类](https://msdn.microsoft.com/en-us/library/system.configuration.install.installer.aspx)使用 CLR 功能。|  
|MIDL 文件 (.idl)|创建一个接口定义语言文件。|  
|模块定义文件 (.def)|创建一个 DLL 导出定义文件。|  
|属性表 (.props)|创建一个 Visual c + + 属性表文件。|  
|注册脚本 (.rgs)|创建一个 ATL 注册脚本文件。|  
|报告定义 (.rdlc)|创建报表文件。|  
|资源文件 (.rc)|创建 Win32 资源文件。|  
|资源模板文件 (.rct)|创建资源模板文件。|  
|功能区 (.mfcribbon ms)|创建功能区文件。|  
|服务器响应文件 (.srf)|创建与 ATL Server 一起使用的服务器响应文件。|  
|所需的 SQL 脚本文件 (.sql)|创建一个 SQL 脚本文件。 **注意：**此模板不是 Professional 版功能。|  
|样式表 (.css)|将添加用于 rich HTML 样式定义的级联样式表。|  
|文本文件 (.txt)|添加空白文本文件。|  
|用户控件|添加用户控件使用 CLR 功能。|  
|Windows 窗体|添加 Windows 窗体上使用 CLR 功能。|  
|XML 文件 (.xml)|将添加一个空白的 XML 文件。|  
|XML 架构文件 (.xsd)|创建用于定义 XML 文档的架构文件。|  
|XSLT 文件 (.xslt)|创建用来转换 XML 文档的文件。|
