# <a name="atl-window-classes"></a>ATL 窗口类

ATL 包含可用于使用和实现 windows 的多个类。 这些类，类似于其他 ATL 类，提供不会施加于你的代码开销的高效实现。

本部分介绍 ATL 窗口类，并说明如何使用它们。

[使用窗口](../atl/using-a-window.md)<br/>
讨论如何使用`CWindow`来操作一个窗口。

[实现窗口](../atl/implementing-a-window.md)<br/>
讨论消息处理程序、 消息映射，以及使用`CWindowImpl`。 包括有关创建超类和子类化的详细信息。

[实现对话框](../atl/implementing-a-dialog-box.md)<br/>
讨论用于添加的对话框类的两种方法，并显示了一个代码示例。

[使用包含的窗口](../atl/using-contained-windows.md)<br/>
讨论包含的窗口在 ATL 中，这是委托给容器对象，而不是在其自身的类中处理它们及其消息的窗口。

讨论在 atl。 窗口特征类 这些类提供标准化的样式用于窗口对象创建一个简单方法。
