# 编译器警告（等级 1）C4187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import 特性“attribute1”和“attribute2”不兼容；均被忽略  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 语句指定了 `no_implementation` 和 `implementation_only` 特性。 忽略这两个特性。
