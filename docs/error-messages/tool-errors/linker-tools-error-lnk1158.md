# <a name="linker-tools-error-lnk1158"></a>链接器工具错误 LNK1158

无法运行 "filename"

[Link](../../build/reference/linking.md)调用的给定可执行文件不在包含链接的目录中，也不在 PATH 环境变量中指定的目录中。

例如，如果尝试在具有32位[操作系统的计算机](../../build/reference/ltcg-link-time-code-generation.md)上使用 PGOPTIMIZE 参数，则会出现此错误。
