# <a name="attributes-by-group"></a>按组分的特性

C + + 特性已组织成以下的功能组。

|特性|描述|
|---------------|-----------------|
|[COM 特性](com-attributes.md)|注入的代码来支持 COM 开发和.NET Framework 公共语言运行时开发的许多方面。|
|[IDL 特性](idl-attributes.md)|启用从一个源代码文件，而不使用向导或不包含结构的知识和该文件的语法的.idl 文件的修改。|
|[OLE DB 使用者特性](ole-db-consumer-attributes.md)|将代码注入-根据[OLE DB 使用者模板](../../data/oledb/ole-db-consumer-templates-reference.md)— 若要创建使用 OLE DB 使用者执行任务，例如打开表执行命令和访问数据。|
|[编译器特性](compiler-attributes.md)|由 Microsoft c + + 编译器提供。|

有关如何定义自定义属性的.NET Framework 应用程序的信息，请参阅[用户定义的特性](../user-defined-attributes-cpp-component-extensions.md)。
