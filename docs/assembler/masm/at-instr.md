# <a name="instr"></a>@InStr
查找的第一个匹配项的宏函数*string2*中*string1*开始，*位置*内*string1*。 如果*位置*未显示，在开始处开始搜索*string1*。 如果返回位置整数或 0 *string2*找不到。  
  
## <a name="syntax"></a>语法  
  
```  
  
@InStr( [[position]], string1, string2 )  
```
