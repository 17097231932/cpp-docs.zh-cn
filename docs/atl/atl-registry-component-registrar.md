# <a name="atl-registry-component-registrar"></a>ATL 注册表组件（注册器）

ATL 注册器提供了通过自定义界面在系统注册表的优化的访问。 在注册机构是自由线程，允许 c + + 客户端的静态链接的代码。

> [!NOTE]
>  可在 atlmfc\include\atliface.h ATL 注册器的源代码。

[设置静态链接到的注册机构代码 （c + +）](../atl/setting-up-a-static-link-to-the-registrar-code-cpp-only.md)<br/>
