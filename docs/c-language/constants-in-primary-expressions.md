# <a name="constants-in-primary-expressions"></a>主要表达式中的常量

常量操作数具有它表示的常量的值和类型。 字符常量具有 `int` 类型。 整数常量具有 `int`、long  `unsigned int` 或 `unsigned long` 类型，具体取决于整数的大小和指定值的方式。 有关详细信息，请参阅[常量](../c-language/c-constants.md)。
