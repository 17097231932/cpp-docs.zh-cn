# Toolbar (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

“图像编辑器”工具栏包含用于绘制、绘画、输入文本、清除和操作视图的工具。  它还包含一个选项选择器，使用该选项选择器可以为使用每种工具选择选项。  例如，可以从各种画笔宽度、放大因子和线条样式中选择。  
  
> [!NOTE]
>  在**“图像编辑器”**工具栏上可用的所有工具也可以从**“图像”**菜单（在**“工具”**命令下）中获得。  
  
 ![“图像编辑器”工具栏](../mfc/media/vcimageeditortoolbar.gif "vcImageEditorToolbar")  
“图像编辑器”工具栏  
  
 若要使用**“图像编辑器”**工具栏和选项选择器，请单击要所需的工具或选项。  
  
> [!TIP]
>  当光标悬停在工具栏按钮上方时会出现工具提示。  这些提示可帮助您识别每个按钮的功能。  
  
 使用“选项”选择器可以指定线条的宽度、画笔笔画等。  “选项”选择器按钮上的图标会根据您选定的工具变化。  
  
 ![“图像编辑器”工具栏上的绘图形状选择器](../mfc/media/vcimageeditortoolbaroptionselector.gif "vcImageEditorToolbarOptionSelector")  
“图像编辑器”工具栏上的选项选择器  
  
 有关将资源添加到托管项目的信息，请参见“.NET Framework 开发员指南”中的[应用程序中的资源](../Topic/Resources%20in%20Desktop%20Apps.md)。有关手动将资源文件添加到托管项目、访问资源、显示静态资源和将资源字符串分配给属性的信息，请参见[演练：本地化 Windows 窗体](http://msdn.microsoft.com/zh-cn/9a96220d-a19b-4de0-9f48-01e5d82679e5)和[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)。
