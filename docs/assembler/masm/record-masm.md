# <a name="record-masm"></a>RECORD (MASM)
声明一包含的指定字段的记录类型。 *fieldname*名称字段，*宽度*指定的位数，和*表达式*提供其初始值。  
  
## <a name="syntax"></a>语法  
  
```  
  
   recordname RECORD fieldname:width [[= expression]]   
[[, fieldname:width [[= expression]]]]...  
```
