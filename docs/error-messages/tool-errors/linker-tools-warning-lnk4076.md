# 链接器工具警告 LNK4076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的增量状态文件“filename”；正在非增量链接  
  
 LINK 无法写入增量状态 \(.ilk\) 文件。  `filename` 已损坏或者不是增量链接数据库。  移除该文件，然后重新链接。
