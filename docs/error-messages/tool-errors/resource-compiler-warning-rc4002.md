# <a name="resource-compiler-warning-rc4002"></a>资源编译器警告 RC4002

“identifier”宏的实参太多

用给定标识符指定的实际参数的数量大于在标识符的宏定义中给定的形参数目。

其他实际参数已收集，但在宏扩展过程中被忽略。
