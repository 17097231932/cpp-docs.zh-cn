# <a name="platformdetails-namespace"></a>Platform::Details 命名空间
此命名空间仅供内部使用，不用于开发。  
  
## <a name="syntax"></a>语法  
  
```cpp  
  
namespace Platform {  
   namespace Details {  
}}  
```
