# <a name="compiler-warning-level-1-c4143"></a>编译器警告（等级 1）C4143

不支持杂注“same_seg”；使用 __based 分配

**#Pragma same_seg** 不再受支持。 请改用 [__based](../../cpp/based-pointers-cpp.md) 关键字。
