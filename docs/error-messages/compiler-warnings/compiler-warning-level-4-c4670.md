# <a name="compiler-warning-level-4-c4670"></a>编译器警告（等级 4）C4670
“identifier”：该基类不可访问  
  
 要在 **try** 块中引发的对象的指定基类不可访问。 不能实例化引发的对象。 检查基类继承了正确的访问说明符。  
  
 下面的示例生成 C4670：  
  
```  
// C4670.cpp  
// compile with: /EHsc /W4  
class A  
{  
};  
  
class B : /* public */ A  
{  
} b;   // inherits A with private access by default  
  
int main()  
{  
    try  
    {  
       throw b;   // C4670  
    }  
    catch( B )  
    {  
    }  
}  
```
