# <a name="c28287"></a>C28287

> 警告 C28287：对于函数，中的语法 \_ 错误 \_ ( # A1 批注 (无法识别的参数名称) 

当使用 `SAL_at` (`__drv_at`) 批注，并且参数表达式无法在当前上下文中解释时，代码分析工具将报告此警告。 这可能包括使用拼写错误的参数或成员名称，或者“return”或“this”关键字拼写错误。
