# <a name="displaybind"></a>displaybind
指示应显示给用户作为可绑定的属性。  
  
## <a name="syntax"></a>语法  
  
```  
  
[displaybind]  
  
```  
  
## <a name="remarks"></a>备注  
 **Displaybind** c + + 属性具有相同的功能[displaybind](http://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL 特性。  
  
## <a name="example"></a>示例  
 请参阅示例[可绑定](../windows/bindable.md)以举例说明如何使用**displaybind**。
