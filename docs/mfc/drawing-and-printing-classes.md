# <a name="drawing-and-printing-classes"></a>绘制和打印类
在 Windows 中，所有图形输出都在一个称为设备上下文 (DC) 的虚拟绘制区域上绘制。 MFC 提供用于封装各种类型的 DC 的类，以及 Windows 绘制工具（如位图、画笔、调色板和钢笔）的封装。
