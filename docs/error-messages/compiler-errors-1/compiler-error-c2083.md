# 编译器错误 C2083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

结构\/联合比较非法  
  
 结构或联合直接与另一个用户定义的类型进行比较 此操作不被允许，除非已定义了比较运算符或存在到标量类型的转换。
