# <a name="synclocktsynclockt-destructor"></a>SyncLockT::~SyncLockT 析构函数
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
~SyncLockT();  
```  
  
## <a name="remarks"></a>备注  
 取消初始化的实例**SyncLockT**类。  
  
 此析构函数还能释放种种当前**SyncLockT**实例。  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers::Details
