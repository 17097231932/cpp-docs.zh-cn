# 每个字符的位数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** 字符中的位数  
  
 字符中的位数由清单常量 **CHAR\_BIT** 表示。  LIMITS.H 文件将 **CHAR\_BIT** 定义为 8。
