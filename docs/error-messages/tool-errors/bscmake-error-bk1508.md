# <a name="bscmake-error-bk1508"></a>BSCMAKE 错误 BK1508
无法删除临时文件 filename [: 原因]  
  
 BSCMAKE 无法删除的一个临时文件。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复  
  
1.  另一个进程已打开该文件。  
  
2.  硬件错误。
