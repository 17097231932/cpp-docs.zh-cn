# <a name="modulemodule-destructor"></a>Module::~Module 析构函数
取消初始化的当前实例**模块**类。  
  
## <a name="syntax"></a>语法  
  
```  
virtual ~Module();  
```  
  
## <a name="requirements"></a>要求  
 **标头：** module.h  
  
 **命名空间：** Microsoft::WRL
