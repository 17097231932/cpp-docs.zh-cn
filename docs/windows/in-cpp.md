# in (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

指示参数是从被调用过程传递给被调用过程。  
  
## 语法  
  
```  
  
[in]  
  
```  
  
## 备注  
 **在** C\+\+ 特性具有与 [在](http://msdn.microsoft.com/library/windows/desktop/aa367051) MIDL 属性相同。  
  
## 示例  
 为的示例演示如何参见 [可绑定](../windows/bindable.md) 使用 **在**。  
  
## 要求  
  
### 属性上下文  
  
|||  
|-|-|  
|**适用对象**|接口参数，接口方法|  
|**可重复**|否|  
|**必需的特性**|无|  
|**无效的特性**|**retval**|  
  
 有关属性上下文的更多信息，请参见 [属性上下文](../windows/attribute-contexts.md)。
