# <a name="windowsfoundation-namespace"></a>Windows::Foundation 命名空间
启用基本 Windows 运行时功能，如对象和工厂创建。  
  
## <a name="syntax"></a>语法  
  
```  
namespace Windows::Foundation;  
```
