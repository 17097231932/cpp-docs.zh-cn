# <a name="compiler-error-c2332"></a>编译器错误 C2332
typedef： 缺少标记名称  
  
 编译器找到的不完整的类型定义。  
  
 下面的示例生成 C2332:  
  
```  
// C2332.cpp  
// compile with: /c  
struct S {  
   int i;  
};  
  
typedef struct * pS;   // C2332  
typedef struct S* pS;   // OK  
  
int get_S_i(pS p) {  
   return p->i;  
}  
```
