# <a name="compiler-error-c2487"></a>编译器错误 C2487
identifier： 不能与 dll 接口声明 dll 接口的类的成员  
  
 您可以声明一个整个类或具有 DLL 接口的非 DLL 接口类的某些成员。 你不能声明具有 DLL 接口的类，然后再声明此类与 DLL 接口的成员。
