# <a name="command-line-error-d8036"></a>命令行错误 D8036

不允许 "/option" 用于多个源文件

这些编译器选项不能与多个源文件一起使用：

- 命名程序集文件列表（/Fa）

- 重命名对象文件（/Fo）

- 创建无局部变量的源浏览器信息（/Fr）

- 创建包含本地变量的源浏览器信息（/FR）
