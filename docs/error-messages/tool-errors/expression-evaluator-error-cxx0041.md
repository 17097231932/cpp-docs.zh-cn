# <a name="expression-evaluator-error-cxx0041"></a>表达式计算器错误 CXX0041

类元素必须是静态成员或成员函数

未指定要使用的类的实例化的情况下使用了类 (或结构或联合) 的非静态成员。

只有静态数据成员或成员函数可以在不指定实例化的情况下使用。

此错误与 CAN0041 相同。
