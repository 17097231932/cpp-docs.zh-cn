# <a name="building-and-testing-the-atl-project"></a>生成和测试 ATL 项目

如 [插入复合控件](../atl/inserting-a-composite-control.md)中所述，项目的一个初始组件是承载新复合控件的默认 HTML 页。 完成修改复合控件后，请从 "**生成**" 菜单中单击 "**生成解决方案**" 或 "**重新生成解决方案**"。 项目成功生成后，将位于复合控件项目根目录中的 HTML 页面加载到 Internet Explorer 或其他浏览器，并测试控件的功能。

你还可以使用 "测试容器" 工具或可承载 ActiveX 控件的任何其他应用程序来测试复合控件。 请参阅 [使用测试容器测试属性和事件](../mfc/testing-properties-and-events-with-test-container.md) 了解有关如何访问测试容器的信息。
