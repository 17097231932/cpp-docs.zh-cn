# <a name="expression-evaluator-error-cxx0059"></a>表达式计算器错误 CXX0059

左的操作数是类不是函数名称

函数调用的左的操作数是类名和无法解析为函数调用。 忽略在表达式中的成员函数的名称会导致此错误。

此错误是与 CAN0059 相同。
