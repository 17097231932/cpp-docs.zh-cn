# Controls in Dialog Boxes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

可以使用[“工具箱”窗口](../Topic/Toolbox.md)中的[“对话框编辑器”选项卡](../mfc/dialog-editor-tab-toolbox.md)将控件添加到对话框。“工具箱”窗口使您能够选择所需的控件并将其拖到对话框上。  默认情况下，此窗口设置为自动隐藏。  当对话框编辑器打开时，此窗口显示为解决方案左边距上的选项卡。  不过，可以通过单击“工具箱”窗口右上角的“自动隐藏”按钮，将此窗口收回到位。  有关如何控制此窗口行为的更多信息，请参见[窗口管理](../Topic/Customizing%20window%20layouts%20in%20Visual%20Studio.md)。  
  
 将控件添加到对话框、重新定位现有控件或将控件从一个对话框移到另一个对话框的最快方法是使用拖放方法。  控件在被放入对话框之前，其位置轮廓是虚线。  当使用拖放方法将控件添加到对话框时，控件具有与该类控件相应的标准高度。  
  
 当将控件添加到对话框或重新定位控件时，该控件的最终位置可能由参考线或边距决定，或者取决于是否打开了布局网格。  
  
 一旦将控件添加到了对话框后，就可以在[“属性”窗口](../Topic/Properties%20Window.md)中更改控件标题等属性。  可以选择多个控件并同时更改这些控件的属性。
