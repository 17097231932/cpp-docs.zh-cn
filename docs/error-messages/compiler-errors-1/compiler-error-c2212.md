# <a name="compiler-error-c2212"></a>编译器错误 C2212
identifier： 不 __based 可用于指向函数的指针  
  
 不能声明为指向函数的指针`__based`。 如果你需要基于代码的数据，使用`__declspec`关键字或`data_seg`杂注。
