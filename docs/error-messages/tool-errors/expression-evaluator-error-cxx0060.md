# <a name="expression-evaluator-error-cxx0060"></a>表达式计算器错误 CXX0060
注册不可用  
  
 将指定的表达式不能使用的寄存器。 可以通过尝试访问不存在运行的计算机的寄存器导致此错误。  
  
 此错误是与 CAN0060 相同。
