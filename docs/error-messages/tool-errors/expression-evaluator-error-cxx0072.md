# 表达式计算器错误 CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

错误：类型信息缺少或未知  
  
 未链接 .pch 文件，或者代码引用的类型所在的模块不是用 \/Zi 编译的。  
  
 该错误与 CAN0067 相同。
