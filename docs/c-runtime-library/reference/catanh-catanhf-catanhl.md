# <a name="catanh-catanhf-catanhl"></a>catanh、catanhf、catanhl

检索复数的反双曲正切值，并沿实轴的间隔 [-1; + 1] 之外的分支剪切。

## <a name="syntax"></a>语法

```C
_Dcomplex catanh(
   _Dcomplex z
);
_Fcomplex catanh(
   _Fcomplex z
);  // C++ only
_Lcomplex catanh(
   _Lcomplex z
);  //  C++ only
_Fcomplex catanhf(
   _Fcomplex z
);
_Lcomplex catanhl(
   _Lcomplex z
);
```

### <a name="parameters"></a>参数

*z*<br/>
表示角度的复数（以弧度为单位）。

## <a name="return-value"></a>返回值

*Z*的反双曲正切值（以弧度表示）。 沿实轴的结果是无限的，并且沿虚轴的间隔 [-i π/2; + i π/2] 中。 如果*z*超出间隔 [-1，+ 1]，将出现域错误。 如果*z*为-1 或 + 1，则将出现极点错误。

## <a name="remarks"></a>备注

由于C++允许重载，因此可以调用**catanh**的重载，该重载采用并返回 **_Fcomplex**和 **_Lcomplex**值。 在 C 程序中， **catanh**始终采用并返回 **_Dcomplex**值。
