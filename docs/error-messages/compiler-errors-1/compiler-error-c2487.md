# <a name="compiler-error-c2487"></a>编译器错误 C2487

identifier: dll 接口类的成员不能与 dll 接口声明

您可以声明整个类或具有 DLL 接口的非 DLL 接口类的某些成员。 您不能声明具有 DLL 接口的类，然后再声明与 DLL 接口的类的成员。
