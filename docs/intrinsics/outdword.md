# <a name="outdword"></a>__outdword
**Microsoft 专用**  
  
 生成`out`指令以将发送双字`Data`输出端口`Port`。  
  
## <a name="syntax"></a>语法  
  
```  
void __outdword(   
   unsigned short Port,   
   unsigned long Data   
);  
```  
  
#### <a name="parameters"></a>参数  
 [in] `Port`  
 要将数据发送到的端口。  
  
 [in] `Data`  
 要发送双字。
