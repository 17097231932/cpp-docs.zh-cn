# operator GE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回 true \(– 1\)，如果 *expression1* 大于或等于 *expression2*，或者返回错误 \(0\)，则不是。  
  
## 语法  
  
```  
  
expression1 GE expression2  
```
