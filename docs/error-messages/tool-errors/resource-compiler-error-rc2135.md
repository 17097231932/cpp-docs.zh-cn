# <a name="resource-compiler-error-rc2135"></a>资源编译器错误 RC2135
找不到文件：filename  
  
 找不到资源编译器命令行中指定的文件。 检查文件是否已被移动到另一个目录和文件名称和路径是否正确键入。  
  
 文件中搜索使用**包括**环境变量或 Visual c + +**包括**设置。
