# Error Information Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

下面选件类指示错误信息的处理方式:  
  
-   [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) 确定对象是否支持 [IErrorInfo](http://msdn.microsoft.com/zh-cn/4dda6909-2d9a-4727-ae0c-b5f90dcfa447) 接口。  **IErrorInfo** 允许错误信息传播到客户端。
