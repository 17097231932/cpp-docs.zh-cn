# <a name="linker-tools-error-lnk1287"></a>链接器工具错误 LNK1287

无效的托管的入口点函数

入口点不是有效的托管映像。 托管的入口点函数的返回类型只能是`void`或`int`。 托管的入口点函数的参数的类型只能`void`或`String []`。
