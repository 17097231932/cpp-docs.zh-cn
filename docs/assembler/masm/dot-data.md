# .DATA
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

当使用 [.MODEL](../../assembler/masm/dot-model.md)，开始每个最近的数据段用于初始化的数据 \(段名 \_DATA\)。  
  
## 语法  
  
```  
.DATA  
```
