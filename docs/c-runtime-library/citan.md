# <a name="citan"></a>_CItan

计算浮点堆栈顶部值的正切值。

## <a name="syntax"></a>语法

```C
void __cdecl _CItan();
```

## <a name="remarks"></a>备注

此版本的 [tan](../c-runtime-library/reference/tan-tanf-tanl.md) 函数具有编译器理解的专用化调用约定。 该函数将加快执行的速度，因为它可防止生成副本和帮助注册表分配。

生成的值被将被推送到浮点堆栈顶部。

## <a name="requirements"></a>要求

**平台：** x86
