# 为提供程序创建项目
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

#### 创建 OLE DB 提供程序将驻留在其中的项目  
  
1.  在“文件”菜单上，单击“新建”\(`New`\)，然后单击“项目”。  
  
     此时将出现**“新建项目”**对话框。  
  
2.  在“项目类型”窗格中单击“Visual C\+\+ 项目”文件夹。  在“模板”窗格中单击**“ATL 项目”**。  
  
3.  在**“名称”**框中输入项目的名称，然后单击**“确定”**。  
  
     即会出现“ATL 项目向导”。  
  
4.  在 ATL 项目向导中，为“服务器类型”选择“动态链接库 \(DLL\)”。  
  
5.  单击**“完成”**。
