# <a name="label"></a>LABEL

通过将当前位置计数器值和给定 *qualifiedType* 分配给 *name* 来创建新标签。

## <a name="syntax"></a>语法

> *名称***标签** *qualifiedType*
>
> *名称***标签**⟦**近** 的  |    |  **过程**⟧ **PTR** ⟦*qualifiedType*⟧
