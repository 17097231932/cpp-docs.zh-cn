# <a name="ml-nonfatal-error-a2133"></a>ML 非致命错误 A2133

**register 值被调用覆盖**

寄存器作为参数传递给过程，但由 [调用](invoke.md) 生成的代码通过其他自变量来传递注册的内容。

汇编程序可以使用 AX、AL、AH、EAX、DX、DL、DH 和 EDX 寄存器来执行数据转换。

使用其他寄存器。
