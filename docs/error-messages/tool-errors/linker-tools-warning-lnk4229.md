# <a name="linker-tools-warning-lnk4229"></a>链接器工具警告 LNK4229

无效的指令 /directive 找到;忽略

通过传递一个指令[注释](../../preprocessor/comment-c-cpp.md)杂注无效。 链接器将忽略`/directive`。
