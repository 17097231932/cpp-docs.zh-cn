# Modifying the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

此时，您的复合控件项目实现您的复合控件中所必需的对象。  下一步是添加该复合控件将包含和处理任何必要的事件的所有控件。  
  
 若要添加其他的ActiveX或Windows控件中，添加一个新的资源脚本然后使用对话框编辑器。  有关添加的更多信息控件\(及相关任务\)，请参见 [对话框编辑器](../mfc/dialog-editor.md)。  
  
 处理从ActiveX控件的所有必要的事件，请参见 [将功能添加到复合控件](../atl/adding-functionality-to-the-composite-control.md)。
