# <a name="tmpmax-ltmpnam"></a>TMP_MAX、L_tmpnam

## <a name="syntax"></a>语法

```
#include <stdio.h>
```

## <a name="remarks"></a>备注

`TMP_MAX` 是 `tmpnam` 函数可生成的最大数量的唯一文件名。 `L_tmpnam` 是 `tmpnam` 生成的临时文件名的长度。
