# 匿名类型初始值设定项在调试窗口中无效
在调试会话期间，已在“监视”或“即时”窗口中输入匿名类型初始值设定项。 匿名类型初始值设定项在调试窗口中不可用。  
  
 **错误 ID：**BC36577  
  
### 更正此错误  
  
-   在代码编辑器中测试匿名类型初始值设定项。
