# MSBuild 错误 MSB3123
**MSB3123: 文件关联数超过限制 \<限制\>。**  
  
 每个文件只允许有固定数目的关联文件扩展名。  当您尝试关联的文件扩展名数超过目标操作系统所允许的数目时，会发生此错误。  
  
### 更正此错误  
  
-   将文件关联数限制为指定数目。
