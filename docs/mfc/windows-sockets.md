# <a name="windows-sockets"></a>Windows 套接字

本系列文章涵盖 Windows 套接字的 MFC 实现。 MFC 提供了两个类以支持通过 Windows 套接字 API 对网络应用程序进行编程。 类 [CAsyncSocket](../mfc/reference/casyncsocket-class.md) 封装 Windows 套接字 API，为高级网络程序员提供最强大的功能和灵活性。 类 [CSocket](../mfc/reference/csocket-class.md) 提供用于将数据序列化到 [CArchive](../mfc/reference/carchive-class.md) 对象的简化接口。
