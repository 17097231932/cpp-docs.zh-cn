# <a name="atl-registry-component-registrar"></a>ATL 注册表组件（注册器）

ATL 注册机构通过自定义界面提供对系统注册表的优化访问。 注册器是自由线程的，允许 c + + 客户端的代码的静态链接。

> [!NOTE]
> 可以在 atlmfc\include\atliface.h. 中找到 ATL 注册器的源代码。
