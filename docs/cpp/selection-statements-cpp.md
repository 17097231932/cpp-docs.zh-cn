# <a name="selection-statements-c"></a>选择语句 (C++)

C + + 选择语句（ [如果](../cpp/if-else-statement-cpp.md) 和 [开关](../cpp/switch-statement-cpp.md)）提供了一种有条件地执行代码段的方法。

[__If_exists](../cpp/if-exists-statement.md)和[__if_not_exists](../cpp/if-not-exists-statement.md)语句允许根据符号是否存在，有条件地包括代码。

有关每个语句的语法，请参阅单独的主题。
