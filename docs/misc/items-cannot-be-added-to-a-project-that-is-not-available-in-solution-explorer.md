# 无法将项添加到在解决方案资源管理器中不可用的项目中。
当已指定项目未在解决方案资源管理器中打开时通常都会发生此错误。 Visual Studio 无法将项添加到当前在解决方案资源管理器中不可用的项目。  
  
### 更正此错误  
  
1.  将指定的项目添加到当前解决方案或打开包含指定项目的解决方案。
