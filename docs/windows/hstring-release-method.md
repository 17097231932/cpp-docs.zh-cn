# <a name="hstringrelease-method"></a>HString::Release 方法
删除基础字符串值，并初始化当前**HString**对象为空值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
void Release() throw()  
```  
  
## <a name="requirements"></a>要求  
 **标头：** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
