# <a name="compiler-warning-level-1-c4027"></a>编译器警告（等级 1）C4027
未用形参表声明的函数  
  
 函数声明没有形参，但在函数定义中有形参或在调用中有实参。 对此函数的后续调用假定该函数采用函数定义或调用中找到的类型的实参。
