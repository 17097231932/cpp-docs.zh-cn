# C.1 Notation
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

语法规则包括名称非终止字符，后跟冒号，后跟在单独的行上的替换选择。  
  
 表达式语法术语opt 指示该术语在替换内是可选的。  
  
 表达式语法 *术语optseq*与 *术语顺序*等效于以下附加opt 规则:  
  
 *术语序列* :  
  
 *术语*  
  
 *术语顺序术语*  
  
 *术语序列* ， *术语*
