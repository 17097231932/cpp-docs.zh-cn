# <a name="cvtres-warning-cvt4001"></a>CVTRES 警告 CVT4001
**未指定; 计算机类型假定**   
 ***type***  
  
 CVTRES 找不到机规范。 它假定给定的计算机类型。 如果默认值是不正确，重新运行 CVTRES 使用 /MACHINE 选项。
