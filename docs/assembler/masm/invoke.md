# INVOKE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

调用过程在 *表达式*会的地址，传递参数在堆栈上或在注册根据调用语言类型的约定该条件。  
  
## 语法  
  
```  
  
INVOKE   
expression [[, arguments]]  
```  
  
## 备注  
 每个参数传递给过程可能是表达式，注册对或地址表达式 \( `ADDR`后面的表达式\)。
