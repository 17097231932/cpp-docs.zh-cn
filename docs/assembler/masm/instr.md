# <a name="instr"></a>INSTR
查找的第一个匹配项*textitem2*中*textitem1*。  
  
## <a name="syntax"></a>语法  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>备注  
 起始*位置*是可选的。 每个文本项可以是文本字符串，前面是一个常数`%`，或通过宏函数返回的字符串。
