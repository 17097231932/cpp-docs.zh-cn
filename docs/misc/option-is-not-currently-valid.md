# Option is not currently valid.
当为命令指定的参数或开关当前不可用时，通常会发生此错误。  例如，如果输入 `>Edit.Find test /sel` 而未在文件中选定内容，则 \/sel 开关不可用。  
  
### 更正此错误  
  
1.  验证输入的参数和开关当前是可用的。  有关更多信息，请参见[带参数的 Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。
