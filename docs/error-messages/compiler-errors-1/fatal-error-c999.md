# <a name="fatal-error-c999"></a>错误 C999

未知消息请选择 Visual C++ "帮助" 菜单上的 "技术支持" 命令，或打开技术支持帮助文件以获取详细信息

此错误通常表示你有具有来自编译器不同版本的混合文件，或者你的编译器安装已损坏。 使用控制面板中的 " **程序和功能** " 小程序来修复或重新安装产品。
