# Map::Size 方法
返回 Map 中的 [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) 元素数目。  
  
## 语法  
  
```  
  
virtual property unsigned int Size;  
```  
  
## 返回值  
 Map 中的元素数目。  
  
## 要求  
 **标头：**collection.h  
  
 **命名空间：**Platform::Collections
