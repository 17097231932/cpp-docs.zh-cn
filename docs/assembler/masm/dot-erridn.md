# <a name="perioderridn-perioderridni"></a>&period;.ERRIDN，&period;ERRIDNI

如果文本项相同，则生成错误。

## <a name="syntax"></a>语法

> **..Erridn** *textitem1* __、__ *textitem2* ⟦ __、__ *message*⟧ \
> **.ERRIDNI** *textitem1* __、__ *textitem2* ⟦ __、__ *message*⟧

## <a name="remarks"></a>备注

如果给定 **&period;ERRIDNI** ，则比较不区分大小写。
