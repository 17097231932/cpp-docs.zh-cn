# <a name="macro"></a>MACRO
标记调用一个宏块*名称*并建立*参数*时宏调用传递自变量的占位符。  
  
## <a name="syntax"></a>语法  
  
```  
  
   name MACRO [[parameter [[:REQ | :=default | :VARARG]]]]...  
statements  
ENDM [[value]]  
```  
  
## <a name="remarks"></a>备注  
 宏函数返回*值*于调用语句。
