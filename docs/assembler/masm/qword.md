# QWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化 8 个字节每 `initializer`的存储。  可以用作类型说明符任何位置类型是非法的。  
  
## 语法  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
