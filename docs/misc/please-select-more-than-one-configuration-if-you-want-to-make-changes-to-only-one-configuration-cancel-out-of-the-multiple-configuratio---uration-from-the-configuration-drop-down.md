# 请选择多个配置。 如果你希望仅对一个配置进行更改，请取消“多重配置”对话框并从配置下拉菜单中选择该配置。
当已在“多重配置”对话框中选择了单个解决方案配置时，通常会发生此错误。  
  
### 更正此错误  
  
1.  选择其他解决方案配置。  
  
     \- 或 \-  
  
2.  如果你希望更新单个配置，请选择“取消”并从“解决方案属性”对话框中的“配置”列表中选择正确的解决方案配置。
