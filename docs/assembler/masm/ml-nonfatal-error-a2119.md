# <a name="ml-nonfatal-error-a2119"></a>ML 非致命错误 A2119

**必须指定语言类型**

没有为过程定义或原型提供语言类型。

如果未指定默认语言类型，则必须在每个过程定义或原型中声明语言类型。 默认语言类型是使用设置的 [。MODEL](dot-model.md) 指令、 **OPTION LANG** 或 ML 命令行选项 **/Gc** 或 **/gd**。
