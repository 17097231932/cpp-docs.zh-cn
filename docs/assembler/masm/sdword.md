# <a name="sdword"></a>SDWORD

分配每个初始值设定项，并选择性地为每个 *初始值设定项* 初始化 (4) 字节的有符号双字。 还可在类型合法的任何位置用作类型说明符。

## <a name="syntax"></a>语法

> *名称* **SDWORD** *初始值设定项* ⟦__，__ *初始值设定项* .。。⟧
