# <a name="fatal-error-c1013"></a>错误 C1013

编译器限制 : 左括号太多

表达式在单个表达式中包含太多级别的括号。 简化表达式，或将其分解为多个语句。

在 Visual C++ 6.0 Service Pack 3 之前，单个表达式中嵌套括号的上限是 59 个。 目前，嵌套括号的上限是 256 个。
