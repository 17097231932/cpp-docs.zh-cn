# <a name="expression-evaluator-error-cxx0022"></a>表达式计算器错误 CXX0022

_main 前的函数调用

C 表达式计算器在调试器进入函数 **_main**之前无法计算函数。 调用 **_main**之前，程序未正确初始化。

此错误与 CAN0022 相同。
