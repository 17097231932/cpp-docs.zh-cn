# <a name="compiler-error-c2773"></a>编译器错误 C2773

\#导入和 #using 仅在C++编译器中可用

C 编译器无法识别 `#import` 预处理器指令。 将源编译为C++。 如有必要，请使用[/tp](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) 。
