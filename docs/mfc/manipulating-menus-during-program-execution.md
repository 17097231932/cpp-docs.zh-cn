# <a name="manipulating-menus-during-program-execution"></a>在程序执行过程中操作菜单
使用类 `CMenu` 即时操作菜单和菜单项。 `CMenu` 将封装 Windows `HMENU` 句柄并提供与菜单一起使用的成员函数。  
  
 请参阅类概述[CMenu](../mfc/reference/cmenu-class.md)有关详细信息。
