# <a name="adding-an-mfc-odbc-consumer"></a>添加 MFC ODBC 使用者

MFC ODBC 使用者包含 ODBC 记录集类和数据绑定访问数据源所必需。

### <a name="to-add-an-mfc-odbc-consumer"></a>若要添加 MFC ODBC 使用者

1. 在类视图中，右键单击该项目。 在快捷菜单上，单击**外**，然后单击**添加类**。

1. 在 Visual c + + 文件夹中，双击**MFC ODBC 使用者**图标或选择它，然后单击**打开**。

     MFC ODBC 使用者向导随即打开。

1. 定义设置，如中所述[MFC ODBC 使用者向导](../../mfc/reference/mfc-odbc-consumer-wizard.md)。

1. 单击**完成**以关闭向导，将在你的项目中插入新创建的 ODBC 使用者代码。
