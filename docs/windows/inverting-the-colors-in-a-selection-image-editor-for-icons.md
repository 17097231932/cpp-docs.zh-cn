# Inverting the Colors in a Selection (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

图像编辑器提供在图像选定部分进行反色操作的便利方法，这样一来，你就能够了解使用反色时图像将如何显示。  
  
### 在当前选定内容中进行反色操作  
  
1.  在**“图像”**菜单上，单击**“反色”**。
