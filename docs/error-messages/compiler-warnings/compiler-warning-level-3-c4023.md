# <a name="compiler-warning-level-3-c4023"></a>编译器警告（等级 3）C4023

“symbol”：基指针传递到非原型函数：参数数目

将基指针传递给非原型函数会导致该指针被正常化，并产生不可预知的结果。

如果你使用传递基指针的原型函数，则可能会修复此警告。
