# clock 函数 (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.12.2.1** `clock` 函数的纪元  
  
 当 C 程序开始执行时，`clock` 函数的纪元将以值 0 开始。  它返回以 1**CLOCKS\_PER\_SEC**（等于 Microsoft C 的 1\/1000）度量的时间。
