# ATL 窗口类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL包括允许您使用和实现窗口的几选件类。  这些选件类，与其他ATL选件类，提供不会开销使您的代码的一个有效的实现。  
  
 本节描述ATL窗口类并说明如何使用它们。

 [使用窗口](../atl/using-a-window.md)

 [实现对话框](../atl/implementing-a-dialog-box.md)  
 讨论添加的对话框选件类两个方法并显示代码示例。  
  
 [使用中包含的Windows](../atl/using-contained-windows.md)  
 在ATL讨论包含窗口，该窗口将其消息为容器对象而不是处理他们的选件类。
