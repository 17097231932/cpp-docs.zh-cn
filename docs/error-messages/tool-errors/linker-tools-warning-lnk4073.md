# 链接器工具警告 LNK4073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法创建 .ILK 文件的映射；正在非增量链接  
  
 共享内存中没有足够大的连续空间供 LINK 创建增量状态 \(.ilk\) 文件。  LINK 执行了非增量生成。
