# 资源编译器错误 RW1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

创建资源名称时出错  
  
 未能创建指定的 .res 文件。  确保不是在只读驱动器上创建它。  使用 \/V 选项查看是否正在创建该文件。
