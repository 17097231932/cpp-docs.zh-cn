# 链接器工具错误 LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效或损坏的文件  
  
 输入文件具有损坏的头，或大小为零或者异常小。  
  
 如果该文件被错误指定为附加依赖项，则可能也会导致 LNK1136。
