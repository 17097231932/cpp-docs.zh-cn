# <a name="compiler-warning-level-1-c4997"></a>编译器警告（等级 1）C4997
“class”：组件类不实现 COM 接口或伪接口  
  
 以 [coclass](../../windows/coclass.md) 特性标记的类不实现接口。  
  
 以下示例生成 C4997：  
  
```  
// C4997.cpp  
// compile with: /WX  
// to resolve this C4997, uncomment all code  
#include <objbase.h>  
  
[ object ]  
__interface I {  
   HRESULT func();  
};  
  
[ coclass ]  
struct C /*: I*/ {  
   /*  
   HRESULT func() {  
      return S_OK;  
   }  
   */  
};   // C4997  
```
