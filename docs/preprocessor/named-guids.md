# <a name="named_guids-import-attribute"></a>named_guids 导入属性

**C++ 专用**

指示编译器以旧样式定义和初始化 GUID 变量，格式为 `LIBID_MyLib` 、、 `CLSID_MyCoClass` `IID_MyInterface` 和 `DIID_MyDispInterface` 。

## <a name="syntax"></a>语法

> **#import** *类型-库* **named_guids**

**结束 C++ 专用**
