# <a name="messages-and-commands-in-the-framework"></a>框架中的消息和命令

为 Microsoft Windows 编写应用程序是"驱动的消息。" 以响应鼠标单击、 击键、 窗口移动等事件，Windows 将消息发送到适当的窗口。 Framework 应用程序处理 Windows 消息类似于 Windows 的任何其他应用程序。 但该框架还提供一些增强功能，使处理消息更轻松、 更可维护性、 更好地封装。
