# <a name="substr"></a>SUBSTR

返回字符串的子*textitem*，开始时*位置*。 *Textitem*可以是文字字符串，前面有一个常量`%`，或通过宏函数返回的字符串。

## <a name="syntax"></a>语法

> 命名 SUBSTR textitem，位置 [[，length]]
