# 编译器警告（等级 4）C4019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

全局范围内的空语句  
  
 全局范围内的分号前面没有语句。  
  
 如果删除额外的分号，则可能会修复此警告。  
  
## 示例  
  
```  
// C4019.c // compile with: /Za /W4 #define declint( varname ) int varname; declint( a );   // C4019, int a;; declint( b )   // OK, int b; int main() { }  
```
