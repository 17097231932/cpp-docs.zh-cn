# ML Nonfatal Error A2137
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**调用的参数太少**  
  
 使用 [调用](../../assembler/masm/invoke.md) 指令传递的参数数目小于在原型中指定所需参数的数目是对调用的过程。
