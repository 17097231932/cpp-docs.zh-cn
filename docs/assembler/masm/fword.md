# <a name="fword"></a>FWORD

分配并根据需要为每个初始化 6 个字节存储`initializer`。

## <a name="syntax"></a>语法

> [[name]]FWORD 初始值设定项 [[，初始值设定项]]...

## <a name="remarks"></a>备注

此外可以用作类型说明符类型是合法的任何位置。
