# <a name="compiler-error-c3721"></a>编译器错误 C3721

"签名"：事件的签名不兼容

事件声明不正确。 有关详细信息，请参阅 [__event](../../cpp/event.md)。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3721。
