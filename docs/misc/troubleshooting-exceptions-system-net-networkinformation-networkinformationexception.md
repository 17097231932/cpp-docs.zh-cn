# 关于异常的疑难解答：System.Net.NetworkInformation.NetworkInformationException
因检索网络信息时发生错误而引发的异常。  
  
## 备注  
 <xref:System.Net.NetworkInformation> 命名空间中的类型会在对 Win32 函数的调用失败时引发此异常。<xref:System.Net.NetworkInformation.NetworkInformationException.ErrorCode%2A> 属性包含该函数返回的结果。
