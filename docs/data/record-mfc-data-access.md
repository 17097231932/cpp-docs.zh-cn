# 记录（MFC 数据访问）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

记录是指收集有关单个实体（例如单个帐户、用户）的数据，存储在表格中（表格行）。  一条记录由一组连续列（有时称为字段）组成，包含各种类型数据。  从数据源中选择的一组记录\(在数据库术语中通常称为结果集\)在 MFC 中称为记录集。  有关更多信息，请参阅[记录集 \(ODBC\)](../data/odbc/recordset-odbc.md)。
