# 编译器错误 C2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“locale”: 区域设置无效  
  
 指定的区域设置无效。  有关所支持的区域设置，请参见“运行库参考”中的[语言和国家\/地区字符串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
