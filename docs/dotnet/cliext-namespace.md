# <a name="cliext-namespace"></a>cliext 命名空间

`cliext` 命名空间包含 STL/CLR 库的所有类型。 有关所有这些类型的列表以及指向 STL/CLR 类型的详细信息的链接，请参阅 [stl/Clr 库参考](../dotnet/stl-clr-library-reference.md)。
