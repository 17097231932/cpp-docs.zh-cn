# <a name="endpropertyset"></a>END_PROPERTY_SET
将标记属性集的末尾。  
  
## <a name="syntax"></a>语法  
  
```cpp
END_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>参数  
 `guid`  
 [in]属性的 GUID 中。
