# 最大数组大小
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.3.4, 4.1.1**整数的类型需要保留数组的最大大小，也就是，“size\_t”的大小  
  
 `size_t` 类型定义是32位x86平台上的一个`unsigned int` 在64位平台上， `size_t` 类型定义是一个**无符号\_\_int64**
