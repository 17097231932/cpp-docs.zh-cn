# <a name="nmake-fatal-error-u1078"></a>NMAKE 错误 U1078

"expression" 处的常量溢出

给定表达式包含超出范围-2147483648 到2147483647的常数。 常数出现在下列情况之一：

- 使用预处理指令指定的表达式

- 使用短划线 (**-**) 命令修饰符指定的错误级别
