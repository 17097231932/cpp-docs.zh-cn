# <a name="linker-tools-error-lnk1248"></a>链接器工具错误 LNK1248

> 图像大小 ( "*size*" ) 超过允许的最大大小 (80000000) 

链接器确定输出文件的大小将超过32位程序映像可能的最大大小。 可能需要将程序转换为多个 Dll。
