# <a name="_rmtmp"></a>_rmtmp

删除临时文件。

## <a name="syntax"></a>语法

```C

int _rmtmp( void );
```

## <a name="return-value"></a>返回值

**_rmtmp** 返回已关闭和删除的临时文件的数目。

## <a name="remarks"></a>备注

**_Rmtmp** 函数将清除当前目录中的所有临时文件。 函数仅删除由 **tmpfile** 创建的文件;仅在创建临时文件的相同目录中使用它。

默认情况下，此函数的全局状态的作用域限定为应用程序。 若要更改此项，请参阅 [CRT 中的全局状态](../global-state.md)。
