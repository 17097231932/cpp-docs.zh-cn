# <a name="filenames"></a>文件名

**ANSI 4.9.3** 撰写有效文件名的规则

文件规范可能包含可选的驱动器号（始终后跟冒号）、一系列可选的目录名称（用斜杠分隔）和文件名。

有关详细信息，请参阅[为文件命名](/windows/desktop/FileIO/naming-a-file)。
