# 关于异常的疑难解答：System.ServiceModel.ProtocolException
当由于数据传输协议不匹配而不能与远程方进行通信时，将在客户端上发生 <xref:System.ServiceModel.ProtocolException> 异常。  
  
## 备注  
 终结点之间的绑定不兼容、使用不正确的远程终结点地址或类似的不正确配置都会导致 <xref:System.ServiceModel.ProtocolException>。
