# <a name="comment"></a>COMMENT

将与分隔符相同的行中的所有*文本*视为注释。

## <a name="syntax"></a>语法

> **注释***分隔符*⟦*text*⟧ \
> ⟦*text*⟧ \
> ⟦*text*⟧*分隔符*⟦*text*⟧
