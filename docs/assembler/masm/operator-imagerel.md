# <a name="operator-imagerel"></a>运算符 IMAGEREL

返回*表达式*的相对偏移量。

## <a name="syntax"></a>语法

> **IMAGEREL** *表达式*

## <a name="remarks"></a>备注

生成的值通常称为 RVA 或相对虚拟地址。

IMAGEREL 仅可用于显示 COFF 对象。
