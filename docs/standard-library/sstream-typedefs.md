##  <a name="istringstream"></a>  istringstream  
 创建专用于 `char` 模板参数的类型 `basic_istringstream`。  
  
```

```

 此类型是模板类 [basic_istringstream](../standard-library/basic-istringstream-class.md) 的同义词，专用于 `char` 类型的元素。
