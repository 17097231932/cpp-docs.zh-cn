# CA0055
未能加载文件: :\<path\>  
  
 代码分析工具无法加载为分析功能指定的目标文件。  如果找不到该文件或者对该文件的访问被拒绝，则会出现此警告。  通常，如果用户选择重写 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 中的输入程序集属性，或者手动调用 fxcopcmd.exe，将会出现此警告。
