# Some Unicode characters in this file will not be saved.
该错误通常发生在文件包含仅可通过应用 Unicode 代码页保存的字符时。  
  
### 保存带有 Unicode 字符的文件  
  
1.  单击**“取消”**。  
  
2.  从**“文件”**菜单中选择**“高级保存选项”**。  
  
3.  在**“高级保存选项”**对话框中，选择适合于文件的编码。
