# ATL 支持 DHTML 控件
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用ATL，可以创建控件以动态HTML \(dhtml\)功能。  ATL DHTML控件:  
  
-   承载webbrowser控件。  
  
-   使用HTML，指定，则，DHTML控件的用户界面\(UI）。  
  
-   访问浏览器对象及其方法通过其接口，[IWebBrowser2](https://msdn.microsoft.com/en-us/library/aa752127.aspx)。  
  
-   管理C\+\+代码和HTML之间的通信。  
  
 DHTML控件类似于其他ATL控件，除此之外，DHTML控件包括其他的调度接口。  为默认DHTML项目提供的接口的插图参见。[标识DHTML控件项目的元素](../atl/identifying-the-elements-of-the-dhtml-control-project.md) 图形。  
  
 可以在浏览器中查看ATL DHTML控件或其他容器，如ActiveX控件测试容器。
