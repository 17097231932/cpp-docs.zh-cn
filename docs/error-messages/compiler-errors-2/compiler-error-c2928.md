# 编译器错误 C2928
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

显式实例化; “identifier”不是模板类“class”的函数或静态数据成员  
  
 无法显式实例化 `class`中不为函数或 `static` 变量的成员。
