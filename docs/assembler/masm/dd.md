# <a name="dd"></a>DD

为每个*初始值设定*分配和（可选）初始化双字（4字节）的存储。 **DD**是[DWORD](dword.md)的同义词。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **DD** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧

## <a name="remarks"></a>备注

在类型合法的任何位置，还可以用作类型说明符。
