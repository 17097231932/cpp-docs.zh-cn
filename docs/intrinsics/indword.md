# <a name="indword"></a>__indword
**Microsoft 专用**  
  
 从指定的端口使用读取的数据的一个双字`in`指令。  
  
## <a name="syntax"></a>语法  
  
```  
unsigned long __indword(  
   unsigned short Port  
);  
```  
  
#### <a name="parameters"></a>参数  
 [in] `Port`  
 要读取的端口。  
  
## <a name="return-value"></a>返回值  
 从端口读取内容单词。
