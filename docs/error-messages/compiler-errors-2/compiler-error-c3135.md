# <a name="compiler-error-c3135"></a>编译器错误 C3135

"property"：属性不能有 "const" 或 "volatile" 类型

属性不允许使用 [const](../../cpp/const-cpp.md) 和 [volatile](../../cpp/volatile-cpp.md) 关键字。
