# <a name="resource-compiler-fatal-error-rc1203"></a>资源编译器错误 RC1203

指定的十六进制默认语言 ID 无效。

指定默认语言（/l）选项后跟无效的十六进制语言 ID。 有关有效的语言 Id 的列表，请参阅《*运行时库参考*中的[语言和国家/地区字符串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
