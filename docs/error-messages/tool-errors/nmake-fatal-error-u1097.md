# <a name="nmake-fatal-error-u1097"></a>NMAKE 错误 U1097

文件名部分语法需要依赖项

当前依赖项没有显式依赖项或隐式依赖项。 文件名部分语法使用百分号（`%`）说明符，表示当前目标的第一个依赖项的组成部分。
