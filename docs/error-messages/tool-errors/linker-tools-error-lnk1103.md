# <a name="linker-tools-error-lnk1103"></a>链接器工具错误 LNK1103
调试信息损坏;重新编译模块  
  
 可以导致此错误，因为之前已创建有效的对象文件编译被终止。  
  
 重新编译给定的对象文件。 如果重新编译不能解决此问题，请注意错误的情况下，尝试隔离问题，并创建可重现的测试用例。 请查阅该技术支持帮助文件或你手册之一中的技术支持部分。
