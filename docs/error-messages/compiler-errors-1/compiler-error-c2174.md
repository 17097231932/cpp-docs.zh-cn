# <a name="compiler-error-c2174"></a>编译器错误 C2174

"function"：实参的类型为 "void"：参数 1-/，形参列表数字2

传递给参数列表的参数 `number1` `number2` 是 `void` 参数。 参数的类型不能 `void`。 请改用 `void*`。
