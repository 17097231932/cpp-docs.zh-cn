# BEGIN_PARAM_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

标记参数映射项的开始。  
  
## 语法  
  
```  
  
BEGIN_PARAM_MAP(  
x  
 )  
  
```  
  
#### 参数  
 *x*  
 \[in\] 用户记录类的名称。  
  
## 备注  
 [命令](https://msdn.microsoft.com/en-us/library/ms724608.aspx)使用参数。  
  
## 示例  
 针对 [BEGIN\_COLUMN\_MAP](../../data/oledb/begin-column-map.md) 宏参见示例。  
  
## 要求  
 **标头:** atldbcli.h
