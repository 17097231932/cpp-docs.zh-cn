# operator .
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

第一个运算符返回 *表达式* 以及按其结构或联合中的 *字段* 。  第二个运算符在位置返回值指向由 *注册* 和按其结构或联合中的 *字段* 。  
  
## 语法  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
