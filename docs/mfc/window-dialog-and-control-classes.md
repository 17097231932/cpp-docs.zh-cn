# 窗口、对话框和控件类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`CWnd` 类，它派生类封装 `HWND`，一个窗口的句柄。  `CWnd` 可以使用单独或作为的基础。派生新类。  类库提供的派生类表示各种窗口。  
  
 [CWnd](../mfc/reference/cwnd-class.md)  
 所有窗口的基类。  可以使用从 `CWnd` 派生的某个类直接或从中派生自己的类。
