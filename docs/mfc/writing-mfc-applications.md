# <a name="writing-mfc-applications"></a>编写 MFC 应用程序
此文章介绍了执行开发应用程序的初始步骤。 首先，你必须决定哪种应用程序正在编写。 多个选项已中所述[应用程序设计选择](../mfc/application-design-choices.md)。 将为你的应用程序：  
  
-   在 Internet 或 intranet 上运行  
  
-   在客户端或服务器上运行  
  
-   运行在浏览器或作为独立的应用程序  
  
-   使用 COM 或 Active 技术  
  
-   下载使用 WinInet 或异步名字对象的数据  
  
 决策确定适用于你的应用程序的类。 你回答还帮助确定在运行应用程序向导，以开始构建您的应用程序时所做的选择。  
  
 有关 Internet 应用程序做出初始设计决策后，你可以使用应用程序向导开始。 使用应用程序向导创建主干应用程序并修改此代码，如以下文章中所述：  
  
-   ActiveX 控件，请参阅[Internet 上的 ActiveX 控件](../mfc/activex-controls-on-the-internet.md)。  
  
-   活动文档，请参阅[Internet 上的活动文档](../mfc/active-documents-on-the-internet.md)。  
  
 以下文章还介绍了如何帮助你开始编程任务：  
  
-   [应用程序设计选项](../mfc/application-design-choices.md)  
  
-   [Internet 上的异步名字对象](../mfc/asynchronous-monikers-on-the-internet.md)  
  
-   [WinInet 基础知识](../mfc/wininet-basics.md)
