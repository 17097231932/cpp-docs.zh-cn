# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
找不到选定的外接程序。  该错误通常发生在选定的外接程序缺少 .dll 或其他必要的组件时。  
  
### 更正此错误  
  
1.  重新安装该外接程序。
