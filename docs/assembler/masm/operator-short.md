# <a name="operator-short"></a>运算符 SHORT

将*标签*的类型设置为 short。 所有带*标签*的跳转必须为 short （范围为-128 到 + 127 字节，范围为-到 + *）。*

## <a name="syntax"></a>语法

> **短***标签*
