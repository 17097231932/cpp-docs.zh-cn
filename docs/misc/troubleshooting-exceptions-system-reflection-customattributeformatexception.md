# 关于异常的疑难解答：System.Reflection.CustomAttributeFormatException
当自定义特性的二进制格式无效时引发的异常。  
  
## 备注  
 导致问题的原因可能是用于创建引发 bug 的该数据类型的工具存在缺陷。
