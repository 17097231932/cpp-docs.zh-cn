# <a name="resources-for-creating-a-c-game-using-directx"></a>使用 DirectX 创建C++游戏的资源

若要创建最佳桌面游戏，请结合使用 Visual C++ 和 DirectX。

## <a name="resources"></a>资源

Visual C++ Studio 的所有版本中都包含视觉对象, 但默认情况下不会安装它。 有关如何安装 Visual Studio 和C++开发工具的详细信息, 请参阅[在 Visual studio 中安装C++支持](../build/vscpp-step-0-installation.md)。

对于 Windows 8.1 和更高版本, DirectX SDK 包含在 Windows SDK 中。 不必下载任何其他内容。 有关详细信息, 请参阅[DIRECTX SDK 在哪里？](/windows/win32/directx-sdk--august-2009-)。

若要创建适用于 Windows 7、Windows Vista 或 Windows XP 的 DirectX 应用程序, 可以下载独立的[DIRECTX SDK](https://www.microsoft.com/download/details.aspx?displaylang=en&id=6812)。

[Directx 图形和游戏](/windows/win32/directx)提供了信息和示例, 可帮助你开始进行 DirectX 编程。

[Win32 Windows 应用程序](../windows/windows-desktop-applications-cpp.md)包含描述如何创建基本 windows 桌面应用程序的文章。

[开发游戏](/windows/uwp/gaming/getting-started)讨论通用 Windows 平台游戏开发, 但是许多概念也适用于桌面游戏。
