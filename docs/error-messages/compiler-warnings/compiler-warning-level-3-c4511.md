# 编译器警告（等级 3）C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“class”: 未能生成复制构造函数  
  
 编译器未能为类生成默认的复制构造函数；基类可以具有私有复制构造函数。
