# <a name="using-frame-windows"></a>使用框架窗口

MFC 框架会创建文档框架窗口 — 及其视图和文档 — 作为其实现的新增和文件菜单打开命令的一部分。 因为框架为您完成大部分框架窗口工作，也只创建、 使用和销毁这些窗口中的小角色。 可以但是，显式创建自己的框架窗口和子窗口的特殊用途。
