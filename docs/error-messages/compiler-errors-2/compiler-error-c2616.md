# <a name="compiler-error-c2616"></a>编译器错误 C2616
conversion： 无法将非-左值 type1 到 type2 隐式转换，不是常量  
  
 无法从非左值初始化引用。  
  
 这是在 ANSI 兼容性错误 ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) 和在 Microsoft 扩展下的警告 (**/Ze**)。
