# <a name="cmultipleresults-class"></a>CMultipleResults 类

如果需要命令处理多个结果集，请将用作 `CMultipleResults` `CCommand` 模板参数 *TMultiple*。

## <a name="syntax"></a>语法

```cpp
class CMultipleResults
```

## <a name="remarks"></a>备注

若要处理多个结果集， [CCommand](../../data/oledb/ccommand-class.md) 必须从此类继承。

## <a name="requirements"></a>要求

**标头：** atldbcli。h
