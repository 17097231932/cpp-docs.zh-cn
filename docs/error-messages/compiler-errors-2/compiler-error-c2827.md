# <a name="compiler-error-c2827"></a>编译器错误 C2827

不能用一元形式全局重写 "operator operator"

运算符不能在对象的外部具有一元形式。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>使用以下可能的解决方案进行修复

1. 使重载运算符成为对象的局部变量。

1. 选择要重载的适当一元运算符。
