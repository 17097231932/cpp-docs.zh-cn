# Command was not specified.
该错误通常发生在向**“命令”**窗口或**“查找\/命令”**框中输入了空字符串时。  
  
### 更正此错误  
  
1.  输入一个有效命令。  有关详细信息，请参阅 [Visual Studio 命令](../Topic/Visual%20Studio%20Commands.md)。
