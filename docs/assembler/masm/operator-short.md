# <a name="operator-short"></a>运算符 SHORT

设置类型*标签*为 short。 所有跳转到*标签*必须很短 (介于-128 到 + 127 个字节从跳转指令到*标签*)。

## <a name="syntax"></a>语法

> 短标签
