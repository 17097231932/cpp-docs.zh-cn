# “文本工具”对话框（图标的图像编辑器）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

使用“文本工具”对话框将文本添加到光标、位图或图标资源。  
  
 要访问此对话框，请打开[图像编辑器](../mfc/window-panes-image-editor-for-icons.md)。  从**“图像”**菜单选择**“工具”**，然后选择**“文本工具”**命令。  
  
 **“字体”按钮**  
 打开[“文本工具字体”对话框](../mfc/text-tool-font-dialog-box-image-editor-for-icons.md)，在其中可以更改光标字体的字体、样式或大小。  更改将应用于在“正文”区域中显示的文本。  
  
 **“正文”区域**  
 显示作为资源的一部分出现的文本。  此区域最初为空。  
  
> [!NOTE]
>  如果设置了**“透明背景”**，则将只在图像中放入文本。  如果设置“不透明背景”，则一个用[背景色](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md)填充的边框将被放在文本的后面。  有关更多信息，请参见[选择透明或不透明背景](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md)。  
  
 可以在“文本工具”对话框上右击以访问默认的快捷菜单，此快捷菜单包含标准 Windows 命令的列表。
