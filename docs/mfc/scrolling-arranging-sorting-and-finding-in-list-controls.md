# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>在列表控件中滚动、排列、排序和查找

默认情况下, 列表控件 ([CListCtrl](../mfc/reference/clistctrl-class.md)) 是可滚动的。 有关详细信息, 请参阅 Windows SDK 中的[滚动位置](/windows/win32/Controls/using-list-view-controls)和[滚动](../mfc/reference/clistctrl-class.md#scroll)成员函数。

您可以调用`CListCtrl`成员函数来排列控件中的列表项、排序项以及查找特定项。 有关详细信息, 请参阅在 Windows SDK 中[使用 ListView 控件](/windows/win32/Controls/using-list-view-controls)和[CListCtrl](../mfc/reference/clistctrl-class.md)成员[排列](../mfc/reference/clistctrl-class.md#arrange)、 [SortItems](../mfc/reference/clistctrl-class.md#sortitems)和[FindItem](../mfc/reference/clistctrl-class.md#finditem)。
