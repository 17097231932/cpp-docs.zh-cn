# Grouped expression is missing &#39;)&#39;.
该错误通常发生在在查找字符串中指定了左括号 `(` 却遗漏了右括号 `)` 时。  
  
### 更正此错误  
  
1.  若要搜索原义字符 `(`，请使用 `\(`。  
  
2.  若要对表达式分组，请输入缺少的 `)`。
