# <a name="compiler-error-c3818"></a>编译器错误 C3818

数组属性声明 "property1" 不应重载索引属性 "property2"

如果某个属性是索引器，而另一个是数组属性，则不能使用重载。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3818。
