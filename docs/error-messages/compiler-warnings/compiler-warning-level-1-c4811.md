# 编译器警告（等级 1）C4811
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

杂注 conform\(forScope, show\) 的值 \=\= value  
  
 你使用 [conform](../../preprocessor/conform.md) 杂注的**“显示”**选项，将发出此警告。*value* 为当前的 conform 值。
