# “自定义颜色选择器”对话框（图标的图像编辑器）
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

此对话框允许您自定义用于图像的颜色。  有关更多信息，请参见[自定义或更改颜色](../windows/customizing-or-changing-colors-image-editor-for-icons.md)。  
  
 **渐变颜色显示**  
 更改选定颜色的值。  将十字线置于要更改的颜色上。  然后上下移动滑块更改颜色的亮度或 RGB 值。  
  
 **光度条**  
 为在“渐变颜色显示”框中选择的颜色设置亮度。  单击并沿该栏向上拖动白色箭头以获得更高的亮度，或向下拖动以降低亮度。  注意“颜色”框显示已选定的颜色和所设置亮度的影响。  
  
 **Color**  
 列出正在定义的颜色的色调（色轮值）。  值的范围为从 0 到 240，其中 0 代表红色，60 代表黄色，120 代表绿色，180 代表青色，200 代表洋红色，240 代表蓝色。  
  
 **色调**  
 列出正在定义的颜色的色调（色轮值）。  值的范围为从 0 到 240，其中 0 代表红色，60 代表黄色，120 代表绿色，180 代表青色，200 代表洋红色，240 代表蓝色。  
  
 **饱和度**  
 指定正在定义的颜色的饱和度值。  饱和度是指定色调中的颜色数量。  值的范围为从 0 到 240。  
  
 **亮度**  
 列出正在定义的颜色的亮度。  值的范围为从 0 到 240。  
  
 **Red**  
 指定正在定义的颜色的红色值。  值的范围为从 0 到 255。  
  
 **绿色**  
 指定正在定义的颜色的绿色值。  值的范围为从 0 到 255。  
  
 **蓝色**  
 指定正在定义的颜色的蓝色值。  值的范围为从 0 到 255。
