# <a name="handling-windows-messages-in-your-dialog-box"></a>处理您对话框中的 Windows 消息

对话框是窗口，因此，如果您提供了合适的处理程序函数，对话框就能处理 Windows 消息。 使用 [类向导](reference/mfc-class-wizard.md)创建对话框类时，向导会向类添加一个空消息映射。 使用向导可映射您希望类处理的任何 Windows 消息或命令。

有关详细信息，请参阅 [将 Windows 消息映射到对话框类](mapping-windows-messages-to-your-class.md) 。
