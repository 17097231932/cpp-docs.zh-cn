# 数字限制 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

这两个标准包含文件（LIMITS.H 和 FLOAT.H）定义了数字限制或给定类型的变量可包含的最小值和最大值。  确保使用与 ANSI C 相同的数据表示形式的任何 C\+\+ 编译器可以移植这些最小值和最大值。  LIMITS.H 包含文件定义[整型的数字限制](../cpp/integer-limits.md)，而 FLOAT.H 定义[浮点型的数字限制](../cpp/floating-limits.md)。
