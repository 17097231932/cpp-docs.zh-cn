# 编译器错误 C2207
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“member”：类模板的成员无法获取函数类型  
  
 类模板的 `member` 以前被分析为非静态数据成员。  不能将它重新定义为成员函数。
