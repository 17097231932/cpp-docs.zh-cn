# <a name="linker-tools-warning-lnk4199"></a>链接器工具警告 LNK4199

/DELAYLOAD：已忽略 dllname;找不到 dllname 中的导入

链接器将忽略 `dllname`，因为它不需要 `dllname` 导出的任何函数。
