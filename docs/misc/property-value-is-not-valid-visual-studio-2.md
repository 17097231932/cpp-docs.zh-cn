# Property value is not valid. (Visual Studio) (2)
该错误通常发生在向属性输入了无效值时。  例如，输入了整型值的错误格式或对短型而言太大的值。  
  
### 更正此错误  
  
1.  输入该属性的有效值。  
  
    > [!TIP]
    >  搜索联机文档以查找该属性的有效值。
