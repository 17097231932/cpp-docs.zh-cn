# NMAKE 错误 U1052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未找到文件“filename”  
  
 NMAKE 未能找到用下列中的某一项指定的文件：  
  
-   **\/F** 选项  
  
-   **\!INCLUDE** 预处理指令  
  
-   响应文件的 At 符 \(**@**\) 说明符  
  
 检查文件是否存在及文件名是否拼写正确。
