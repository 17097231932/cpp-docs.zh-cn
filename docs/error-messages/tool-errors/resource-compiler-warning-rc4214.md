# 资源编译器警告 RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

代码页无效 : 被忽略  
  
 .rc 文件包含代码页参数而指定的代码页无效。  有关更多信息，请参见 [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674)。
