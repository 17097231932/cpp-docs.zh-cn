# <a name="compiler-error-c2487"></a>编译器错误 C2487

"identifier"： dll 接口类的成员不能用 dll 接口声明

您可以使用 DLL 接口声明整个类或非 DLL 接口类的某些成员。 不能使用 DLL 接口声明类，然后使用 DLL 接口声明该类的成员。
