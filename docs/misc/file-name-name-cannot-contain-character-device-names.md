# File name &lt;name&gt; cannot contain character device names.
该错误通常发生在输入的文件名实际上是字符设备名时。  字符设备名包括但不限于 CON、PRN、AUX、CLOCK$、NUL、LPT1、LPT2、COM1、COM2、COM3 和 COM4。  
  
### 更正此错误  
  
1.  输入不同的文件名。
