# <a name="compiler-error-c2459"></a>编译器错误 C2459
identifier： 正在定义;无法将添加为匿名成员  
  
 类、 结构或联合内重新定义其自己的范围由匿名联合的成员。  
  
 下面的示例生成 C2459:  
  
```  
// C2459.cpp  
// compile with: /c  
class C {  
   union { int C; };   // C2459  
   union { int D; };  
};  
```
