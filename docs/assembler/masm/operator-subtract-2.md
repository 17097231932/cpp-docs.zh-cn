# <a name="operator--"></a>运算符 -

返回 *表达式* 2 减 *表达式*。 第二个运算符反转 *表达式* 的符号。

## <a name="syntax"></a>语法

> *expression1* **-** *expression2*
>
> __-__*表达式*
