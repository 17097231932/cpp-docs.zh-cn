# File name or default path is too long.
当尝试创建代码注释 Web 报告但“生成注释网页”对话框的“Web 页保存在”框中指定的文件名或路径太长时，通常会发生此错误。  
  
### 更正此错误  
  
1.  指定少于 260 个字符的文件名或路径。
