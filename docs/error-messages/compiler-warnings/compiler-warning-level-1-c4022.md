# 编译器警告（等级 1）C4022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 指针与实参“number”不匹配  
  
 实参的指针类型与相应的形参的指针类型不同。  实参传递时未经更改。
