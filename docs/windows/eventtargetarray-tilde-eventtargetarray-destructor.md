# <a name="eventtargetarrayeventtargetarray-destructor"></a>EventTargetArray::~EventTargetArray 析构函数
支持 WRL 基础结构，不应在代码中直接使用。  
  
## <a name="syntax"></a>语法  
  
```  
~EventTargetArray();  
```  
  
## <a name="remarks"></a>备注  
 取消初始化当前`EventTargetArray`类。  
  
## <a name="requirements"></a>要求  
 **标头：** event.h  
  
 **Namespace:** Microsoft::WRL::Details
