# <a name="named_guids-import-attribute"></a>named_guids 导入属性

**C++相关**

指示编译器以旧样式定义和初始化 GUID 变量`LIBID_MyLib`, 格式为`IID_MyInterface`、 `CLSID_MyCoClass`、和`DIID_MyDispInterface`。

## <a name="syntax"></a>语法

> **#import***类型库***named_guids**

**结束C++特定**
