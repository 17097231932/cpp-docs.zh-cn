# <a name="setting-the-dialog-boxs-background-color"></a>设置对话框的背景色

您可以通过处理对话框窗口 WM_CTLCOLOR 消息来设置对话框的背景色。 您设置的颜色仅用于指定的对话框。

有关示例，请参阅 [codexpert 博客](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) 。
