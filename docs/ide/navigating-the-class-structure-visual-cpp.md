# <a name="navigating-the-class-structure-visual-c"></a>导航类结构 (Visual C++)
您可以导航项目类结构，包括函数、 继承和接口使用[类视图](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925)或[对象浏览器](http://msdn.microsoft.com/en-us/f89acfc5-1152-413d-9f56-3dc16e3f0470)。  
  
 通过双击类视图中的函数名称，可以转到函数声明。 通过双击类视图的类名称上，你可以转到类定义。
