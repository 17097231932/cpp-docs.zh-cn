# <a name="expression-evaluator-error-cxx0066"></a>表达式计算器错误 CXX0066
静态成员不存在  
  
 类的静态成员未找到或未定义。 此错误可能是由于从静态类而不声明的成员定义，或仅定义，并引用不包含调试信息的模块中。  
  
 此错误是与 CAN0066 相同。
