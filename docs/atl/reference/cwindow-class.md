# <a name="cwindow-class"></a>CWindow 类

此类提供用于操作窗口的方法。

> [!IMPORTANT]
>  此类及其成员不能用于在 Windows 运行时中执行的应用程序。

## <a name="syntax"></a>语法

```
class CWindow
```

|[CWindow：： m_hWnd](#m_hwnd)|与 `CWindow` 对象关联的窗口的句柄。|
|[CWindow：： rcDefault](#rcdefault)|包含默认窗口尺寸。|

## <a name="remarks"></a>备注

`CWindow` 提供了在 ATL 中操作窗口的基本功能。 许多 `CWindow` 方法只是包装一个 Win32 API 函数。 例如，比较 `CWindow::ShowWindow` 和 `ShowWindow`的原型：

|CWindow 方法|Win32 函数|
|--------------------|--------------------|
|**BOOL ShowWindow （int** `nCmdShow` **）;**|**BOOL ShowWindow （HWND** `hWnd` **，int** `nCmdShow` **）;**|

`CWindow::ShowWindow` 通过将 `CWindow::m_hWnd` 作为第一个参数传递来调用 Win32 函数 `ShowWindow`。 直接包装 Win32 函数的每个 `CWindow` 方法都传递 `m_hWnd` 成员;因此，很多 `CWindow` 文档将向您介绍 Windows SDK。

> [!NOTE]
>  并非每个与窗口相关的 Win32 函数都由 `CWindow`包装，并且不是每个 `CWindow` 方法都包装 Win32 函数。

`CWindow::m_hWnd` 存储用于标识窗口的 HWND。 当你执行以下操作时，会将 HWND 附加到你的对象：

- 在 `CWindow`的构造函数中指定 HWND。

- 调用 `CWindow::Attach`。

- 使用 `CWindow`的**operator =** 。

- 使用派生自 `CWindow`的以下类之一创建或划分窗口的子类：

[CWindowImpl](../../atl/reference/cwindowimpl-class.md)允许您创建新窗口或将现有窗口划分为子类。

[CContainedWindow](../../atl/reference/ccontainedwindowt-class.md)实现一个包含在另一个对象中的窗口。 您可以创建新窗口或将现有窗口划分为子类。

[CDialogImpl](../../atl/reference/cdialogimpl-class.md)允许您创建模式对话框或无模式对话框。

有关 windows 的详细信息，请参阅 Windows SDK 中的[windows](/windows/win32/winmsg/windows)和后续主题。 有关在 ATL 中使用 windows 的详细信息，请参阅[Atl 窗口类](../../atl/atl-window-classes.md)一文。

##  <a name="create"></a>CWindow：： Create

创建一个窗口。

```
HWND Create(
    LPCTSTR lpstrWndClass,
    HWND hWndParent,
    _U_RECT rect = NULL,
    LPCTSTR szWindowName = NULL,
    DWORD dwStyle = 0,
    DWORD dwExStyle = 0,
    _U_MENUorID MenuOrID = 0U,
    LPVOID lpCreateParam = NULL) throw();
```

### <a name="parameters"></a>parameters

*lpstrWndClass*<br/>
中指向窗口类的指针。

*hWndParent*<br/>
中父窗口或所有者窗口的句柄。

*rect*<br/>
中指定窗口位置[_U_RECT](../../atl/reference/u-rect-class.md)类型的变量。 默认值为 NULL。 当此参数为 NULL 时，将使用 `CWindow::rcDefault` 的值。

*szWindowName*<br/>
中指定窗口的名称。 默认值为 NULL。

*dwStyle*<br/>
中窗口的样式。 默认值为0，表示未指定样式。 有关可能值的列表，请参阅 Windows SDK 中的[CreateWindow](/windows/win32/api/winuser/nf-winuser-createwindoww) 。

*dwExStyle*<br/>
中扩展的窗口样式。 默认值为0，表示未指定扩展样式。 有关可能值的列表，请参阅 Windows SDK 中的[CreateWindowEx](/windows/win32/api/winuser/nf-winuser-createwindowexw) 。

*MenuOrID*<br/>
中类型为的变量， [_U_MENUorID](../../atl/reference/u-menuorid-class.md)指定菜单或窗口标识符的句柄。 默认值为0U。

*lpCreateParam*<br/>
指向[CREATESTRUCT](/windows/win32/api/winuser/ns-winuser-createstructw)结构中包含的窗口创建数据的指针。

### <a name="return-value"></a>返回值

如果成功，则为由[m_hWnd](#m_hwnd)指定的新创建的窗口的句柄。 否则为 NULL。

### <a name="remarks"></a>备注

`CWindow::rcDefault` 定义为 `__declspec(selectany) RECT CWindow::rcDefault = {CW_USEDEFAULT, CW_USEDEFAULT, 0, 0};`。

有关详细信息，请参阅 Windows SDK 中的[CreateWindow](/windows/win32/api/winuser/nf-winuser-createwindoww) 。

**注意**如果将0用作*MenuOrID*参数的值，则必须将其指定为0u （默认值），以避免出现编译器错误。
