# <a name="nil-structure"></a>Nil 结构

支持 WRL 基础结构，不应在代码中直接使用。

## <a name="syntax"></a>语法

```cpp
struct Nil;
```

## <a name="remarks"></a>备注

用于指示未指定可选的模板参数。

**Nil**是一个空的结构。

`Nil`
