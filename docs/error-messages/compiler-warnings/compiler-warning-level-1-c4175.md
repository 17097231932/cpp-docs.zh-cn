# <a name="compiler-warning-level-1-c4175"></a>编译器警告（等级 1）C4175

\#杂注 component(browser, on)： 浏览器信息最初必须指定命令行上

要使用 [component](../../preprocessor/component.md) 杂注，必须在编译过程中生成浏览信息 ([/FR](../../build/reference/fr-fr-create-dot-sbr-file.md))。
