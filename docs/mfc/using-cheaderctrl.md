# <a name="using-cheaderctrl"></a>使用 CHeaderCtrl

使用由类[CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)表示的标头控件显示列列表的列标题。 例如，标头控件可用于在电子表格中实现列控件。

标题控件通常分为部分，称为"标题项"，每个部分都带有文本或数字关联列的标题。 根据设置的样式，您可以为用户提供许多直接方法来操作标头项。

> [!NOTE]
> [CListCtrl](../mfc/reference/clistctrl-class.md)提供嵌入式标头控件[，CListView](../mfc/reference/clistview-class.md)封装`CListCtrl`在 MFC 类中。 通常，考虑使用标记`CHeaderCtrl`您打算自己绘制的列表。
