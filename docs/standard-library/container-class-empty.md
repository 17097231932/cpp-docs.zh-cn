# <a name="container-classempty"></a>Container Class::empty
> [!NOTE]
>  本主题位于 Visual C++ 文档内，作为在 C++ 标准库内使用的容器的非功能性示例。 有关详细信息，请参阅 [C++ 标准库容器](../standard-library/stl-containers.md)。  
  
 对于空的受控序列，该成员函数返回 **true**。  
  
## <a name="syntax"></a>语法  
  
```  
 
bool empty() const;
```
