# <a name="resource-compiler-error-rc2124"></a>资源编译器错误 RC2124

不允许空菜单

**最终**中定义任何菜单项之前，将显示关键字**菜单**语句。 资源编译器不允许使用空菜单。 请确保你没有任何左引号内**菜单**语句。
