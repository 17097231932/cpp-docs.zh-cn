# Please specify valid search string.
该错误通常发生在尝试查找但未在“查找符号”对话框的“查找内容”中输入要查找的字符串时。  
  
### 更正此错误  
  
1.  在“查找内容”框中键入正确的文本并重新选择“查找”命令。
