# <a name="expression-evaluator-error-cxx0062"></a>表达式计算器错误 CXX0062

不支持的构造函数调用

表达式所做的调用构造函数。 表达式不能进行显式调用构造函数或使需要构造函数调用的转换。

此错误是与 CAN0062 相同。
