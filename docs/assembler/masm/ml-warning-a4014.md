# <a name="ml-warning-a4014"></a>ML 警告 A4014

说明和初始化不支持在 BSS 段中的数据

尝试定义已初始化的 BSS 节中的数据。  BSS 部分被指一个类，其名称是 BSS。  这包括简化的段`.data?`。
