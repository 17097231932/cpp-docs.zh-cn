# <a name="oleautomation"></a>oleautomation

指示接口使用自动化兼容。

## <a name="syntax"></a>语法

```cpp
[oleautomation]
```

## <a name="remarks"></a>备注

**Oleautomation** c + + 属性具有相同的功能[oleautomation](/windows/desktop/Midl/oleautomation) MIDL 特性。

## <a name="example"></a>示例

请参阅有关[defaultvalue](../windows/defaultvalue.md)并[nonextensible](../windows/nonextensible.md)的示例使用**oleautomation**。
