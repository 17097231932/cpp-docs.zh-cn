# CEnumeratorAccessor::m_bIsParent
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

如果行是枚举数，变量指示枚举数是否是父枚举数。  
  
## 语法  
  
```  
  
VARIANT_BOOL m_bIsParent;  
  
```
