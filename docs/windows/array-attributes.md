# Array Attributes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

以下特性应用于数组或 **SAFEARRAY**s。  
  
|特性|说明|  
|--------|--------|  
|[library\_block](../windows/library-block.md)|放在 .idl 文件的库中的构造块。|  
|[satype](../windows/satype.md)|指定 SAFEARRAY 结构的数据类型。|  
|[string](../windows/string-cpp.md)|指示一维 `char`、 `wchar_t`、 **字节** \(或等效\) 数组或指针到此数组必须将字符串。|
