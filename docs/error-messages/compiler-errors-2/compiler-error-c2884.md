# <a name="compiler-error-c2884"></a>编译器错误 C2884
name： 引起与局部函数 function 使用声明冲突  
  
 你尝试以不止一次定义的函数。 第一个定义是本地的定义。 第二个是与一个命名空间中`using`声明。  
  
 下面的示例生成 C2884:  
  
```  
// C2884.cpp  
namespace A {  
   void z(int);  
}  
  
void f() {  
   void z(int);  
   using A::z;   // C2884 z is already defined  
}  
```
