# <a name="punctuators-c"></a>标点符号 （c + +）
在 C++ 中，标点符号相对于编译器来说具有语法意义和语义含义，但是它们本身不会指定一个产生数值的操作。 某些标点符号（单独或组合）也可以是 C++ 运算符或对预处理器很重要。  

 以下任意字符都被视为标点符号：  

```  
! % ^ & * ( ) - + = { } | ~  
[ ] \ ; ' : " < > ? , . / #  
```  

 标点符号**[]**， **（)**，和**{}**必须成对出现在[翻译阶段](../preprocessor/phases-of-translation.md)4。
