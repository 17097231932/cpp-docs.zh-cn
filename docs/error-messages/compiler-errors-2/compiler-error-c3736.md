# 编译器错误 C3736
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“event”: 必须是方法；或者在托管事件的情况下，也可以是数据成员  
  
 Native 和 COM 事件必须是方法。.NET 事件也可以是数据成员。  
  
 下面的示例生成 C3736：  
  
```  
// C3736.cpp  
struct A {  
   __event int e();  
};  
  
struct B {  
   int f;   // C3736  
   // The following line resolves the error.  
   // int f();  
   B(A* a) {  
      __hook(&A::e, a, &B::f);  
   }  
};  
  
int main() {  
}  
```
