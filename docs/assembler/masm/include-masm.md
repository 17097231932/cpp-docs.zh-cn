# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

插入源代码从 *文件名* 给定的源文件添加到当前源文件在程序集内。  
  
## 语法  
  
```  
  
INCLUDE filename  
```  
  
## 备注  
 ，如果它包含一个反斜杠，分号，大于号，小于符号、单引号或双引号，在尖括号必须将 *文件名* 。
