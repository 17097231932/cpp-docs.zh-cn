# <a name="schema-rowsets"></a>架构行集合

架构 （或元数据） 是数据库的结构或组织，如提供程序、 行集、 表、 列的信息有关的任何信息或除实际的数据库内容之外的其他信息。 OLE DB，可获得此类信息，而不必打开行集，通过使用架构行集。  
  
有关如何使用架构行集的信息，请参阅[用架构行集获取元数据](../../data/oledb/obtaining-metadata-with-schema-rowsets.md)。
