# 项目生成错误 PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

您的系统缺少 NULL 设备。无法启动生成。  
  
 该错误可能由系统资源不足引起。  关闭一些应用程序或重新启动系统。  
  
 如果对计算机上的 NULL 设备没有足够的特权，也可能会导致该错误。
