# <a name="compiler-error-c3815"></a>编译器错误 C3815
方法 get_accessor 返回类型必须与 setter 的最后一个参数的类型匹配  
  
 在声明属性的返回值时`get_accessor`方法必须与匹配的 set 访问器方法的声明中的最后一个参数。  
  
 C3815 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
