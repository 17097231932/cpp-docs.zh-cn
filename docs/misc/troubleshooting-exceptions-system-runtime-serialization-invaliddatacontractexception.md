# 关于异常的疑难解答：System.Runtime.Serialization.InvalidDataContractException
在序列化和反序列化过程中，当 <xref:System.Runtime.Serialization.InvalidDataContractException> 或 <xref:System.Runtime.Serialization.DataContractSerializer> 遇到无效的数据协定时，将引发 <xref:System.Runtime.Serialization.NetDataContractSerializer> 异常。
