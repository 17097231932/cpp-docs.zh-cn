# 字符串，ATL 属性页向导
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

提供与属性页关联的文本。  
  
 **标题**  
 设置属性页选项卡上显示的文本。  
  
 **文档字符串**  
 设置描述页的文本字符串。  该字符串可显示在属性表对话框中。  属性框架可以使用状态行或者工具提示中的说明。  标准属性框架目前不使用该字符串。  
  
 **帮助文件**  
 设置描述如何使用属性页的帮助文件的名称。  此名称不应包括路径。  当用户按下“帮助”按钮时，框架打开 HelpDir 项值（位于该按钮 CLSID 下的属性页注册表项中）中命名的目录中的帮助文件。
