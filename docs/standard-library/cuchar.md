# <a name="ltcwchargt"></a>&lt;cwchar&gt;

包含标准 C 库标头\<uchar.h >，并将添加到关联的名称`std`命名空间。

## <a name="requirements"></a>要求

**标头：** \<cuchar >

**命名空间：** std

## <a name="member-functions"></a>成员函数

|||
|-|-|
|[c16rtomb](../standard-library/cuchar-functions.md#c16rtomb)|无说明。|
|[c32rtomb](../standard-library/cuchar-functions.md#c32rtomb)|无说明。|
|[mbrtoc16](../standard-library/cuchar-functions.md#mbrtoc16)|无说明。|
|[mbrtoc32](../standard-library/cuchar-functions.md#mbrtoc32)|无说明。|

## <a name="remarks"></a>备注

包含该标头还将确保使用标准 C 库标头中的外部链接声明的名称在 `std` 命名空间中声明。
