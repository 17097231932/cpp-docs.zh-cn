# <a name="compiler-warning-level-3-c4543"></a>编译器警告（等级 3）C4543

特性 no_injected_text 禁止注入的文本

[No_injected_text](../../windows/no-injected-text.md)属性出现在源代码中，这意味着编译器将阻止属性注入代码。

此警告是提醒您属性将无法再将代码注入。
