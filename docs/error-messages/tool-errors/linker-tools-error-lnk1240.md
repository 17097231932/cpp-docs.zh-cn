# 链接器工具错误 LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未能编译 IDL 内容  
  
 链接器生成了 MIDL 以编译嵌入的 IDL，但出现了问题。  检查 MIDL 指定的错误。
