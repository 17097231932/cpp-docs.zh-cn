# <a name="compiler-error-c3815"></a>编译器错误 C3815

方法 "get_accessor" 的返回类型必须与 setter 的最后一个参数的类型匹配

声明属性时，方法的返回值 `get_accessor` 必须与 set 访问器方法声明中的最后一个参数匹配。

只能使用过时的编译器选项 **/clr： oldSyntax** 来访问 C3815。
