# 表达式计算器错误 CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不支持运算符  
  
 表达式中指定了不受支持的 C 运算符。  使用受支持的 C 运算符写一个等效表达式。  
  
 该错误与 CAN0005 相同。
