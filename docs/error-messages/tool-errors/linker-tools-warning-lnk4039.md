# <a name="linker-tools-warning-lnk4039"></a>链接器工具警告 LNK4039

部分用 /SECTION 选项指定的 name 不存在

[DUMPBIN](../../build/reference/dumpbin-reference.md)或[EDITBIN](../../build/reference/editbin-reference.md)找不到一个名为部分`name`输入文件中。

若要查看文件中的部分，请运行与 DUMPBIN [/HEADERS](../../build/reference/headers.md)选项。
