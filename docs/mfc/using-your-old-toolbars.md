# <a name="using-your-old-toolbars"></a>使用您的旧工具栏

如果已使用早期版本的 Visual C++ 创建自定义工具栏，则 [CToolBar](../mfc/reference/ctoolbar-class.md) 类的新实现可能会导致出现问题。 因此您无需放弃您的旧工具栏来使用新的功能，以前的实现仍受支持。

DOCKTOOL 示例不使用旧式工具栏，仅使用新式工具栏。

您无法使用工具栏资源编辑器编辑旧式工具栏。
