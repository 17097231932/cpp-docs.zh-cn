# <a name="compiler-error-c2667"></a>编译器错误 C2667

"function"：所有数字重载都没有最佳转换

重载的函数调用不明确，无法解析。

与函数调用中的参数匹配的实际参数所需的转换必须严格优于所有其他重载函数所需的转换。
