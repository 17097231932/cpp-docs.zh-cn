# <a name="catstr"></a>CATSTR

连接文本项。

## <a name="syntax"></a>语法

> *name* **CATSTR** ⟦*textitem1* ⟦__，__ *textitem2* .。。⟧⟧

## <a name="remarks"></a>备注

每个文本项可以是文本字符串、前面有的常量 **%** 或宏函数返回的字符串。 **CATSTR** 是 [TEXTEQU](textequ.md)的同义词。
