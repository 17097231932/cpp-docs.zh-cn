# <a name="dt"></a>DT

为每个*初始值设定项*分配10个字节的存储并根据需要对其进行初始化。 在类型合法的任何位置，还可以用作类型说明符。 **DT**是[TBYTE](../../assembler/masm/tbyte.md)的同义词。

## <a name="syntax"></a>语法

> ⟦*name*⟧ **DT** *初始值设定项*⟦ __，__ *初始值设定项*.。。⟧
