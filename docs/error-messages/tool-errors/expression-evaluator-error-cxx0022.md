# 表达式计算器错误 CXX0022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在 \_main 之前的函数调用  
  
 C 表达式计算器无法在调试器进入 **\_main** 函数之前计算函数。  直到调用 **\_main**，程序初始化才正确。  
  
 该错误与 CAN0022 相同。
