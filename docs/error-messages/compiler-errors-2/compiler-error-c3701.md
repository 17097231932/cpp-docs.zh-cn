# 编译器错误 C3701
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: event\_source 没有事件  
  
 尝试将 [event\_source](../../windows/event-source.md) 用于没有事件方法的类。  若要修复此错误，请向该类添加一个或多个事件。  
  
 下面的示例生成 C3701：  
  
```  
// C3701.cpp  
[ event_source(native) ]  
class CEventSrc {  
public:  
   // uncomment the following line to resolve this C3701  
   // __event void fireEvent(int i);  
};   // C3701  
  
int main() {  
}  
```
