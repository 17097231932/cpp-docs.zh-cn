# NMAKE 错误 U1097
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

文件名部分语法要求相关  
  
 当前依赖项没有显式从属项或隐式从属项。  文件名部分的语法使用百分比 \(`%`\) 说明符，表示当前目标的第一个从属项的组件。
