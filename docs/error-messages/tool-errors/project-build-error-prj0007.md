# <a name="project-build-error-prj0007"></a>项目生成错误 PRJ0007

无法创建输出目录 "directory"。

视觉C++对象未能创建输出目录。 可能的原因包括：

- 要在其中创建输出目录的目录是只读的。

- 计算机的磁盘空间不足。

- 指定的目录路径无效。

- 用户权限不足，无法创建目录。
