# <a name="array-attributes"></a>数组特性

以下属性应用于数组或`SAFEARRAY`s。

|特性|描述|
|---------------|-----------------|
|[library_block](../windows/library-block.md)|将放置在.idl 文件的库块中的构造。|
|[satype](../windows/satype.md)|指定的数据类型`SAFEARRAY`结构。|
|[string](../windows/string-cpp.md)|指示的一维**char**， **wchar_t**， `byte` （或等效） 必须作为字符串处理数组或指针，指向此类。|
