# <a name="compiler-warning-level-2-c4309"></a>编译器警告（等级 2）C4309
conversion： 常量值的截断  
  
 类型转换会导致要超过为其分配的空间的常量。 你可能需要使用更大的类型常量。  
  
 下面的示例生成 C4309:  
  
```  
// C4309.cpp  
// compile with: /W2  
int main()  
{  
   char c = 128;   // C4309  
}  
```
