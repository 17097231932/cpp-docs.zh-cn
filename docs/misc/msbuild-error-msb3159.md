# MSBuild 错误 MSB3159
**MSB3159: 文件“\<file\>”中的 Xml 验证错误:“\<error\>”**  
  
 此错误在产品文件或 package.xml 文件不符合 XML 架构时产生。  此错误的后半部分是具体的错误消息。  
  
### 更正此错误  
  
-   更新 XML 文件以解决该验证错误。
