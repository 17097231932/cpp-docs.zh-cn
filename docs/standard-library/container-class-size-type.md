# <a name="container-classsize_type"></a>Container Class::size_type

> [!NOTE]
> 本主题在 Microsoft c + + 文档中作为 c + + 标准库中使用的容器的非功能性示例。 有关详细信息，请参阅 [C++ 标准库容器](../standard-library/stl-containers.md)。

描述可表示任何受控序列长度的对象。

## <a name="syntax"></a>语法

```cpp
typedef T0 size_type;
```

## <a name="remarks"></a>备注

此处描述为未指定类型的同义词 `T0` (通常 `Alloc::size_type`) 。
