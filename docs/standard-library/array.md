# &lt;array&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

定义容器模板类 `array` 和几个支持的模板。  
  
## 语法  
  
```  
#include <array>  
```  
  
### 类  
  
|||  
|-|-|  
|[array](../standard-library/array-class-stl.md)|定长存储元素序列。|

|[tuple\_size](../standard-library/tuple-size-class-array.md)|包装数组元素的范围。|
