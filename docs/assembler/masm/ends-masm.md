# <a name="ends-masm"></a>ENDS (MASM)
将标记的末尾段、 结构或联合*名称*以前始于[段](../../assembler/masm/segment.md)，[结构](../../assembler/masm/struct-masm.md)，[联合](../../assembler/masm/union.md)，或简化的线段指令。  
  
## <a name="syntax"></a>语法  
  
```  
  
name ENDS  
```
