# <a name="dt"></a>DT
分配和 （可选） 为每个初始化 10 个字节的存储`initializer`。 此外可作为类型说明符类型是合法的任何位置。 `DT` 同义词[TBYTE](../../assembler/masm/tbyte.md)。  
  
## <a name="syntax"></a>语法  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
