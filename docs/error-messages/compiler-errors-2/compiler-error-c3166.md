# <a name="compiler-error-c3166"></a>编译器错误 C3166
指针： 不能将内部 __gc 指针指向的指针声明为 type 的成员  
  
编译器找到了无效的指针声明 (`__nogc`指向`__gc`指针。)。 
  
C3166 才可访问使用过时的编译器选项**/clr:oldSyntax**。
