# 数组、列表和映射的模板类
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

这些集合类是参数确定在聚合存储的对象的类型的模板。  `CArray`、`CMap`和 `CList` 类使用必须通常自定义全局的 Helper 函数。  有关这些帮助程序函数的更多信息，请参见 [集合类帮助器](../mfc/reference/collection-class-helpers.md)。  类型化类指针是包装其他类的类库中。  通过使用这些包装，则注册编译器的类型检查有助于避免错误。  有关使用这些类的更多信息，请参见 [集合](../mfc/collections.md)。  
  
 这些类提供可以使用创建数组的模板列表，并映射使用您喜欢的任何类型。  
  
 [CArray](../mfc/reference/carray-class.md)  
 使的数组模板类任意类型。  
  
 [CList](../mfc/reference/clist-class.md)  
 使的列表模板类任意类型。  
  
 [CMap](../mfc/reference/cmap-class.md)  
 使的映射模板类与任意键和值类型。  
  
 [CTypedPtrArray](../mfc/reference/ctypedptrarray-class.md)  
 类型安全数组的指针。模板类  
  
 [CTypedPtrList](../mfc/reference/ctypedptrlist-class.md)  
 类型安全列表的指针模板类  
  
 [CTypedPtrMap](../mfc/reference/ctypedptrmap-class.md)  
 安全类型映射的模板类的指针。
