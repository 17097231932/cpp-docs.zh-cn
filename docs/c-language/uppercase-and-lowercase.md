# <a name="uppercase-and-lowercase"></a>大写和小写
**ANSI 3.1.2** 大小写区别是否重要  
  
 Microsoft C 将编译单元中的标识符视为区分大小写。  
  
 Microsoft 链接器区分大小写。 您必须根据情况一致地指定所有标识符。
