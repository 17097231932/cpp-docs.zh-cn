# &lt;typeinfo&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

包括标准头定义多种类型的 `<typeinfo>` 与类型，[typeid 运算符](../cpp/typeid-operator.md)运算符标识提供有关静态和动态类型的信息。  
  
## 语法  
  
```  
  
#include <typeinfo>  
  
```  
  
## 备注  
 有关在 \<typeinfo\>定义的类的信息，请参见下列主题：  
  
-   [bad\_cast 异常](../cpp/bad-cast-exception.md)  
  
-   [bad\_typeid 异常](../cpp/bad-typeid-exception.md)  
  
-   [type\_info 类](../cpp/type-info-class.md)
