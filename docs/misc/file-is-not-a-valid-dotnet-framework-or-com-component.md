# File is not a valid .NET Framework or COM component
您尝试在未包含有效对象的“对象浏览器”中查看文件。  在对象浏览器中只能查看 [!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)] 和 COM 组件。  
  
### 更正此错误  
  
1.  如果选错了文件，请选择正确的文件并重试。  
  
2.  如果在**“打开方式”**对话框中误选了“对象浏览器”，请选择用来打开该文件的其他程序。
