# <a name="compiler-error-c2875"></a>编译器错误 C2875

using 声明导致 class::identifier 的多个声明

声明会导致相同的项定义了两次。

下面的示例生成 C2875:

```
// C2875.cpp
struct A {
   void f(int*);
};

struct B {
   void f(double*);
};

struct AB : A, B {
   using A::f;
   using A::f;   // C2875
   using B::f;
};
```
