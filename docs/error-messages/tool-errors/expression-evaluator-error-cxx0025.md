# <a name="expression-evaluator-error-cxx0025"></a>表达式计算器错误 CXX0025

运算符要求结构/联合

采用的表达式的运算符`struct`或**union**类型应用到不是表达式`struct`或**联合**。

组件的类、 结构或联合变量必须具有完全限定的名称。 组件不能输入而无需完整规范。

此错误是与 CAN0025 相同。
