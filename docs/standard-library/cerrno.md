# <a name="ltcerrnogt"></a>&lt;cerrno&gt;

包含标准 C 库标头\<errno.h >，并将添加到关联的名称`std`命名空间。 包括此标头中声明标准 C 库标头中使用外部链接声明的名称将确保`std`命名空间。

## <a name="syntax"></a>语法

```cpp
#include <cerrno>
```
