# <a name="ml-fatal-error-a1005"></a>ML 错误 A1005

**组装器限制： 宏参数名称表完整**

过多的参数、 局部变量或宏标签定义的宏。 宏名称表时没有更多空间。

定义短或更少名称或删除不必要的宏。
