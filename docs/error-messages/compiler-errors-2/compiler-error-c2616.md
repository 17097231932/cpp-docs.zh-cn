# <a name="compiler-error-c2616"></a>编译器错误 C2616

"转换"：无法将非 lvalue "type1" 隐式转换为不是常量的 "type2"

无法从非 lvalue 初始化引用。

这是在 ANSI 兼容性 ([/za](../../build/reference/za-ze-disable-language-extensions.md)) 下的错误和 Microsoft 扩展 (**/ze**) 下的警告。
