# <a name="fatal-error-c1045"></a>错误 C1045

编译器限制 : 链接规范嵌套太深

嵌套的外部对象超过编译器限制。 允许嵌套的外部项使用外部链接类型，如 `extern` “C++”。 减少嵌套的外部项的数量以解决该错误。
