# 编译器警告（等级 1）C4116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

括号中的未命名类型定义  
  
 在附加说明的表达式中定义了无名称的结构、联合或枚举类型。  类型定义是无意义的。  
  
 在 C 函数调用中，定义具有全局范围。  在 C\+\+ 函数调用中，定义具有与被调用函数一样的范围。
