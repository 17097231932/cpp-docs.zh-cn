# <a name="compiler-error-c2411"></a>编译器错误 C2411
identifier: context 中的非法结构/联合成员  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>通过检查以下可能的原因进行修复  
  
1.  该标识符不是可见的结构或在此上下文中的联合的成员。  
  
2.  该标识符不是结构或使用成员选择 （.） 运算符指定的联合的成员。
