# <a name="sword"></a>SWORD
分配和 （可选） 为每个初始化存储的有符号的字 （2 个字节） `initializer`。 此外可作为类型说明符类型是合法的任何位置。  
  
## <a name="syntax"></a>语法  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
