# <a name="project-build-error-prj0047"></a>项目生成错误 PRJ0047

未能继续挂起的进程。  生成失败。

在开发环境中出错。

若要解决此错误，请关闭并重新加载项目。 如果有必要，请退出并重新启动 Visual Studio。
