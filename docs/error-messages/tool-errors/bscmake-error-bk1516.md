# <a name="bscmake-error-bk1516"></a>BSCMAKE 错误 BK1516

bscfile 已损坏;无法增量更新

.Bsc 文件已损坏，可能是生成过程中发生系统错误。 删除 .bsc 文件，重新生成所有 .sbr 文件，然后重新生成 .bsc 文件。
