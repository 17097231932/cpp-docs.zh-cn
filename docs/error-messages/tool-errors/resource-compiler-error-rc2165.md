# <a name="resource-compiler-error-rc2165"></a>资源编译器错误 RC2165

在 DLGINCLUDE 语句中找不到字符串

语句未指定有效的包含文件。

**DLGINCLUDE** 语句必须使用以下语法：

**DLGINCLUDE "**<em>filename</em>**.h"**
