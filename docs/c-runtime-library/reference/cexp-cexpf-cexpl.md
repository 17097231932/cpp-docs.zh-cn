# <a name="cexp-cexpf-cexpl"></a>cexp、cexpf、cexpl

计算复数的以 e 为底的指数。

## <a name="syntax"></a>语法

```C
_Dcomplex cexp( _Dcomplex z );
_Fcomplex cexpf( _Fcomplex z );
_Lcomplex cexpl( _Lcomplex z );
```

```cpp
_Fcomplex cexp( _Fcomplex z );  // C++ only
_Lcomplex cexp( _Lcomplex z );  // C++ only
```

### <a name="parameters"></a>参数

*z*<br/>
表示指数的复数。

## <a name="return-value"></a>返回值

值**e**的幂*z*。

## <a name="remarks"></a>备注

由于 c + + 允许重载，你可以调用的重载**cexp**采用并返回 **_Fcomplex**和 **_Lcomplex**值。 在 C 程序中， **cexp**始终采用并返回 **_Dcomplex**值。
