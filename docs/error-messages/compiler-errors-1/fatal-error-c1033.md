# <a name="fatal-error-c1033"></a>错误 C1033
无法打开程序数据库 pdb  
  
 可以通过磁盘错误导致此错误。  
  
 在 Visual c + +.NET 2002 中，用户区域设置时，必须设置正确的文件名称 （或指向的文件名称的目录路径） 含有 MBCS 字符。 设置系统区域设置是不够的；必须设置用户区域设置以处理 MBCS 字符。  
  
 有关详细信息，请参阅[http://support.microsoft.com/default.aspx?scid=kb;en-us;246007](http://support.microsoft.com/default.aspx?scid=kb;en-us;246007)。
