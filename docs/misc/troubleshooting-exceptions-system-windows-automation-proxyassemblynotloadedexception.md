# 关于异常的疑难解答：System.Windows.Automation.ProxyAssemblyNotLoadedException
若在加载包含客户端提供程序的程序集时出现问题，则会引发某种异常，而 <xref:System.Windows.Automation.ProxyAssemblyNotLoadedException> 异常将包含有关这种异常的信息。
