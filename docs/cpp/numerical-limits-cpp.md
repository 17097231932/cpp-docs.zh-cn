# <a name="numerical-limits-c"></a>数字限制 (C++)

这两个标准包含文件 \<限制 .h > 和 \<float >，定义给定类型的变量可以包含的数值限制或最小值和最大值。 这些最小和最大的可移植到使用C++与 ANSI C 相同的数据表示形式的任何编译器。\<限制 .h > 包含文件[为整型类型定义数值限制](../cpp/integer-limits.md)，\<float > 定义[浮动类型的数值限制](../cpp/floating-limits.md)。
