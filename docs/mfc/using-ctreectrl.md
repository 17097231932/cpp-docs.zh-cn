# <a name="using-ctreectrl"></a>使用 CTreeCtrl

类表示一个树控件[CTreeCtrl](../mfc/reference/ctreectrl-class.md)，是一个窗口，显示磁盘上的项，如在文档中，标题索引，或文件和目录中的项的层次列表。 每个项包含一个标签和一个可选位图图像，且每个项可以有一个与之关联的子项列表。 通过单击项，用户可以展开和折叠关联的子项列表。 资源管理器的左侧窗格中的目录树是树控件的一个示例。
