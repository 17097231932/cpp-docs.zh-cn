# MSBuild 错误 MSB4137
**MSB4137: 在配置文件的“\<x\>”处指定的值无效。  属性名和工具版本均不能为空字符串。**  
  
 工具集定义包含无效的属性名或值。  
  
### 更正此错误  
  
-   确保工具集定义不包含无效或空的属性名或值。
