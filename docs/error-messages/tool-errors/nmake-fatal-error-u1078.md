# NMAKE 错误 U1078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

常数在“expression”处溢出  
  
 给定的表达式包含超出范围（– 2,147,483,648 至 2,147,483,647）的常数。  该常数在下列某种情况中出现：  
  
-   用预处理指令指定的表达式  
  
-   用短划线 \(**\-**\) 命令修饰符指定的错误级别
