# <a name="compiler-warning-level-3-c4622"></a>编译器警告（等级 3）C4622
覆盖创建对象文件的预编译头期间生成的调试信息：“file”  
  
 它用编译时丢失指定的文件中的 CodeView 信息[/Yu](../../build/reference/yu-use-precompiled-header-file.md) （使用预编译标头） 选项。  
  
 重命名对象文件 (使用[/Fo](../../build/reference/fo-object-file-name.md)) 创建或使用预编译标头文件中，并使用新的对象文件进行链接时。
