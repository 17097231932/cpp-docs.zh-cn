# 编译器错误 C3322
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“property”：不是特性“attribute”的有效属性  
  
 向特性传递的属性（或参数）无效或无法识别。 请查看该特性的文档。
