# <a name="compiler-warning-level-1-c4098"></a>编译器警告 （等级 1） C4098

function: void 函数返回一个值

返回类型声明的函数[void](../../cpp/void-cpp.md)具有`return`语句返回的值。 编译器将假定该函数返回类型的值`int`。
