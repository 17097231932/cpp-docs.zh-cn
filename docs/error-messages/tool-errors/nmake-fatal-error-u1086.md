# <a name="nmake-fatal-error-u1086"></a>NMAKE 错误 U1086
**推理规则不能有依赖项**  
  
 冒号 (**:**) 中的推理规则后面必须跟以下方法之一：  
  
-   换行符  
  
-   分号 (**;**)，可以跟命令  
  
-   数字符号 (**#**)，可以跟注释
