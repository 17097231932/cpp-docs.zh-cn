# 编译器错误 C2309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

catch 处理程序需要带括号的异常声明  
  
 catch 处理程序没有带括号的类型。  
  
 下面的示例生成 C2309：  
  
```  
// C2309.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch C {}   // C2309  
   // try the following line instead  
   // catch( C ) {}  
}  
```
