# 编译器警告 C4961
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

没有将任何配置文件数据合并到“.pgd file”，因此已禁用按配置文件优化  
  
 没有可用的配置文件数据（.pgc 文件），因此按配置优化不会发生。
