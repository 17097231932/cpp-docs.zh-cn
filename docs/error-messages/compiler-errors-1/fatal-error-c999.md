# Fatal Error C999
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

未知消息    请在 Visual C\+\+ 帮助菜单上，选择技术支持命令，或打开技术支持帮助文件了解详细信息。  
  
 此错误通常表示你有具有来自编译器不同版本的混合文件，或者你的编译器安装已损坏。  使用控制面板中的“程序和功能”小程序修复或重新安装产品。
