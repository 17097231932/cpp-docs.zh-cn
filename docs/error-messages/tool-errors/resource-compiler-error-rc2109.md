# <a name="resource-compiler-error-rc2109"></a>资源编译器错误 RC2109

预期数值对话框常量

一个**对话框中**语句要求的整数值*x、 y、 宽度*，并*高度*字段。 请确保这些值是包含后**对话框**关键字，它们不负。
