# <a name="namedguids"></a>named_guids
**C + + 专用**  
  
告知编译器定义和初始化旧样式的窗体的 GUID 变量`LIBID_MyLib`， `CLSID_MyCoClass`， `IID_MyInterface`，和`DIID_MyDispInterface`。  
  
## <a name="syntax"></a>语法  
  
```  
named_guids  
```  
  
## <a name="remarks"></a>备注  

**结束 c + + 专用**
