# <a name="code"></a>.CODE

与一起使用时[。模型](../../assembler/masm/dot-model.md)，指示在代码段的开始。

## <a name="syntax"></a>语法

> .CODE [[name]]

#### <a name="parameters"></a>参数

|参数|描述|
|---------------|-----------------|
|`name`|可选参数，用于指定代码段的名称。 默认名称是小型、 小型、 简洁和平面 _TEXT[模型](../../assembler/masm/dot-model.md)。 默认名称是*modulename*_TEXT 对于其他模型。|
