# <a name="adding-atl-support-to-your-mfc-project"></a>向 MFC 项目添加 ATL 支持

如果你已创建基于 MFC 的应用程序，然后您可以添加支持的活动模板库 (ATL) 轻松地通过运行到 MFC 项目向导的添加 ATL 支持。

> [!NOTE]
>  ATL 和 MFC 不通常支持在 Visual Studio 速成版中。

> [!NOTE]
>  此支持仅适用于添加到 MFC 可执行文件或 DLL 项目的简单 COM 对象。 可以将其他 COM 对象 （包括 ActiveX 控件） 添加到 MFC 项目中，但这些对象可能不会按预期运行。

### <a name="to-add-atl-support-to-your-mfc-project"></a>若要向 MFC 项目添加 ATL 支持

1. 在解决方案资源管理器，右键单击你想要添加 ATL 支持的项目。

1. 在快捷菜单中，依次单击“添加”  和“添加类”  。

1. 选择**向 MFC 项目添加 ATL 支持**图标。

    > [!NOTE]
    >  此图标位于 ATL 文件夹**类别**窗格。

1. 出现提示时，单击**是**添加 ATL 支持。

添加 ATL 支持 MFC 项目的代码更改的详细信息，请参阅[详细信息的 ATL 支持 ATL 向导添加](../../mfc/reference/details-of-atl-support-added-by-the-atl-wizard.md)
