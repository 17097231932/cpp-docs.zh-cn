# <a name="math-error-m6205"></a>数学错误 M6205

function: _TLOSS 错误

总丢失基数 （精度） 发生。

此错误 cos，可能会导致的给定的操作数的 sin，非常大或 tan，因为该操作数必须减小为 0 到 2 之间的数字 * pi。
