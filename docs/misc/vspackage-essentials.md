# VSPackage 要点
VSPackage 是软件模块，它们通过 UI 元素、服务、项目、编辑器和设计器来组成并扩展 Visual Studio。 VSPackage 是 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 的中央体系结构单元。[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 本身多半以 VSPackage 集合的形式进行编写。  
  
 要创建 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 扩展性应用程序，请首先创建 VSPackage。 随后可以添加菜单命令、工具窗口、文档窗口或其他代码以扩展 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] IDE。  
  
 要使用 VSPackage，需要安装 Visual Studio SDK。 要按照本演练的步骤操作，必须安装 Visual Studio SDK。 有关详细信息，请参阅[Visual Studio SDK](../Topic/Visual%20Studio%20SDK.md)。
