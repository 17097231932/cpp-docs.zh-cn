# <a name="version"></a>\@版本

在命令行上报告的 MASM 的主要版本和次要版本，作为 (文本宏) 的单个数字。 例如，MASM 版本14.23.28107.0 生成 "1423"。

## <a name="syntax"></a>语法

> **\@Version**
