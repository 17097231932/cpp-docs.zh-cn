# <a name="compiler-error-c3645"></a>编译器错误 C3645
function: __clrcall 不能用于函数编译为本机代码  
  
 在函数中的某些关键字的存在将导致要编译到本机的函数。  
  
## <a name="example"></a>示例  
 下面的示例生成 C3645。  
  
```  
// C3645.cpp  
// compile with: /clr /c  
#pragma unmanaged   
int __clrcall dog() {}   // C3645  
```
