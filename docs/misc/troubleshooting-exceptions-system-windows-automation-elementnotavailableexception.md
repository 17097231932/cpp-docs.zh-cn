# 关于异常的疑难解答：System.Windows.Automation.ElementNotAvailableException
当尝试访问与不再可用的某个用户界面部分对应的 UI <xref:System.Windows.Automation.ElementNotAvailableException> 时会引发某种异常，<xref:System.Windows.Automation.AutomationElement> 异常包含有关这种异常的信息。  
  
## 备注  
 如果元素所在的对话框已关闭，或者元素所在的应用程序已关闭或终止，则会引发此异常。
