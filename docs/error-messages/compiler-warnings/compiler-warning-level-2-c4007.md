# 编译器警告（等级 2）C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 必须是“attribute”  
  
 某函数的所需特性未显式说明。  例如，**main** 函数必须有 `__cdecl` 特性。  编译器强制该特性。
