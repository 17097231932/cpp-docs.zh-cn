# <a name="ml-nonfatal-error-a2004"></a>ML 非致命错误 A2004

**符号类型冲突：标识符**

[EXTERNDEF](../../assembler/masm/externdef.md)或[label](../../assembler/masm/label-masm.md)指令用于在同一模块中定义但类型不同的变量、符号、数据结构或标签。
