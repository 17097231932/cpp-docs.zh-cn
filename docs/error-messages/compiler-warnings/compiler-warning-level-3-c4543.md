# 编译器警告（等级 3）C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

特性“no\_injected\_text”禁止注入的文本  
  
 [no\_injected\_text](../../windows/no-injected-text.md) 特性出现在源代码中，这意味着编译器将防止特性插入代码。  
  
 此警告用于提醒您特性将无法插入代码。
