# <a name="resource-compiler-error-rc2114"></a>资源编译器错误 RC2114

应为控件类名

`class`**对话** 语句中控制语句的字段必须是以下类型之一：按钮、组合框、编辑、列表框、滚动条、静态或用户定义。 请确保类拼写正确。
