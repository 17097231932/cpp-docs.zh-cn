# <a name="compiler-error-c2155"></a>编译器错误 C2155
'？ ': 无效的左侧操作数，预期算术或指针类型  
  
 在左侧的表达式中`?`不能为零进行比较。 你必须使用可以比较为零的算术或指针表达式。
