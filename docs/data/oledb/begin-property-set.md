# BEGIN_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

标记的属性集的开头。属性集映射。  
  
## 语法  
  
```  
  
BEGIN_PROPERTY_SET(  
guid   
)  
  
```  
  
#### 参数  
 `guid`  
 \[in\] GUID属性。  
  
## 示例  
 参见 [BEGIN\_PROPSET\_MAP](../../data/oledb/begin-propset-map.md)。  
  
## 要求  
 **头文件：** atldb.h
