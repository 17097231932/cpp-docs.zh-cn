# <a name="compiler-error-c2517"></a>编译器错误 C2517
identifier： 右侧的:: 未定义  
  
 范围解析运算符右侧的标识符 (`::`) 必须是定义的类、 结构或在左侧的联合成员。 如果名为任何类、 结构或联合，则必须具有全局作用域声明在右侧的标识符。
