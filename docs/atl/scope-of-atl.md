# <a name="scope-of-atl"></a>ATL 范围

ATL，可轻松地创建 COM 对象、 自动化服务器和 ActiveX 控件。 ATL 提供了对许多基本的 COM 接口的内置支持。

ATL 附带源代码，其中包括在应用程序中。 ATL 还使 DLL 可 (atl90.dll)，其中包含可以在组件之间共享的代码。 但是，不需要此 DLL。
