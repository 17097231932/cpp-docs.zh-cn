# <a name="compiler-warning-level-1-c4025"></a>编译器警告（等级 1）C4025
“编号”: 基指针传递给带变量参数的函数: 参数号  
  
 将基指针传递给使用变量参数的函数会导致指针被规范化，结果不可预知。 请勿将基指针传递给带变量参数的函数。
