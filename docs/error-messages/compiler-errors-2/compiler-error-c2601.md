# <a name="compiler-error-c2601"></a>编译器错误 C2601
function： 局部函数定义是非法的  
  
 代码尝试定义函数中的函数。  
  
 或者，C2601 错误的位置之前在源代码中可能有一个额外的大括号。  
  
 下面的示例生成 C2601:  
  
```  
// C2601.cpp  
int main() {  
   int i = 0;  
  
   void funcname(int j) {   // C2601  
      j++;  
   }  
}  
```
