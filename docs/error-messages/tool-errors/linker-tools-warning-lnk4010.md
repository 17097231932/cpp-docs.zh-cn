# <a name="linker-tools-warning-lnk4010"></a>链接器工具警告 LNK4010

无效的子系统版本号号;假定为默认子系统版本

可以 ([/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)) 为映像的子系统指定版本。 每个子系统的最低版本要求。 如果指定的版本低于最小值，则会出现此警告，链接器将只使用默认子系统。
