# CVTRES 错误 CVT1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法读取文件名  
  
 当 CVTRES 尝试读取给定文件时发生无法恢复的错误。  
  
 此错误可能是由于文件损坏而造成的。
