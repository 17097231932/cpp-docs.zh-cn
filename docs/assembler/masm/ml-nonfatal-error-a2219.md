# <a name="ml-nonfatal-error-a2219"></a>ML 非致命错误 A2219

> 错误中的偏移量的对齐方式展开代码

## <a name="remarks"></a>备注

操作数[ &period;ALLOCSTACK](../../assembler/masm/dot-allocstack.md)并[ &period;SAVEREG](../../assembler/masm/dot-savereg.md)必须是 8 的倍数。  操作数[ &period;SAVEXMM128](../../assembler/masm/dot-savexmm128.md)并[ &period;SETFRAME](../../assembler/masm/dot-setframe.md)必须是 16 的倍数。
