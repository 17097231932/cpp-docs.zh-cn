# <a name="nosearchnamespace"></a>no_search_namespace
**C + + 专用**

具有相同的功能[no_namespace](../preprocessor/no-namespace.md)属性，但在您使用的类型库使用`#import`指令与[auto_search](../preprocessor/auto-search.md)属性。

## <a name="syntax"></a>语法

```
no_search_namespace
```

**结束 c + + 专用**
