# <a name="nolistif"></a>.NOLISTIF

取消其条件计算结果为 false (0) 的条件块的列表。

## <a name="syntax"></a>语法

> **.NOLISTIF**

## <a name="remarks"></a>备注

这是默认值。 与相同 [。.SFCOND](dot-sfcond.md)。
