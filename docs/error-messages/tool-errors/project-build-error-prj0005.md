# 项目生成错误 PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无法在目录“directory”中创建临时文件。  
  
 创建临时文件的调用失败。  失败的原因包括：  
  
-   临时文件名称已用尽。  
  
-   临时目录是只读的。  
  
-   没有临时目录或 TMP 环境变量。  
  
-   计算机的可用磁盘空间不足。
