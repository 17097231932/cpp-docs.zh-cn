# 链接器工具错误 LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

无效的映像基址  
  
 您为图像指定的基址无效。  基址必须是 64KB 对齐的（最后四个十六进制数字必须为零），且映像基必须在 32 位有符号值或无符号值的范围内。
