# 处理您对话框中的 Windows 消息
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

对话框是窗口，因此，它们可处理窗口消息，如果提供了相应处理程序函数。  当使用添加类向导创建了对话框类，则向导添加空的消息映射到类。  使用属性窗口映射您希望处理类的所有窗体消息或命令。  
  
 有关更多信息，参见 [映射窗口消息到对话框类](../mfc/mapping-windows-messages-to-your-class.md)。
