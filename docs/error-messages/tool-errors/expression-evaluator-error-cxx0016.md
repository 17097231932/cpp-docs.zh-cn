# <a name="expression-evaluator-error-cxx0016"></a>表达式计算器错误 CXX0016

常量太大

C 表达式计算器无法接受大于 4294967295 (0FFFFFFFF 十六进制) 的无符号整数常量，或大于大约 1.8 E + 308 的大小的浮点常数。

此错误与 CAN0016 相同。
