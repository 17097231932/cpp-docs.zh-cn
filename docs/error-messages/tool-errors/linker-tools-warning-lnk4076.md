# <a name="linker-tools-warning-lnk4076"></a>链接器工具警告 LNK4076
无效的增量状态文件 filename;非增量链接  
  
 链接无法写入增量状态 (.ilk) 文件。 任一`filename`损坏，或者不是增量链接数据库。 删除文件并重新链接。
