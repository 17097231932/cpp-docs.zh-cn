# .NOLISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

取消条件计算结果为 false 的列表条件块 \(0\)。  
  
## 语法  
  
```  
  
.NOLISTIF  
  
```  
  
## 备注  
 这是默认值。  和 [.SFCOND](../../assembler/masm/dot-sfcond.md)相同。
