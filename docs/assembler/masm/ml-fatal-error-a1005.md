# <a name="ml-fatal-error-a1005"></a>ML 错误 A1005
**汇编程序限制： 宏参数名称表完整**  
  
 针对宏定义了太多的参数、 局部变量或宏标签。 在宏名称表中时没有更多空间。  
  
 定义更短或更少名称或删除不必要的宏。
