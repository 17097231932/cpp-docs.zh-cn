# <a name="linker-tools-error-lnk1296"></a>链接器工具错误 LNK1296

无法加载文件名

给定的 DLL 不可用。

DLL 的可能位置是当前目录、系统目录、Windows 目录和 PATH 环境变量中指定的目录。
