# <a name="compiler-error-c2163"></a>编译器错误 C2163
function： 不可用作内部函数  
  
 `intrinsic`或`function`杂注列出在内部形式中不可用的函数。 例如，某些内部函数并非编译使用 /clr 编程的程序时。
